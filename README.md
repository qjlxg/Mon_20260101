随着人工智能逐渐成为科技主导权竞争的新战场，投资者正权衡中美两国在这一领域的发展前景。在美国，几家科技巨头正投入数千亿美元建设数据中心和算力基础设施，竞相开发最先进的模型。相比之下，中国互联网公司投入规模较小，更侧重于将人工智能应用嵌入现有平台。
Pzena增持阿里巴巴和腾讯的部分资金来自减持三星电子和台积电，因为基金认为这两家公司的投资吸引力已不如从前。以三星为例，随着人工智能需求推动存储芯片价格上涨，相关投资逻辑比预期更早实现。
Pzena认为，中国互联网巨头的AI潜力还没有被市场充分定价，现在买入成本不高，而在AI发展的早期阶段，仍难以判断最终赢家和输家的阶段，投资这些大型平台公司是一种风险较低、回报潜力较高的策略。
彭博汇编的数据显示，Pzena的新兴市场价值型基金过去五年表现优于97%的同类基金，今年以来的表现也超过90%的同行。该基金的主要持股包括三星电子、台积电和阿里巴巴。
