英国健康数据泄露 一度在阿里巴巴网站上公开出售
英国一项50万人参与、涉及衰老和疾病相关研究的健康记录，一度在阿里巴巴网站上被公开出售。在英国政府要求下架数据后，阿里巴巴已将数据删除。
英国科技部长默里（Ian Murray）星期四（4月23日）向下议院议员发的声明中表示，数据库所有者英国生物样本库（UK Biobank）通知英国政府，上述数据曾被发布在阿里巴巴网站上的三个在线列表中。
英国生物样本库表示，受影响的旗下全球最大的健康、生活方式和生物数据资源库，的数据来自分享个人健康信息的志愿者，旨在追踪人们随着年龄增长的长期健康状况。
上述数据资源库收集的数据，包括磁共振成像（MRI）扫描和生物标志物等直接医疗信息，以及自我报告的生活方式信息。
默里表示，在英国政府要求下架数据后，阿里巴巴已将数据删除。他表示，数据中不包含参与者的姓名、地址或联系方式。
尽管如此，英国生物样本库此前曾警告，如果参与者自行发布家谱信息，仍可能被他人通过数据交叉比对识别出来。
阿里巴巴方面则未回应置评请求。
数据似乎并未遭到黑客攻击或窃取。默里表示，这些信息是由三家能访问英国生物样本库的机构研究人员泄露的，但他没有透露这三家机构的名称。

华盛顿——美国一名官员4月23日表示，美国将邀请俄罗斯总统普京出席在迈阿密举行的2026年G20峰会，但总统唐纳德·特朗普则表示怀疑普京会来。
作为2026年G20东道主，美国此举将显著缓解对普京的国际压力——自2022年入侵乌克兰以来，普京被西方多数国家孤立，但并未被特朗普排斥。白宫一位高级官员称，所有G20成员都将被邀请参加部长级会议和领导人峰会。峰会拟定于12月14–15日在迈阿密举行。俄罗斯副外长潘金对俄新社表示，俄方已获“最高层”邀请。
在接受记者询问时，特朗普说他不知道普京是否会来，并直言“我怀疑他会来”，但同时表示支持邀请，称若普京出席“可能很有帮助”。他还批评前总统奥巴马因2014年乌克兰局势将俄罗斯逐出当时的G8，认为普京因此受辱。去年8月，特朗普曾在阿拉斯加接待普京，为自入侵以来普京首次踏上西方土地。
自2025年重返白宫后，特朗普一直尝试改善与俄关系、推动结束乌克兰战争，曾承诺可在24小时内结束冲突，但迄今成效有限，尽管莫斯科与基辅曾多次会谈。特朗普多次将战争责任归咎于乌克兰总统泽连斯基不愿割让领土。
克里姆林宫发言人佩斯科夫表示，普京尚未决定是否出席。2013年国际刑事法院于2023年对普京发出逮捕令，令其避免前往可能会依法逮捕他的国家；美国并非国际刑事法院成员，且特朗普长期反对该法院。普京自2019年后未参加过G20峰会，原因先为新冠疫情，随后为对乌战争。

华盛顿消息——美国司法部内部监察机构4月23日表示，将调查司法部是否遵守《爱泼斯坦档案透明法》（Epstein Files Transparency Act）对公开相关档案的规定。
该法案于2025年11月通过，要求司法部在30天内公开与性犯罪嫌疑人杰弗里·爱泼斯坦相关的全部文件。爱泼斯坦为亿万富豪，2019年在狱中等待性贩运审判时死亡。司法部在一系列文件公开行动中，于1月底释放了约300万页材料，但随之遭到强烈批评。
国会两党议员指出，公开文件中存在不当删节，同时司法部一度公布了此前未公开受害者的姓名与照片，引发愤怒，这类敏感内容随后被移除。该法要求对受害者的姓名及其他可识别信息进行涂黑，但规定不得以“窘境、名誉受损或政治敏感”为由为与爱泼斯坦关系密切的权势人物提供保护。
司法部监察长办公室表示，此次调查将评估司法部识别、涂黑与发布相关记录的流程，并在完成后发布最终报告。此前2月，获准在司法部查阅完整未删节档案的议员亦曾投诉，公开版本仍存在不当遮蔽。自1月30日大规模资料释放后，若干知名人士因与爱泼斯坦的关联受到震荡，引发欧洲等地的刑事调查、逮捕与辞职。
