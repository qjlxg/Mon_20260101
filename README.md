16被告涉非法募资314亿余人民币 一人无期徒刑
中国有16人涉嫌非法集资，募资金额达到314亿余元（人民币，下同，58亿新元），其中一人被判无期徒刑，其余被告则监禁五年至15年不等。
综合新华社和央视新闻报道，长沙市中级人民法院星期一（4月27日）公开宣判这起集资诈骗、非法吸收公众存款罪一案，对其中一名被告刘必安以集资诈骗罪判处无期徒刑，剥夺政治权利终身，并处没收个人全部财产；对其余15名被告分别以集资诈骗罪、非法吸收公众存款罪判处五年至15年不等有期徒刑，并处罚金。
16名被告涉嫌自2014年8月起在不具备吸收公众存款资格的情况下，以中战华信集团及多家关联公司名义，通过虚构投资项目、公开虚假宣传等方式，以还本付息、许诺高额回报为诱饵，向社会公众募集资金314亿余元。
大部分集资款用于兑付到期本息、发放员工工资及提成、个人使用和挥霍等，造成集资参与人经济损失61亿余元。
案发后，办案机关追缴部分资金、不动产、机动车、购买的公司股权等资产。现追赃挽损工作仍在进行中。追缴到案的资产将在判决生效后依法执行，按集资额比例返还集资参与人。
法院认为，被告以非法占有为目的，使用诈骗方法非法集资，数额巨大，造成集资参与人财产损失，严重扰乱国家金融管理秩序，应依法严惩。

据路透社报道，中国商务部上星期五（4月24日）将七家欧盟企业列入出口管制管控名单，理由是相关企业“曾参与对台军售或与台勾连”。这是北京罕见地对与台湾相关的欧洲企业实施制裁。
台湾的武器装备来源主要来自美国，欧洲过去30年因担心触怒北京，未向台北出售战机等大型武器装备。
顾立雄星期一（4月27日）到立法院外交国防委员会备询时说，这并非中国大陆首次因涉台军售实施制裁措施。
他补充说：“不过，据我了解，此举不影响我们持续从相关的多元渠道持续获得武器装备”，但未进一步予以说明。
虽然欧洲多数国家为避免遭受北京反制，不愿与台湾开展防卫合作，但自2022年俄罗斯入侵乌克兰以来，中东欧部分国家对台湾伸出援手。这次入列制裁名单的七家欧盟企业，其中四家来自捷克。
美国主要军火商因对台军售，一再面对中国大陆的制裁，最近一次是去年12月，当时美国公布对台111亿美元（141亿新元）的军售案。

华盛顿希尔顿酒店——4月26日晚白宫通讯员晚宴发生枪击的地点——再次成为犯罪现场。该酒店也是1981年对时任总统罗纳德·里根发动刺杀未遂的地点，里根当时在酒店外被击中，重伤入院。
当晚总统唐纳德·特朗普曾在宴会厅台上，事发后被迅速护送离开，未受伤。1981年，里根在离开酒店走向加长车时被一颗.22口径子弹击中肋部，子弹穿破肺部，他被紧急送往乔治华盛顿大学医院并接受两小时手术，事后据称对第一夫人南希说“亲爱的，我忘了弯腰”。枪手约翰·H·辛克利还击中白宫新闻秘书詹姆斯·布雷迪，致其部分瘫痪，并伤及一名特勤和一名哥伦比亚特区警察。2000年，克林顿总统以布雷迪命名白宫新闻发布厅。
这家建于六十年前、外形“M”字的酒店常年承办白宫通讯员晚宴、第一夫人午宴和全国祈祷早餐等重要活动，位于杜邦圈社区，距白宫约一英里。1981年的枪击是半个世纪以来对总统的首次未遂刺杀（此前为1975年对福特总统的未遂枪击）。辛克利以精神错乱为由被判无罪，后在2022年解除多项法院限制。
联邦调查局称，4月26日案发后有嫌疑人被控制；特勤称事发地点靠近安检区域。晚宴服务继续进行；特朗普当地时间21:45返回白宫，22:34在詹姆斯·S·布雷迪新闻发布厅向记者简短说明，并表示“这是个危险的职业”。（来源：纽约时报）

华盛顿 — 英国国王查尔斯三世于4月27日开始对美国进行为期四天的国事访问。尽管两天前在一场有总统特朗普出席的晚宴发生枪击事件，行程仍照常进行，此次访问旨在纪念美英历史性纽带并配合美国建国250周年。

外交观察称查尔斯外事能力老练，预计在国会演讲中以含蓄方式触及“尴尬议题”。此外，已故美国富商杰弗里·爱泼斯坦丑闻及其与查尔斯弟弟安德鲁王子往来仍对访问构成潜在影响；安德鲁二月遭逮捕后仍在调查中，未被起诉，去年已被国王剥夺头衔。
行程安排严密可控；白宫表示，特朗普与查尔斯的椭圆形办公室会面仅允许摄影记者入内。
