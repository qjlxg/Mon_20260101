随着美国最高法院裁定撤销大范围关税，外界普遍认为中美贸易摩擦在两国领导人4月31日至4月2日（注：原文为3月31日至4月2日）北京峰会前，正从升级走向谨慎稳定。
该裁决降低了实际税率，使中国对美商品的有效关税从去年釜山会晤后的约20%降至约15%，与其他贸易伙伴趋于一致。分析人士称，这一法律逆转提升了中国的谈判地位，同时中国在稀土等关键矿产上的筹码以及更有底气的反制手段，也令局势更具平衡性。
与此同时，美国正在从遏制转向维护稳定，强调关键物资供应可靠性，双方更可能在峰会上延长一项脆弱的休战，而非达成重大让步。
曾任美国贸易代表办公室对华高级贸易代表的萨拉·舒曼在华盛顿国际贸易协会年会上表示：“现在中国与其他贸易伙伴基本处于更平等的竞争环境。这会令政府担忧，不排除其试图通过其他方式将中国的关税水平再拉高，以与主要贸易伙伴区分开来。”
