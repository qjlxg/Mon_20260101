国民党评台美贸易协定：15%不是胜利而是沉重代价
台湾在野的国民党认为，台美对等贸易协定涉及的层面远超关税协商范畴，15%的税率“不是胜利，而是沉重代价”，导致食安防线松动与劳工、农民权益的长期隐忧。国民党称将在国会对协定内容启动严格的监督机制。
台湾星期五（2月13日）宣布与美国签署“对等贸易协定”，将下调关税至15%、扩大美国商品准入，并向美国能源与科技项目投入数十亿美元（10亿美元约等于12.6亿新元）。
国民党同日发布声明称，协定相关内容表面上谈的是关税，实际上牵动的却是攸关食安民生以及劳工权益的问题。协定涉及层面横跨医疗、食安、能源、劳动与产业政策，已远超单纯关税协商范畴。
国民党认为，协议并非政府所宣称的“重大突破”，而是一项可能让民众在不知情下承担风险的安排，并批评执政的民进党政府急于将协定包装为外交与经贸成果，而对可能衍生的食安疑虑与长期冲击说明不足。
国民党因而表示，将在国会启动严格的监督机制，严格把关相关法规与执行细节，确保台湾人的餐桌安全与孩子校园饮食健康不被牺牲，坚决保障受台美贸易协定冲击重创而失业的劳工权益。
在星期五发布的新闻稿中，国民党要求政府关注受协定冲击而可能失业的劳工权益，尤其是汽车及零组件业、农业与畜产、屠宰业、医疗保健业等相关产业；同时强调保障劳工与农民利益，必须与经贸开放同步推行，避免贸易协定造成产业、劳动与民生的二次伤害。
基于以上关切，国民党针对协定具体内容提出十项“重大疑虑”，要求台湾行政部门逐一解释。这些疑虑包括台湾卫福部食药署针对新药和医疗器材上市的审查权会否遭到架空、数百亿美元的能源采购与净零目标的政策矛盾、为美农产品提供“优惠市场准入”对本土食品表示和审查制度的影响等。

【美媒：美方已下令向中东派遣第二艘航空母舰】金十数据2月13日讯，据美联社援引知情人士报道，美国最大型的航空母舰——“福特”号航空母舰已被下令从加勒比海驶往中东海域，此举正值特朗普总统考虑对伊朗采取可能的军事行动之际。这是美国在中东地区部署的第二艘航母，此前，美国已在中东部署“亚伯拉罕·林肯”号航空母舰。第二艘航母的部署反映了华盛顿在伊朗核问题等紧张局势中加大军事压力的趋势，标志着“福特”号的快速调动，此前该航母曾被调往加勒比海参与针对委内瑞拉的军事行动。

【英媒：特朗普拟缩减部分钢铁和铝制品关税 以缓解“可负担性危机”】金十数据2月13日讯，据英国金融时报报道，特朗普计划缩减部分钢铁和铝制品关税，因为他正面临一场“可负担性危机”，而这场危机已削弱了他在11月中期选举前的支持率。去年夏天，美国总统对钢铁和铝制品征收了高达50%的关税，并将这些税收扩大到包括洗衣机、烤箱在内的多种金属制品。但据三位知情人士透露，特朗普政府正在审查受关税影响的产品清单，并计划对部分商品予以豁免，停止扩大征税清单，而是对特定商品发起更有针对性的国家安全调查。知情人士称，美国商务部和美国贸易代表办公室的贸易官员认为，这些关税通过提高派盘、食品和饮料罐等商品价格，正在伤害消费者。知情人士称，包括英国、墨西哥、加拿大以及欧盟成员国在内的国家，可能从美国对钢铁和铝制品关税的放松中受益。
