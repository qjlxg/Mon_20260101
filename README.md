【Kpler：遭无人机袭击后，俄炼油量降至21年来最低水平】金十数据7月14日讯，大宗商品数据与分析公司Kpler周二表示，乌克兰无人机袭击造成的破坏已将俄罗斯的炼油量推低至21年来的最低水平，为每天约380万桶。Kpler预计，由于持续的无人机袭击和不断的维修工作限制了炼油厂的运营，俄罗斯炼油厂的开工率在第三季度将保持低迷，仅会出现有限的回升。Kpler表示，袭击造成的累积影响导致炼油厂停工数量急剧增加。截至7月中旬，处于停工状态或受袭击影响的产能约为每天430万桶，约占俄罗斯炼油总产能的58%；其中，估计有每天150万至200万桶的加工能力实际上处于停运状态。Kpler预计，随着检修活动的减少，8月份炼油开工率将小幅回升至每天约430万桶，尽管持续的无人机袭击意味着下行风险依然很大。

中汽协：“新能源车平均车龄仅1.8年”等报道相关内容引市场误读
中国汽车工业协会后市场分会星期二（7月14日）针对“新能源车平均车龄仅1.8年、消费者3-5年置换电车”等报道称，相关内容引发市场误读。
后市场分会在中汽协官网发声明称，上述报道是引用该分会发布的《2025中国汽车后市场年度发展报告》中部分数据发布的。
分会进一步说道，网传报道推导得出的“电车快速迭代更换” 结论失真，属于统计概念误用。
分会也说，报告显示，传统动力类型乘用车在用车平均车龄约8.2年，新能源乘用车普及时间短、近年销量高速增长、新增上牌车辆占比高，因此整体存量新能源乘用车平均车龄约1.8年。“该数值仅反映的是行业转型发展现阶段特征，与当前市场情况一致，符合客观事实和现实体感，不代表车主仅1.8年就更换车辆，也不代表车辆使用寿命为平均1.8年。”
此外，分会也说：“汽车行业统计和调研数据具有特定的专业口径和适用范围，对于本报告中数据标注解释不够到位所引发的误读，我分会特此说明并深表歉意。”

美国移民执法人员枪击移民惹议
（比迪福德综合电）美国移民执法人员星期一在缅因州开枪打死一名26岁男子，这是不到一周内第二起涉及联邦执法人员的致命枪击事件。
一名26岁哥伦比亚男子星期一（7月13日）在缅因州比迪福德（Biddeford）遭移民执法人员开枪击毙，预计将进一步加剧外界对总统特朗普大规模驱逐非法移民政策，以及美国移民与海关执法局（ICE）执法方式的批评。
ICE发言人说，事件发生在当地时间约早上7时（新加坡时间晚上7时）。执法人员当时在一名已收到遣返令人士的最后住址附近展开监视，并试图拦截一辆驶离现场的汽车。
发言人称，有关车辆企图驾车逃离，执法人员“基于公共安全考虑”开枪，司机中弹后伤重不治。
缅因州参议员安格斯·金早前说，死者是因移民身份而成为ICE的逮捕对象，但他后来证实，死者其实并非ICE这次逮捕令的目标。他要求对此案展开全面、透明且公开的调查，并称涉案执法人员当时似乎没有佩戴随身摄像机。
据缅因州移民权益组织透露，死者获准在美国合法工作，并批评事件“令人震惊且不可接受”。案发后，当地约200名示威者上街抗议，高喊“ICE滚出缅因州”等口号。
上周，一名墨西哥男子在得州一次车辆拦截行动中，被移民执法人员开枪打死。

在股指持续上涨、人工智能和中东战争相关市场波动持续之际，高盛集团股票交易业务收入刷新其自身创出的华尔街纪录，达到74.2亿美元。高盛第二季度业绩显示，其股票业务连续第三个季度创下历史最高纪录。仅过去三个月的收入，就超过该业务2019年全年四个季度的总和。高盛周二在声明中表示，股票业务收入同比增长72%，融资以及对安排押注获利共同推动了这一增长。利率交易部门在第一季度表现令人失望后也超过预期；投资银行部门在并购咨询和承销业务推动下，录得2021年以来最高费用收入。高盛公布的利率交易收入为45.9亿美元。投行业务费用收入总计34亿美元，超过彭博汇总的分析师共识预估。高盛股票交易收入再创新高，证实了彭博新闻社6月的报道。这一纪录意味着高盛当季表现极为强劲，不过摩根大通的股票交易员录得更大增幅。摩根大通周二早些时候公布，其股票交易收入增长86%，至60.3亿美元。高盛首席执行官苏德巍6月表示，市场中“贪婪多于恐惧”。高盛投行团队在第二季度牵头了SpaceX创纪录的首次公开募股以及Alphabet Inc.的股票融资，在排行榜上大幅领先同行。该行股票承销业务收入较上年同期跃升130%。数据显示，高盛在并购市场的份额超过三分之一，今年已为1万亿美元交易提供咨询，达到任何银行有记录以来最快速度。高盛资产管理部门也在快速增长。该部门第二季度所管理资产规模为40.4亿美元，较上年同期增加逾7亿美元。业绩还显示，高盛净营收达到创纪录的203亿美元。支出也有所上升，但员工薪酬增速低于收入增速。收入同比增长39%，薪酬增长30%。
