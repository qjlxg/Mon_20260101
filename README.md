报道也提到，围绕经济社会发展重点问题建言献策是无党派人士和党外知识分子的重要职责和突出优势。培训班围绕无党派人士工作、党外知识分子工作分别安排专题辅导；组织学员开展专题调研，针对如何形成优秀建言献策成果等进行专门指导。

菲律宾劳工与就业部（DOLE）已预先拨出12亿比索（P1.2B），用于“进一步强化并调整”针对弱势及受冲击劳动者的相关方案，此举与政府因中东冲突宣布的全国能源紧急状态相呼应。 
DOLE表示，此举配合政府推出的统合生活、产业、粮食及交通方案（UPLIFT），以确保国内能源供应稳定、维持经济活动并保障民生。资金将主要用于两大社会保护项目：TUPAD临时就业计划和DOLE综合生计计划（DILP），以支援可能因能源紧急状况而失业或被迫转业的劳动者。

DOLE并持续在全国举办月度职场博览，自2026年1月以来已举办144场，涵盖高中应届、替代教育体系毕业生及被迫停工的建筑工人等群体；并将召集国家三方产业和平委员会，拟定更多应对策略与行动方案。

鉴于近期媒体报道涉及洛佩斯家族内部争端并提及 ABS-CBN，本董事会郑重声明：ABS-CBN 不是该案件的当事方。我们在此就法庭记录中涉及公司的两项指控作出澄清。
首先，关于“未解决的审计事项”一说，实情是没有审计发现，也就无从谈起“解决”。此项说法毫无根据。
其次，关于“拟注资将用于向若干高管支付款项”的指控，公司未发生此类支付，亦无此类计划，该指控同样站不住脚。
自 2020 年失去广播牌照以来，ABS-CBN 面临前所未有的挑战，但公司从未停止为延续经营而努力，也未停止服务公众。我们不希望公司和员工在与其无关的争端中被误传或被牵连。
董事会成员继续支持公司管理层履行为菲律宾人民服务的使命。
为菲律宾人民服务——今天、明天、永远。

阿布扎比发生三起火情，致6人受伤，包括印巴两国公民
阿布扎比当局称，位于哈利法经济区（KEZAD）附近接连发生三起由被拦截弹道导弹残骸引发的火灾。空防成功拦截导弹后，落下的碎片引燃多处火点。最初确认5名印度籍人员受轻至中度伤，随后又有一名巴基斯坦籍人员受伤，受伤总数增至6人。
阿布扎比媒体办公室表示，三处火灾已被控制，正在进行冷却作业。官方敦促公众关注权威通报，避免传播未核实信息。
当局此前连发多次安全警报，称已拦截来自伊朗方向发射的导弹与无人机。相关警报分别于凌晨3:26、4:05、6:24和下午16:25发布。国防部确认，各地听到的响声来自空防拦截弹道导弹，战机同时对巡航导弹和无人机实施拦截。官员提醒民众不要接近或搬动残骸，遇有相关情况及时向应急部门报告，局势正被密切监控，后续将持续通报。
应急期间注意事项：
• 以安全为先，不要在警报时拍摄或传播视频，听从指示；
• 驾驶人员安全驶离并在抵达后寻求掩护，切勿在路上停留；
• 留在安全场所或室内，直至官方发布“解除警报”；
• 仅在真实紧急情况拨打999，保持热线畅通；
• 仅转发官方核实的信息，避免传播谣言。

阿联酋国防部称，周四凌晨在连续发布多次安全警报后，阿联酋防空系统对来自伊朗的导弹和无人机进行了拦截。最新警报在上午6:24发布，确认正在应对导弹威胁，敦促民众就近避险并通过官方渠道获取最新信息。
这是自凌晨以来的第三次警报，前两次分别在凌晨3:26和4:05发布。国防部表示，国内多地听到的响声源于防空系统拦截弹道导弹，战斗机则对巡航导弹和无人机实施了拦截。
当局提醒公众保持警惕，不要接近、触碰或拍摄拦截残骸，相关事件应立即向应急部门报告，并由专业队伍处理。阿联酋表示将继续密切监控局势，未来数小时可能发布进一步警报或更新，以保障平民和关键基础设施安全。
