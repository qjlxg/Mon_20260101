马尼拉消息 — 菲律宾最高法院裁定，同居的同性伴侣如能证明实际出资，可依《家庭法》第148条被承认为共有财产共有人。
高院第二分庭由大法官Jhosep Lopez撰写判决，准许一名前伴侣的财产分割请求，确认其为奎松市一处房屋土地的共有人。两女在交往一年后共同购屋，因便于办理银行手续，产权登记在一人名下；分手后曾约定出售并平分款项，但产权登记者拒绝出售并否认对方为共有人。前伴侣曾在地籍上标注不利权利主张并以签署的“付款约定书”作为实际出资的证据向地方法院提起诉讼。
地方法院以缺乏出资证明驳回并判赔；上诉法院维持驳回但撤销赔偿。最高法院撤销前两级裁定，认定该签署的承认书为有约束力之承认，足以证明实际出资并确立共有权。
高院进一步阐明，《家庭法》第147条适用于可合法结婚的未婚伴侣，其同居期间取得之财产推定为共同财产；第148条适用于不得结婚者，只有能證明之实际出资才为共同财产。因现行法仅承认一男一女婚姻，同性伴侣应属第148条适用范围。
高院同时指出，涉及同性伴侣权利的政治、道德与文化问题需由国会等政治机关处理，法院无法独力解决全部政策问题。大法官Marvic Leonen与法官Amy Lazaro‑Javier在协同意见书中亦表示，第148条不应以性别区别，同性同居关系应受该条保护，不能在对待同性伴侣上出现不合理差别。
