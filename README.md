法官裁定特朗普政府大幅抬高H1B签证费违法
美国联邦法官裁定，特朗普政府将H-1B签证申请费用大幅提高至10万美元（约12万9000新元）的政策违法，认定此款项属于未经国会授权的税收。白宫表示将对裁决提起上诉。
彭博社报道，美国马萨诸塞州联邦地区法院法官索罗金星期一（6月8日）裁定，政府大幅提高这种热门签证成本的法令属于非法征税，必须予以撤销。这一裁决对特朗普限制移民、提振美国劳动力需求的举措无疑是一次打击。
裁决说，特朗普政府的“涨价”政策未经国会授权，国务院和美国公民及移民服务局不得执行这一政策。
裁决说：“从这笔10万美元款项的实质和用途来看，无论它叫什么，都是税款的一种。”裁决还说，联邦移民法赋予总统限制部分外国人入境的权力，但并未授权总统以此征税。
这起诉讼由20名民主党籍州检察长提起。原告认为，特朗普政府相关政策超越行政权力，将严重影响各州雇用高技能外国专业人才，对教育和医疗保健等关键公共部门造成特别不利的影响。
新华社报道，H-1B签证是美国的一种临时工作签证，属于非移民签证，有效期通常为三年，可延长至六年，原本的基本申请费不超过1000美元。这种签证允许美国公司雇佣外籍专业技术人员，以填补国内难以找到合适人才的职位空缺，主要被美国科技企业用于吸引高学历外籍人才。
美国总统特朗普2025年9月签署公告，将企业为新H-1B签证申请人支付的费用从数千美元提高至10万美元，引发震荡。同年10月，美国商会、美国大学协会以及美国汽车工人联合会、美国大学教授协会等数家机构针对特朗普政府提高H-1B签证费提起诉讼。同年12月，美国加利福尼亚州等20个州起诉特朗普政府，希望阻止后者提高H-1B签证费用。

陕西塌桥致62死亡失踪事故 调查称检测验收全造假
两年前中国陕西一启用不足六年的公路桥梁发生坍塌，导致62人死亡和失踪。官方最新通报指出工程有多项造假，包括施工单位不按图纸施工、第三方验收造假等。
2024年7月19日，陕西商洛市柞水县境内丹宁高速水阳段公路桥梁，因突发山洪发生垮塌，造成62人死亡及失踪。
本周一（6月8日），中国国务院安委会办公室通报安全生产领域部分弄虚作假典型案例，其中提到，这条公路桥梁工程背后存在严重的监理与检测舞弊。
调查评估发现，监理单位北京华宏工程咨询有限公司违规同意施工单位不按图纸施工将系梁抬升，不如实记录监理日志；实际情况与设计不符，却按原施工图进行验收，并在竣工前复测检验时违规签字同意。
第三方试验检测单位原陕西交建公路工程试验检测有限公司项目负责人发现事发桥梁实际桩长与设计不符，擅自修改桩长和原始超声波图形，公司发现后也未纠正并出具检测报告。
灾害发生后，涉事项目旁站监理、涉事第三方试验检测服务项目负责人涉嫌刑事犯罪，被司法机关采取强制措施；陕西省交通运输厅对北京华宏工程咨询有限公司、陕西交控工程技术有限公司（原陕西交建公路工程试验检测有限公司）以及相关责任人进行了行政处罚。
安委办表示，近年来，在多起事故灾害调查中发现，一些单位在勘察、设计、施工、监理、评价、检测、检验等多个环节存在弄虚作假问题，成为有关事故灾害发生的重要诱因，暴露出这些单位安全生产底线意识淡薄、责任落实虚化，性质严重、影响恶劣。

2025年，绝大多数北约盟国同意将核心国防支出提高到GDP的3.5%，并另拨1.5%用于国防相关开支，以满足特朗普提出的总计5%投入军费的要求。欧盟方面，欧盟委员会拟通过修订网络安全法禁止中国电信供应商进入通信网络，但德国与西班牙反对，担忧此举将引发北京报复。委员会已将华为与中兴列为“高风险供应商”。
德国近年多次讨论用公款资助电信运营商替换华为设备。北约领导人峰会将于七月在土耳其召开，资金议题将再度成为焦点。美国曾批评盟国将与国防关联不大的项目计入1.5%目标，驻北约大使马修·惠特克曾抨击盟国对此有“过于宽泛”的理解，意大利曾一度考虑把建造世界最长斜拉桥计入军费，后则放弃。

罗马，6月8日——意大利检方周一对以色列极右翼国家安全部长伊塔玛尔·本-吉夫（Itamar Ben-Gvir）展开调查，涉案疑点包括对上月加沙救援船队激进分子的待遇。司法人士证实，检方正就涉嫌对包括意大利公民在内的激进分子施加酷刑和绑架进行调查，若证据充分，检方可提出起诉请求。
对此调查，本-吉夫发表声明称：“我不会回避任何调查，将继续自豪地站在我们的战士一边。”此前他在社交媒体发布的一段视频显示，警方在国际水域截获援助船队后，拘押的激进分子被迫单膝下跪、双手反绑；视频中一名女子高呼“解放、解放巴勒斯坦”后被警员按倒在地。组织者称约有430名示威者被以方扣押，包含意大利和韩国公民。
意大利总理梅洛尼政府称此类对待“不可接受”，已召见以色列大使并请求欧盟讨论对本-吉夫的制裁；法国已决定禁止其入境。船队方面表示，目的是冲破对加沙的封锁并运送人道援助；尽管自2025年10月促成的美方斡旋停火承诺增加援助，但救援仍显不足。以色列则称其对加沙的海上封锁合法。来源：路透社
