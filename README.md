斯里兰卡中央银行行长韦拉辛格（Dr. Nandalal Weerasinghe）25日表示，尽管中东冲突持续，汇款并未减少，近期反而略有回升。韦拉辛格在中央银行举行的记者会上称，目前未见因冲突而自中东大规模遣返或回国的斯里兰卡劳工报告。
他同时承认入境游客有所减少，估计受中东局势影响，游客到访量下降约17%。中央银行昨日决议维持隔夜政策利率（OPR）在7.75%。委员会在考量国内外形势及中东冲突带来不确定性后作出决定。
中央银行指出，全球能源价格飙升和贸易中断已推动国内能源价格上调，但今年2月通胀仅为1.6%，相较5%的目标有回旋余地，预计通胀将在2026年第二季度回升至5%，此后维持在目标附近。
央行并提到，尽管受年末“迪特瓦”气旋影响，斯经济在2025年仍实现5.0%的强劲增长，2026年初显示出良好复苏态势；外部部门在2026年前两月保持稳健，出口、汇款和旅游收入增多，官方储备至2月底升至73亿美元，央行亦在市场购入大量外汇。但中东冲突通过能源、旅游、贸易和汇款渠道对外部前景构成风险，若冲突长期化将拖累国内经济。央行称将视情况采取适当政策，确保通胀回稳并支持经济发挥潜力。

锡兰石油公司（CPC）表示，目前该国燃油订购量约比实际需要多出10%，但只要避免紧急囤积，就不会发生燃料短缺。
CPC经理马尤拉·奈特提库马拉格（Dr. Mayura Neththikumarage）在Ada Derana“ Big Focus”节目中说，国内燃油储备充足，至少可满足到5月中旬的需求。他强调，目前的问题不是油品短缺，而是囤积和分配管理不当。
他指出，二维码系统可确保合理配给，防止普遍性短缺。举例称，一艘载运4万吨的油轮即可弥补相应缺口。过去柴油价格曾上涨150%，引发超量采购——本应取40却取到60。若囤购继续加剧，可能出现个别日子的供应紧张。
他还称，个别车辆少量超用不会形成大问题，但有人索取18张燃油券（正常应为3张）并宣称要用六个月，这类囤积不得允许。对普通消费者而言，若无大规模囤积，供应不会中断。
