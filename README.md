“大润发之父”台湾企业家尹衍樑逝世
被誉为“大润发之父”的台湾企业家、北京大学光华管理学院创始董事长尹衍樑星期二（5月26日）在台北逝世，享年76岁。
综合彭博社和澎湃新闻等报道，润泰集团发布声明称，尹衍樑星期二在台北荣民总医院去世，但未公布死因。
尹衍樑1950年出生于台北，长期担任润泰集团总裁。润泰集团由他的父亲尹书田于1943年在上海创立，1945年迁至台湾。尹书田1991年去世后，尹衍樑接掌集团，并将它从一家纺织企业发展为横跨房地产、金融、零售、生物科技及医疗保健等领域的大型商业帝国，成为台湾最大企业集团之一。
润泰集团旗下上市公司包括润泰创新国际、润泰全及润弘精密工程等，同时还掌控台湾第三大寿险公司南山人寿。根据彭博亿万富豪指数，尹衍樑去世时身家约97亿美元（123亿新元），主要财富来源即为南山人寿。
尹衍樑创立的大润发量贩体系于1997年在上海开设中国大陆首家门店，随后迅速扩张，一度成为中国大陆最大的外资零售企业之一，因此他被外界称为“大润发之父”。
北京大学光华管理学院也发布讣告悼念尹衍樑。讣告称，尹衍樑1989年创立光华教育基金会，率先在北京大学设立光华奖学金，并在1994年慨然捐资、鼎力襄助，推动前北京大学工商管理学院更名组建北京大学光华管理学院，并出任学院创始董事长，“开启中国商学教育的崭新篇章”。
讣告指出，尹衍樑毕生恪守“取之于社会，回馈于社会”的人生信条，持续助力中国多所高校学科建设与教育发展，创设享誉全球的唐奖，尊崇学术、赓续文脉、奖掖群英，为两岸教育交融、中华文化弘扬、全球学术进步作出了卓越且深远的贡献。
尹衍樑也是台湾知名发明家之一，在19个国家拥有超过650项专利。他在2012年创立唐奖，被视为“亚洲版诺贝尔奖”，表彰永续发展、生技医药、汉学及法治等领域的杰出成就。

中国4月工业利润创逾两年来最快增速

中国国家统计局星期三（5月27日）公布的数据显示，4月全国规模以上工业企业利润同比增长24.7%，明显高于3月的15.8%，创下逾两年半来最快增幅。
今年前四个月，工业企业利润同比增长18.2%，高于第一季度15.5%的增幅。
受全球人工智能投资热潮，以及买家担忧美以与伊朗冲突可能进一步推高投入成本、提前采购等因素推动，中国多个行业上月出口增长加快。不过，强劲出口表现仍不足以抵消国内需求持续疲弱的影响，这一趋势在汽车行业尤为明显。
中国最大电动车制造商比亚迪第一季度净利润同比下滑55.4%，创2020年以来最大跌幅，尽管海外销量占总销量比重创下新高。
与此同时，被视为比亚迪新兴竞争对手的零跑汽车，今年第一季度营收创历史同期新高，主要受销量增长及出口大幅增加带动，但净亏损较去年同期进一步扩大。

中区加州联邦检察官第一助理比尔·埃赛利（Bill Essayli）表示：“大学有义务为所有学生维护安全、包容的校园环境。对反犹行为反复不加防护的高校将被追究责任。”
UCLA尚未对本案置评。司法部在提起诉讼前不久，一联邦上诉法院于5月26日部分维持一项禁令，要求恢复加州大学在2025年被特朗普政府以支持“多样性、公平与包容”（DEI）为由撤销的拨款。
这是特朗普政府自2023年10月7日哈马斯袭击以色列后，针对美国高校所谓反犹问题采取的一系列法律行动之一。2023年袭击导致逾1200人死亡，随后引发以色列—加沙冲突，迄今已造成数万人伤亡。冲突爆发后，美国及欧洲多地数百名学生举行支持停火的示威，部分人士占领校园建筑。 路透社
