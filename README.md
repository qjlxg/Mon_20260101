【股价盘中触及跌停，业绩扭亏受益库存白银上涨，福达合金回应】
6月9日，福达合金盘中触及跌停。截至收盘，公司股价报89.2元/股，跌0.89%。消息面上，公司昨晚公告提示风险。对此，福达合金方面独家回应中国证券报记者，股价受多重因素影响。对于市场关心的白银价格对业绩波动的影响，福达合金方面回应记者，白银价格波动对利润确实有影响，但影响不大。公司的定价模式是银的市场价再加上加工费。因此，客户下单的时候，公司会同步向供应商下单，是随行就市的购买。因此，风险敞口已经被承担了一大部分，只有剩余的风险敞口会影响利润。“随着银价回调，公司也会战略性地囤一部分货，但整体规模不大，因为白银毕竟是贵金属，多囤占用资金比较大。”福达合金方面告诉记者。(中证金牛座)

香港公务员统一加薪2% 低于委员会建议
香港公务员表现长期遭社会诟病，港府星期二（6月9日）宣布全体公务员今年只加薪2%，并计划在10月推行新措施，表现最差的部分公务员将不获加薪。
香港公务员薪酬架构设有两个独立且不同的机制。一是当局每年因应经济、通胀和私人市场等因素，调整十多万名公务员薪酬。二是设立增薪点制度，未达顶薪点的公务员，当年即使冻薪，也会自动提高薪酬。
港府辖下的薪酬趋势调查委员会早前建议，高、中、低层公务员今年分别加薪4.12%、2.64%及1.17%。但港府最终决定，今年公务员统一加薪2%，生效日期追溯至今年4月1日。总公帑开支涉及60亿港元（9.84亿新元）。
当局日前也向立法会提交文件，表示拟于今年10月推行六级评核制度优化计划，实施“指导性分布机制”，公务员表现被评为第4至6级、即最差的5%至10%公务员将不获增薪点。
港府公务员事务局局长杨何蓓茵星期二说，今年公务员薪酬涨幅没有跟从薪酬趋势调查委员会建议，是因为香港未来发展仍需要庞大财务承担，且地缘政治变化可以在短时间内影响民生。
香港去年年底发生大埔宏福苑大火，揭示政府多个部门失职。杨何蓓茵被问到这起事件对公务员加薪的影响时，重申当局作出决定前会考虑六大因素，很难机械划分每项因素占比有多大。
香港特首李家超则指出，10月推行新的公务员评核制度，是希望提高政府治理，透过有效制度区分公务员表现，令市民更支持热心工作的优秀公务员；同时分辨出工作欠佳的公务员，要求他们改进。
不过，香港警察队员佐级协会主席梁俊杰日前引述多名警务人员意见指出，有人担心新制度会衍生“马房文化”及“擦鞋文化”，将原本针对工作能力的评核扭曲为对员工的人脉关系考验，甚至动摇公务员团队经年累月建立的廉洁文化，认为当局有责任设立机制严防有人以权谋私。
香港大专院校讲师樊智伟接受《联合早报》访问时分析，新的公务员表现评核计划打破以往自动加薪文化，符合大众期望。“香港公务员表现无论好坏，皆能自动加薪，一直为人所诟病；再加上宏福苑火灾事件带来的公众观感，若引入六级评核制，并将增薪点与绩效挂钩，可以提高市民对公务员问责性及工作效率的期望。” 
樊智伟同时指出，强制比例可能让部门领导为“凑人数”而将合格员工评为不合格，或衍生“办公室政治”，且很多专业工作难以简单量化和衡量，如何平衡问责机制与公务员整体团队士气，是政府需要仔细考量的事。

德黑兰表示，若不将黎巴嫩纳入停火协议，其与华盛顿的谈判将受影响。伊朗议会议长兼首席谈判代表盖里巴夫称，美国基地和以色列目标因“在黎巴嫩违约等敌对行为”可被视为合法打击对象。  
美伊双方称已接近一项旨在结束冲突的初步协议，涉及重新开放霍尔木兹海峡等内容；同时华盛顿考虑动用被冻结的伊朗资产补偿海湾国家，德黑兰警告任何转动资产为非法并将采取应对措施。以色列总理内塔尼亚胡因在2026年选举前就黎巴嫩停火作出的决策遭到国内政敌批评。

OPEC+于6月7日决定自7月起每日增产18.8万桶，旨在应对因霍尔木兹海峡实质封锁引发的供应紧张局面。美国与伊朗尚未就停火并彻底重开海峡达成协议，使此次增产在很大程度上具有象征性。
财团在声明中表示将继续密切监测市场并采取谨慎做法。此次远程会议的成员包括阿尔及利亚、伊拉克、哈萨克斯坦、科威特、阿曼、俄罗斯和沙特等国。
自2月28日美以对伊朗发动打击、伊朗以关闭海峡报复以来，中东冲突推动油气价格飙升并加剧通胀担忧。封锁前，霍尔木兹海峡曾输送约五分之一的全球能源，封锁导致沙特、伊拉克、阿联酋和科威特被迫减产；阿联酋5月退出OPEC也冲击了以沙特为首的产油联盟。
华盛顿研究机构ClearView能源合伙公司全球油气主管Jacques Rousseau指出，此次增产“意义不大”，因为受阻通道使大量增产原油无处可去。他并警告，海峡关闭越久，补充全球库存所需时间越长，油价可能持续高位。来源：纽约时报

华盛顿消息——据知情人士透露，美国特朗普政府敦促北约盟国将国防经费用于拆除并替换中国华为在网络与关键基础设施中的设备。
美国长期将华为及其他中国厂商视为国家安全风险并将其排除于本国网络，国务院对华事务协调员约书亚·杨（Joshua Young）在五月布鲁塞尔会见官员时，建议将北约用于“国防相关”1.5个百分点的开支用于替换华为设备。发言并未点名具体国家，但据一位人士指涉意在劝说德国。国务院未予置评。
2025年，绝大多数北约盟国同意将核心国防支出提高到GDP的3.5%，并另拨1.5%用于国防相关开支，以满足特朗普提出的总计5%投入军费的要求。欧盟方面，欧盟委员会拟通过修订网络安全法禁止中国电信供应商进入通信网络，但德国与西班牙反对，担忧此举将引发北京报复。委员会已将华为与中兴列为“高风险供应商”。
德国近年多次讨论用公款资助电信运营商替换华为设备。北约领导人峰会将于七月在土耳其召开，资金议题将再度成为焦点。美国曾批评盟国将与国防关联不大的项目计入1.5%目标，驻北约大使马修·惠特克曾抨击盟国对此有“过于宽泛”的理解，意大利曾一度考虑把建造世界最长斜拉桥计入军费，后则放弃。

罗马，6月8日——意大利检方周一对以色列极右翼国家安全部长伊塔玛尔·本-吉夫（Itamar Ben-Gvir）展开调查，涉案疑点包括对上月加沙救援船队激进分子的待遇。司法人士证实，检方正就涉嫌对包括意大利公民在内的激进分子施加酷刑和绑架进行调查，若证据充分，检方可提出起诉请求。
对此调查，本-吉夫发表声明称：“我不会回避任何调查，将继续自豪地站在我们的战士一边。”此前他在社交媒体发布的一段视频显示，警方在国际水域截获援助船队后，拘押的激进分子被迫单膝下跪、双手反绑；视频中一名女子高呼“解放、解放巴勒斯坦”后被警员按倒在地。组织者称约有430名示威者被以方扣押，包含意大利和韩国公民。
意大利总理梅洛尼政府称此类对待“不可接受”，已召见以色列大使并请求欧盟讨论对本-吉夫的制裁；法国已决定禁止其入境。船队方面表示，目的是冲破对加沙的封锁并运送人道援助；尽管自2025年10月促成的美方斡旋停火承诺增加援助，但救援仍显不足。以色列则称其对加沙的海上封锁合法。来源：路透社
