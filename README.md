目前泰国家庭负债已达GDP约87%，旧债多来自疫情期，近期因能源价格上涨、生活成本上升而更显吃紧。
由民间网络与公平金融组织（FFT）推动的新法，曾因两次议会解散而中止，再次成为新政府议程焦点。与1940年以大额企业（逾1000万铢）为对象的旧法不同，新法允许个人债务人在债务10万至100万铢区间内申请自愿重整；法院受理后启动暂缓执行（Automatic Stay），债权人暂不得提告或强制执行，债务人可在3–5年内按实际收入还款并保留住所与生计工具，未还清部分得以豁免。
在FFT座谈会上，主持重整草案审议的国会特委会主席塔威指出，债务结构多由政策造成，学生贷款借款人逾700万，强制执行案件高达25万亿铢，超出GDP。该法能在不动用公款下助债务人自救，并可保留公务员职务，改善还款优先从本金计入的做法。
FFT研究主管沙瑞妮称，现有救助不足，自愿重整可提高债权人回收率并帮助债务人复生；当前由油价带来的冲击或较疫情更难预估，法案亦可培育“债务管理员”。
泰国银行公会代表帕西特表示，国际趋势倾向“给予重生”（Fresh Start），虽对担保人免责的条款引发金融机构担忧，但银行总体认同赋予51%债权人通过重整计划以提高程序效率。
消费者基金会副主任纳罗蒙披露，过去十年消费者申诉逾4,811件，信用卡占33%，被起诉者逾千件，常见不公如小额债务导致高额房产被没收、利息与费用暴涨、购车分期与非法借贷案件增加。
