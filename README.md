中国限制关键矿产出口 对日本企业带来冲击
（东京路透电）中国限制关键矿产出口引发的供应短缺开始波及日本整体经济，让日本政府更迫切地寻求替代方案。
中国海关上月公布的数据显示，自去年11月至今年5月，中国未向日本出口铽（terbium）或氧化镝（dysprosium oxide）；自去年12月以来，氧化钇（yttrium oxide）的出口量也微乎其微，切断制造高性能磁铁所需的原料供应。
在5月和6月近200份提及稀土的日本企业文件中，超过三分之二称中国的出口管制已对业务造成负面影响，或未来可能带来冲击。
瑞穗研究技术首席经济师东深泽武史指出，在2010年中国实施贸易限制期间，日本经济曾损失约相当于国内生产总值（GDP）的0.9%，而稀土如今已成为供应链中不可或缺的一环，这次中国禁令造成的冲击可能更大。
乐天证券大宗商品分析师吉田悟认为，各行业受影响程度不一，主要是各公司的稀土储备量不同。“供应受到限制，但每个人都开始使用稀土，这只会让稀土更加稀缺。”
日本政府对此采取一些相应措施，包括去年10月与美国签署一项框架协议，旨在协调关键矿产和稀土资源，同时讨论联合开发深海矿藏。然而，商业规模的生产仍需数年时间。 
日本也已启动一些稀土回收项目，同时寄望于七国集团6月达成的一项加强储备协调计划。
不过，Yuri Group咨询公司首席执行官亨伯说，这些项目何时投入运营和最终产量如何仍不明朗。“我认为长达一年的出口限制会造成很大的问题，而我们已经历四到五个月。”
他也认为，日本政府显然希望保持低调，以免引发恐慌，也避免让中国过早取得优势。
去年11月高市早苗将“台湾有事”列为日本的“存亡危机事态”后，中国今年1月起加强管制对日本出口两用物项。

        北京时间7月11日凌晨3点,西班牙和比利时将迎来本届世界杯的1/4决赛,争夺一张半决赛门票。
