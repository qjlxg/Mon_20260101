上海市公安局城市轨道和公交总队官方微博星期四（1月15日）消息，称近日“上海地铁有偿代坐”的相关帖文引起网民关注。经轨交公安调查，发布贴文的郭姓女子（19岁）和翁姓男子（24岁）是出于猎奇心理在网上发布相关内容，均没有实际成交。警方已对二人予以训诫。
据经视直播此前报道，二手交易平台闲鱼上有大量提供上海地铁“代坐”服务的帖子。发帖者多从这些线路的起点站附近上车，表示若有人购买服务，可将座位让出。

杨敏指出，闲鱼作为平台方负有监管责任，公众可向平台投诉，平台应采取措施如冻结账户等，以遏制此类交易。

截至新加坡时间星期四（15日）傍晚5时，白银挫跌2.06%至每安士89.50美元，但近六个月以来，白银价格仍上扬超过134%。在星期三，白银价格上涨4.1%至每安士90.52美元，创下新高。
美国总统特朗普在星期三发布声明指出，将通过谈判达成双边协议，来确保关键矿产的充足供应。 
据彭博社星期四报道，道明证券（TD Securities）大宗商品策略师加利（Daniel Ghali）指出，这一份声明缓解市场对于美国向白银、铂金和钯金提高关税的担忧，同时触发白银市场的套利活动。
此前，由于地缘政治局势不稳，加上全球供应链风险重重，投资者纷纷涌入黄金、白银、铜等贵金属期货市场。
此外，特朗普频频威胁美国联邦储备局的独立性，也使得更多资金流入贵金属市场。
至于黄金价格，则下滑0.62%至每安士4607.20美元；近六个月来看，金价则走高超过38%。

中国国家公务员局称，本次笔试广大考生认真遵守考试纪律，总体考试秩序良好。对于违规违纪人员，有关部门依照公务员法和公务员录用有关规定，作出了考试成绩为零分、取消考试资格、限制报考等处理，严肃考风考纪、确保公平公正。

调查：逾三成韩企预料未来五年裁撤在华业务
韩国调查报告显示，超过三成的韩国企业预计未来五年将裁撤在华业务。
据韩联社报道，韩国产业研究院北京分院星期三（1月14日）发布的2025年韩国企业在华经营环境状况调查结果显示，32.4%的韩企预计将在未来五年内撤离、转移或缩小在华业务规模。
具体数据显示，9.7%韩企将撤离中国市场，将在华业务迁移至其他地区的韩企占1.8%，20.9%韩企将缩小在华业务规模。未来五年将维持现有规模的韩企占48.6%，与2024年的49.2%基本持平；计划扩大在华业务的韩企比例则由13.8%上升至19.1%。
从行业来看，对在华发展前景持悲观态度的韩企主要集中在显示面板、手机与家电，以及批发零售与流通行业，挖掘机和造船行业相对看好未来五年在华业务发展前景。
上述韩企指出，考虑撤离或转移业务的主要原因，包括市场竞争加剧、中国生产成本上升以及接班困难等。迁移目的地以东南亚为主，占59%，返回韩国占17%。
这次年度调查由韩国产业研究院北京分院、大韩商工会议所北京代表处和中国韩国商会在2025年8月至11月联合进行，调查对象总计455家在华韩企。
韩国经济人协会去年11月发布的报告显示，韩国10大出口产业中有一半已在企业竞争力方面被中国赶超，预计五年后10大产业竞争力可能全线失守。
中国企业在钢铁、通用机械、二次电池、显示器、汽车和零部件五个领域领先于韩企；半导体、电子和电机、船舶、石油化学和石油产品、生物健康五个领域仍落后于韩企。
