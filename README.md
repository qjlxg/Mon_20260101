秘鲁总统私会中国商人 被国会罢免
上任仅四个月的秘鲁总统赫里，因卷入与一名中国商人进行未公开会面的丑闻而被国会罢免。
路透社报道，秘鲁近10年来因总统频繁更迭、国会与政府冲突加剧、高层腐败丑闻，陷入严重的政治动荡。
秘鲁国会星期二（17日）召开特别全体会议，以75票赞成、24票反对、3票弃权的表决结果，通过对总统赫里的弹劾动议。
代理国会主席罗斯皮廖西宣布星期三（18日）举行国会主席选举，进而确认接任总统职务的人选。
秘鲁国会共130个席位，按照法律规定，弹劾总统动议需要至少87名议员支持才能获得通过。不过，赫里是以国会主席身份接任总统职位，只需半数以上赞成即可同时解除其国会主席职务和总统的职务。

被指卖非法产品和成瘾性设计 欧盟调查希音
欧盟星期二（2月17日）针对中国跨境快时尚巨头希音（Shein）被指销售非法产品，以及可能令人成瘾的平台设计，对希音展开调查。
据路透社报道，欧盟是根据《数字服务法》展开调查。《数字服务法》要求在线平台采取更多措施，打击非法和有害内容。
法国去年11月敦促欧盟委员会严厉打击希音平台出现销售带有儿童特征的色情玩偶的情况。希音此后已停止在全球范围销售所有色情玩偶。
欧盟科技事务负责人维尔库宁（Henna Virkkunen）在声明中说：“《数字服务法》保障消费者的安全，保护他们的福祉，以及针对他们接触到的算法，赋予他们了解关于这些算法的信息权利。我们将评估希音是否遵守了这些规则和其责任。”
希音说，将继续与欧盟监管机构合作，并已投入大量资金加强对《数字服务法》的遵守力度，包括进行系统性风险评估和缓解框架，以及加强对年轻用户的保护。
公司称，“除了强化检测工具外，针对有年龄限制的产品，我们还加快推出额外安全措施”，包括年龄验证措施，以防止未成年人查看或购买有年龄限制的内容或产品。
欧盟委员会说，将调查希音在欧盟建立旨在限制销售非法产品的系统，包括潜在的儿童性虐待材料。
这项调查也将重点关注希音的成瘾性设计，包括通过积分或奖励鼓励用户进行互动，这可能对用户的身心健康产生负面影响。
欧盟也将针对希音用于推荐内容和产品的推荐系统的透明度，进行审查。

美国得克萨斯州对路由器制造商TP-Link提起诉讼，指控公司在营销网络设备时存在误导行为，并允许中国政府方面访问美国消费者的设备。
