随着独居人口激增，中国“孤独经济”正在快速崛起。一款名为Are You Dead?（中文名“Sileme”，全球品牌为Demumu）的打卡应用近期走红，短暂登顶中国内地及其他市场付费榜，随后被苹果公司从中国区App Store下架，但在境外仍可使用。该应用要求用户定时点击确认安全，若超过48小时未应答，会向预设紧急联系人发出警报。

消费端已现变化：麦当劳等连锁在中国推出隔断单人用餐位；独身就餐被认为推动了外卖市场繁荣。来自中国互联网络信息中心与中国饭店协会外卖委员会的数据显示，2024年外卖用户达5.45亿，市场规模约1.2万亿元，日均外卖消费约33亿元。
