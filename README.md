尼科西亚/5月23日 — 国际货币基金组织（IMF）在一份提交给欧盟财政部长的报告中警告称，未来15年欧盟各国将在国防、能源和养老金方面面临巨额开支，如果不采取行动，公共债务将走上不可持续道路。

部分共和党人反对该基金可能向涉1月6日国会冲击者或已被赦免者支付赔偿，北卡参议员汤姆·蒂利斯称此举“荒谬”。宾夕法尼亚众议员布莱恩·菲茨帕特里克与纽约民主党众议员汤姆·斯沃齐联手拟法案，禁止向该基金提交的索赔付款。内布拉斯加退休众议员唐·培根称该舞厅与救助基金已成为众议院共和党人的“毒丸”。
尽管怀疑者认为共和党长期对特朗普保持忠诚，不易真正决裂，但包括亚利桑那与田纳西等地的共和党议员已公开力挺总统。代表1月6名被告的律师彼得·蒂克廷则自信其客户仍能获赔。
民主党人借此将公众生活困境与总统奢侈开支及可能流向其盟友的巨额资助做对比。参议院民主党领袖舒默将共和党形容为为舞厅与所谓“私设基金”而陷入崩溃。国会复会后，双方可能通过为基金设立监督委员会标准、司法审查等“护栏”寻求折中；民主党已拟出多项修正案，包括禁止向攻击执法人员的1月6参与者付款、禁止动用纳税人资金及要求公开支付名单等。来源：路透社
