综合“长安街知事”微信公众号和纵览新闻报道，青海省数据局上星期三（1月7日）在青海省公共资源交易网发布《关于4名评标专家在评标过程中身体出现不适症状送医救治情况的通报》。
青海省数据局称，1月4日，在开展跨省远程项目评标过程中，随机抽取的四名评标专家“因送餐员送错地方致使送餐延时，出现低血糖、头晕等不适症状，无法继续履行评标职责，提出呼叫救护车进行医疗救助需求，四名专家就餐后称症状未有缓解，坚持要求120急救服务，拒绝继续评标”。
通报续称，交易场所工作人员呼叫救护车接离评标现场，送青海大学附属医院予以救治后自行离开医院。项目招标人重新补充抽取评标专家后完成评标。
青海省数据局强调，评标工作责任重、强度高，评标时长不确定，时常需加班加点延时评标，甚至隔夜评标，对评标专家身体健康状况要求较高，专家身体健康是确保评标活动顺利进行和评标质量的根本保证。《评标专家和综合评标专家库管理办法》规定“入库专家应当具备正常履行职责的身体条件”，《青海省评标专家和综合评标专家库管理办法》规定“入库专家须身体健康”。
数据局也说，本着对评标专家个人身体健康和生命安全负责，保障每一个招标项目评标活动顺利进行，避免评标期间发生人身意外，根据相关规定，省数据局研究决定，不再将上述四名专家聘任为省综合评标专家库评标专家，并调整出库。
数据局呼吁所有在库评标专家，务必高度重视自身健康管理，定期自行体检，若发现高血压、糖尿病、冠心病等疾病影响评标工作，不再具备正常履行评标职责的身体条件，无法继续胜任评标工作的，应及时向省数据局提出退出省综合评标专家库申请，依法依规办理相关手续。
数据局也说：“隐瞒或不申请依法依规退出省综合评标专家库，出现影响正常评标工作的身体健康状况，省数据局将及时解除聘任关系，并调整退出省综合评标专家库；评标时发生意外情况的，一切后果由评标专家个人自行承担。”
针对这份通报，网上出现不少声音，质疑事情呈现出的过程太过巧合。也有网友质疑，这个项目或是内定标，四名专家因此拒绝进行评标。
青海省数据局开标评标中心工作人员星期二（13日）告诉记者，“那天确实就是四位专家岁数有点高了，他们有点低血糖了，就难受得不行。”对于吃饭时间晚了多久，工作人员回应“晚了不到半小时”。
