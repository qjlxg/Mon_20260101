全球最大路由器制造商之一的普联技术（TP-Link Systems）30年前在中国创立，虽已把总部迁到美国加利福尼亚州，仍持续面对特朗普政府发起的国家安全调查。

美国议员曾对中国制造的路由器提出安全隐患。众议院中国问题特别委员会主席穆勒纳尔对联邦通信委员会的这项禁令表示赞赏，称此举保护美国免受中国持续不断的网络攻击，并指中国制造的路由器应被排除在美国关键基础设施之外。

政府与媒体座谈中东冲突应对与对外沟通，称石油储备充足并拟发动全国性节能运动
（2016年3月23日）3月23日14时，政府府邸召开由总理办公室大臣苏帕玛斯·伊撒帕吉主持的媒体座谈会，邀请国家广播委员会、泰国电视台、记者协会、在线新闻生产者、主要电视台、数字平台及相关经济与安全部门，商讨中东战事信息发布与影响应对。
会议指出，中东地区冲突已带来三方面影响：一是能源，影响生活与企业成本；二是宏观经济与物价波动；三是海外泰人安全。内阁已把情势管理与对外沟通列为“国家紧急事项”，成立统一指挥的情势管理与追踪中心以提升协作效率。
在三项具体工作上，能源部表示正加速改善燃油配送、提升物流效率，并确认国家石油储备充足，同时筹划“全国节能”运动以缓解长期影响；商业部强调加强稽查、打击囤货与哄抬物价，维护民生必需品价格；外交部则全力协助在外泰民撤离与领事协调。
媒体提出多项建设性建议，包括建立油料储备实时报告机制以稳民心、公布民生必需品参考价以防止压榨消费者、推动切实可行的节能措施，以及实施更积极的危机期信息澄清策略。苏帕玛斯表示这些意见对政府决策与对外沟通非常有帮助，政府将予以整合并纳入后续措施。
