美NGO报告：低碳工业项目由中国主导 美国落后
美国一家非政府组织的最新报告显示，过去六个月已获得融资的低碳工业项目数量同比增加逾一倍，总值430亿美元（约550亿新元），其中大部分位于中国。
据路透社，美国非政府组织“使命可能伙伴关系”（Mission Possible Partnership，简称MPP）星期一（6月8日）发布报告称，从去年11月至今年4月的全球低碳工业投资项目中，有13个位于中国，涵盖甲醇、铝等领域；美国只有一个项目。这一时期共有19个低碳工业项目获得融资，高于一年前同期的八个。
报道指出，在美国和以色列袭击伊朗、推高化石燃料价格期间，绿色工业投资有所加速。
MPP首席执行官德拉萨尔（Faustine Delasalle）说：“在日益分化且不稳定的环境中，依赖化石燃料一次又一次显示出会带来价格冲击、供应中断和经济危机的风险。”
MPP是一家总部设在美国的非营利组织，致力于推动低排放工业发展。
报告显示，目前全球已宣布、处于不同推进阶段的低碳工业项目共有969个，涉及化学品、航空、水泥和金属等行业。其中中国占已宣布项目中的170个；包括印度和巴西在内的所谓“阳光地带”国家有318个，欧洲有211个，美国则有72个。

英国《每日电讯报》星期天（6月7日）引述知情人士说，美国白宫正在考虑一项绕开英国、直接与毛里求斯达成“收购”查戈斯群岛协议的方案。

路透社向英国外交部寻求置评，尚未获得回应。
一名美国官员告诉路透社，“特朗普总统坚持认为英国不应放弃英属印度洋领土，包括我们在迪戈加西亚岛（Diego Garcia）的美英联合军事设施。”
这名官员说：“迪戈加西亚岛位于印度洋战略位置，它对美国国家安全至关重要，是不可或缺的军事设施。”他还表示，美国将继续与英国保持定期磋商，以维护这个岛作为区域安全平台的作用。

2025年5月，英国和毛里求斯就查戈斯群岛主权移交问题签署协议。根据协议，迪戈加西亚军事基地将由毛里求斯租借给英国和美国。英国将以99年的租约继续控制迪戈加西亚岛上具有重要战略意义的军事基地，美国也能在这个基地进行军事行动。
今年1月，特朗普指英国把查戈斯群岛主权交还给毛里求斯“愚蠢至极”。英国政府于今年4月暂停移交查戈斯群岛主权的协议。英国媒体称，由于美国方面拒绝履行必要步骤，英国政府被迫搁置将查戈斯群岛主权移交给毛里求斯的相关法案。同月，毛里求斯外交、地区一体化和国际贸易部长兰富尔说，毛里求斯将“不遗余力”收回查戈斯群岛。

伊斯坦布尔讯——过去二十年国家大力投入，使土耳其成长为无人机及军工出口大国。土耳其现向近40个国家，主要为海湾、非洲、亚洲及部分欧洲国家，提供被视为更便宜、交付更快且更易定制的武器装备。路透社统计显示，自2021年以来，土耳其防务出口已超过三倍，至2025年达100亿美元，约占该国出口总额的3.7%，对欧美出口近四倍增至56亿美元。 
本国成熟的军工体系包括Baykar、土航（Turkish Aerospace Industries）、Arca Defense与Kale等企业；持续的国家支持、灵活供应链与为买家定制系统的能力，使其能迅速填补西方供应短缺或冗长采购周期留下的市场空白。土方防务机构表示，目标在两年内将防务出口翻倍，以获取偿债与研发资金。 
土耳其地处乌克兰与伊朗之间，安全利害相关，其在防空、战机与坦克发动机等领域的不足亦可通过贸易与技术合作弥补。哈德逊研究所资深研究员Can Kasapoglu称，土耳其通过出口先进系统，尤其是空中无人机，实现了“重大跨越”。 
土耳其约占全球武装无人机使用量的65%，也是弹药重要出口国，正规划或生产护卫舰、航母、防空系统与装甲车。印尼2025年拟购48架仍在研制的土耳其战机。五月在伊斯坦布尔展示的一款洲际弹道导弹原型及其宣传片引发外界质疑。 
土耳其盼借7月7–8日在安卡拉举行的北约峰会（并拟接待美国总统特朗普与其他领导人）扩大在西方市场的军售与联合生产，但在欧盟面临成员国专属防务项目与政治阻力——包括对土外交分歧与民主倒退的担忧，令其在欧安保行动中多被排除。尽管如此，土耳其仍与波兰、西班牙、葡萄牙、罗马尼亚达成供货协议，Baykar收购意大利Piaggio Aerospace并与莱昂纳多合作生产。 
斯德哥尔摩国际和平研究所指出，全球军费在2021–2025年增长24%至近2.9万亿美元，欧洲增幅达75%。同期土方防务机构将资助的研发项目增至1400余项。防务分析师Arda Mevlutoglu表示，如能克服政治障碍，欧洲与土耳其的更深层合作具备条件，但需高层政治意愿推动。路透社报道。
