印度飞行员要求继续调查空难 或因电气故障
印度飞行员联合会向印度民航部提交一封信函，指出去年印度航空空难的罪魁祸首可能是电气故障，而非飞行员操作失误。
法新社报道，去年6月12日，印度航空一架波音787-8梦想客机在起飞后不久坠毁，造成260人遇难。
根据国际法规定，印度航空事故调查局在事故发生一个月后的7月12日发布了初步报告。报告指出，飞机发动机的燃油供应在撞击前瞬间被切断，让外界质疑飞行员可能操作失误。
报告并未提及燃油开关的关闭是否可能是飞行员操作失误或故障造成的。
法新社看到的这份日期为5月1日的信函提供了一份“技术说明”，其中“暗示了一个可信的原因”，需要进一步调查。
信中写道：“起飞前的电气干扰可能导致意外的继电器操作和双发燃油切断，与飞行员无关。”
信中补充道：“媒体报道仍然暗示是飞行员的操作。然而，国际民用航空组织要求首先排除所有可信的技术原因。”
最终报告预计将于6月发布。
信中指出：“我们认为，这可以被视为一个假设，并通过详细的电气分析进行检验……在完成这项分析之前，不能排除任何技术原因。”

美情报评估显示：美伊战争对伊朗核计划破坏有限
（华盛顿路透电）美国情报评估显示，自去年夏天以来，伊朗制造核武器所需的时间并未发生变化。
路透社引述消息人士说，美国和以色列去年6月对伊朗发动12天战争之前，美国情报机构判断，伊朗可能在三到六个月内生产出足够制造核武器的高浓缩铀，并完成核弹制造。不过，伊朗的纳坦兹、福尔多和伊斯法罕核设施遭到袭击后，情报评估把时间表推迟约九个月至一年。
然而，美以今年2月底对伊朗发动新一轮战争期间，虽然袭击了伊朗的一些重要核设施，但对德黑兰核计划的整体评估仍未发生变化。这意味着，若要显著阻碍伊朗核计划，可能须销毁或移除伊朗剩余的高浓缩铀库存。
消息人士认为，对伊朗制造核武器所需时间的估计保持不变，部分反映了美以最新军事行动的重点。
虽然以色列3月下旬摧毁一座伊朗铀加工厂等核相关目标，但美国的打击主要集中在伊朗的常规军事能力、领导层及军工基础上。
一些分析人士认为，估计时间不变也可能是因为去年6月的军事行动之后，已经没有多少容易打击的伊朗重要核设施目标了。
曾领导伊朗核计划评估工作的美国前高级情报分析师布鲁尔指出，评估结果没有改变并不令人意外，因为近期美国的打击行动未优先考虑与核相关的目标。
他说：“据我们所知，伊朗仍然拥有全部核材料。这些材料可能储存在美国弹药无法穿透的深层地下设施中。”
近几周来，美国官员一直在考虑采取一些危险的行动，包括对伊朗进行地面突袭，以回收据信储存在伊斯法罕隧道群中的高浓缩铀，但派出地面部队的风险非常高。
不过，即使是全球最先进的情报机构，也难以精确评估伊朗实际的核能力。
多家美国情报机构分别对伊朗核计划进行研究，尽管总体存在共识，但仍可能出现不同的评估结果，也有可能伊朗的核能力实际受损情况，比目前情报评估的还要严重。
美国多次强调发动战争的目的是要阻止伊朗发展核武，而美国国务卿鲁比奥等官员认为，美方对伊朗防空系统的打击削弱伊朗保护核设施的能力，从而降低了核威胁。
此外，以色列暗杀伊朗核科学家也产生影响。联合国前核查员、现任科学与国际安全研究所所长奥尔布赖特说，这些行动给伊朗制造可正常运作核弹的能力带来不确定性。“知识无法被轰炸摧毁，但技术诀窍是可以被破坏的。”

西棕榈滩/迪拜 5月2日—美国总统唐纳德·特朗普表示，他已被告知一项与伊朗有关的“方案构想”，但正在等待具体措辞，同时警告若德黑兰“表现不当”，美方可能会恢复对伊朗的打击。

据路透社报道，伊朗官员称，将核问题留到最后谈判是为促成协议创造更有利氛围：在一项框架下，战事结束并获得美以不再攻击的保证，伊朗开放海峡、美国解除封锁，随后就核项目限制与制裁解除进行谈判，伊朗要求即便暂停核计划也保留和平浓缩铀的权利。

华盛顿——随着5月3日进入距11月中期选举最后六个月，控制国会的争夺进入关键阶段，这次选举可能重塑特朗普第二任期并改变华盛顿权力平衡。
中期选举一向被视为总统的公投，民主党希望借经济不满和特朗普约40%的低支持率收复众议院与参议院，共和党则力争在逆风中保住席位。
掌控国会不仅关系立法权，更决定能否对总统展开调查、阻挠提名等，影响其任期走向。早期迹象显示共和党面临挑战，通胀与伊朗战争相关开支等经济问题正侵蚀民众信心。
民调显示民主党在总体投票意向上略占优势，部分调查也显示选民在经济问题上对民主党信任度上升。亲共和团体AFP Action警告，参议院多数党面临风险，称存在结构性逆风和选民冷漠。
本次中期选举将改选全部435个众议院席位、约三分之一的100席参议院以及多数州长职位。民主党若要掌控两院，需在众院净胜三席、参院净胜四席。
民调机构Impact Research的莫莉·墨菲指出，当总统支持率在或低于40%时，更容易在传统偏共和的地区看到席位变动。
选情仍然流动：共和党受益于相对有利的参议院版图和经过严重划分的选区，但多州正在进行年中重划，包括德州、加州、北卡、俄亥俄、佛罗里达、密苏里、犹他和弗吉尼亚，最高法院限制基于种族的重划也改变格局。
参议院领袖舒默指责共和党通过过度选举安全和移民执法措施削弱投票权，称这是“操纵制度”的企图。共和党则依靠资金优势并强调移民与国家安全议题，认为最终取决于投票率与选民对其执政成果的认可。众议院多数党领袖斯卡利斯表示，选情将取决于选民投票热情和对共和党“扭转局面”成果的评价。

霍尔木兹海峡航运几乎停滞，尽管美国总统特朗普日前承诺将推动恢复通航。4日并未出现通航回升迹象，MarineTraffic数据显示，进入阿曼湾的仅有一艘受制裁的小型液化石油气油轮、几艘货船和一艘敷设电缆船。未见油轮排队等候过航，德国航运集团赫伯罗特（Hapag-Lloyd）表示，由于缺乏关于安全通行的明确程序，其船舶仍无法通行。 
美国中央司令部（CENTCOM）称将于4日开始协助恢复海峡通航，同时继续对伊朗港口实施封锁。航运业尚未收到关于美方行动意图和指引的说明，国际海运经纪与顾问协会（BIMCO）警告称，若未获伊朗同意，尚不清楚是否能削弱或压制对商船的威胁；BIMCO为业界提供安全警示。国际海事组织表示，数百艘商船和最多2万名海员因伊朗冲突无法通行该水道。 
美方主导的联合海事情报中心将海峡海上安全威胁评为“危急”，建议船舶考虑绕行阿曼领海（位于交通分离带以南）。CENTCOM将其行动描述为“防御性”，并称会将外交努力与军事协调相结合。伊朗则警告美军远离海峡，要求商船与其军方协调通行，并公布新控制区地图。 
巴基斯坦表示，上月被美军登船扣押的伊朗籍集装箱船Touska的22名船员已撤离至巴基斯坦并将遣返，船舶经修复后返还船东，巴方称此举为“增进互信”的措施。自4月13日以来美方在伊朗港口实施的海上封锁亦已导致德黑兰的石油出口收缩。路透社
