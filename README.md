据央视新闻客户端消息，由中国国家海洋环境监测中心等七家单位共同编制的《2025南海海洋生态环境质量状况报告》星期五（3月20日）发布。
报告基于2025年船舶定点监测、走航监测和卫星遥感监测结果，从海水环境质量、海面漂浮垃圾和微塑料、温室气体、岛礁生态环境等方面分析了南中国海海洋生态环境状况。

报告分析，南中国海海域在监测期间整体表现为大气二氧化碳的弱源。监测的南中国海主要岛礁生态环境状况稳定，植被覆盖面积较大的岛礁为东岛、美济礁（Mischief Reef）、永兴岛和渚碧礁（SubiReef）。

内贸司重申瓶装棕榈油暂未涨价，警告商家不得囤货哄抬价格
泰国内贸司司长威塔功·马尼内特表示，针对近期有关瓶装棕榈油将因部分生棕榈油用于政府生物柴油（B7/B10）政策而涨价的报道，部门已密切追踪产量、库存与价格；目前市售价格仍维持在每瓶42–50泰铢，厂商、批发与零售渠道均未确认调价。政府正从生产端到终端加强监控，防止借机牟利。
瓶装棕榈油为政府列管商品，内贸司作为价格委员会秘书处，有权督导并建议检讨管制措施，以兼顾生产成本、农民收益与民众生活负担。内贸司将尽快修订相关管控办法，确保价格监督与时俱进。
司长呼吁业者勿无故涨价或囤货，民众如发现哄抬、囤积或拒售，可拨打1569或向各省商务办举报，部门将依法严肃处理。
目前泰国生棕榈油（CPO）库存约35万吨，足以应付内需；预计4至7月每月将有不低于40万吨的新季产量入市，连同既有库存可满足家庭消费与能源用油需求，整体供应有望维持平衡并支撑棕榈果价回稳。

本周《Sustainable together》专栏随泰国PTT公司代表团赴日，考察东京燃气（Tokyo Gas）与北海道燃气（Hokkaido Gas）的能源设施，学习跨季节LNG（液化天然气）库存管理模式，并向IHI Corporation取经其高安全性储罐技术。
PTT国际贸易业务副总经理Jaturong Warawitsurawattana在参观位于北海道石狩市的Ishikari LNG Terminal时介绍，该终端由Hokkaido Gas运营，坐落于石狩湾新港的深水港与重要产业区，是北海道天然气供应与能源安全的关键基础设施。
该终端可接纳国际LNG运输船，通过装卸、储存与再气化（regasification）后并入管网，供家庭、商业、工业及发电用能。因北海道气候严寒且未与本州连通管网，地区高度依赖船运LNG，Ishikari遂成为重要的“能源储备库”。
PTT认为，利用泰国与日本季节性用能差异——日本冬季LNG需求高峰、泰国夏季用气用电高峰——可发展跨国库存管理与互借机制，在需求低谷期间互相存放LNG，提高储罐利用率（utilization），无需新增储运投资即可提升资产效率并加强能源安全。
