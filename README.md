关税政策被判违宪 特朗普誓言加征10%全球进口关税
美国总统特朗普对最高法院裁定他援引《国际紧急经济权力法》向全球征收关税违反宪法勃然大怒，他称一些大法官令他感到可耻，并宣布他将签署行政令，对全球商品加征10%的进口关税，以取代被最高法院认定违法的一些紧急关税。 
路透社报道，美国最高法院星期五（2月20日）以6比3的结果，裁定美国《国际紧急经济权力法》（IEEPA）没有授权总统征收大规模关税，此举超越总统职权，已构成违宪。这项裁决不仅重创特朗普执政的核心经济武器，更可能引发高达1750亿美元（约2217亿新元）的税款退还潮。
特朗普当天下午在白宫的临时新闻发布会说：“我对最高法院的某些成员感到可耻，绝对可耻，因为他们没有勇气为我们国家做正确的事。”其中两名由特朗普提名的大法官戈萨奇和巴雷特和保守派的首席大法官罗伯茨加入三名自由派大法官，裁定特朗普动用紧急权力法对世界各国加征关税，超越总统职权。
他称这一裁决令一些国家欢喜若狂，甚至“在街上跳舞”，并指责最高法院受到外国势力影响，但未提供证据。
特朗普同时宣布，他将援引1974年《贸易法》第122条对美国贸易伙伴征收10%的全面关税。特朗普是第一位援引《贸易法》第122条的美国总统。
特朗普还说，他将研究动用其他权力征收新一轮关税。如果他的计划奏效，最高法院20日的裁决对市场的长期影响可能有限。
由首席大法官罗伯茨撰写的多数判决书指出，特朗普援引1977年《国际紧急经济权力法》加征关税是不合法的。罗伯茨指总统必须得到国会的明确授权才能行使加税权，宪法规定这项权力属于国会，而非白宫。
特朗普说：“他们非常不爱国，对我们的宪法不忠。我认为最高法院受到了外国利益集团和一个远比人们想象中小得多的政治运动的影响。”
特朗普当天还在社交媒体发文，称美国所有以“国家安全”为由征收的关税，以及根据1962年《1贸易扩展法》第232条和1974年《贸易法》第301条征收的关税将继续有效。
另外，特朗普在记者会上承认，最高法院裁决将引发一场旷日持久的法律缠斗，关于联邦政府是否必须向美国企业退返数以十亿美元计的关税税款，可能“要打五年官司”。

分析：中国或因美高院裁决减少采购美大豆
分析认为，中国或因美国最高法院对总统特朗普关税政策的裁决减少采购美国大豆。
据路透社报道，在美国最高法院星期五（2月20日）表决通过，裁定特朗普对全球发起关税政策违宪之前，特朗普已持续表明中国将增加对美国大豆的采购。
特朗普2月4日在社媒Truth Social发文称，中国将额外采购800万吨美国大豆。在那之后，美国大豆的交易期指上扬8.49%，但多数大豆活跃交易期指在星期五上午微幅下滑。
报道引述美国商品经纪公司高级对冲顾问费斯勒（Darin Fessler）说，“特朗普一直尝试向中国施压增购美国大豆，而如今我们要问的是，这样的结果（美高院裁决）会否让中国减少接收美国大豆？”
费斯勒说：“美国大豆仍比巴西的稍微昂贵。中国若未被逼迫，为何要选择购买美国大豆？”
即便美国大豆交易期指持续上扬，但多数分析师和贸易商质疑中国将加大采购美国大豆。
中国去年持续数月停止采购美国大豆，直至10月中美元首韩国会晤达成贸易休战后，才恢复购买美大豆，迄今为止已采购了1200万吨。中国国有大豆买家中储粮集团也已在去年底重启大豆拍卖，为美国进口大豆腾出空间。
没有关税政策施压，美国大豆将难以与巴西大豆匹敌。目前正值巴西大豆盛产期，且其价格也远比美国大豆低廉。
美国最高法院星期五裁定总统特朗普动用紧急权力法单方面对世界各国加征关税，超越了总统职权，已构成违宪。这一裁决引申出疑问，特朗普政府会否或如何通过合法途径推进新的关税政策。 
对密切关注世界最大大豆进口商中国动向的市场交易商来说，上述裁决给已动荡的市场增添不确定性。
贸易商称，他们将密切追踪美国关税政策的变动，以及中国是否将根据特朗普的意愿购买美大豆，或是持续转向采购巴西和阿根廷大豆。
美国芝加哥农业资源公司总裁巴塞（Dan Basse）说，“关税大棒从总统手中被夺走，他还可用什么工具确保关税政策持续运作？这确实给这一议题增添阴霾。”

美国最高法院周五裁定，唐纳德·特朗普以《国际紧急经济权力法》（IEEPA）为依据大规模征收关税超越总统权限，阻断了他用关税推进经济议程的关键手段。法院以6比3裁定“IEEPA不授权总统征收关税”，首席大法官约翰·罗伯茨在意见书中指出该法案未提及关税或税率。三名自由派大法官加入三名保守派大法官形成多数；布雷特·卡瓦诺、克拉伦斯·托马斯和塞缪尔·阿利托表示反对。
去年重返总统后，特朗普曾以紧急经济权力对几乎所有贸易伙伴加征“对等”关税，并针对墨西哥、加拿大和中国就毒品走私与移民问题实施另行关税。下级贸易法院今年5月曾裁定这些普遍性关税越权并阻止其生效，政府上诉后最高法院维持该判决。法院同时强调，此裁决不影响此前针对钢铝等特定行业的部门性关税，数项相关调查仍在进行中。
