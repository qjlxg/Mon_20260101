中国多位顶尖AI科学家在北京清华大学主办的AGI-Next峰会上表示，尽管中国近年来在AI领域取得进展，但短期内很难赶超美国。
阿里巴巴Qwen团队技术负责人林君阳称，任何中国公司在未来3到5年内超越谷歌DeepMind或OpenAI的概率不到20%，这一估计已“极为乐观”。他指出，美国在计算资源上比中国多一到两个数量级，OpenAI等正投入大量算力进行下一代研究，而中国目前几乎把大部分算力都用于满足日常需求。
智谱AI（Z.ai）联合创始人兼首席科学家唐杰也表达了类似担忧。他指出，虽有第三方基准显示中国模型与美国领先模型差距缩小，但美国还有大量未公开的模型，差距可能并未真正收窄。
腾讯新任首席AI科学家、前OpenAI研究员姚顺宇则更为乐观，认为未来3到5年世界领先的AI公司或许会来自中国，理由是中国在电动汽车和制造等领域展现了快速规模化能力。但他也提到，中国需克服极紫外光刻机等先进芯片制造短板、企业级AI采纳不足以及基础研究投入有限等问题，并呼吁更多敢于冒险的创新精神。
唐杰补充说，90后和00后新一代研究者若在更优的创新环境与更明确的巨头与初创竞争格局下专注核心研究，可能推动突破。
