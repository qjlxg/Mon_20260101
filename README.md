综合香港01、星岛头条等媒体报道，火灾独立委员会首场听证会星期四上午10时开始，下午4时20分结束。委员会首席代表大律师杜淦堃表示，已收集“海量”文件，资料总量超越1TB，包括火灾当日的闭路电视片段、从未披露的政府楼宇工程监督工作内部记录等。

第四，一些阻燃棚网在大风预警后被更换为非阻燃棚网，且且居民窗户被发泡胶遮蔽，促使火灾蔓延至居民家中。
第五，工地吸烟。棚架上有大量烟头，居民多次投诉但没有得到正视。
杜淦堃表示，宏福苑火灾蔓延、造成巨大伤亡，就现阶段的证据来看，是多项因素重叠的后果。
由香港特区政府成立、高等法院原讼法庭法官陆启康任主席的独立委员会，星期四起至4月2日将举行八场听证会，通过证人的口述证供、书面陈述等厘清大火的相关事实，查找大火起因、维修工程是否涉及围标或不当合谋、监督有否疏忽等各方责任，并审视现有制度和法例等，在九个月内提交报告。

IMO称，目前约有约20,000名船员滞留在霍尔木兹海峡以西约3,200艘船舶上。根据UKMTO、IMO及伊拉克、伊朗当局资料与AFP汇总，自冲突爆发以来至少有21艘船遭袭、被锁定或报告遇袭。阿联酋提交文件称“超过18艘各国商船遭到弹片、导弹、无人艇及海底炸弹攻击，至少8名船员确认死亡、4人失踪”。 
英国、法国、德国及多国（含海湾国家）敦促IMO强烈谴责伊朗对邻国与民用航运基础设施的“不可原谅”袭击，并谴责其所谓封锁霍尔木兹海峡的行为。伊朗在提交文件中则将当前海上安全恶化归咎于以色列与美国的袭击，称海上后果是“直接且不可避免”的结果。 
日本、巴拿马、新加坡和阿联酋等国建议IMO推动建立框架，便于将商船和船员从高风险海域安全撤离。航运业团体则呼吁国际协调安保措施并关注船员福利，包括保持与家人通信、促进换班和上岸、以及保障足够物资供给。

马尼拉—菲律宾中央银行（BSP）周三表示，正在密切关注中东冲突升级，称这可能在4月23日政策会议前，对国内通胀与金融稳定构成风险。 
央行重申，维持物价稳定为首要任务，目前正评估油价飙升是否会通过化肥、运输费及一般消费品价格产生二次冲击。 
BSP同时关注冲突对经常账户的影响，尤其是贸易往来及派驻该地区的海外菲律宾劳工（OFW）汇款流动。 
针对近期汇率波动，央行重申采取弹性汇率制度，仅在出现“过度波动”时入市干预以维护市场秩序，而非守住比索的具体水平。彭博报道指，BSP本周早些时候在比索接近1美元兑60比索时曾出手干预——总统马科斯曾表示不希望比索突破该价位。 
央行表示“政策将以数据为依据”，暗示利率决定将取决于地缘政治对菲律宾经济的实际冲击。回顾货币政策：央行在2月将基准利率下调25个基点至4.25%，去年曾多次降息，但在2023年10月也曾一次性上调25个基点以抑制上升通胀与食品价格。行长伊莱·雷莫洛纳年初曾表示宽松周期接近尾声，而今年通胀在中东冲突推高油价前已呈回升迹象。 
此外，美联储在美以—伊紧张局势冲击油市与供应链、经济数据走弱之际，预计将维持利率不变。美联储为期两天的会议周二开幕，基准利率决定将于周三美东时间下午2点（菲律宾时间次日凌晨2点）公布。
