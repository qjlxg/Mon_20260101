中国2025年创万亿美元贸易顺差
尽管面临美国关税政策和持续的贸易施压，中国在2025年的出口表现依然强劲，全年实现创纪录的万亿美元级贸易顺差。
中国海关总署星期三（1月14日）公布的数据显示，中国2025年全年贸易顺差达1.189万亿美元，规模相当于全球前20大经济体之一沙特阿拉伯的国内生产总值（GDP）。
数据显示，以美元计价，中国12月出口同比增长6.6%，高于11月的5.9%，也显著超出路透社调查经济学家预期的3%。进口同比增长5.7%，较上月的1.9%明显回升，同样高于市场预期的0.9%。
从全年走势看，中国2025年月度出口顺差超过1000亿美元的月份多达七次，而2024年仅出现一次。这一表现部分受益于人民币贬值，也显示尽管美国关税政策在一定程度上抑制了对美出口，中国整体对全球市场的贸易能力并未受到明显削弱。
经济学家预计，中国今年仍将继续扩大在全球市场的份额。一方面，中国企业加快在海外布局生产基地，以更低关税进入美欧市场；另一方面，对低端晶片及其他电子产品的强劲需求，也将继续支撑出口增长。
在房地产长期低迷、内需复苏乏力的背景下，出口成为中国经济增长的重要支柱。不过，这一创纪录的贸易顺差也可能进一步加剧外部摩擦，部分经济体对中国的贸易行为、产能过剩以及对关键中国商品的高度依赖表示担忧。

马永火山活动增温，位于6公里永久危险区（PDZ）内的数千名农户被禁止下田种植，当地农业官员周三表示。 
阿尔拜省农业官员切里尔·雷贝塔（Cheryll Rebeta）称，因火山出现轻微降灰与落石，已有2261名种植稻米、玉米、剑麻、白菜、佛手瓜、茄子、番茄、四季豆及根茎类作物的农户被禁止进入农地。她并引述一名农民的话说：“那里很热，我们不希望土地被水泥化。” 
雷贝塔指出，因位于永久危险区，这些农田无法投保；区内家畜将被迁至危险区外的集中饲养点。她同时表示，农民常在火山平静时重返PDZ耕作，因为土壤肥沃、无需施肥，且有不少买家期待来自马永的农产品。 
菲律滨火山与地震研究所（Phivolcs）截至1月14日记录到2次火山地震、209次落石事件及46次火山碎屑密度流（uson）。 
马尼拉——社会福利与发展部（DSWD）周一表示，如马永活动持续数月，已备妥粮食与其他救援物资，目前仍有数百人滞留避难所。国家减灾委员会（NDRRMC）数据显示，受影响有1,131个家庭、逾4,000人，多数在阿尔拜的避难中心。

能源部对Leandro Leviste创办的Solar Philippines Power Projects Holdings开出240亿披索罚单，理由是未履行既定交付承诺。
能源部长Sharon Garin表示，能源部已取消共163份电力供应服务合约，合计1.7万兆瓦（MW）。其中Solar Philippines持有28份合约，合计1.2万MW，占能源部预估容量的60%。
Garin指出，公司将面临对政府的多项财务责任，且若不惩处恐导致其他业者效法，拿到合约却不交付。若供应商未在2025年12月如期交付，将导致电力储备吃紧、价格上扬，并可能使供需缺口持续到2029至2030年。
能源部同时以合约附带的履约保证金和年度发展费等為由计罚，并已与司法部、监察署及律师总办公室就此事展开协商。
Leviste于2013年创立Solar Philippines，曾在就任国会前将SP New Energy的164.4亿股以204.9亿披索出售给Meralco PowerGen。ABS-CBN已向Leviste及Solar Philippines征询回应。
另SP New Energy Corporation（SPNEC）向菲律宾证券交易所说明，公司并非新闻中所指之“Solar Philippines”，但确认Solar Philippines Power Project Holdings为其股东。SPNEC并称已接获其Sta. Rosa项目服务合约的终止通知，计划请求复议。

菲律宾国家警察（PNP）周三表示，将对一名警官在马尼拉涉嫌强奸一案展开彻底且公正的调查。
代理PNP首长何塞·梅倫西奧·納爾塔特茲中將（Jose Melencio Nartatez Jr.）表示，警方将全面配合正在进行并行行政调查的国家警察委员会（NAPOLCOM）。
受害者及其伴侣已被接待到位于克拉梅营（Camp Crame）的代理首长办公室，相关官员亲自听取了他们的陈述，并再次承诺对涉案PNP人员进行公正、不偏不倚的调查。
纳爾塔特茲在声明中说：“我们的首要任务是受害者的福祉。我们会进行彻底且公正的调查，查明真相、协助受害人，并确保伸张正义。”
据初步报告，事件疑于2025年12月上旬发生：嫌疑人为现役警官，邀受害人在聚会场所与友人相聚；受害人随后在社交场所失去意识，事后发现遭到性侵。
PNP强调对队伍内部不当行为零容忍，并表示将采取一切必要措施，协助受害者及其家属，同时维护调查完整性，确保涉案者依法追究责任。
