马尼拉 — 菲律宾人权委员会（CHR）敦促国会通过立法，取消对诽谤与网络诽谤的刑事追诉。CHR表示，这些条款长期侵蚀言论与新闻自由，并被用作对记者、活动家及普通公民的法律骚扰。 
CHR指出，《修订刑法典》第355条与2012年《反网络犯罪法》第4(c)(4)条已被“工具化”，用于压制批评与限制公共话语，并强调对表达的限制必须符合法律、必要性与相称性原则。 
委员会建议修法，取消对诽谤与网络诽谤的监禁刑，同时保留对已证实诽谤行为的民事赔偿，并建立相称的民事制裁制度（含赔偿金）作为刑事起诉的替代方案。CHR亦主张对法官、检察官和执法者进行以人权为本的相关培训，强化对媒体工作者、社会倡导者和人权捍卫者免受法律骚扰及报复性网络诽谤诉讼的保护。 
CHR并敦促第20届国会优先并整合现有法案：众议院有HB 1188（Brian Poe Llamanzares）、HB 441（Antonio Tinio、Renee Co）及HB 6187（Rufus Rodriguez、Maximo Rodriguez）；参议院则有SB 810（Loren Legarda）、SB 476（Jinggoy Estrada）与SB 250（Erwin Tulfo）。上述案均在委员会审议中。 
CHR在立场书中援引多起案件，说明刑事化的“寒蝉效应”：如诺贝尔奖得主、Rappler创办人玛丽亚·雷萨及前研究员Reynaldo Santos Jr.于2021年被判诽谤；报导者Frank Cimatu在2022年因2017年Facebook贴文被前农业部长Manny Piñol提告并定罪。CHR批评此类惩罚性做法过度、不相称，助长对人权与公民自由的侵蚀，并被用于加剧对异议者的红色标记。 
“废除诽谤与网络诽谤的刑责，是赋权社群、保护监督者并确保公民参与免于恐惧的重要一步。”CHR如是表示。

开放问题｜弗朗西斯科·乌尔迪涅斯谈委内瑞拉如何检验中国在拉美的经济推进
作为研究中拉关系的权威学者，弗朗西斯科·乌尔迪涅斯提出了“经济位移”（economic displacement）概念，并于今年11月在剑桥大学出版社出版同名著作。他认为，中国的崛起使得与北京的经济合作对许多拉美国家更具实质影响，从而削弱了美国的相对重要性。但在美国对加拉加斯的行动及尼古拉斯·马杜罗被绑架的背景下，委内瑞拉的局势可能会动摇这一框架，改变外界对美中在拉美竞争格局的认知。
为量化这种变化，乌尔迪涅斯构建了一个指标，将贸易、援助、金融与投资等变量按各国GDP加权合成，用以衡量中国相对于美国在某国的经济影响力。他发现，从2000年代到2020年间，美国的相对经济分量缩减约一半，而同期中国在拉美的经济权重增长了约15倍，这一趋势在全球其它地区也可观察到。乌尔迪涅斯指出，从2001年中国入世到他在2023年完成分析，这一增长具有史诗般的规模，但委内瑞拉事件或将成为检验和重塑这一格局的重要变量。

联合国秘书长警告：若会员国不缴费，联合国面临“迫在眉睫的财政崩溃”。
安东尼奥·古特雷斯在写给193个会员国的信中称，联合国正陷入不断加深的财务危机，威胁项目执行，资金可能在7月耗尽。他要求各国履行强制缴费义务，或彻底修改财务规则以避免崩溃。
古特雷斯指出，2025年已缴款仅占应缴总额的77%，创纪录欠款。现行一项规定要求将未能执行项目的未用款退回给会员国，导致组织不得不“归还并未实际收到的资金”——仅本月按2026年评估就被迫退回2.27亿美元。

古特雷斯警告称，当前形势与以往财政危机“截然不同”。他强调二选一：要么会员国按时足额缴费，要么共同根本性改革财政规则，否则将面临“迫在眉睫的财政崩溃”。
