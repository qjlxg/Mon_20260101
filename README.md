根据独立研究机构IEEFA于5月13日发布的报告，2026年第一季度欧盟从俄罗斯进口的液化天然气（LNG）达到了自2022年俄乌战争以来的最高水平。
报告显示，欧盟在2026年前三个月自俄进口天然气约69亿立方米，较2025年同期增长16%；IEEFA并称4月份进口同比亦增长17%。尽管通过管道的俄罗斯天然气几近停止，但法国、西班牙和比利时等国仍通过油轮大量采购俄罗斯LNG。
