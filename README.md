【乘联分会崔东树：2025年以来全国乘用车行业促销与降级均回归理性】金十数据6月7日讯，乘联分会崔东树发文表示，2025年以来全国乘用车行业促销与降级均回归理性，市场秩序明显改善。由于新能源车购税恢复征收，指导价对消费者购车交税影响体现。2026年5月降价规模20款，较同期多7款。其中纯燃油车型7款，较同期多5款；插混车型7款，较同期多4款。燃油车和狭义插混的压力太大，价格调整可以理解。2026年降价规模77款，较同期少4款，其中常规燃油车降价32款，较同期增13款；混合动力燃油车降价4款，较同期增加1款；插混燃油车降价16款，较同期持平；增程式车型降价3款，较同期减少4款；纯电动车型降价22款，较同期减少14款。2026年5月份，新能源车新车降价车型均价21.5万元的降价力度算术平均达到2.1万元，降价力度达到9.6%的较高水平。2026年1~5月份，新能源车新车降价车型均价24.9万元的降价力度算术平均达到3.1万元，降价力度达到12.5%。

辽宁将撤并28家省级经济开发区
中国辽宁省在因产业基础不扎实和产业同质化等问题突出，而撤并21家省级经济开发区后，将继续大刀阔斧改革，再撤并28家省级经开区。
央视《焦点访谈》栏目星期六（6月6日）报道，经济技术开发区原是改革开放的试验田、经济发展的增长极，但一段时间以来，个别地方也出现了只铺摊子、不顾质量的问题，一些开发区体量偏小、质效不高、定位模糊、布局分散、产业趋同。
为了让开发区实现高质量发展，辽宁省过去几个月撤并了21家省级经济开发区。
辽宁省商务厅副厅长刘殿辉说，开发区多了，每一个开发区都会出台相关的政策，同样的项目来到这个地区之后，就容易引起互相争抢、竞相出台更加优惠的政策，就会形成政策内卷。
报道指出，辽宁省先后撤并的21家省级经济开发区，普遍具有成长性差、规划布局不合理等发展质效问题。
刘殿辉解释说，一是开发区的产业基础不扎实，比如有的缺少产业发展规划，谋划的主导产业不聚焦、数量多，没有形成真正的主导性产业、特色性产业；二是产业同质化问题较突出，没有形成差异化发展，同质化严重容易带来内卷，整个规模小、发展能力弱，招商也困难，总体上质效比较差。
除了对21家省级经开区进行撤并，辽宁也对其他省级经济开发区开展了多项改革措施，包括精简内设机构、剥离社会管理职能、扩大经济管理权限、建立与地方政府财力分成机制，目的是使开发区聚焦在经济发展、招商引资、涉企服务等主责主业上，实现高质量发展。
刘殿辉说，此轮开发区改革坚持刀刃向内，通过改革倒逼开发区加快高质量发展，倒逼市县区治理模式转变和产业发展全域统筹，为辽宁全面振兴贡献更大的开发区力量。
下一步，辽宁省还将撤并28家省级经济开发区，基本完成阶段性整合优化目标，让经开区主业更加聚焦，开放底色更加鲜明。
报道称，辽宁省果断撤并整合省级经济开发区，就是要从过去的“铺摊子”转向未来的“上台阶”。撤掉的开发区，由地方政府继续为企业提供全流程优质服务，留下的开发区要巩固提升先进制造业产业基础，因地制宜发展新质生产力。

批评者指出，美方措施更以贸易规模与地缘政治为导向，而非按剥削严重性精准制裁，未必能改善情况，反而可能激化部分国家的政治抵抗。欧盟反驳称其将于2027年生效的强迫劳动法规程序更为严格，覆盖范围广，且与美国在2025年达成的贸易协定将大多数欧盟商品对美关税上限定为15%。国际商会与人权组织也对关税的任意性与有限效果表示担忧。

伊朗6月6日向海湾盟友巴林和科威特发射多枚导弹，进一步破坏本已脆弱的停火。美国中央司令部称，伊朗发射7枚弹道导弹，其中6枚被拦截、1枚未达目标。巴林谴责此次袭击为“公然侵略”，并指出这是三天内对两国的第二次攻击；科威特外交部警告称此举“构成严重升级”，威胁平民生命安全。首都发生爆炸声并响起空袭警报，民众惊恐。 
上月8日名义上的停火仍在，但紧张局势在6月5日加剧：美方称在击落飞往霍尔木兹海峡的无人机后对伊朗雷达目标实施打击。伊朗革命卫队表示，次日以导弹打击“该地区敌方基地”，作为对美方针对西里克和凯什姆岛行动的回应。美军称目前无美方人员伤亡，伊朗关于摧毁驻巴林第五舰队总部的说法为假。 
与此同时，美方已为赴世界杯的伊朗国家队发放签证，部分随队行政管理人员获拒；一名美方官员称决不允许借体育签证潜入恐怖分子。美总统特朗普表示，伊朗仍保留大约20%左右的导弹库存。谈判陷入僵局，伊朗高层顾问呼吁释放约240亿美元被冻结资产。 
此外，黎巴嫩局势亦受牵连：6月6日黎南部遭以军轰炸致3名黎军人死亡，以军称正在调查并表示其行动针对真主党。以色列与真主党在新一轮停火协议被拒后仍有交火。来源：法新社。

以色列空袭致南黎巴嫩车辆被击中，黎巴嫩军方称3名士兵遇难
黎巴嫩军方6日表示，南黎巴嫩Khardali—纳巴蒂耶公路上一辆军用车辆遭以色列空袭，导致包括一名军官在内的多名军人“殉职”，其中3人死亡。军方在社交平台X的声明称，此次袭击“野蛮且蓄意”，意在破坏恢复稳定的努力。 
以色列军方回应称，遭击中的车辆在一个已撤空的“活跃战区”内“行踪可疑”，并强调其行动针对“真主党恐怖组织，而非黎巴嫩军队”，目前正在审查此事件。以军阿拉伯语发言人阿维查伊·阿德拉伊在Telegram上称，因真主党违反停火，以军被迫采取行动。 
今年4月17日生效的停火未获严格遵守，双方多次相互指责。美国华盛顿本周宣布一项有条件停火方案，要求真主党停火并从靠近以色列边境地区撤出，黎巴嫩军队将在若干“试点区”部署并行使管控；真主党拒绝该协议，要求以色列全面撤出黎巴嫩领土。 
同日，以色列下令南部和东部五个村庄民众撤离至Zahrani河以北，并在南黎巴嫩开展多次空袭，黎巴嫩官方媒体报道。 
黎巴嫩总统米歇尔·奥恩在6日谴责此次袭击为“公然侵犯主权”，并在6日播出的CNN采访中呼吁伊朗停止干预，称“这是我们的国家，不是你们的”，指责伊朗把黎巴嫩当作与美国谈判的筹码；他并敦促真主党通过谈判与外交寻求解决，称大多数黎巴嫩人已厌倦战争。 
自3月2日真主党为报复2月28日伊朗最高领导人遇害而对以色列发动攻击后，黎以冲突升级，以色列随后进行大规模空袭与地面行动，造成逾3500人死亡。 （法新社）

伊斯坦布尔讯——过去二十年国家大力投入，使土耳其成长为无人机及军工出口大国。土耳其现向近40个国家，主要为海湾、非洲、亚洲及部分欧洲国家，提供被视为更便宜、交付更快且更易定制的武器装备。路透社统计显示，自2021年以来，土耳其防务出口已超过三倍，至2025年达100亿美元，约占该国出口总额的3.7%，对欧美出口近四倍增至56亿美元。 
本国成熟的军工体系包括Baykar、土航（Turkish Aerospace Industries）、Arca Defense与Kale等企业；持续的国家支持、灵活供应链与为买家定制系统的能力，使其能迅速填补西方供应短缺或冗长采购周期留下的市场空白。土方防务机构表示，目标在两年内将防务出口翻倍，以获取偿债与研发资金。 
土耳其地处乌克兰与伊朗之间，安全利害相关，其在防空、战机与坦克发动机等领域的不足亦可通过贸易与技术合作弥补。哈德逊研究所资深研究员Can Kasapoglu称，土耳其通过出口先进系统，尤其是空中无人机，实现了“重大跨越”。 
土耳其约占全球武装无人机使用量的65%，也是弹药重要出口国，正规划或生产护卫舰、航母、防空系统与装甲车。印尼2025年拟购48架仍在研制的土耳其战机。五月在伊斯坦布尔展示的一款洲际弹道导弹原型及其宣传片引发外界质疑。 
土耳其盼借7月7–8日在安卡拉举行的北约峰会（并拟接待美国总统特朗普与其他领导人）扩大在西方市场的军售与联合生产，但在欧盟面临成员国专属防务项目与政治阻力——包括对土外交分歧与民主倒退的担忧，令其在欧安保行动中多被排除。尽管如此，土耳其仍与波兰、西班牙、葡萄牙、罗马尼亚达成供货协议，Baykar收购意大利Piaggio Aerospace并与莱昂纳多合作生产。 
斯德哥尔摩国际和平研究所指出，全球军费在2021–2025年增长24%至近2.9万亿美元，欧洲增幅达75%。同期土方防务机构将资助的研发项目增至1400余项。防务分析师Arda Mevlutoglu表示，如能克服政治障碍，欧洲与土耳其的更深层合作具备条件，但需高层政治意愿推动。路透社报道。
