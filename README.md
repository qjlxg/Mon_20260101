据澎湃新闻报道，新田县营商环境建设局揭牌仪式2月14日在县政务中心举行。微信公众号“新田发布”刊文指出，新田县营商环境建设局的正式挂牌成立，充分释放了县委、县政府大抓经济、优化环境的坚定决心，更彰显了将营商环境打造成核心竞争力的坚定决心，将有效整合各方面资源，进一步理顺体制机制，凝聚共建营商环境的合力，推动新田县营商环境不断提档升级。

法国巴黎 — 一项周五发布的研究指出，尽管开发者不断夸大自主“代理”（agent）的能力，但对其潜在安全问题和风险披露甚少。研究者呼吁对这类用于处理复杂任务的聊天型自主系统加强透明度与研究。
剑桥大学研究员莱昂·斯陶弗（Leon Staufer）说，过去一年涌现出“能在更长时间内在无人干预下完成任务的自主代理”，但关于这些系统安全性的公开信息并未随之增加。研究团队来自英国、美国（包括斯坦福、麻省理工）、中国和以色列，基于截至去年12月的工具样本编制了“AI代理指数”。
研究发现，许多以OpenAI、Anthropic、Google等先进大模型为基础的代理系统，缺乏发布方和第三方的安全评估：30个样本中有25个“未披露内部安全结果”，23个“无第三方测试信息”。斯陶弗指出，关键的安全行为来源于代理自身的规划、工具、记忆和策略，而非仅模型本身，但很少有开发者分享这些评估数据。
研究同时警示，已有事件显示代理的记忆与工具结合会产生用户可能未预料的行为。为例，一位美国开发者斯科特·香鲍（Scott Shambaugh）称某代理曾试图通过在博客撰写抹黑文章来强迫他接受对开源项目的修改；代理所有者则匿名回应称仅给过一般性指令。斯陶弗称这是“真实且迫在眉睫的威胁”，并强调需开展更多、基于具体使用场景的评估，目前这类工作主要在大型AI公司内部进行。

受中国需求推动，津巴布韦在上一个销售季创造了3.527亿公斤的烟草产量，价值约12亿美元。

【玉渊谭天：美国最高法裁决后，对华IEEPA关税应自动取消】金十数据2月22日讯，玉渊谭天发文称，自去年2月以来，美国依据IEEPA对华加征的新关税主要有两部分。一是所谓的“芬太尼关税”，目前对华税率是10%；二是所谓的“对等关税”，目前对华执行其中的10%税率，余下24%税率暂缓执行。按照美国行政令的说法，这些援引美国《国际紧急经济权力法》实施的关税将“尽快终止征收”。在美国最高法院作出裁决后，也就意味着美方根据最高法院判决发布的取消IEEPA关税的行政命令同样也适用于中美关税安排中的相关部分，对华IEEPA关税应自动取消。

【玉渊谭天：美国通过IEEPA路径对外国产品加征关税已难以为继】金十数据2月22日讯，玉渊谭天发文称，目前看，美国通过IEEPA路径对外国产品加征关税已难以为继。美国最高法院认为，征收和收取“税收、关税、进口税和消费税”的权力属于国会，IEEPA虽赋予总统在宣布国家紧急状态后处理“异常和特殊”外国威胁的权力，但并未明确授权总统在和平时期单方面征收关税。多数意见还援引“重大事项原则”，强调凡涉及重大经济和政治后果，必须有国会清晰授权。美国政府一直宣称，即便不能实施IEEPA关税，也可以援引其它法律对进口商品加征关税。但问题是，这些法律工具规定了严格的实施条件和实施前提，仅为替代IEEPA关税而强行加征新名目的进口关税，必将给美国企业和消费者带来新的伤害，进而引发新的司法诉讼，浪费司法资源。

【印度贸易代表团推迟访美】金十数据2月22日讯，据外媒报道，在美国最高法院裁决后，印度新德里官员透露，印度贸易官员将推迟前往美国的行程，此前计划敲定印美达成的临时贸易协议。会谈将重新安排在较晚的日期。印度本月早些时候与美国达成了一个临时框架，特朗普将印度商品的税率从25%降至18%，并取消了额外的25%惩罚性关税。印度官员们表示，此次访问将在两国研究和评估事态发展及其影响后进行。

【以色列称不会向所谓“和平委员会”提供资金】金十数据2月22日讯，当地时间22日获悉，以色列已通知美国政府，不会为美国总统特朗普设立的所谓“和平委员会”提供资金。以色列安全内阁成员泽夫·埃尔金当天在接受采访时证实相关消息。他表示，以色列“没有理由”为该委员会提供资金，并称“以色列遭到攻击，不应承担加沙重建费用”。有以色列政府官员透露，这一决定也与以色列国内政治考量有关。决定被认为有助于安抚极右翼的以色列国家安全部部长伊塔马尔·本-格维尔和以色列财政部长斯莫特里赫。此前，此二人并不支持以色列加入“和平委员会”。
