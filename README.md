马尼拉消息——棉兰老岛以外的民多罗省官员强烈反对近期航运票价上调，业者称此次涨价系中东冲突推高国际油价所致，个别航线涨幅最高达25%。
安东尼·亚普（Anthony Yap，东方民多罗第二区议员）与西民多罗议员西里洛·特霍索（Cirilo Tejoso Jr.）已向省议会（Sangguniang Panlalawigan）提交决议，敦促海事工业局（MARINA）检讨并调查多家航运公司此次涨价理由。
特霍索批评涨价过于仓促，将加重民众负担，并质疑油价回落时公司是否会下调票价；亚普则指出此次涨幅缺乏充分依据，称航运业在2023和2024年已于2022年油价曾升至每桶139美元后多次涨价，而当前国际油价约为每桶90美元，尚不足以支持新一轮上涨，并建议油价恢复后应向乘客退款。
地方官员强调，海运仍是民多罗居民往返大陆及运送农产品的主要交通方式，票价上涨影响重大。
交通部长乔瓦尼·洛佩斯（Giovanni Lopez）向ABS-CBN表示，尽管船运票价属放宽管制行业，但MARINA仍需判断涨幅是否合理，该议题将于周四与MARINA会商。
航运业方面，菲律宾岛际航运协会主席兼FastCat业者菲律宾群岛渡轮公司（Philippine Archipelago Ferries Corp.）业主克里斯托弗·帕斯特拉纳（Christopher Pastrana）为涨价辩护，称燃油成本激增，占营运成本约80%。他称民多罗等地油价从每升约53—55披索在一周内飙升至约89—90披索，价格每日攀升。
帕斯特拉纳表示，尽管调价，但目前收费仍在MARINA于2022年核准的公布票价内，且此前实际收取票价曾低于公布价约50%。以巴坦加斯—卡拉潘（Batangas–Calapan）航线为例，2022年公布经济舱票价为648披索，公司在危机前仅收540披索，现已调至648披索；私家车公布价为4,320披索，危机前收2,640披索，现调为3,168披索，仍低于公布价。
他建议政府可通过临时减税或降低港口费等措施协助业者减负，并透露多家公司已将日班次从8班降至5班以节省燃油。帕斯特拉纳并承诺，对符合法律规定的乘客折扣将继续执行。

马尼拉讯 — 菲律宾交通部（DOTr）正考虑下调航空相关收费，以稳定机票价，交通部长乔瓦尼·洛佩兹周三表示，此举旨在缓解中东冲突导致的油价与供应链冲击对乘客票价的影响。

交通部：将重新核算运价 避免加重民众负担
马尼拉—交通部（DOTr）周三证实将出现调涨车费的情况，但强调相关单位须重新核算最终金额，以免增加民众负担。陆运特许与监管委员会（LTFRB）已提交加价建议，交通部长乔瓦尼·洛佩兹（Giovanni Lopez）表示，鉴于当前中东冲突引发国际油价飙升，必须谨慎检视数据，调整幅度要兼顾通膨与民众承受能力。洛佩兹指出，政府正审慎评估数字，避免本意为助益反而加剧问题。总统费迪南德·马科斯（Ferdinand Marcos Jr.）已下令对公车司机发放5000比索补助，预计下周启动以缓解运输冲击。
众议院二读通过临时停征燃油特税授权法案
马尼拉—在油价居高不下之际，众议院周三以二读通过一项法案，授权总统在国家或全球经济紧急状况下，临时停征或减征石油產品燃油特税。根据众议院第8418号法案，总统可在发展预算协调委员会（DBCC）并与能源部长协商后启动权力；触发条件包括：迪拜原油月均价达或超过每桶80美元，或总统宣布的国家紧急或灾难导致国内油价异常上涨，且经能源部长证明。停征可针对特定燃油品项，采取全部或部分减免，期限最长期限为6个月，国会可延长但累计不得超过1年；此授权效期至2028年12月31日。法案并要求财政部长在下达停征或减征后15日内向国会两院提交事由、估计損失與对通膨及经济影响报告，并每月更新。众议院稅收委員會主席米羅·昆博（Miro Quimbo）预期法案将在下周休會前三读通过；若众参两院版本有异，则将召開兩院協商會議以求一致，最终送交总统签署。

菲律宾政府宣布，从下周起向公共载具（PUV）司机发放每人5,000比索的燃油补助，以应对中东局势推高油价的影响。首批补助将通过地方政府发放给三轮车司机，社会福利与发展部（DSWD）部长雷克斯·加查利安表示，政府已从总额600亿比索的“个体危机援助”（AICS）中拨出一半用于该计划。
加查利安在总统府简报中说，鉴于公共运输司机生计受损，可视为“处于危机中的个体”，因此适用AICS支援。他并表示，马尼拉大都会市长们已达成共识，补助将于下周二起开始发放；当日下午他将与大都会的社福官员会面，敲定可领取补助的三轮车司机名册。
运输部（DOTr）部长乔瓦尼·洛佩兹称，吉普尼、计程车、公交与网约车司机也将在随后数周（预计四月）领取5,000比索的现金援助。他强调采取现金发放以减轻民众负担，受助者可自行决定将款项用于燃油、医药或食品等开支。
对于非公共运输行业的中产阶层，加查利安表示，DSWD仍有其他救助措施可提供：任何处于危机的公民均可向DSWD报到，申请现金救助或其他福祉服务。
此外，洛佩兹说交通部正与收费站营运单位协调，对运输食品及农产品的卡车给予通行费减免，以抑制物价上涨。地方政府也获指示动用地方政府支持基金（LGSF）发放米粮补贴。
圣胡安市市长、菲律宾城市联盟主席弗朗西斯·萨莫拉表示，若有中产选民确实需要协助，也可列入领取大米补贴的名单。
根据2026年国家预算，LGSF编列578.7亿比索，其中374.9亿用于地方政府财政援助（FALGU）、113亿用于成长基金、80亿支持村里事务、10亿用于参与式预算。地方政府将继续关注波斯湾局势，评估是否需要第二轮现金援助。

社保局提升数字化服务，启用网站应用运营战情室（SSO Core）以保障雇主、被保险人及医疗机构的稳定使用。
3月11日，社保局副秘书长农拉坤·郭瓦鲁库接秘书长康差那·普勒考委托，在位于暖武里府提瓦农路社保局วิทุรแสงสิงแก้ว大楼一层，正式启用支持网站应用服务的战情室（SSO Core），并组织现场巡检。
