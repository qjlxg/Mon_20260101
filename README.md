中国官方：推动增收 让居民能消费敢消费愿消费
中国国家发改委表示，将制定实施扩大消费“十五五”（2026年至2030年）规划，使居民有稳定收入能消费、没有后顾之忧敢消费、消费环境优获得感强愿消费。
据21世纪经济报道，中国国家发展改革委副主任王昌林星期五（4月17日）在“开局起步‘十五五’”系列主题新闻发布会上，介绍“十五五”时期推动居民消费率明显提高的重点安排。
王昌林表示，目前中国超大规模市场优势还没有充分发挥出来，扩大消费特别是服务消费潜力巨大。下一步，将深入实施提振消费专项行动，更好发挥消费对经济发展的基础性作用。
王昌林从四方面展开阐述。一是增强居民消费能力，具体措施包括健全就业促进机制，强化劳动者技能培训，促进劳动者技能就业、技能增收，稳步提高最低工资标准。多渠道增加城乡居民财产性收入，持续提升居民消费能力。
二是提升消费品质，用好消费品以旧换新政策，以高质量供给提高居民消费意愿；开展服务业扩能提质行动，增加生活性服务业高品质多样化便利化供给等。
三是注重培育消费场景。充分发挥中国超大规模市场和丰富应用场景优势，打造更多互动式、智能化消费新场景。
四是注重优化消费环境。从“硬投资”来讲，要统筹各类资金渠道加快消费基础设施建设，包括中央预算内投资支持养老、托育、文旅、体育等设施建设，发行消费基础设施房地产投资信托（REITs）等，运用超长期特别国债资金支持线下消费商业设施设备更新。
从“软建设”来讲，要积极推动各地优化限行限购、新型消费、市场准入等政策措施，加快清理优化消费领域不合理限制性措施，共同营造愿消费、敢消费、乐消费的良好环境。
王昌林也表示，鼓励更多有条件的地方积极探索中小学春秋假，同时做好学生假期与家长带薪年休假的衔接，增加亲子游、研学游、假期托管等服务供给，让家长有更多时间陪伴孩子出行。

中国多地加强对退休干部党员的教育管理监督
中国多地加强对退休干部党员的教育管理监督，有地方除了规范因私出国行为，也建立思想动态摸排机制，以发现“苗头性、倾向性问题”。

据报道，《若干措施》从“严明纪律规矩、加强纪律教育、严格管理监督、压实工作责任”四个方面提出明确要求，推动“全面从严治党”进一步向退休干部党员延伸覆盖。
针对有干部退休后对自身要求放松、初心使命淡化等问题，文件明确提出“十个自觉、十五个不得”要求，涵盖政治纪律、中共中央八项规定精神、组织纪律、廉洁纪律、群众纪律、工作纪律及生活纪律等多个方面，强调退休干部党员要始终坚持离岗不离党、退休不褪色。
报道称，《若干措施》的出台，标志着湖南在加强退休干部党员教育管理监督上迈出重要一步，“为教育引导广大老同志严守党规党纪、坚守入党初心提供了有力制度保障”。
湖北、福建、广东、山东等中国多地的老干部局，过去一段时间以来均在官方微信公众号发文，提出加强对退休干部党员的教育管理监督。
中共湖北省委老干部局星期四在微信公众号发布题为《省纪委监委机关推动退休干部教育管理监督工作走深走实》的文章称，始终把政治建设摆在首位，教育引导退休干部保持政治坚定。
文章介绍了“全周期、全覆盖的退休干部教育管理体系”，以及委机关、党总支、党支部、在职党建联络员“四级联动”管理机制，以此严格落实中共的组织生活制度，并制定《离退休干部服务手册》，明确政治待遇、因私出国（境）等十余项行为规范，建立重要事项事前申报制度，前移监督关口。
文章透露，当局也建立退休前谈话制度，由委领导开展谈心谈话，重申纪律要求。退休后，建立动态管理台账，对长期异地居住、出国（境）党员实行“一对一”联系，确保教育管理不断线。建立思想动态摸排机制，对苗头性、倾向性问题早发现、早提醒。近年来，对三名情节轻微、主动配合的机关退休干部予以批评教育，对一人予以党纪处分。

中国七家电商平台涉幽灵外卖被罚近36亿人民币
中国官方公告，中国七家电商平台涉嫌“幽灵外卖”系列案，被罚近36亿（人民币，下同，6.71亿新元）。
中国国家市场监督管理总局星期五（4月17日）在官网公告上述消息。
根据公告，中国市场监管总局当天依据《中华人民共和国食品安全法》第一百三十一条、《中华人民共和国电子商务法》第八十三条的规定，对上海寻梦信息技术有限公司（拼多多）、北京三快科技有限公司（美团）、北京京东叁佰陆拾度电子商务有限公司（京东）、上海拉扎斯信息科技有限公司（原饿了么，现淘宝闪购）、北京抖音科技有限公司（抖音）、浙江淘宝网络有限公司（淘宝）、浙江天猫网络有限公司（天猫）等7家电商平台“幽灵外卖”系列案，作出行政处罚决定，责令家电商平台改正违法行为，暂停新增蛋糕店铺三至九个月不等，并处以罚没款共计35.97亿元。
中国市监管总局同时也依据《中华人民共和国食品安全法实施条例》第七十五条的规定，对七家平台企业法定代表人和食品安全总监合计处以罚款1968.74万元。
公告称，经查，上述七家电商平台对入网食品经营者许可证审核把关不严，未依法履行资质审查义务；与转单平台签订合作协议，明知或应知转单行为侵害消费者合法权益，但未采取必要措施；七家电商平台法定代表人和食品安全总监，负有食品安全管理责任，但未全面履行有关岗位职责。
上述行为严重违反《中华人民共和国食品安全法》《中华人民共和国电子商务法》《中华人民共和国食品安全法实施条例》有关规定。
案件启动调查后，中国市监管总局第一时间责成电商平台立行立改，七家电商平台均已下架未经审核的有关“幽灵店铺”，停止与相关转单平台的餐饮转单合作。
所谓“幽灵外卖”，是指没有营业执照、食品经营许可证等证照，或不具备餐饮经营资质的商户，通过假证、套证、借证等违法违规手段在外卖平台提供违规餐饮服务的现象，此类商户借助虚假证照与地址信息违规上线经营，埋下食品安全隐患。

四川资中八天连发23起小型地震 官方称大震可能性不大
中国四川省内江市资中县八天内连续发生23起3级以上的小型地震，当地官方通报称，发生大震的可能性不大，民众无需恐慌。
中国地震台网中心历史记录显示，上周五（4月10日）至星期四（16日）的连续七天内，资中县境内共计发生20起3级以上地震，其中两次地震超4级，最大震级为4.2级。
星期五（17日）凌晨6时22分起，资中县境内地又相继发生三起地震，分别为4.4级、4.0级、3.3级，震源深度都在6至7公里不等。
根据中国地震台网速报目录，震中周边200公里内近五年来发生3级以上地震共212次，最大地震是2021年9月16日在四川泸州市泸县发生的6.0级地震，距离本次震中68公里。
