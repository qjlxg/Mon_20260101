根据美国民意调查机构盖洛普（Gallup）和医疗费用研究机构“西区健康”（West Health）去年对2万名美国人进行的调查，估计有超过8200万美国人为了支付医疗账单而至少做出了一项牺牲。
这项星期四（3月12日）发布的调查结果显示，最弱势和没有医疗保险的人群最有可能为了省钱而做出妥协，但也有11%的富裕家庭表示他们也做出了类似的权衡。
西区健康政策中心主席拉什在一份声明中说：“当各个收入阶层的家庭都被迫在医疗账单和支付取暖费或电费之间做出选择时，这并非个人预算问题，而是系统性缺陷。”
这一调查发现正值美国民众持续担忧生活成本上涨之际，预计这将再次成为美国选民在11月中期选举投票时关注的关键问题。
西区健康政策中心去年底针对5660名美国人进行的另一项调查发现，由于医疗费用高昂，许多美国人正在推迟治疗和手术。
根据调查，医疗费用导致14%的受访者推迟购房计划，9%的受访者推迟退休，另有6%的受访者推迟生育或领养子女的计划。

报告总结称，能源进口、汇款、财政补贴、汇率和国际融资等渠道，可能把该冲突的风险传导到部分新兴市场。（综合）

联合国安理会敦促伊朗停止对海湾国家的袭击
（2026年3月12日）联合国安理会周三通过一项决议，要求伊朗立即停止针对巴林、科威特、阿曼、卡塔尔、沙特阿拉伯、阿拉伯联合酋长国和约旦的所有攻击。决议以13票赞成、2票弃权通过，但未提及美以对伊朗的打击，引发伊朗代表强烈不满。
决议同时谴责伊朗任何旨在关闭、阻碍或干扰霍尔木兹海峡国际航行的行为。提出该议案的巴林常驻联合国大使贾迈勒·法雷斯·阿尔罗瓦埃伊表示，海湾安全关系到全球经济与能源安全，因此是国际共同责任。中国和俄罗斯因决议未指责美以而弃权。
伊朗常驻联合国大使阿米尔·赛义德·伊拉瓦尼称该决议是“对安理会职责的公然滥用”，并抨击其对伊朗不公。美国常驻联合国大使迈克·沃尔茨表示，投票结果反映了对伊朗袭击的广泛谴责。
此前，伊朗为报复其所称导致伊朗最高领袖阿里·哈梅内伊遇害的美以袭击，曾多次打击海湾国家并向霍尔木兹海峡过往商船开火，企图影响全球能源与经济。该决议由135国共同发起并由巴林提交。来源：法新社（AFP）
