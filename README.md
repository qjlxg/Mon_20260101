中国供应商上调价格 美国消费者恐迎昂贵圣诞季
沃尔玛和好市多等美国大型零售商的中国供应商称，随着原材料成本飙升，加上工厂为圣诞节加紧生产，他们多年来首次上调价格。
彭博社报道，根据过去两周对十多家供应商的采访，涨价幅度因产品和客户类型而异。其中五名受访者说，中国制造商对大型连锁零售商部分产品的涨价幅度最高约为5%；另有11名受访者称，对小型商家的涨价幅度最高可达15%。
五名因信息未公开而要求匿名的人士称，节庆靠垫等家居用品是受冲击最严重的产品之一，价格上涨了10%至15%。
报道认为，当前时机相当敏感，因为全球零售业正进入关键阶段；现在下达的生产订单，将决定商品能否赶在圣诞节前上市。这意味着，随着节日季临近，原本已令家庭预算承压的通胀压力，可能进一步加剧。
义乌市圣诞用品行业协会秘书长蔡勤亮说：“对于这些‘你不可能跳过不买’的节日商品，只要供应商不开价过高，买家对涨价的容忍度还是相对较高的。”
蔡勤亮说：“他们其实也没有太多选择，对吧？全球和中国的原材料价格都在上涨，因此拒绝接受更高价格，实际上就等于放弃今年的生意。”
蔡勤亮也说，人造圣诞树的价格涨幅最高达到10%。
美国零售商大多仍未自行提价。彭博行业研究高级分析师巴塔舒斯说，大型零售商商品种类繁多，这意味着即使需要涨价，也能通过对多种商品小幅提价来分散影响，相较于单一商品价格大涨，对消费者造成的冲击可能较小。
不过，随着汽油价格飙升，美国消费者的担忧情绪正不断加剧。
卡夫亨氏和麦当劳等面向消费者企业的高管警告称，消费者已开始缩减开支，这使零售商面临一项微妙任务，既要缓解利润压力，又不能让消费者因价格上涨而却步。
彭博社引述荷兰国际集团（ING）大中华区首席经济师宋林分析：“价格上涨最终将转嫁给消费者，尤其是在战争持续拖长、能源价格维持高位的情况下。”
宋林也指出，人民币走强同样推高了中国出口商品的价格。“到了圣诞季，这些涨价影响很可能至少会有一部分真正显现出来。”
也有迹象显示，随着伊朗冲突持续扰乱供应链，零售商担心未来几个月价格可能进一步大幅上涨。一些制造商说，他们早在4月中旬就已开始收到圣诞季生产订单，比往年惯常时间提前了数周。
