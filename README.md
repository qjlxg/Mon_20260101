总理出席并主持全国市镇联盟年会开幕，强调重视地方政府并承诺全力协助
（摘自 4月3日 2569 / 上午11:24）—— 总理阿努廷·詹维鲁（Anutin Charnvirakul）于3月4日上午9时在曼通他尼IMPACT会展中心Challenger 2馆主持“泰国市镇联盟学术会议暨年会（财政年度2569）”开幕。内政部及地方治理相关高层、全国各市镇长、议会主席、镇公所负责人等出席。
阿努廷在致辞中表示，地方政府是国家治理的基石，政府重视并将持续强化地方自治体的能力与预算，推动快速决策与自主解决问题，以带动基层就业、旅游与地方经济发展。他指出，在全球经济、科技与气候快速变动下，政策能否转化为惠及民众的实际行动，关键在于最贴近人民的地方单位——市镇、乡镇与村长。
总理强调，本次会议须重视灾害管理，要求地方从被动防御转为积极预防，建立明确的应变计划、有效预警与常态演练，并加强跨部门协作、动员社区参与，提升救援与救济效率。
他同时指出，供水、公共设施、环境管理与老年人照护等基础公共服务仍是市镇工作的核心，必须以善治、透明与审慎的预算使用来赢得民众信任；地方越强，国家越稳。
最后，总理承诺，不论以总理或内政部长身份，都会与地方站在同一阵线，推动未竟事项延续与落地，凡是能帮助民生的事将不遗余力，目标是实现国民福祉与国家长治久安。
