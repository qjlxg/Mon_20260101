北京警方通报盲人女孩盲道被撞反被骂视频是虚假摆拍
一则关于中国一名盲人女孩在北京盲道上行走时，被一辆骑行在盲道上的电动自行车撞倒并遭骑车人斥责的视频在网上走红，引发广泛关注。北京警方随后通报称，上述视频是虚假摆拍，涉事男女已被依法采取刑事强制措施。
在简介称患有一级视力障碍的短视频博主“抱抱盲兔”星期五（5月8日）发布的视频显示，她在北京旅游期间，沿盲道正常行走时被一辆电动自行车撞倒，导致手部受伤、盲杖脱落。
视频显示，骑车男子在事故发生后没有道歉，反而质问她“不看路啊”。女孩回应“我走的是盲道”后，对方又反问“什么盲道”，随后骑车离开现场。
视频发布后迅速引发热议，北京市公安局公安交通管理局旗下的抖音账号“北京交警”也在视频评论区说：“您好，您反映的情况我们已关注。请您通过私信将您的联系方式、原始视频提供给我们，我们将尽快核实情况，依法处置。”
女孩随后回复称，“感谢大家关心，昨晚已经收到了当事人的道歉，也有相关部门联系过了。”
北京市公安局旗下的微信公众号“首都网警”星期六（5月16日）发文称，为净化网络环境，北京警方聚焦人民群众反映强烈的网络乱象，持续加大打击整治力度，积极营造清朗网络空间，并通报三起典型案例。
其中案例一显示，26岁的刘姓男子和24岁的江姓女子为吸粉引流、博取关注、牟取私利，在朝阳区某路旁虚假摆拍“盲人在盲道行走被电动自行车撞击并遭骑车人斥责”内容，并通过短视频平台对外发布，误导大量网民关注和讨论，造成恶劣影响。通报称，两人已被公安机关依法采取刑事强制措施。
目前，“抱抱盲兔”账号已被禁止关注。

中国第二艘国产大邮轮开启试航
中国第二艘国产大型邮轮“爱达·花城号”，星期六开启为期12天的试航。
综合新华社和中国央视新闻报道，“爱达·花城号”星期六（5月16日）从中船集团上海外高桥造船5号码头启航，开始为期12天的试航任务。
这是“爱达·花城号”继3月20日出坞后的又一里程碑式生产节点，也是邮轮正式交付前的最后一个重要节点。
此前，中国首艘国产大型邮轮“爱达·魔都号”分两次完成了全部出海试航任务。“爱达·花城号”将优化了测试流程，把两次试航合并为一次进行。
上海外高桥造船组织来自12个国家共计937人的工程技术团队参加这次试航。工作人员将对邮轮建造质量与系统可靠性进行实战检验，通过对减摇鳍等系统及设备的试验，同步验证振动噪声、航速等关键指标，以达到验证邮轮安全性、舒适性、可靠性的目的。
按计划，“爱达·花城号”将在试航结束后逐步向船东完成项目报验，计划于今年11月6日完成交付，较原计划提前近两个月，交付后将从广州南沙国际邮轮母港开启首航季。
“爱达·花城号”全长341米，总吨位14.19万吨，拥有2130间客房和套房，满载可容纳5232名宾客。作为“爱达·魔都号”的姐妹船，“爱达·花城号”在空间设计、智能科技与邮轮体验上进行了全面升级。

中国U17男足时隔22年再跻身亚洲杯四强
时隔22年，中国U17男足再次跻身亚洲杯四强。
综合北京青年报和中新社报道，中国队当地时间星期五（5月15日）在沙特阿拉伯吉达进行的2026年亚足联U17男足亚洲杯复赛中，以3比1逆转战胜此前三战不败的东道主沙特阿拉伯队，时隔22年再次跻身四强。
本届U17男足亚洲杯共有16队参赛，分四个小组进行角逐。小组赛阶段，中国队取得1胜2负，凭借净胜球优势位居B组第二取得出线权，同时也获得2026年国际足联U17男足世界杯正赛资格，这是中国男足时隔21年再次进入国际足联U17男足世界杯正赛。
中国U17男足在星期五的复赛中迎战A组头名沙特队，对方第21分钟率先破门，以1比0领先。中国队调整状态，更加积极主动组织进攻。第27分钟，受伤缺席小组赛的邝兆镭复出比赛，在右路横传，为队友万项助攻，后者在禁区内冷静扣射破关，扳平比分。
沙特队随后掀起反攻，但始终显得较为急躁，多次进攻威胁均被中国队防守球员化解。第58分钟，万项底线横传，何思凡后点头球摆渡，赵松源铲射破门，中国队以2比1反超。完场补足时间内，何思凡杀入禁区爆射，皮球击中横梁后反弹入网，为中国队以3比1锁定胜局。
中国U17男足将在当地时间5月19日进行的半决赛中，对阵越南队与澳大利亚队复赛的优胜队。

对于特朗普是否可能与赖清德讨论军售问题，陈明祺回应称，台湾方面仍需进一步了解特朗普相关言论的“真实意图”。

30多岁男子在西澳大利亚遭鲨鱼袭击 目前情况危急
一名30多岁的男子在西澳大利亚遭鲨鱼袭击，目前情况危急。
法新社报道，西澳大利亚圣约翰救护机构一名发言人说，这起事件发生在星期六（5月16日）早上10时（新加坡时间早上10时），地点位于珀斯市附近的罗特内斯特岛附近海域。
发言人说，救援直升机、警方和岛上紧急救援人员抵达现场展开救援。
这起袭击事件是两天内记录的四起鲨鱼袭击事件之一，促使当局关闭了市内数十个海滩。这也是这个市近期发生的第三起与鲨鱼相关的死亡事件。
根据一个记录鲨鱼与人类遭遇事件的数据库显示，自1791年以来，澳大利亚已发生近1300起鲨鱼袭击事件，其中超过260起死亡事件。
最近一起死亡事件是一名12岁男孩，他于今年1月在悉尼港遭到鲨鱼袭击后身亡。　　
澳大利亚科学家认为，日益拥挤的水域和不断上升的海洋温度正在改变鲨鱼的迁徙模式，这可能是导致鲨鱼袭击事件增加的原因之一。

国际货币基金组织（IMF）表示，欢迎美中两国高层展开建设性对话，并认为缓解两国紧张与不确定性有利于两国及全球经济。
IMF发言人朱莉·科扎克在华盛顿新闻发布会上就特朗普与习近平的初步会晤成果表示，双方在最高层交往十分重要，任何有助于降低贸易紧张和不确定性的举措，都将惠及两国并利好全球经济。
科扎克指出，由于中东冲突及霍尔木兹海峡局势导致原油价格维持在每桶100美元以上，IMF在4月《世界经济展望》中列出的三种情形正朝中间的“逆风情形”演进。该情形下，全球实际GDP增速今年将降至2.5%，而更温和的参考预测为3.1%，2025年预计为3.4%。
IMF称该逆风情形假定全年油价为每桶100美元，并伴随金融状况收紧与通胀预期上升。科扎克表示，尽管短期物价预期受推高，但中期通胀预期总体仍然锚定，全球金融环境仍较为宽松。
基金组织正与成员国讨论对受高能源和大宗商品价格冲击的国家提供可能的财政援助，但未披露具体国家或细节。此前IMF总裁格奥尔基耶娃在春季会议上曾估计，至少有12个国家可能需要国际货币基金组织与世界银行合计约200–500亿美元的援助规模。IMF同时建议各国避免大规模燃料补贴，以免耗尽财政资源并推高油价。

美国华盛顿 — 5月13日，美国参议院的共和党人阻止了由民主党推动、要求未经国会授权即结束对伊朗战争的最新决议，但随着第三名共和党参议员投赞成票，该决议仍更接近通过。参院以50比49投票不推进这项由俄勒冈州参议员杰夫·默克利提出的战争权力决议，投票几乎沿党派划分进行。肯塔基的兰德·保罗、缅因的苏珊·柯林斯和阿拉斯加的丽莎·墨科斯基三名共和党人加入了除宾夕法尼亚民主党人约翰·费特曼外的全部民主党人行列支持该议案。
这是今年参院第七次由共和党人阻止类似决议，也是自5月1日冲突触及《1973年战争权力法》规定的60天期限以来的首次投票。该法规定，总统在未获国会授权时只能发动60天军事行动，或在因“不可避免的军事必要性”撤军时请求延长30天。特朗普此前称停火“终止”了敌对行动，民主党人则反驳称冲突仍在继续，指出美方封锁伊朗港口并袭击伊朗船只，伊朗封锁霍尔木兹海峡并攻击美方船只和油轮。默克利称“双方仍在交战，60天时钟不应被中止”。
参院民主党人表示将于下周及以后每周再次提出相同决议，直到战争结束或特朗普向国会寻求授权。众院民主党人也提出了相关决议，但同样被共和党阻止。民主党强调宪法将宣战权赋予国会，警告总统可能将美国拖入长期冲突；共和党和白宫则称总统作为三军统帅有权下达有限军事行动。路透社

官员表示，此行旨在规避中国大陆对台湾日益强化的外交与空域限制，突破孤立努力，确保总统能安全抵达非洲友邦并完成外访任务。

基辅消息——乌克兰总统泽连斯基5月15日表示，俄方正试图将白俄罗斯进一步卷入对乌战争，并正在权衡从白俄罗斯境内对乌克兰北部或直接对北约国家发动攻击的方案。
泽连斯基在与军方和情报官员会面后于Telegram称，乌方持续记录俄方拉拢白俄总统卢卡申科参与“新一轮进攻行动”的尝试，并掌握俄白间为此进行的额外接触。他指出，俄方正在考虑从白俄南部或北部方向展开行动——可能针对乌克兰的切尔尼戈夫—基辅方向，或直接从白俄罗斯领土对某一北约国家发动攻势，但未披露更多细节。
白俄罗斯南接乌克兰，北及西部邻接北约成员国波兰、立陶宛与拉脱维亚。对此，莫斯科和明斯克尚未立即回应。俄罗斯对乌军事计划列为国家机密。
卢卡申科作为普京的亲密盟友，曾允许俄军在2022年2月入侵期间利用白俄领土，但未派白方部队参战。明斯克随后同意在境内部署俄方战术核武和“奥列什尼克”高超声速导弹。
泽连斯基补充称，4月已获情报显示俄方正为再度牵扯白俄做准备，并警告若卢卡申科支持俄方，乌克兰必将捍卫国家与人民。他已指示国防部制定应对方案，加强切尔尼戈夫与基辅北部防御。  路透社
