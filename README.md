世行：中东战事不可避免加剧粮食安全
世界银行、国际货币基金组织和联合国世界粮食计划署周三警告，中东战争引发的石油、天然气和化肥价格飙升，将不可避免地导致食品价格上涨和粮食安全问题加剧。
世行、国际货币基金组织（IMF）和联合国世界粮食计划署（WFP）在星期三（4月8日）发表的联合声明中指出，最脆弱的群体，特别是低收入、依赖进口的经济体，将承受最沉重的负担。它们将继续密切关注事态发展，并“协调运用一切可用工具，为受危机影响的人们提供支持”。
联合声明指出：“中东战争正在颠覆该地区乃至更广泛地区人民的生活和生计。它已经引发了现代史上全球能源市场最严重的动荡之一。”

三机构领导人承诺，将根据各自的职责提供支持，并为实现强劲复苏奠定基础。

研究：长期由AI陪伴 或影响心理健康
最新研究显示，人工智能（AI）陪伴虽可在一定程度上缓解孤独、提供情感支持，但长期使用可能对用户心理健康和现实社交关系产生负面影响。
据新华社报道，芬兰阿尔托大学（Aalto University）牵头的一项研究发现，AI的陪伴与互动能够为用户提供安慰和支持，但与此同时，用户语言中也出现更多焦虑、孤独和抑郁，甚至是自我伤害念头的迹象。
据阿尔托大学周二发布的公报，这项研究基于一款被设计为用户的虚拟朋友、导师甚至情感伴侣的AI聊天机器人Replika。
数据显示，在开始使用这类AI系统后，用户在网络平台上的发帖内容更加频繁地涉及人际关系话题，但表达中负面情绪信号也有所增加。
研究人员指出，AI陪伴能够提供持续、无条件且不知疲倦的回应，这对存在社交困难的人群具有较强吸引力。但随着时间推移，它也可能抬高用户面对现实人际关系的心理门槛，使人更难适应现实关系中的复杂性、不确定性和情感投入，进而减少与他人的主动联系。
研究人员提醒，随着AI快速发展，人们在使用相关技术时需更加谨慎。

周三，中国人工智能公司智谱AI开源其最新旗舰模型GLM-5.1，并同步将API价格上调10%，显示在与美国对手竞争加剧的背景下，开始加强对高级模型的商业化变现。此次为公司今年第二次涨价：2月对编程订阅的调整曾上调逾30%，本次幅度较小但覆盖包括云端输入token在内的多项计费类别。消息发布后，智谱港股盘中上涨约11.5%。
定价方面，智谱对输入token每百万收费约1.40美元、输出约4.40美元；而Anthropic的Claude Opus 4.6截至2026年2月分别为每百万输入5美元、输出25美元，差距仍在。GLM-5.1上月末先作为付费编程订阅产品推出，此举引发其从开源策略向更强调变现的猜测。
基准测试机构Artificial Analysis认为，按综合智能指标计算，GLM-5.1为全球最强开源模型，领先国内竞争者如MiniMax，但仍落后于OpenAI与Anthropic的顶级模型。智谱在X上称，GLM-5.1可在复杂工作流中持续运行约8小时，能从零构建可用的Linux桌面，反映对可长时自治运行的agent型AI工具的需求。
智谱定位为中国硬件生态中的重要软件合作方，并称已与多家国产芯片企业合作适配其GLM模型，包括华为、摩尔线程与寒武纪。摩尔线程表示其MTT S5000服务器已实现对GLM-5.1的“Day-0”兼容性。
智谱CEO张鹏在3月中关村论坛表示，依赖低价竞争不利行业长期发展，提价旨在建立“更健康的商业闭环”，支持高质量AI服务的长期交付；小米AI模型负责人罗福立也表达了类似看法。
