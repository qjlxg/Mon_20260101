3月中国国内能源价格指数环比大涨16.5%

中国物流与采购联合会大宗商品交易市场流通分会副会长周旭表示，3月，中国大宗商品价格指数大幅上涨是国内大宗商品市场景气水平回升、政策效应逐步显现，以及中东地缘政治持续紧张等因素共同作用的集中体现。

周旭指出，当前外部不确定不稳定性因素交织，部分行业生产成本上涨压力较大。面对新的形势，产业企业需进一步加强外部输入性风险研判与管控，拓宽原材料来源和替代性资源利用，增强市场风险抵御应对能力。

3日（泰历2569年）——民主党副党魁กรณ์·查蒂卡瓦尼ช向新任财政部长兼油价成本审查委员会主席เอกนิติ致意，并呼吁重新检视成品油定价的成本结构，指出过去一个月民众单方面承受了过高油价。
กรณ์提出须公开且透明查证的7项要点：一、炼油厂实际购入原油成本为多少；二、市场价上涨后炼厂实际购油价与用于计算加工费的参考价差异；三、关联交易商（trader）对炼厂的买卖是否存在转售利润；四、所谓从新加坡计入的“假想”运输费数额；五、炼油的真实加工成本；六、本月异常加工费产生的炼厂利润与正常期比较；七、每日进出油库的库存数据。
若查证发现炼厂获利异常，กรณ์建议两项主要对策：一、修订加工费计算公式，使之反映合理成本；二、征收“风fall税”（Windfall Tax），追回超额利润以直接冲减油基金债务。同时他主张仿效他国，将燃油特别消费税下调6铢并显著下调增值税，且追溯三个月以缓解民众负担。此类措施参考国别包括已降税的印尼、澳大利亚；采用风fall税的印度与英国；以及正在审议的菲律宾、韩国与欧盟国家。
กรณ์强调，全面公开相关数据是确保此次危机能被透明、可检验地解决的关键。

“ถอดรหัสกติกาประชามติ 69”座谈会：修订规则以便利公民投票
4月3日13:00，泰国法政大学（Thammasat University）法学院在塔帕差尼中心211室举办题为“ถอดรหัสกติกาประชามติ เพื่อแก้ไขการลงประชามติในครั้งต่อไป”的座谈会。与会者包括国会议员 ภัณฑิล น่วมเจิม（พรรคประชาชน）、Dr. อุษณีย์ ปฐพีศรีกิจ、iLaw 活动官 เกวลิน ถนอมทอง、以及法学院副教授 รศ. ปริญญา เทวานฤมิตรกุล 等。
会议由เกวลิน列举了影响公投投票的八大问题：一是仅允许在“外县”（นอกเขตจังหวัด）投票，不能在同一省内跨区投票；二是公投媒体宣传少于选举；三是示范选票印刷遗漏导致废票；四是选举委员会（กกต.）禁止政党引导，但无相关法律依据；五是现场官员有引导选民投“不赞成”的情况；六是误导选民在“赞成”栏打勾；七是部分地区存在买票诱导投“不赞成”；八是假消息鼓动投“不赞成”。
Dr. อุษณีย์提到曾尝试网上登记未果并截图，但法院以其未尽充分努力驳回延长登记请求，引发社会呼吁选举委员会行使权限延长登记期。iLaw执行董事นายยิ่งชีพ指出，选举委员会提供的不利信息可能误导选民，未来公投争议将更多。รศ. ปริญญา强调，提前投票登记与异地公投登记时间不对等（前者17天，后者仅3天），选举委员会可依规放宽或延长登记，他主张开通邮寄或线上投票，并呼吁公民社会与政党共同修法，优先考虑公民便利，只修必要条文。
