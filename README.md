特朗普：美国不会让中国接管巴拿马运河

世行：资源丰富的巴布亚新几内亚让国民共享经济增长成果
世界银行周四发布报告警告，作为南太平洋地区最大的国家，巴布亚新几内亚虽受益于黄金和天然气产业带动的经济增长，但大多数国民并未从中受益。
法新社报道，得益于黄金、铜和液化天然气出口价格的飙升，巴布亚新几内亚去年的经济增长强劲，增速从上一年的3.4%升至5.6%。
但世行在星期四（7月2日）发布的报告中指出，这些行业仅提供了6%的正规就业岗位，且外资天然气项目的大部分收益都流向海外。
世行发现，若计入人口快速增长因素，巴布亚新几内亚去年的就业率实际上下降了0.3%。
报告描绘了一幅严峻图景：巴布亚新几内亚每两名5岁以下儿童中就有一人发育迟缓，近四分之三的10岁儿童在离校时仍不识字；而在有工作的民众中，80%从事非正规工作。
报告形容，巴布亚新几内亚是个“经济在增长，却将大多数民众抛在身后”的国家。
世行说，巴布亚新几内亚需要制定更明确的就业议程，加强基础设施建设，改善教育与医疗条件，并实施改革，为私营企业投资和招聘创造更有利条件。

他说，台湾拥有优秀人才、完善基础设施和成熟产业生态，具备建立全球竞争优势的条件；美国则拥有庞大的国防需求、系统整合能力及研发资源，能够协助台湾厂商对接民主国家市场。双方可共同打造以“民主阵营”为基础的无人机生产体系，进一步强化自由世界的整体威慑能力。

他说：“对台湾而言，最有效的吓阻方式，莫过于把台湾打造成为一个由空中、水面和水下无人机组成的‘蜂巢’。”

台湾行政部门随后提出总额2100亿元新台币（85亿新元）的无人机发展计划，拟在2031年底前投入监侦无人机、岸防攻击无人机及小型无人水面载具等项目。

赖清德星期三（7月1日）在民进党中执会上指出，无人机建设已刻不容缓。他说：“面对地缘政治局势变化及现代战争形态演进，建构不对称作战能力是一项与时间赛跑的国防工程。”
赖清德还重申，不接受北京对台湾的主权主张，强调只有台湾民众有权决定自己的未来。
