农业部周日表示，国内进口洋葱库存仅足以支撑到本地收获季开始。
截至2月13日16:15，对约82%冷藏库的监测显示，红洋葱库存为4,454.09公吨，黄洋葱为5,271.15公吨，几乎全部为进口货。
以红洋葱平均库价每公斤30.36披索计，红洋葱库存预计可维持到2月19日，黄洋葱可维持到3月15日。农业部称，依据现有数据，国内并无进口洋葱过剩。
国家植物检疫部门的资料显示，约8,000公吨红洋葱获卫生与植物卫生进口许可，许可有效期至2月15日，且须在本地收获高峰前到港。植物局指出，即便剩余船货本月全部到港，进口红洋葱库存也可能仅能支撑到3月6日，恰逢收获进入高峰期。
产区估算显示，今年1—3月红洋葱产量仅9,325.14公吨，仅约占52,034.4公吨的18%；用于商业用途的黄洋葱本季度供应预计仅1,645.25公吨。
农业部长弗朗西斯科·蒂乌·劳雷尔表示，“数字显示，当前进口库存并未淹没市场，只是在填补供给缺口。”他并称，农业部将进一步查明农民所称的“季节性降价”原因。
