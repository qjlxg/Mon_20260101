美国总统特朗普的2026年贸易政策议程进一步强调并在一定程度上重塑了对华方针，主张在强化执法的同时推动“可管理”的美中贸易关系。
美国贸易代表詹姆森·格里尔（Jamieson Greer）周一向国会提交了2026年贸易政策议程和2025年年度报告，称这是美中贸易关系的重要转向。议程将“为实现互惠与平衡而管理对华贸易”列为六大优先事项之一，报告同时称贸易逆差已下降，并在特朗普与习近平会晤前承诺加强对《贸易法》第301条的执法力度。

德黑兰 — 三名知情伊朗内部商议的官员称，负责选举下一任最高领袖的高级神职人员于3月3日召开会议，已明显倾向于被杀前最高领袖的次子莫杰塔巴·哈梅内伊（Mojtaba Khamenei）。
官员们说，宗教领袖们曾考虑在3月4日上午宣布莫杰塔巴为继任者，但有人担心此举会让他成为美以的目标；相关人士均要求匿名以讨论敏感内部事务。
据称专家议会（Assembly of Experts）在上午和晚上各举行一次视频会议。与伊朗伊斯兰革命卫队关系密切的法尔斯通讯社报道，以色列曾轰击设在什叶派重镇库姆的一栋拟供议会开会的建筑，建筑当时无人。
现年56岁的莫杰塔巴长期在已故父亲阿里·哈梅内伊的权力体系幕后活动，性格低调但影响力大。伊朗政府称，2月28日美以袭击中除阿里·哈梅内伊外，莫杰塔巴的妻子扎赫拉、母亲曼苏雷和一名儿子也被击毙。
三名官员称，掌握武装力量的革命卫队主张提名莫杰塔巴，认为他熟悉安全和军事协调，适合在危机中掌舵。约翰斯·霍普金斯大学伊朗问题专家瓦利·纳斯尔表示，若他当选，意味着更强硬的卫队派系正在掌权。
德黑兰分析人士预测，部分民众会强烈反对并产生反弹，而政府支持者则会把他视为“殉难者”的延续迅速支持。其他入围人选包括过渡领导三人委员会成员、法学家阿里雷扎·阿拉菲和被视为相对温和的哈桑·霍梅尼（霍梅尼孙子）。
与莫杰塔巴关系密切的政治人物阿卜杜勒雷扎·达瓦里曾称，如若继位，他可能像沙特的穆罕默德·本·萨勒曼那样推进变革并边缘化强硬派。美国总统在华盛顿发布会也警告，自2月28日美以攻势以来，许多潜在接班人已被斩断，局势充满不确定性。
专家议会由88名经公选产生的什叶派高级神职组成，依宪负责任免、监督与罢免最高领袖。这将是伊斯兰共和国47年历史上第二次产生最高领袖。

华盛顿 — 五名知情消息人士称，美国政府计划于3月6日在白宫与主要国防承包商高管会面，商讨加快武器生产以补充因近期对伊朗等军事行动而消耗的弹药库存。
受邀参会的企业包括洛克希德·马丁和雷神母公司RTX等关键供应商。与会人士表示，会议将强调华盛顿在伊朗行动大量动用弹药后，迅速恢复库存的紧迫性，并可能敦促军工企业提高产能和提速交付。
消息称，国防部副部长史蒂夫·范伯格正牵头制定约500亿美元的追加预算请求，或于3月6日一并公布，用于替换近期冲突中消耗的武器。该数字为初步估算，或有变动。
报道指出，自俄乌战争和以色列在加沙的军事行动以来，美国已动用数十亿美元库存，包括火炮、弹药和反坦克导弹；对伊朗的打击消耗了更远程的导弹。2月28日美军对伊朗出动战斧巡航导弹、F‑35战机及小型无人攻击机后，生产压力进一步加大。雷神与五角大楼已有协议，最终目标将战斧年产量提升至1000枚；五角大楼计划2026年采购57枚，单价约130万美元。
白宫、五角大楼与洛克希德未立即回应置评请求，RTX拒绝评论。特朗普3月2日在社交媒体称美弹药“几乎无限”，可支持长期作战。政府也在施压军工企业将产能置于股东分红之上，并已签署行政令识别绩效不佳的承包商，相关公司将被要求在15天内提交整改计划，否则可能面临合同终止等执法措施。  路透社

特朗普对伊朗的打击震动了俄罗斯国内强硬派。2025年他重返白宫时，部分鹰派曾对其不可预测性和交易性抱有谨慎乐观，认为可能有利于莫斯科在乌克兰问题上取得好处。但这次对伊朗的军事行动使许多人改观，开始把他视为对俄罗斯本身的威胁。
民族主义富商马洛费耶夫称“没有原则的美国威胁全世界”，认为华盛顿既想要一个弱小的欧洲，也想要一个弱小的俄罗斯。网红战争博主“卡萨德上校”称特朗普为“怪物”，不可信。学者西多罗夫在国营电视上更直言，特朗普是“危险人物”，并对其在2024年幸存刺杀表示遗憾。
克里姆林宫虽指责美方为“无端侵略”，但避免直接抨击特朗普，并未对伊朗提供实质性援助，仍表示有继续推进乌克兰和谈的利益。同时，分析人士认为莫斯科目前难以在实质上帮助德黑兰。另有观点称，油价走高与美国在中东长期投入可能在财政与军援分配上给俄方带来短期利好。
强硬派担心：特朗普正在系统性削弱俄罗斯的盟友——叙利亚、委内瑞拉、伊朗乃至古巴的命运令他们不安。极端民族主义者杜金警告，若伊朗倒下，“下一步可能是我们”。（路透）
