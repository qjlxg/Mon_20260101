这是井泉自去年12月在马尼拉履新以来，首次被小马可斯召见。井泉的前任黄溪连自小马可斯政府于2022年上台后曾被召见六次，其中五次由菲外交部出面。
在与小马可斯进行数小时的闭门会谈后，井泉向当地媒体形容此次会面“非常具有建设性”。他澄清：“这并非某种外交抗议或交涉，没有相互指责，也没有激烈的言辞交锋。我认为这是一次深入的意见交换。”

贝森特：美国将审查中国AI模型是否窃取知识产权
随着中美在尖端人工智能（AI）领域的竞争日趋激烈，美国财政部长贝森特说，美国将仔细审查来自中国的开源AI模型，查看是否存在窃取知识产权的行为。
据彭博社报道，贝森特星期二（7月21日）接受福克斯商业频道采访时说：“我们看到很多有关开源模型正不断涌现，并对美国大型语言模型构成威胁的讨论。本届政府支持开源模型，但我们不支持窃取知识产权。尤其是如果我们发现海外模型窃取我国优秀企业的成果，我们有能力对它们实施制裁。”
中国AI企业月之暗面近期发布先进的开放权重模型Kimi K3，引发市场对美国可能在AI竞争中失去领先优势的担忧，同时也加剧了外界对硅谷企业AI基础设施巨额投资回报前景的疑虑。

贝森特说，美国政府在许多中国AI模型中发现美国大型语言模型的水印，这种情况“不可接受”。他说：“因此，我们将在未来几天或几周内对此展开调查。”
贝森特还称，美国是AI领域的“发源地”。作为财政部长，他近年来积极参与AI和对华贸易相关事务，曾与银行和投资机构讨论AI安全风险，也参与推动设立独立监管机构，以评估AI模型的安全性。
他也提出，使用中国AI模型的企业是否应向客户披露相关情况，并称“你不能使用假冒产品”。
此外，美国贸易代表格里尔星期二接受美国消费者新闻与商业频道采访时说，特朗普政府也将审查中国是否通过不公平手段，在全球AI领导地位的竞争中扩大对美国的优势。
格里尔还说：“我们正密切审视中国如何推动AI发展，以确保我国企业能够重新在公平的竞争环境中参与竞争。”

香港高院再驳回宗馥莉上诉申请 维持冻结18亿美元资产
中国饮料巨头杭州娃哈哈集团的继承人宗馥莉，就解冻汇丰银行账户内18亿美元（约23亿新元）资产向香港高等法院提出的上诉申请再次被驳回。这意味着汇丰银行账户的保全令及配套披露令继续有效。
综合澎湃新闻和《上海证券报》报道，香港高等法院星期二（7月21日）就已故娃哈哈创始人宗庆后的家族遗产纠纷案颁布最新裁决。法庭驳回被告宗馥莉及建浩创投（Jian Hao Ventures Limited）的上诉申请，维持2025年8月1日作出的资产保全及披露命令，并下令宗馥莉须承担25万港元（约4万1200新元）的诉讼费。
案缘2025年8月1日，香港高等法院颁布判案书，支持作为原告的宗庆后三名境外子女宗继昌、宗婕莉、宗继盛的诉求，确认存在信托争议的汇丰账户的资产保全令。
这意味着，宗馥莉和建浩创投作为被告在杭州相关诉讼取得进一步结果前，不得提取或抵押账户的相关资产，同时需披露账户的最新余额、资产去向及收支完整账目。
截至2024年5月31日，这一账户的净资产约达18亿美元。
裁决出炉后，宗馥莉去年9月提出上诉申请被拒，并于去年10月再度提交申请。在最新的上诉申请中，宗馥莉提出了五项理由，包括原审法官错误认定原、被告之间就汇丰账户资产存在信托及受信关系；此外，法庭不应配套批出披露令，因为披露内容与杭州诉讼的诉求存在重复，且在无资产转移风险的情况下无需强制披露。
不过，最新的判案书指出，宗馥莉提出的五项上诉理由均不具备合理胜诉的可能性。
法官在裁决中强调，法院拥有配套作出披露命令的自由裁量权，目的是保障资产保全令能够落地执行。尽管原审法官认可披露内容与杭州诉讼程序可能存在部分重叠，但这并不构成香港法院拒绝发出披露令的法定障碍。法官重申，披露令的唯一目的是保障保全令的有效执行，而非预先裁决杭州实体争议。
原告宗继昌、宗婕莉、宗继盛此前向杭州法院提起诉讼，认为相关资产涉及宗庆后生前安排，主张宗馥莉应按照相关文件约定履行信托义务。宗馥莉方面则认为，“宗氏三兄妹”并不享有相关账户资产权益。
此次香港上诉庭裁定针对的是资产保全程序问题，并不意味着法院已经认定相关资产归属于哪一方。围绕协议效力、信托关系是否成立以及相关权益如何分配等问题，仍需等待杭州诉讼进一步审理。
