阿尔及利亚颁布法律 认定法国殖民统治构成犯罪
阿尔及利亚正式颁布法律，认定法国对阿尔及利亚长达132年的殖民统治构成犯罪。法律条文已生效。
新华社引述阿尔及利亚星期五（5月22日）报道，阿尔及利亚当天颁布的法律规定，法国从1830年至1962年对阿尔及利亚的殖民统治及“持续至今的直接和间接影响，”构成“国家犯罪”，并列出了故意杀人、蓄意对平民发动军事袭击、使用非常规和国际禁用武器等31项“无追诉期限”的殖民罪行。
法律还规定，阿尔及利亚将采取一切法律和司法手段，确保法国“正式承认”其殖民历史，并要求法国妥善处理涉核试验有关事宜并归还“被盗”的国库资产。
根据该法律，以任何手段宣扬殖民主义、旨在为法国殖民行为辩护或否认其犯罪性质的人，都将被处以监禁和罚款。
4月，阿尔及利亚参议院通过法案，认定法国对阿尔及利亚的殖民统治构成犯罪。阿众议院已于3月通过同一法案。最终通过的法律删除了此前要求法方正式道歉和赔偿的条款，但保留了法国在撒哈拉地区进行核试验的相关条款。
据阿尔及利亚媒体报道，法国在阿尔及利亚的殖民统治伴随严重暴行，包括在阿境内的撒哈拉地区进行多次核试验，给当地居民健康和环境造成危害。
