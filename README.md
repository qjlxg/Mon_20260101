昆明检测列车撞人致11死 官方调查后惩处34人
中国官方公布昆明一检测列车撞人酿11死两伤事故调查结果，对34名有关人员给予党纪政务处理。
中国国家铁路局网站星期五（7月3日）公布中国铁路昆明局集团有限公司洛羊镇站“11·27”重大铁路交通事故调查报告。
报告称，2025年11月27日，云南省昆明市呈贡区洛羊街道辖区内，中国铁路昆明局集团有限公司55537次检测列车行至洛羊镇站内时，与昆明南工务段施工作业人员相撞，造成11人死亡、两人受伤，直接经济损失1452万元（人民币，约276万新元）。
调查将事故定性为因施工过程中违章指挥、冒险作业，现场未建立有效安全防护体系，企业日常安全管理不严不实，行业监管责任落实不到位等造成的重大铁路交通责任事故。

调查认定，昆明局集团公司及相关单位对事故发生负有责任，建议由成都铁路监督管理局对有关单位存在的违法行为依法给予行政处罚，并责令国铁集团向国务院作出书面检查，昆明局集团公司和成都铁路监督管理局分别向国铁集团和国家铁路局作出书面检查。
调查组对涉嫌重大责任事故罪的驻站联络员移送司法机关处理，将事故调查过程中发现的有关部门和单位的公职人员履职方面的问题线索及相关材料按管理权限，分别移交中央纪委国家监委驻交通运输部、国铁集团纪检监察组，对负有责任的昆明局集团公司、成都铁路监督管理局等单位的34名有关人员给予党纪政务处理。
