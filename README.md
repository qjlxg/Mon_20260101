中国自欧盟进口再度下滑 双方贸易失衡进一步扩大
中国自欧盟的进口三个月来首次下降，双方贸易失衡再次扩大，这一失衡并正导致中欧走向摩擦轨道。
中国海关总署星期二（6月9日）公布的初步和部分数据显示，中国自欧盟进口在经历此前连续两个月增长后下滑1.3%，与中国5月整体进口大幅增长形成鲜明对比。
中国对欧盟的出口增速并放缓至7.6%，为去年10月以来最低增速。与此同时，中国对欧盟的贸易顺差较4月也略有扩大，维持在300亿美元（386.1亿新元）以上。
彭博社指出，这种贸易失衡的程度将继续把外界关注点集中在中欧贸易失衡，这种状况并正促使欧洲官员开始考虑针对中国采取新的限制性措施。
随着双方对峙加剧，中国官方已表示，如果欧盟推进新的贸易限制措施，北京将采取反制行动。
据美国电子媒体“政治报”（Politico）报道，法国总统马克龙计划主持七国集团（G7）与中国之间的一场视频会议，讨论全球贸易失衡问题。
中国海关最新公布的数据显示，中国5月自德国进口同比下降6.2%，扭转了此前连续两个月增长的态势。中国自法国的进口增长为24%，增速与4月大致相同；自荷兰的进口则恢复增长，增幅8.8%。更详细的数据预计将在本月晚些时候公布。

郑丽文坦言两岸分歧逾百年 难以通过和平统一公投
台湾在野党国民党主席郑丽文坦言，两岸分歧已逾百年，现阶段若在台湾举行公投，和平统一的选项很难通过。
综合台湾《联合报》《自由时报》报道，郑丽文于美东时间星期一（6月8日）在纽约出席亚洲协会（Asia Society）座谈会时指出，多年来台湾社会一个稳定趋势是，大多数民众属意“维持现状”。如果现在要求台湾人就“独立或统一”进行表态，支持明确统独的是极少数。
她坦言，两岸分歧已超过一个世纪，而过去10年台湾政府切断所有形式的两岸交流，导致双边敌意上升。因此，若现在在台湾进行民调或公投，“要通过和平统一是很困难的”。但她也强调，大多数台湾民众依然支持两岸对话与交流。
郑丽文补充说，只要确保两岸不会走向永久分裂，双方就能争取到足够的时间与空间，慢慢通过对话化解分歧。她呼吁两岸建立更多机制以稳定交流，并以更多创意构建人类历史上不曾出现的制度，来处理两岸问题。
郑丽文也坦言北京所提的“一国两制”，在台湾非常难被接受。“没有人可以把台湾交还给中国”，台湾人民的意愿至关重要。未来现状的改变要符合两岸人民的意愿，不管什么样的安排，最重要的是台湾人民的意愿必须被尊重。
她重申，“中华民国宪法”是一部“一中宪法”，九二共识完全符合“中华民国宪法”。在九二共识的基础及两岸同属一个中华民族的身份认同上，“维持现状可以有很大的空间”，可以建立一个制度化、和平的两岸关系。
郑丽文强调，国民党所期待的两岸关系，不是某一方牺牲、妥协、或放弃自己的制度。

中国5月份PPI连续第三个月上涨
中国官方公布，5月份的全国工业生产者出厂价格（PPI）连续第三个月上涨，并写下近四年新高。
中国国家统计局星期三（6月10日）公布，5月份的官方PPI同比上涨3.9%，高于路透调查预估的3.8%，创下2022年7月以来的新高，涨幅也较4月份的2.8%进一步扩大；全国PPI环比上涨0.5%，涨幅比上月回落1.2个百分点。

统计局城市司首席统计师董丽娟分析，算力需求增长等带动有色金属、电气机械和计算机相关行业价格上涨，且国际原油价格波动传导影响国内相关行业价格由涨转降。

数据显示，5月份的全国CPI环比下降0.1%。其中城市下降0.1%，农村下降0.1%；食品价格下降0.4%，非食品价格下降0.1%；消费品价格下降0.2%，服务价格下降0.1%。
路透社指出，CPI和工业生产者购进价格指数均上涨，显示高企的全球能源价格正持续推高制造企业和普通家庭的成本。

中国联通警告美拟议电信禁令恐将扰乱全球通信
中国联通美国分公司警告称，特朗普政府拟禁止美国电信运营商与被视为“国家安全风险”的中国电信企业互联互通，此项提议恐将对全球通信网络造成严重破坏。
据路透社报道，中国联通在星期二（6月9日）提交给美国联邦通信委员会（FCC）的一份文件中表示，该禁令将“对在中国拥有重大商业和供应链利益的美国公司造成伤害”。
FCC此前提议以国家安全风险为由，禁止美国本土及其他在美运营的电信企业与中国联通、中国移动、中国电信等中资公司进行互联。
中国联通对此回应称，中资电信运营商共同构成了中美这两个全球最大经济体之间通信流量的主要网关。“全面切断与这些实体的互联，将从根本上割裂全球通信网络的关键一环。”
此外，代表AT&T、威瑞森（Verizon）、T-Mobile等美国运营商的行业组织美国电信协会（USTelecom）也向FCC提交文件，反对撤销对中国运营商的互联授权。
该组织指出，这项提议“可能会降低美国网络服务提供商对高风险流量的可见度，迫使通信数据通过外国或缺乏监管问责的中间商进行传输，不仅给日常商业和运营通信带来不确定性，还会破坏那些依赖已知联系节点和既定上报路径的技术保障措施。”
近年来，FCC持续加强对中国电信企业的限制。2021年，FCC将华为、中兴通讯、海能达、海康威视和大华股份的电信及视频监控设备列入受限清单，认定这些企业对美国国安构成风险。2022年11月，FCC决定不再批准这些企业新型号设备在美国销售或进口。

费城 — 随着人工智能在可见规模上替代劳动力，华盛顿正紧急寻求应对方案。
特朗普准备在白宫会见顶级AI公司领导人，讨论一项曾被视为边缘的想法——政府入股AI企业以分享其收益。
生成式AI自2024年商业化成熟，至2026年大批岗位被取代，已造成超过115,000个工作岗位流失。宾夕法尼亚大学2026年2月民调显示，仅17%美国人认为AI在未来十年会带来积极影响；对医学研究持乐观者为57%，对政府效率、经济及美中关系的期待分别仅24%、19%和5%。
对政策制定者的不满跨越党派：近三分之二受访者认为政府对AI监管做得太少。
工会要求更多话语权与保障：美国劳工联合会—产业组织（AFL-CIO）要求AI使用透明、防止因AI导致裁员或降薪，并禁止将招聘、解雇与纪律决策完全自动化；通信工人联合会和新闻工会在合同与仲裁中争取AI保护；代表22.5万注册护士的全国护士联盟要求任何影响病患护理的技术须经工会批准。
布鲁金斯学者达雷尔·韦斯特指出，工会措施必要但不足，必须配套政府主导的再培训，而特朗普政府已削减相关联邦项目。
数据中心成为社区矛盾焦点：大型数据中心被指噪音大、耗水耗电、推高居民费用，环保活动家艾琳·布罗科维奇收集了近4,000起相关投诉，执法部门亦警告这些设施可能成为极端分子的目标。
政策回应零散：参议员伊丽莎白·沃伦主张对企业、资本利得和AI基础设施加税；伯尼·桑德斯提出“美国AI主权财富基金”法案，拟对每家AI公司一次性征收50%股票并入公有基金，赋予公众表决权、董事席位和未来分红，以共享AI财富。
该设想在市场经济中具争议，也有AI公司高管表示支持。特朗普表示支持此类让公众分享AI利益的做法，但批评者警告这会产生监管与投资的利益冲突，并指出入股无法阻止岗位流失，仅能为再培训或失业补偿提供资金。
布鲁金斯研究显示，高AI暴露度地区多为民主党倾向的城市中心，这些地区对AI驱动的岗位流失更为焦虑。2026年中期选举或将成为检验AI议题政治影响的首场战役，率先提出解决方案的一方可能掌握话语权，受失业或担忧影响的选民正密切关注。

华盛顿，6月9日——美国政府问责局（GAO）周二发布报告称，位于德州、由特朗普政府建成的联邦移民拘留中心“东蒙大营”（Camp East Montana）在规划与监管上存在“重大、普遍性问题”。
报告指出，拘留中心未能提交使用武力报告，未向病情严重的被拘人员按时提供药物，并因加急合同流程导致数千万纳税款浪费。该营区于2025年8月在埃尔帕索的布利斯堡军事基地兴建，ICE通过加速的军事承包程序将合同授予此前无拘留经验的小型公司Acquisition Logistics LLC。
GAO对2026年1月发生的两起死亡事件处理提出质疑：一案被裁定为他杀，拘留中心未向ICE提供使用武力或死亡报告，且证据“丢失或被销毁”；另一案为自杀，涉事人员被置于医疗留置室而非防自杀牢房，且无人看管超过15分钟。
此外，ICE医务人员在2025年12月视察时发现，医疗承包方未能为慢性病被拘者提供治疗方案——例如糖尿病和HIV患者均无治疗计划。
ICE与Acquisition Logistics未立即回应置评。该营区合同在2026年3月被突然转交给Amentum Services Inc。报告还指出，特朗普政府正寻求拘押创纪录的移民人数；据两名知情人士，截至6月初约有57,000名被拘者，较2025年上任时的约40,000人大幅上升。
