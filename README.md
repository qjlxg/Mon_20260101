中国外交部发言人林剑星期二（2月10日）在例行记者会上说，日本如果真心想发展中日战略互惠关系，很简单也很明确，就是撤回高市早苗错误涉台言论，恪守中日四个政治文件和日方作出的政治承诺，以实际行动展现对话的诚意。

本周，高市早苗领导的自民党在众议院选举中取得大胜后，林剑星期一（2月9日）在记者会上说，中国对日政策始终保持稳定性和连续性，不会因日本某一次选举而发生变化，并再次敦促日本撤回高市的“涉台错误言论”。
