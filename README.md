美国本流感季已有病例至少2000万起
美国疾病控制和预防中心星期五（1月30日）公布的最新数据显示，估计本流感季美国累计流感病例已有至少2000万起，其中住院病例至少27万起，死亡病例至少1万1000起。 
新华社报道，美国疾控中心说，全美季节性流感活动目前仍处于高位。在连续三周下降后，截至1月24日的一周内，流感活动再次出现回升。甲型H3N2流感病毒是本流感季占主导地位的流行毒株。
数据显示，截至24日的一周为自2010年至2011年流感季以来，流感相关住院人数第二高的一周，其中18岁以下儿童的住院率达到自2010年至2011年流感季以来的最高水平。当周美国新增八起与流感相关的儿童死亡病例，本流感季儿童死亡病例总数升至52起。
据美疾控中心网站介绍，美国流感季通常在秋冬两季，每年12月至次年2月为高峰期。
美疾控中心建议每年接种流感疫苗，这不仅有助于预防流感感染，还可降低重症、住院和死亡风险。此外，感染流感后应尽早开始使用抗病毒药物，这对老年人、慢性病患者等发生严重并发症风险较高的人群尤为重要。
