受人工智能提振 台湾6月外销订单创历年单月新高
受人工智能（AI）热潮的强力提振，台湾6月外销订单增幅远超预期，创下历年单月新高。
综合路透社和台湾《经济日报》报道，台湾经济部星期二（7月21日）公布的数据显示，6月份外销订单金额达952.6亿美元（约1229亿新元），较去年同期大幅增长59.4%。
这一表现不仅大幅优于分析师预期的49.5%，还连续第17个月实现正增长。
按产品类别观察，新兴科技应用需求扩增带动了相关供应链的接单畅旺。6月资讯通信及电子产品接单分别同比暴增81.9%及79.9%。相较之下，仅光学器材因面板备货需求降温，接单同比微降3.3%。
传统产业方面，同样受惠于AI产业的蓬勃发展，相关供应链需求维持高位。加上电子及工业用化学、塑料制品接单增长，带动机械、基本金属、化学品、塑料橡胶制品6月接单分别同比增长17.6%、16.5%、3.4%及1.8%。
从主要接单地区来看，来自美国的订单继5月增长63.9%后，6月进一步飙升83.6%。来自中国大陆的整体订单则同比增长37.2%，与5月份的37.5%基本持平。
此外，欧洲和日本的订单也分别录得53.9%和37.9%的显著增长。
路透社分析指出，尽管市场对AI泡沫、全球通胀及中东冲突重燃的担忧，正考验着当地晶片与科技产业的增长前景，但最新贸易数据表明，台湾的出口动能依然强劲。
经济部数据显示，台湾第二季外销订单达2721.9亿美元，上半年累计接单5041亿美元，双双刷新历年同期纪录。
展望未来，台湾经济部预估，7月外销订单金额将落在935亿至955亿美元之间，同比增长54.5％至57.8％。
台湾经济部分析，尽管地缘政治风险与全球贸易政策等不确定性将继续拖累全球经济复苏，但云端服务供应商扩大资本支出（capex）等有利因素，有望继续支撑目前的强劲接单势头。

一份研究报告指出，过去六年，中国批准上市的抗癌新药数量超过美国，显示对中国生物科技行业的投资正在提升其全球竞争力。
据彭博社报道，上述报告星期一（7月20日）发表在《卫生事务》（Health Affairs）指出，在2020年至2025年间，中国国家药品监督管理局共批准94款新型肿瘤药上市，美国食品药品监督管理局则在同期批准87款。中国在2025年批准的抗癌药数量接近美国的三倍。
上述研究由宾夕法尼亚大学和癌症研究之友（Friends of Cancer Research）的研究人员完成。
报告的数据显示，在研究覆盖时段的2020年至2022年间，美国食品药品监督管理局批准的药物数量均高于中国，但这一局面在2023年出现逆转，此后直至2025年中国一直领先美国。
不过，数据也显示，中国获批的许多肿瘤药物已是在美国上市一段时间、中国本土版本。在将同类首创药物推向市场方面，美国仍占据主导地位。
报告称，在研究覆盖时期内，在36款用于人体特定癌症通路的同类首创新药中，有30款首先在美国获批上市。按中位数计算，美国食品药品监督管理局审批新药上市申请所需时间要比中国国家药品监督管理局短117天。

中国推动平台让利个体工商户 降低佣金抽成减轻创业成本
中国官方出台新方案，要求引导平台企业为新入驻个体工商户提供让利服务，降低佣金和抽成等费用，减轻商户运营初期成本压力。
中国市场监管总局星期一（7月20日）在官方发布《推动平台企业赋能个体工商户发展行动方案》，聚焦个体工商户数字化转型中的痛点堵点，从降成本、增加流量、教技能等方面推出系统性帮扶举措。
方案明确，到2027年年底前，滚动形成一批帮扶举措、一批优质商户、一批培训课程、一批应用场景、一批典型案例等“五个一批”工作成果，探索构建平台企业赋能个体工商户发展的长效实践路径，进一步降低个体工商户数字化转型成本。
针对个体工商户创业初期成本压力较大的问题，方案提出，鼓励平台企业简化入驻流程，优化智能辅助功能，缩短信息填报和审核时间；同时，引导平台企业为新入驻个体工商户提供让利服务，合理降低佣金、抽成、会员费、技术服务费、信息服务费和营销推广费等收费，减轻个体工商户运营初期成本压力。
在经营扶持方面，方案要求，支持平台企业结合自身经营特点和优势，为新入驻个体工商户提供更多曝光机会；鼓励平台企业将扶持举措、获取方法等基础性内容纳入新入驻经营主体培训。
此外，《行动方案》还部署开展能力提升行动，提出鼓励平台企业加大对平台内个体工商户培训力度，强化经营指导，帮助个体工商户了解掌握市场行情动态、电商运营策略、数字化工具使用等方面技巧知识，为个体工商户成长提供专业化、体系化指引。

英国政府启动降低生活费计划 削减能源账单
英国新任首相伯纳姆星期二（7月21日）启动降低民众生活费的政策计划，其中包括拟取消家庭电费账单中的增值税。
法新社报道，伯纳姆说，政府将从10月起取消电费账单中的增值税。这项政策预计将使英国政府本财年减少约8亿5000万英镑（约14亿7000万新元）的税收。
伯纳姆在声明中说：“我们正在立即采取行动，降低能源账单的税费，让民众口袋中有更多钱。”
英国政府说，这项措施平均每年将为普通家庭节省约45英镑。“这将惠及最贫困家庭，因为他们的收入中用于能源支出的比例更高。”
这项措施预计将在10月开始实施，恰逢英国能源监管机构上调电力和天然气价格上限。此次能源价格上涨与美伊冲突有关。
伯纳姆还说，希望减轻小型企业面临的压力。这些企业此前受到前工党政府加税政策的冲击。
伯纳姆还承诺遵守斯塔默政府时期实行的严格财政纪律，即确保政府税收收入与日常公共支出保持平衡。
英国国家统计局星期二公布的官方数据显示，6月份公共部门的净借款为160亿英镑，同比下降三分之一；失业率维持在4.9%不变。
英国金融市场星期二整体保持稳定。
伯纳姆的长期目标是推动英国“再工业化”，并大规模建设公共住房。
尽管工党重新执政两年来，英国经济一直未能实现强劲增长，但国际货币基金组织预计，今年英国经济增速将超过其他主要经济体。

中国海关查获藏匿在旅客裤袋两濒危活鹦鹉
中国海关查获藏匿在旅客裤袋的两只濒危活体鹦鹉。
中国海关总署办公厅星期二（7月21日）在微信公众号通报，罗湖海关在旅检进境渠道，连续查获两名女性旅客在各自裤子口袋内藏匿活体鹦鹉。
根据通报，海关关员在对进境旅客进行监管时，发现两名旅客相继进入海关监管区，均未主动申报。两人裤子宽松，口袋处有明显鼓起，海关关员立即对两人进行拦截检查。
关员在两名旅客的裤子口袋内，各发现一个黑色密封塑料袋，并感受到袋内有活物蠕动迹象。拆开后，关员发现用半透明纱网包裹的活体鹦鹉两只，脚部均被捆绑。
