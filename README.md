台行政院副院长：台湾为全球首个获美232关税最优惠待遇
台湾行政院副院长郑丽君在越洋记者会上说，台湾为全球首个获得美国232关税最优惠待遇。
综合联合新闻网和中时新闻网等台媒报道，台美关税谈判经过多轮磋商后，在美东时间星期四（1月15日）达成协议，台湾获得与日、韩、欧盟齐平的15%不叠加对等关税。
由台湾行政院副院长郑丽君、行政院经贸谈判办公室总谈判代表杨珍妮所率领的台方谈判团队，于美东时间1月15日与美国商务部长卢特尼克、美国贸易代表格里尔率领的美方谈判团队达成贸易协议，并共同见证驻美国台北经济文化代表处、美国在台协会于美国商务部共同签署投资合作备忘录。
郑丽君当天在台驻美代表处召开的记者会上说，经过九个月的谈判，台湾达成预定的四项目标，包括对等关税调降为15%，且不叠加原MFN（最惠国待遇）税率，获得美国主要逆差国中“最优惠盟国待遇”，与日、韩、欧盟齐平、成为全球第一个“为本国对美投资业者争取到半导体、半导体衍生品关税最优惠待遇的国家”、以“台湾模式”引领业者进军美国供应链，打造产业聚落、促成台美高科技领域相互投资。
郑丽君指出，台美已经完成谈判，“我们是全球第一个获得美方所承诺的，不论未来第二阶段半导体及其衍伸品的关税为何（232条款相关关税），都将获得最优惠待遇。”　
行政院指出，台湾的半导体与科技企业将对美进行2500亿美元（3220亿新元）的新增直接投资，台湾政府协助提供赴美投资的企业2500亿美元的信用担保。
行政院强调，美方认同“台湾模式”，台湾产业投资也全由企业自主规划，不受美国政府项目清单限制、与日韩有所不同。
对于台湾企业对美投资达2500亿美元，外界担忧产业外移，台湾是否被掏空，郑丽君说，以台湾模式跟美国磋商供应链合作，最重要的战略目标是：这不是产业外移，而是科技产业的延伸跟扩展，“供应链合作不是move，而是build。”

两名开发者（化名王、李）于2023年开发“情感陪伴”机器人AlienChat（AC），2024年4月因模型聊天记录中出现露骨内容被警方拘留。法院文件显示，至被捕时该机器人有11.6万注册用户，其中2.4万为付费用户。徐汇区法院对150名付费用户中12,495条对话抽样，认定3,618条含“淫秽内容”。法院称两被告“编写并修改系统提示以规避大模型伦理约束”，将AC训练为可持续输出淫秽内容的工具。 
被告人王在法庭上否认有意开发色情机器人，其辩护律师周晓阳称，王修改提示是为提升模型的智能性与情感回应能力，所用基础模型本身易生成色情内容，相关技术专家的证词亦支持这一点。辩方并指出，多为一对一私密对话、由用户引导，难以准确评估对社会造成的损害，主张以警告或整改等行政手段为宜，而非刑事处罚。 
此案在法学界引发争议：有观点认为即便对话私密，模型被设计为生成可被用户获取的露骨内容，已扰乱社会管理秩序并存在社会危险；也有学者如海南大学颜二鹏认为，开发者更多提供工具，是否应承担与内容创作者同等责任值得商榷。 
背景方面，近年来中国各地推动AI产业发展，但相关监管相对滞后。2023年8月发布生成式AI条例性文件，要求服务提供者承担网络信息安全责任，11月又提出更具体的过滤与拒答要求。此案之后，业界出现更广泛的自我审查与敏感词、屏蔽机制升级。辩护律师认为，考虑到领域新颖与开发者并非构建基础模型，适用刑法并不合适。
