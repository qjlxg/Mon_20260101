担忧高油价推高通胀 多名美联储官员认为可能有必要升息
（华盛顿综合电）美国联邦储备局越来越多官员认为，可能需要调高利率以抑制高油价所造成的通货膨胀压力。
美联储星期三（4月8日）公布的3月17日至18日联邦公开市场委员会（FOMC）会议纪要显示，更多美联储官员对通胀感到担忧，敦促同事们考虑在会后声明中提及，若出现特定情况，可能调高利率。
会议纪要写道：“一些与会者认为，有充分理由在会后声明中对委员会未来利率决策采用双向表述，以表明如果通胀维持在目标水平之上，上调联邦基金利率目标区间可能是适当的。”
相比之下，在1月会议纪要中，认为可能须要加息的官员仅有“几位”。在美联储的用语中，“一些”（some）所指的人数多于“几位”（several）。
在3月会议上，美联储连续第二次将利率维持在3.5%至3.75%区间不变，但警告中东战争为经济前景带来新的不确定性。美国和以色列2月28日对伊朗开战，伊朗随即封锁石油运输咽喉要道霍尔木兹海峡，导致油价飙涨。 
美联储肩负着双重使命，既要将通胀率降至2%的目标，也要最大程度促进就业。加息虽有助于抑制通胀，但会加剧经济下行压力，推高失业率；降息可刺激经济活动和就业，但会推高物价。
根据会议纪要，美联储多名官员指出，在油价持续上涨的情况下，通胀高企的风险可能会持续比预期更长的时间。
他们担忧中东冲突引发的高油价效应可能外溢到不包括食物和能源价格的核心通胀，也担忧通胀连续几年高于目标水平后，民众的长期通胀预期可能对能源价格的上涨更为敏感。
他们指出，若战争旷日持久，通胀向2%目标回落的速度可能比此前预期更慢，并认为通胀持续高于2%的风险已有所上升。
桑坦德集团首席美国经济师斯坦利说，会议纪要显示，美联储认为通胀和就业方面的风险都在上升，同时也担心战争若拖得更久，风险将进一步加剧。
在3月会议上，大多数美联储官员预期失业率会保持平稳，但认为就业市场的风险偏向下行。 
在会议过后发布的预测中，美联储官员暗示今年可能会有一次降息，这与去年12月的预测相同。不过根据联邦基金期货市场的反应，投资者不认为美联储会在今年内降息。

北京未成年人严重犯罪数逐年减少
北京检方发布的白皮书显示，北京检察机关去年办理的未成年人犯罪案件呈现涉校园暴力犯罪大幅下降、办理未成年人八类严重犯罪数逐年减少。
综合北京日报客户端和中新社报道，北京市检察院星期三（4月8日）发布的《北京市未成年人检察工作白皮书（2025）》显示，北京检察机关去年办理的未成年人犯罪案件呈现出以下八个特点。
一是受理审查逮捕和受理审查起诉未成年犯罪人数双下降；二是办理未成年人犯罪案件类型较为集中；三是办理低龄未成年人犯罪数上升；四是办理未成年人八类严重犯罪数逐年减少；五是办理未成年人利用网络实施犯罪数量小幅下降；六是涉校园暴力犯罪大幅下降；七是对涉罪未成年人的帮教力度大幅提升；八是对未成年人的刑罚以轻缓刑为主。
白皮书显示，北京检察机关去年共批准逮捕未成年犯罪嫌疑人109人，提起公诉133人，不起诉202人，做出附条件不起诉决定70人。
根据白皮书，近五年来，北京检察机关受理审查起诉涉嫌故意杀人、故意伤害致人重伤或死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投放危险物质等八种严重犯罪的未成年人数呈逐年下降趋势，2025年同比下降16.67%。

中油油轮有望在美伊暂时停火后返航 台湾加大力度化解“塑胶袋之乱”
美伊刚达成停火两周协议，伊朗同意在此期间安全开放霍尔木兹海峡，台湾经济部长龚明鑫透露，台湾规模最大的石化能源公司中油一艘在波斯湾装载200万桶原油的油轮，有望在这期间返台，舒缓台湾炼油厂储备压力，稳定短期油价走势。
针对台湾近期的“塑胶袋之乱”，龚明鑫也说，中油将提供5000吨平价乙烯给台塑公司，“台塑明天（4月10日）开始出货，速度还蛮快的”。
立法院经济委员会星期四（4月9日）邀请龚明鑫等相关部会首长，就中东冲突加剧市场不确定性，导致包括乙烯、塑胶、化肥、沥青等石化原材料及物流成本增加，并频频传出缺料、囤货等问题的因应策略、现况等进行报告，龚明鑫在会上作上述表示。
美国和以色列在2月底对伊朗发起攻击，中东战事导致霍尔木兹海峡运输近乎停摆，由于该海峡承载全球近五分之一的石油和天然气运量，原油等价格近期一路飙升。
台湾官方为稳定民生物价、缓和通膨压力，虽然宣布暂时冻涨民生用电、桶装瓦斯与天然气等价格，但石化基本原料乙烯价格从2月底战事开打至3月底，大涨至每吨1400美元，涨幅逾97%。这也在台湾引发塑胶袋、医疗耗材等民生物资短缺的“塑胶袋之乱”。
龚明鑫星期四在立法院报告时指出，即便中东局势暂时缓解，国际石化乙烯原料价格大涨，直接影响塑化产业链供需平衡，造成上游原材料价格上涨，引发下游制品短缺疑虑。
他称，为因应塑胶袋短缺问题，政府针对生产端、渠道端提出五大强化措施，包括上游业者扩大生产、下游业者扩大产量、渠道业者稳定供货、设置原材料供应通报网、强化查处遏止不法行为。
在协调上游业者扩大生产方面，龚明鑫说，已协调中油高雄林园石化厂的第四轻油裂解厂提前到4月1日投入运作，预计4月增加乙烯产量1万9000万吨，达到7万9000吨，5月可再增加至9万吨。
他也说，在4月增加的1万9000吨乙烯产量中，会拨出5500吨让台塑和台聚公司生产聚乙烯，作为塑胶原料使用，塑胶原料将从3月的2万2000吨增至4月的2万7500吨，因此可扩大供应。
龚明鑫还称，下游业者也会扩大产量，先由中油专案提供平价乙烯，再由台塑与台聚供应平价聚乙烯给经济部产业发展署合作的20家塑胶制造商，制造相对平价的塑胶袋，该专案可增加 12.5亿个一斤装的塑胶袋产量，快速提供给商圈及其他市场渠道。

以OpenClaw为例，上月约有近千人在腾讯深圳总部排队申请免费安装，本地社交平台上也出现收取数百元安装费的服务，反映出中国市场对AI工具的强烈需求。

中国为在人工智能领域取得突破、应对与美国加剧的科技竞争，正加紧从海外吸纳半导体人才；对此，台湾当局也强化对大陆疑似挖角行为的调查。
台湾法务部调查局（MJIB）上周一表示，已对11家大陆企业展开调查，控诉这些企业以隐瞒大陆背景、设立空壳公司并未经核准在台开展业务等方式，非法挖角半导体及其他高科技人才。MJIB称，自2020年以来已处理约100起相关案件；去年也曾调查大陆最大晶圆代工厂中芯国际（SMIC）涉嫌挖角案。
加拿大顾问机构The Geopolitical Business地缘战略师阿比舒尔·普拉卡什（Abishur Prakash）称，这是一场“低调的科技战”，与美中围绕出口管制和资本等议题的公开对峙不同，大陆更着眼于推动下一波AI创新所需的人才。台湾当局对此高度警觉，调查行动反映出大陆企业对具备经验的人才在半导体与AI领域的强烈需求，即便面临美国对高端技术的限制，这类人才流动仍在加剧。

多家中国企业包括阿里云和知谱科技近期选择不将部分最新人工智能模型开源，转而通过官方付费渠道获取使用价值。 
本周阿里发布三款仅能在其云平台或聊天机器人网站访问的专有模型，其中包括增强编程能力的Qwen3.6-Plus与可处理文本、音频、图像和视频的多模态模型Qwen3.5-Omni。

周三，中国人工智能公司智谱AI开源其最新旗舰模型GLM-5.1，并同步将API价格上调10%，显示在与美国对手竞争加剧的背景下，开始加强对高级模型的商业化变现。此次为公司今年第二次涨价：2月对编程订阅的调整曾上调逾30%，本次幅度较小但覆盖包括云端输入token在内的多项计费类别。消息发布后，智谱港股盘中上涨约11.5%。
定价方面，智谱对输入token每百万收费约1.40美元、输出约4.40美元；而Anthropic的Claude Opus 4.6截至2026年2月分别为每百万输入5美元、输出25美元，差距仍在。GLM-5.1上月末先作为付费编程订阅产品推出，此举引发其从开源策略向更强调变现的猜测。
基准测试机构Artificial Analysis认为，按综合智能指标计算，GLM-5.1为全球最强开源模型，领先国内竞争者如MiniMax，但仍落后于OpenAI与Anthropic的顶级模型。智谱在X上称，GLM-5.1可在复杂工作流中持续运行约8小时，能从零构建可用的Linux桌面，反映对可长时自治运行的agent型AI工具的需求。
智谱定位为中国硬件生态中的重要软件合作方，并称已与多家国产芯片企业合作适配其GLM模型，包括华为、摩尔线程与寒武纪。摩尔线程表示其MTT S5000服务器已实现对GLM-5.1的“Day-0”兼容性。
智谱CEO张鹏在3月中关村论坛表示，依赖低价竞争不利行业长期发展，提价旨在建立“更健康的商业闭环”，支持高质量AI服务的长期交付；小米AI模型负责人罗福立也表达了类似看法。
