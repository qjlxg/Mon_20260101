在伊朗战事持续、霍尔木兹海峡仍被封锁背景下，越共总书记苏林希望通过本周与中共总书记习近平的会晤，深化越中能源合作。
据彭博社报道，上周当选越南国家主席的苏林从星期二（4月14日）起访华四天，将于本周与中国国家主席习近平会面。
报道引述知情人士称，苏林预计将见证包括越南廉航越捷航空与中国商飞在内等越中潜在合作协议的签署，但不清楚是否包含采购或租赁商飞客机。越中也将签署在胡志明市兴建地铁，以及能源和通讯合作等协议。
澳大利亚新南威尔士大学名誉教授泰尔（Carl Thayer）说：“双方将为现有经贸和投资合作受到最低限度的干扰寻求保证。随着伊朗战事的持续，苏林极可能寻求中国加大对铁路连接和能源合作的投资。”
这是苏林集党政大权于一身后的首次外国访问。越南国会上星期二（4月7日）全票选举越共总书记苏林为国家主席。68岁的苏林在今年1月的越共十四大上已成功连任总书记，如今再出任国家主席，未来五年将集党政大权于一身。
在大范围收紧出口的情况下，中国已向东南亚国家提供限量能源援助，包括给越南运送10万桶馏分燃料。
越南称，尽管高成本和运输干扰风险等因素已导致航班被取消，以及加油站大排长龙，该国有足够的燃料撑过4月。
越南高度依赖进口石油和天然气，已推出冻结税负和绿色能源等应急措施。
苏林预计也将寻求中国加大对越南铁路基础设施的投资。在习近平去年访问越南时，越中扩大主要铁路走廊合作，包括耗资78亿美元（99亿新元）建设一条连接越南港口城市海防市和中国边境的铁路。两国也在今年3月签署额外技术协议，中国企业已参与河内的地铁工程。
