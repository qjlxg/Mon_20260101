据彭博社报道，知情人士透露，协会对部分承销商给予窗口指导，城投发行人债务融资工具以2023年3月31日余额为上限，若余额超出，发行人将不能在银行间市场发行新债。
上述知情人士不愿具名，因未获授权公开评论。银行间市场交易商协会是中国银行间市场的自律组织。
报道称，随着10年化债进入收尾阶段，监管层近期正通过一系列举措以管控高成本融资，以及遏制地方融资平台过度举债行为。上周协会要求部分银行避免承销两年期以内的短期债券，叠加本次新规，弱资质城投的银行间市场债务置换压力可能进一步上升。
除上述措施外，国家发改委近期还要求一些中资投行，不要承销收益率高于4%的离岸人民币债券及收益率高于5%的美元债券，以降低高成本借款及过度举债融资，尤其是针对地方政府融资平台。
彭博汇总数据显示，中国地方融资平台整体有16万亿元（人民币，约3万亿新元）存续债券，目前有约5万亿元债券将在2027年底前到期，其中银行间市场约占52.7%。

台积电位于凤凰城外的第一座晶圆厂已于2024年底开始量产4纳米晶片，第二座晶圆厂预计明年下半年开始生产更先进的3纳米晶片。

中国外交部：“朝贡体系”曲解当代中国外交平等相待核心底色
对于亚洲是否会开启新的“朝贡体系”时代，中国外交部称，所谓“朝贡体系”的论调，曲解了当代中国外交平等相待的核心底色。
中国外交部发言人林剑星期四（7月16日）主持例行记者会。林剑说，所谓“朝贡体系”的论调，不过是西方国家用自身霸权的逻辑，镜像中国外交的伪命题，无视中华文明和而不同的历史基因，曲解当代中国外交平等相待核心底色，更背离国际关系基本准则，不符合中国同亚洲乃至世界各国交流合作的现实。
林剑说，中国从不认同国强必霸。中国追求的从来不是威慑他国的霸权，而是和衷共济的智慧和天下大同的愿景。国与国无论大小一律平等，正确的相处之道在于相互成就，而不是单向依附。中国外交始终锚定相互尊重、公平正义、合作共赢，进行真正的多边主义，致力于推动构建人类命运共同体。
他说：“我们也一以贯之秉持睦邻安邻富邻、亲诚惠容、命运与共的周边外交理念方针，是地区安全的稳定锚、发展繁荣的发动机。中方将同各方共同努力，继续为亚洲和世界的和平稳定与发展繁荣注入更多确定性。”
彭博电视《华尔街周刊》（Wall Street Week）节目今年5月15日播出对全球最大对冲基金桥水创始人达利欧的采访。达利欧认为，美国作为一个愿为捍卫自身利益而战的全球大国，其信誉正在下降；与此同时，中国正在积累财富和影响力，从根本上重塑世界其他国家对中美两国的看法，并开启一个类似“朝贡体系”的时代。

港府拟下月开放平台公司申请网约车牌照
香港特区政府将于下月邀请平台公司申请网约车牌照，预计从11月底起陆续批出牌照。
据香港政府新闻公报，港府星期四（7月16日）表示，与规管网约车服务有关的四项附属法例已经完成审议，立法会也已就上述附属法例完成察悉议案辩论。
立法会2025年10月通过《2025年道路交通（修订）（网约车服务）条例》（《修订条例》），订定规管网约车服务的法律框架。
为进一步落实规管制度，港府今年5月27日向立法会提交四项附属法例，以订定规管制度的细节，包括牌证有效期和续期安排、各项费用等、引入德士和网约车的合并驾驶测验、指明网约车车辆许可证的数目的限额，以及订明《修订条例》各项条文的生效日期。
与网约车服务规管制度相关的大部分法律条文将于今年8月3日生效，余下法律条文将于2027年8月22日开始实施。
香港运输及物流局发言人说，随着网约车规管大部分法律条文将于今年8月3日生效，运输署将于星期五（17日）推出网约车服务专属网页，让有兴趣申请牌证者了解相关信息。
发言人称，政府计划最早在今年8月邀请有兴趣的平台公司申请网约车服务牌照，并最早在今年第四季邀请网约车车辆许可证的申请及开始接受网约车车辆驾驶许可证的申请。
发言人续称，港府预计最早在今年11月底起陆续批出网约车服务牌照，让持牌平台随即开展筹备工作，包括招募持有有效许可证的车主兼司机于平台进行登记，并进行认证。
发言人介绍，为便利有意投身个人化点对点服务行业的人士，运输署将于今年8月3日推出德士及网约车综合笔试，并于同日开始接受申请，预计首批考生将于今年9月中开始应考经优化的综合笔试。综合笔试主要考核考生对提供德士和网约车载客服务的基本知识，以及主要地点和路线等实务知识的掌握，以及对《道路使用者守则》的了解。
在综合笔试中及格并完成修习德士司机或网约车司机职前课程的人士，视乎所修习的职前课程，可向运输署申请德士正式驾驶执照或网约车车辆驾驶许可证。
此外，为加强打击非法取酬载客，《修订条例》已订明因非法取酬载客而被定罪的司机，其被取消驾驶资格的期限，应不少于12个月，最长为三年。有关条文将于今年8月3日生效。

美国拟向绿卡申请者征10万美元押金
（华盛顿综合电）《华尔街日报》引述消息报道，美国国务院正考虑向一些通过美国驻外领事馆申请绿卡居留证的人，征收10万美元（约12万9000新元）的押金。
这是特朗普政府持续限制那些财力有限的外来移民所采取的措施之一，以确保在美国的移民有能力维持生计。
消息人士称，押金数额可依据个别情况高于或低于10万美元，并可能先在一小部分国家试行。
几名知情者透露，申请者预计得在正式成为美国公民之后，才可取回押金，这个过程一般须至少五年。这样一来，如果绿卡持有者到美国后无法自食其力，这笔押金便可作为担保资金。此外，移民的亲属也可以代为缴纳押金。
美国国务院发言人皮戈特说，总统特朗普已表明，有意移民美国的外来者必须在经济上自给自足。他称，国务院正在研究如何利用《移民与国籍法》赋予的权限，要求一些签证申请者支付押金，以此证明他们有维持自身生活所需的资金。
特朗普政府今年1月已经暂停针对75个国家的移民签证审批，以限制低收入外国人移民美国。这项措施影响巴基斯坦、尼日利亚和巴西等人口大国。 
美国移民律师协会的政府事务部主任德伊尼认为，高昂的押金会产生威慑作用，并指设押金的目的，似乎是为了将特定类型的移民拒之门外。
征收押金的计划势必引来移民权益倡导者的反对，因为大多数有意移居美国的外国人，他们的工资低于美国人，无力支付高昂的押金。
