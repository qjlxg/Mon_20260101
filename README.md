最高检：中国技术外流风险加剧
中国最高人民检察院指中国技术外流风险加剧，并强调检察机关从严打击商业间谍犯罪。
据中国检察日报正义网报道，最高人民检察院知识产权检察厅副厅长刘太宗星期二（2月24日）在一场访谈活动中指出，2021至2024年全国检察机关受理审查起诉侵犯商业秘密犯罪1262人，2025年前11个月受理232人，数量呈上升趋势。
他举例称，尊湃侵犯华为海思晶片技术商业秘密案涉及国产晶片技术，涉案金额3.17亿（人民币，5813万新元），社会各界高度关注。上海检察机关以侵犯商业秘密罪提起公诉，14名被告人均被判处刑罚。
刘太宗说，当前中国技术外流呈现多维度风险加剧特征，检察机关依法适用刑法第219条之一，守护国家科技安全和经济安全。他也举例称，浙江检察机关办理的张姓被告为境外非法提供商业秘密案，张姓被告是中国某著名半导体企业员工，离职后违反保密约定，接受境外组织委托非法泄露原单位商业秘密，经检察机关提起公诉，被判处有期徒刑并处罚金。
报道称，在刑事打击侵犯商业秘密犯罪同时，最高检还加强技术案件民事行政诉讼监督，对确有错误的案件提出抗诉或者再审检察建议，促进法律适用标准统一。
