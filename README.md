阿耳忒弥斯2号飞越月球背面 创下人类最远太空飞行纪录
美国宇航局“阿耳忒弥斯2号”（Artemis 2）任务的四名宇航员星期一（4月6日）飞越月球背面，并直接目睹了流星撞击月球背面阴影区时产生的“撞击闪光”。
路透社报道，这次飞掠历时六小时，飞船掠过月球表面，最近时距离月面4070英里（约6400公里）。这次飞行是自阿波罗计划半个多世纪以来，人类首次将宇航员送往月球附近海域。
“阿耳忒弥斯2号”的宇航员们星期一开始了太空飞行的第六天。醒来后，他们听到了已故NASA宇航员洛弗尔预先录制的一段信息。
“欢迎来到我的老家。今天是历史性的一天，我知道你们会很忙，但别忘了欣赏美景……祝你们好运，一路顺风。” 
数小时后，“阿耳忒弥斯2号”航天器，以25万2756英里的距离，创造了人类太空飞行的历史，成为迄今为止距离地球最远的人类。
此前的纪录约为24万8000英里，由阿波罗13号于1970年创造。
在前往月球背面途中，“阿耳忒弥斯2号”的宇航员们为一些此前没有正式名称的月球地貌赋予了新的临时名称。 
在发给休斯顿任务控制中心的无线电信息中，宇航员汉森建议将其中一个陨石坑命名为“诚信”（Integrity），与乘坐的猎户座飞船同名；另一个陨石坑则以宇航员怀斯曼已故的妻子卡罗尔的名字命名，卡罗尔于2020年因癌症去世。
汉森后来说，宇航员们观测到了许多“人类从未见过的月球地貌，即使是阿波罗计划也没有”。
当猎户座飞船高速绕月飞行时，宇航员们拍摄到了一个罕见的瞬间：由于与地球的距离创下纪录，地球显得格外渺小。随着飞船绕月飞行，地球与月球地平线一同升起落下，这与通常在地球上看到的月升日落景象截然相反，形成了引人注目的天文奇观。
由于月球自转和绕地球公转的速度相同，它的背面始终背对着地球，只有阿耳忒弥斯和阿波罗的宇航员曾直接观测过月球表面。

中国为在人工智能领域取得突破、应对与美国加剧的科技竞争，正加紧从海外吸纳半导体人才；对此，台湾当局也强化对大陆疑似挖角行为的调查。
台湾法务部调查局（MJIB）上周一表示，已对11家大陆企业展开调查，控诉这些企业以隐瞒大陆背景、设立空壳公司并未经核准在台开展业务等方式，非法挖角半导体及其他高科技人才。MJIB称，自2020年以来已处理约100起相关案件；去年也曾调查大陆最大晶圆代工厂中芯国际（SMIC）涉嫌挖角案。
加拿大顾问机构The Geopolitical Business地缘战略师阿比舒尔·普拉卡什（Abishur Prakash）称，这是一场“低调的科技战”，与美中围绕出口管制和资本等议题的公开对峙不同，大陆更着眼于推动下一波AI创新所需的人才。台湾当局对此高度警觉，调查行动反映出大陆企业对具备经验的人才在半导体与AI领域的强烈需求，即便面临美国对高端技术的限制，这类人才流动仍在加剧。

多家中国企业包括阿里云和知谱科技近期选择不将部分最新人工智能模型开源，转而通过官方付费渠道获取使用价值。 
本周阿里发布三款仅能在其云平台或聊天机器人网站访问的专有模型，其中包括增强编程能力的Qwen3.6-Plus与可处理文本、音频、图像和视频的多模态模型Qwen3.5-Omni。 
今年9月发布的上一代多模态模型Qwen3-Omni为开源版本，但阿里表示新版不会开源，理由是基于开源平台Hugging Face的下载数据，Omni系列在开发者中的受欢迎程度较低。 
业内认为，顶级模型规模不断增大、难以在本地硬件上部署，使厂商更倾向于通过集中托管和商业化服务变现，同时并未明确放弃开源策略。 
阿里云为阿里巴巴集团旗下的人工智能与云计算业务单元，阿里巴巴集团为《南华早报》的所有者。
