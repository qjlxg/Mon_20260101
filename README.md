全球最大集装箱航运公司开辟新航线 绕过霍尔木兹海峡
（利雅得综合电）全球最大的集装箱航运公司地中海航运宣布，5月10日起另辟新航线绕过霍尔木兹海峡，推出连接欧洲与中东的新服务。
地中海航运（MSC）星期六（5月2日）发布公告指出，新航线将利用横跨沙特阿拉伯的陆路卡车运输，结合波斯湾的小型支线船，将欧洲与运输受阻的中东港口连接起来，以替代被封锁的霍尔木兹海峡航线。
据规划，首趟航次将从比利时的安特卫普（Antwerp）出发，航线停靠多个欧洲港口，包括德国、意大利、立陶宛和西班牙。船只穿过苏伊士运河进入红海后，将在沙特西海岸的两个主要港口，吉达和阿卜杜拉国王港卸货。
货物随后将改由卡车跨越沙特内陆，抵达沙特东海岸达曼（Dammam），再由小型支线船运往阿布扎比、迪拜杰贝阿里（Jebel Ali），以及巴林、伊拉克和科威特等港口。这条“陆桥”走廊全长约1300公里，中途经过沙特首都利雅得。
霍尔木兹海峡是全球能源与贸易命脉。自今年2月28日美国和以色列对伊朗发动袭击以来，海峡航运一直受到严格限制，且短期内无重启迹象。
彭博社报道指，尽管陆路替代方案的成本更高、耗时更长且碳排放量大，但面对“中东局势的严峻挑战”，各大航运巨头不得不寻求避险之道。除MSC外，赫伯罗特（Hapag-Lloyd）与马士基（Maersk）此前也建立了横跨沙特与阿曼的“陆桥”解决方案。
目前，位于霍尔木兹海峡外的阿曼及阿联酋东海岸港口正迎来货运量激增，当地正紧急扩充卡车运输能力以应对转运压力。

霍尔木兹海峡封锁 德国加速生产化肥以弥补供应缺口
（柏林法新电）伊朗封锁霍尔木兹海峡引发全球经济动荡，对德国化肥生产商和农民带来冲击。
全球三分之一的化肥通常要经过霍尔木兹海峡，世界贸易组织早前警告，霍尔木兹海峡运输中断威胁着全球粮食安全，尤其是对非洲和南亚地区。
德国化肥巨头SKW近来全力投入生产，以弥补供应缺口。SKW发言人说：“这就是为什么在欧洲设立生产基地如此重要的原因。”
SKW占地220公顷的庞大厂区内，一条23公里的铁路网络运输尿素、氨和成品肥料，运往德国各地以及欧洲其他地区的各个厂区。
尽管公司预计今年的收入或许猛增高达20%，但公司首席执行官弗兰茨克称，公司并非“战争投机分子”，若将飙升的能源成本也计算在内，收益可能仅能勉强收支平衡。
SKW约80%的生产依赖天然气，而自2月28日冲突爆发以来，天然气价格已翻了一番。 
与德国许多工业企业一样，SKW此前已因乌克兰战争引发的能源危机而举步艰难，这场危机暴露了德国对俄罗斯天然气的依赖。由于德国努力摆脱对廉价俄能源供应的依赖，SKW已连续三年亏损。 
目前，SKW从挪威、荷兰和美国进口天然气，但最近中东冲突引发的多米诺骨牌效应，导致全球市场价格上涨，使它蒙受亏损。
德国化肥生产商协会指出，近年来，由于成本问题，一些欧洲工厂已经关闭。“如果没有本地生产者和具有竞争力的农业，欧洲的粮食安全将受到严重威胁。”

俄乌战争持续延烧 双方都加大无人机攻势
（基辅／莫斯科综合电）俄乌战争持续延烧，乌克兰南部城市赫尔松星期天遭俄军无人机袭击，造成平民死伤。同时，莫斯科称拦截300多架乌无人机，其中一些无人机靠近首都莫斯科和圣彼得堡等大城市。
自2022年入侵乌克兰以来，俄罗斯几乎每晚都对乌克兰发动数百架无人机攻击，而乌克兰也持续展开报复性反击。近几个月来，双方都加大了无人机攻势。 
法新社对乌空军公布的数据进行分析显示，俄罗斯4月向乌克兰发射的远程攻击无人机数量创下新高。根据数据，俄罗斯4月共发射6583架远程无人机，比3月的发射数量多2%。
乌克兰前线城市赫尔松（Kherson）的军方官员称，一架俄无人机星期天（5月3日）击中一辆私企公司车，造成一死四伤，其中包括一名63岁妇女。
俄国防部则称，俄防空部队夜间在西部边境约15个地区拦截并摧毁334架乌无人机。部分无人机在离前线较远的莫斯科附近被击落，另有59架无人机在列宁格勒州被击落。
乌总统泽连斯基3日在社交媒体发文称，乌军在新罗西斯克港口附近水域袭击了两艘俄“影子船队”油轮，造成油轮无法使用。
据俄电视台报道，克里姆林宫发言人佩斯科夫3日说，如果乌克兰继续袭击俄石油基础设施，全球油价可能会进一步上涨。
他说：“如果俄石油减少出口，油价将从目前的水平（已经超过每桶120美元）进一步上涨。这意味着即使出口量减少，俄公司也能赚到更多钱，国家也能获得更多收入。”

机构：特朗普拟上调汽车关税，或使德国产值损失近 180 亿美元
一家经济机构周六分析称，美国总统唐纳德・特朗普宣布对欧盟汽车和卡车加征关税，或导致德国产值损失近 150 亿欧元（约 175.8 亿美元）。基尔世界经济研究所（IfW）的这一估算，凸显了欧盟最大经济体对美国进口关税的脆弱性 —— 此类关税已令德国汽车业蒙受数十亿欧元损失。该研究所分析称，“影响将十分显著”，长期来看，产值损失将升至约 300 亿欧元。基尔世界经济研究所所长莫里茨・舒拉里克表示。特朗普周五宣布，下周将把欧盟汽车关税从此前商定的 15% 上调至 25%，理由是欧盟未遵守与华盛顿达成的贸易协定。基尔世界经济研究所经济学家朱利安・辛茨称：“德国本就低迷的经济增速将遭受重创。”该研究所目前预计，德国经济今年将增长 0.8%。研究所补充道，意大利、斯洛伐克、瑞典等汽车产业规模较大的欧洲经济体，也可能蒙受重大损失。德国经济部长首席顾问延斯・苏德库姆建议对特朗普的举动保持谨慎。苏德库姆向表示：“欧盟目前只需静观其变。众所周知，特朗普动辄撤销或撤回其高调的关税威胁。”他指出，特朗普必须说明为何认为欧盟未遵守现有贸易协定，且最新关税威胁是否具备法律依据也尚不明确。这位顾问称：“这一切似乎都相当冲动。”

特拉维夫——彩色鸽子图案为舞台背景，耶路撒冷青年合唱团以希伯来语和阿拉伯语演唱，为“人民和平峰会”主会场揭幕。第四月30日，第三届年会在特拉维夫举行，数百人出席。大会由“It’s Time”联盟主办，该联盟汇集80余个组织，旨在通过协议结束以巴冲突，保障两国人民的自决权。
会场气氛带着倔强的希望。与会者承认，临近十月大选、在以色列史上之一的极右翼联合政府下，左翼和平运动面临更大挑战；正如一位台上发言人所言，“和平已成一个污名”。
著名歌手亚希诺安·尼尼（艺名Noa）表示：“总有乐观的理由，也总有为和平努力的理由。”她并称，在对外多线作战的同时，以色列国内也在为能否保持自由民主而斗争，“临选只会更糟”。
大会虽规划为以巴联合活动，但与会者多为以色列和平阵营的老面孔。和平现在、并肩而立、丧亲者论坛等组织在主厅设展台。自2023年10月7日哈马斯袭击及随后加沙战争以来，约旦河西岸和加沙的活动人士因通行许可受阻未能到场，但仍有数十名青年运动成员及以色列的巴勒斯坦公民参加。

在主会前的一个论坛上，法国外交部长让-诺埃尔·巴罗通过视频宣布，法国将在六月主办一场致力于“两国方案”的国际会议，距上次类似会议已有一年。
以色列媒体对本次峰会报道寥寥，反映出和平阵营在国内话语中的边缘化；议会中间派与右翼主流代表未见到场，仅有少数左翼与阿拉伯议员出席，其中远左哈达什党议员奥费尔·卡西夫表示：“我预计最坏，但希望最好。我们将坚持信念，为两国人民争取更好未来，终将胜出，只是时间问题。”（法新社）
