路透社报道，澳洲国防部长马尔斯星期五（7月24日）接受澳洲广播公司（ABC）电台采访时说：“在我们看来，这根本毫无道理。”
另据法新社报道，澳大利亚贸易部长法雷尔指美国以强迫劳动问题对澳洲加征关税是错误的。他说：“这些关税是不合理的，违反了我们的自由贸易协定，应该被撤销。”
美国贸易代表办公室星期四（23日）发布公告说，美国将依据《1974年贸易法》第301条，以“强迫劳动”为由，对60个国家和地区征收10%或12.5%的关税。 
钢铁、铝、汽车及零部件等商品不受新关税影响，部分食品、农产品、化肥及能源产品也获豁免。
一匿名美国高级官员说，新关税将在纽约时间7月24日零时01分（新加坡时间24日中午12时01分）生效。在此之前已装船的部分货物不受新关税影响。
