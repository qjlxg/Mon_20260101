他指出，中国支持IAEA履行职责，对福岛核事故处置各方面问题保持长期审议，按照全球核安全最高标准、最严要求和各国最佳实践，监督日本应对福岛核电站退役、高放废物处置等复杂挑战。
李松还呼吁各国秉持理性、协调、并进的核安全观，切实履行维护核安全使命，以人民为中心保障核能事业安全、健康、可持续发展，共同构建人类核安全命运共同体。
2011年3月11日，东日本发生9级大地震和大海啸，摧毁了东北沿海地区，也导致福岛第一核电站发生堆芯熔毁和辐射外泄事故，大量辐射物质流入海洋。

马云与阿里和蚂蚁核心管理层罕见一起亮相谈AI
中国科技巨头阿里巴巴创始人马云，与阿里巴巴和蚂蚁集团核心管理层，罕见齐聚由他创办的杭州云谷学校，探讨人工智能（AI）带来的挑战和机会。
据“云谷教育”微信公众号星期二（3月3日）消息，阿里和蚂蚁最核心的管理团队当天齐聚云谷学校，探讨AI带来的挑战和机会。出席者除了马云外，还包括阿里巴巴集团主席蔡崇信、CEO吴泳铭、风险委员会主席邵晓锋、电商事业群CEO蒋凡，蚂蚁集团的董事长井贤栋以及CEO韩歆毅。
他们一行人与校长和老师们交流了一个多小时。根据“云谷教育”，马云说，AI时代已经快速到来，对社会的冲击超出想象，谁都没有做好足够的准备。但是对10几岁的孩子来说，他们最有改变的希望和机会。
他称，此次到云谷学校，目的就是把阿里巴巴最近对AI越来越清晰的洞察和老师们分享。
马云说，AI带来了一个让教育回归教育本身的机会，并建议死记硬背的时间，刷题的时间可以释放出来，用来培养创造力和想象力；孩子们可以有更多时间来玩，来学习音乐、绘画、运动，从中学会分享、学会感受和体验，学会倾听，以及学会理解。
马云也提到，看一所学校是不是属于AI时代的学校，并不是去看一所学校有多少AI服务器，有多强的AI技能。他解释，因为AI拥有的是晶片，而人类拥有的是心，AI时代对教育最大的改变，是老师们可以完完全全去做“灵魂工程师”，而不是做知识的灌输者。
马云还说，未来不是让孩子去和AI比拼计算和记忆，而是让孩子保持好奇，学会共情和担当，拥有体验感。因为，好奇心、想象力、创造力、判断力还有审美能力，才是AI时代教育需要赋予孩子的真正的能力。
“云谷教育”称，他们一行人在讨论中提到，AI的迭代以周计算，能力还在不断增长，此次的技术革命对生产效率和社会方方面面带来的变革是历史性的，以后可能一天不用工作八小时，但是很多工作种类都会消失。
此前，马云公益基金会官方微信公众号1月27日发文称，马云在线参加马云公益基金会“腊八之约”活动，期间分享了一些他对AI的思考。
马云说：“AI时代，未来我们的教育不是让孩子背得更多，记得更多，而是让他们想得更有趣，想得更有创意，想得更加独特。AI时代，我们的教育不是让一千个学生给出同样一个正确的答案，而是让一千个学生学会提出一万个不同的好问题。”

人工智能算力高需及国产替代驱动下，A股半导体存储行业迎来高度景气周期，上市公司佰维存储今年头两个月的归母净利达到去年全年的两倍。
综合财联社和证券时报网等报道，半导体储存器企业佰维存储星期二（3月3日）晚间发布业绩预告，预计2026年1月至2月实现营业收入40亿元（人民币，下同，7.4亿新元）至45亿元，同比增长340%至395%；归属于上市公司股东的净利润为15亿元至18亿元，同比扭亏为盈。
公司不久前发布的2025年业绩快报显示，2025年归母净利润8.67亿元，同比增长437.56%。这意味着，佰维今年前两个月的净利润，已接近或超过去年全年的两倍。

美国总统唐纳德·特朗普周二（3月3日）表示，他认为伊朗准备先动手，这是他就美方发动对伊朗攻击所作的最新解释。特朗普在白宫椭圆办公厅会见德国总理梅尔茨时说，他认为如果美国不先下手，以色列或伊朗会先发起攻击，他“可能迫使了以色列先动手”。
这一说法与国务卿马可·鲁比奥此前的表述相左。鲁比奥周一（3月2日）对记者称，美方发动打击是出于担忧：若以色列对德黑兰采取行动，伊朗会报复美国部队，若不先发制人，美军伤亡将更大。
自周末冲突爆发以来，特朗普关于行动目的和时间线的表述多次变化：他曾对媒体称行动可能在数周内结束，也表示会持续到达到目标为止。白宫称目标是摧毁伊朗的核与远程弹道导弹能力，防止其成为对中东及美国的“不可承受”的威胁。
国防部长皮特·赫格塞思称，美军行动“聚焦于摧毁伊朗进攻能力”，并非所谓的“政权更迭”战争，但他也表示伊朗领导层已受到重创。特朗普2月28日发布视频，形容伊朗当局“凶狠残暴”，并警告袭击将持续至其导弹和核能力被消除，同时劝说伊朗军警放下武器并呼吁平民“趁机推翻政府”。
