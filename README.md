农历新年首两天 未见大陆军机飞越台海中线
台湾国防部公布的台海周边海空动态显示，今年大年初一（2月17日）和初二（18日）均未侦测到中国大陆军机在台海周边活动，也未出现穿越台海中线的情况。

以色列斋月每日允万名巴勒斯坦人赴阿克萨回教堂礼拜
以色列在回教斋月之际宣布，允许1万名巴勒斯坦信徒前往东耶路撒冷的回教堂参加礼拜。
法新社报道，回教斋月星期三（2月18日）开始，以色列当天宣布，整个斋月期间，每天会发放专用许可证，允许1万名巴勒斯坦信徒进入位于耶路撒冷旧城的圣殿山，参加周五礼拜。
以当局同时对阿克萨回教堂区域实施出入限制，仅允许55岁以上男性、50岁以上女性及12岁以下儿童进入；儿童必须有直系亲属陪同。
阿克萨回教堂是回教第三大圣地，它所在的区域有多个犹太教和回教圣地，具有高度的地缘政治意义，因此一直是以巴冲突的主要争议焦点。
以色列于1967年占领这片地区，随后实施国际社会不承认的吞并。
2022年，阿克萨回教堂爆发回教徒与犹太人之间的暴力事件，导致数十名巴勒斯坦人受伤。
巴勒斯坦耶路撒冷省本周称，以色列当局阻止由约旦管理的伊斯兰宗教基金会（负责管理阿克萨回教堂的机构）在斋月前进行常规准备工作，包括安装遮阳设施和设立临时医疗站。
阿克萨回教堂高级伊玛目谢赫（Sheikh Muhammad al-Abbasi）透露，他本人被禁止进入回教堂建筑群一周，理由未知，且禁令可能延长。

中国春节假期期间 金银价格双双下跌
黄金价格在中国春节假期期间跌至每盎司4900美元（6194美元）附近，亚洲多数市场因农历新年休市。白银价格也同样回落。
据彭博社报道，现货黄金农历大年初一星期二（2月17日）一度跌2.7%，创逾一周盘中最低水平。此前一个交易日该金属已下跌1%。
金价此前经历了持续多年的强劲上涨，1月份涨势加速，但当月月底突然遭遇抛售。从每盎司5595美元以上的峰值，金价在两天内骤降至4400美元附近。此后金价虽有所回升，但波动性依然显著。
作为实物黄金两大市场，中国和印度强劲零售需求是近几个月金价的重要支撑。
印度进口数据显示，1月份黄金和白银进口量接近纪录水平。该国当月黄金进口额超过120亿美元，创下有记录以来第三高的月度进口额。白银进口额则飙升至20亿美元以上。
法国巴黎银行、德意志银行和高盛等多家银行预测，金价将重拾升势，支撑黄金持续上涨的因素预计将持续存在。
与此同时，白银价格星期二一度跌5%，随后收复部分失地。由于市场规模较小且流动性较低，这种白色金属的价格波动向来比黄金更为剧烈。不过，白银行情近期出现自1980年以来最剧烈的波动 ，其幅度和速度仍引人注目。

西班牙去年出生人数10余年来首次上升
官方数据显示，面临欧洲最严重人口危机之一的西班牙，2025年新生儿数量时隔10多年首次回升。
法新社报道，西班牙国家统计局初步数据显示，去年西班牙共诞生32万1164名婴儿，较2024年增加近3200人，增幅约为1%。
这是自2014年以来，西班牙年度出生人数首次出现增长。
不过，2025年西班牙死亡人数较上年增长2.5%，导致自然人口减少12万2167人。这是连续第九年西班牙出现自然人口减少。
面对人口老龄化与低生育率的双重压力，西班牙近年来推行开放移民政策，与西方国家形成鲜明对比。
首相桑切斯1月称，西班牙计划为50万无证移民（主要来自拉丁美洲）办理合法身份。
这项政策遭到右翼政党强烈批评。桑切斯则强调，移民有助于维持劳动力供给和养老金体系的运转。
西班牙国家统计局上周公布的数据显示，截至1月1日，西班牙总人口为4950万，其中外国人720万，占总人口14.6%。

备忘录为保护媒体工作者建立机制，包含情报共享、联合侦查及运用NBI专业技术支援工作组。代理局长安赫利托·马尼奥（Atty. Angelito Magno）表示，NBI已指派主要部门牵头，并指定专责人员负责案件监控与与成员机构的协调。“NBI加入工作组后，我们致力于维护新闻自由并为媒体工作者伸张正义。”他补充，根据第10867号法案，NBI对与媒体相关的谋杀案拥有调查管辖权。
双方合作目标是加速记者案件侦办进程，并强化政府打击针对媒体犯罪的问责力度。
