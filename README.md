阿里与百度拟将芯片设计子公司赴港上市，可能加剧国内高性能AI芯片市场竞争，挑战英伟达在中国的主导地位。
据彭博引述知情人士报道，阿里巴巴旗下芯片设计部门“T-Head”正筹备上市。阿里（南华早报母公司）对置评请求未回应。消息发布后，阿里香港股价周五收涨2.3%，报168.50港元。
百度方面，其芯片设计臻麟科技（Kunlunxin）本月已在香港秘密提交IPO申请，百度称发行规模与结构尚未最终确定。百度股价周五在港基本持平，报160.50港元。
分析人士称，两家公司计划上市反映出投资者对AI芯片企业的强烈兴趣与国内对替代计算资源的需求增长，同时也是北京在美方技术制裁下推动科技自主可控的大背景下的体现。此前在美中监管加强前，英伟达曾宣称在中国高端芯片市场占有约95%的份额。
T-Head 2018年成立，去年被中国官媒报道其专用集成电路PPU在性能上可与英伟达2024年为中国市场推出的H20 GPU相媲美。T-Head亦加大对开源指令集RISC‑V的投入。杰富瑞研究称，T-Head是阿里云全栈基础设施的自然延伸，并采用多芯片策略以满足不同业务需求。
近期已有多家中国GPU设计商成功上市，包括慕尼黑线程（Moore Threads）、MetaX、上海佰仁（Biren）等，连同获得腾讯支持的Enflame，被称为中国GPU市场的“四小龙”。其中Enflame的上海科创板上市申请已获批；寒武纪（Cambricon）早在2020年就在科创板上市。
有分析认为，T-Head上市不仅有利于其独立IP与终端推理市场的发展，也将助力阿里构建“云＋芯＋算”全栈能力，推动其在国内AI赛道的战略布局。纳蒂克西斯高级经济学家Gary Ng表示，美国技术封锁加速了中国在价值链上移动并增强自主生产能力的紧迫性。
阿里去年2月曾承诺在三年内至少投入3800亿元人民币用于云计算与AI基础设施，显示大型民营企业在国家计算力建设中的重要角色。
