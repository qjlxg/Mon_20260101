IMF警告人工智能威胁全球金融稳定
（华盛顿综合电）国际货币基金组织（IMF）警告，由先进人工智能（AI）工具驱动的网络攻击正对全球金融稳定构成风险，呼吁各国加强合作应对这个问题。
基金组织周四（5月7日）发表报告说，他们的分析显示，极端网络事件造成的破坏，可能引发融资紧张、加剧偿付能力担忧，并扰乱更广泛的市场。
报告说，全球金融体系高度互联，使风险进一步放大，更令人担心的是，先进AI模型能够大幅降低利用系统漏洞的时间和成本。
基金组织也警告，新兴和发展中国家往往面临更严重的资源限制，可能更容易成为攻击目标，尤其是那些防御较弱的地区。
报告说，这些系统性的风险涉及多个行业，一旦出事就会加以扩散。报告说：“防御措施不可避免地会被突破，因此确保韧性也必须成为优先事项，尤其要限制事故的传播范围，并确保快速恢复”。
基金组织总干事格奥尔基耶娃上月接受哥伦比亚广播公司（CBS）采访时已警告，全球金融体系尚未做好准备，应对AI带来的网络安全威胁。她希望各国更加重视制定保障措施，以保护金融稳定，并呼吁就此问题展开全球合作。

杜尔逊补充说，目前正计划于6月进一步增加运力。他指出，每列从西安开出的火车大约可以运载50个标准40英尺集装箱，而一艘远洋集装箱船可以装载数千个。

俄罗斯向基辅的外国使馆发出警告，称若乌克兰在本周末扰乱莫斯科5月9日二战纪念活动，俄方将对基辅发动打击。俄方在5月6日的通知中要求各国及国际组织“及时撤离”使馆和在基辅的人员，并宣布将在5月8日至9日单方面停火以配合纪念活动。俄方同时威胁将对基辅实施“报复性打击”，包括针对决策中心。乌方未即时回应。

英国选民于5月7日参加地方与地区选举，普遍预期将对首相基尔·斯塔默领导的工党造成重创，并再度引发其执政能力的质疑。此次投票包含英格兰近5,000个议会席位以及苏格兰和威尔士议会，若选民转向民粹和民族主义政党，可能动摇英美长期的两党格局。

若工党大量失守英格兰议会席位、在威尔士议会丧失主导地位，或在苏格兰仅列第三，斯塔默将面临更大辞职压力，甚至被要求公布离任时间表。近期市场亦因担忧其可能被更左倾的领导人取代而推高英国借贷成本。

党内基层与议员反映选举时常遭遇选民愤怒，若选情不佳，可能出现敦促斯塔默公布离任时间的书面要求。但要替换他并不容易，潜在接替人选如曼彻斯特市长安迪·伯纳姆和前副首相安吉拉·雷纳目前尚不足以发起挑战。前工党副领袖汤姆·沃森则劝告同僚勿重蹈2006年草率求仕的覆辙，警告内斗只会让选民侧目。 路透社
