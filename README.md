中国中东部87城启动重污染天气预警
1月下旬以来，中国中东部地区出现细颗粒物（PM2.5）污染过程，污染程度有所加重。
据中新社报道，截至星期一（1月26日），京津冀及周边地区、长三角、长江中游地区共87个地级及以上城市启动重污染天气预警应急，共同应对本次跨区域重污染过程。
1月21日至22日，河北、河南、山西、陕西等地部分地区出现PM2.5轻度污染。23日至24日，PM2.5污染范围扩大至长三角中北部、山西中南部、湖北中西部。25日，晋冀鲁豫交界地区中重度污染形势维持。
分析本次污染过程的主要原因，专家指出，一是中国中东部地区气象条件持续不利。二是区域污染排放处于高位。大范围雨雪降温天气，导致1月下旬初期火电供热行业活动水平上升到入冬以来最高水平。同时，区域内钢铁焦化、平板玻璃等大量生产工序不可中断行业的活动水平维持稳定。火电供热、工业炉窑、柴油车等排放的氮氧化物向硝酸盐的化学转化，是PM2.5污染加重的主要因素。
为共同应对本次跨区域重污染过程，京津冀及周边地区、长三角、长江中游地区共87个地级及以上城市启动重污染天气预警应急。截至25日，河南省安阳、鹤壁、新乡市，河北省邢台市启动了红色预警，天津等启动了橙色预警，江苏、浙江、江西、湖南部分城市启动了黄色预警。评估结果显示，启动预警后，重点工业源污染排放明显减少。
中国环境监测总站预计，本轮污染过程将持续至1月下旬后期，29日起，受东北方向冷空气及雨雪天气影响，污染形势有望逐步缓解。

奥尔西计划在人民大会堂会见习近平，双方将“深入交换”区域与国际问题看法。委内瑞拉危机料将在私下讨论，中国谴责美方行动侵犯主权并坚持支持马杜罗，乌拉圭则曾暗示若有助于民主转型则可视为“好消息”，引发加拉加斯不满，双方随后就措辞产生交锋。
近年来，蒙得维的亚与北京关系显著发展。此前乌拉圭曾在拉卡列·波政权时期推动与华自贸谈判，后改为通过南方共同市场（南方共同市）框架推进合作。双方在海关现代化、数字单一窗口等领域已签署多项协议，农业、肉类加工、基础设施、可再生能源和软件等部门将携技术方案随队访华。

【新一轮专项债收储工作陆续启动】金十数据1月27日讯，近期，2026年专项债土地收储工作在多地陆续落地。东方金诚研究发展部执行总监冯琳预计，各地将继续加大收储力度，“考虑到更加积极的财政政策基调，2026年新增专项债限额料继续上调，用于收储的专项债发行规模也有望进一步增加”。克而瑞研究中心研究员马千里表示，土地收储有效化解房企闲置土地压力，缓解资金沉淀难题，稳定市场预期；盘活低效土地资源，推动城市土地利用向存量提质转型；但还需注意的是，区域分化显著，落地规模不及预期，大量计划待后续落实。在此背景下，2026年土地储备专项债发行进一步提速。不过，土地收储还面临一些阻碍。有业内人士坦言，闲置土地所有者不少是在价高时拿地，可能因成本、利益考量，提出过高的价格要求，而地方政府需在合理价格范围内进行谈判，双方短时间内难以就价格达成一致。“此外，按国家政策要求，土地收储价格需按土地取得成本与市场评估价就低确定，但有的房企拿地时间在几十年前，按成本价评估不足市场价30%，如果参考成本价，部分闲置土地面临以‘腰斩式’‘膝斩式’收购，企业不可能同意。”
