谷歌与SpaceX之间既有合作，也有竞争。
根据星期五披露的协议，任何一方有权提前90天通知终止安排——这与Anthropic协议中的条款相同。

天气干旱中东冲突 澳洲小麦产量预计大幅下降
（悉尼综合电）由于天气干旱，加上中东冲突推高燃料和化肥价格，澳大利亚本季小麦产量预计将下降超过四分之一。
澳洲农业部6月2日发布报告指出，澳洲本季小麦收成预计为2670万吨，与去年同期相比下降26%，也比过去五年平均水平少23%。
产量下降的部分原因是种植面积减少，农业部估计本季小麦种植面积将缩减12%至1090万公顷，创2019/2020年度以来的最低水平。
农民减少种植小麦，主要因为全球小麦价格疲软，且相对于其他作物，种植小麦的利润空间不断收窄，尤其是在霍尔木兹海峡贸易受阻，导致燃料和化肥价格飙升之后。农业部指出，化肥变得更贵，预料会促使农民减少施肥，进而降低作物产量。
澳洲是全球最大的小麦出口国之一，也是东南亚、中东以及中国等经济体的重要供应国。澳洲农业部说，由于美伊战争已进入第四个月，全球燃料和化肥供应出现重大干扰。如果中东冲突持续，投入成本可能会在更长时间内维持高位，从而对产量构成压力。
气候因素也影响小麦产量。澳洲部分地区包括新南威尔士州和昆士兰州南部出现极端干旱，促使农民缩减种植面积。预计厄尔尼诺现象也将使澳洲东部各州在重要的春耕期间降雨量减少，影响小麦生长。西澳和南澳部分地区则老鼠增多，鼠患可能危及作物产量。

中国证监会主席：坚决打击操纵市场行为
中国证监会主席吴清指出，将持续强化交易监管机制，坚决打击操纵市场等行为，并呼吁业者为投资者创造更可持续的中长期收益，不要走回“冲规模、赚快钱”的老路。

他还围绕加快建设一流投资机构提出，要坚持规范为先，不断夯实合规风控内在基础；坚持客户为本，始终注重投资者利益优先；坚持功能为要，更好发挥培育新质生产力作用；坚持创新为基，持续增强行业高质量发展内生动力。

吴清认为，注重权益投资是行业服务投资者的关键所在，要下更大力气加强投研专业能力建设，强化权益投资的行业影响力和竞争力；要加强供需适配，推出更多适配中长期资金的投资产品，更好对接不同群体的差异化需求，同时加快推进向投资者回报导向的买方投顾转型，不断提升投资者满意度。
中国国务院办公厅星期五（6月5日）发布关于加强监管防范风险、促进私募投资基金高质量发展的指导意见，在加强政府投资基金和国有企业投资基金管理等方面进行具体安排。
