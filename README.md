中国据报收紧房企境外再融资 限制“1+N”贷款结构
知情人士透露，中国进一步加强境外借款监管，限制房地产企业采用“1+N”结构进行境外再融资。这种结构过去曾帮助部分房企缓解房地产危机期间的资金压力。
彭博社引述知情人士报道，负责监管中企外债的中国国家发改委近几个月已告知部分银行业者，不再允许境内房业，或以中国大陆房产作为抵押品的跨境借款人，采用“1+N”的再融资结构。
这类贷款的期限通常为364天，后续可选择进一步展期。这实际上允许借款人、尤其是房地产开发商，在获得发改委外债额度批准前签订贷款协议。
随着中国收紧期限超过一年的境外借款审批，企业面对的融资延误日益普遍，“1+N”结构可协助借款人填补等待审批期间的资金缺口。
报道认为，最新限制将进一步增加中国房企的债务再融资难度。尽管楼市出现企稳迹象，但近期仍有多家在中国大陆和香港经营、陷入财务困境的开发商，在与贷款机构谈判数月后，于债务到期前紧急寻求再融资。
知情人士说，发改委希望借此促使企业和银行更早规划债务再融资，以加强风险管理。目前，部分其他行业暂不受新限制影响。
此前，发改委已开始要求借款人提供更详细的还款计划和资金用途资料，使期限一年或以上的境外债券和贷款审批时间延长至四至六个月。
监管趋严促使部分中企将重点转向发行短期债券，也增加了企业平衡投资需求与现金流的难度。

台湾行政院星期四（7月23日）在官网发布新闻稿，公告上述消息。
台行政院会当天通过卫生福利部拟具的《食品安全卫生管理法》部分条文修正草案，将送请立法院审议。
行政院长卓荣泰说，关于中联油脂公司部分沙拉油检验出不符规定案，卫福部已于7月20日公布4至6月份生产油品与相关产品检验结果，并于21日召开食品风险评估咨议会，公布检验后处理原则。
他指出，在《食品安全卫生管理法》部分，现行制度于食品业者自主检验、实验室管理、食品安全异常通报及第三方验证等面向都有精进空间，卫福部已同时启动修正草案研议，并于21日由行政院政务委员陈时中召开修法会议，邀请六都代表出席，以建立更完善的食品安全管理制度。

修正条例的主要内容，包括在发生重大食安事件时，中央得成立指挥中心，地方主管机关有动员、通报中央和依指挥中心指示办理的应变责任；符合一定类别及规模的食品业者，应将食安监测计划报经中央核定，发现食品有危害之虞时，应于24小时内通报。
其他内容包括食品业者如违反自行或送检规定、未设置实验室等规定，直接开罚3万至300万元新台币，违反有危害之虞时的停工、通报义务等规定，最高罚锾3000万元新台币。并增订财产执行保全机制，以避免受处分人有隐匿、转移财产或逃避执行之虞。为强化网路食品管理，明定网际网路业者之管理义务及风险管控责任。

中信证券公告称，公司境外全资子公司中信证券国际的附属公司CSI MTN Limited于2026年7月23日在中票计划下发行500万美元票据，由中信证券国际提供担保。截至公告日，实际为其提供的担保余额（不含本次）为29.00亿美元，公司及控股子公司对外担保总额为2089.55亿元，占最近一期经审计净资产的比例65.31%，无逾期担保。本次担保在公司授权范围内，风险可控。

LCK次级联赛宣布试行新的教练语音规则，无时间和次数限制
LCK次级联赛官方宣布将试行新的教练语音制度，自游戏BP开始到比赛结束，选手和教练间的语音沟通将没有时间限制和次数限制。这一规则也和现有各大赛区一级联赛教练语音制度的规则不同。
LCK次级联赛官方表示这一举措是为了通过实时反馈提升选手的表现，并加速潜力新秀的成长。
