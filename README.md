Globe首席人工智能官兼首席资安官Anton Bonifacio称，设备能实时识别并定位假基地台，显著提升响应速度和执法效率，帮助查明可疑作业车辆或携带背包的人员位置，以利逮捕。
CICC执行主任Aboy Paraiso与CIDG参谋长Joselito Esico均表示，将立即部署设备以追查并逮捕策划诈骗的个人与犯罪团伙。Globe同时强调会持续与政府及执法单位共享情报、强化执法并改善防护机制。
Globe指出，公司在2025年已封锁逾9.67亿则诈骗与垃圾简讯，并停用逾24.4万张涉诈SIM卡。
