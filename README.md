澳维州新法9月生效 员工每周可居家办公两天
（墨尔本综合电）澳大利亚维多利亚州将立法规定，员工每周可居家办公两天。新法将在9月1日生效。 
维州州长亚兰星期三（3月4日）发布声明说，新法将纳入《平等机会法》，使维州成为澳洲第一个引入居家办公权利的地区。
她说：“居家办公对家庭有益，因为它能节省时间和金钱，还能让更多家长投入职场。”
这项新政策适用于所有工作场所，不过雇员少过15人的商家可延迟至2027年7月1日才落实。
新政策引起当地商界反弹，认为这将影响招商。维州首府墨尔本在冠病疫情期间经历了全球最漫长的封锁期之一，经济回弹缓慢。墨尔本委员会首席执行官温克尔说：“如果经商变得太困难，商家们会转移到别处。”
不过亚兰反驳称，没有证据显示居家办公法律会驱赶投资，而且想要吸引最优秀人才的雇主，都已推行居家办公和灵活工作安排。
她也指出，数据显示，员工居家办公可推高生产力20%，此外新政策实施后，估计每户家庭每年可节省多达5000澳元（约4500新元）。
维州将于今年11月举行州选举。亚兰领导工党寻求第四个任期，她的竞选纲领预料会以落实居家办公为核心。
在野的自由党-国家党联盟去年在联邦大选中惨败，主要原因在于它主张取消联邦公务员的居家办公，要求所有员工每周五天都要到办公室上班。这个政策非常不受欢迎，竞选期间就被丢弃了。

2月28日开战时双方曾将更迭政权作为目标，但在3月2日白宫讲话中，特朗普并未将推翻伊朗政权列为首要任务，而是称美方目标是摧毁伊朗的导弹与海军能力并阻止其获得核武器。五角大楼首席皮特·赫格塞斯也表示，这不是所谓的“政权更迭战争”。相对而言，内塔尼亚胡在3月2日晚仍呼吁伊朗民众上街推翻统治者，并对福克斯新闻表示要为伊朗人民掌控命运创造条件。
一位熟悉白宫意图的美国官员告诉路透，双方军事目标并不相同，“政权更迭是他们的目标之一”。在开战前，内塔尼亚胡说服特朗普这是阻止德黑兰取得核武与毁灭弹道导弹能力的“现在或永不”时刻；特朗普则称行动可能需“四五周”或“无论多久”。但以方私下承认，何时结束战争最终由特朗普决定，前美驻以大使丹·夏皮罗认为特朗普可能寻求“提前下坡路”。
战争拖延还给特朗普带来国内压力：路透/益普索民调显示仅四分之一美国人支持对伊开火，初选已于3月3日在可能左右国会的德州和北卡州开跑；航运与能源受扰、油价上涨也将成为选民切身负担。皮尤研究中心10月民调显示，对以色列政府持不利看法的美国人由一年前的51%升至59%。白宫与内塔尼亚胡办公室未就此置评。
据多名消息人士透露，自2025年特朗普重返白宫后，内塔尼亚胡曾至少与其会晤七次并多次通话，将注意力从加沙冲突引向伊朗导弹与核问题；两国数月来秘密筹划军事行动，攻击时机数周前已确定。内塔尼亚胡欲借此在10月大选前巩固政治遗产，但他仍面临右翼联盟裂缝、贪腐审判及2023年10月哈马斯突袭与随后加沙战争造成的创伤——即便在伊朗取得军事成果，也难以完全化解国内愤怒。

迪拜/特拉维夫讯——美军与以色列3月4日持续对伊朗展开昼夜打击，以军称已对伊朗导弹发射场和防空体系发动新一轮打击。美军中央司令部司令、整体作战指挥布拉德·库珀上将称行动“超前于作战计划”，称有5万名美军、200架战机和2艘航母参与“从海底到太空及网络空间的24/7打击”，并称伊朗防空体系严重受损、海军关键水域已无战舰可用，已打击超过2000个目标，更多兵力在途中。 
尽管美国原希望迅速推翻德黑兰神职领导层，但伊朗持续对以色列及海湾地区报复，并誓言选出接替2月28日首轮轰炸中被击毙的最高领袖阿里·哈梅内伊的新人。战事进入第五天，市场担忧冲突长期化：亚洲股市暴跌，首尔创纪录重挫，黄金与美元因避险需求走强，航旅业应对逾2万次航班取消并撤离滞留旅客。 
以色列称当日打击目标涵盖导弹发射阵地、防空系统与指挥中心，国防部长以色列·卡茨称“以摧毁并威胁自由世界为目标的伊朗领导层是明确的消灭对象”。伊朗谴责美以攻击为无端侵略。 
能源方面，伊朗攻击船只与能源设施并封锁霍尔木兹海峡，区域产出停摆，油价飙升。伊朗革命卫队称完全控制海峡并警告船只勿通行，美总统特朗普表示美海军可必要时护航油轮。法国总统马克龙将派航母入地中海协助通航，但称美以行动“违反国际法”。 
华盛顿方面，部分共和党议员与民主党人要求国会批准海外用兵；美方已确认首批4名遇难美军士兵，并警告将有更多伤亡。伊朗方面3月3日称死亡人数达787人，其中首日一所学校被炸致165名女童遇难。特朗普否认是以色列迫使其出手，称因核谈判停滞“感觉”德黑兰会发动攻击而下令开打。 来源：路透社

罗马报道——多名匿名消息人士周三称，意大利政府可能向因应伊朗空袭而请求装备支援的海湾国家提供军事援助，或包括可拦截弹道导弹的SAMP/T防空系统。消息称最终决定尚未敲定，受援国与拟调动的具体电池尚未公布。
第三位消息人士表示，意大利也可能迅速提供反无人机系统，相关技术评估仍在进行中。意大利目前已向乌克兰供应SAMP/T以支持其防空，消息人士排除从基辅调拨装备来支援海湾。
国防部长古伊多·克罗塞托表示，罗马会已收到海湾方面关于防空与反无人机系统的请求；内阁次长阿尔弗雷多·曼托瓦诺称政府正在评估这些请求。随着冲突向地中海蔓延，英、法、希三国已表示将向塞浦路斯派遣防空力量，此前该岛的英国阿克罗蒂里空军基地遭无人机袭击。
意大利外交部长安东尼奥·塔亚尼说，迄今尚未收到使用意大利境内美军基地对伊朗开展行动的请求，如有将予以评估。伊朗的报复性打击已影响海湾港口、城市和石油设施；意大利能源部长称，尽管对海湾能源依赖有限，但若冲突引发能源危机，意大利可重启部分燃煤电厂。

俄罗斯无人机击中乌南部空载客运列车，副总理通报
3月4日，乌克兰副总理奥列克西·库列巴表示，周三凌晨一架俄罗斯无人机在乌南部尼古拉耶夫（Mykolaiv）地区击中一列空载客运列车，造成一名铁路工人受伤。
库列巴称，周二晚间对一列运行于东中部第聂伯罗（Dnipro）与西北科韦尔（Kovel）之间的列车也发动了无人机袭击，但被铁路人员及时应对，无人机在机车数米外坠落，未造成人员重大伤亡。
乌克兰国家铁路公司（Ukrzaliznytsia）称，俄罗斯对铁路基础设施的无人机攻击有所加剧，机车、货车及用于修复基础设施的专用设备是主要目标。本月以来已记录18次袭击，损坏41处设施，铁路车场和桥梁也遭到打击。
此前周一，第聂伯罗彼得罗夫斯克地区一列通勤列车的无人机袭击已造成1人死亡、7人受伤。　路透社
