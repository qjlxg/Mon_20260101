美国商务部长卢特尼克将加拿大总理卡尼近日加强与中国贸易关系的举动斥为“政治噪音”，并警告这种做法可能会影响今年晚些时候就新版北美贸易协定展开的谈判。

今年1月，卡尼访问中国，并在与中国国家主席习近平会面后说，预计中国将在3月1日前对加拿大油菜籽征收的关税，从85%降至约15%，中国也将让加拿大公民免签入境。加拿大则将允许4.9万辆中国电动车进入加拿大市场，关税税率约为6.1%，远低于目前的100%。

他质疑：“如果渥太华选择进口中国电动汽车，并采取其他强化与北京贸易关系的措施，你认为美国总统会在USMCA谈判中说，你们应该继续保持全球第二好的协议吗？”
卢特尼克还透露，USMCA的重新谈判可能将在今年夏末或夏中前后进行。
加拿大财政部长弗朗索瓦-菲利普·香槟（Francois-Philippe Champagne）指出，七国集团每个成员国都在与中国的关系上制定自己的战略路径，加拿大也不例外。他说：“我们将继续与美国伙伴密切合作。同时，我认为加拿大人现在已经明白，多元化是关键。我们需要更具韧性。”

第一财经报道，中国航空运输协会星期五（1月23日）公告，正组织各航空公司研究制定《公共航空运输企业航班预留座位规则》团体标准，并将于近日向社会公开征求意见。
据微信公号“民航观察”，航司航班预留座位现象已被多个央媒点名。
据江苏省消保委调查，中国国内航司经济舱锁座比例普遍达30%至60%，远超国际航协建议的5%至10%合理范围，部分航司甚至锁定超六成优质座位。这些座位多为前排、靠窗及过道等偏好座位，仅开放机尾、中间区域供免费选择，航司靠过度锁座获利。
