担忧数据安全 波兰禁止中国制造汽车进入军事设施
出于对车载传感器可能被用于收集敏感数据的担忧，波兰军方已禁止中国制造车辆进入军事设施。
据路透社报道，波兰军方星期二（2月27日）发布声明公布上述决定，并称在关闭特定功能、落实各设施安全规定所要求等其他防护措施后，相关车辆仍可能获准进入管制区域。
为降低机密信息外泄风险，波兰军方同时禁止将单位手机连接至中国制造车辆的车载娱乐系统。

IMF警示日本保持央行独立 控制财政扩张避免削减消费税
国际货币基金组织（IMF）星期二（2月17日）发布报告，警示日本政府应保持日本央行的独立性，控制财政扩张，避免以削减消费税方式应对民生问题。
新华社报道，这一报告发布恰逢日本首相指名选举之际。路透社称，市场密切关注高市早苗是否会反对央行进一步加息，以及高市早苗此前提出的为期两年的“食品消费税归零”承诺。
货币政策方面，IMF指日本央行保持独立性与公信力，有助于稳定通胀预期，并表示日本央行“应继续退出货币宽松，使政策利率在2027年达到中性利率水平”。 
财政政策方面，IMF认为，短期内财政政策不宜进一步宽松。这一点与高市早苗提出的“负责任且积极的财政政策”相悖。
IMF认为，日本目前虽有一定的财政空间，但仍须保持财政克制，以稳固财政缓冲、维持应对冲击的能力。IMF预测，长期看日本政府财政赤字将扩大，支出压力将增加，公共债务总额将进一步增长。
IMF指出，债务水平持续高企，叠加财政收支状况恶化，使日本经济面临一系列冲击。
针对高市早苗提出的为期两年的“食品消费税归零”竞选承诺，IMF直言“应避免削减消费税”，称“这一缺乏针对性的措施会压缩财政空间，增加财政风险”。
