英媒：外国汽车经由中国流入俄罗斯以规避制裁
路透社报道称，在“灰色市场”操作下，数以万计的汽车正从中国出口至俄罗斯，绕开西方和亚洲政府的制裁，也打破了此前汽车制造商退出俄罗斯市场的承诺。
2022年俄乌战争爆发，欧美制裁俄罗斯，多家汽车企业退出俄罗斯市场。但路透社报道指出，包括丰田、马自达以及德国豪华品牌在内的车辆交易仍在持续，部分原因是俄罗斯经销商通过中国中介订购车辆。
报道称，这些销往俄罗斯的车辆大多在中国生产，或在其他地区生产后经由中国转运。而且，越来越多的车辆为所谓的“零里程二手车”，即新车先在中国被经销商或贸易商登记为已售出，随后被重新归类为二手车并出口。
零公里二手车是中国高度补贴、竞争激烈的汽车市场的产物，使车企和经销商得以抬高销量数据、获取补贴并出口过剩车辆。
一名前四川汽车贸易商的出口人员表示，将欧洲、日本和韩国品牌汽车从中国运往俄罗斯的贸易商，会把新车归类为二手车，以免去对俄销售批准程序，“这样更容易出口。”
零里程二手车在中国往往大幅折价，但在俄罗斯的售价与从未登记过的新车相近。路透社查阅的运输文件和一名俄罗斯经销商证实了这一点。
奔驰、宝马、大众等来自对俄实施制裁地区的车企表示，他们禁止对俄销售，并正尽最大努力防止未经授权的出口，包括通过对经销商培训以及在合同中设置条款。但他们也强调，调查潜在违规行为难度很大。奔驰在声明中称，此类调查“耗时且复杂”，需要第三方协助。
宝马表示，已要求其中国零售业务“坚决反对任何潜在的对俄车辆出口”，并称若车辆仍以灰色渠道进入俄罗斯，“这发生在我们影响力之外，也明确违背我们的意愿”。
一名俄罗斯经销商告诉路透社，他在符拉迪沃斯托克的经销店不批量进口外国汽车，而是为满足客户订单，逐辆从中国贸易商处采购。
俄罗斯的汽车市场研究与数据分析机构Autostat数据显示，中国进口车辆在俄罗斯登记的所有西方或日本品牌汽车中占比不断上升，韩国品牌的销量也保持在较高水平。
数据显示，自俄乌战争以来，外国品牌在俄累计销量已超过70万辆。去年俄罗斯购买的丰田数量，仅次于中国品牌，位居所有外国品牌之首。但丰田在声明中表示其已于2022年停止向俄罗斯发货，“丰田不向俄罗斯出口新车”，并未回应Autostat的数据。马自达也作出类似表态，并补充称，在俄罗斯销售的任何新马自达“均通过不受马自达控制的第三方转售”。

马尼拉 — 菲律宾天主教主教会议（CBCP）呼吁信徒在四旬期实行“数字媒体禁食”。CBCP指出，禁食应回应当代影响人类生活的新现实，其中包括数字媒介。传统上禁食多指节制饮食以示悔改与灵修，但今天也应包括“从一切使心远离上主的事物中抽离”，如持续使用手机、社交媒体、串流、游戏等，这些会分散注意力并削弱内心生命。 
CBCP表示，数字媒体禁食旨在“重新发现沉默、祈祷、默想与真实的人际关系”，不是对身体的惩罚，而是解放心灵。教会同时肯定负责任使用科技的益处，但警示过度使用常导致分心、疲乏、注意力减弱、人际关系疏远与灵性生活受损，许多人因屏幕持续索取注意力而难以在上帝与他人面前同在。 
实践建议包括：睡前与醒后避免使用手机、限制社交与串流时间、在用餐和聚会时实行无设备、偶尔进行24小时或周末的数字禁食、删除易干扰的应用，并以祈祷、读经、礼仪、服侍或面对面交谈替代屏幕时间。 
CBCP强调这不是剥夺而是转化，能为祈祷、读经、圣体敬礼、慈善和有意义的相遇腾出时间。预期成效有：与天主更亲近、思路更清晰、情绪平和、休息与体力改善、家庭和社区关系增强，以及对受造物与现实相遇的重新珍惜。 
教会呼吁信徒以喜乐的心实行禁食，借由远离数字过度重获内在宁静、深化关系，并在日常生活中重新发现天主的临在。
