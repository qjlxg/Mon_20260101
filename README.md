广东公司悬赏60万人民币寻找走失模特犬
中国广东惠州一家公司为了寻找走失的模特犬，公布悬赏60万元（人民币，下同，11万新元），引起中国网民质问60万元是否能兑现。
据中国《现代快报》报道，一则“60万元重金寻狗”的启事在社交平台引发广泛关注。发帖寻狗的是广东惠州一家公司，走失的是一只明星模特犬。
公司相关负责人付先生称，这只犬的价值高，若能成功找回，60万元悬赏金一定会兑现。不过，有网友表示疑惑，质问是真的吗？如果找到，60万元确保兑现吗？
付姓负责人说，走失的犬只名为辛巴，是一只4岁半的成年公犬，属于伯恩山犬品种，体型较大，体重约37公斤，以黑、白、棕三色为主，性格温顺亲人、乖巧听话，是公司团队的重要一员，更是陪伴大家多年的“明星模特犬”。“我们从小看着它长大，养了四年多，感情非常深厚。” 
这名负责人也说，2月1日下午3时左右，工作人员带着辛巴前往上海虹桥机场附近的南石桥路参加一场宠物相关活动，其间辛巴不慎走失，工作人员立即在当地寻找，但始终无果，随后报警求助。如今辛巴走失已近20天，派出所民警告知付姓负责人，近期犬只走失警情较多，目前暂未反馈到与辛巴相关的有效线索。
《现代快报》引述通过“企查查”系统查询到的信息报道，付姓负责人所在公司是惠州一家聚焦宠物智能科技的初创企业，成立于2025年7月，注册资本500万元。
江苏尚学律师事务所陈鹏律师认为，公开悬赏寻狗是有法律效力的，依据中国《民法典》，悬赏人以公开方式声明对完成特定行为的人支付报酬的，完成该行为的人可以请求其支付；根据《民法典》，权利人悬赏寻找遗失物的，领取遗失物时应当按照承诺履行义务。因此，公开悬赏等同于单方允诺/要约，完成行为（找到并送还）等同于承诺，合同即成立。
陈鹏说，找到狗的人有权要求全额兑现，如悬赏人拒付等同于违约，找到狗的人可以起诉，法院会判决全额支付悬赏金。

美联储官员对利率展望分歧扩大 有几人不排除升息
（华盛顿综合电）美国联邦储备局的会议纪要显示，官员对利率展望的分歧扩大，有几人认为若通货膨胀持续居高不下，当局或须调高利率。这可能导致它与强力主张降息的特朗普总统冲突加剧，让联储主席提名人选沃什上任后的任务更加复杂。
根据美联储星期三（2月18日）发布的1月底政策会议纪要，负责表决利率政策的联邦公开市场委员会（FOMC）多数成员都认为，就业市场在去年呈现疲弱走势后，下行风险已在今年1月趋缓，但通胀居高不下的风险则继续存在，当中有几人因此对调高利率的选项保持开放。
会议纪要说：“几位成员警惕，在通胀数据仍居高的前提下进一步放宽利率政策，可能会被错误解读为暗示决策者对维持2%通胀目标的承诺已动摇。”
美联储内部追踪的通胀指标，目前比2%的目标高出约1个百分点。虽然大部分官员都认为通胀放缓的进度比预期慢，但还是有几人表态，若通胀走低符合预期，他们对进一步降息保持开放。
战略咨询机构安永—博智隆首席经济师达科在发给客户的分析报告中说：“会议纪要显示了官员更加趋向鹰派的立场，这会在沃什被确认为美联储主席后形成一个有趣的互动局面。”
针对最新发布的美联储会议纪要，白宫发言人德赛周三称，由于特朗普刺激经济供给侧的政策奏效，最新的价格数据证明通胀如今已“冷却并平稳下来”。他说：“美联储是时候承认这个再明显不过的事实，削减利率为美国购房者和商家带来更多经济上的舒缓。” 
特朗普已多次公开表态，要下任美联储主席降息。他是在FOMC的1月份会议后两天，于1月30日宣布提名沃什。FOMC当时以10票对2票，通过维持联邦基金利率在3.5%至3.75%的区间，只有理事沃勒和米兰主张降息0.25个百分点。
美联储主席鲍威尔当时在记者会的正式发言，其实已透露了FOMC内部对未来利率走向的分歧扩大，预示沃什在5月中接过棒子后，要说服官员同意降息，或面对不小的挑战。
鲍威尔任内还将主持3月和4月的两次政策会议，根据芝加哥商品交易所（CME）的观察工具FedWatch，市场仍大概率预测两次会议都会继续保持利率不变。
至于沃什预期上任后首次主持的6月份会议，市场预测当局有49%的概率会降息0.25个百分点，高于预测维持不变的40%。

去年中国规上有色金属工业企业利润同比增长25.6%
2025年中国规模以上有色金属工业企业实现利润总额达到5284.5亿元（人民币，约970亿新元），较上年增长25.6%，创历史新高。
据新华社报道，记者从中国有色金属工业协会获悉，从金属品种来看，铝、黄金、铜表现亮眼，实现利润增加额分别为380亿元、212亿元、177亿元，对规上有色金属企业实现利润增长的贡献率分别为35%、20%、16%。
中国有色金属工业协会副会长陈学森说，支撑2025年中国有色金属行业效益较快增长的核心因素主要是有色金属价格高位运行、政策红利持续释放、新兴产业需求拓展等。
陈学森说，在新兴产业需求方面，下游应用领域的升级迭代和拓展，为有色金属行业创造了新的增长极，航空航天、新能源、新一代信息技术、人工智能等战略性新兴产业的快速发展，对有色金属产品的市场需求持续扩大。

路透社梳理资料后发现，北京正借特朗普政策所带来的不确定性，尝试推动中国庞大的制造业深度融入欧洲联盟、海湾国家和跨太平洋伙伴全面进展协定等。
尽管外界对中国产能过剩、市场准入不均和国内需求疲软等表达高度关切，中国仍加速达成总计20项贸易协定，大都经过多年筹划部署。
路透社梳理了具官方背景的贸易学者自2017年以来撰写的100篇中文论文，发现中国政策顾问正在系统性地对美国贸易政策进行逆向分析，以化解华盛顿对华所采取的遏制战略。
中国目前已将所制定的计划蓝图付诸于行动。受访的10名中国官员和贸易外交官称，加拿大总理卡尼1月访华与北京达成协议，调降中国出口电动汽车关税，是中国多项旨在突破美国经贸影响力的首个贸易成果。

路透社审阅的论文源自中国社会科学院和北京大学认可的2000多篇贸易战略论文。这些文章为最高领导层提供建议，显示政策圈内人士普遍认为，痛苦的结构性变革，是中国长期主导全球商业格局值得付出的代价。
两名西方外交官称，如果取得成功，北京可能颠覆美国10多年的贸易政策，将自身置于一个以中国为中心的新型多边秩序核心。
对于中国所采取的这项策略，一名美国官员对路透社说，拥有巨额贸易顺差的国家力图维持全球化，并不足为奇。
这名官员说：“特朗普正在解决全球化给美国带来的问题，其他国家则在自由市场准入远离美国之际，试图加倍押注全球化。”
