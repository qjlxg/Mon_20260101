台汉光演习模拟断网 总统府：只是降速不是完全断网
台湾军方将在8月举行的“汉光”演习实兵操演期间，首度模拟战时手机断网情景。针对在野党对该行动法律依据的质疑，台湾总统府澄清，此次演习并非完全“断网”，而是采取网络降速。
综合台湾《联合报》《自由时报》报道，台军汉光演习将于8月5日至14日展开，其中城镇韧性演习将首度纳入网络受阻演练：中部及北部县市将分别在8月10日和13日下午的演习期间，面临移动电话上网受阻的状况。
国民党立委葛如钧质疑，若有人因此错过求救、医疗警示或重要交易，谁要负责？民众党立法院党团总召陈清龙也质疑，实施30分钟“行动断网”的法源依据为何？谁来决定与监督？国民党立委赖士葆质疑，只在北中断网，是民进党制造战争恐惧，另类的认知作战。
台湾总统府发言人郭雅慧星期二（7月21日）解释，此次演练并非全面切断网络，而是通过网络降速，模拟天然灾害或大规模网络攻击等情境。演练旨在测试移动通信受影响时，能否顺利切换至固网、Wi-Fi及其他备援通信方式，让政府部门及民众熟悉应变流程，强化跨机关的协调能力。
她指出，通信演习并非台湾首创。日本、韩国以及欧盟等，都有针对断电、网络攻击等通信受阻情况进行模拟演练，这是提升社会韧性的普遍做法。
至于为何此次断网演习仅涵盖中北部，郭雅慧说明，中北部人口占全台73%，手机使用密度最高；且南部及离岛地区此前已因海底电缆断裂及台风等灾害，经历过真实的通信受阻情境。她强调，演练优先从中北部起步，未来将循序渐进推广至全台其他县市。
台湾防长顾立雄星期一（20日）说明，本次演练属初步验证，配合汉光演习针对部分地区与特定功能降载移动网络，固网不受影响。此举也将考验台军在模拟情境下，指挥管制通信的备援能力。

中朝关系升温 两国贸易额升至2017年以来新高
随着中国与朝鲜关系持续回暖，两国贸易额升至2017年联合国对朝实施大规模制裁以来的最高水平。
据彭博社报道，中国海关星期二（7月21日）公布的数据显示，今年上半年，中朝贸易额同比增长19.6%，达到15亿美元（19亿新元）。其中，朝鲜对华出口大增54.1%，不过中国对朝鲜仍维持贸易顺差。
这是中朝关系持续升温的最新迹象。今年6月，中国国家主席习近平时隔七年首次访问平壤；随后，由中共政治局常委、全国政协主席王沪宁率领的中国高级代表团上周访问朝鲜，进一步凸显双方加强合作的意愿。

台修法松绑外资低轨卫星业者限制 为星链入台铺路
台湾立法院三读通过有“星链条款”之称的《电信管理法》修正案，允许低轨卫星业者不受现行外资直接持股不得超过49%、以及董事长须具台湾籍等限制，此举可望为美国太空探索技术公司（SpaceX）旗下“星链”（Starlink）进驻台湾市场扫除法律障碍。
综合台湾《联合报》和法新社报道，台湾立法院星期二（7月21日）三读通过修正案，规定申请以卫星通信技术设置使用电信资源的公众电信网络者，经主管机关审酌台湾安全、公众电信网络安全、电信资源使用及电信产业发展等核准后，得不受董事长须具台湾籍及外资持股比率限制。
这一修法被视为让星链进驻台湾扫除法律障碍。按照台湾现行的《电信管理法》，设置使用电信资源的公众电信网络者，其董事长应具台湾籍，外国人直接持股不得超过49%，直接及间接持股合计不得超过60%。
SpaceX于2024年曾在社交平台X上表示，星链不在台湾运营，仅仅是因为没有获得许可；该公司无法遵守台湾一项要求任何卫星互联网提供商必须由本地资本拥有多数股权的规定。
台湾目前没有自主的低轨卫星，其与全球的网络连接严重依赖海底通信电缆。过去几年，台湾的中华电信已与包括欧洲公司Eutelsat在内的多家外国卫星公司达成协议，构建多轨卫星通信网络，以便在战争或自然灾害发生时提供备用通信服务，但主导着全球卫星通信市场的“星链”却不在合作伙伴之列。
“星链”是世界上最强大的近地卫星互联网系统，已拥有超过6000颗在轨运行的近地卫星，网络覆盖全球数十个国家。
国民党立委黄健豪表示，引进低轨卫星通信系统可确保在紧急状况时，台湾仍有可用且安全的通信网络。如果爆发两岸冲突，大陆切断台湾所有的海底电缆，台湾将几乎无法与外界进行任何通信。
不过，即便法规有所调整，尚不清楚“星链”是否有意向台湾提供卫星服务。
台湾民主、社会与新兴科技研究中心的政策分析师Cathy Fang向法新社表示，电信运营商在进入台湾市场时还面临其他障碍：“运营商仍需解决频谱使用权、地面基础设施、终端部署、网络安全与数据治理要求、市场规模以及商业可行性等问题。”
她说：“此次修法将消除一项重大的法律障碍，但最终决定仍取决于SpaceX的商业考量。”
