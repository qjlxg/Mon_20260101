研究：睡眠差或提高年轻人患癌风险
（休斯敦讯）美国研究发现，睡眠质量差可能是导致全球50岁以下人群患癌率上升的原因之一。
过去30年，年轻人患上癌症的病例增加了近80%。全球早发性癌症病例从1990年的182万起上升到2019年的326万起，而40多岁、30多岁及更年轻人群的癌症死亡率上升了27%。
《卫报》报道，在芝加哥举行的美国临床肿瘤学会年会上提交的两项研究显示，年轻人不规律的睡眠模式可能是造成这种情况的一个因素。
由得克萨斯州休斯敦的MD安德森癌症中心牵头开展的两项研究，分析了美国1800多万名18至50岁成年人的健康数据。
研究员发现，睡眠习惯不佳的人更容易患上早发性肠癌、乳腺癌、子宫癌或卵巢癌。在某些情况下，被诊断为失眠的50岁以下人群，在五年内患癌的概率是一般人的三倍。
研究员说：“这些发现显示，睡眠紊乱可能是早发性癌症风险分层中一个具临床意义且潜在可改变的风险因素，值得进一步研究。”
查明年轻人癌症发病率上升的原因已成为全球卫生领域的优先事项。《英国医学杂志·肿瘤学》发表的一项研究显示，每年有超过100万50岁以下者死于癌症。
英国布里斯托一家睡眠专科诊所主管加利指出，有关研究只是发现睡眠与癌症有关联，并非证明睡眠紊乱会导致年轻人患癌的因果关系。
他说，这种关联可能有多种解释，如睡眠不足可能带来生理变化，或是较难保持健康生活方式，例如饮酒增多、肥胖率更高、运动量减少、吸烟增多等，而这些因素也会增加患癌风险。
英国癌症研究中心健康信息经理温特则说：“这些研究探讨了50岁以下人群失眠与某些癌症之间是否存在关联，但在得出结论之前须进行更多研究，特别是长期跟踪研究。”

新加坡 — 意大利國防大臣圭多·克羅塞托（Guido Crosetto）表示，西歐政府能理解美國要求增加國防開支的理由，但選民並不買帳。5月30日於香格里拉對話會（SLD）期間，他在接受《海峽時報》和聯合早報採訪時指出，全球能源危機使在經濟和民生壓力下，把資源用於軍費變得更難向公眾說明。克羅塞托援引霍爾木茲海峽被封鎖所衍生的危機，稱在經濟困難時期更難為軍費辯解，並表示這一觀點在與美國“戰爭部長”皮特·赫格塞斯（Pete Hegseth）同席的部長圓桌午餐中已有提出。
他還強調，民眾對軍費的接受度與所處地緣位置有巨大差異，越遠離俄羅斯或中國邊界的社會，越難說服民眾增加軍費。赫格塞斯當日則批評西歐國家推動軍費落後，並稱亞洲盟友在面對區域威脅時是“可靠的制衡力量”，警告“那些繼續靠美國納稅人慷慨搭便車的人，該醒了：那個時代結束了。”
在北約2%國防支出基準上，義大利直到2025年才首次達標；北約已把目標提高到5%。本國媒體報導克羅塞托在國內為提升軍費與面臨2027年大選的總理喬治亞·梅洛尼（Giorgia Meloni）存在分歧。對此他回應，梅洛尼理解國防投資的重要性，但歐盟關於將預算赤字限制在國內生產總值3%以內的財政規定，實際上限制了投資空間；義大利2025年赤字達3.1%。他同時重申，義大利的國防支出走向將遵守其對歐盟和北約的承諾。

新加坡——美国在5月30日举行的第23届香格里拉对话会上表示，将优先与那些响应美国、在安全开支上分担更多责任的“模范盟友”合作。美国国防部长皮特·黑格塞斯在致辞中称，符合条件的国家将被“提前安排”，享受加速军售、深度产业合作和扩大情报共享等优待。 
他列举了澳大利亚、印度、印尼、日本、马来西亚、菲律宾、韩国、新加坡、泰国和越南为正提升防务支出的国家，但未提及台湾与新西兰；在印太仅有澳、日、菲、韩、泰属于与美签订条约的盟国。对日本，黑格塞斯表示期待仍高，尽管东京正在改革战后防务架构，防务开支仍不足2%。 
在对华态度上，他较2025年语气缓和，称美中需达成“有利且持久的战略稳定”，并将以“强而沉稳、明确”的行动维护地区稳定，此番演讲也是习近平与特朗普北京会晤后美方首次重要防务表述。 
关于台湾，他回避了议会关于对台军售的提问；台前国防部长杨念祖指出，特朗普对台支持的态度最终还要由总统作答。特朗普曾对可能达140亿美元的对台军售表态不确定，令地区不安；参议员彼得·里克茨则称，去年获批的110亿美元军售显示美国立场坚定。台湾关系法仍要求美方提供台自卫所需资源。 
会外，黑格塞斯与英澳 counterparts宣布三国在AUKUS框架下共研无人水下航行器，推进第二支柱的前沿防务技术合作，同时第一支柱仍聚焦向澳提供常规武装核动力潜艇。 
黑格塞斯总结称，美方欲避免“不必要对抗”，在实力无可置疑的情况下并与负担公平分担的伙伴同盟行动；当利益一致则共同作为，分歧时则务实调整。

联合国卫生署署长谭德塞（Tedros Adhanom Ghebreyesus）于5月30日抵达刚果民主共和国东部最受影响省份伊图里省省会布尼亚，现场有法新社记者目击。谭德塞在当地表示，国际社会正在协助刚果当局应对疫情，但“社区自主管理同样重要”，他此行旨在与社区沟通、评估防控进展并提供支援。
世卫组织和非洲疾控中心数据显示，自5月15日宣布暴发以来，刚果已有至少1,077例疑似埃博拉病例、246人死亡（截至5月28日）。疫情已波及该国东部三省及邻国乌干达，乌干达确认9例感染、1人死亡。世卫警告真实规模可能更大，因疫情在被发现前就已传播，而当地实验室检测能力有限。
冲突加剧防控难度。伊图里、北基伍和南基伍长期受武装组织活动影响，数百万民众流离失所、营地卫生条件差，近100万流离人员在伊图里，营地疫情扩散引发担忧。营地居民多卡斯·马彭齐（Dorcas Mapenzi）称：“如果埃博拉来，我们会被挤得像沙丁鱼一样被消灭。”
乌干达本周关闭与刚果边境并对入境者实施21天隔离。世卫27日通报一名确诊患者已两次核酸阴性，于5月27日出院，这是本轮暴发首例出院确诊者。埃博拉通过密切接触和体液传播，过去50年在非洲致逾1.5万人死亡。导致本次暴发的邦迪布焦（Bundibugyo）毒株尚无特效治疗或现成疫苗，非洲疾控中心负责人称有望在年底前准备好疫苗。

德国南部慕尼黑机场5月30日上午短暂停飞约一小时，原因是两名飞行员报告在机场上空疑似发现无人机，机场当局和警方证实了德媒的初步报道。
停飞期间，20余班原计划降落慕尼黑的航班被改降至其他机场。警方称，约上午9时，分别处于两架飞机上的两名飞行员报告在停机坪上空看到疑似无人机，随后采取了停飞措施。
约一小时后航班恢复。警方表示，尽管动用了包括警用直升机在内的大规模搜查，但未发现可疑目标；在排除对航空安全构成威胁后，于10时05分解除停飞，并已对事件展开调查。
此前，2025年10月慕尼黑也曾因无人机目击在短期内多次被迫关闭，影响更为严重。

在香格里拉对话会上，AUKUS防务伙伴关系获得急需推进。澳大利亚、英国和美国公布了加强海底情报侦察与保护海底关键基础设施（如光缆和管道）的行动方案。AUKUS自2021年成立以应对印太地区中国影响力增长，分两大支柱：第一支柱由英美协助澳大利亚获取核动力潜艇，第二支柱则致力于联合发展先进防务能力。近期外界对第二支柱进展迟缓表示担忧。
英国参谋长理查德·奈顿空军元帅透露，第二支柱的首个签署项目将部署超大型无人水下航行器（XLUUV），预计自2027年起服役，具备长时间水下持续监视能力，将为AUKUS伙伴提供重要保障。他并称这只是第一步，还在规划后续项目。
澳大利亚副总理兼国防部长理查德·马尔斯在论坛上强调海底基础设施是脆弱环节，近年地区及全球多条海底电缆被切断，澳大利亚约99%的国际流量经由仅15条海底电缆承载，金融、医疗、通信等均高度依赖不可移动的海底资产，容易遭破坏。
5月30日在新加坡美驻馆会谈后，马尔斯与英方国防大臣约翰·希利及美方代表一致表示，第一支柱仍在按计划推进，同时重申要加快第二支柱先进能力的交付。英国下议院国防委员会4月发布报告也警告，若不加速，第二支柱在先进技术方面已丧失公信力。美方代表在论坛强调联盟须为互惠关系，各方应承担应有责任，不能搭便车。

以色列军队于5月31日在黎巴嫩南部的中世纪要塞博福特城堡升起国旗，同时警告当地居民撤离，以便展开地面行动。城堡周边可闻炮声并冒出浓烟，该堡在以色列此前两度占领期间被用作军事基地，地点俯瞰南黎巴嫩。
