联合国决定维持大堡礁世界遗产地位 澳洲政府表示欢迎
（悉尼综合电）联合国教科文组织（UNESCO）决定维持大堡礁的世界遗产地位，暂不将它列入濒危世界遗产名录，澳大利亚政府星期六（7月4日）对此表示欢迎。 
自2021年以来，教科文组织每年都会评估大堡礁的状况，并多次指大堡礁面临被列入濒危名录的风险。
教科文组织星期五（3日）指出，澳洲正持续努力应对气候变化、改善水质、推动可持续渔业管理及管控土地开垦，但在2024年至2025年间，大堡礁的硬珊瑚覆盖率大幅萎缩。这主要是因为海水温度持续高于平均水平，引发自2016年以来第六次大规模珊瑚白化事件。 
此外，极端天气、陆地径流、沿海开发以及棘冠海星大量繁殖，也持续对大堡礁的生态系统施加压力。
教科文组织说：“虽然大堡礁仍展现出一定的生态韧性，但它承受这类事件并从中恢复的能力正不断减弱，这令人深感担忧。”
澳洲助理旅游部长格林回应时说，教科文组织的决定肯定了澳洲在管理这些风险方面所做的大量努力，但她也坦承，气候变化在未来将继续威胁大堡礁。
澳洲海洋保护协会活动经理辛德勒则认为，澳洲在应对大堡礁面临的主要威胁方面仍存在严重不足。
她说：“大堡礁每年为澳洲经济贡献约90亿澳元（约80亿新元），是全国第五大就业支柱，支撑着7万7000个工作岗位。我们不应该等到教科文组织来提醒，才懂得必须采取更多措施来保护它。”

据报道，特朗普政府内部人士期望教宗能出席7月4日的美国庆典，但美国并没被列入良十四世2026年的出访日程。
在岛上，教宗与一个移民家庭交谈，随后牵着孩子的手，与他们怀孕的母亲一起站在“欧洲之门”（Porta d’Europa）纪念碑前。这座纪念碑旨在纪念那些为了追求更美好生活而冒着生命危险横渡大洋的人。
一如其前任方济各，良十四世将捍卫移民权益视为其任期的核心支柱。他在访问中赞扬那些帮助弱势群体的人，并谴责美国大规模驱逐移民的行为。
兰佩杜萨岛常住人口约6000人，位于突尼斯海岸145公里处，是一个以渔业和旅游业为主的社区。舆论普遍预计，教宗将利用这次的半天访问，再次呼吁国际社会建立安全且合法的移民途径。
联合国难民署发言人温加罗接受访问时指出，当前全球关于移民的政治辩论往往只围绕边界和威慑，而非保护和共同责任。他强调，教宗的到访传递了一个明确的信息。

中国前五个月失业保险支出创新高
中国今年前五个月的失业保险支出为880.8亿元（人民币，下同，167亿新元），创下2013年有数据以来同期新高。
中国人力资源和社会保障部星期五（7月3日）在官网发布今年1月至5月人力资源和社会保障主要统计数据。其中，失业保险基金支出为880.8亿元，扣除1月至4月已支出的708.6亿元，5月单月支出为172.2亿元。
长期统计和分析公开数据的微信公众号“数据GO”星期六（7月4日）发文指出，与历年5月相比，失业保险基金支出的上一轮高点出现在2022年5月，当月支出为174.6亿元。由于中国在2022年底放开冠病疫情封控，今年5月已成为疫情结束后失业保险基金支出最高的一个5月，同比增加13.96%。
从历年前五个月数据来看，今年1月至5月失业保险基金支出为880.8亿元，超过2020年同期的879.2亿元，创下2013年有数据以来同期新高。
不过，失业保险基金收入也同步增加。今年1月至5月，失业保险基金收入为887亿元，支出为880.8亿元，累计净收入6.2亿元，结束此前连续八个月净收入为负的局面。
文章还指出，截至今年5月底，中国失业保险参保人数为2亿4885万人，同比增加1.62%，但较年初累计减少33万人。这可能反映退休人员数量多于新增就业并缴纳失业保险的人数。文章称，能够缴纳失业保险的单位一般属于正规单位。
