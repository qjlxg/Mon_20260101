刘国中北京调研 要求抓好水库安全度汛工作
中国副总理刘国中星期天（7月12日）在首都北京调研时要求，全力以赴抓好水库安全度汛工作。
据中新社报道，刘国中星期天先后赴北京平谷区滑子水库和彰作水库，实地察看土石坝、溢洪道、放水设施和下游河道，了解防汛责任落实、雨水情预报和安全管理等情况。他称，要牢固树立底线思维、极限思维，全面压实政治责任，落实落细暴雨洪水防御应对措施，全力以赴抓好水库安全度汛工作。
刘国中说，中国水库数量多、分布广，当前正值主汛期，安全度汛任务重。要完善雨水情监测预报体系，落实水库运行管理规定和各项预案，健全指挥决策机制，提升洪水防御和险情处置能力。
刘国中称，要全面落实水库大坝安全责任制，强化防洪安全管理，对受暴雨影响区域水库提前预泄腾库，严禁违规超汛限水位运行。要加强病险水库、中小水库监管，病险水库原则上一律空库运行。要加强水库大坝、险工险段等重点部位巡查，及时发现和消除风险隐患，保障溢洪道行洪畅通，提前备足抢险物资装备和人员。
刘国中也强调，要强化监测和联合会商，提升短临预报预警能力，及时发布暴雨、洪水、台风等预警，严格落实临灾预警“叫应”和跟踪反馈机制，确保预警信息第一时间直达基层。
刘国中还说，要抓好山洪灾害和中小河流洪水防御，强化宣传教育，做好人员转移避险工作。要统筹流域上下游、左右岸、干支流，科学调度水利工程拦洪削峰错峰，充分发挥防灾减灾作用。要强化城市洪涝联排联调，严防内涝风险。

中国6月冠病疫情升温 新增近八万确诊病例
中国疾控中心通报，中国6月冠病疫情升温，新增近八万起确诊病例。

在哨点医院中，今年第23周（6月1日至7日）至第26周（6月22日至28日），门急诊流感样病例冠病病毒阳性率连续上升，分别为2.9%、4.1%、5.3%、6.4%。
本土病例病毒变异监测情况显示，全国31个省（自治区、直辖市）及新疆生产建设兵团6月共报送3741例本土病例冠病病毒基因组有效序列，均为奥密克戎变异株。
优势流行株为NB.1.8.1及其亚分支。根据采样日期，第23周至第26周NB.1.8.1及其亚分支占比分别为98.5%、98.1%、98.8%、95.9%。现有证据未提示出现可能构成额外公共卫生风险的新变异株。
中国疾控中心上星期四（9日）发布的今年第27周（6月29日至7月5日）急性呼吸道传染病哨点监测数据显示，哨点医院门急诊流感样病例呼吸道样本检测阳性率前三位病原体为流感病毒（11.6%）、冠病病毒（9.6%）、肠道病毒（9.1%）。
为防范呼吸道传染病疫情，中疾控建议公众在就医过程中需全程佩戴口罩；在人群密集场所或乘坐飞机、火车、地铁等时佩戴口罩，尤其是老年人和一些有慢性基础性疾病的患者，以减少感染风险。
中疾控也建议免疫力较弱人群，如孕妇、低龄儿童、老人及慢性病患者等积极主动接种疫苗。
