中东战事引发外资汇出 台湾外汇储备创15年最大降幅
中东战事引发外资汇出，台湾外汇储备于3月份大减少86.01亿美元（109.7亿新元），创2011年欧债危机以来最大单月减少金额。台湾央行进场卖汇，阻止台币贬值。
据台湾央行新闻稿，截至3月底外汇储备降至5968.86亿美元，为2025年5月以来最低水位。新闻稿指出，其他影响外汇储备因素包括投资运用收益，以及主要货币对美元的汇率变动。
彭博社引述台湾央行外汇局局长蔡烱民在记者会表示，维持外汇市场秩序进场调节，是外汇储备减少最主要原因，出口商亦卖出不少美元。他并强调，“有效汇率（Nominal Effective Exchange Rate，简称NEER）仅下跌1%”，台币尚属稳定。
美国与以色列在2月底对伊朗发动空袭，战事迟未结束，使得外资大举撤出台股。
长期以来偏向稳定增长的台湾外汇储备，罕见出现大额减少的状况，反映出台湾央行频频进场抚平外汇市场波动。据彭博数据，台币年初至今的表现在主要亚币居中段班。
台湾证交所数据显示，外资3月份累计卖超台股9682亿元新台币（388.7亿新元），创彭博有纪录最大。在庞大的资金汇出压力下，新台币跟进重贬2.28%，靠着出口商卖汇与央行干预暂守住32元整数关卡。
台湾央行总裁杨金龙近日接连表示，密切关注外汇市场，会维持汇率稳定防止输入性通胀。至于外资方面，尽管汇出资金庞大且集中，但尚属可控，央行也有进场调节以稳定新台币。

根据国民党提供的讲稿，郑丽文在中山陵结束谒陵行程后发表公开谈话，先从1925年孙中山逝世谈起，指当时台湾已沦为日本殖民地长达30年，台湾人无法如大陆民众般公开悼念孙中山，只能在日本当局钳制下举行追悼会，连悼歌、悼词都遭禁止与检查。
她又细数辛亥革命后，台湾民间悬挂“庆祝汉族光复”布条遭拆除、戏班改编黄花岗故事遭查禁等情节，凸显日本对台湾社会与民族情感的压制。

中国为在人工智能领域取得突破、应对与美国加剧的科技竞争，正加紧从海外吸纳半导体人才；对此，台湾当局也强化对大陆疑似挖角行为的调查。
台湾法务部调查局（MJIB）上周一表示，已对11家大陆企业展开调查，控诉这些企业以隐瞒大陆背景、设立空壳公司并未经核准在台开展业务等方式，非法挖角半导体及其他高科技人才。MJIB称，自2020年以来已处理约100起相关案件；去年也曾调查大陆最大晶圆代工厂中芯国际（SMIC）涉嫌挖角案。
加拿大顾问机构The Geopolitical Business地缘战略师阿比舒尔·普拉卡什（Abishur Prakash）称，这是一场“低调的科技战”，与美中围绕出口管制和资本等议题的公开对峙不同，大陆更着眼于推动下一波AI创新所需的人才。台湾当局对此高度警觉，调查行动反映出大陆企业对具备经验的人才在半导体与AI领域的强烈需求，即便面临美国对高端技术的限制，这类人才流动仍在加剧。

多家中国企业包括阿里云和知谱科技近期选择不将部分最新人工智能模型开源，转而通过官方付费渠道获取使用价值。 
本周阿里发布三款仅能在其云平台或聊天机器人网站访问的专有模型，其中包括增强编程能力的Qwen3.6-Plus与可处理文本、音频、图像和视频的多模态模型Qwen3.5-Omni。 
今年9月发布的上一代多模态模型Qwen3-Omni为开源版本，但阿里表示新版不会开源，理由是基于开源平台Hugging Face的下载数据，Omni系列在开发者中的受欢迎程度较低。 
业内认为，顶级模型规模不断增大、难以在本地硬件上部署，使厂商更倾向于通过集中托管和商业化服务变现，同时并未明确放弃开源策略。 
阿里云为阿里巴巴集团旗下的人工智能与云计算业务单元，阿里巴巴集团为《南华早报》的所有者。
