狂风暴雨袭粤 多地提醒民众到室内躲避
受强对流云团影响，中国广东广州、深圳、汕尾、揭阳、清远等地星期天（4月5日）下午出现暴雨、雷电大风、冰雹等强对流天气。多地也提醒民众到室内躲避。
据中新社报道，位于粤北的河源市连平县当天下午狂风大作，当地气象部门说，此次冰雹天气可能给当地造成雹灾，并于傍晚6时28分发布连平县冰雹橙色预警信号。梅州、汕尾等地下午也发布了冰雹预警信号。
广州同日下午乌云密布，该市天河区、越秀区出现中到强雷雨，伴有6级短时大风和强雷电。广州气象部门傍晚5时多发布提醒称，由于雷雨云团正影响广州大部，局地伴有短时强降水、强雷电及6至8级的短时大风，在外的民众请快到安全的室内躲避。
广州市民陈思蓓告诉记者：“这雨太大了，我打雨伞根本没用。”她当日原计划外出与朋友聚会，但由于雨势太大，她一出门就被雨“淋”回家，聚会也取消了。
当日的深圳同样是暴雨倾盆。深圳气象部门也发布紧急提醒称，该市光明、宝安、南山、福田等区的民众需注意，还在户外活动的，尽快转移到室内。
广东省气象部门说，预计星期一（6日）全省韶关大部、河源和梅州的北部市县仍将有大雨局部暴雨。民众外出踏青、祭扫需警惕雷击、冰雹、树木和建筑物倒塌造成的伤害，注意防范局地强降水引发的城乡积涝、山洪和滑坡、崩塌等地质灾害。

中国学者陈群任联合国教科文组织教育助理总干事
中国学者陈群获任联合国教科文组织（UNESCO）教育助理总干事。这是第二次由中国人担任该职位。

陈群曾于2012年至2017年担任设立在华东师范大学的联合国教科文组织教师教育教席主持人，并自2021年起就任教科文组织教师教育中心理事会主席。

《沉默的荣耀》主角吴石故居挤满参观者
聚焦1949年潜伏台湾的中共地下党员吴石的电视剧《沉默的荣耀》播出后，吴石故居受到关注。适逢星期天（4月5日）清明节，吴石故居迎来许多参观者。
据极目新闻报道，位于福建福州仓山区螺洲古镇吴厝村的吴石故居，星期天迎来了络绎不绝的参观者。
报道引述到访的福州民众周姓男子称，他携妻子和女儿一同祭拜吴石。这名周姓男子说，早在《沉默的荣耀》播出之前，他便知晓吴石的事迹，并称十几年前曾到访故居。“很多往事一直埋在心底，直到这部剧播出，才特意带着妻女一起来看看。”
另据星岛头条、《福建日报》和福州新闻网报道，吴石故居始建于清末，历经百年风雨，虽然建筑风貌保存尚好，却面临木质结构老化、展示陈列设施老旧等问题。福州市相关部门去年11月开始对吴石故居展开系统性修缮。经过两个多月修缮，故居已从2月7日起对外试运行。
吴石是中共在台潜伏最高情报员，《沉默的荣耀》剧情围绕1949年吴石奉命赴台任国防部参谋次长展开，讲述他在台期间多次向中共华东局传递关键军事情报的故事。

“ถอดรหัสกติกาประชามติ 69”座谈会：修订规则以便利公民投票
4月3日13:00，泰国法政大学（Thammasat University）法学院在塔帕差尼中心211室举办题为“ถอดรหัสกติกาประชามติ เพื่อแก้ไขการลงประชามติในครั้งต่อไป”的座谈会。与会者包括国会议员 ภัณฑิล น่วมเจิม（พรรคประชาชน）、Dr. อุษณีย์ ปฐพีศรีกิจ、iLaw 活动官 เกวลิน ถนอมทอง、以及法学院副教授 รศ. ปริญญา เทวานฤมิตรกุล 等。
会议由เกวลิน列举了影响公投投票的八大问题：一是仅允许在“外县”（นอกเขตจังหวัด）投票，不能在同一省内跨区投票；二是公投媒体宣传少于选举；三是示范选票印刷遗漏导致废票；四是选举委员会（กกต.）禁止政党引导，但无相关法律依据；五是现场官员有引导选民投“不赞成”的情况；六是误导选民在“赞成”栏打勾；七是部分地区存在买票诱导投“不赞成”；八是假消息鼓动投“不赞成”。
Dr. อุษณีย์提到曾尝试网上登记未果并截图，但法院以其未尽充分努力驳回延长登记请求，引发社会呼吁选举委员会行使权限延长登记期。iLaw执行董事นายยิ่งชีพ指出，选举委员会提供的不利信息可能误导选民，未来公投争议将更多。รศ. ปริญญา强调，提前投票登记与异地公投登记时间不对等（前者17天，后者仅3天），选举委员会可依规放宽或延长登记，他主张开通邮寄或线上投票，并呼吁公民社会与政党共同修法，优先考虑公民便利，只修必要条文。

卡斯将中美差异概括为“中国是以消费者为中心的技术文化（techno-centric consumer），美国则更偏向以企业为中心的技术文化（techno-centric enterprise）”。他指出，中国消费者普遍比美国更热衷新技术，这与科技在中国推动脱贫和壮大中产阶级的作用有关。
卡斯在2021至2023年间任职OpenAI并参与了2022年11月ChatGPT的推出，目前经营自己的咨询公司。他还称，美国消费者对大科技更为怀疑，担忧社交媒体对儿童的影响及公众信任下降，这也解释了为何一些AI应用在中美两地的热度差异。
以OpenClaw为例，上月约有近千人在腾讯深圳总部排队申请免费安装，本地社交平台上也出现收取数百元安装费的服务，反映出中国市场对AI工具的强烈需求。

Meta宣布聘请中国资深高管许锐（Xu Rui）出任旗下超智能单元的AI硬件团队负责人，负责开发独立于Reality Labs的AI设备团队。该消息由Business Insider披露，南华早报亦对其背景进行了梳理。
许锐毕业于哈尔滨工业大学与首尔国立大学，历任多家中国大型科技公司的关键岗位。2015至2017年，他在小米担任全球拓展总监，推动公司智能电视海外布局并与谷歌达成将首款Android设备引入美国的合作。
2020至2022年，许锐在字节跳动旗下硬件部门New Stone Lab任总经理，期间推出包括Dali智能台灯在内的多款消费产品；2022至2024年曾在腾讯的扩展现实（XR）设备团队任职。2024年他加盟帕洛阿尔托的人形机器人初创公司K-Scale Labs，该公司于11月宣布破产，随后他加入AI代理初创公司Dreamer，后者于今年3月被Meta收购。
业内认为，Meta此番任命反映其在AI硬件领域的持续投入与竞争布局。

港股上市、吉利入资的曹操出行押注重资产、车队优先战略，力争在中国机器人出租车赛道中脱颖而出，目标到2030年部署10万辆自动驾驶车辆。 
公司CEO龚欣在接受南华早报采访时表示，机器人出租车的未来取决于以车辆制造、自动驾驶技术和车队运营构成的闭环“金三角”资产管理模式。曹操正在完善其L4级自动驾驶系统，并于2025年底在杭州先行投放了100辆机器人出租车；尽管中国多数车辆仍需安全员看守，曹操计划今年实现无人化运营。 
4月1日，公司获批在杭州开展无人路测，成为该市首家获批企业。面对Pony.ai、WeRide等本土竞争者，曹操强调其“完全为机器人出租车打造”的车型是核心竞争力：该车过去两年从头设计、软硬件高度集成，预计今年亮相并于2027年上半年进入量产。
