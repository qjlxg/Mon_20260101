德国绿色小村电力自足自给 免受中东能源危机干扰
（柏林法新电）当全世界都被中东战争推高能源价格所困扰，德国首都柏林附近一个小村庄费尔德海姆（Feldheim）却因早早便转向可再生能源而免受波及。 
费尔德海姆只有130名村民，被风力发电机环绕，太阳能板随处可见，拥有自己的电力和暖气网络，完全由在地产出的廉价电源支持。
担任村长逾20年的纳普不无自豪地说：“世界其他地方正在发生的事，对我们来说其实无所谓。”
费尔德海姆早在1990年代、东西德统一后不久就开始转向清洁能源，并在2010年取得“能源自足自给”，自此成为德国向全世界展示如何摆脱传统化石能源的示范项目。
费尔德海姆的成功得力于地理环境和显著投资。它地处广阔且风势强劲的平原，让数十个风力发电机可源源不绝地放送电力。
村子的电力只需一个发电机就足够，剩下的都通过营运商支付费用，让村民卖电赚钱。还有多余的能源用电池储存起来，在缺乏日照或风力的时节使用。村子的暖气设施则是通过一个使用农业废料的沼气厂来发力。
跟几年前爆发的俄乌战争一样，中东战争再次暴露了德国对进口能源的依赖。不过纳普坚信，费尔德海姆的模式有机会成为德国“成功的出口产品”。

因中东冲突影响，燃油价格波动，政府在3月25日深夜将油价一次上调每升6铢，随即召开特别内阁会议通过7项紧急措施，缓解油价冲击。副总理兼交通部长毕帕表示，将对运输业实施定向补贴，主要在4月1日至30日间按实际行驶量补助，依据GPS及追踪系统核算。
主要措施如下：
- 非定期货车（黄牌70）287,175辆，陆运局GPS资料显示日均行驶约3,100万公里、平均油耗4公里/升、日用油约780万升，政府补贴燃油成本每升6铢，旨在降低生活必需品运输成本，防止物价跟涨。
- 公共客运与外送骑手分项补助：客运2、3类11,395辆补助每升4铢；客运4类（如面包车）19,414辆补助300铢/车/日；Rider约114,653辆补助300铢/车/月，用以稳定票价并减轻民众出行负担，尤其在泼水节期间。
补助将按实际行驶追踪支付，若业者无GPS，则以车上QR码记录替代。长途及跨省客运维持现有票价，油价基准按每升33铢计算，超出部分由政府补偿，自4月1日起实施。另将调整油品储备并推广B20柴油，预计比一般柴油每升便宜约5铢。交通部设立战情室密切监控，对哄抬价格者将严厉处罚。询问请拨陆运局热线1584。
