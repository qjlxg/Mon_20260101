中国工信部长：应提升传统产业竞争力 不应简单淘汰
中国工信部长李乐成撰文指出，传统产业是现代化产业体系的基础，中国应着力提升相关产业竞争力，而非将这简单视为“低端产业”加以淘汰。
中共中央机关刊物《求是》星期六（5月16日）出版的最新一期，发表李乐成撰写的文章。
李乐成在文中说，中国当前正处于由制造大国向制造强国转变的关键时期，制造业比重必须保持在合理区间，不能任它持续下滑，否则将埋下产业安全隐患，削弱经济抗风险能力和国际竞争力。
他提到，国际经验表明，在迈向现代化的进程中，一个国家如果不能与时俱进地实现产业结构调整升级，就可能导致经济长期徘徊不前、落入“中等收入陷阱”。
李乐成指出，中国制造业目前整体还处在全球价值链中低端，高技术产业占比仍然较低，反映到市场上表现为中低端供给过剩、高端供给不足。此外，一些发达经济体近年来纷纷推进“再工业化”、“制造业回流”，一些新兴经济体为承接产业转移而出现同质化竞争，对中国制造业发展形成双重挤压。
他呼吁加强创新，特别是在核心技术和基础技术领域，并推动人工智能与工业更深层次融合。
李乐成认为，产业结构优化升级是提高中国制造业综合竞争力的关键举措；要把握全球产业发展和变革大趋势，瞄准世界产业发展制高点，以提高技术含量、延长价值链、增加附加值、增强竞争力为重点，推动实现产业体系整体跃升。
他建议大力发展先进制造业，因为这是大国科技和产业博弈的主战场，并呼吁要统筹推进传统产业、新兴产业、未来产业发展，加快构建以先进制造业为骨干的现代化产业体系。
李乐成也说，传统产业是现代化产业体系的基底，决不能当成“低端产业”简单退出，要推广先进适用技术，提升传统产业在全球产业分工中的地位和竞争力。
他还呼吁推进信息化和工业化深度融合，撑腰深入实施“人工智能+制造”专项行动，促进智能产业化、产业智能化；加快新一代信息技术全方位全链条普及应用，大力发展工业互联网，加快制造业数智化转型。

艾利森称峰会开局正确，双方正认真寻求重塑竞争关系。习近平在晚宴上首次以“让美国再度强大”（MAGA）措辞表示，中国发展可与美方共同前行。艾利森指出，此次北京会更像“经贸型”峰会，商业在外交中占据更高位，反映在特朗普随团的企业领袖阵容，包括马斯克、贝莱德首席拉里·芬克、英伟达黄仁勋与苹果库克等。峰会上达成的商务细节尚未公开。
对习近平而言，此次峰会有利于维持中国继续崛起的宽松环境；对特朗普而言，推动强劲经济对其11月3日中期选举及树立“和平特使”历史定位至关重要。报道：海峡时报美国首席记者Bhagyashree Garekar。

台北 — 一架借来的公务机近日秘密载着台湾总统赖清德飞往非洲，机上只有随身行李、没有大型行李箱，以节省燃油来完成绕开受限空域、长达约2.4万公里的行程。 
这次高度保密的出访由随行官员披露，官员本人不仅参与航班并对外说明过程，还出示了相关文件作为佐证。 
官员表示，此行旨在规避中国大陆对台湾日益强化的外交与空域限制，突破孤立努力，确保总统能安全抵达非洲友邦并完成外访任务。

基辅消息——乌克兰总统泽连斯基5月15日表示，俄方正试图将白俄罗斯进一步卷入对乌战争，并正在权衡从白俄罗斯境内对乌克兰北部或直接对北约国家发动攻击的方案。
泽连斯基在与军方和情报官员会面后于Telegram称，乌方持续记录俄方拉拢白俄总统卢卡申科参与“新一轮进攻行动”的尝试，并掌握俄白间为此进行的额外接触。他指出，俄方正在考虑从白俄南部或北部方向展开行动——可能针对乌克兰的切尔尼戈夫—基辅方向，或直接从白俄罗斯领土对某一北约国家发动攻势，但未披露更多细节。
白俄罗斯南接乌克兰，北及西部邻接北约成员国波兰、立陶宛与拉脱维亚。对此，莫斯科和明斯克尚未立即回应。俄罗斯对乌军事计划列为国家机密。
卢卡申科作为普京的亲密盟友，曾允许俄军在2022年2月入侵期间利用白俄领土，但未派白方部队参战。明斯克随后同意在境内部署俄方战术核武和“奥列什尼克”高超声速导弹。
泽连斯基补充称，4月已获情报显示俄方正为再度牵扯白俄做准备，并警告若卢卡申科支持俄方，乌克兰必将捍卫国家与人民。他已指示国防部制定应对方案，加强切尔尼戈夫与基辅北部防御。  路透社
