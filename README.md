澳洲年轻人心理健康问题加剧 2022年逾三分一患焦虑等精神障碍
澳大利亚一项研究显示，在2022年，超过三分之一的澳洲年轻人符合焦虑症、情绪障碍或物质使用障碍的诊断标准。
新华社报道，根据澳洲悉尼大学星期五（7月24日）发布的声明，研究发现在16岁到24岁澳洲年轻人当中，患有焦虑症、情绪障碍或物质使用障碍的占比，从2007年的23.4%增至2022年的36.2%。
这项发表于《柳叶刀区域健康—西太平洋》的研究也发现，严重精神障碍病例比例从14.7%升至24%，年轻人的自杀念头在这段期间增加超过一倍。焦虑症比例更从12.2%增至29.1%，情绪障碍则从4.9%升至13.3%。
研究人员分析2007年至2022年的全国调查数据后发现，2022年约每五名年轻澳洲人中就有一人同时患有多种精神障碍，男女皆呈上升趋势。
研究共同首席作者、悉尼大学马蒂尔达心理健康与物质使用研究中心教授斯莱德说，严重精神障碍比例上升，可能严重影响日常生活，并“对身心健康造成持久影响”。
研究没有调查具体成因，但指出经济不稳定、工作和住房缺乏保障，以及数码媒体影响等因素可能起到推波助澜的作用。研究人员说，63%的精神障碍在25岁前首次出现。

（华盛顿彭博电）美国汽车安全监管机构将开始制定新规定，以确保司机和乘客能在发生意外后安全离开车子。
这是美国针对一系列事故所采取的重要措施，在这些事故中，车内人员因无法逃出车辆而被困，有时甚至导致致命事故。
根据一份监管文件，美国国家公路交通安全管理局（NHTSA）星期四（7月23日）透露，将启动制定新的联邦法规程序，并批准一名申请者提出的要求，即要求该机构强制所有车子配备可靠且明显易操作的车门逃生系统。如果新规定获得通过，美国所有新车都必须符合相关要求。
特斯拉和其他汽车制造商采用的隐藏式、电子控制车门把手设计，近年来涉及多起人员被困车内的事故。在好些案例中，特斯拉车辆的乘员虽然在高速撞击中幸存，却因无法逃离车辆，在随后发生的火灾中死亡或受重伤。

美国汽车安全中心执行主任布鲁克斯说，启动规则制定程序是加强电动车门及逃生安全的重要第一步。
