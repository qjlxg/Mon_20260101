人民币有望超越日元  成为外汇期权市场第二大交易货币
根据伦敦清算所（LCH）的数据，人民币有望在外汇期权市场中超越日元，成为美元兑换的第二大交易货币。
据彭博社报道，LCH外汇清算部门负责人安德鲁·巴奇勒（Andrew Batchelor）预计，等到国际清算银行（BIS）2028年发布下一份三年期外汇和场外衍生品市场调查报告时，美元兑换的离岸人民币交易很可能升至第二位，仅次于兑换欧元。
根据国际清算银行去年发布的调查，使用人民币的外汇期权全球日均交易量为820亿美元（约1046亿新元）。同期，日元为1020亿美元，欧元为2360亿美元。
巴奇勒在上海接受采访时说：“人民币国际化是真实的趋势。这不一定是为了取代美元，而是为了实现货币使用的多元化。市场希望能够以不同的货币进行交易和结算。”
巴奇勒说，人民币的使用受到多种因素推动，包括实体经济贸易和跨境流动的增加。其他驱动因素还包括国际市场日益增长的风险管理需求等。
中东冲突也提振了美元和人民币的使用。根据全球金融信息服务机构Swift的数据，3月份美元在国际交易中的占比升至创纪录的51.1%。与此同时，中国跨境人民币支付系统（CIPS）也在4月初创下了有史以来最大的单日交易量。
彭博社的报道称，近年来，随着中国经济的增长以及政府推出一系列支持人民币全球化的措施，人民币在全球金融市场中的使用量不断增加。中国人民银行前行长周小川本月表示，由于美国政策削弱了美元的信誉，当前是推动人民币国际化的黄金窗口。

何瑞修说：“我们跟欧洲市场很多项目开发商交流，当地具备额外竞争的需求。本土化将是推动公司在欧洲市场销售涡轮机的关键因素。”

美国商务部长办公室与贸易代表办公室对CBC的报导未立即置评。但美国贸易代表贾米森·格里尔在国会作证时称，特朗普政府不会“照单全签”续签USMCA，协议部分条款需作重大修改，并批评加拿大在全球化问题上的立场与美方难以契合。 
卡尼就任一年多来对特朗普的威胁态度坚定，强调必须大幅降低对美经济与安全的依赖。4月19日他在全国讲话中称，过去依赖与美国紧密关系的优势已成为“我们必须纠正的弱点”。 
特朗普对加拿大关键行业加征高额关税，但总体上仍遵守USMCA，使逾85％的美加贸易保持免税。汽车和钢铁关税对加拿大最大省安大略打击严重，省长道格·福特以禁止所有美国酒类和葡萄酒销售作为反制。美国商务部长在国会称该禁令“令人愤慨”，福特则表示在美方撤销所有关税前不会让步，并直言“我不信任特朗普总统”。（美联社）

巴黎消息——分析人士称，尽管存在派系分歧且部分关键决策者长期不露面造成混乱，伊朗在这场中东战争中总体仍维持统一立场，目前尚无明显领导层重大分裂迹象。

哈梅内伊之子莫杰塔巴尚未公开亮相，新任革命卫队司令艾哈迈德·瓦希迪和新安全主管穆罕默德·巴格尔·佐尔加德多以书面慰问代替公开活动。仍留任的高层如议长加利巴夫、外长阿拉格奇与总统马苏德·佩泽什基安大多支持以对美外交结束冲突且不让步的统一策略。

萨贝特还表示，莫杰塔巴虽对决策有广泛监督，但难以像其父那样日常掌控，导致瓦希迪领导下的意识形态化革命卫队更直接地行使权力。新华社/法新社
