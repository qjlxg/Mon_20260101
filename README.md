科学家：气候变化加剧欧洲干旱 非降雨量偏低
国际科学家星期四发布的一项分析指出，人为造成的气候变化加剧了欧洲各地的干旱。导致极端干燥状况的主要因素是高温，而不是降雨量偏低。
欧洲大陆继6月遭遇历史性热浪后，正陷入严重干旱。从法国到罗马尼亚，各国纷纷面对河流干涸、用水限制和破坏性野火等挑战。
世界天气归因组织星期四（7月23日）发布报告指出，全球气候变暖加快欧洲土壤、湖泊和河流的水分流失，增加出现极端干旱的可能性。
报告联合作者、伦敦帝国理工学院的扎卡里亚（Mariam Zachariah）说，虽然欧洲大部分地区今年春季降雨量低于平均水平，但气候变化造成的高温才是令干旱“更加严重”的主要因素。
她说：“总体而言，我们的研究结果表明，这场干旱主要并非由降雨量少引起，而是由大气变暖导致陆地水分亏缺加剧所致。这意味着，即使在降雨量变化不大，或预计季节性降雨量会增加的地区，干旱风险仍在持续上升。”
科学家指出，气温每升高1摄氏度，大气容纳水蒸气的能力就会增加约7%。
自1850年至1900年以来，受燃烧煤炭、石油和天然气影响，全球平均气温已上升约1.4摄氏度。
