2026.4.17【午评：用生命线理论研究创业板】
         生命线 名词是我发明的，但生命线股票走势规律是自然存在的，我在长期实战和研究中发现：有一类中长线的牛股或指数经常沿着某一条均线长期上涨，走势极具规律性，我把这类股票或指数形态称为生命线股票或指数。 
         在契丹10条黄金均线中（5、10、20、55、89、144、233、377、450、610）中短期生命线主要是5、10、20这三条，周期再长的严格说就是波段交易了，长周期黄金均线不具备生命线操作含义，比如610牛熊分界线是判断股市和股票长期牛熊特点的均线，虽然重要，但不会有股票沿着这种长期均线上涨。              
       严格说刚形成生命线初期交易者并不能知道股票已经走出生命线，因为这时候短中长期均线排列都是比较错乱的，但你一定要牢记生命线雏形特点：5、10、20三条均线必须多头排列，而且股价站在5日线上，这时候可以尝试性买入，如果买入后股价一直沿着这三条均线的某一条向上运行，确定这就是生命线股票。卖出的条件：股价有效跌破生命线卖出。有效跌破的含义：当日收盘前放量跌破生命线超过3%幅度，或者放量跌破生命线后连续三天收盘在生命线之下！
         如图，创业板日线图，就目前看创业板有生命线雏形，类似指数这种生命线形态的个股一并这样处理，假设你看好某类生命线股票，可以遵循【线上持股线下抛】的口诀来应对，当然我个人建议做多选择恒生科技指数，那个安全可靠，但目前创业板的形态可以用生命线理论跟踪！

法国财政部长罗兰·勒斯科尔指出，G7须随时准备采取行动，应对因能源与供应中断带来的经济与通胀风险；三月在G7支持下，国际能源署曾创纪录释放战略石油储备以缓解霍尔木兹海峡相关供应断裂。与会者同意继续监测形势并在必要时采取措施，强调应保障霍尔木兹海峡航行自由，船只不应为通行向伊朗“付一分钱”。
法国央行行长维洛鲁瓦表示，各国央行承诺防止能源与大宗商品冲击通过二次效应嵌入核心通胀，必要时将果断行动，但需更多数据评估影响。
会议还重申将持续支持乌克兰、推动其在80亿美元IMF计划下的经济改革、维护对俄经济压力并参与切尔诺贝利封存穹顶修复。G7将继续推动针对稀土等关键矿产的具体替代供应链措施，并在六月法国依云召开的领导人峰会前提出可行方案。 Reuters

法国总统马克龙与英国首相基尔·斯塔默将于4月17日主持一次盟友会议，商讨在伊朗与美以冲突结束后，派遣多国部队保障霍尔木兹海峡安全与航行自由的方案。会议自中午（新加坡时间晚20时）开始，大部分以视频方式举行，预计约有30国领导人出席。 
伊朗自2月28日美以对其开战后对这一重要航运通道实施封锁，导致国际能源价格飙升；尽管目前存在脆弱停火，美国也对伊朗港口实施了封锁。欧洲担忧若封锁持续，将通过更高通胀、食品短缺和因航空燃油短缺而取消航班影响民众生活。 
爱丽舍宫邀约显示，与会者将呼吁恢复全面航行自由，并在“条件具备时”准备部署一支“严格防御性”的多国军事行动，以确保航道畅通。出席者强调，任何部署都将在战争结束后进行。 
英国方面称，斯塔默将主张“无条件、立即”重开海峡，称这是“全球共同责任”。法方要求获得伊朗不袭击过境船只及美国不封锁进出船只的承诺。会议还将讨论被封锁船只上逾2万名海员的人道与后勤问题。 
德国总理梅尔茨与意大利总理梅洛尼将亲自出席。英方表示，已在筹划“一旦条件允许即可展开的联合军事行动”，相关军方将在下周于诺斯伍德总部继续磋商。法方称，此举意在走出“对伊最大压力与重启战争之间的第三条道路”。

华盛顿 — 路透社审阅的一份电报显示，特朗普政府已指示驻外外交官在联合国推动一项“贸易优先于援助”的宣言，作为重塑美国对外援助支出模式的一部分。电报由国务卿马可·鲁比奥于4月15日下发，要求外交官在各自国家以适当最高层级传达该信息，并在4月20日前收集签署。
电报称，此举旨在通过联合国体系推广“美国优先”价值观并为美企创造商机；随文附上宣言全文，并指出将于四月底举办发布活动。白宫和国务院方面强调，贸易与自由市场是促进繁荣的最佳路径，美国从不吝慷慨，但反对“援助非贸易”的做法，指其助长腐败的非政府组织利益链条。
批评者警告，这一做法可能削弱人道救援与发展援助的及时性与公信力。洛克菲勒基金会副总裁埃里克·佩洛夫斯基表示，拿饥饿儿童作商机的说法背离美国传统价值，也可能危及国家安全。
路透社称，该电报信息此前由《华盛顿邮报》先行报道。美国驻联大大使迈克·沃尔茨在4月15日谈及联合国改革时曾提到，通过吸引私营部门、降低资本壁垒和推动亲商改革来“以贸易代替援助”的思路。

美国国防部长皮特·海格塞斯（Pete Hegseth）4月16日在五角大楼简报中援引圣经抨击媒体，将记者比作寻思“如何除掉他”的犹太法利赛人。海格塞斯的话旨在反击他认为对美以与伊朗战争的负面报道，并正值美国总统唐纳德·特朗普与“首位出生在美国的天主教领袖”里奥教皇（Pope Leo）争执升级之际。
海格塞斯在开场讲话中回顾周日讲道，引用“法利赛人看见奇迹后仍心硬，‘就商议如何除掉他’”，称“我们的媒体就像这些法利赛人”，但补充并非指所有人，只指“憎恨特朗普的传统媒体”。他批评媒体“只寻找负面，将每一善举挑剔为违规”，称其“唯有贬损之心”。
近来海格塞斯与特朗普多次以基督教语言谈论战争，两人皆称复活节当天在伊朗获救的美军飞行员为“奇迹”。海格塞斯3月在祈祷仪式上曾为部队祈求能对“不应获怜悯者施以压倒性暴力行动”。历史学者约翰·菲亚指出，历届美政府在战争时常诉诸基督教，但特朗普政府用词更为绝对。
五角大楼简报会结束不到一小时后，出生于芝加哥的教皇在X上发文警告：“祸哉那些为军事、经济、政治利益操弄宗教和上帝之名，把神圣拖入黑暗与污秽者。”海格塞斯长期批评媒体反特朗普，并就五角大楼记者证政策与媒体展开法律争斗——联邦法官3月裁定该政策违宪，五角大楼正在上诉。路透社

布加勒斯特，4月17日 — 罗马尼亚总统尼库索尔·丹表示，由四个亲欧盟政党组成的广泛执政联盟必须继续执政，尽管该联盟中最大党可能会在周一要求总理伊利·博洛扬下台。
消息称，左翼社会民主党（PSD）多名成员预计将于周一投票撤回对推动改革的自由党总理博洛扬的支持，此举可能引发政治危机。该联盟在十个月前组建，当时经历了一场因疑涉俄方干预而一度取消、随后重办并由中间派的丹获胜的高度对立的总统选举。四党当时曾同意由博洛扬轮换让位给社会民主党，时间定在2027年春。
但PSD与博洛扬在削减开支等问题上多次发生冲突，本次内部表决也违反了十个月前签署的联盟协议。丹在当地电台表示，来自相同政党的联合才是治理罗马尼亚的唯一选择，他不愿站队，将努力调解已显现的分裂。作为总统，丹有权提名总理，去年曾明确支持博洛扬。
知情人士称，PSD或可放弃轮换安排，前提是自由党提名非博洛扬人选；博洛扬表示无意辞职。分析人士认为，若无PSD这一议会最大党，亲欧盟多数无法维系，尽管该党当前民调明显落后于极右反对派。罗马尼亚将于2028年举行议会选举。评级机构在博洛扬政府提高税收并开始削减欧盟最大财政赤字后仍将该国列为最低投资级，同时警告政治不稳定是主要风险。——路透社

维尔纽斯，4月17日（路透）— 爱沙尼亚国防部长汉诺·佩夫库尔16日在立陶宛首都对路透表示，他确信若俄罗斯发动进攻，美国会出手相助，但警告欧洲目前无法单独在军事上抗衡莫斯科。
他还提到，爱沙尼亚情报部门2月警告称俄罗斯已为未来战争囤积弹药；俄罗斯则称有关其可能攻击北约的指控是荒谬的。
佩夫库尔说：“我信任美国，也信任我们的所有盟友。”他认为美欧在军事上相互需要，因此“不相信北约会解体”，并将当前摩擦比作长期婚姻中的波折，需要共同化解。
他指出，北约多数成员尚未兑现2025年将国防投入提高到至少占GDP 5%的承诺；爱沙尼亚2026年预算将达5.1%，位居前列。
谈及更广泛的局势，佩夫库尔表示北约应致力促成解决伊朗冲突，这将为美国把更多关注转回乌克兰创造条件。路透社报道。

阿比让（科特迪瓦）— 人权组织大赦国际4月16日呼吁布基纳法索执政军政府撤销对100余个非政府组织和社团的解散决定，称此举是对基本权利的公然攻击。
媒体称，军方在前一天下令解散118个组织，并禁止这些团体在国内开展活动，其中许多从事人权辩护工作。大赦驻萨赫勒区高级研究员乌斯曼·迪亚洛指出，解散NGO与布基纳法索宪法相抵触，亦不符合该国的国际人权义务，呼吁当局立即撤销相关决定。
自2022年政变上台以来，军政府加强对非政府组织、工会、集会自由及反对派的打压。布基纳法索长期与被指与“基地组织”和“伊斯兰国”有关的武装分子作战，获得国际资助的NGO常遭指控为恐怖分子提供情报或勾结。
据悉，军政府首领、易卜拉欣·特拉奥雷上尉于2025年7月签署了限制权利组织和工会运作的法律；今年4月，特拉奥雷曾对民众表示应“忘记”民主。—法新社

巴塞罗那，4月17日——西班牙首相佩德罗·桑切斯与巴西总统路易斯·伊纳西奥·卢拉·达席尔瓦将于周五和周六在巴塞罗那主导一系列全球左翼集会，旨在捍卫多边主义并动员左翼力量应对极右翼抬头。此次活动由西班牙及若干左翼政治网络组织，正值美国总统特朗普大幅削减人道援助、加强军事干预并威胁退出北约，动摇国际秩序之际，引发全球盟友重新评估立场。
周五启动的“全球进步动员”意在凝聚左翼倡导者，并在捍卫民主、推进绿色转型等议题上形成共同行动宣言。周六题为“为捍卫民主”的会议由西班牙政府承办，是卢拉与桑切斯在2024年发起峰会的第四次延续。
两位领导人均公开批评特朗普，且在各自国家面临日益强劲的极右翼挑战。南非总统西里尔·拉马福萨和墨西哥总统克劳迪娅·谢因鲍姆将出席——谢因鲍姆为自2018年以来首位访西班牙的墨西哥总统。桑切斯在访华期间表示，进步政党应团结，向公众表明其力量超越国内政治，并在北京与习近平承诺深化关系。
在匈牙利民族主义者维克多·欧尔班周日选举落败后，桑切斯称此事证明“浪潮可以被遏制”。由西班牙社会党主办的为期两天活动将有约3000人参加，包括现任及前任国家元首、约400位市长、工会、活动家与政党代表，桑切斯与卢拉将为大会闭幕。欧洲社会党秘书长贾科莫·菲利贝克表示，必须向民众展示“存在另一种选择”。（路透）
