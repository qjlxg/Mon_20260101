广东警方：有不法分子搭建虚假考试网非法牟利
中国今年高考将于星期天（6月7日）开考，警方称，有不法分子假冒官方网站、或利用信息网络生成虚假考试成绩等非法牟利，已被采取刑事强制措施。
“公安部网安局”公众号星期六（6日）发布消息称，高考在即，公安网安部门紧盯仿冒官方网站、虚假查分链接、涉考引流诈骗等突出风险。
广东公安网安部门发现，网上有多条涉仿冒教育考试类官网“假查分、伪造成绩”帖文。
调查发现，罗姓违法行为人从今年1月开始，仿照“中国教育考试网”等教育考试相关官方站点，搭建虚假教育考试网站，并通过社交平台进行非法引流，以提供“定制版成绩查询页面”“定制版成绩查询截图”等虚假成绩为名，收取费用、非法牟利，严重扰乱招生考试秩序。
目前，属地公安机关已对罗姓违法行为人采取刑事强制措施，相关虚假网站、引流账号均被关停清理。
警方披露，涉仿冒教育考试网站类违法行为分为三个步骤，包括：一是克隆官网，伪造“官方背书”，诱导网民误以为其为官方子站，骗取信任；二是瞄准“逃避型”心理，专门瞄准考试失利、压力过大的考生及家长，将虚假网站包装成可以“解决麻烦”的灰色渠道；三是贩卖假象，收割“遮羞费”，以提供“定制成绩页面”“生成逼真截图”为由收取费用，不仅骗取钱财，更诱导考生使用虚假成绩欺骗家人。
网警也提醒考生和家长查询成绩、确认录取进度，务必要认准官方渠道，切勿点击不明链接。
2026年全国统一高考将于6月7日开考。中国教育部星期三（3日）公布的数据显示，报考人数较去年减少45万人，降至1290万人，是高考报名人数连续第二年下滑。

业内：相关政策研究和制定要精细化
近段时间以来，加大购房补贴力度成为多地楼市政策的新着力点。中指研究院的数据显示，今年前5个月全国各省市（县）出台政策超430条。从政策方向上看，涉及优化限制性政策、调整公积金贷款、发放购房补贴、完善城市更新相关配套政策等内容。多位购房者表示，期待更多降低购房压力的新政策，包括房贷利率和交易税费能够进一步下调、发放购房补贴等。广东省住房政策研究中心首席研究员李宇嘉提出，购房补贴政策可能存在一些问题。例如补贴可能导致后续的需求提前释放，而不是产生增量需求，导致补贴一停，销售态势就恢复到常态。如果补贴一直延续下去，暂不论地方财力支持。即便有财政实力，但对于市场来说，就认为是新常态，自然也不会因为有补贴而加速购房。此外，补贴还要选择时机，也要精细化研究，哪些人群对补贴政策比较敏感，有针对性设计补贴发放范围。“总之，政策研究和制定要精细化，将宝贵的财政资金用在刀刃上。”（证券时报）

阿尔巴尼亚首都地拉那——数千名民众6月4日晚在地拉那街头集会，抗议一家与美国总统女婿贾里德·库什纳有关联的公司拟在亚得里亚海沿岸敏感生态区建设豪华度假项目，成为本周最大规模示威。
该项目由库什纳的投资公司Affinity Partners主导，投资额约为14亿欧元，拟在一座岛屿及靠近南部Vjosa‑Narta受保护湿地的未开发海岸带建设。当地湿地是火烈鸟、海豹和海龟产卵地，环保人士担忧数百公顷原始海滩和每年经过、繁殖的数千只火烈鸟将遭破坏。
上周工地动工和重型机械进场引发当地抗议，随后演变为地拉那大规模街头示威。6月4日晚，抗议者在总理埃迪·拉马办公处外聚集，举着粉色充气火烈鸟，高呼“革命”“停止项目”，并拉出“埃迪·拉马，下台”的标语。
作家林迪塔·科马尼（Lindita Komani）参加游行并表示：“阿尔巴尼亚不是出卖的地方，属于阿尔巴尼亚人民，我们决定这里的未来，不是腐败政治家可以任意处置我们的自然与文化遗产。”
拉马为项目辩护，开发方称将坚持“负责任的管理与环境改善”。阿尔巴尼亚经济与创新部长德莉娜·易布拉希玛伊表示，拟建项目正在编制环境影响评估，必须全面遵守环境立法并保护当地栖息地；欧盟环保指令和阿尔巴尼亚法律将为受保护泻湖及周边生态提供法律保障。
库什纳于2024年公布该度假区计划，属更大投资案的一部分，曾包括塞尔维亚贝尔格莱德的一处前军方总部。库什纳在2025年因当地抗议已放弃塞尔维亚项目。来源：路透社。

伦敦报道—总统唐纳德·特朗普威胁对被指未能遏制强迫劳动的贸易伙伴征收新关税，但专家、商界及部分人权组织认为此举难以遏制现代奴役，甚至可能适得其反。
美国贸易代表办公室提议对60国进口商品加征10%或12.5%关税，理由是这些国家未能阻止强迫劳动的商品贸易。新加坡贸易与工业部6月4日表示拒绝美方指控，并称三分之一对美出口或遭12.5%关税。
此举源于301条不公平贸易调查，旨在恢复今年2月被美国最高法院推翻的紧急关税。国际劳工组织最新估计显示，全球被强迫劳动者达2760万，比2016年增加约270万，近半数集中在制造、建筑、农业、渔业和采矿等出口相关行业。
批评者指出，美方措施更以贸易规模与地缘政治为导向，而非按剥削严重性精准制裁，未必能改善情况，反而可能激化部分国家的政治抵抗。欧盟反驳称其将于2027年生效的强迫劳动法规程序更为严格，覆盖范围广，且与美国在2025年达成的贸易协定将大多数欧盟商品对美关税上限定为15%。国际商会与人权组织也对关税的任意性与有限效果表示担忧。
