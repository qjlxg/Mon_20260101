南中国海仲裁案10周年 台湾主张应被纳入争端解决机制
南中国海仲裁案届满10周年，台湾外交部重申对南中国海诸岛的主权主张与相关政策没有改变，并呼吁将台湾纳入多边争端解决机制，主张任何有关南中国海区域行为规范的讨论均不应排除台湾。
南中国海仲裁案裁决星期天（7月12日）届满10周年。荷兰海牙仲裁庭当年裁定，中国大陆以“九段线”对南中国海的主权声索无法律依据，同时裁定台湾管辖的太平岛由“岛”降格为“礁”，意味着台湾无法对太平岛周围广阔水域提出专属经济控制权。
对此，台湾外交部星期天说，台湾对南中国海议题秉持“四点原则”，包括南中国海争端应依据国际法及海洋法，包括“联合国海洋法公约”，以和平方式解决；台湾应纳入多边争端解决机制；相关国家有义务维护南中国海航行及飞越自由；台湾主张应以“搁置争议、共同开发”方式处理南中国海争端，并愿在平等协商基础上，和相关国家共同促进南中国海区域和平与稳定，并共同保护及开发南中国海资源。

台湾外交部强调，作为国际社会负责任成员及南中国海区域重要利益关系方，任何涉及台湾权益及南中国海区域行为规范的讨论，均不应排除台湾参与。
台湾外交部称，反对任何一方在南中国海采取单边扩张主义、灰色地带胁迫、军事化部署及其他破坏和平的行为，并呼吁相关各方恪守国际法及和平解决争端原则，避免采取任何升高紧张、制造冲突或破坏区域和平稳定的作为。
目前，中国大陆、台湾、菲律宾、越南、马来西亚及文莱等均在南中国海存在主权争议。其中，台湾在争端中扮演的角色长期以来在很大程度上被北京的论述所掩盖。
对于南中国海仲裁案，北京则坚称“不参与、不接受”裁决。菲律宾外长拉扎罗强调，这项裁决不容谈判或妥协，并再次强调将继续依据国际法维护菲律宾的海洋权益。

不丹颁布生育补贴政策 应对人口下降
由于出生率急剧下降，加上越来越多年轻人为寻求发展机会而选择移居国外，不丹政府正在向家庭提供现金奖励，鼓励他们多生育子女。
法新社报道，过去十年间，不丹的年出生率下降超过四分之一。
据不丹国家广播公司报道，首相策林·托杰多次就人口骤减发出警告，称其为“生存危机”。
托杰在政府于6月推出的“第三胎及以上”计划介绍中写道：“证据非常明确——不丹的生育率已下降到接近或低于更替水平。”
这项计划为每个第三胎及以后的孩子提供每月105美元的补贴，直至孩子年满三岁。
报告显示，自2020年以来，不丹第三胎及以上出生的比例下降27%。
联合国估计，不丹的生育率已降至每名妇女约生育1.8个孩子，低于人口更替水平，而65岁及以上人口的比例预计将从约6%上升到2050年的17%。
首相说：“这些并非抽象的统计数据，它们对不丹的劳动力、财政可持续性以及全国各地社区的社会结构构成了切实且日益加剧的压力。”
政府担心，出生率下降加上持续的人口外流，将导致劳动年龄人口过少，难以支撑经济发展和应对人口老龄化问题。
专家认为，不丹家庭规模缩小的原因很复杂，从生活和育儿费用的上升到生活优先级的变化，不一而足。

中国新能源车平均车龄仅1.8年 把电车当手机换
一份最新行业报告揭示，中国新能源乘用车的平均车龄仅为1.8年，远低于传统燃油乘用车的8.2年。这显示新能源汽车正成为如智能手机般的“快速消费品”。
综合第一财经和《21世纪经济报道》报道，中国汽车工业协会（中汽协）联合和君咨询发布的《2025中国汽车后市场年度发展报告》显示，中国汽车市场的油电产业生命周期与后市场需求已走向全面分化。
数据显示，目前燃油乘用车平均车龄达8.2年，七年以上老车占比接近六成；相比之下，新能源乘用车平均车龄仅1.8年，其中一至三年车龄的车辆占比高达90%。燃油乘用车的平均车龄是新能源车的4.5倍。
乘联分会秘书长崔东树认为，出现这样的现象一方面是因为新能源车在近五年才迎来爆发式普及，市场存量整体偏新；另一方面，更深层的原因在于新能源汽车极快的技术迭代速度。
麦肯锡的一份报告指出，目前中国新能源新车的开发周期已大幅压缩至约24个月，远短于传统燃油车时代约60个月的开发周期。由于三电（电池、电机、电控）及智能驾驶技术迭代迅猛，上市仅两三年的车型在续航、算力及补能效率上便会出现显著的技术代差。叠加置换政策刺激，直接导致车主的换车周期大幅缩短。
中国汽车流通协会数据显示，传统燃油车换车周期约为六至八年，而新能源车换车周期已被压缩至三至五年。在二手车成交端，燃油二手车平均成交车龄为8.6年，新能源二手车则仅为3.4年。
极速的迭代与缩短的换车周期，正对新能源汽车的保值率造成冲击。
6月份中国汽车保值率研究报告显示，所有三年车龄的新能源二手车保值率均出现环比下滑。其中，受插混新车技术快速推陈出新的影响，二手插电混动车型贬值加速，保值率已下探至44.2%。
《中国新闻周刊》和央视新闻媒体报道，二手车商目前普遍不愿收购车龄六年以上的新能源车。对于使用七至八年的新能源车，残值通常仅剩新车指导价的两成左右；八年或12万公里，已是新能源车的“斩杀线”。
