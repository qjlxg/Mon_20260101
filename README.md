法新社报道，以色列总理内坦亚胡办公室星期天（1月11日）发表声明说：“伊朗和伊朗的代理人包括黎巴嫩真主党、哈马斯、胡塞武装，不仅威胁以色列，还威胁地区稳定和国际安全。

“以色列的敌人要知道：我们无处不在，时刻紧盯着他们。”

内坦亚胡在另一份声明中说：“我认为德国和以色列是天然合作伙伴。我们在‘箭-3’（Arrow III）反弹道导弹防御系统方面开展合作，还在很多领域方面进行合作。”

上月，德国批准了一项价值31亿美元（约40亿新元）的箭-3反弹道导弹防御系统合同扩充计划。此系统由以色列制造，并在美国的支持下研发。这项最初于2023年签署的合同，如今价值约65亿美元。以方称，这是以色列迄今为止最大的军事出口合同。

观点：人工智能竞赛对美、中与世界的三种可能结局
全球人工智能竞争仍在加剧，目前可见三种主要走向。第一种是美中主导的“双雄格局”。斯坦福2025年AI指数显示，美国在重要模型数量和投资额上领先，中国在论文和专利上占优，且中国模型性能近年迅速提升，已逼近美方水平。更深层的差异在于两国的经济模式：中方以国家主导的投资动员资源、技术和资本，受限于高端芯片但正快速补足；美方依赖科技巨头、风险投资与政府支持，但存在泡沫化和以复杂债务扩张数据中心的隐忧。
第二种情形是在AI泡沫破裂后出现的“三极格局”，欧盟凭借以伦理为导向的监管（如AI法案）获得更大影响力。尽管欧在性能上仍落后，但其规则制定可能在某些关键但争议较小的领域（如AI安全）构建跨区监管纽带。正如前美国国家安全顾问杰克·沙利文所言，美中不应在不交涉风险的情况下一路竞进，亦应就气候与公共卫生等跨国挑战中的机遇对话。
第三种是技术性颠覆：如低成本大模型、本地化处理、稀疏模型和合成数据等突破，可能大幅降低构建和部署成本，侵蚀美中先发与规模优势，使中等实力的公私主体迎头赶上，同时带来更广泛的风险与经济机遇。
结论：AI走向仍不确定，技术可能最终胜出于政策，但现在对不同情景的深入思考有助于政府与企业应对未来不确定性。
