中国出台反外国不当域外管辖条例
《中华人民共和国反外国不当域外管辖条例》公布，当中明确中国政府有权对与中国存在适当联系的行为实施域外管辖措施。
据新华社星期一（4月13日）下午报道，中国总理李强签署国务院令，公布《中华人民共和国反外国不当域外管辖条例》（简称《条例》），自公布之日起施行。
《条例》旨在维护国家主权、安全、发展利益，保护中国公民、组织合法权益，维护以国际法为基础的国际秩序。《条例》共20条。
首先，明确适用范围和工作机制。外国国家违反国际法和国际关系基本准则，实施不当域外管辖措施，危害中国国家主权、安全、发展利益，损害中国公民、组织合法权益的，中国政府有权采取相应的措施。
明确中国政府有权对与中国存在适当联系的行为实施域外管辖措施。规定国家建立健全应对外国不当域外管辖有关工作机制，统筹协调相关工作。中国国务院有关部门按照职责分工承担应对外国不当域外管辖具体工作。
其次，建立外国不当域外管辖措施识别、阻断和反制制度。建立外国不当域外管辖措施识别制度，明确识别工作的程序、考虑因素等。明确任何组织和个人不得执行或者协助执行外国不当域外管辖措施，并规定了相关豁免制度。
同时，明确国家层面采取的反制和限制措施。针对推动实施或者参与实施外国不当域外管辖措施的外国组织、个人，建立恶意实体清单制度，明确相关程序、采取的措施和豁免制度等。针对执行或者协助执行外国不当域外管辖措施的组织、个人，规定了禁执令制度，并明确相应的法律责任。
最后，健全服务保障制度。明确任何组织、个人执行或者协助执行外国不当域外管辖措施，侵害中国公民、组织合法权益的，中国公民、组织可以依法起诉。省级以上人民政府有关部门提供指导服务。行业协会商会发挥行业自律和协调作用，为会员提供与应对外国不当域外管辖有关服务。

公安部称，此次联合行动是中斯两国深化执法合作、加大打击跨境电信诈骗力度的成果。自今年年初以来，双方针对部分涉诈人员及相关地下产业由他国转移至斯里兰卡的态势加强了协作。
公安部透露，斯里兰卡警方近期多次开展清查行动，摧毁数个诈骗窝点并抓获多名涉案人员。参与此次遣返工作的中国湖北省公安机关正在对相关案件进行侦查。
公安部一名官员表示，将继续加强与更多国家的执法合作，共同构建打击跨国犯罪的务实高效新机制。来源：新华社

深圳报道 — 中国国务院总理李强在周五举行的高层座谈会上，要求加快在制造业推广工业人工智能，推动制造业的数字化与智能化升级。李强在与新兴独角兽Moonshot AI创始人杨志林等企业家和业界专家座谈时指出，要更深入落实“AI+”战略，加快把人工智能嵌入制造环节，强化先进制造与现代服务的融合，增强产业体系整体实力。 
李强并表示，尽管外部环境今年更为复杂多变，但在及时出台积极的宏观政策支持下，经济已开局良好。他还强调要抓住新兴产业发展机遇，扩大就业、提升收入、拉动内需，并鼓励企业家在创新和带动就业方面发挥更大作用。 
受邀参加座谈的还有广发证券首席经济学家郭磊、中国人民大学金融学教授马广荣以及招商局港口控股董事长冯伯明等。

美国联邦通信委员会（FCC）周四表示，正考虑更广泛的措施，禁止中国电信运营商在美运营数据中心，并进一步限制其接入美国网络与基础设施。草案同时建议禁止美国运营商与这些被认为存在“重大国家安全风险”的公司互联互通。中国移动、中国电信和中国联通均被列入受限实体名单，可能被要求出售、转让或关闭在美的数据中心和网络节点，甚至被迫退出美国市场。
分析人士称，此举标志着华盛顿对中国科技企业多年打击的升级，尽管短期内对相关公司财务影响有限。塔霍塔律师事务所合伙人王宇祥表示，草案将管制从禁止向公众直接提供服务和限制硬件，进一步延伸到对底层互联网基础设施和互联协议的更深层控制。

美伊在伊斯兰堡举行的首轮面对面谈判首阶段已结束，双方专家团队继续就关键问题交换书面文本。伊朗国家广播电视台（IRIB）援引接近谈判团队的消息称，谈判可能在今晚晚些时候或周日继续。
