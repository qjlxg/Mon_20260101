别车指的是前车故意阻碍后车，或在超车时故意挡住被超车辆，属于开“斗气车”的典型危险行为。常包含突然变道、紧急制动等操作，极易引发交通事故。

中国官媒报道并未明确透露事发时间，仅指发生在几年前。红河舰是2022年12月入列服役，意味着这起事件应该是近三年发生。
至于涉事的外国军舰，有中国军事自媒体根据官媒披露的信息，推测应是美军的阿里·伯克级驱逐舰。
054A型护卫舰满载排水量4000余吨，最高航速约28节。阿里·伯克级驱逐舰排水量则逾9000吨，最高航速超过30节。

此前，伊朗司法医学组织负责人在4月12日通报的死亡人数为3,375人；总部在美的人权活动者新闻社（HRANA）4月7日的统计则称至少有3,636人丧生，其中包括1,701名平民（至少254名儿童）、1,221名军人及714名身份未明者。法新社指出，由于报道受限，无法进入袭击地点或在伊朗境内独立核实伤亡数字。

伦敦，4月18日 — 英国副首相大卫·拉米周六表示支持首相基尔·斯塔默，就任命彼得·曼德尔森为驻美大使引发的争议力挺首相。
曼德尔森由斯塔默任命，后因与已故性犯罪者杰弗里·爱泼斯坦的关联被解除职务，此事令斯塔默的判断遭到质疑。政府透露，安全审查曾建议不宜任命，但外交部官员在未告知首相的情况下推翻了该建议，周五这一情况使斯塔默再度面临辞职压力。

曼德尔森的律师在《卫报》最先报道他未通过部分审查时未予置评。路透社报道。

联邦官员已将位于纽约州韦斯特切斯特县多布斯费里的一个移民儿童收容所中的未成年移民转移至其他地点，原因是接到该处存在虐待的报告。该所由儿童村（Children’s Village）运营。 
CNN于4月16日首次报道了对该设施的虐待指控。美国卫生与公众服务部（HHS）在致《纽约时报》的声明中表示，已注意到相关指控，并将这些无陪伴儿童转移至其他地点，同时已通知调查机构。对此设施的审计于1月20日完成，随后一周儿童被移出。无陪伴移民儿童由卫生与公众服务部下属的难民安置办公室（ORR）负责。 
HHS发言人安德鲁·尼克松称，“ORR在管护下儿童的安全与福祉是首要任务，任何可信的关切都会迅速且彻底地处理。” 
到访该所的一位律师称，曾被拘留的儿童遭受约束和隔离惩罚。代表政府监管下儿童的非营利组织Children’s Rights首席法律顾问利西亚·韦尔奇表示，青少年曾投诉被单独关押多天，称之为“红屋”。她警告称，长期拘留会加剧儿童行为问题，这类有害做法可能导致严重受伤。 
儿童村发言人克里斯托弗·鲁卡斯表示，这些指控令人不安，“我们对任何形式的惩罚零容忍，所有受托青少年都应得到最高水平的关怀、支持与专业对待。” 
负责保护特殊需求人员的纽约州司法保护中心表示已收到关于该收容所的投诉并在调查中，但未作进一步评论。

布达佩斯——选举办公室4月18日表示，经邮寄票、海外使领馆票与移交票计入后，彼得·马嘉尔领导的中右翼“蒂萨”（Tisza，Respect and Freedom）党在199席议会中扩大至141席，取得三分之二以上的超级多数。4月12日大选中，蒂萨以压倒性胜利结束了维克多·欧尔班16年的执政；在创纪录的高投票率下，欧尔班迅速承认败选。马嘉尔称这是“前所未有的多数、前所未有的授权，也意味着责任”。
对比上届，欧尔班的执政党Fidesz在2022年曾在106个单一选区中赢得87席，本次仅得10席，全国议员将降至52人。马嘉尔的胜利推动匈牙利资产上涨，市场预期与欧盟可能恢复因法治争议而被暂停的数十亿欧元资金有关。Capital Economics分析师Liam Peach指出，若成功解锁欧盟资金将支持投资并降低主权风险溢价，但对增长的影响主要在中期，短期仍受伊朗冲突等外部因素及国内财政政策影响。
初步统计曾显示蒂萨为138席，最终增至141席，使马嘉尔有能力推翻欧尔班的宪法修订并展开反腐行动。马嘉尔承诺将于5月9或10日上任后启动大规模反腐，以争取欧盟资金并重振近三年停滞的经济。欧尔班虽一再否认有不当行为，但在4月16日的在线访谈中表示，关于与Fidesz关系密切商人财富的媒体报道可能促成其败选；他未对报道真实性发表评论。路透社

华盛顿/伊斯兰堡讯——伊朗首席谈判代表穆罕默德·巴格尔·加利巴夫表示，与美国的会谈已取得“进展”，但在核问题和霍尔木兹海峡问题上仍存在较大分歧；美国总统特朗普则称与德黑兰有“非常好的对话”，同时警告不要以海峡为“勒索”工具。
双方在4月18日均未就谈判细节公布更多信息。冲突进入第八周以来，战事已造成数千人伤亡，并蔓延至黎巴嫩；霍尔木兹海峡事实性关闭导致油价飙升，该海峡战前曾承运全球约五分之一的原油。
4月18日，伊朗决定重申对海峡的控制，再次关闭这一能源咽喉，称此举是回应美国对伊朗港口的持续封锁并违反停火；伊朗最高领袖莫杰塔巴·哈梅内伊表示海军已准备对敌人给予“新的惨痛打击”。特朗普称此举为“敲诈”，但同时为会谈点赞，并威胁若在4月22日停火到期前未达长期协议，将“重新投弹”。
与此同时，至少两艘船只在4月19日试图通过海峡时遭到炮击，印度为此召见伊朗驻印大使并表示严重关切。美军中央司令部称美方正在执行对伊朗的海上封锁，但对伊朗最新行动未作评论。
上周在伊斯兰堡的会谈中，美国提出对伊朗核活动作出为期20年的中止提议，伊朗则建议停止3至5年；伊朗副外长赛义德·哈蒂布扎德称尚未确定下一轮谈判日期，须先达成一项理解框架。
在停火前景与国内政治压力交织下：特朗普为保共和党在11月中期选举的议席，面临油价高企、通胀上升及支持率下滑等压力。尽管4月17日因海峡恢复通航预期油价下跌约10%并推高全球股市，但海湾仍有数百艘船只和约2万名海员滞留等待通行。
