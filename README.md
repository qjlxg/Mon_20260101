全球化肥市场短缺之际 中国进一步收紧化肥出口
（北京路透电）中国正在收紧化肥出口以保护国内市场，因中东战事陷入全球化肥短缺的局面，将因此雪上加霜。
知情人士称，北京在3月中旬禁止了氮钾混合肥和某些磷酸盐肥料的出口，叠加此前对尿素出口的禁令和配额，目前中国仅有硫酸铵等少数几种化肥可正常出口。
最新禁令意味着去年中国的化肥出口量中，约有一半到四分之三受到限制，可能高达4000万吨。
中国是全球最大化肥出口国之一，去年出口额超过130亿美元（166亿新元），过往一直通过管控出口来保持国内化肥价格稳定。
经由霍尔木兹海峡运输的化肥，约占全球海运供应量的三分之一，伊朗、沙特、卡塔尔三国均位列全球前十大尿素出口国。随着这一关键通道实质处于封锁状态，全球化肥供应陷入紧张，国际尿素价格已较战前水平上涨约40%。
在上海星期三（3月18日）举行的一场化肥会议上，多名销售人员预计，在6月至8月的中国出口高峰期之前，化肥出口禁令不会解除。
据国际贸易中心的数据，去年巴西、印尼和泰国进口的化肥中，约五分之一来自中国，马来西亚和新西兰的这一比例为三分之一。
印度关键化肥供应也高度依赖进口。印度贸易数据显示，去年自中国进口化肥比例占总量的16%，从中东进口了超过40%的尿素和磷酸二铵，该国目前已要求中国对尿素实行出口配额。

今年出台的新法赋予联邦安全局（FSB）要求运营商断开用户连接的权力，并准许建立由其管辖的审前拘留网络。匿名外交官称，此举旨在战争背景下稳固国内控制：战事拖延或结束都可能引发民意波动，官员们希望预防可能出现的异议，准备在网上实施“大规模镇压”的选项。 
俄罗斯调查记者安德烈·索尔达托夫指出，安全部门记得1989年阿富汗战争结束后和1991年苏联解体带来的动荡，正努力构建防止类似后果的体系。 
两名知情人士称，俄方借鉴中国和伊朗做法，着手能在移动与固网大范围屏蔽网络同时控制在线通信的手段。自2022年起，俄方颁布一系列强化审查与FSB权力的法律。今年2月24日，普京在FSB年会上要求加强反恐与信息领域管控。克里姆林宫称所有措施均属维护安全的合法行为。 
近月行动加剧：据《商业报》统计，截至今年1月中旬已封禁逾400个VPN，比去年底增长约70%。当局近期放慢了在俄和乌均广泛使用的Telegram并将其创始人帕维尔·杜洛夫列入涉及“恐怖主义”的刑事调查，称该平台被乌克兰及北约情报机构渗透；Telegram否认并指责当局试图强推亲国有的Messenger“MAX”。上月WhatsApp因未遵法被全面屏蔽，母公司Meta谴责此举削弱公众安全。 
部分年轻人转换VPN以继续使用受限的Instagram、Snapchat等西方应用。一位只愿以名叫安德烈自述的青年讽刺道：“如果这些老政治家要把所有东西都封掉，为什么不能做出有趣的俄国应用？”
