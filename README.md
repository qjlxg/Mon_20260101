据报道，习近平曾于2023年与美国前总统拜登在旧金山会晤时，呼吁美国“支持中国和平统一”，拜登团队并未接受。 
《华尔街日报》分析，对北京而言，即便只是措辞上的微小让步，也将是重大心理胜利，可能影响台湾内部舆论。不过消息人士指出，习近平并不希望特朗普亲自介入有关台湾未来的谈判。
知情人士称，习近平与特朗普会晤时，也可能会试图将台湾现任总统赖清德提出的两岸互不隶属言论，界定为具有挑衅意味的“新两国论”，是造成区域不稳定的因素。

菲律宾中央银行（BSP）预计，受中东冲突引发的油价冲击影响，2026年全年平均通胀率将升至5.1%。该行指出，美、以二月下旬对伊朗的攻击引发地区紧张，推动油价及其他大宗商品（尤其是化肥）上涨，这将推高食品和燃料价格。

马尼拉 — 菲律宾能源监管委员会（ERC）已暂停全国批发电力现货市场（WESM）运作。WESM为国内电力交易中心，负责电力供需调度并由市场与商业力量决定电价。此次暂停后，政府拟改采经调整的行政定价机制，相关方案正与利害关系人磋商，可能于4月1日前后定案。 
拟议措施包括：燃煤电厂按固定费率支付；天然气电厂按既有合约价结算；水电、地热等可再生能源实施行政定价并享有优先调度；燃油机组则在发电调度或合约时间按行政价支付。 
ERC指出，按现行规则以一、二月电价作为行政定价基础，已无法反映燃料价格受限下的当前情势，因此需修订定价机制。ERC委员Francis Saturnino Juan表示，此举旨在保护菲律宾消费者并确保电力供应稳定可靠；新规将持续生效，直至当局认为可以恢复正常市场运作。 
总统费迪南德·马科斯（Ferdinand Marcos Jr.）周二宣布全国能源紧急状态，称中东持续动荡对国家能源供应的可获性与稳定构成“迫在眉睫”的威胁。
