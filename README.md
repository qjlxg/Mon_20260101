林毅夫：中国基础科研经费占比应翻倍至最少15%
中国著名经济学家林毅夫认为，在许多产业领域，中国已从落后于发达国家变为“并跑”，而处在世界最前沿时，必须自己发明。他建议中国将基础研究经费的占比，从目前的7%增加一倍多到至少15%。
世界银行前首席经济学家、北京大学新结构经济学研究院院长林毅夫接受澎湃新闻访问时，发表上述观点。
他指出，中国在许多产业已经与发达国家“并跑”，甚至在一些领域开始“领跑”，比如家电产业、造船、新能源汽车等。
林毅夫强调，世界正面临第四次工业革命，这一次中国与发达国家站在同一起跑线上。如果能抓住机遇，就能实现民族复兴；如果错过，就可能再次落后。
在科研投入方面，2025年中国基础研究经费占全社会研发经费的比重超过7%。
林毅夫说，发达国家基础研究的经费占比普遍在15%至20%。作为大国，且处于百年未有之大变局，他认为中国应将这个占比逐步提升到15%至25%。
在科研经费的资金来源上，发达国家政府占比约40%，企业40%，社会20%。林毅夫说，鉴于中国企业过去习惯了引进吸收，积极性不高，社会基金会也较少，政府必须做得更多，占比应达到50%甚至更高。
他同时强调，政府的资金是有限的，不能“撒胡椒面”，必须挑选对国家经济发展、国家安全等至关重要的领域，比如晶片、关键材料、生物医药、能源安全、农业育种等。
林毅夫认为，政府要支持那些具有共性和重大平台作用的基础科研项目，一次突破能带动多个领域。

苹果公司据报正研发新一代iPhone防盗功能。未来若系统侦测到手机疑似遭人抢夺，设备或可自动锁定，以弥补现有保护机制在手机处于解锁状态时仍可能被滥用的漏洞。 
财经杂志《福布斯》（Forbes）报道，研究人员在苹果系统代码中发现，这项功能已进入开发阶段。据悉，苹果现有的“被盗设备保护”（Stolen Device Protection）虽然可以有效保护用户数据，但若歹徒在手机处于解锁状态时迅速抢走设备，仍可能在系统锁定前利用短暂时间窗口进行操作。
新功能据报将利用iPhone内置的加速度计，侦测设备是否出现类似被突然夺走的异常动作，这个原理与安卓系统现有的“盗窃侦测锁”（Theft Detection Lock）相似。
此外，系统也可能结合其他资讯进行判断，包括iPhone与Apple Watch之间的距离变化，以及用户是否身处熟悉地点、是否连接常用无线网络等因素，以提高判断准确度。
若系统综合评估后认为手机可能遭抢夺，将立即锁定设备，并限制用户修改Apple ID密码等关键设置。同时，银行应用程序、密码管理工具等敏感应用，也需通过Face ID或Touch ID验证才能开启。
网络安全公司ESET全球网络安全顾问摩尔（Jake Moore）认为，若新功能能够结合熟悉地点以及与Apple Watch的距离等额外数据进行判断，将进一步提升防盗效果。
不过，他提醒，即使手机被锁定，犯罪分子仍可能利用部分破解工具尝试取得设备资料，因此用户在手机失窃后的“黄金一小时”内，仍应尽快停用SIM卡、联系银行并锁定重要账户。
摩尔也建议用户采用多重验证机制，并为银行、密码管理等敏感应用单独启用Face ID等生物识别验证，以进一步降低资料遭盗用的风险。

五角大楼忙于补充消耗殆尽的弹药弹体，军人家属承受长期部署带来的压力。伊朗继续对地区盟友发起反击，曾于6月5日对巴林和科威特发射弹道导弹。特朗普4月宣布停火，但冲突陷入僵持，霍尔木兹海峡基本处于关闭状态，特朗普警告若谈判失败将恢复全面轰炸。
美军必须保持高度戒备：储备导弹与拦截器、依靠无人机与卫星情报更新伊朗目标名单。匿名官员称维持“最高警戒”极为紧张，前中司令约瑟夫·沃特尔警告目前阶段“极其危险”。五角大楼发言人肖恩·帕内尔表示将尽一切方式支持前线部队。
伤亡方面，陆军预备役军士长科里·希克斯在一次伊朗无人机袭击中受伤并患上创伤性脑损伤，正在马里兰州沃尔特里德医院治疗。冲突约有400名美军受伤，多为脑伤，超过90%已复员执行任务，13人遇难。
军属对停火中的不确定性感到恐慌。伊朗官方媒体频繁宣称袭击美舰机，伊朗称6月5日在阿曼湾向美舰示警射击，美方否认。加州军属雅迪拉·德桑坦说“不知道发生何事很可怕”，她每天给儿子发早安短信以表慰问。路透/益普索五月民调显示，仅四分之一受访者认为对伊朗的军事行动值得。
谈判若能开放霍尔木兹海峡（战前约占全球原油运输20%），协议可能延长停火但推迟核问题等棘手议题。国防部长皮特·赫格塞斯说，补足导弹和拦截器库存可能需数年。战略与国际研究中心的汤姆·卡拉科指出，战争耗损不仅是库存，还有装备与人员的磨损。
希克斯仍与海外同袍保持联络，许多人被延长部署令沮丧。他怀念在科威特袭击中丧生的六名战友，其中包括39岁的妮可·阿摩尔军士长，“这是我余生都要面对的记忆。”
