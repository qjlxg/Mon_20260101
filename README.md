欧洲运输与环境联合会：伊朗战争推高全球航运成本
欧洲运输与环境联合会发布报告说，受美国和以色列对伊朗的战事影响，全球航运业燃料成本大幅攀升，在推高行业运营压力的同时，也为加快能源转型提供契机。

航空燃油短缺冲击 或从亚洲扩散至欧美
（伦敦彭博电）随着旅游旺季到来，伊朗战争引发的燃油短缺危机对亚洲航空业的打击预料将进一步恶化，并可能扩散到欧洲等其他地区。
随着燃油价格创新高，亚洲航空公司纷纷取消航班，各国政府也开始采取防御措施。中国已限制燃油出口，韩国则探讨把原本销往国外的燃油保留给国内市场。
越南民航局警告，4月初可能会开始出现燃油短缺。越南航空已暂停部分国内航线，廉价航空公司越捷航空则削减部分国际航班。菲律宾航空公司则说，燃油供应只能保障至6月底，之后还未有着落。
新西兰航空公司已取消1100趟国内航班；澳大利亚最繁忙机场悉尼机场也警告，下月能否获得航空燃油供应还是未知数。
能源数据分析公司Kpler首席研究分析师利托里亚说，目前的短缺现象是不均衡的，而供应紧张最严重的是东南亚等依赖进口的地区。
不过，欧盟和英国再过几周可能也会面临类似情况，因为他们也依赖波斯湾炼油厂的供应。
能源分析公司Sparta Commodities高级石油分析师琼斯-勒克斯说，如果霍尔木兹海峡持续关闭，欧洲将在5月开始出现燃油短缺。他说，无论欧洲炼油商采取什么行动，如提高开工率、推迟检修或是调整产出以偏向航空煤油，都无法弥补霍尔木兹海峡关闭所造成的供应短缺。
美国虽然是石油净出口国，也不免受到冲击。美国西岸、夏威夷和阿拉斯加去年的航空煤油消耗量有逾18%从国外进口，而这些进口绝大部分来自韩国，因此易受供应冲击影响。美国一些航空公司包括联合航空已因为燃油价格上涨，而取消部分不赚钱的航线。
根据能源数据分析公司Energy Aspects的数据，全球航空燃料和煤油炼厂3月的日产量预计将比上月减少约60万桶，尽管降幅仅约7%，但此时正值夏季旅游旺季来临前，需求逐步上升的阶段。
即使中东战火能快速降温，伊朗解除对霍尔木兹海峡的封锁，遭受破坏的全球供应链仍需数周或数月才能完全回归常态。
大宗商品交易公司Alkagesta创始人兼首席执行官鲁斯塔莫夫指出，这是因为贸易流需要时间逐步恢复正常，炼油厂须调整产出结构，航空公司也要一段时间重建航班时刻表。

3月28日10:30，安努廷在“记者见面会：一个月的全球危机——泰国的应对”上表示，受中东冲突影响，全球能源安全紧张、油价普遍上涨，泰国与各国一样面临挑战。政府为此成立中东局势应对与监控中心（ศบก.），每日更新措施，重心放在外交、能源安全、物价管控与减轻民生冲击四方面的协调与应对。
他回顾称，政府曾在3月前15天暂时维持油价以缓和民生影响，但随冲突延长不得不调整政策，停止价格固定以遏止囤货和走私，并减少燃油基金補贴（由每升约24铢降至16铢）。同时通过外交斡旋，已争取到伊朗允许泰国油轮安全通行霍尔木兹海峡，缓解运输风险与加速供应恢复，部分地区油站的短缺已见缓和。
安努廷呼吁民众勿恐慌抢购，指出危机前日均用油约6700万升，目前一度升至8200万升，若能回落至6700万升即可维持正常生活。他强调泰国油价仍低于大多数邻国（仅高于印尼与文莱两国），政府将持续打击囤积和非法外销，保障国内供给与民生稳定。

2016年3月28日，王普拉查多克研究所秘书长伊萨拉·赛里瓦塔纳武提（Isara Sereewatthanawut）在脸书发文，提出对国会议员（下议院）餐饮经费的五项改革构想，主张以制度减少浪费与腐败。
伊萨拉指出，目前议员餐饮预算名义约每年80多万泰铢，实际使用约50万，另加委员会餐饮约20多万。通常按每人每日三餐上限1,000铢，若开会至晚间再加两餐，造成备餐过量与大量食物浪费，且承包商因雇工与餐具流失等成本常把价码抬高。
他提出五条改革：
1. 改为个人额度卡（每日上限不超1,000铢），按实际消费结算，剩余返还国库；不愿领取者须在四年任期内一次性声明放弃。 
2. 每日/每餐开支应公开透明，接受社会监督。 
3. 餐饮承包成本需合理检讨，包含临时雇工与容器管理等问题。 
4. 严禁将餐食带出餐厅，服务应在指定场所内完成，避免人数估算被无限放大。 
5. 将此项节省放在更大反腐框架下看待——与全国性贪腐、回扣问题相比，节省的意义更大。
他强调，真正的改变不在于让议员自带饭，而是要全体500名议员自觉不贪，减少将政治当生意的动机，才是最有效的节支之道。
