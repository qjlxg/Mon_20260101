中国副外长：中国任今年APEC东道主意义远超亚太范畴
中国外交部副部长马朝旭受访时说，中国担任2026年亚太经济合作组织（APEC）东道主，意义远超亚太范畴。
综合新华网和中新社星期天（2月8日）报道，马朝旭就亚太经合组织“中国年”在北京接受媒体采访时说，当今世界变乱交织、动荡加剧，单边主义、保护主义抬头，国际经济贸易秩序遇到严峻挑战。亚太地区面临何去何从的抉择，国际合作也面临不进则退的考验。在这一大背景下，中国担任2026年APEC东道主，意义远远超出APEC本身，也远远超出亚太范畴。
马朝旭指出，作为区域合作的引领者、多边主义的捍卫者、世界经济的主引擎，中国举办APEC第三十三次领导人非正式会议，充分体现了负责任大国的担当，不仅将为亚太区域合作注入强大动力，也将为全球经济增长与国际合作提供更多的确定性、稳定性和正能量。
2026年APEC第一次高官会即将在广州举行。马朝旭说，这次会议将开启APEC“中国年”。
他说，APEC“中国年”主题为“建设亚太共同体，促进共同繁荣”，在去年12月举行的非正式高官会上，这一主题得到APEC成员一致同意和普遍支持。
马朝旭介绍，“开放、创新、合作”是APEC“中国年”的三大优先领域。“开放”是构建开放型经济，反对保护主义，促进贸易投资自由化便利化。“创新”是坚持创新驱动，发展新质生产力，打造APEC合作新的增长点。“合作”是挖掘APEC成员比较优势，开展政策对接和经验分享，实现合作共赢。
他说，作为APEC东道主，中国将发扬历史主动，引领塑造区域经济一体化，推动建设亚太共同体，为构建新发展格局、促进高质量发展营造良好外部环境。

【分析师：近期科技股抛售并不意味着AI投资热潮结束】金十数据2月9日讯，M&G Investments分析师法比亚娜·费德利表示，近期全球科技股抛售并不意味着人工智能投资热潮终结。她指出，这更像是“一次有意义的市场调整，而非AI投资周期的结构性断裂”。费德利认为，投资机会远不止集中在少数几家受到市场关注的美国大型科技公司。在消费、传媒和金融等多个行业，企业正积极部署AI以优化成本结构和提升收入表现。她补充道，AI投资热潮的最大受益者可能并非那些投入巨资的公司，而是最善于抓住AI机会、具备战略布局优势的企业。
