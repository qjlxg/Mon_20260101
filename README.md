菲律宾劳工与就业部（DOLE）已预先拨出12亿比索（P1.2B），用于“进一步强化并调整”针对弱势及受冲击劳动者的相关方案，此举与政府因中东冲突宣布的全国能源紧急状态相呼应。 
DOLE表示，此举配合政府推出的统合生活、产业、粮食及交通方案（UPLIFT），以确保国内能源供应稳定、维持经济活动并保障民生。资金将主要用于两大社会保护项目：TUPAD临时就业计划和DOLE综合生计计划（DILP），以支援可能因能源紧急状况而失业或被迫转业的劳动者。 
同时，DOLE正与外劳部（DMW）及海外劳工福利署（OWWA）协作，识别受中东冲突影响被遣返的海外菲劳，并推动职缺媒合、转介技术教育与技能发展署（TESDA）的培训机会及生计扶助。作为国家再融入网络的一部分，DOLE将参与下月的“Bayanihan para sa Balikbayang Manggagawa”活动，提供遣返劳工及其家庭一站式再融入服务。 
DOLE并持续在全国举办月度职场博览，自2026年1月以来已举办144场，涵盖高中应届、替代教育体系毕业生及被迫停工的建筑工人等群体；并将召集国家三方产业和平委员会，拟定更多应对策略与行动方案。 
总统马科斯本周宣布国家能源紧急状态，称中东战事已对国内能源供应的可得性与稳定性构成「迫在眉睫的危险」。警方称，截至周五已有425座加油站因燃油耗尽暂时停业；油价上涨部分归因于霍尔木兹海峡关闭——全球约20%的石油经此通行。

埃及政府宣布，自2026年3月28日起实行临时节能措施，要求商店、商场、餐馆和咖啡馆提前营业结束，周一至周日统一在晚上21:00关门，周四和周五延长至22:00。该措施初步为期一个月，届时将评估对能源消耗和市场活动的影响，再决定是否延长或取消。
政府表示，这是一个可随经济与用能情况调整的灵活方案。内阁总理穆斯塔法·马德布利称，目的是在保持经济活动的同时降低能源使用，措施将逐步实施以避免对市场造成突发影响。
多类关键行业获豁免，确保基本服务不受影响，包括医院、药房、医疗中心，以及超市、杂货店、面包房和果蔬店；工业厂区和工厂维持原有作业时间，燃料站、水务和燃气等必需服务亦不受限制。
违规者将根据2019年第154号《商店法》面临最高2万埃镑罚款，并可能被行政查封或撤销执照；依据紧急法规，还可处300至4000埃镑罚款，严重者甚至面临刑事处罚。
配套措施还包括研究部分政府机构每周一至两天远程办公（不含工业与必要服务）、将新行政首都政府区晚6点关闭、将路灯降至最低安全照明并关闭道路广告灯箱。

阿布扎比当局称，位于哈利法经济区（KEZAD）附近接连发生三起由被拦截弹道导弹残骸引发的火灾。空防成功拦截导弹后，落下的碎片引燃多处火点。最初确认5名印度籍人员受轻至中度伤，随后又有一名巴基斯坦籍人员受伤，受伤总数增至6人。

当局此前连发多次安全警报，称已拦截来自伊朗方向发射的导弹与无人机。相关警报分别于凌晨3:26、4:05、6:24和下午16:25发布。国防部确认，各地听到的响声来自空防拦截弹道导弹，战机同时对巡航导弹和无人机实施拦截。官员提醒民众不要接近或搬动残骸，遇有相关情况及时向应急部门报告，局势正被密切监控，后续将持续通报。
应急期间注意事项：
• 以安全为先，不要在警报时拍摄或传播视频，听从指示；
• 驾驶人员安全驶离并在抵达后寻求掩护，切勿在路上停留；
• 留在安全场所或室内，直至官方发布“解除警报”；
• 仅在真实紧急情况拨打999，保持热线畅通；
• 仅转发官方核实的信息，避免传播谣言。
