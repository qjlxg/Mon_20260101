中国多家航司和线上平台被约谈 限期自查整改机票超售问题
中国北京、天津和河北三地消协组织星期五（7月24日）约谈10家航空公司和五家线上售票平台，要求这些企业限期完成机票超售问题自查整改。
据中国央视新闻报道，7月初京津冀三地消协组织联合开展民航机票超售认知专项调查，并针对调查中发现的问题，对这些企业开展联合约谈，要求上述它们在8月31日前完成自查整改。
这10家航空公司为中国国际航空、中国南方航空、中国东方航空、海南航空、厦门航空、天津航空、四川航空、吉祥航空、春秋航空、河北航空。五家线上售票平台为同程、携程、飞猪、去哪儿、美团。
在约谈前的沟通环节中，春秋航空的服务热线无法联系上人工客服，消协组织发函后也未收到任何回复。京津冀三地消协组织对此予以公开批评，要求春秋航空公司正视自身在服务响应和消费者权益保障方面存在的不足，主动与消协组织对接沟通，按要求提交整改材料，确保整改工作落到实处。
除了督促整改落实，三地消协组织也依据向上述航空公司和线上售票平台提出整改建议，包括完善购票事前告知，保障旅客知情权。线上购票渠道须采取醒目独立的提醒方式，完整公示超售规则、现场处置流程及分级现金补偿标准。不能以小字、折叠隐藏、浅灰色淡化等方式弱化提示。
其他建议包括优化现场处置流程，尊重旅客选择权。处置过程中应优先征集自愿放弃座位的旅客，不可未经协商即强制拒绝旅客登机。
因机票超售导致旅客无法正常登机的，航空公司应提供多种解决方案供旅客自主选择，包括全额免费退票或免费改签至当日最早可用航班等，不得收取手续费及差价，不得强制安排凌晨或深夜航班。
在现金补偿方面，航空公司应制定并公开阶梯式现金补偿标准，旅客可自主选择是否现金赔付，不能以里程或代金券强制替代。旅客当日无法成行的，航空公司应免费提供住宿、餐饮服务及机场往返接送。
京津冀三地消协组织要求各相关企业，根据上述建议逐项落实整改，明确整改措施、责任部门和完成时限。对约谈后整改不佳或持续侵害消费者权益的企业，组织将采取进一步措施。
