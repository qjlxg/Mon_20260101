美电动车企里维安拟自研激光雷达 或与中企合作
美国电动汽车制造商里维安正考虑自行生产激光雷达传感器，并可能与中国企业合作。
据路透社报道，里维安（Rivian Automotive）首席执行官斯卡林奇（RJ Scaringe）星期二（5月5日）在旧金山受访时透露上述消息，并称计划在今年晚些时候推出的R2车型版本将配备激光雷达传感器。这类传感器可协助自动驾驶车辆获取道路三维视图。
里维安并未公布具体供应商，但展示车辆所使用的激光雷达体积明显小于字母公司旗下自动驾驶部门Waymo自动驾驶出租车在旧金山等地所使用的大型旋转式设备。
目前，禾赛科技和速腾聚创等中国供应商，已在体积更小、成本更低的激光雷达市场占据主导地位。不过，这一趋势也引发美国部分立法者对供应链安全的担忧。
斯卡林奇说，与其直接从中国供应商采购，里维安正考虑在美国本土利用中国技术自行生产激光雷达，甚至通过合资形式实现本地化制造。他指出，在汽车制造商所需的“数百美元价位”传感器市场，“几乎所有真正的选择都来自中国”。
他还透露，公司正与多家激光雷达企业进行密集磋商，并认为这一方向可能吸引其他车企共同参与。“不少车企都在思考，是否可以联合推进，或至少形成某种协同，共同在美国，或至少在中国以外地区，建立相关产能。”
此外，里维安也在推进自研晶片战略，以提升自动驾驶系统竞争力，对标特斯拉等竞争对手。公司去年已投入数亿美元开发定制晶片项目，首款自动驾驶处理器RAP-1预计将在今年推出。

中国最高法明确“开门杀”等交通事故赔偿责任
对于“开门杀”等交通事故，中国最高人民法院发布司法解释，明确责任主体、赔偿计算等问题。
据新华社客户端报道，最高法星期三（5月6日）发布《最高人民法院关于审理道路交通事故损害赔偿案件适用法律若干问题的解释（二）》，共12条，从责任主体、责任认定、赔偿计算、程序规定等方面作出规定，自今年6月30日起施行。
司法解释对于机动车租赁、借用等情形下的责任承担，明确规定被侵权人（即受害人）一并请求使用人与所有人、管理人承担责任的，由使用人承担侵权人应承担的全部责任；所有人、管理人对损害的发生有过错的，在其过错范围内与机动车使用人共同承担赔偿责任。
日常生活中，车内人员疏于观察，贸然打开车门与他人发生碰撞造成交通事故，俗称“开门杀”。
关于“开门杀”事故，司法解释明确，被侵权人主张乘车人责任属于该机动车一方责任，并请求保险公司在交强险责任限额范围内以及按照商业三者险合同的约定赔偿的，人民法院应予支持。
对于非营运机动车无偿搭载他人，发生交通事故造成搭乘人损害的，民法典规定，在机动车使用人没有故意或者重大过失的情形下，应减轻机动车使用人的赔偿责任。司法解释明确，“好意同乘”情形下，人民法院应当综合公安交管部门作出的认定、事故形成原因、机动车使用人的具体行为等，判断机动车使用人是否构成“故意或者重大过失”。

贝森特指出，中国购买伊朗约90%的能源，“因此在资助最大的国家级恐怖主义赞助者”，并呼吁中国“加入我们的国际行动”以开放海峡，但未具体说明北京应采取何种措施。他还要求中俄不要在联合国阻挠相关倡议，如鼓励保护霍尔木兹海峡商船的决议。
贝森特表示，川普与习近平已就伊朗局势交换意见，5月14至15日峰会将继续面对面磋商，同时强调双方将努力维持自2025年10月釜山达成贸易休战以来的美中关系稳定。他称美国通过对伊朗航运的封锁“完全控制”霍尔木兹海峡，并表示美军新行动将引导船只通行、压低油价，认为当前高油价是“暂时异常”，数周或数月内将回落。
（路透）

伊朗外交部发言人巴盖伊（Esmaeil Baghaei）5月4日在国营电视的简报中表示，美方必须放弃对伊斯兰共和国的“过高要求”，目前首要任务是结束战争。他说：“对方应采取合理态度，放弃对伊朗的过分要求。”
自4月8日停火生效以来，双方谈判陷入僵局，迄今仅进行过一轮直接和谈。争议焦点之一是，自2月28日美以发动袭击以来，伊朗对霍尔木兹海峡的控制导致石油、天然气和化肥等重要货物流受阻；与此同时，美国对伊朗港口实施封锁。
美国总统特朗普5月3日表示将开始对通过该海峡的船只进行护航，此举招致伊朗军方的攻击威胁。巴盖伊警告称：“美国人应当明白，不能用威胁和武力对付伊朗民族。”他并称，伊朗视自己为霍尔木兹海峡及该重要航道的守护者。
巴盖伊还指出，战前该海峡是国际航运的安全通道，并呼吁国际社会追究美国及以色列对该水道不安全负责的责任。他表示，德黑兰已向巴基斯坦斡旋方提交一份以“结束战争”为重点的14点建议书，华盛顿已通过巴基斯坦方面作出回应。消息来源：法新社

布鲁塞尔消息——特朗普在德国裁军的决定，再次暴露出说服美国继续留在北约的努力存在边界。5月1日宣布从德国撤走5000名美军、并放弃在德部署远程“战斧”导弹的实质并不令人完全意外，但此次决定的做法更令欧洲震惊——事前几乎没有通报或磋商，美方还将此与德国总理梅尔茨对美伊战争的批评联系起来。社民党资深议员西姆特耶·默勒称，令人担忧的不是数字，而是华盛顿发出的政治信号：长期可靠的伙伴关系似乎被任意决策取代。
特朗普曾指责盟友对美在伊朗战争中的支持不足，甚至暗示美国或不再必须遵守北约第五条集体防御条款，并一度以掠夺格陵兰岛威胁丹麦，所幸荷兰首相吕特出面缓和。外交人士担心，在7月安卡拉峰会前，如果伊朗战争尚未结束，特朗普可能再次采取考验盟友的举动。“北约和欧洲盟友的长远棋局是熬过安卡拉，”一名欧洲外交官表示。
欧洲方面不得不尽力留住美国：多国在伊朗战事期间允许美军使用境内基地或领空（西班牙例外），包括英国、克罗地亚、法国、德国、希腊、意大利、黑山、葡萄牙和罗马尼亚等国。官员们还在向美方、国会议员及保守智库论证：坚持北约符合美国利益，并推动战后霍尔木兹海峡可能的护航任务。荷兰、比利时与德国表示将派舰艇参与，法国与英国主导相关方案。吕特称，欧洲已收到美国明确信息，正为可能的后方与后战争行动做预置。  REUTERS

霍尔木兹海峡航运几乎停滞，尽管美国总统特朗普日前承诺将推动恢复通航。4日并未出现通航回升迹象，MarineTraffic数据显示，进入阿曼湾的仅有一艘受制裁的小型液化石油气油轮、几艘货船和一艘敷设电缆船。未见油轮排队等候过航，德国航运集团赫伯罗特（Hapag-Lloyd）表示，由于缺乏关于安全通行的明确程序，其船舶仍无法通行。 
美国中央司令部（CENTCOM）称将于4日开始协助恢复海峡通航，同时继续对伊朗港口实施封锁。航运业尚未收到关于美方行动意图和指引的说明，国际海运经纪与顾问协会（BIMCO）警告称，若未获伊朗同意，尚不清楚是否能削弱或压制对商船的威胁；BIMCO为业界提供安全警示。国际海事组织表示，数百艘商船和最多2万名海员因伊朗冲突无法通行该水道。 
美方主导的联合海事情报中心将海峡海上安全威胁评为“危急”，建议船舶考虑绕行阿曼领海（位于交通分离带以南）。CENTCOM将其行动描述为“防御性”，并称会将外交努力与军事协调相结合。伊朗则警告美军远离海峡，要求商船与其军方协调通行，并公布新控制区地图。 
巴基斯坦表示，上月被美军登船扣押的伊朗籍集装箱船Touska的22名船员已撤离至巴基斯坦并将遣返，船舶经修复后返还船东，巴方称此举为“增进互信”的措施。自4月13日以来美方在伊朗港口实施的海上封锁亦已导致德黑兰的石油出口收缩。路透社

海湾再度濒临全面战争：伊朗向阿联酋石油设施发射巡航导弹与无人机后，美国总统特朗普威胁若伊朗扩大对美舰的攻击，就要将其“从地球上抹去”。
这一轮冲突打破了自4月8日开始的脆弱停火。霍尔木兹海峡的封锁已成难以维持的僵局，双方可能在数日内被迫让步或重返高强度对抗。
事态源于二月底美以对伊空袭后，伊朗封锁这一经由全球五分之一石油与液化天然气的国际航道；四月美国则对伊港口与海上交通实施报复性封锁。双方均试图以经济脆弱性为筹码，伊朗寄望能源短缺迫使美军撤退，美国则欲切断伊朗油资使其崩溃。
外交上亦已分歧：伊朗仅愿就海峡问题协商、拒谈核问题，美国视此为不可接受且令谈判破裂。经济压力正迅速累积——据估计仅四月失油收入达48亿美元，更严重的是储油设施紧缺，可能在五月中旬耗尽，迫使弃井停产并造成长期损害；伊朗通胀约70%，里亚尔暴跌。
鉴于此，双方均认为需以军事行动打破僵局。特朗普提出的“自由行动计划”（Project Freedom）欲为海湾滞留船只提供“引导”，意在以军力撬开伊朗对海峡的控制；伊朗则通过袭击阿联酋富查伊拉石油工业区向周边施压，传达“我不能出口，别人也别想出口”的信息。
若短期内无重大外交突破，冲突有回到全面战争的危险，阿联酋等海湾邻国也不太可能坐视不理。
（作者：乔纳森·艾亚尔，伦敦/布鲁塞尔）

世界卫生组织（WHO）5日表示，怀疑豪华邮轮MV Hondius上发生少数人际传播病例，受感染者多为密切接触者。该船已有七例确诊或疑似汉坦病毒病例：一对荷兰夫妇和一名德国乘客已死亡，一名英国乘客已被撤离并在南非重症监护。船上仍有三名疑似病例，其中一人仅有轻微发热。
该船被困在佛得角海域，不获准让乘客上岸。WHO称，当前重点是将船上两名病人撤回荷兰，随后让邮轮驶往加那利群岛；但西班牙卫生部尚未决定是否接收。佛得角正与相关方商讨撤离方案。
WHO重申汉坦病毒主要通过感染啮齿动物及其排泄物传播，人际传播罕见但安第斯病毒株在极少数暴露密切的接触者（如同房夫妻）间有有限传播记录。WHO流行病学负责人玛丽亚·范·凯霍夫表示，初步判断荷兰夫妇可能在阿根廷上船前已被感染，其他病例或在观鸟登陆岛屿时暴露。
邮轮自3月自阿根廷最南端乌斯怀亚出发，约150名乘客多为英国、美国和西班牙人，行程曾到访南极半岛、南乔治亚和崔斯坦达库尼亚等偏远地区。南非已确认被撤离的英国患者汉坦病毒呈阳性，荷兰亦证实一名死亡荷兰女乘客感染。南非传染病研究所与塞内加尔巴斯德研究所正进行病毒测序和确证工作，结果预计数日内公布。
WHO估计全球每年汉坦病毒病例在1万至10万例之间，并指出阿根廷在美洲病例最多，安第斯毒株致死率约32%，高于其他株。

英国地方选举将于5月7日举行，民调显示执政工党面临沉重失利，可能加剧对首相基尔·斯塔默的党内不满，并推动左右民粹势力抬头。此次投票是斯塔默自2024年7月大选大胜以来的最大一次选民检验，涉及苏格兰、英格兰和威尔士数百万选民。民调普遍预测对工党不利，可能引发领导权质疑，但党内尚无明显接班人。
极右的改革党（Reform UK）与左翼绿党预计将有所斩获，选民对建制两党流失支持。政治学者蒂姆·贝尔指出，这既与政府上任初期作出若干不受欢迎决定和领导人沟通失效有关，也反映选民忠诚度下降。
民调显示工党可能在威尔士失去执政权，若成真将是威尔士自27年前设立议会以来首次失守，卡迪夫大学政治学者劳拉·麦卡利斯特称其后果“灾难性”。在苏格兰，民族党（SNP）预计将延续掌权，民调甚至显示改革党可能挤下工党位居第二。
英格兰方面，民调员罗伯特·海沃德预计工党约失去1,850个其防守的地方议席，全国约有5,000席位竞选；改革党或从工党与保守党手中夺得约1,550席，绿党则可能在伦敦等地凭借亲加沙立场获得数百席位。
斯塔默自结束保守党14年执政以来屡遭政策争议，近期又陷入被解职驻美大使彼得·曼德尔森相关丑闻，经济提振乏力，但在对美伊问题上有抵抗美国的正面评价。媒体有报道称若选举失利，部分前内阁成员或会推动罢免，但任何挑战者需获得20%议员提名；政府部长史蒂夫·里德斥该类说法“无稽”。
