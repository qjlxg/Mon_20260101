【国家药监局：暂停一款印度原料药进口】金十数据4月3日讯，今天从国家药监局获悉：近期，国家药监局组织对印度一家企业（名为Cadila Pharmaceuticals Limited）的苯磺酸氨氯地平原料药开展现场检查，经查，上述产品部分工序实际生产地址与注册信息不一致，生产过程中还存在多项变更未按要求及时申报、实际生产工艺与批准工艺不一致等情况，不符合我国《药品生产质量管理规范（2010年修订）》及附录有关要求。根据《中华人民共和国药品管理法》第九十九条、《药品医疗器械境外检查管理规定》第三十条等有关规定，国家药监局决定，自即日起：一、暂停进口上述原料药，各口岸药品监督管理部门暂停发放该原料药产品的进口通关单。二、上述原料药不得在境内销售、不得用于药品制剂生产，对已使用上述原料药生产的制剂不得放行；已上市放行的制剂，药品上市许可持有人应当立即开展调查与评估，并根据评估结果采取必要的风险控制措施。

中国拟到2029年推进石化装置升级改造 淘汰落后产能
中国计划到2029年对部分老旧石化装置实施更新改造，并逐步淘汰落后产能。
中国工业和信息化部等七部门星期五（4月3日）印发《加力推进石化化工行业老旧装置更新改造行动方案（2026—2029年）》，提出加快推进老旧装置改造，提升本质安全水平，促进行业智能化、绿色化、融合化优化升级。

文件未披露具体涉及升级或淘汰的装置数量。

副总理兼财政部长เอกนิติ下令重整油价结构，削减炼厂与营销费用，拟于4月6日提请内阁审议
2日，副总理เอกนิติ在主持首场“燃油成本定价适当性研究委员会”会议后表示，已邀请炼油厂代表说明数据，要求能源部尽快检视并调整批发油价的计价结构，特别是“炼制费”和“营销费”，以反映真实成本并减轻民众生活负担，准备在4月6日的内阁会议上提出。
会议认定，当前计价公式将运输费与保险费并入批发价，可能导致成本被高估，已指示能源部剔除相关项目，预计可立即压低炼制费。同时要求审查所谓的战争风险溢价（War Premium），鉴于泰国炼厂并非仅从中东单一来源进口，应核实真实采购成本，避免将不当成本转嫁给消费者。
会议要求炼厂于4月3日前提交资料，委员会将在4月6日之前完成讨论并向内阁汇报，预期调整将直接降低油价并缓解民众负担。
能源部常务次长普拉瑟回应称，目前营销费自1月1日至4月2日平均约1.95铢/升，低于先前研究提出的2.45铢/升上限；而表面上达到13–14铢的炼制费实为原油与成品油价差，并非全部为纯利，其中包含原料成本与战争期间的溢价。按过去五年正常情形，炼制费平均约2.45铢/升，与营销费相近。

菲律宾劳工部（DMW）提醒在阿联酋及中东地区的海外菲律宾工人（OFWs）网络谨慎，警告转发未经证实的信息可能带来严重后果。DMW呼吁“点开前先三思”，指出在地法律严格，即便出于善意，转发虚假或未经核实的新闻、照片或视频也可能构成违法行为。
