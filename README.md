深圳出台提示规范企业个人黄金业务
中国深圳市十部门发提示，规范企业和个人黄金业务，包括要求企业不得违规开展黄金预定价交易，个人不得开发、兜售非法黄金交易软件等。
据深圳市地方金融管理局官网消息，深圳市地方金融管理局、中国人民银行深圳市分行、深圳市市场监督管理局、中国国家金融监督管理总局深圳监管局、中国证券监督管理委员会深圳监管局、深圳市互联网信息办公室、深圳市公安局、深圳市商务局、深圳市工业和信息化局和深圳市通信管理局10个部门星期四（2月12日）就企业、个人、金融机构及非银行支付机构从事黄金业务禁止性行为联合发布公开提示。
此举是为进一步规范深圳市黄金市场经营秩序，防范化解市场风险，保护消费者合法权益，促进黄金市场健康发展。
针对企业黄金业务，提示列出，企业不得违规开展黄金预定价交易、杠杆交易、延期交易等非法黄金交易活动；不得违规开展以黄金托管、租赁、回购等为名义约定给予客户固定收益回报“保本付息”的非法集资活动；不得利用互联网直播等渠道违规宣传、开展黄金产品销售业务；不得违规开展黄金委托投资等活动等。
对于个人黄金业务，提示表明，个人不得组织或参与非法黄金预定价交易、以黄金为名义的非法集资、非法黄金委托投资等各类非法交易活动；不得开发、兜售非法黄金交易软件、APP或小程序，不得为相关非法黄金交易软件、APP或小程序提供信息服务和技术支持，不通过各类软件、APP及小程序参与非法黄金投资行为。
至于金融机构及非银行支付机构黄金业务，提示指出，金融机构未按规定向金融监管部门备案或获得相应审批，不得开展黄金业务，不得违反大额交易报告制度及可疑交易报告制度，发生异常情况不得迟报、漏报、瞒报；金融机构、非银行支付机构不得为未经依法设立或者从事非法经营活动的商户提供服务，不得开展非法黄金活动宣传，不得为非法黄金活动宣传提供便利。

综合台湾《联合报》《自由时报》等报道，台海委会海巡署舰队分署星期五（2月13日）在中信造船高鼎厂区举行第11艘600吨级巡防舰CG613兰屿舰交船典礼。海委会为了彰显对原民文化的认同、呼应向海致敬的政策，将船取名为兰屿巡防舰，未来将在台湾东部水域执勤。
海委会副主委张忠龙在交船仪式上致词时说，海巡署兰屿巡防舰将带着守护美丽家园的心巡航在台湾周边海域，同时也具体呈现海巡同仁专业的航海技术与执法的决心。
张忠龙强调，政府将全力支持筹建海巡舰艇前瞻发展计划及国安韧性特别预算，打造科技化、立体化的海巡舰队。随着新舰陆续服役，海巡署将以更强韧的编装，展现海上不缺席的执法决心。
兰屿巡防舰是海巡安平级巡防舰系列的第11艘，舰体全长约65公尺、宽约14.8公尺，最高航速可达40节，续航力达2000海里（3704公里），能执行长时间海上任务。舰上配备高压水炮与警备艇，具备消防救援、海上执法及救生救难等多元勤务能力，在应对大陆灰色地带袭扰方面，具有速度优势。
