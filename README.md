陆委会咨委：大陆将台海“内水化”企图影响台地方选举
台湾政府的大陆委员会咨询委员认为，中国大陆透过常态化执法将台海“内水化”，展现对台具管辖权，并可能提升对台灰带侵扰力度，企图影响台湾底的地方选举。
台陆委会星期一（3月23日）在官网发布新闻稿介绍，该会日前举行咨询委员会议，讨论对中国大陆“两会”的观察，学者与咨委参与讨论。
学者指出，今年大陆“两会”整体政策方向未见结构性改变，大陆政府工作报告虽将经济增长目标下修到4.5至5%，创1991年以来最低水准，但仍继续采取更积极的财政政策、适度宽松的货币政策，高赤字率及特别国债支出成为新常态。
学者称，有党政官员持续高举“习核心”领导地位；“十五五”（2026年至2030年）规划纲要重申发展新质生产力、科技自立自强、提振内需消费、绿色转型、防风险促安全等政策目标，在短期内不会有重大调整。对台政策提及打击“台独”、推动同等待遇、弘扬中华文化等，也为其一贯立场，未见明显变化。
部分咨委认为，今年“两会”政策基调由先前的积极扩张转趋保守防御，凸显将迈入第四任期的中共总书记习近平更重视经济韧性、政治安全、军队忠诚。此一转变可能反映中国大陆政经社军各领域均面临重大转型，也可能是因国际局势变化，如美以袭击伊朗等，强化对美斗争准备。
有咨委观察到，中国大陆当前面临多重结构性困境，包括经济成长动能不足、财政压力加剧、消费疲弱、产能过剩、“内卷”竞争及人口老化等问题，若加强发展新质生产力，恐影响传统产业，衍生失业问题。
部分咨委指出，习近平近期高度关注军队整顿与忠诚问题，警告解放军不能“怀有二心”，重申政治建军，显示对军队的要求将以政治忠诚为首要，强调“红大于专”。在近期大规模肃清后，二十届五中全会甄补情形值予关注。大陆2026年军费增长7%，高于GDP增长目标，反映其将持续推动军队现代化建设。
有咨委提醒，“两会”通过《民族团结进步促进法》，将裁罚对象由过去的“作为犯”扩大至“不作为犯”；若被认定不积极促进民族团结都可能被推定为“不作为”，且鼓励民众相互举报并允许地方自订法规，对艺文、新闻工作者等的影响不容小觑。
部分咨委分析，当前大陆对台工作重点包括“反独促统”法制化、推动两岸共同市场、争取台青认同。

全球贸易局势承压 澳洲欧盟敲定贸易协议

路透社报道，在美国特朗普政府向全球各国大幅提高关税后，澳洲和欧盟加紧了贸易谈判，最终达成这项协议。

她也说，取消从澳洲进口关键矿产的关税，对于摆脱对中国的依赖、实现供应链多元化至关重要。
协议将取消欧盟出口至澳洲商品99%以上的关税，每年为企业削减10亿欧元（约14亿8000万新元）的关税。预计未来十年，欧盟对澳州的出口将增长高达33%。
澳洲总理阿尔巴尼斯在新闻发布会说，澳欧协议每年将为澳洲经济带来约100亿澳元，他也说，欧盟取消对几乎所有澳洲关键矿产的进口关税将有助于稳定全球供应链。

人民网评姚晨官宣离婚：不该借用官宣这一公共化权威性词汇
中国著名女演员姚晨上周官宣离婚，相关话题一度登上微博热搜榜。中国官媒人民网星期一（3月23日）发布评论文章指出，离婚纯属私事，不属于公众知情权范围，没有义务在公共社交平台公开，并指不该借用“官宣”这一公共化、权威性词汇，“让‘官宣’回归专属权威发布的本真含义”。
人民网星期一发布题为《人民网三评“所谓官宣”之一：自爆私事，何来“无意”！》的评论文章。文章首先写道，近日姚晨与曹郁被传离婚的消息引发网络热议，相关话题迅速登顶热搜。文章接着指出，放眼当下的网络舆论场，这类明星私事霸榜的场景屡见不鲜。“官宣”二字，更是成为明星、名人、网红们频繁使用的热词。
文章称，这些“官宣”看似是真实信息发布，本质上却是不折不扣的“伪事件”，并解释“伪事件”是指并非自然发生的社会事实，而是人为建构、专为传播与围观而制造的媒介现象。
文章进一步称，“官宣”这一党政机关、国家部门发布重要政务信息、权威事项的专属公文用语，自带严肃、权威、正式的属性。如今却被肆意套用在明星婚恋、网红私生活、商业代言等私人领域，被当成了部分群体放大私人声量、收割流量红利的“密码”。
文章写道：“原本庄重的公共话语，在滥用中逐渐失去了应有的严肃性。”
文章指出，必须看到，愈演愈烈的“官宣”闹剧，不仅消解了公共话语的严肃性，更严重扰乱了公共空间的信息传播秩序。
文章以姚晨“官宣”离婚一事为例称，离婚纯属私事，不属于公众知情权范围，没有义务在公共社交平台公开，更不该借用“官宣”这一公共化、权威性词汇。文章批评，刻意公开后引发的热搜霸榜、舆论狂欢，更是让公众被动裹挟进流量洪流，既浪费了公共资源，也破坏了正常的信息传播生态。
文章呼吁让“官宣”回归专属权威发布的本真含义；让公共舆论空间摆脱流量绑架，重回清朗有序的状态；让真正有价值的公共信息，能够被公众看见、关注。
文章最后写道，这需要全社会形成共识，共同发力。“是时候整治这股‘所谓官宣’的乱象了！”
姚晨上星期一（3月16日）和摄影师曹郁在微博发布联合声明宣布离婚：“多年前，我们结束了婚姻关系，考虑到当时孩子年幼，未做对外公布。”相关话题当天登上微博热搜榜，截至星期二（3月24日）下午12时16分，阅读量有3亿次。

台湾官方证实，“百亿赌王”林秉文弃保潜逃，在柬埔寨被枪杀。

刑事局表示，该局驻胡志明市联络官目前已与外交部驻胡志明市办事处建立联系窗口，共同协助家属前往当地处理遗体指认及后续相关事宜。
56岁的林秉文早年混迹新北市芦洲。由于行事作风江湖味十足，因此有了“鲈鳗”的外号。
2008年，林秉文卷入中华职棒米迪亚暴龙队假球案，被控以威胁、恐吓、利诱等方式，逼迫球员打假球，被判刑两年，易科罚金73万元（新台币，下同，2万9172新元）。出狱后，林秉文投资博弈产业、获得庞大利益，因而有“百亿赌王”之称。
2021年，林秉文公开表示帮艺人澎恰恰处理高达上亿新台币的巨额债务，并透过旗下公司PGTalk协调债权人。后来，林秉文被控处理债务跳票，并与部分债权人失联，他才表示已不帮澎恰恰处理债务，双方合作关系终止。
