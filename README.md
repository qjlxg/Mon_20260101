经营者强烈反对将五大经济中心转交新公司管理
斯里兰卡达姆布拉经济中心的摊贩在与贸易、商业、粮食安全与合作发展部长瓦桑萨·萨马拉辛赫（Wasantha Samarasinghe）31日的座谈会上，明确表示强烈反对政府拟将五个主要经济中心从现有管理架构转入新成立的公司。
会上通报拟成立新管理公司、任命七名董事，并将丹布拉（Dambulla）、努瓦拉埃利亚（Nuwara Eliya）、桑布特加玛（Thambuttegama）、凯佩蒂波拉（Keppetipola）与纳拉亨皮塔（Narahenpita）五个中心移交该公司的计划。副部长苏尼尔·比扬维拉及相关政府与管理信托代表、摊贩协会成员出席。
摊贩指出，这些市场由交易者通过管理信托运营已逾30至35年，强调不能在未事先通知、未充分调查的情况下被“突袭”接管。达姆布拉中心自称29年间未领政府拨款，信托存款约2.5亿卢比，并为农户提供无息贷款、种子等支持。
部长表示，设立新公司旨在规范管理、维护公平市场，并宣布自下月起上述中心的月度税收将缴入该公司，且不会剥夺农民或交易者既有权利。但交易者反驳称已对此前提交内阁审批的公司提起诉讼，指控在内阁批准后更改公司名称、未重新取得批准即另行设立新公司。
记者质询内阁批准公司与被起诉公司名称不一致时，部长回应称公司名称已更改，愿日后就变更名称再次寻求内阁批准。鉴于摊贩强烈反对，部长表示将择期继续磋商，寻求与交易者和公司的共识。
