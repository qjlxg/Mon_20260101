中国证监会发布完善上市公司再融资规则建议，寻求社会公众意见，其中提及建立再融资定向增发储架发行制度，优化小额快速再融资制度。
据中国证监会官网消息，为增强国内资本市场竞争力、吸引力，提高资本市场制度包容性、适应性，中国证监会星期五（7月3日）发布完善上市公司再融资规则，现向社会公开征求意见，意见反馈截止时间为8月2日。
规则的主要修改内容，包括建立再融资定向增发储架发行制度，优化小额快速再融资制度，实行统一的市价发行定价机制和强化可转债监管要求等。

规则要求所有上市公司定增须以发行期首日作为定价基准日确定发行价格，推动定价市场化，并完善锁定期安排，更加体现对中小投资者的保护。
规则也明确，沪深可转债与定增、增发、配股适用相同的再融资间隔期要求，并加强对发行可转债相关偿债能力约束的要求。
今年2月，沪深北交易所推出优化再融资一揽子措施突出扶优、扶科、限劣导向，明确对优质上市公司进一步提高再融资效率，优化未盈利创新企业再融资间隔期要求。

金融监管赶不上AI发展 欧洲决策者促建防线控风险
（伦敦讯）欧洲金融业决策者指出，金融监管已难以跟上人工智能（AI）快速发展的步伐，他们在努力推动AI应用的当儿，也致力管控AI对市场诚信和稳定的潜在风险。
英国金融行为监管局首席执行官亚迪（Nikhil Rathi）星期四（7月2日）接受美国消费者新闻与商业频道（CNBC）访问时说，在AI技术尤其是代理式AI（Agentic AI）快速发展的年代，传统的监管立法周期已行不通，必须换一个不同的思考方式来应对。
他举例说，英国已成立AI安全研究所，协助决策者、监管机构和企业更深入了解AI风险，以安全地运用AI。
欧洲央行行长拉加德（Christine Lagarde）同天在法国《回声报》刊出的访谈中也点出，AI能提升生产力和增值，但也构成显著风险。
她说：“过去10年我们一直在谈论网络安全、黑客和数据盗窃等风险，但随着AI模型的加速深化发展，我们如今面临更严峻的风险，因为技术发展非常快，但防御风险的方式，以及支持防御的资金都还很稀缺。”
在欧洲央行本周于葡萄牙举行的年会上，AI对生产力和市场诚信的影响是主要讨论点之一。
英国央行副行长布里登（Sarah Breeden）在会上发表演讲时警告，代理式AI在金融市场的应用，可加倍放大市场动荡时产生的波动。
目前，交易机构主要使用自动化AI处理低风险的功能如研究，但布里登认为，情况可能很快改变。一旦代理式AI的应用更加普及，金融市场可能须要更大力道的监管，如类似于股市熔断机制的防护栏，“在AI模型出错导致市场崩盘时，能限制或终止整个市场交易”。

（悉尼综合电）联合国教科文组织（UNESCO）决定维持大堡礁的世界遗产地位，暂不将它列入濒危世界遗产名录，澳大利亚政府星期六（7月4日）对此表示欢迎。 
自2021年以来，教科文组织每年都会评估大堡礁的状况，并多次指大堡礁面临被列入濒危名录的风险。
教科文组织星期五（3日）指出，澳洲正持续努力应对气候变化、改善水质、推动可持续渔业管理及管控土地开垦，但在2024年至2025年间，大堡礁的硬珊瑚覆盖率大幅萎缩。这主要是因为海水温度持续高于平均水平，引发自2016年以来第六次大规模珊瑚白化事件。 
此外，极端天气、陆地径流、沿海开发以及棘冠海星大量繁殖，也持续对大堡礁的生态系统施加压力。
教科文组织说：“虽然大堡礁仍展现出一定的生态韧性，但它承受这类事件并从中恢复的能力正不断减弱，这令人深感担忧。”
澳洲助理旅游部长格林回应时说，教科文组织的决定肯定了澳洲在管理这些风险方面所做的大量努力，但她也坦承，气候变化在未来将继续威胁大堡礁。
澳洲海洋保护协会活动经理辛德勒则认为，澳洲在应对大堡礁面临的主要威胁方面仍存在严重不足。
她说：“大堡礁每年为澳洲经济贡献约90亿澳元（约80亿新元），是全国第五大就业支柱，支撑着7万7000个工作岗位。我们不应该等到教科文组织来提醒，才懂得必须采取更多措施来保护它。”
