具体到专业调整方面，探索建立新专业增设快速响应通道，面向新兴产业和未来产业，重点增设低空经济、人工智能、高端装备、城市更新、民生紧缺等领域新专业。

【土耳其外长：美国可能对伊朗铀浓缩的部分要求保持灵活】金十数据2月12日讯，据英国金融时报，土耳其外交部长哈坎·费丹表示，美伊双方似乎都准备在核协议上做出妥协，但警告称，如果谈判扩大至涉及伊朗弹道导弹计划，只会“带来另一场战争”。费丹表示，美国已表明愿意在伊朗停止所有铀浓缩的关键要求上保持灵活。费丹还表示，他相信伊朗“真心希望达成真正的协议”，并会接受类似2015年美伊及其他世界大国协议中规定的浓缩水平限制和严格的检查机制。“美国似乎愿意在明确设定的界限内容忍伊朗的铀浓缩，这是积极信号，”费丹说，“伊朗现在意识到需要与美国达成协议，而美国也理解伊朗有一定底线。试图强迫他们毫无意义。”他警告称，如果美国“坚持同时处理所有问题（指伊朗的弹道导弹库存及对地区武装组织的支持），我担心即使是核问题也难以推进……结果可能是该地区爆发另一场战争”。
