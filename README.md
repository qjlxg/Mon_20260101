中国进口俄罗斯液化天然气激增 需求强于预期
中国去年12月从俄罗斯进口的液化天然气（LNG）出现超预期激增，显示作为亚洲最大进口国的中国需求强于市场判断。
据彭博社报道，中国海关总署数据显示，去年12月自俄罗斯进口的LNG攀升至创纪录的190万吨，远高于彭博根据船舶追踪数据估算的85万吨；11月海关数据也比预估数据高出约90%。
这一差距表明，在经历约一年进口下滑后，中国在去年最后两个月的LNG需求超出市场低预期。造成这一现象的原因可能包括：俄罗斯使用更多“影子船队”运输受制裁天然气，以隐藏航行位置；或因货物抵达中国与海关清关时间存在差异。
中国自去年8月起开始进口被列入制裁名单的俄罗斯LNG，并以大幅低于现货价格的折扣，在南部北海接收站接收约二十多船货物。同时，运输这些燃料的“影子船队”也在加紧规避西方限制。
根据中国海关数据，中国去年共进口980万吨俄罗斯LNG，创历史新高。同期，由于需求疲软、库存高企以及管道天然气进口强劲，中国全年LNG总进口量同比下降11%。

中国商务部强调反对欧盟将部分中企列为高风险供应商
中国官方星期四（1月22日）强调，坚决反对欧盟将部分中国企业列为高风险供应商。
据中国央视新闻报道，中国商务部新闻发言人何咏前在新闻发布会上说，欧盟发布相关文件，强制要求成员国在能源、交通、ICT服务管理等18个关键行业排除所谓“高风险供应商”。中国对此表示严重关切。
何咏前指责欧盟在毫无事实依据的情况下，将部分中国企业列为高风险供应商，限制中国企业参与第五代移动通信技术（5G）建设。中国方面坚决反对欧方对中国企业的歧视行为和将经贸问题政治化、泛安全化的做法。
路透社早前报道，根据布鲁塞尔星期二（20日）公布的提案草案，欧盟计划在关键行业​​逐步淘汰来自高风险供应商的零部件和设备。有可能受此举影响的中企华为公司批评这一举措。
欧盟委员会针对修订《欧盟网络安全法》一事提出这一措施，也没有点名任何公司或国家。
不过，欧洲已经在加强针对中国技术的审查。德国任命了一个专家委员会，重新评估对华贸易政策，并禁止在未来的6G电信网络中，使用中国零部件。
美国2022年禁止审批华为及中兴通讯的新电信设备，并敦促欧洲盟友效仿这一做法。

【亚虹医药：APL-2401在澳大利亚获得开展Ⅰ期临床试验许可】金十数据1月22日讯，亚虹医药公告，子公司Asieris Pharmaceuticals (Aus) Pty Ltd 收到澳大利亚人类研究伦理委员会签发的关于批准公司开发的产品APL-2401（药物名称为ASN-8639 片）在FGFR2/3 驱动的晚期实体瘤患者中开展Ⅰ期临床试验的临床试验伦理许可，并于澳大利亚卫生部所属的Therapeutic Goods Administration（即澳大利亚药品管理局）完成临床试验备案。按照澳大利亚药品注册相关法律法规，公司已获得该产品I 期临床试验许可。
