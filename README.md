官方宣布油价上涨前 中国石化罕见提醒车主错峰加油
受中东战事影响，中国官方将在星期一午夜宣布油价上涨。此前，中国石化罕见提醒车主错峰加油。
综合快科技和新浪证券报道，按照10个工作日一调整的原则，中国国内成品油新一轮调价窗口将于星期一（3月23日）午夜12时开启。本轮油价将迎来今年内第五次上调，同时是涨幅最大的一次。
多家机构预测，92号汽油价格或将全面迈入“9元时代”。据测算，本轮成品油对应涨幅或达每吨2200元（人民币，下同，409新元）左右，折算为升价，92号汽油、95号汽油、0号每升分别上调1.73元、1.83元、1.87元。
按此计算，若以50升油箱为例，加满一箱92号汽油将多花约86.5元，95号汽油则将增加约91.5元。
值得注意的是，中国石化日前已向全国会员发送提醒短信，提示油价将于3月23日24时上调，预计涨幅较高，建议车主合理安排出行时间，提前错峰加油。
不少网民称，这是第一次收到来自的“错峰加油”提醒，汽车博主初晓敏也发文称，“印象中好像是第一次收到”。
据中国国家发改委3月9日发布，受美伊冲突持续升级影响，国际油价大幅上涨，自3月9日午夜开始，中国国内汽油与柴油零售价格每吨分别上调695元与670元。
平均来看，各型号汽油与柴油每升涨幅均超过0.5元，意味着此轮调价落地后，车主加满50升油箱，将较之前多花费近30元。
这次上调是中国国内油价2022年3月以来的最大涨幅，也是成品油零售价格今年的第四次上涨。

军事与政治分析人士警告，随着导弹射程和攻击范围扩大，冲突对地区安全与全球能源供应构成严重威胁。

伊朗向距伊朗约3,800–4,000公里的美英军事基地迪戈加西亚发射导弹未中，引发其真实射程的质疑。
多方报道显示，伊朗发射了两枚弹道导弹：一枚飞行中失效，另一枚被拦截或未达目标。若属实，此次打击意图几乎是其官方宣称约2,000公里自我限制的两倍。
迪戈加西亚为美军在印太地区的重要枢纽，支持中东、南亚和东非行动，长期承担远程轰炸机、侦察与后勤任务。针对该目标即便未命中，也释放出威胁远距打击的信号。
分析认为，这可能反映未公开的更远射程能力、现有系统的改装，或对中程/远程技术的早期测试；也可能是战略模糊的一部分，用以增加对手防御负担。
同时，伊朗高层警告称“公园、旅游地等非军事目标”也可能不再安全，显示冲突有向更广泛目标扩散的风险。对地区及美军部署而言，关键问题是：伊朗的实际打击距离是否被低估？在答案明朗之前，相关国家须在防御部署上预设更远的威胁。

罗马——意大利媒体周六报道，两名无政府主义者在罗马郊区一处简易房发生爆炸身亡，疑似在制造炸弹时发生意外。死者为萨拉·阿迪佐内（Sara Ardizzone）和亚历山德罗·梅尔科利亚诺（Alessandro Mercogliano），事发于周五。
报道称，该对情侣为无政府主义运动活跃分子，警方已对此有所了解。媒体指出，他们可能正准备一场旨在为被囚无政府主义者领袖阿尔弗雷多·科斯皮托（Alfredo Cospito）发声的抗议行动。
科斯皮托现年58岁，因一系列包裹炸弹及针对当局的袭击被判入狱20年，正被施以针对恐怖分子和黑手党等的高度限制性羁押措施，人权组织曾批评其过于严苛。三年前，科斯皮托曾为抗议这类措施发起绝食。法院将于五月对是否放宽其羁押条件作出裁定。
《共和报》称，阿迪佐内和梅尔科利亚诺筹划的行动并非以杀人为目的，而是一种“抗议行为”。据《晚邮报》报道，阿迪佐内在2025年庭审中曾表示，她与梅尔科利亚诺“既是生活伴侣也是斗争伙伴”，并为暴力抗议辩护，称被压迫者的暴力有其道德准则，而压迫者的暴力则无此约束。

韩国大田一汽车零部件厂周五发生大火，官方周六将死亡人数由11人上调至14人，另有59人受伤，约170名工人事发时在厂内。内政部负责火灾与灾害事务的官员称，未发现失踪人员。
救援初期因厂房有倒塌风险且现场储存的金属钠存在爆炸危险，消防无法立即进入搜救。目击者称曾听到爆炸声；当局亦在起火约下午1时（格林威治时间0400）后出动吊车从高处喷水扑救，浓黑浓烟可见。周六下午火势被扑灭，起火原因尚未公布。
总统李在明已到场慰问遇难者家属，并在社交平台表示政府将全力支援受害者并彻查原因、采取根本性防范措施。
韩国劳工安全纪录时有争议，官方统计显示2000至2024年间约有一万余人死于工地事故。今年9月华城一锂电池厂大火造成22人遇难，法院认定企业将利润置于安全之前，判处涉事电池厂CEO15年徒刑。
