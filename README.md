柬埔寨官方报告显示，当局去年将近5200违法的名中国大陆与台湾人驱逐出境。
《柬中时报》星期一（1月12日）引述柬埔寨移民总局发布的报告指出，2025年，共有1万3557名外籍人员因在柬埔寨从事违法犯罪活动被驱逐出境。
其中有4806名中国大陆人、383名台湾人；其余人数较多的国籍依次为3456名越南人、1476名印尼人、963名巴基斯坦、534名泰国人、345名印度人、247名缅甸人、241名孟加拉人、229名菲律宾人、222名韩国人、191名马来西亚人、40名日本人等。
数据显示，2025年被驱逐出境的外籍人员数量较2024年同期增加7705人，增幅达56.8%。
柬埔寨移民总局同时指出，自2014年至2025年11月30日，柬埔寨累计驱逐4万7134名非法外籍人员，来自112个国家和地区。
此外，报告显示，2025年柬埔寨移民总局共发现并打击外籍人员违法案件5011起，涉及9857名外籍人员。其中非法偷渡案件556起，涉及1315人；非法居留及非法就业案件465起，涉及3176人；逾期停留案件3567起，涉及3567人；刑事犯罪案件419起，涉及嫌疑人1795人。
另一方面，柬埔寨金边市政府星期一召开专题会议，部署打击电诈犯罪相关工作。西哈努克省等重点省份亦同步作出具体安排。内政部上周连续召开高层会议，指挥全国反电诈工作。
柬埔寨打击网络诈骗委员会秘书处星期天（11日）明确表示，将持续保持高压态势，对电诈犯罪持“零容忍、严打到底”态度。

特朗普伊朗关税警告或波及中国 中美贸易前景生变
美国总统特朗普宣布将对与伊朗开展贸易的国家加征关税，外界认为此举不仅针对德黑兰，也可能波及中国这个伊朗原油最大买家，进而动摇中美之间来之不易的贸易休战安排。 
根据彭博经济研究，中美去年10月达成贸易休战后，美国对中国商品的平均关税税率已从40.8%降至30.8%。若再推出新关税，或打乱这一协议，并为特朗普4月访华安排增添变数。 
目前尚不清楚特朗普会否在现有关税基础上叠加新措施，或为中国设定豁免。特朗普政府此前曾警告，提高关税或反噬美国自身利益。市场对特朗普加征关税的举措反应有限，股市和债市波动不大。

伊朗长期是中美关系中的敏感议题。华盛顿一贯支持以色列和沙特，北京则逐步靠近伊朗，通过接收近九成伊朗原油出口，成为支撑德黑兰经济的重要生命线。
尽管中国自2022年6月以来未正式进口伊朗原油，但第三方数据提供商和交易员指出，在美国制裁之下，相关原油流向依然保持韧性。这是因为中国建立一套不受西方控制的供应链，每日支撑逾100万桶原油进口。
大幅折价出售的伊朗原油对中国民营炼油行业至关重要，也是中国经济的重要燃料来源。原油抵达后通常存放于离岸储油设施。数据显示，截至去年12月下旬，离岸储量升至两年半高点，超过5000万桶。
除贸易外，伊朗在战略层面也是中国的重要伙伴。大西洋理事会“全球中国中心”高级研究员罗伯茨说：“关键问题在于，特朗普是否真的会付诸行动……如果他突然宣布中国不受影响，我也不会感到意外。谁知道接下来会发生什么？”
