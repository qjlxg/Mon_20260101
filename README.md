据中新社报道，在星期四（6月11日）举办的“2026·全球人权治理高端论坛”开幕式上，国家人权行动计划联席会议机制成员单位代表共同发布了《国家人权行动计划(2026－2030年)》。
在适当生活水准权方面，行动计划指出，持续巩固拓展脱贫攻坚成果。统筹建立常态化防止返贫致贫机制，坚持精准帮扶，完善兜底式保障，强化产业、就业等开发式帮扶，增强内生动力，分层分类帮扶欠发达地区，健全乡村振兴重点帮扶县支持政策，确保不发生规模性返贫致贫。
围绕工作权，行动计划提出，加强劳动者权益保障。完善劳动就业法规体系，完善劳动者权益保障检察公益诉讼制度，加强集体协商制度建设，提升劳动人事争议调解仲裁效能，健全劳动保障监察体制，督促企业依法落实工时和休息休假制度，有效治理就业歧视、欠薪欠保、违法裁员等。完善劳动人事争议多元处理机制。
在社会保障权方面，行动计划提出，加强家庭发展政策支持和生育友好型社会建设。深入实施育儿补贴等政策，减轻家庭养育负担。支持多子女家庭子女同校就读，住房保障和购房政策向多子女家庭倾斜。保障法律法规规定的产假、生育奖励假、陪产假、育儿假等生育假期落实到位，建立生育奖励假等假期成本合理共担机制。支持夫妻共担育儿、赡养老人责任，缩小男女两性家务劳动时间差距。
在健康权方面，行动计划提出，努力使人均预期寿命提高到80岁，新生儿死亡率降低至3‰，婴儿死亡率降低至3.5‰，5岁以下儿童死亡率降低至6‰，孕产妇死亡率降低至12/10万以下，主要健康指标进入高收入国家行列。
针对受教育权，行动计划提出，促进学前教育优质普惠发展。实施“十五五”学前教育优质普惠发展行动计划，优化学前教育资源配置，大力发展公办幼儿园，提高公办幼儿园在园幼儿占比，健全学前教育保障机制与实施免费学前教育相结合，推动学前教育高质量发展。

四年开店1万4000家 中国自助麻雀室品牌“四个朋友”爆火
随着中国年轻人对线下社交需求的持续攀升，主打“24小时无人自助”模式的麻将室品牌“四个朋友”在中国迅速扩张。短短四年间，全国门店数突破1万4000家，扩张速度超越茶饮巨头蜜雪冰城。
据《都市快报》报道，“四个朋友”发源湖南长沙德思勤商圈写字楼内的一间六包厢样板房。有别于传统麻将室繁杂的人工服务流程，“四个朋友”主打极简自助体验：从开门、选房、计费、核销与售后全部自动化。
消费者只需通过美团、抖音或微信小程序提前下单锁定时段，到店后扫码即可开门，进入完全私密的麻将室。门店内不设前台，茶水全程自助。
凭借极低的人力成本，“四个朋友”将人均消费压缩至仅15到20元人民币（约2.85至3.8新元），对追求轻量化社交、私密休闲及低成本解压的年轻群体极具吸引力。
报道称，“四个朋友”的客群往往不是重度麻将玩家，而是以下班后的同事聚会、周末的闺蜜局，以及情侣或家庭休闲为主。门店工作人员透露，每逢周末或节假日，麻将室基本爆满。
此外，相比许多知名餐饮品牌动辄数十万的启动资金，“四个朋友”的加盟门槛较低，达到一定包间数后可免服务费，还提供系统支持与选址指导。
得益于此，“四个朋友”自2022年开业以来迅速扩张。据官网数据，该品牌已在全国300余城市开了1万4000家店，仅用两年半就实现万店规模。相比之下，蜜雪冰城花了13年，正新鸡排则用了六年。
但在迅速扩张之下，“四个朋友”也面临如何管理加盟店并确保服务质量的难题，同时还需应对平台流量波动、监管措施随时可能落下等挑战。

费城 — 随着人工智能在可见规模上替代劳动力，华盛顿正紧急寻求应对方案。

华盛顿，6月9日——美国政府问责局（GAO）周二发布报告称，位于德州、由特朗普政府建成的联邦移民拘留中心“东蒙大营”（Camp East Montana）在规划与监管上存在“重大、普遍性问题”。
报告指出，拘留中心未能提交使用武力报告，未向病情严重的被拘人员按时提供药物，并因加急合同流程导致数千万纳税款浪费。该营区于2025年8月在埃尔帕索的布利斯堡军事基地兴建，ICE通过加速的军事承包程序将合同授予此前无拘留经验的小型公司Acquisition Logistics LLC。
GAO对2026年1月发生的两起死亡事件处理提出质疑：一案被裁定为他杀，拘留中心未向ICE提供使用武力或死亡报告，且证据“丢失或被销毁”；另一案为自杀，涉事人员被置于医疗留置室而非防自杀牢房，且无人看管超过15分钟。
此外，ICE医务人员在2025年12月视察时发现，医疗承包方未能为慢性病被拘者提供治疗方案——例如糖尿病和HIV患者均无治疗计划。
ICE与Acquisition Logistics未立即回应置评。该营区合同在2026年3月被突然转交给Amentum Services Inc。报告还指出，特朗普政府正寻求拘押创纪录的移民人数；据两名知情人士，截至6月初约有57,000名被拘者，较2025年上任时的约40,000人大幅上升。

伊朗表示在美方报复性空袭后袭击了约旦与巴林境内的美军基地。美方证实一架阿帕奇直升机被击落，机组获救；巴林、约旦称拦截多枚来袭导弹或空中目标，科威特亦称出动防空。美中东司令部称已打击霍尔木兹附近的伊朗防空与雷达设施，并在阿曼湾击中一艘违反禁运的油轮；伊朗总统警告不可以民生基础设施为目标，俄中呼吁降级局势。
德黑兰坚持任何停火须包含黎巴嫩停战——3月2日真主党向以色列发射火箭后黎巴嫩被卷入战事。以色列的空袭与地面行动已致逾3600人死亡，南黎巴嫩交火持续。6月10日南部袭击致12人死亡，西顿市中心遭毁伤。以色列总理内塔尼亚胡呼吁黎巴嫩民众反对真主党，泰尔全市被令撤离，居民表示“我们已收拾行李，要走了”。

美国参议院民主党人指责特朗普政府在选举安全通报上冷落国会，要求在11月中期选举前定期提供相关简报，他们表示迄今仅收到一次简报。
民主党议员在致行政部门的信中称，国会通常会就关键基础设施威胁或外国势力干预选举等问题获得多方通报，但来自联邦调查局和国土安全部下属网络安全与基础设施安全局（CISA）的类似请求被忽视。他们写道：“未采取任何有意义措施协调行政部门机构间的信息共享与事件响应，也未确保及时向选举相关方通报信息。”
信中还指出，特朗普政府去年削减了联邦调查局和国土安全部负责选举保障的关键办公室人员。民主党人并对政府重新调查已被驳回的“特朗普赢得2020年大选”说法表示担忧。——路透社
