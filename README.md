沙特王储穆罕默德·本·萨勒曼4月28日在吉达主持了海湾阿拉伯国家合作委员会（GCC）的一次协商会议，这是自两个月前各国成为伊朗战争前线以来的首次领导人面对面会晤。
一位不愿具名的海湾官员称，会议旨在就自2月28日美以对伊发动攻势后，海湾国家遭遇的数千次伊朗导弹与无人机袭击，商议应对方案。沙特官方表示，峰会讨论了“与地区和国际形势相关的议题及协调应对”。
冲突已造成六个GCC成员国的关键能源设施受损，连同与美方有关的企业、其他民用和军事设施也遭到攻击。自4月8日美伊达成停火以来袭击有所减少，但各国仍担心冲突重燃，旨在达成永久停火的美伊谈判尚未取得实质性成果。
沙特媒体称，卡塔尔埃米尔、科威特王储、巴林国王与阿联酋外长已抵达吉达参会，阿曼是否派代表尚不清楚。阿联酋高级官员安瓦尔·加加什此前批评GCC在政治与军事立场上“历史性地薄弱”，称虽有后勤支持但总体作为不足。

英国首相斯塔默的前幕僚摩根·麦克斯威恩（Morgan McSweeney）4月28日在议会委员会作证时，为曾支持任命彼得·曼德尔森出任驻华盛顿大使一事承担责任，称这是“严重的判断失误”，并因此丢掉了工作。麦克斯威恩同时否认曾指示官员无视程序或强行放行安全审查。
他的证词在与前外交部官员关于任命失误责任以及首相是否被告知相关风险的争论中，支持了斯塔默的说法。麦克斯威恩表示：“我曾建议首相支持该任命，我做错了。但我没有负责国家安全审查，也没有要求官员跳过程序或暗示必须不惜一切通过审查。”
他称已向首相说明任命的利弊，但斯塔默最终选择了他认为最适合特朗普政府的人选。此任命因曼德尔森与已故美国性犯罪者杰弗里·爱泼斯坦的已知关联而引发争议，要求斯塔默辞职的呼声随之高涨。
斯塔默已承认任命错误并表示遗憾，但称程序已被遵循，并指责官员未告知有安全审查机构曾劝阻任命。前外交部常务次官菲利普·巴顿（Philip Barton）称，任命时间存在“紧迫感”，目的是在或接近唐纳德·特朗普于2025年1月20日就职时完成交接，且他并未被就该政治决定征询意见，斯塔默办公室对审查过程“不感兴趣”。
曼德尔森于2024年底被任命，斯塔默当时称其“经验无与伦比”。今年9月斯塔默解除其职务，原因是电子邮件暴露了曼德尔森与爱泼斯坦的密切关系。英国警方于今年2月以涉嫌玩忽职守逮捕曼德尔森，但未对其提出指控，且未涉及性行为指控。
另一名前外交部高官奥利·罗宾斯（Olly Robbins）也称曾遭遇“持续压力”加快任命，斯塔默否认相关指控。此事促使外界要求斯塔默就误导议会负责，但议会在4月28日晚间可能会否决就此事对斯塔默进行调查的动议。 路透社
