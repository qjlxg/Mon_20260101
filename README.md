【上海发布楼市“沪七条”】为更好满足居民刚性和改善性住房需求，促进房地产市场平稳健康发展，2月25日，市住房城乡建设管理委、市房屋管理局、市财政局、市税务局、市公积金管理中心等五部门联合印发《关于进一步优化调整本市房地产政策的通知》（以下简称《通知》），自2026年2月26日起施行。
一是缩短非沪籍居民购买外环内住房所需缴纳社保或个税年限。对非本市户籍居民家庭或成年单身人士购买外环内住房的，购房所需缴纳社会保险或个人所得税的年限，调整为购房之日前连续缴纳满1年及以上。
二是符合条件非沪籍居民可在外环内增购1套住房。对在本市缴纳社会保险或个人所得税满3年及以上的非本市户籍居民家庭或成年单身人士，在执行现有住房 
#限购
 政策基础上，可在外环内增购1套住房。
三是符合条件的持《上海市居住证》群体可在本市购买住房。对持《上海市居住证》满5年及以上的非沪籍居民家庭或成年单身人士，在本市限购1套住房，无需提供缴纳社保或个税证明。
按照上述规定，非沪籍居民家庭或成年单身人士，自购房之日前在本市连续缴纳社会保险或个人所得税满1年及以上的，在外环外购买住房不限套数，在外环内限购1套住房；连续缴纳社会保险或个人所得税满3年及以上的，在外环内限购2套住房。持《上海市居住证》满5年及以上的，在全市范围内限购1套住房。

海南三亚一在职教师报考其他单位被开除 引教师不能跳槽争议
海南三亚一名民办中学教师因报名参加其他单位的招聘考试，被学校以“严重违纪”为由开除，引发网民关于教师是否不能跳槽的争议。三亚教育局回应称，已介入关注。
据央视新闻，三亚市教育局星期三（2月25日）发布情况通报称，近日有媒体报道三亚市某民办学校，一在职教师未经学校批准报考其他单位被解除劳动关系一事引发关注。
通报称，经核查，目前该争议已经进入劳动争议仲裁程序，事件正在进一步审理中。三亚市教育局表示将全程持续关注事件进展，依法依规督导涉事双方在法律法规框架内妥善解决，切实保障双方合法权益。
据《现代快报》，通报中提到的某民办学校为三亚中学。被开除的教师因“在2025年9月至10月期间，未经学校批准报名了其他单位的招聘并参加考试”，被校方认定违反学校《教职工管理规定》，属于严重违纪行为，决定予以开除处分。
三亚教育局工作人员对《现代快报》表示，该教师于2025年7月入职三亚中学，聘用合同期限为3年，被校方开除后，对方已经申请劳动仲裁。
据悉，该教师和三亚中学的劳动已写明，合同期限内未经单位允许，不能去报考其他单位。三亚教育局工作人员表示，关于这样的约定合理与否，要等劳动部门去裁决。
工作人员也表示，三亚中学属于民办学校，由于教师流动性比较大，校方为了方便管理，才有这样的规定。
不少网民批评这不是教师合同，而是“卖身契”，一般企业都能“骑驴找马”，但民办学校不属于体制内，也要遵守这么严格的规定，并不公平。

海牙——在国际刑事法院就指控前菲律宾总统杜特尔特犯有危害人类罪进行的确认起诉听证会第2日，检方与受害人律师团呈现证据，称其为一场有组织、全国范围的杀戮行动，并强调受害家庭的人道代价。
检方在三场听证中围绕危害人类罪的背景要件、所谓“大叻模式”被复制到全国的指控及受害者家属证词展开。受害人公诉律师办公室（OPCV）敦促法官确认起诉，称证据显示对弱势社区的系统性侵害。
检方律师埃德华·杰里米称，杜特尔特就任总统后把达沃时期的做法扩展并复制至全国，任命当时的同谋担任要职，按照相同计划和手法行动。内部证人称，“neutralize（中和）”在实务上即意味着“杀死”，并把名单称为“死亡名单”。
检方举例，莱特省阿尔布埃拉市长罗兰多·埃斯皮诺萨2016年投案后仍在监狱被杀，事后监控被关闭、枪械被栽赃、悬赏支付。另指出行动偏向贫困社区，所谓“Tokhang”对象多为无力申诉者。
检方列举作案模式：通过观察名单确定目标、行动中杀害嫌疑人、栽赃武器与毒品、并以“开枪反抗（nanlaban）”为案情叙述。援引案例包括2016年被多次枪击的本杰明·维斯塔、被监控录像推翻警方自辩的17岁少年基安·德洛斯·桑托斯，以及2018年疑似被绑缚勒杀的未成年案件。
检方并引用数据与官方数字：独立组织记录至少5,823宗反毒杀戮，政府2017年报告承认3,967死，菲律宾禁毒署報告5,281死。检方称多起警方声明与公开言论显示有杀戮指示与绩效压力。
OPCV主辩保利娜·马西达强调人间代价，称受害者多来自贫困家庭，许多无重大犯罪记录，家属目睹武装人员闯入、亲人哀求仍被枪杀或看到警察摆放毒包与枪械的证词，证明这些并非孤立事件，而是有指挥、有组织的攻击。
听证会将评估现有证据能否构成“有实质根据”的起诉，以决定案件是否进入正式审判程序。

美国AI实验室Anthropic指控三家中国公司通过“蒸馏”手段获取其Claude模型能力，暴露了业界训练实践的灰色地带。Anthropic周一在博客中称，DeepSeek、Moonshot AI和MiniMax AI使用约2.4万名虚假账号，与Claude模型产生超1600万次对话，构成未经授权的“蒸馏”——即将大模型能力迁移到小模型的技术。
Anthropic称，DeepSeek生成约15万次侧重推理的对话，Moonshot和MiniMax分别生成约340万次和1300万次、主要针对代码与其它功能。三家公司未立即回应置评请求；Moonshot研究员在社交平台转发指控并配以翻白眼表情。Moonshot创始人杨智林此前称，其Kimi K2.5模型在多项基准上胜过Claude，且此前出现自称为Claude的情况可能因“互联网上的代码数据”进入训练集所致。
蒸馏（knowledge distillation）是AI领域常用方法，早在2015年即被提出并广泛应用。Anthropic表示蒸馏本身并非法，但其服务条款禁止竞争对手用其模型进行蒸馏，并以国家安全为由限制来自中国的访问。OpenAI也在致美国国会的备忘录中点名DeepSeek曾利用美方模型做蒸馏。专家指出，此类做法在行业内较为普遍，且辨别来源并非易事。
此事引发业界对训练数据来源与规范的争论。xAI创始人埃隆·马斯克指责Anthropic曾大规模使用有版权的训练数据，Anthropic暂未回应。与此同时，投资者对中国AI公司的兴趣仍高，MiniMax股价周二收于港币880元，较发行价上涨155%；此前被指蒸馏美方模型的智谱AI股价亦上涨近12%，收于港币628元。

近期数日，政府接连遭遇冲击：最高法院裁定其全球关税计划无效；最新经济数据显示增长放缓、通胀回升；国土安全部因国会两党就移民执法冲突而几近停摆；路透/益普索民调称六成美国人认为特朗普随年龄变得反复无常，其中包括30%共和党人。针对已定罪性犯罪者杰弗里·爱泼斯坦的丑闻，特朗普仍难摆脱指控人物影响，民主党人邀请多名受害者出席抗议。
白宫官员称，特朗普将宣布一项计划，要求科技公司为在当地新建的人工智能数据中心增加的电力成本承担更多费用；福克斯新闻称他还将呼吁新一轮个人与企业减税。
对伊朗问题，官员表示特朗普正向中东增兵并酝酿可能包括政权更迭的方案，周二演讲或首次公开为军事干预作陈述。当天恰逢俄乌战争爆发四周年，亦将成为他检视“促成和平”记录的场合。
特朗普预计会对最高法院关税裁决提出异议并提出替代法律路径，同时强调股市涨幅、私营投资与减税成果为其经济成就证据，并重申强硬边境与驱逐政策。幕僚敦促他聚焦经济议题，但他历来喜欢即兴发挥，演讲可能较长。
超过20位众参两院民主党人将缺席并在国家广场举行集会，称将提供对特朗普执政更“诚实的描述”。弗吉尼亚州长阿比盖尔·斯潘伯格将代表民主党发表官方回应，加州参议员亚历克斯·帕迪利亚将发表西班牙语反驳声明。

【上海：完善个人住房房产税政策】金十数据2月25日讯，上海市住房城乡建设管理委等五部门联合印发《关于进一步优化调整本市房地产政策的通知》。《通知》规定自2026年1月1日起，对本市户籍居民家庭中的子女成年后，购买住房属于成年子女家庭唯一住房的，暂免征收个人住房房产税。即对购房人于未成年时（或于本市个人住房房产税试点前）已与父母、（外）祖父母共同拥有住房的，在本市新购或置换住房后，该住房仍属于成年子女家庭唯一住房的（除上述共同拥有住房外），暂免征收个人住房房产税。购房人家庭住房情况发生变化，符合上述条件的，可向应税住房所在地税务机关重新办理个人住房房产税纳税信息申报、认定，从税务机关重新认定之次月起调整纳税，并退还所属期在2026年1月1日以后多缴纳的税款。
