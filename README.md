日内瓦 — 无国界医生（MSF）4月28日发布报告指责，以色列当局系统性剥夺加沙民众赖以生存的水资源，将其作为对巴勒斯坦人的“集体惩罚”。
题为《以水为武器》的报告称，2024至2025年收集的证词与数据表明，广泛摧毁平民供水设施并阻碍水源进入，与直接杀害平民、摧毁医疗与住宅一起，构成“对加沙巴勒斯坦人口故意施加毁灭性与不人道生活条件”，甚至是“种族灭绝的组成部分”。
报告援引联合国、欧盟与世界银行数据称，以色列已损毁或破坏近90%的供水和卫生基础设施，海水淡化厂、井、管网和污水系统不可用或无法进入。MSF记录到其标识明显的水车与水井遭射击或摧毁，民众为取水受伤甚至死亡。
MSF称其为加沙仅次于地方当局的主要饮水供给者，3月每日供应逾530万升，满足约40.7万人（约五分之一）的最低需求。但战争期间，以军的撤离命令使其团队无法进入原先供应地区，约三分之一引入关键水处理与卫生物资的请求被拒或未获答复。
MSF紧急事务经理克莱尔·桑·菲利波警告，断水叠加恶劣居住条件、严重拥挤和医疗体系崩溃，将为疾病蔓延酿成“完美风暴”。MSF呼吁以色列立即恢复加沙所需水量，并督促以色列盟友利用影响力，施压以方停止阻碍人道援助。AFP

此次演讲为英国君主第二次在国会发表讲话（上一次为伊丽莎白二世1991年），当地时间15:00开始，预计约20分钟，议题包含北约、中东与乌克兰等。王室方面称，稿件在政府建议下起草，但措辞与语气多来自国王本人，核心在于面对共同挑战时捍卫共享价值，维护国际安全与繁荣，并称英美伙伴关系“堪称人类历史上最重要的同盟之一”。
尽管特朗普长期公开赞赏王室并称查尔斯为“伟人”，但他与英国首相斯塔默存在摩擦。斯塔默政府希望此行能修复近期紧张的跨大西洋关系；访英恰逢美以对伊朗行动引发的争议，以及一封涉及福克兰群岛立场的五角大楼内部邮件引发的关注。
访问于4月27日开始，王室在白宫与特朗普夫妇共进私人茶会，并出席英国大使官邸的花园招待会；4月29日将赴纽约悼念2001年9·11遇难者，4月30日于弗吉尼亚会见环保保护人士，行程以国事晚宴等活动为高潮。来源：路透社

美国财政部长斯科特·贝森特4月27日警告称，与伊朗航空公司有业务往来的企业可能面临美国制裁，指出这是对德黑兰施加经济压力的一部分。贝森特在社交平台X上表示，与受制裁的伊朗航空实体开展业务——包括提供航空燃油、机上餐饮、停场费或维修等服务——都可能招致美国制裁，财政部将对为伊朗实体提供便利的第三方采取行动。
一位特朗普政府官员补充称，财政部将对继续为伊朗活动提供便利的外国金融机构实施次级制裁。伊朗国营媒体周末报道，德黑兰国际机场自战争爆发以来首次恢复商业航班，航线包括伊斯坦布尔、阿曼马斯喀特、沙特麦地那，以及飞往伊拉克和卡塔尔的航班。
背景方面，美以于2月28日对伊朗发动攻击，伊朗随后对以色列及拥有美军基地的海湾国家进行还击。近三周来虽出现脆弱停火，但美以对伊朗的打击以及以色列在黎巴嫩的行动已导致数千人死亡、数百万人流离失所。
