日本现“隐形贫困群” 漂泊无依以车为家
日本的汽车休息站，白天是游客和司机歇脚的地方，夜晚却成了一些贫困者的栖身之所。日本放送协会（NHK）此前对日本1160个路边休息区进行调查。结果显示，其中335个（占总数的29%）出现以车为家的流浪人。
这些人的苦难不为外界所知，成为日本社会里的“隐形贫困群”。其中有一些是贫困妇女，在休息站内艰难度日。　　
日本2002年出台《无家可归者自立支援特别措施法》，官方将“无家可归者”定义为在城市公园、河流、道路、火车站等公共场所生活的人。然而，这一定义近年被指已跟不上现实变化，因为新型贫困群体的居所已逐渐转移到网吧和漫画咖啡馆。
东京都政府此前的调查显示，每天约有4000人因没有固定住所，只能在这些场所过夜。与此同时，日本汽车休息站也正成为贫困者新的栖身地点。　　
静冈县民间救贫组织“POPOLO”秘书长铃木和树（44岁）指出，日本的无家可归现象已演变成一种“混合型”现象。
除了网吧和24小时便利店，也有约一成无家可归者睡在车里。白天，他们把车开到商场、公园等地消磨时间；到了晚上，再把车停在休息站内，过着孤独而漂泊的生活。
“POPOLO”发现，不少车居流浪者的生活状况已跌至谷底。透过车窗可见，车内毯子、衣物和生活垃圾混杂堆放。
一名68岁的货车司机失业后，每月只能依靠约10万日元（约800新元）养老金生活。由于付不起房租，只好住在车里。他在车内生活了一年半，体重骤减近30公斤。车检过期后，他的车就未曾离开过路边休息区。　　　　　　
调查还显示，在以车为家的贫困者中，约两成为女性。一名40多岁的女性长期在群马县和长野县乡间的休息站之间辗转。当社工接近她时，她说：“我已经在车里生活三年了。”
社工观察到，她通常会把车停在休息区灯光较亮的地方，以保护自身安全。她只在明亮、容易被人发现的地方过夜；但她也担心长时间停留在同一个休息站会引人注意，只能开着车住不同的休息站。
《读卖新闻》指出，随着互联网普及，流浪者的生活范围正在扩大，栖身场所也日益多样化。日本虽设有支援无家可归者的机制，例如提供临时住宿等救助措施，但以车为家的流浪者不在现行福利救助范围内，他们因而成为社会中孤立、难以获得救助的“隐形贫困群体”。
