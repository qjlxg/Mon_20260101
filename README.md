北京丰台区人民法院星期一（2月2日）通过微信公号“北京丰台法院”，通报上述消息。
根据通报，北京市丰台区法院当天宣判，以重大责任事故罪对被告人汪文杰等19人依法判处有期徒刑六年半至二年不等刑期。
法院经审理查明，2023年4月18日12时50分许，在北京长峰医院改造工程施工现场，施工单位违规进行自流平地面施工和门框安装切割交叉作业，环氧树脂底涂材料中的易燃易爆成分挥发、形成爆炸性气体混合物，遇角磨机切割金属净化板产生的火花发生爆燃；引燃现场附近可燃物，产生的明火及高温烟气引燃楼内木质装修材料，部分防火分隔未发挥作用，固定消防设施失效，致使火势扩大、大量烟气蔓延；加之未能有效组织高楼层患者疏散转移等原因，造成29人死亡、42人受伤和严重经济损失。
法院经审理认为，各被告人在生产、作业中违反有关安全管理的规定，因而发生重大伤亡事故，情节特别恶劣，依法均应予以惩处。综合考虑各被告人犯罪的事实、情节、危害后果以及是否自首、认罪悔罪、赔偿经济损失等，依法作出上述判决。
新华社报道，4月18日案发当日，北京市公安局丰台分局以涉嫌重大责任事故罪对相关责任人员立案侦查，后对北京长峰医院及施工单位15人刑事拘留。中国最高检5月决定对事故挂牌督办，要求北京市检察机关充分发挥检察职能作用，查明各方责任，惩处相关犯罪。
同年10月，中国官方通报，包括北京副市长靳伟在内等40余名公职人员被问责，包括北京市长峰医院股份有限公司法定代表人汪文杰、长峰医院院长王晓玲在内等20人涉嫌重大责任事故罪，被公安机关立案侦查，其中19人已被检察机关批准逮捕。

法新社报道，根据澳洲政府去年12月10日生效的社媒禁令，包括Snapchat、Meta、TikTok和YouTube在内的平台，必须禁止未满16岁的用户持有账号。若没有采取合理措施遵守规定，相关公司将面临最高4950万澳元（约4400万新元）的罚款。

Snapchat发声明称，截至1月底，Snapchat已在澳洲封禁或停用41万5000个属于16岁以下用户的账号，并称“我们每天仍在封禁更多账号”。

Snapchat还表示，不认为澳洲全面禁止未成年人使用社媒是正确做法。
“以Snapchat为例，它主要是一款供年轻人用来与亲密朋友和家人保持联系的即时通讯应用。我们不认为切断青少年与这些关系的联系，会让他们更安全、更快乐，或带来任何实质性的好处。”
