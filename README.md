（首尔／东京综合电） 高度依赖中东原油的日本与韩国相继启动紧急应对机制。两国通过释放国家战略储备、重启价格管制及提升核电利用率等措施，力求稳定国内能源供应与物价，防范战事外溢风险重创东亚经济。
韩国总统李在明星期二（3月24日）在内阁会议宣布，启动政府紧急应变机制，并指示各部门制定具体对策，以应对能源供应紧张及更广泛的经济冲击。据悉，韩国政府拟于本周五（27日）发布第二次最高油价公开通知。
为减轻能源价格上涨对民生造成的压力，李在明敦促国会尽快通过补充预算，协助受能源形势影响的群体。政府也正考虑发放额外能源券，支援弱势家庭。
李在明还呼吁全民参与节约能源，并要求公共机构率先实施五天制车辆轮流限行制度，降低能源消耗。
在供应端，韩国正放宽燃煤发电限制，将核电站利用率提升至80%的最高水平，以增强国内发电能力。
美以伊冲突导致油价暴涨后，韩国已重启中断近30年的价格管制。本月13日，当局自1997年石油价格自由化以来，首次对炼油厂批发价设定法定上限，并将每两周调整一次。
日本方面也在加强能源风险防范措施。
首相高市早苗星期二在阁僚会议上指示经济产业部长赤泽亮正，对石油相关产品的供应链展开全面评估。
高市会后指出，政府正以高度紧迫感应对能源安全挑战，并将继续推动稳定中东局势的外交努力。
日本上周一（16日）开始释放民间持有的石油储备，并定于本周四（26日）起动用国家战略储备。
根据国际能源署（IEA）数据，日本为IEA协调的创纪录石油储备释放行动贡献的总量将近8000万桶，主要是原油。
此外，日本也准备动用与沙特阿拉伯、阿联酋及科威特共同持有的1300万桶储备（约七天供应量），其中约五天供应量将投放国内市场。
为避开局势凶险的霍尔木兹海峡，日本正调整航运路径。当局披露，两艘分别从沙特延布港及阿联酋富查伊拉出发的油轮正绕道驶往日本，预计于本周及4月初抵达。
除释放储备外，日本政府还考虑采取更直接的市场稳定措施。日本财政部正征询市场意见，研究是否介入原油期货市场，以减轻战事对油价及汇率波动的影响。
日本财长片山皋月周二称，原油期货市场的投机交易已开始影响外汇市场走势，政府准备在必要时采取一切可能措施稳定市场。

能源部副部长费利克斯·威廉·弗恩特贝拉（Felix William Fuentebella）称，尽管大都会马尼拉柴油每升已飙升至约134比索，但电网仍然稳定。他表示，停电并非由“油荒”引起，并承诺若出现供给问题将保持透明通报。 
另一位副部长亚历山德罗·塞尔斯（Alessandro Sales）称，电力部门可能会受到有限影响。他指出，菲律宾高度依赖进口煤炭，但来自印尼的运费上升已导致燃料成本边际性上涨约10%至20%。国营PNOC-EC已越过常规定触发机制寻求新供货商，本周预计有专用柴油货船抵达苏比克以充实储备。 
目前汽油储备约为38天、柴油约42天。政府亦放宽规定，允许老旧吉普尼和发电机使用欧2标准柴油以缓解压力。 
参议员对此提出警示，指现行补贴已被市场消化。里萨·洪蒂维罗斯（Risa Hontiveros）称，近期減免特別消費稅对每升最高上涨达18比索的冲击不够；洛伦·莱加尔达（Loren Legarda）主张暂停12%增值税，预计每升可降约15.6比索。能源部官员则建议谨慎，倾向通过零税率安排以维护供应链稳定。 
参议员班姆·阿基诺（Bam Aquino）提醒，若国际局势缓和，油价回落也不会立刻发生，可能需数月时间。

马尼拉消息——菲律宾总统费迪南德·马科斯周二在接受采访时警告，由于燃油供应“短缺”，多班从菲律宾出港的航班可能面临停飞的“明确可能性”。
马科斯在接受彭博社采访时称，他与航空业者交谈后发现，“一些国家表示无法为我们的飞机加油，要求携带返程燃油”。他并指出，长途航班的影响将更为严重。
被问及是否会因此在菲律宾境内停飞航班时，马科斯表示“我们希望不会，但这是一个可能性，一个明确的可能性”。
菲律宾最大航空公司宿务太平洋已因燃油价格飙升而暂停若干航线并减少航班频次。政府方面则一再强调并非能源危机，通信副部长克莱尔·卡斯特罗称目前是燃油“价格扰动”，国内石油供应充足。
为缓解国际油价上涨冲击，马科斯此前已将一项允许行政部门暂停或降低油品消费税的法案列为紧急法案；该法案已在国会通过，但距通过一周后仍未签署生效。

河内——越南民航局表示，由于中东冲突导致航空燃油（Jet A-1）供应受限，越南国家航空公司自4月1日起将临时停飞多条国内航线，合计每周约23班。
民航局称，中东战争爆发三周以来喷气燃油价格大幅上涨，推高油价并引发燃料短缺风险，因此不得不调整航班安排；主要国内干线和国际航班将继续保障运行。
越航及其他航空公司正研究对国际航线加收燃油附加费，预计可能在4月实施。越南已向卡塔尔、科威特、阿尔及利亚和日本等国寻求燃油支援，并于周一与俄罗斯签署两国油气生产合作协议。
区域方面，缅甸国家航空公司周日亦宣布因“不可避免的情况”取消部分国内航班。美国联合航空上周表示，因喷气燃油成本上升将缩减运力，预计在海湾冲突持续影响下油价仍将攀升。

菲律宾能源部表示，受中东冲突扰乱天然气供应、液化天然气(LNG)价格飙升影响，政府将提高燃煤电厂产能以抑制电价上涨。
能源部长沙龙·加林（Sharon Garin）周二对媒体说，菲律宾约有60%的发电依赖煤炭，短期内不得不“更依赖”这种高碳燃料，同时尽量“最大化”本地煤炭利用，并保留向主要供应国印尼增购的选项。
加林称，当局已与发电公司商谈，评估燃煤厂可提高的发电量，这一“临时措施”最早可于4月1日启动，目的在于减缓因中东冲突带来的电价上涨。
印尼方面已向菲律宾保证，目前对印尼煤炭订单没有限制，加林并指出因此不一定需要大幅增购。
菲律宾长期依赖进口燃料并时常面临停电风险。今年1月，总统马科斯宣布在日益枯竭的马兰帕亚（Malampaya）海上天然气田附近发现“重大”天然气藏，盼望为向来为吕宋岛约40%电力供应的该气田延寿。

农业部称将于4月发放约100亿比索补助，以缓解因燃油价格飙升对农渔业的冲击。
农业部长弗朗西斯科·提乌·劳雷尔（Francisco Tiu Laurel Jr.）在参议院例会及部门简报会上表示，这笔“总统对农渔民援助”来自本年度农业部预算，预计“希望在4月6日当周”开始发放，仍待财政部发出支出授权（SARO），财政部长Toledo承诺在4月2日前释放资金。
农业部拟向登记在《农业基本部门名册》（RSBSA）的约417.5万名受益者每人发放2,325比索；其中约219.7万为稻农、147.5万为玉米种植者、10.2万为甘蔗农；另有近40万名渔民也将受惠。
劳雷尔并指出，来自往年燃料补贴的余款规模有限，已于3月19日开始先行发放，首批约2.5万人受益，累计约4万名受益者将分批次收到往年补贴。
他提醒，霍尔木兹海峡受阻将影响全球尿素与化肥运输，推高国内成本，进而波及农业和渔业。农业部应对六大优先项包括：强化国内生產與良种推广、以燃料补贴及生物/有机替代品降低投入成本、完善冷链与港口运输协调、分散进口来源、通过价格监测与定向救助保护消费者并必要时动用战略储备及价格上限。
劳雷尔呼吁各界提高效率、支持本地生产并团结应对挑战。国家首都区小型渔民因燃油涨价被迫缩短出海、靠近近海作业，收入减少难以维持生计。菲律宾农业与食品商会则表示，尽管燃料涨价可能推高食品价格，但本季农产品供应仍充足。

洛杉矶讯 — 在一起具有标志性的社交媒体诉讼中，陪审团周二恢复合议，但就两被告之一——Meta与YouTube——能否达成一致存在分歧。
陪审员通过便条向主审法官卡罗琳·库尔（Carolyn Kuhl）表示：“就某一被告难以达成共识，请问有何建议？”库尔法官要求他们继续合议，并提醒若无法达成裁决，将需从同一社区另行挑选陪审团重审，增加各方费用。
周一下午未达成裁决，陪审团将于周二返回继续商议。上周完整一周的合议在周五结束时，陪审团曾就计算赔偿额向法官询问，显示部分陪审员认同一方或双方平台存在过失或有害设计，应予以警示。
这起诉讼是数百起类似案件中的先行案，原告指控社交平台通过设计吸引未成年用户上瘾，导致抑郁、饮食失调、住院治疗乃至自杀等后果。
被告长期依赖美国《通信道德法》第230条阻隔责任，但本案主张这些公司应对其“有缺陷”的产品负责，其商业模式刻意延长用户停留并推送可能损害心理健康的内容。
加州20岁原告Kaley G.M.在庭上称，自童年起YouTube和Instagram助长了她的抑郁与自杀念头，称六岁起便沉迷社媒；交叉质询中她也提到曾遭家庭冷落与指责。
陪审表格要求裁定Meta或YouTube是否应当预见其服务对儿童构成危险、设计是否存在过失；若成立，还要认定其是否为造成Kaley困境的“重大因素”以及应承担的赔偿额。
此案被选为“风向标”案件，其判决可能影响其他指责社媒导致精神创伤的诉讼；但若陪审团就某被告无法达成一致，可能由另一案件来确立先例。原告律师马克·兰尼尔称，目前形势尚不明确，也“远未接近误审”。
（法新社）
