大陆国台办：和平统一将是台湾基建焕然一新的起点
中国大陆国台办发言人朱凤莲星期三（3月25日）表示，和平统一将是台湾基础设施、新型基础设施焕然一新的起点。她指出，和平统一后，大陆强大的基础设施建设能力将为台湾基础设施更新升级提供强大支持，台湾交通、能源、农林、水利、城市更新等基础设施将得到重建和升级。
大陆国台办星期三举行例行新闻发布会，朱凤莲会上就大陆基建相关问题回答记者提问。她表示，“十五五”（2026年至2030年）时期，中国大陆将加快构建现代化基础设施体系，为强国建设、民族复兴伟业提供坚强支撑。
朱凤莲指出，在“十五五”规划109项重大工程项目中，与基础设施直接相关的有23项。这不仅将进一步提升交通、水利、能源等基础设施建设，同时还将加强重大科技基础设施、信息通信网络、算力网络等新型基础设施建设。

中国科学院院士、中国杰出的微电子科学家吴德馨因病医治无效，于星期一（3月23日）下午在北京逝世，享年90岁。
中国科学院微电子研究所星期二（24日）发布讣告，公布吴德馨逝世的消息。
中新社指出，吴德馨是中国半导体与集成电路研究的开拓者之一，为中国微电子事业发展作出了卓越贡献。
吴德馨1936年12月生于河北乐亭，1961年毕业于清华大学无线电电子工程系，成为清华大学第一批半导体专业毕业生。
她毕业后到中国科学院半导体研究所工作，1986年调入中国科学院微电子研究所前身中国科学院微电子中心工作，1991年当选为中国科学院院士（学部委员）。
在中国科学院半导体研究所工作期间，吴德馨作为课题负责人，承担了中共建政初期“12年科学规划”中的“平面型高速开关晶体管的研究”，独立自主解决了提高开关速度的关键问题，开关速度达到当时国际同类产品水平。
该技术打破了西方国家对中国的封锁，为“两弹一星”配套的“109丙”计算机提供了核心器件。
在中国科学院微电子中心，吴德馨成功开发出双层多晶硅和差值氧化工艺，打破中国大规模集成电路成品率长期低下的局面，为中国集成电路工业生产奠定基础。
她并领导开发出0.8微米CMOS全套工艺，科研成果和产品开发等效益不断增加。
吴德馨先后荣获中国国家科技进步奖二等奖、中国科学院科技进步一等奖、何梁何利基金科学与技术进步奖等。
