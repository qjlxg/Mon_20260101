华盛顿——总统唐纳德·特朗普周六将进口商品的全球关税提高到15%，在最高法院日前大体裁定其关税计划违法后，继续推行强硬贸易政策。
特朗普在社交平台发布声明称，法院周五做出“极不利于美国”的裁决后，政府经审查决定将关税上调到“法律允许并经检验”的15%。此前最高法院以6比3否定他依据1977年紧急经济权力法案征税的权限，特朗普曾短暂改宣布以另一法律途径征收10%全球关税，随后改为15%。
他同时对支持多数意见的保守派大法官发起人身攻击，指责其“不忠”，并称他们是“愚人和走狗”。这一判决被视为自他重返白宫以来在最高法院遭遇的最大挫败，可能加剧全球贸易不确定性。
白宫称新关税为期150天，医药等正在单独审查的行业以及根据美墨加协定进入美国的商品可获豁免；此前与美方达成单独关税协议的贸易伙伴也将受影响。
国际反应方面，巴西总统卢拉呼吁对各国一视同仁、避免新冷战；德国总理梅尔茨表示将与欧盟盟友商讨统一立场，并计划三月初访美。宾夕法尼亚州长乔什·夏皮罗敦促特朗普听从最高法院、结束混乱关税，停止对农民、小企业和家庭造成伤害。
法院裁决未影响特朗普另行对钢铁、铝等实施的行业性关税，正在进行的调查仍可能产生更多部门性关税。特朗普对三位支持其权力的大法官（克拉伦斯·托马斯、塞缪尔·阿利托和布雷特·卡瓦诺）表示感谢，同时指控六人多数受到“外国利益”影响。
市场反应方面，周五华尔街小幅上扬，商界普遍欢迎判决，全国零售联合会称此举为企业带来急需的确定性。司法程序是否需退还已征关税未获法院裁定，白宫称若关税被判非法将退还，特朗普预计相关诉讼将持续多年，卡瓦诺曾警告退税流程可能“一团乱”。
