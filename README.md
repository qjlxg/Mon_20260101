本届世界杯首次由美国、加拿大、墨西哥三国联合举办，决赛定于7月19日在美国新泽西州大都会人寿体育场（MetLife Stadium）举行。国际足联主席因凡蒂诺说，美国总统特朗普将出席世界杯大决赛，把世界杯奖杯颁给冠军球队。
