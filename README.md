国际劳工组织通过全球首个保障零工权益公约
（日内瓦综合电）联合国旗下国际劳工组织（ILO）周五批准全球首个保障数码平台零工权益的具约束力国际公约；条约一获得核准，全球数以亿计通过数码平台工作的人员如外卖配送员和网约车司机，在薪酬、安全和社会福利方面的权利将有望得到保障。
世界银行2023年估计，全球有多达4亿3500万在线零工劳动者。他们的任务和薪酬很大程度由平台控制，但平台公司通常将他们归类为独立承包商，而非雇员。这使得平台得以规避最低工资要求、工作场所安全以及社会保障覆盖等问题。
根据国际劳工组织通过的文本，《平台经济中体面工作公约》（Decent Work in the Platform Economy Convention）适用于所有数码劳工平台和所有数码平台员工，无论他们是雇员还是独立承包。它也首次确立一套适用于任何就业地位的保护措施，包括职业安全与健康、最低报酬以及防止无理终止或停用账户的保护措施。 
在周五（6月12日）的表决中，406个成员包括中国、日本、德国和法国投了赞成票，八个成员包括美国和新西兰投了反对票，另有英国和印度等36个成员弃权。
美国代表里博尼说，美国不支持在快速发展的经济领域，制定具有规定性的约束力公约，因为过于僵化的规则会阻碍创新，并损害它要帮助员工的本意。
国际工会联合会说，公约将有助确保数百万平台员工能够享受到所有员工应得的权利、保护和尊严。
公约还有待各国核准，只要有两个国家向国际劳工组织登记批准了公约文本，它就能在全球生效。生效日期为各国登记批准公约满12个月后。
如果一个国家批准了公约并将它转化为国内法，那么个人可能就能够将平台公司告上法庭。

中国出台金融信息服务数据管理指引 加强网络安全监管
中国官方出台金融信息服务数据分类分级指引，以强化数据安全管理，并规范行业发展。
综合新华社、彭博社和路透社报道，中国国家互联网信息办公室、中国人民银行、国家金融监督管理总局、中国证券监督管理委员会、国家统计局、国家外汇管理局星期六（6月13日）联合印发《金融信息服务数据分类分级指南》。
《指南》适用于在中国境内开展业务的金融信息服务机构，明确了数据分类分级及安全保护要求。
根据《指南》，金融信息服务数据被划分为业务数据、用户数据和企业数据三大类，并根据数据泄露、篡改或滥用可能造成的影响，分为核心数据、重要数据、敏感一般数据和一般数据四个等级。
官方称，出台《指南》旨在帮助金融信息服务机构在数据规模快速增长的背景下，按照中国数据安全法律法规加强数据管理，为数据分类分级和安全保护提供统一标准，提升数据安全保障水平，促进数据依法合规使用。
监管部门在配套问答文件中指出，随着金融信息服务行业数据规模扩大、流动频繁，推进数据分类分级管理、实现数据治理规范化的需求愈发迫切。
根据《指南》要求，相关机构需开展数据资产梳理，完成数据分类分级，建立分类分级清单，并报送重要数据目录。监管部门指出，此举将有助于规范数据处理活动，进一步强化数据安全管理。

苏黎世，6月12日 — 瑞士将于周日就一项将人口上限设为1000万的公投做出决定。有评论将此与英国脱欧相提并论，认为结果可能对经济及与欧盟关系产生深远影响。

背景与动因：自2002年人员自由流动生效以来，瑞士人口增长明显快于欧盟。右翼民粹政党瑞士人民党（SVP）发起此案，称大规模移民给公共服务、住房带来压力，并导致犯罪上升。较高薪酬与低税率吸引了大量移民，亦促进公司落地与对技术人才的需求。
通过条件与影响：该提案需获多数选民与多数州（州）支持方可通过。瑞士人口老龄化严重，经济对欧盟移民依赖较大；外籍居民占常住人口近28%，一项研究显示近四成公司创始人为外籍。欧盟是瑞士最重要的贸易伙伴，2024年底两者达成深化经贸一体化的双边协议，SVP反对，该协议正进入议会并可能面临艰难公投。限制欧洲劳动者入境或将动摇双边关系。
