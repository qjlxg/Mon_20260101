英国气象局星期三发布年度报告说，极端气候在英国正日益成为“常态”，与上世纪典型的英国天气相比，如今英国的气候已发生显著变化，气温持续升高，极端天气事件频发。伦敦气温超过30摄氏度的天数比上世纪80年代多了三倍。去年也是英国有记录以来最热的一年
报告指出，自1980年代以来，英国气温每10年上升约0.25摄氏度，从沿海到高地无一处不受影响。以伦敦所在的英格兰东南部为例，最热日子里的气温比过去高了4.5摄氏度。热浪也在向北蔓延，侵蚀寒冷的山顶。
目前英国及西欧大部分地区正遭受三轮致命热浪袭击。屡创新高的气温给电网、交通运输和医疗体系带来压力，扰乱了欧洲各地民众的日常生活。

美国众议院通过法案 将夏令时日光节约永久化
美国国会众议院通过法案，将夏令时或日光节约（Daylight saving）的时间永久固定。法案若获得参议院通过，美国民众从此无须因为冬天太阳升得晚而在每年11月的首个星期天将时间往回调一小时，隔年3月的第二个星期天再往后调一小时。
只不过，参议院会否通过法案仍高度不确定。
众议院星期二（7月14日）以308票赞成、117票反对，通过这项被称为《阳光保护法案》（Sunshine Protection Act）的法案。除非个别州属在法案生效前选择豁免，否则美国全国将全年采用目前仅在3月至11月期间采用的时间。
星期二在众议院主持现场投票的田纳西州共和党众议员德斯贾拉斯（Scott DesJarlais）在宣读最终计票结果时，用手机播放披头士的歌曲《太阳出来了》（Here Comes the Sun）。
此前，美国总统特朗普已公开敦促国会通过这项法案。他5月21日在自家社媒贴文称：“人们是时候不再为‘时钟’而烦恼了，更不用说为了这场每年两次、荒谬至极的折腾所耗费的精力和资金了。这对于共和党来说也将是一个非常漂亮的胜利。拿下它！”
《阳光保护法案》目前已提交至参议院审议。
去年10月，由于阿肯色州的共和党参议员科顿（Tom Cotton）反对通过“一致同意”程序快速通过法案后，参议院版本的《阳光保护法案》陷入停滞。
科顿当时说，转为永久夏令时可能会带来一些被忽视的负面后果。他说，在全国一些地区，太阳要到上午9点或更晚才会升起，这可能导致清晨通勤环境黑暗且危险，而且一些人也不得不在没有阳光的清晨开始工作。
一名国会高级助理星期二透露，科顿对这项提案仍持有与以往“相同的担忧”，且两党多名参议员已在委员会中对参议院版本的《阳光保护法案》表示反对。
这名助理透露，科顿将要求参议院多数党领袖图恩（John Thune）不要将该法案提请表决。
田纳西州的共和党众议员伯切特（Tim Burchett）说，他的选民非常渴望将夏令时永久化，但他对参议院能否通过法案持怀疑态度。
