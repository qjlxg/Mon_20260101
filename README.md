美国国家海洋和大气管理局下属的国家气象局气候预测中心说，厄尔尼诺现象将持续增强至今年年底，并有极大概率持续至2027年初春。
新华社报道，美国国家海洋和大气管理局（NOAA）下属国家气象局气候预测中心星期四（7月9日）发布报告说，过去一个月，赤道太平洋中部和东部海表温度持续升高，海洋与大气耦合特征显示，厄尔尼诺现象正在增强。最新每周尼诺3.4区海表温度指数为正1.2摄氏度，表明该区域海表温度较常年同期偏高1.2摄氏度。
报告说，厄尔尼诺现象预计将在今年下半年继续增强，并有97%的概率持续至2027年初春。今年10月至12月出现强厄尔尼诺事件的概率为81%，强度可能跻身1950年以来最强厄尔尼诺事件之列。
尼诺3.4区位于赤道太平洋中东部，是国际上监测厄尔尼诺—南方涛动（ENSO）的关键海区，其海表温度异常情况是判断厄尔尼诺强度的重要指标。当这个区域平均海表温度连续数月达到或超过常年平均水平0.5摄氏度时，即表明厄尔尼诺现象形成。
今年6月，美国家海洋和大气管理局宣布热带太平洋已形成厄尔尼诺现象，并发布厄尔尼诺预警。
美国国家海洋和大气管理局同日发布的另一份气候分析报告说，2026年6月是自1850年有记录以来全球第二热的6月，仅次于2024年6月。6月全球海洋表面温度则创下有记录以来同期最高水平。

泉州市长就晋江鞋厂大火向遇难者家属和全社会道歉
中国福建省泉州市市长蔡战胜就晋江鞋厂火灾致歉。
福建泉州星期五（7月10日）上午召开新闻发布会，介绍晋江鞋厂火灾相关情况。新闻发布会开始前，全体人员向遇难者默哀。泉州市市长蔡战胜鞠躬致歉，“在这里，泉州市委市政府向所有遇难者表示深切哀悼，向所有遇难者家属、向全社会诚恳道歉。”
星期四（9日）中午12时4分许，位于福建泉州晋江市的辉腾鞋业有限公司发生火情，事故发生时工厂内共有237名员工和两名外来人员，现场疏散救出213人，其中两人送医不治身亡；另外26名失联人员确认遇难，共造成28人死亡。
另据央视新闻报道，星期五上午，中国应急管理部派出的八人专家调查组前往事故现场开展事故勘验工作。央视现场报道称，发生火灾的辉腾鞋厂厂房内部，仍有很大的烟雾在弥漫，很刺鼻，所以需要戴口罩进行防护。地上堆积的物料因为过火已经炭化，旁边放物料的架子也有一些变形。屋顶上的换气扇电扇被熏黑，电扇也已经被烧得变形。过火之后，厂房内的地面上还有很多积水。专家介绍，辉腾鞋厂初始的起火位置应该就在一层。
报道称，该企业主要从事鞋制造、服装制造、服饰鞋帽批发，厂房建筑为钢筋混凝土结构，单层面积为1300平方米，共五层。着火部位为一层冲床车间，燃烧物质为鞋材，鞋材易燃蔓延快。

日本6月企业物价涨7.1% 创逾三年新高
6月日本企业物价指数同比上涨7.1%至135.4，涨幅较5月的6.6%进一步扩大，刷新2023年3月以来最高值。
新华社报道，日本央行星期五（7月10日）公布的初步统计结果显示，在纳入统计的23类商品中，有22类价格同比上涨。其中，受中东局势推高能源成本影响，石油和煤炭制品价格同比上涨22.8%，涨幅较5月明显扩大；有色金属、信息通信设备价格分别同比上涨39.2%和14.5%。
日元贬值进一步推高进口成本。6月以日元计价的进口物价指数同比上涨29.7%，创2022年11月以来最大涨幅；出口物价指数同比上涨20.7%。
企业物价指数反映企业间交易的商品价格水平，是日本衡量通胀水平的重要指标之一。

欧盟对北京鸭产品展开反倾销调查
中欧贸易争议蔓延至餐桌，欧盟对中国北京鸭肉展开反倾销调查。
据欧盟星期四（7月9日）在官方公报发布的通知，欧盟已就中国是否向欧盟市场倾销北京鸭产品展开调查。

美联储成立五工作组 推进货币政策机制
美国联邦储备委员会公布了最近设立的五个独立工作组负责人名单及工作目标，以改进货币政策执行机制。
新华社报道，美国联邦储备委员会星期四（7月9日）发布公告说，这五个工作组将审视美联储在政策制定关键领域的做法，负责人包括知名学者、前央行官员和企业高管。五个工作组将分别围绕货币政策沟通、资产负债表政策、经济数据、生产力与就业、通胀框架展开评估。
美联储主席沃什当天说，各工作组将仔细考虑政策制定者的手段和方法、分析工具以及政策路径是否有改进空间，目标是帮助美联储更有效实现维持物价稳定和促进充分就业的双重使命。
沃什在6月17日货币政策会议后的记者会上宣布，美联储将设立五个独立工作组，推进改革议程。按计划，各工作组将于今年底提交相关领域的研究报告。
分析人士认为，沃什此次广泛引入美联储体系外人士加入工作组，显示这轮改革不仅着眼于货币政策制定机制，也关注人工智能（AI）、生产力提升及数据质量等议题，未来或将对美联储产生深远影响。
由于美国货币政策对全球经济具有外溢效应，美联储下一步的改革方向受到全球资本市场密切关注。

中国首次实现运载火箭可控回收 预计年底前完成火箭复用飞行
中国运载火箭在海南发射升空后完成分离，一子级实现回收。这是中国运载火箭首次实现可控回收，预计在今年前完成火箭复用飞行。
据中国航天科技集团在官方微博发布的消息，长征十号乙运载火箭星期五（7月10日）中午12时14分在海南商业航天发射场发射升空，将卫星顺利送入预定轨道。
火箭一二级分离约六分钟后，一子级垂直返回，在海上回收平台通过网系捕获方式成功回收，发射及一子级回收任务取得圆满成功。
这是中国首次运载火箭可控回收，也是全球首次运载火箭网系回收，标志着中国在重复使用火箭技术领域取得历史性突破，将为加快提升中国进出空间能力奠定坚实基础。长征十号乙运载火箭成为中国首型成功实施回收的重复使用运载火箭。
据中国央视新闻报道，长征十号乙运载火箭研制团队接下来将持续优化火箭性能，加快重复使用火箭技术的迭代升级，预计将在今年底前完成一子级火箭复用飞行。这是长征系列运载火箭的第657次发射。
长征十号乙运载火箭由中国航天科技集团有限公司所属中国运载火箭技术研究院抓总研制，为5米直径两级串联构型大型液体运载火箭。
此次任务是继长征十号系列运载火箭低空演示验证飞行并在海上安全溅落后，对重复使用火箭回收技术的进一步验证。

重庆公安局长悬空 中国公安部长助理谭权出席重庆市委会议

此前，重庆官媒5月7日晚间引述“重庆市张安疆同志治丧工作小组”消息称，张安疆“因突发疾病经抢救无效，于2026年5月7日10时52分不幸离世，享年55岁”。
官方关于张安疆离世的通报中，仍以“同志”称呼他。官方通报发布前，坊间星期四一度流传张安疆“出事”的消息。中国全国人民代表大会常务委员会6月公告他的代表资格自然终止时，未按惯例对他的去世表示哀悼。
张安疆是重庆直辖以来第八任公安局长，也是首位没有公安政法系统背景的公安局长。他过去长期在发改系统及地方政府任职，拥有经济和科技产业管理经验。
张安疆之前的七任局长中，朱明国、王立军、何挺和邓恢林均被查，山城“刀把子”因此被形容为“高危岗位”。张安疆的突然离世，再次引起外界对这一岗位敏感性和特殊性的讨论。
