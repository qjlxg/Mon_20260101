中国全国“两会”将于下周在北京召开，北京市公安局官方微信公众号“平安北京”星期五（2月27日）发布《关于加强2026年全国“两会”期间空中安全管理工作的通告》。
通告称，为确保全国“两会”期间北京地区的空中安全，自3月1日至12日，在北京行政区域内，禁止升放气球、风筝等影响飞行安全的物体。
通告也提醒，根据《北京市人民政府关于公布无人驾驶航空器管制空域范围的通告》，北京市行政区全域为无人驾驶航空器管制空域，未经空中交通管理机构批准，不得在管制空域内实施无人驾驶航空器飞行活动。
警方同时说明，根据《关于在北京地区禁止生产销售燃放孔明灯的通告》，在北京行政区域内，禁止燃放孔明灯。
通告最后说：“请相关单位和个人严格遵守各项规定，对于违反相关法律法规的，由有关部门依法予以处罚；构成犯罪的，依法追究刑事责任。”
