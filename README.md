杨志勇：房地产回不到过去高杠杆高周转的老路
中国财政科学研究院院长杨志勇星期三（4月22日）说，房地产“回不到过去高杠杆、高周转的老路”，转向房地产发展新模式是必然选择。
据香港《明报》报道，杨志勇在中国记协举办的“继续实施更加积极的财政政策为十五五规划落实提供有力保障”的新闻茶座上，作出上述表示。他称，过度依赖房地产会挤占科技等关键领域的资源与资金，影响国家长期竞争力。
杨志勇也说，过去以土地出让收入为主的土地财政模式不可持续，须建立更可持续的地方财政结构，同时确保房地产调整过程总体平稳有序。
中国房地产市场已持续多年低迷，不仅加剧通缩压力，也抑制居民消费。在各地持续推出支持措施、带动部分购房需求回流后，房价跌势有所放缓。
官方数据显示，3月一线城市新建商品住宅价格环比由上月持平转为上涨0.2%。其中，上海、广州和深圳分别上涨0.3%、0.3%和0.2%。
麦格理首席中国经济学家胡伟俊说，中国房地产市场尚未触底，但正逐步接近底部，一旦出现明确且显著的政策调整，“房地产市场将见底”。
他提到，重大政策转向可能要等到中国出口明显走弱之时才会出现，因为届时官方或将加大对房地产的支持力度。但中国一季度出口表现仍然强劲，短期内房地产政策大幅调整的可能性不高。

大陆批民进党政府：妄想不承认九二共识恢复两岸协商对话
台湾七大工商团体之一的商业总会，本周呼吁台湾官方回应中国大陆公布的10项惠台措施，同时指有政府部门会前致电以政逼商。大陆国台办星期三（4月22日）批评民进党政府，只会一味抹黑攻击大陆惠台措施，借机进行政治操弄，妄想在不承认“九二共识”的情况下，恢复两岸协商对话。
台湾在野的中国国民党主席郑丽文与中共总书记习近平会晤后，中国大陆官方4月12日宣布10项两岸交流措施，包括推动福建沿海和金门、马祖通水、通电，重启上海、福建居民赴台自由行等。
据《联合报》、民视新闻网等报道，商总星期一（4月20日）率领七大行业工会召开记者会，提出产业界希望促进交流的诉求。商总理事长许舒博透露记者会前接到政府电话关切，外界质疑官方以政逼商。
台湾政府的大陆委员会主委邱垂正星期二（21日）受访时证实，记者会前确实有致电向业者了解情况，并称只表达关怀与支持，没有施压。
邱垂正表示，台湾政府支持业者，但认为两岸经贸交流不应被政治化，应一视同仁，不应区别对待。他指出，从历史经验可以看到，包括陆客赴台、农渔产品输往大陆等，北京曾把这些正常经济交流政治武器化，说停就停、说断就断，导致台湾弱势的农民渔业者血本无归、哭诉无门。
邱垂正表示，希望两岸能通过政府与政府的协商机制，以及制度化的保障，减少台湾业者的风险及不确定性。他说：“（台湾）政府支持在对等尊严之下，进行健康有序的两岸交流。但也不希望附带压缩‘中华民国台湾’生存空间的政治前提，或使两岸交流沦为政党之间、政治交易的筹码。”
大陆国台办发言人张晗星期三在例行新闻发布会上回应，表示台湾商业总会及相关产业界表达的心声，充分反映了近期大陆发布10项促进两岸交流合作的政策措施，符合台湾产业复苏需求，有利于解决经济民生痛点。
张晗批评民进党政府无视台湾民意，无视台湾民众利益福祉，对大陆释出的善意诚意和出台的惠台政策措施，只会一味抹黑攻击，并借机进行政治操弄，妄想在不承认“九二共识”的情况下，恢复两岸协商对话。
张晗认为，这些惯用伎俩骗不了人，也不可能得逞，只会让民进党政府越来越站在台湾主流民意的对立面。
张晗说，交流融合是两岸民众的愿望和福祉所在，两岸民众走亲走近走到一起的大潮流，谁也阻挡不了。民进党政府应停止政治操弄，顺应民心所向和大势所趋，在促进两岸交流合作上转变态度。
