据法新社报道，美国司法部上个月公布了最新一批爱泼斯坦档案，当中涉及超过300万份与调查这名声名狼藉金融家相关的文件、照片和视频。
法新社在查阅相关文件后发现，达赖喇嘛的名字被提及超过150次，但没有提及他与爱泼斯坦会面或有任何互动的内容。
法新社报道称，仅在爱泼斯坦档案中出现某人的名字，本身并不意味着此人涉及任何不当行为。
达赖喇嘛办公室在X平台发表声明称，最近一些媒体报道和社交平台贴文围绕“爱泼斯坦档案”，试图将达赖喇嘛与爱泼斯坦牵扯在一起。
声明还补充说，可以明确地确认，达赖喇嘛从未与爱泼斯坦见过面，也从未授权任何人代表他与爱泼斯坦进行会面或互动。
中国国际电视台（CGTN）上星期四（2月5日）报道称，通过美国司法部网站核实，在最近公布的爱泼斯坦档案中，“达赖喇嘛”这个名字至少出现169次。
CGTN也提到了一封电邮，其中一名身份不明的发件人告诉爱泼斯坦，正考虑出席一场达赖喇嘛将参加的活动。
观察者网上星期四也报道，通过检索发现，目前达赖喇嘛在爱泼斯坦档案里出现的次数为168次。观察者网指出，这一出入可能是由于美国司法部仍在调整公布的文件内容。
不过，法新社查阅美国司法部公布的爱泼斯坦档案后发现，其中有154条提及达赖喇嘛的记录。
多年来令美国上流社会颜面尽失的爱泼斯坦案，在2026年再度掀起巨大波澜。美国司法部日前公布迄今规模最大的相关档案，披露这名已故性犯罪者与多位政商名流的往来细节，重新引发全球关注。
