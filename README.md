海关局局长阿里尔·内波穆塞诺表示，根据2025年记录，燃油特赦税收入约为2200亿至2600亿比索，石油产品的增值税最高可达1600亿比索，合计可能减少3300亿至4100亿比索。
部分团体呼吁暂停或废除相关税收，以缓解因美以与伊朗冲突导致的全球油价上涨对民众的冲击。但总统费迪南德·马科斯表示，他较倾向于下调燃油特赦税，并将把一项授予其临时权力以调降税率的法案列为紧急事项。
内波穆塞诺强调，无论采取何种措施，政府须寻找替代财政来源，确保海关局能在2026年继续达到对国库10000亿比索的纳税目标。他称：“我们目前担忧的是，来自石油产品的正常税收（VAT、特赦税及其他税费）每年将减少至少3300亿至4100亿比索。若因危机而减收，必须另觅收入渠道。”
为防止业者趁机牟利，海关局已开始巡查全国油库，协助能源部掌握实际库存，并防止将旧货以高价出售规避监管。内波穆塞诺指出，必须确认各储油库和仓储的基线库存，杜绝“以旧充新、抬高价格”的情形。
海关将与农业部及贸易工业部合作，打击农产品及食品囤积行为，避免危机期间出现投机行为，加剧低收入民众的生活负担。

纽约讯 — 总统府周一（3月9日）在马科斯为期三天的工作访问期间提醒民众，尽管油价飙升，仍无需恐慌性抢购，保证民生物资和石油产品“充足”。
总统府新闻处副秘书长克莱尔·卡斯特罗在纽约的记者会上表示，尽管中东紧张局势持续，国人确实需有所准备，但“没有必要发生恐慌性抢购，因为供应是完整且足够的，尤其是燃油和基本商品。”
她援引贸易和工业部的说法称，大都会马尼拉及其他地区的生活必需品价格“尚无变动”。卡斯特罗并呼吁民众如遇未经授权的涨价，应向贸易部和能源部举报。
对于周二即将生效的油价上调，能源部长莎朗·加林承诺将由各加油公司分批實施，以减轻对消费者的冲击。卡斯特罗指出，目前情势难以预测，政府主要是提醒民众做好应对准备。

马拉坎南宫（总统府）周一表示，若菲律宾比索因中东冲突持续对美元下跌，将采取包括提高利率在内的干预措施。
比索在周一收盘创历史新低，报59.50比1美元。
总统府新闻官副秘书长克莱尔·卡斯特罗在马科斯三天工作访问纽约期间的记者会上称，中东局势可能引发此类后果，央行已就此表态，若走势持续将会有干预，政府亦会支持。
总统费迪南德·马科斯·小马此前表示不希望比索跌破每美元60元。央行行长埃利·雷莫洛纳曾指出，比索可能触及60元但“不会很快”，该说法是在战争爆发前发表的。
过去一年利率因通胀温和而回落，但有经济学家警告若油价持续大幅上涨，可能再次推动通胀上升。一些分析师称，多家大型菲律宾企业在规划时已将比索贬至60纳入考量。冲突爆发前，比索曾处于升值态势。
