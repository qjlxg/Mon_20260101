特朗普名字已从肯尼迪中心外墙上移除
工人已经将美国总统特朗普的名字从华盛顿著名的肯尼迪中心的外墙上移除。

截至星期六下午，肯尼迪中心外墙上仍然覆盖着一块白色防水布，这块防水布是移除特朗普名字工程的一部分。
美国哥伦比亚特区巡回上诉法院星期五（12日）驳回美国政府和肯尼迪表演艺术中心董事会的上诉，维持联邦地区法院此前的裁决，下令拆除场馆上特朗普的名字。
一群反特朗普人士星期五一直守候在施工现场，直到深夜的雷暴迫使建筑工人停工。除了在场聚集的人群以外，还有数万人通过直播远程观看。
本周早些时候，肯尼迪中心已从网站上删除特朗普的名字。
自去年1月重返白宫以来，特朗普一直积极地将自己融入美国政治体制。他任命自己为该艺术中心的主席，并在理事会中安插许多忠实拥护者。
随后，理事会在去年12月投票决定将艺术中心更名为“唐纳德·J·特朗普与约翰·F·肯尼迪表演艺术纪念中心”。
特朗普名字迅速以醒目的大字出现在肯尼迪中心的外墙上，位于肯尼迪名字的上方，导致演出纷纷取消，剩余活动的门票销量也大幅下滑。
在华盛顿的其他地方，现已关闭的美国和平研究所也被更名为特朗普研究所，他的头像也出现在司法部和农业部外墙上的巨幅横幅上。
42岁的华盛顿居民塞尔巴罗利称，肯尼迪中心是“我最喜欢的地方之一”，而移除特朗普的名字是“一大胜利”。
65岁的马丁对此表示赞同。他说：“有些人可能觉得这只是个无关紧要的……标志。但事实是，我们不能剥夺国会授予这座美丽建筑的荣誉，因为这是为纪念约翰·F·肯尼迪总统而设立的。肯尼迪总统当之无愧，他的名字应该受到保护。”

特朗普将赴法国出席七国集团峰会 会晤中东伙伴
美国总统特朗普将于下周在法国举行的七国集团（G7）领导人峰会期间会见中东伙伴，凸显出伊朗战争继续扮演着举足轻重的角色，眼下欧洲盟友正努力应对冲突带来全球经济冲击。

陆台企业签约对接 张荣恭：没什么力量能阻止自发性互利行为
中国大陆在“习郑会”后发布“十项促进两岸交流合作的政策措施”，陆台企业在海峡论坛期间正式对接签约。赴陆出席论坛的国民党副主席张荣恭说，只要对两岸基层民众交流合作有利，就没有什么力量能阻止自发性的互利行为。
国民党主席郑丽文与中共总书记习近平会晤后，中国大陆4月12日发布10项惠台措施，内容涵盖两岸直航、台湾农渔产品产销、中小企业赴陆、影视交流以及恢复上海及福建居民赴台湾个人游试点等面向。
新华社消息，相关措施的对接签约会星期六（6月13日）在福建厦门举行。大陆企业与台湾多地企业和商协会进行签约，采购台湾台东、云林、南投等县市凤梨释迦、文旦柚、茶叶等特色农渔产品。
中国国务院台办主任宋涛在签约会上致辞时说，中华民族伟大复兴势不可挡，台湾的前途命运系于祖国大陆发展进步、国家统一强盛。只有坚持“九二共识”、反对“台独”，认同两岸同属一个中国，两岸关系才能和平发展，台海才有和平安宁，两岸同胞利益福祉才有保障。
第18届海峡论坛大会星期六在厦门举行，签约会同日傍晚举行。
台湾《太报》报道，张荣恭会上致辞时说，台湾农渔民的产品行销到大陆，只是为了生计，若还要遭抹黑打压，那绝对“天理不容”。
张荣恭说，自海峡论坛开办以来，台湾各县市首长率团参加已成惯例，论坛是一个很好的平台和机会，对两岸人民都有利，绝对没有什么坏处。
对于此次海峡论坛首次发生台湾陆委会禁止台湾县市首长参与一事，张荣恭坦言遗憾，但强调“历史不能倒退”。
张荣恭也说，两岸农渔产交流则是促进两岸人民相互理解重要渠道，是增进台湾基层生活福祉最实在方式，签约仪式开启台湾农产品拓展大陆市场新阶段。

蒙古国总统：不会因同其他国家关系做损害中方利益的事情
中国官媒报道，蒙古国总统呼日勒苏赫会见到访的中国外长王毅时说，对华友好是蒙古国外交政策的首要方针，蒙方不会因为同其他国家关系做任何损害中方利益的事情。
新华社消息，呼日勒苏赫星期六（6月13日）在首都乌兰巴托会见王毅。
报道引述呼日勒苏赫说，蒙中关系已成为地区国家间关系的典范。两国始终相互理解、相互信任，尊重彼此独立、主权和领土完整，互利合作不断深化，今年有望实现双边贸易额200亿美元（257亿新元）目标。
呼日勒苏赫称，中国是蒙古国永远的好邻居。蒙方理解尊重中方的立场和关切，坚定恪守一个中国原则，认为台湾是中国领土不可分割的一部分，不支持任何形式的“台独”行径，涉港、涉藏、涉疆问题也都是中国内政。蒙古国愿同中方在国际地区事务中加强合作、共同践行。

王毅表示，中国对蒙政策保持稳定性连续性，始终把发展中蒙关系置于周边外交重要位置，有意愿也有能力做蒙古国可以依靠的邻居、值得信赖的朋友和加快发展的伙伴。中方尊重蒙古国的独立、主权和领土完整，尊重蒙方自主选择的发展道路。
他指出，中国愿同蒙古国加强发展战略对接，深挖合作潜能，推进互联互通、能源矿产、贸易投资等既有合作，在重要矿产、绿色发展、数字经济等领域打造更多合作增长点。
王毅也说，中国也愿同蒙方在中俄蒙、上海合作组织等多边平台加强团结协作，同全球南方国家联合自强，推动构建更加公正合理的全球治理体系。
访蒙期间，王毅也将会见蒙古国总理乌其尔勒，并同蒙外长巴特策策格举行会谈。

河南纸厂三员工窒息死 省委书记：严肃追究违法操作责任

苏黎世，6月12日 — 瑞士将于周日就一项将人口上限设为1000万的公投做出决定。有评论将此与英国脱欧相提并论，认为结果可能对经济及与欧盟关系产生深远影响。

据透露，备忘录拟要求恢复霍尔木兹海峡通航并解除美国对伊朗港口的海上封锁，作为交换，美国将释放被冻结的数十亿美元资产并放宽对伊朗石油的制裁；伊朗核问题将在随后60天内谈判，双方分歧包括高浓缩铀库存处置等，最终仍需伊朗最高领导人批准。

Anthropic 6月12日表示，应美国政府要求，公司将“立即禁用”其最先进的AI模型对所有用户的访问。政府向公司下达出口管制指令，要求暂停外国人对Fable 5和Mythos 5的访问，但未提供具体国家安全细节。Anthropic称，政府认为存在可能绕过或“越狱”某项保护措施的方法，使Fable 5可用于识别软件漏洞。
此次命令出台之际，Anthropic与美方的紧张关系已有前情：2026年公司拒绝允许军方将其模型用于国内监视和全自主武器后，被列入供应链黑名单。Anthropic称政府仅给出“口头证据”表明存在一处可能的、非普遍性越狱，反对以此为由召回已部署的商用模型。
本周早些时候，Anthropic推出名为Claude Fable 5的“Mythos级”模型，并设有禁止用于网络安全等高风险领域的防护措施，部分用户认为限制过宽。专家警告，这类模型若被滥用，可能显著加速针对如银行等关键行业的复杂网络攻击。Anthropic称发布前已与美方沟通安全问题，并指出若此标准普遍适用，将基本停止前沿模型的新部署。
Anthropic称为遵守指令不得不立即切断Fable 5和Mythos 5的访问，其他模型不受影响，并正努力恢复接入。亚马逊云AWS称应Anthropic要求已撤销所有地区用户对两款模型的访问。商务部一名官员证实已下达相应出口管制指令。前白宫官员Dean Ball指出，这意味着非美国人（包括在美居住者）将受限，可能需出示国籍证明。若干核心员工出生于美国之外，路透无法确认其国籍，Anthropic对此拒绝置评。
