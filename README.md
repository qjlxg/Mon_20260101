中国车企小鹏启动Robotaxi内测 拟下半年载客示范运营
（深圳／北京综合讯）中国电动车企小鹏汽车的自动驾驶出租车（Robotaxi）业务开启员工内测，集团董事长何小鹏说，Robotaxi是小鹏从新能源汽车迈向“机器人汽车”的重要一步。
综合《证券时报》与新华财经报道，小鹏集团星期四（7月9日）召开Robotaxi业务首次全员会，何小鹏在会上说，Robotaxi是小鹏“物理AI”版图的关键一子，并指未来十年，所有具身智能载体“本质上都会成为机器人”。
小鹏Robotaxi基于该公司旗舰车型GX打造，搭载四颗自研图灵人工智能（AI）晶片，本地有效算力达全球最高的3000TOPS，具备L4级自动驾驶能力。
总部位于广州的小鹏去年11月宣布开启Robotaxi项目，今年1月进行常态化道路测试，5月首台量产车下线，自启动到内测仅用了八个月。何小鹏说，这一速度超出最初预期。
