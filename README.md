70余名滞留中东台湾人抵达上海后返台
中国大陆媒体报道，70余名滞留中东的台湾民众星期二（3月10日）抵达上海，并将在同日搭乘航班返台。
据央视新闻报道，搭载这70人的东方航空MU704航班，星期二凌晨4时许平安抵达上海浦东国际机场。这是近期中东局势升级以来，经大陆中转返台人数最多的一批台湾民众。
报道称，受地区局势影响，多国关闭领空、机场停运，这批台湾人原计划经阿联酋阿布扎比转机返台，因航班取消滞留海外。经大陆有关方面协调，他们改乘东航航班从土耳其伊斯坦布尔飞抵上海中转，将于10日上午分别搭乘航班飞往台北松山机场、桃园国际机场返台。
中东局势升温后，中国大陆驻以色列大使馆在微信公众号宣布，自3月1日起开放人员登记，呼吁“自行无法安排而又有意愿撤离前往埃及的中国公民（包括港澳台同胞）注册登记”，并说登记人员“须持有中国护照（含香港特区护照、澳门特区护照、台胞证等）”。
