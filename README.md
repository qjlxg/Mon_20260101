中国仲裁机构首次聘请外籍专家担任执行机构负责人
广州仲裁委员会聘请美籍专家担任执行机构负责人，是中国大陆首例。
据中新社报道，广州仲裁委员会星期二（2月3日）发布消息，委员会聘请北京大学国际法学院执行院长、法学教授马克·费尔德曼（美国籍）担任执行机构负责人，这是中国大陆仲裁机构首次聘请外籍专家担任执行机构负责人。
新一届委员会由13名委员组成，背景涵盖普通法系的代表国家、“一带一路”共建国家、金砖国家，以及香港和中国大陆。
广州仲裁委员会是中国司法部首批培育的中国特色国际一流仲裁机构之一，设有七个分支机构、五个专业仲裁平台，上线全球首个APEC成员经济体跨境在线商事争议解决平台。数据显示，广州仲裁委员会2025年受理案件达2.7万件。

赛义夫的律师在脸书上证实他死亡的消息，但也没透露细节。
利比亚媒体Fawasel则引述曾代表卡扎菲参与联合国政治对话的阿卜杜拉（Abdullah Othman Abdurrahim）报道，赛义夫是在家中遭武装人员杀害，检察官正在调查此案。

中央一号文件再提整治农村高额彩礼
中共中央一号文件再提整治农村高额彩礼，称要引导树立正确的婚恋观、生育观、家庭观，培育简约文明的婚俗文化。
据新华社， 2026年中央一号文件《中共中央国务院关于锚定农业农村现代化扎实推进乡村全面振兴的意见》星期二（2月3日）发布，这也是中国“十五五”首个中央一号文件。文件提出，持续整治农村高额彩礼，加强省际毗邻地区联动治理。引导树立正确的婚恋观、生育观、家庭观，培育简约文明的婚俗文化。
中国农业农村部农村经济研究中心乡村治理与社会文化研究室副研究员庞静泊说，整治农村高额彩礼，本质上是扭转乡土风俗陋习、推动乡村文化治理，体现出党和国家对培育文明婚俗、涵养良好家风的重视。
2019年的一号文件首次明确把 “天价彩礼” 等不良婚俗纳入中央层面治理范围，之后的三年的中央一号文件继续多次提到推进农村移风易俗、治理高价彩礼等问题。
在新华社相关报道的微博下，许多留言称，一号文件的提法治标不治本，整治农村高额彩礼，就“不要认为女方是嫁到男方家的说法，双方父母都要出资买房及需要品”，“彩礼和嫁妆都是为了新组建家庭能承担房屋和生育的压力”。

港澳办：若巴拿马一意孤行 必将在政治经济上都付出沉重代价
在巴拿马最高法院裁定，香港长和子公司运营巴拿马运河港口的合同违反宪法后，负责香港事务的中国国务院港澳办发文指出，巴拿马应当认清形势、迷途知返，并警告如果一意孤行，执迷不悟，巴拿马必将在政治和经济上都付出沉重代价。
港澳办星期二（2月3日）晚在微信公众号发表署名“港澳平”的评论文章称，巴拿马最高法院以所谓“违宪”为由，裁决有关香港企业巴拿马运河港口特许经营合约续约无效。有关裁决罔顾事实、背信弃义，严重损害香港企业合法权益，理所当然受到中国政府和香港特区政府的坚决反对。
文章进一步指出，有关企业的港口特许经营权合约乃由双方公平自愿签订，历经巴拿马多届政府和国会认可，现在巴拿马却裁定合约违宪，意味将来同样可以裁定巴拿马任何时期批准的其他特许经营权合约违宪，这向国际投资者释放的信号是，巴拿马实际上无法为投资者提供任何保障。
文章批评此举可谓自毁国家信用，将对巴拿马营商环境和经济发展造成深远损害，并严重破坏国际贸易规则。
文章最后写道，中国从来不会屈从于强权政治和霸权主义，有足够的手段和工具、足够的实力和能力维护国际经贸秩序公平公正。巴拿马应当认清形势、迷途知返。如果一意孤行，执迷不悟，“必将在政治上、经济上都付出沉重代价！”
巴拿马高院上星期四（1月29日）裁定，长和子公司运营巴拿马运河港口的特许经营权合同违宪。裁决书称，相关合约存在“过度偏袒”长和的条款。
长和旗下巴拿马港口公司（PPC）自1997年起获巴拿马港口的运营合同，并在2021年于上一届巴拿马政府执政期间获续约，为期25年。
关于撤销巴拿马港口公司合约一事，最早由巴拿马审计部门去年提出。审计部门认为特许经营权违宪，并指长和未向巴拿马政府支付12亿美元（约15亿新元）税金。
据法新社报道，巴拿马海事局上星期五（1月30日）宣布，马士基集团旗下港口运营商APM码头公司，将担任位处巴拿马运河两端的巴尔博亚港和克里斯托瓦尔港的“临时管理方”。

浙江夫妇吃网购娃娃菜中毒案反转 实为二人合谋骗赔偿
中国浙江省一对夫妇因食用网购娃娃菜后出现中毒症状入院，警方调查曾指向商家包装不当、丈夫投毒，一度引发舆论关注。
但最新进展显示，中毒是夫妻二人事先合谋，故意制造“食用网购蔬菜中毒”的假象，企图以此向平台或商家要挟索取高额赔偿。目前，两人已被警方刑事拘留。
综合红星新闻、澎湃新闻报道，浙江省台州天台县杨女士和丈夫去年12月网购娃娃菜，食用后夫妇俩同时出现全身出血症状，入院抢救。杨女士在ICU（重症监护病房）抢救一个多月才好转，丈夫出现全身出血、肺积水、呼吸衰竭等症状。
医院诊断结果显示，夫妇俩均为老鼠药中毒。随后警方也在网购娃娃菜中检测出老鼠药，并受案。
上个月媒体披露，警方锁定毒源为包裹娃娃菜的旧报纸在回收储存中沾染杀鼠剂，商家为压缩成本用其包装，导致有毒物质渗透污染蔬菜。涉事商家随后下架失联，平台已提供店主信息，警方正追查全链条责任。
媒体星期二（2月3日）报道，警方进一步侦查发现，下毒者是杨女士的丈夫，目前涉事男子已被警方采取刑事强制措施。
2月3日晚，媒体再度披露，警方已侦查识破，丈夫的下毒行为系夫妻双方共谋，两人试图通过网购娃娃菜食用后制造中毒迹象，以此要挟，向平台或商家骗取巨额赔偿。
浙江天台公安星期二晚也发布警情通报称，针对近期网络关注的”夫妻网购娃娃菜中毒事件”，专案组开展了溯源调查，现场勘查，检验鉴定，调查走访等工作。通报称，杨女士夫妻因涉嫌敲诈勒索罪，已被依法采取刑事强制措施，案件正在进一步侦办中。

【大选前夕头寸频繁调整 日本国债价格早盘承压】金十数据2月4日讯，由于日本2月8日大选前可能出现的头寸调整，日本国债价格在东京早盘小幅走低。野村外汇研究分析师在评论中表示，媒体报道强烈倾向于执政的自民党将获得众议院多数席位。分析师表示：“如果执政联盟赢得三分之二以上席位的风险情景成为现实，这将增强首相高市的决策权，最初可能会引发支撑高市政策的交易。”但他们补充称，采取宽松财政和货币政策的压力减小，可能会获得反对党的支持，这有望使日本国债逐渐稳定。
