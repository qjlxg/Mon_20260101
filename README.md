中国体育媒体“懂球帝”报道，WTT官网显示，前中国国乒选手方博以哈萨克斯坦运动员身份注册参加WTT赛事。此前方博在个人直播中提及，今年将重返乒超赛场，并且很快将参加WTT赛事。 ​
方博星期一（4月20日）更新个人微博，表示正式宣布复出，将代表哈萨克斯坦出战，与Nomad 乒乓球学院达成合作，预计6月开始参加WTT赛事，但强调自己仍是中国国籍，不参加国际三大赛。“我复出的初衷仍然是热爱这乒乓球这项运动，我想重新挑战一下自己的极限。”       
方博曾是2009年的世界青年乒乓球锦标赛多项赛事冠军，包括男团、男单、男双、混双。
2021年，29岁的方博退役。去年8月，方博在直播中回应了自己早退役的原因，他表示，是中国前体育总局局长苟仲文直接不让他们这一波球员打了，让他们给后面的球员让道。

去年5月，中国十四届全国政协原常委、民族和宗教委员会原副主任苟仲文被提起公诉。检察机关起诉指控，苟仲文利用担任北京市政府副市长，北京市委常委，北京市委副书记，国家体育总局党组书记、局长等职务上的便利，以及职权或者地位形成的便利条件，为他人谋取利益，非法收受他人财物，数额特别巨大。

首届化石燃料退出大会将于4月28至29日在哥伦比亚圣玛尔塔举行，超过50国派出部长级代表参会。
会议在伊朗战争引发的燃料短缺与价格飙升背景下召开。国际能源署称，这次由霍尔木兹海峡局势触发的石油供应冲击是有史以来最大一例，凸显各国对煤、油、气的依赖。
该次会议源于对联合国气候谈判难以在化石燃料退出上达成共识的失望。组织者表示，能源危机反而加强了推动逐步淘汰化石燃料的理由，但也有国家计划在短期内增加煤炭使用以保障供应。
主办方哥伦比亚与荷兰指出，参会的“战略性国家”约占全球化石燃料产量的五分之一、消费近三分之一。澳大利亚、加拿大、挪威等主要产国与安哥拉、墨西哥、巴西、土耳其、越南等将出席，但美、中、沙特、俄等最大生产国缺席。
批评者担心产油方可能影响议程，气候学者比尔·黑尔警告，参与国越多利益越分散，难以形成明确成果。哥伦比亚环境部长伊雷内·委莱斯称，此次会议是打破“禁忌”、建立更诚实对话空间的重要一步；图瓦卢和瓦努阿图等低洼岛国也表示支持。
尽管预计不会有重大宣告，会议建议将纳入巴西牵头的自愿“脱碳路线图”。分析人士认为，能源安全问题与气候目标将同样主导讨论；自COP28以来各国虽承诺转向非化石能源，但落实仍面临强烈阻力。

梅尔茨4月19日在汉诺威工业博览会发言时（与巴西总统卢拉同场）称：“我们将很快在柏林召集国家安全委员会会议。”他指出，该委员会有权在必要时采取即时措施，保障能源安全，确保汽油、柴油和航空燃料等关键产品供应。
梅尔茨称当前市场形势紧张但德国供应安全，如形势恶化“我们已准备好采取措施”，但未说明具体措施。与此同时，受航空燃料储备减少影响，德国汉莎等航空公司已宣布削减航班。

汉诺威展会亦被德国经济增长乏力所笼罩，执政联盟在刺激经济承诺上进展有限。卢拉则称巴西是“不稳定世界中的可靠伙伴”，可帮助欧盟降低能源成本，并批评美方对伊朗的战争政策。两国将于4月20日联合举行内阁会谈，推动原材料、防务、数字化转型与气候等一系列合作项目。
德国政府数据称，巴西是德国在南美的最大贸易伙伴，贸易额超€200亿，且在德国减少对中国稀土依赖的努力中占据重要地位。BLOOMBERG
