## 来源: WorldSpotNews

波士顿大学全球发展政策中心周三发布报告称，2024年中国对非洲贷款近乎腰斩至21亿美元，低于2023年的39亿美元，延续十年下滑态势。自2006年起对非贷款总体上升，2012—2018年受“一带一路”推动曾年均超100亿美元，2016年高峰达288亿美元，但自2020年以来未再超过50亿美元。
报告指出，北京正从向政府提供数十亿美元大额贷款转向更小且更具战略与商业可行性的项目，并更多采用人民币计价以规避美元波动风险。以肯尼亚为例，2024年所有中国基础设施贷款均以人民币计价，并将标准轨铁路未偿债务转换为人民币，预计年偿债成本可减少2.15亿美元。研究同时警示，对与中国贸易中人民币储备有限的国家，这一转变可能引入新的货币风险。
中心“全球中国倡议”研究员岳梦迪表示，这不是撤出而是“再校准”，反映了对2010年代债务可持续性与风险管理的教训。她并称，尽管绿色能源领域的主权贷款仍有限，支持方式可能由直接放贷转向资助前期可行性研究与项目识别，以培育清洁能源生态并吸引私人投资。
中国对非贷款高度集中于安哥拉、埃塞俄比亚、肯尼亚、赞比亚、尼日利亚和埃及六国。自2000年以来，安哥拉一国占总额逾四分之一，约490亿美元；埃塞俄比亚约140亿美元。2024年安哥拉仍获最大份额14.5亿美元，其中7.6亿美元用于输电线，6.9亿美元用于卢安达周边的房地产、公路与港口等基础设施。全大陆仅有六个项目获批，单笔贷款无一超过10亿美元。报告称，十亿美元项目时代正在消退，中国正在进入更为选择性的参与阶段。
