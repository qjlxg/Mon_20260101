新规将适用于欧盟委员会认定的18个关键领域，包括检测设备、互联和自动化车辆、电力供应与储能系统、供水系统，以及无人机和反无人机系统；云服务、医疗设备、监控设备、航天服务和半导体也被列为关键领域。

欧盟负责科技事务的委员维尔库宁（Henna Virkkunen）在声明中指出，随着新的网络安全一揽子方案出台，欧盟将具备更充分的工具，以更好地保护关键的信息与通信技术供应链，并更有力地应对网络攻击。
对此，华为一名发言人指出，基于供应商原产国、而非事实证据和技术标准来限制或排除非欧盟供应商的立法提案，违反了欧盟关于公平、非歧视和比例原则的基本法律原则，也违背了其在世界贸易组织（WTO）框架下的义务。
发言人还说，华为将密切关注相关立法进程的后续发展，并保留一切权利以维护自身合法权益。
早在2020年，欧盟已推出5G安全“工具箱”，以遏制华为等被视为高风险供应商的使用，理由是担忧破坏或间谍活动。然而，由于更换成本高昂，一些成员国至今仍未完全移除相关设备。

观点：能源转型的下一个大挑战是系统整合
未来十年不仅由新增清洁能源装机定义，更取决于电网、储能与需求侧系统能否高效衔接与优化。过去十年，讨论集中在清洁技术能否规模化——现在这一问题正得到肯定。光伏、风电、电池和电动汽车已成为主流，成本更低、效率更高、部署更快；在许多市场，这些已是现实而非未来。
挑战从证明单一技术可行，转向在跨部门的电力与工业系统中整合这些技术，既要满足快速增长的需求，又要保持可靠、可负担和安全。这标志着能源转型进入新成熟阶段，系统设计的重要性已与技术创新并重。该演进正通过三重重叠的浪潮展开。
第一波侧重于规模带来的可行性。过去十年制造与部署推动若干核心技术成本大幅下降，中国的扩张尤为显著：光伏、风电、电池与电动汽车将曾经的高端选项转为可大规模生产的平价产品。随着规模增大、价格下降、供应链成熟与学习曲线加速，结果不仅是排放减少，也重新定义了“可负担的能源”。更便宜的光伏改写了全球电力市场，电池成本下降则使各收入层都能普遍采用电动出行——去碳化从环境选择转为经济选择。第一波证明，脱碳可以与经济增长并行。
