北京市获批启动互联网诊疗首诊试点
中国首都北京市获批开展互联网诊疗首诊试点，从今年1月开始为期一年。明日（2月1日）起，患者可在北京两家医院的部分科室进行互联网首诊。
据央视新闻消息，中国国家卫生健康委星期六（1月31日）回复北京市卫生健康委《关于申请开展互联网诊疗首诊试点工作的请示》，同意北京市卫健委依托首都医科大学附属北京儿童医院和首都医科大学附属首都儿童医学中心，开展儿童生长发育专业、儿童营养专业、儿童皮肤疾病专业互联网诊疗首诊试点。
国家卫健委称，此举是为充分利用信息技术疏解北京非首都功能，满足外地患者就医需求。
国家卫健委也对北京市卫健委提出一些具体要求，包括加强信息化监管和穿透式监管，确保医疗机构的执业信息、人员资质、诊疗数据和活动过程全程留痕、可追溯；将互联网诊疗纳入质量控制体系，确保试点医院遵守医疗质量、医疗安全、处方管理办法、网络安全等有关法律法规和规定。
国家卫健委也提出，要加强对试点医院和医务人员管理。医师开展互联网诊疗应当依法取得相应执业资质，具有三年以上独立临床工作经验，实行“白名单”动态管理；当患者病情出现变化或存在其他不适宜互联网诊疗的情况时，接诊医师应立即终止互联网诊疗活动，引导患者到实体医疗机构就诊；要制定负性事件处置预案，畅通咨询和投诉渠道，及时处置违法违规行为，防范不良舆情发生。
另据新京报报道，从北京市卫健委了解到，从星期六下午开始，首批进行互联网首诊试点的两家知名三甲医院——北京儿童医院及首都儿童医学中心开放预约，患者最快可在2月1日进行互联网线上首诊。
据悉，中国的互联网诊疗一般限制在“复诊”范畴，即必须在某家医院有过首诊记录，方可在之后的复诊中使用该院的互联网诊疗。如今这一限制正试点放开。
