业内将中国BCI初创企业视为可能挑战埃隆·马斯克旗下Neuralink的竞争者。受监管利好和资金推动，相关公司股价周五在内地交易所普遍上涨，深圳上市的英康生命科技（Inkon Life Technology）股价单日涨幅超过10%。
国家药监局表示，此次对纽莱科的批准，标志着全球首次允许有创脑机接口作为商业产品面向患者销售和使用。纽莱科成立于2011年，为私营企业。
公司创始人徐宏来拥有清华大学生物医学工程博士学位。上月，纽莱科与中信证券签署了科创板首次公开发行辅导协议，已启动上市进程。
纽莱科获批的植入式手部运动功能系统为一枚硬币大小的无线装置，安放于脑表外侧而不穿透脑组织，旨在读取神经信号并将其转化为手部动作，帮助脊髓损伤患者恢复部分运动能力。

受全球储能需求上升带动，中国锂电龙头宁德时代（CATL）业绩亮眼，推动港股及内地电池股周三持续上扬。
宁德时代周三早盘股价最高涨逾10%至608港元，为五个月高位，收盘报599.50港元，单日上涨9%，此前周二亦涨9%。
公司本周披露，2025年净利润同比增长42%，达到722亿元人民币；旗下储能业务毛利率高于其电动车电池主业。首尔咨询机构SNE Research称，宁德时代已连续五年位居全球储能电池出货量首位。
在香港举行的Battery Show Asia 2026上，宁德时代总法律顾问权俊洙表示，目前公司业务中电动汽车电池与储能电池占比分别约80%与20%，未来有望向50:50转变，长期储能或将超越车用电池。
分析指出，全球光伏和风电等间歇性可再生能源加速部署，以及以人工智能、云计算和数字化为动力的数据中心爆发式增长，正共同推动储能需求攀升。

中国电动汽车厂商蔚来（Nio）预计2026年第一季度交付量将激增至8万辆至8.3万辆，同比增长90%至97%，显示出在刚实现首个季度盈利后强劲反弹，亦在多数国内对手遇冷的背景下逆势而上。
