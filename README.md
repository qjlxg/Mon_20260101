华盛顿讯 — 美国总统唐纳德·特朗普在4月25日晚白宫记者协会年会发生枪击未遂约一小时后，于白宫记者会上将此次事件解读为其“影响力”的证明。许多出席晚宴的约2,600名宾客当时被迫趴地，特勤局护送总统和其他要员撤离。

特朗普在记者会上既表示慰问也迅速将焦点移回自身与政绩，强调正因其政策改变国家才成为攻击目标，并再次为他主导兴建的价值4亿美元（约5.11亿新元）白宫宴会厅辩护，称该厅将有地下掩体、防无人机屋顶与防弹玻璃，声称特勤与军方需要该设施，但未提供证据。

特朗普还提到2024年两次未遂刺击（宾州巴特勒集会受伤、佛州高尔夫球场开枪被捕事件），并希望此案促使民主党放弃对移民执法的额外监督并批准国土安全部新拨款。报道：路透社

华盛顿——白宫记者协会晚宴外发生枪击后，美方执法机构正在重新审视总统安保部署。
据报道，4月25日晚，枪手在华盛顿希尔顿酒店附近开火，但联邦特勤等人员在其进入酒店地下层（总统原定演讲地点）前将其制止，未造成对总统及主要官员的直接伤亡。尽管如此，部分宾客能听到枪击击中一名特勤人员的声音，暴露出安保薄弱环节。
当晚仅在宴会厅入口设置了金属探测门，进入酒店则只需持票。有消息称有人尝试用2025年票入场；嫌疑人据称数日前已在酒店登记并持多把武器突袭，绕过了基本防线。
前特勤人员与高层官员表示，事后审查可能要求将安检与金属探测设备前移、扩大外围管控，即便会给公众和酒店带来不便；同时需加强各保护机构之间的疏散协调。事发后，不同执法机构分别护送宾客撤离，显示应对存在不够统一的问题。
视频与音频分析显示，最后枪声后约30秒特朗普被迅速带离，而卫生与公共服务部长离场用时约100秒，国务卿和五角大楼官员约150秒。
特朗普事后称华盛顿希尔顿“并非特别安全”。此前2024年竞选期曾发生两起针对特朗普的刺杀未遂，已促使安保加强。枪手在一份宣言中也批评现场安保松懈。部分保守派与官员借机呼吁在白宫地面增建宴会厅，但联邦法庭曾以未经国会批准下令暂停该工程，后在上诉中暂缓执行。
此外，有官员指出，自今年2月伊朗冲突升级以来，部分内阁成员的安保已被加强，未来围绕总统与内阁的安保评估与调整仍将继续。

华盛顿消息——4月25日晚白宫记者协会晚宴发生枪击后，社交媒体迅速被各类阴谋论和指责淹没。
在此类突发事件中，意见领袖和网红常以炒作填补信息真空，吸引注意力与粉丝。此次在X、Facebook、TikTok等平台上，左右两端的用户都参与制造混乱，出现“为政客或其他人策划、以转移视线”为由的“作秀”说法，也有人无证据地将枪手与以色列相关立场挂钩，并用疑似经AI处理的图像佐证，俄媒RT在X上放大了部分说法。
多人错误传播称枪手当场被击毙，实际上其被逮捕；事后虽有澄清，但阅读量远不及错误信息。类似现象此前也出现在对特朗普的暗杀未遂及委内瑞拉前总统马杜罗被捕的报道中。
密歇根大学信息学院副院长Cliff Lampe指出，人们倾向寻找能证实既有偏见的信息，而非客观证据。康涅狄格大学研究大规模枪击与阴谋论的Amanda Crawford说，核实事实需要时间，但公众缺乏耐心，空白处由迎合情绪的叙事迅速填满。
事件发生后，特朗普线上积极参与，将此事用于支持其在白宫场地建造“金碧辉煌舞厅”的主张，右翼网红广泛转发相关说法（晚宴实际在华盛顿希尔顿酒店举办）。个别网红如Mario Nawfal在X上发布多种无根据猜测后又声明并不认同，但此类内容仍获高流量。
福克斯新闻转播的一段现场电话连线因中断被部分用户指责为遭到压制，连线记者Aishah Hasnie随后在X上解释当时所在球厅信号极差。
专家认为，社交平台上的注意力与收益驱动了谣言传播，使得真相常被延迟或淹没。

斯德哥尔摩——斯德哥尔摩国际和平研究所（SIPRI）4月27日发布报告称，2025年全球军费开支接近2.9万亿美元，已连续第11年上升。美国、中国和俄罗斯三国合计开支约1.48万亿美元，占全球总额逾半。
报告显示，尽管美国军费下降7.5%至9540亿美元（主要因未批准新的对乌军事援助），全球军费仍较2024年增长2.9%。SIPRI研究员洛伦佐·斯卡拉扎托表示，美方减少被欧洲和亚洲开支上升所抵消，全球“军费负担”（占GDP比重）升至2009年以来最高，反映安全感下降。
地区方面，欧洲（含俄乌）为增长主因，军费增14%至8640亿美元。德国上升24%至1140亿美元，西班牙跃升50%至402亿美元，军费占GDP首次自1994年以来超过2%。俄罗斯增5.9%至1900亿美元（占GDP约7.5%），乌克兰增20%至841亿美元（约占GDP 40%）。
中东整体仅微增0.1%至2180亿美元，伊朗名义值因高通胀实际下降至74亿美元，以色列因停火后强度减弱降4.9%至483亿美元，但较2022年仍增长97%。
亚太及大洋洲军费增8.5%至6810亿美元，中国为主要驱动，约3360亿美元；日本增9.7%至622亿美元（占GDP 1.4%，为1958年以来最高），台湾增14%至182亿美元。
