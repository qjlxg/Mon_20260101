机构：特朗普拟上调汽车关税，或使德国产值损失近 180 亿美元
一家经济机构周六分析称，美国总统唐纳德・特朗普宣布对欧盟汽车和卡车加征关税，或导致德国产值损失近 150 亿欧元（约 175.8 亿美元）。基尔世界经济研究所（IfW）的这一估算，凸显了欧盟最大经济体对美国进口关税的脆弱性 —— 此类关税已令德国汽车业蒙受数十亿欧元损失。该研究所分析称，“影响将十分显著”，长期来看，产值损失将升至约 300 亿欧元。基尔世界经济研究所所长莫里茨・舒拉里克表示。特朗普周五宣布，下周将把欧盟汽车关税从此前商定的 15% 上调至 25%，理由是欧盟未遵守与华盛顿达成的贸易协定。基尔世界经济研究所经济学家朱利安・辛茨称：“德国本就低迷的经济增速将遭受重创。”该研究所目前预计，德国经济今年将增长 0.8%。研究所补充道，意大利、斯洛伐克、瑞典等汽车产业规模较大的欧洲经济体，也可能蒙受重大损失。德国经济部长首席顾问延斯・苏德库姆建议对特朗普的举动保持谨慎。苏德库姆向表示：“欧盟目前只需静观其变。众所周知，特朗普动辄撤销或撤回其高调的关税威胁。”他指出，特朗普必须说明为何认为欧盟未遵守现有贸易协定，且最新关税威胁是否具备法律依据也尚不明确。这位顾问称：“这一切似乎都相当冲动。”

跌幅最大的是由军政府掌控的尼日尔（第120位，下跌37位），RSF表示这反映了萨赫勒地区近年来在武装团体和军政府打压下，获取多元平衡信息权利的全面衰退。AFP报道。
