中国去年整治内卷式竞争十大案例 货拉拉列首席
中国国家市场监督管理总局公布去年整治内卷式竞争十大案例，科技物流平台货拉拉位列首席。
中国市监管总局星期三（1月21日）在微信公众号“市说新语”通报上述情况。
