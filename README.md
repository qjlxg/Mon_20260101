全球变压器短缺 中国出口额创历史新高
去年，随着数据中心快速建设和电气化进程加速，全球电力变压器供应趋紧、价格上涨，中国从电力变压器出口中斩获创纪录收益。
彭博社报道，据中国海关总署星期天（1月18日）公布的数据，去年中国变压器出口额达到创纪录的646亿元人民币（119.26亿新元），同比增长近36%。变压器用于将高压电转换为低压电，供家庭或工厂使用。
出口额大幅增长之际，全球正面临电网基础设施所需关键设备的短缺。能源咨询公司伍德麦肯兹估计，作为全球最大的变压器买家，美国面临约30%的供应缺口，这一情况可能持续至2030年代。
随着海外电网升级改造订单增加，以及中国对以可再生能源为主的超级电网加大投资，中国制造商从中受益。
海关总署的数据显示，去年中国出口的单台变压器平均价格上涨至20.5万元人民币，同比上涨约三分之一。彭博新能源财经称，在美国，受制造产能持续紧张影响，变压器价格比2021年之前的水平高出约79%，欧洲和澳大利亚也面临类似压力。

韩国有逾320万人选择“尊严死去”
韩国最新官方数据显示，已有超过320万名韩国人正式登记，在罹患不可治愈疾病时选择放弃维持生命治疗。
综合韩国《先驱报》和亚洲新闻网络报道，韩国国家生命维持治疗管理机构星期一（1月19日）公布的数据显示，截至去年12月，已有320万韩国人登记参与临终医疗决定制度，其中女性约213万人，男性约107万人，总登记人数持续上升。
这项制度常被称为允许“尊严死去”。65岁及以上长者占登记总人数逾三分之二，约237万人。
制度自2018年2月实施以来，登记人数快速攀升：首年约8万6000人，2021年8月破百万，2023年10月达200万人，并于2025年8月突破300万人。

现行法律仅允许已进入生命末期的患者，决定放弃维持生命治疗。根据韩国法律第二条的定义，这类患者须患有不可治愈疾病、预计数月内死亡，且无康复可能。
去年，执政党共同民主党国会议员南仁顺提出修法，主张放宽门槛，将尚未进入终末期、但被评估为病情不可逆、明显走向死亡的患者，也纳入适用范围。
若修法通过，患者可在疾病较早阶段作出放弃维持生命治疗的决定，不需被迫等到临终阶段。

中国制定发布中医体质分类与判定国家标准
中国官方制定并发布了中医体质分类与判定国家标准，明确中医九种体质类型以及有关判定方法。
新华社星期一（1月19日）引述中国国家市场监督管理总局报道，市场监管总局（国家标准委）近日批准发布《中医体质分类与判定》推荐性国家标准。这一标准由中国国家中医药管理局组织起草，将自4月1日起正式实施。
《中医体质分类与判定》规定了中医九种基本类型的特征、中医体质分类的判定，主要具有三方面特点：第一，应用场景全域覆盖。标准适用于医疗、公共卫生、科研、教学及健康产业全领域，为治未病中心、体检机构、基层医疗等提供判定依据；第二，体质特征循证革新。根据2009至2024年的最新中医体质研究结果，对平和质、气虚质、阳虚质、阴虚质、痰湿质、湿热质、血瘀质、气郁质和特禀质等中医体质的特征表述，选择了具有较好循证证据的最新结果作为依据。
第三，确定了体质辨识方法。标准中将复杂专业的体质类别判定通过选取代表不同体质的典型体感问题，并匹配相应的等级计分算法，形成了标准化的判定方法，促进了体质辨识的开展与推广。
报道称，标准的发布实施，标志着中医体质学走向更加规范化、标准化的发展新阶段。

北京人工智能公司智谱AI宣布，其新图像生成模型GLM-Image全部在华为计算栈上训练完成，成为首个在全国产训练链上开发出的主要开源多模态模型。
公司表示，GLM-Image的整个训练流程——从数据准备到最终训练——均在华为Ascend Atlas 800T A2服务器上进行，采用华为自研的Ascend系列AI处理器和MindSpore框架。智谱称此举证明在美方限制高端芯片出口的背景下，国内算力可支撑强大多模态模型的开发。
该模型采用自回归与扩散混合架构，具备图文联合生成能力。智谱称其在开源模型中文字生成与文本呈现任务中取得行业领先的自报成绩，但仍落后于字节跳动的闭源图生模型Seedream 4.5；字节未披露其训练所用芯片。
尽管华为Ascend对小型模型训练表现良好，但能否用于打造如下一代GLM-5这类旗舰大模型尚无定论。当前国产芯片尚难撼动英伟达在前沿模型训练中的主导地位。此前英伟达游说下获准向中国出售H200芯片，但据路透报道，中国方面已指示海关限制该芯片进口，并仅在例外情况下批准H200订购。
此外，北京鼓励企业采用MindSpore等国产机器学习框架替代TensorFlow、PyTorch，但MindSpore的总体采用率仍低于美国主流框架。
