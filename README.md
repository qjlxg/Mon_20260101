张国清调研雅鲁藏布江下游水电工程 吁规范用好资金
中国国务院副总理张国清上周在西藏林芝调研雅鲁藏布江下游水电工程建设等情况，要求压紧压实安全生产责任措施，确保进度服从质量和安全，集约规范用好资源资金，打造廉洁工程。
雅鲁藏布江下游水电工程（简称雅下水电工程）是世界最大最贵的水坝工程，去年7月19日在西藏林芝市开工，中国总理李强当天出席开工仪式，要求确保生态环境不受损害，把雅鲁藏布江下游水电工程建成重大标志性工程。
为保障雅下水电工程顺利建设运营，官方成立新央企中国雅江集团。副总理张国清去年7月19日出席在北京举行的成立大会，并为公司揭牌。
据新华社报道，雅下水电工程主要采取截弯取直、隧洞引水的方式开发，建设五座梯级电站。它的总投资约1.2万亿元人民币（2243.7亿新元），是三峡大坝建设成本（2542亿元人民币）的五倍。工程生产的水电以外送消纳为主，兼顾西藏本地自用需求。
张国清上个星期五至星期天（4月10日至12日）在西藏林芝调研和雅下水电工程，调研中国雅江集团组建运营和雅下水电工程建设等情况。他先后来到工程现场和集团总部，实地了解施工进展、装备运行、工程调度、公司管理、生活保障等情况，与技术骨干、施工人员交流。
张国清强调，雅江集团要以对历史、对人民高度负责的态度，坚决扛牢工程建设主体责任，把雅下水电工程建成新时代的重大标志性工程。
他并要求雅江集团坚持质量为先、安全为本，严格落实工程建设标准规范和生态环保要求，强化对工程设计、施工组织、材料供应等全过程质量管控，压紧压实安全生产责任措施，确保进度服从质量和安全。
要坚持中共领导、加强党建，全面提升公司运营管理能力，构建智能化指挥调度系统和穿透式监管系统，健全科学高效的运行机制，集约规范用好资源资金，打造廉洁工程。
张国清还要求各有关方面强化一盘棋思想，为工程建设和公司运营提供有力支持保障，共同服务国家战略、促进西藏经济社会发展。
调研其间，张国清还到林芝消防救援机动支队了解队伍建设、保障施工安全等情况。他强调，要着眼于提升高原复杂环境下应急救援攻坚能力，常态化开展实战实训，前置先进适用装备，快速响应、高效救援，全力保障民众生命财产安全。

周三，中国人工智能公司智谱AI开源其最新旗舰模型GLM-5.1，并同步将API价格上调10%，显示在与美国对手竞争加剧的背景下，开始加强对高级模型的商业化变现。此次为公司今年第二次涨价：2月对编程订阅的调整曾上调逾30%，本次幅度较小但覆盖包括云端输入token在内的多项计费类别。消息发布后，智谱港股盘中上涨约11.5%。

智谱定位为中国硬件生态中的重要软件合作方，并称已与多家国产芯片企业合作适配其GLM模型，包括华为、摩尔线程与寒武纪。摩尔线程表示其MTT S5000服务器已实现对GLM-5.1的“Day-0”兼容性。
智谱CEO张鹏在3月中关村论坛表示，依赖低价竞争不利行业长期发展，提价旨在建立“更健康的商业闭环”，支持高质量AI服务的长期交付；小米AI模型负责人罗福立也表达了类似看法。

美国联邦通信委员会（FCC）周四表示，正考虑更广泛的措施，禁止中国电信运营商在美运营数据中心，并进一步限制其接入美国网络与基础设施。草案同时建议禁止美国运营商与这些被认为存在“重大国家安全风险”的公司互联互通。中国移动、中国电信和中国联通均被列入受限实体名单，可能被要求出售、转让或关闭在美的数据中心和网络节点，甚至被迫退出美国市场。
