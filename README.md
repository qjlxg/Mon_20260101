中国研推广部分地方”扫码飞“以解决飞行审批难
中国官方正研究推广部分地方”扫码飞“等经验和做法，以解决无人机飞行活动审批难的问题。
据中国网报道，中国国家发展改革委低空经济发展司司长郑剑，星期五（4月17日）在新闻发布会上说：“我们注意到社会舆论反映存在无人机飞行活动审批难的问题，我们正在与有关部门，研究推广部分地方‘扫码飞’等好经验好做法，提高飞行计划审批效率。”
随着中国近年无人机保有量快速上升，扰航事件频繁出现，民航安全压力加大，无人机监管逐渐收紧，2026年开年的一系列措施被舆论视为最严无人机监管新规。
《财经》杂志曾在2月报道，新规落地过程中，监管方式与复杂的现实需求产生摩擦，审批的“最后一公里”暂未打通，部分规定缺乏梯度与精度。
新规要求管制区等空域飞行必须报批，然而报批流程繁琐、审批慢、通过难等问题，让合规飞行变得困难。不少用户选择搁置或者出售手中的无人机，市场一度出现抛售潮，并直接影响整体销量。
新规发布后，部分大疆经销商门店的销量直线下降，有经销商近两个月的无人机销量下降了近50%。二手无人机市场也受到影响，一名二手大疆无人机企业负责人说，店内无人机销量大幅下滑，咨询卖飞机的客户却多了起来。在闲鱼、转转等二手平台上，多个品牌的无人机转卖量大幅上涨，价格明显回落。
不过，江西南昌、上海、广州市白云区等多地开始通过搭建地方政务小程序平台，集成无人机飞行报备服务，以提升审批效率与便民程度。
在各地的飞行报批平台搭建探索中，江西南昌受到的好评尤其多。多个航拍摄影师实测，在南昌通过“昌通码”申报无人机飞行，提前五分钟报备即获秒审通过。即使是管制空域特殊飞行，审批只需两小时。
不少用户也反映，当前不少地方政府提供了小程序平台等飞行审批渠道，但审批效率依旧很难，审批通过率低，“飞行难”问题依旧没有完全解决。
《财经》也报道，当前中国无人机保有量已经超过500万架。2025年实名登记无人机总数突破328万架，累计飞行4530万小时，同比增长近70%。

无偿献血可优先用血 中国400毫升献血证被炒到2000元
中国媒体爆料，有中介组织外来务工人员卖血后，将献血证倒卖给有用血需求的患者家属。经中介层层加价后，原本对应400毫升献血的证件，可被卖到高达2000元（人民币，372.6新元）一张。
上游新闻报道，有中介在江苏等地通过社交平台发布“献血兼职”信息，组织人员跨省前往上海献血。参与者每献400毫升全血，大约可以拿到400元左右补贴，甚至提供接送和简单食宿。这些人多为外来务工人员，对献血证后续用途并不完全清楚，只是把它当成一份短期收入。
报道称，中介拿到献血证后，转卖给需要用血的患者家属。在医院里还有人发放所谓“爱心血互助”小卡片，引导家属通过中介购买献血证。经过中介层层加价，一张原本对应400毫升献血的证件，最终被卖到约2000元。
此外，中介会教唆不符合条件的人“吃药后再去献血”，甚至暗示通过“红包”等方式蒙混过关，以确保能顺利拿到献血证。
上游新闻指出，这种倒卖已经不是个别现象，而是形成了跨省运作的产业链，即中介上游招募“献血兼职”，然后转手倒卖，下游是急需用血、愿意花钱购买“优先用血资格”的患者家属。
中国多城市有供血紧张的问题。极目新闻指出，要彻底斩断这一非法牟利的链条，杜绝“鲜血变商品”，就要从根本上解决血液供需失衡的问题。
根据《中华人民共和国献血法》，无偿献血者及其直系亲属在一定额度内享有免费或优先用血权利。通常献血者本人可无限量或累计献血量倍数免费用血，直系亲属（配偶、父母、子女）可等量或一定比例免费，且同等条件下优先用血。

伊朗称霍尔木兹海峡开放；特朗普称“很快”可达成协议

墨西哥城湖区发现上千骨骸 令世界杯前景再添阴影
墨西哥城——距世界杯开幕仅数周，墨西哥城湖区附近发现大量人骨碎片，引发对该国毒品战争遗留问题的关注。寻找失踪亲属的志愿者团体称，在查尔科湖（Lake Chalco）及周边搜出的骨骸超过1000片，部分出现在此前已被官方检查过的区域，反映出“一场无法估量的法医危机”。团体指责当局希望淡化此事，家属则要求向世界揭示首都发生的悲剧。
市检方4月13日公布，已在湖滨地带发现约300块骨片，怀疑可能属于三人；当局上周已开始进行挖掘。抗议者4月17日与市府会面，要求持续不断地搜寻直至彻底勘查完毕。
自2006年政府动用联邦军队打击贩毒集团以来，墨西哥已有逾48万人遇害、13万人失踪。联合国专家委员会称失踪危机为“反人类罪”，并批评公职人员的消极与失责；总统克劳迪娅·谢因鲍姆对该报告提出反驳，称忽视了新出台的援助政策。
墨西哥城与瓜达拉哈拉将于6月举办世界杯赛程，国内两城均有示威者抨击政府未能妥善调查失踪案。美加为本届杯赛共同主办方。 (法新社)

基辅消息——乌克兰总统泽连斯基4月17日在Telegram上表示，依据乌军总指挥奥列克桑德尔·谢尔斯基提供的情报，乌方认为俄方正在准备，可能再次尝试将其盟友白俄罗斯卷入与莫斯科的战争。
泽连斯基称，情报显示白俄边境地区存在通往乌克兰的修路工程与火炮阵地建设，乌方因此认为俄方会再次动用白俄。他表示已向白俄领导层发出警告，强调乌克兰准备捍卫领土与独立。
他还指出情报表明俄方正在进行部队重组，可能是为弥补兵员短缺，这也解释了俄在白俄境内活动的增加，但未提供更多证据。
白俄总统卢卡申科是克里姆林宫的亲密盟友，曾在2022年允许俄军利用其领土发动对乌入侵。卢卡申科虽称不打算派兵参战，但已同意在境内部署俄方战术核武器及高超音速“奥雷什尼克”导弹，并长期因支持战争与人权问题遭西方制裁。
美国方面正寻求改善与白俄关系并促成释放被视为政治犯的在押者——3月已释放约250人；美方同意取消部分制裁，但明确要求白俄不得协助规避制裁或支持对乌战争。来源：路透社

华盛顿，4月17日 — 白宫周五与人工智能公司Anthropic首席执行官达里奥·阿莫代（Dario Amodei）会面，双方就合作进行了首次公开讨论，此前数月美军与该公司围绕AI用途发生争执。
此次会谈正值外界对Anthropic最新模型“Mythos”可能加剧网络攻击风险的担忧升温之际。中央银行、企业界及监管机构纷纷加快对该模型的评估，认为银行业因长期遗留系统与新技术并存而尤为脆弱。至少美、加、英三国政府已与主要银行高层就Mythos带来的威胁举行会谈。
白宫发言称，财政部长斯科特·贝森特（Scott Bessent）与幕僚长苏西·怀尔斯（Susie Wiles）参加了与阿莫代的会谈，双方就合作机会、应对技术扩展带来的挑战的共同行动与协议进行了讨论，认为会谈“富有成效且具有建设性”，并将与其他AI公司继续对话。Anthropic也表示会谈富有成效，讨论了网络安全、美国在AI竞赛中的领导地位及AI安全等共同优先事项。
Anthropic于4月7日宣布推出“Mythos”，并在名为“Project Glasswing”的受控项目下向部分企业提供尚未公开的Claude Mythos Preview模型，用于查找网络安全漏洞。公司称该模型在编码与自主代理任务方面能力最强，但专家警告其高阶编码能力可能前所未有地识别并利用漏洞，给金融机构带来风险——Guardrail Technologies首席执行官TJ Marlin指出，银行业的遗留系统可能暴露大量弱点。
早在Mythos推出前，美方与Anthropic已就AI使用原则存在分歧。五角大楼对该公司施加了供应链风险限制，因其拒绝移除防止将AI用于自主武器或国内监控的限制。特朗普曾下令联邦机构停止使用Anthropic工具，并在社交平台上严厉抨击该公司；Anthropic于3月对国防部的“黑名单”决定提起诉讼。被记者在凤凰城问及此次会谈时，特朗普回应称“我不知道”。
（路透社）

乌克兰总统泽连斯基周五呼吁各国共同行动，组建一支有效机制以保障霍尔木兹海峡航行自由，并表示基辅在黑海的战时经验可以提供帮助。
他在由法国与英国主持、50国参加的视频会议中表示，目前对霍尔木兹的决定将影响其他强势行动者是否会在其他海峡或战线制造麻烦，强调必须尽可能具体明确，避免六个月后陷入类似加沙的局面。
泽连斯基还称，霍尔木兹面临的安全挑战并非仅靠政治决定即可解决；乌克兰在与俄罗斯四年战争中已在黑海执行过类似任务，具备护航商船、排雷、防空以及协同指挥等实战经验。
他并说，乌克兰已派专家遍及中东，协助各国应对由伊朗设计的俄制无人机，愿为海上安全贡献力量。乌克兰已与沙特、卡塔尔、阿联酋签署安全合作协议，并与阿曼、科威特、巴林保持磋商。

喀麦隆杜阿拉 — 教宗利奥十四4月17日在喀麦隆警告，人工智能热潮可能助长“对立、恐惧与暴力”，并指向现实被其模拟逐步取代的危险。此次表态发生在教宗与美国总统特朗普争论不断之际——特朗普此前发布后又删除了一张疑似将其塑造成耶稣形象的AI合成图片，引发宗教界谴责。
在杜阿拉为逾12万名信众主持弥撒后，教宗在首都雅温得的中非天主教大学对师生发表讲话，称AI带来的挑战超出表面，不仅是技术使用问题，更是我们与真相关系的转变，可能导致极化与冲突。
本次为期11天的非洲之行中，教宗多次就世界和平、权力滥用和环境问题发表直言。此前他批评美以对伊战争，遭特朗普反击并指其“对犯罪软弱、外交糟糕”。教宗还抨击“少数暴君”祸害世界，亦引来特朗普回击。
教宗在喀麦隆受到热烈欢迎，民众挥舞“和平之枝”和梵蒂冈旗帜。有民众称被信息感动，另有人称如愿一睹教宗终生夙愿。
同时也有人担忧此次访问可能美化执政自1982年的保罗·比亚总统的形象。去年10月其连任后，杜阿拉等地曾发生示威被严厉镇压，目击者称安全部队开枪，官方承认数十人死亡。
教宗亦谴责为支撑AI发展的稀土等矿产开采造成的环境破坏，要求结束采矿业的腐败与外力掠夺，点名对非洲资源有重要影响的外国势力（以中国为首）。他在4月15日抵达后敦促喀麦隆领袖肃清腐败与以秩序之名的暴行。
天主教在喀麦隆社会中影响深远，全国3000万人中逾三分之一为天主教徒。利奥十四将于4月18日清晨在当地举行弥撒后启程前往安哥拉，最终抵达赤道几内亚，结束横跨1.8万公里的行程。此前他还访过阿尔及利亚，该行程曾遭两起自杀式爆炸袭击。

利马，4月17日 — 随着总统选票计票进程缓慢并出现被指不规范的情况，要求撤换秘鲁选举机构国家选举办（ONPE）负责人皮耶罗·科尔韦托的呼声周五升温。由于尚未出现能在6月第二轮挑战保守派领跑者凯科·藤森的明确对手，局势更加不确定。 
ONPE数据显示，截至周五已计票93.3%，藤森以约17%领先；左翼的罗伯托·桑切斯以12.0%紧随其后，极右前利马市长拉斐尔·洛佩斯·阿里亚加以11.9%位居第三，二者相差约1.3万张选票。地方观察组织Transparencia表示，最终结果可能需要多达两周才能公布。 
ONPE称约5%的选票因投票站记录缺失或错误被标记须复查，这部分选票将由特别选举陪审团审定后并入最终计票。企业界和跨党派议员纷纷要求科尔韦托下台，商会CONFIEP主席豪尔赫·萨帕塔警告称，“如此严重的错误是有后果的”。 
科尔韦托本周承认在利马等地曾出现后勤延误，致使投票延长一天，但否认存在舞弊。洛佩斯·阿里亚加曾指控舞弊并要求暂停计票。国家选举法庭已向检方对科尔韦托提起刑事控告，指称包括侵犯投票权在内的违规行为。科尔韦托方面尚未回应路透社询问。 
警方表示，周四在利马一条公路上发现四个投票站的选举材料，目前正调查此事；ONPE称这些站点的票数已被记录。欧盟观察团本周未发现舞弊证据。 
市场方面，桑切斯上升为第二名曾引发对其拟改写宪法和加强国有资源控制的担忧，但对2026–2031届国会可能偏右的预期部分缓解了投资者顾虑。周五午间索尔币小幅波动，利马基准股指上涨0.4%。 
秘鲁过去十年内政局动荡，经历八任总统，但经济仍是拉美较为稳健的之一。来源：路透社
