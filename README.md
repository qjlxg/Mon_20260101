韩国失业人数虽下降 长期失业人口却暴增近四成
虽然韩国整体失业人数有所回落，但失业六个月以上的长期失业人口却在4月同比大增超过三成，在整体失业者中所占比重创下22年来同月最高纪录。
其中，15岁至39岁人群占长期失业者近五成七，显示长期失业问题在较年轻年龄层中更为突出。韩国国家数据处分析认为，中东战争的影响开始波及就业市场，可能推高长期失业人数。
韩国国家数据处国家统计门户网站（KOSIS）与就业动向数据显示，今年4月韩国失业人数同比减少2000人，降至85万3000人。不过，失业六个月以上的人数同比增加37.6%，达到10万8000人。
长期失业者在整体失业人口中的比重升至12.7%，创下2004年同月13.6%以来最高纪录。
韩联社对经济活动人口调查微数据分析发现，截至上月，失业六个月以上人员中，15岁至29岁青年为2万9000人，30多岁人群为3万2000人，两者合计6万1000人，占长期失业人员总数的56.5%。
针对这一趋势，韩国国家数据处分析认为，中东战争爆发的影响开始波及就业市场，可能成为长期失业人员增加的因素之一。

马英九基金会专案组调查出炉 无证据证明萧旭岑王光慈私吞财物
马英九基金会风波持续延烧三个月，针对前执行长萧旭岑、王光慈涉嫌违反财政纪律案，基金会三人专案小组的调查报告星期天（5月24日）出炉，表示已完成调查报告，明确指出无证据证明萧旭岑、王光慈私吞财物，也无法证明两人违反财政纪律。
马英九基金会风波于今年3月爆发、并在5月急剧升级为台湾政坛焦点事件，核心围绕前总统马英九、马英九重要幕僚金溥聪等，与马英九多年心腹旧属萧旭岑、王光慈之间的“违反财政纪律”指控，并牵扯出关于马英九是否失智及家族内讧的罗生门。
马英九今年3月以董事长身份调整马英九基金会人事，撤换追随他10多年的原执行长萧旭岑与副执行长王光慈等核心幕僚，改由戴遐龄接任。马英九一方指控萧、王二人涉嫌侵占与背信，存在严重财务纪律问题，并扬言移送司法调查。
萧旭岑对此坚决否认，称自己从未经手财务，并质疑马英九遭“有心人”（指重返马系核心的前国安会秘书长金溥聪等人）挑拨操控。
据联合新闻网报道，三人小组发布声明指出，此次调查程序以基金会在4月20日寄送给董事的纸本资料、即内部调查资料与指控事实作为基础，展开历时一个月的调查，其中包括进行10多次调查会议，以及访谈六组人员的严谨过程。
三人小组表示，并无具体及确切的客观证据，能证明萧旭岑、王光慈人有私吞财物之情事。加上萧旭岑并未经手基金会财务；负责财务的王光慈也提出完整的相关账务纪录和凭据，多可证明两人被指控的现金使用于董事长马英九、机要及随扈等的公务用途，因此无法证明萧、王两人违反财政纪律。
三人小组表示，依据董事会决议，三人小组应于调查完成后报请董事会讨论，再由董事会采取适当处理措施，包括对社会大众说明。
不过，三人小组也指出，根据上个星期五（22日）多家媒体报道，马英九的长姐马以南已委任律师龙毓梅向法院声请，寻求裁定马英九需要辅助宣告，由辅助人替他处理事情；台北地方法院也证实确有此情。
据此，三人小组建议应等待台北法院对辅助宣告的声请做出判断后，再召开董事会商讨调查报告。
辅助宣告是台湾为保护精神障碍或心智缺陷（如早期失智症、智能障碍等）导致判断、表达能力显有不足的成年人，所设立的法律保护制度。
三人小组重申，调查过程并不顺畅，包含索取资料未获基金会完整提供，拟约谈人员也未全数配合约谈。
他们举例指出，自称受马英九授权办理萧、王涉嫌违反财政纪律事件的金溥聪，自始至终并未出示授权书。

极端热浪席卷印度多地 气温超45度夺16命
印度大片地区持续受热浪笼罩，南部特伦甘纳邦已至少有16人因中暑死亡。
随着部分地区气温持续超过45摄氏度，当地官员呼吁提高警惕，防范高温对公共健康造成影响。
法新社报道，特伦甘纳邦税务部长雷迪发声明说，本轮高温强度已达到前所未有的水平，并要求官员提前发布预警信息。当地政府同时建议年长者、儿童及孕妇等高风险群体，除非必要，否则避免在白天外出。
卫生专家指出，极端高温可能导致人体严重脱水，使血液变得黏稠，严重时甚至会引发器官衰竭。
印度气象局本周稍早预测，全国多个地区将出现高于常年的气温及强烈热浪天气。首都新德里及周边城市本周气温持续超过40摄氏度，推动电力需求升至历史高位。与此同时，夜间最低气温也维持在较高水平，民众几乎难以从酷热中获得喘息空间。

印度是全球第三大温室气体排放国，发电高度依赖煤炭。印度承诺于2070年实现净零排放目标，较多数西方工业化国家晚约20年。
官方数据显示，印度有记录以来最高气温为51摄氏度，于2016年在拉贾斯坦邦法洛迪测得。

尼科西亚/5月23日 — 国际货币基金组织（IMF）在一份提交给欧盟财政部长的报告中警告称，未来15年欧盟各国将在国防、能源和养老金方面面临巨额开支，如果不采取行动，公共债务将走上不可持续道路。

具体措施包括：提升劳动力在27国间的流动性并鼓励企业雇佣；一体化能源市场；促进居民储蓄跨境流向高回报投资；统一目前各国存在差异的法律；推进养老金改革并提高退休年龄；对低碳与抗气候风险项目提供政府担保以吸引私人资本。
IMF还认为，应把创新、能源与国防视为欧盟公共产品，并通过共同举债来融资。对此，欧盟内部存在明显分歧：西班牙、意大利、法国等支持共同债务，而德国及若干北欧国家强烈反对。欧元区财政部长会议主席Kyriakos Pierrakakis表示，这将是未来数月讨论的议题。
IMF警告称，即便实施改革，多数国家仍需财政整顿以使债务呈下降趋势；改革越有力，所需整顿越少。报告批评“边走边看”的做法已接近极限，碎片化、零敲碎打的调整难以应对不断上升的支出压力。来源：路透社。

民主党人借此将公众生活困境与总统奢侈开支及可能流向其盟友的巨额资助做对比。参议院民主党领袖舒默将共和党形容为为舞厅与所谓“私设基金”而陷入崩溃。国会复会后，双方可能通过为基金设立监督委员会标准、司法审查等“护栏”寻求折中；民主党已拟出多项修正案，包括禁止向攻击执法人员的1月6参与者付款、禁止动用纳税人资金及要求公开支付名单等。来源：路透社
