【德明利股价连续跌停 回应称二季度净利润环比下滑受多因素影响】
7月16日，德明利再次跌停，连续两日“一字”跌停。德明利股价大幅下挫的背后，部分原因或与公司日前披露的2026年半年度业绩预告相关，特别是公司二季度预计净利润出现环比下滑，导致前期乐观预期落空。部分行业人士认为，存储模组类上市公司持续的股价波动背后，是市场对存储模组厂盈利可持续性的担忧——前期低价囤货的红利正在消退，未来采购成本逐步上升，盈利空间可能收窄。16日，针对公司二季度净利润环比下滑、存储行业后市研判等问题，记者致电德明利，公司进行了回应。针对市场关注的公司二季度净利润环比下滑的情况，德明利回应称，二季度净利润表现受多方面因素影响：一是费用端有所抬升，包括研发投入增加、产能建设带来计提增加、股份支付费用提升等；二是公司持续看好后续行业景气度继续向上，积极加大战略储备支撑业务快速扩张，存货战略储备充足；三是客户拓展投入增加，公司企业级存储、嵌入式存储，内存等业务仍处于加速扩张阶段，此类产品认证周期较长、前期适配测试投入较大，且研发费用、销售费用亦有所增加，业务扩张期相应投入需逐步兑现业绩。上述投入均为长期战略布局，并已在二季度开始逐步产生积极贡献，后续将为公司业绩释放奠定扎实基础。(上海证券报)

其中《2026年吸烟（公众卫生）条例（修订附表2）令》指定建筑地盘为禁止吸烟区，包括大厦维修工程地盘。
《2026年定额罚款（吸烟罪行）（指明主管当局及公职人员）（修订）公告》则指明劳工处职业安全主任可依据《定额罚款（吸烟罪行）条例》，执行所有涉及在建筑地盘吸烟罪行的执法工作。
任何人在建筑地盘吸烟即属违法，职安主任可向违例者发出定额罚款通知书，定额罚款3000港元（493新元）。地盘承建商也须确保没有人在地盘内吸烟，违规承建商可罚款40万港元。若地盘有人吸烟而构成严重火警风险，劳工处会按相关条例起诉涉事的东主或雇员，东主可判处罚款300万港元及监禁六个月，雇员则可判处罚款15万港元及监禁六个月。
上述修订法例决议案星期三（15日）在立法会获得议员通过，将从星期五起实施。
对此，香港劳工及福利局局长孙玉菡说，新法规将于星期五刊宪后正式生效，意味着执法工作立即展开，包括不事先通知日常进行的突击巡查，也不会有缓冲期和警告期。
他也说明，举证方面，如果有工人违规吸烟的相片、影片作为证明，能辨别违规者的样貌身份、事发地点和时间，例如地盘出入纪录等，以上证据齐全，港府就会执法。

世卫组织：45%失智症风险可避免或延缓
（日内瓦法新电）世界卫生组织说，高达45%的失智症风险与可改变的风险因素有关，通过健康生活方式、疾病管理和减少环境风险等措施，可预防或延缓失智症。
世卫组织星期三（7月15日）发布关于预防和延缓失智症的更新指南。世卫组织指出，虽然目前尚无治愈失智症的方法，但吸烟、饮酒、社会孤立、缺乏运动、空气污染以及高血压、糖尿病等非传染性疾病都是可改变的风险因素。
新指南强调，预防仍然是减少失智症病例的最有效方法。通过增加体育活动、戒烟、减少饮酒、保持健康饮食以及减少空气污染，可降低失智风险。
对于出现轻度认知障碍的成年人，指南建议对他们开展认知刺激和训练。更有效控制高血压、糖尿病和高胆固醇也有助于降低失智风险。
失智症由脑部疾病引起，会影响记忆、思维和生活自理能力。在全球老龄人口当中，失智症是造成死亡和残疾的第七大导因，目前全球有5700万人患上失智症，每年有近1000万人确诊。阿尔茨海默症是最常见的失智症类型，占病例总数的60%至70%。
据估计，失智症每年给全球经济造成约1.3万亿美元（约1.68万亿新元）损失，其中约一半来自家庭成员和朋友提供的无偿照护。

北京时间7月15日，世界杯半决赛迎来焦点大战，阿根廷与英格兰展开强强对话。比赛中英格兰率先打破僵局，第55分钟取得领先，但阿根廷随后展开疯狂反扑。
第85分钟，恩佐·费尔南德斯帮助球队扳平比分，补时阶段劳塔罗·马丁内斯完成绝杀，最终阿根廷2比1逆转击败英格兰，连续闯入世界杯决赛。
另一场半决赛中，西班牙展现强大控制力，以2比0击败法国，成功拿到决赛门票。
西班牙凭借稳定的整体发挥，一路淘汰强敌，成为本届世界杯最大热门之一。
