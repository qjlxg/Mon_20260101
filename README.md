特朗普称会“铭记”不​​申请关税退税公司
美国总统特朗普说，他会“铭记”没有就最高法院裁定为非法关税付款寻求退款的公司。不过，他并没有具体说明放弃申请关税退款会有什么好处。 
路透社报道，特朗普星期二（4月21日）在美国消费者新闻与商业频道（CNBC）现场直播的电话采访中说：“如何他们不申请退款，那是非常明智的。”
美国海关与边境保护局20日开始接受进口商的电子退税申请。特朗普告诉CNBC，他从媒体报道得知，苹果公司、电商平台亚马逊和其他一些大公司尚未申请关税退款，他很高兴。 
特朗普说：“如果他们不申请退款，我会记住他们，因为我正在努力让这个国家强大起来。”
关税争议的焦点是特朗普政府向美国进口商征收了约1660亿美元（约2114亿新元）的关税，这些税收是来自特朗普去年4月援引《国际紧急经济权力法》（IEEPA），对几乎所有贸易伙伴征收的所谓“对等”关税。
美国最高法院今年2月裁定《国际紧急经济权力法》并未授权总统征收大规模关税，指特朗普越权，构成违宪。 
特朗普曾将美国进口商缴纳关税形容为爱国之举，但他在21日的电话采访中似乎把申请关税退税的美国公司称为“敌人”。
特朗普说：“在许多情况下，敌人——这笔钱是落入敌人手里。
“那些人仇恨美国，我们却给他们开出数十亿美元的支票。真令人悲哀。”
特朗普所指的显然是申请关税退款的公司，但他没有点名具体公司或进口产品的原产国。
特朗普说，最高法院“本可以帮助我们”，维持他全面征收的全球关税政策。
开市客（Costco）、联邦快递（FedEx）、食品大厂亿滋（Mondelez）等美国大公司，起诉美国政府以维护它们获得关税退款的权利。 
苹果、亚马逊、美国零售连锁巨头塔吉特（Target）、沃尔玛尚未这么做，但它们仍可通过美国海关与边境保护局门户网站CAPE提交关税退款申请。
