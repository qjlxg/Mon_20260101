世界卫生组织总干事：刚果（金）疑似埃博拉死亡达220人，疫情蔓延速度超过应对能力
日内瓦讯 — 刚果民主共和国（DRC）东部冲突区当前爆发的罕见Bundibugyo型埃博拉已造成重大影响。卫生部5月24日数据称，三省11个卫生区已报告逾900例疑似病例；世卫组织总干事特德罗斯·阿达诺姆·盖布雷耶苏斯5月25日表示，目前疑似死亡达220人，因病例发现延误，救援工作正在“追赶疫情”。他并敦促与刚果接壤国家立即采取行动，并于5月26日启程前往刚果。
世卫组织已将此次Bundibugyo株疫情列为国际关注的突发公共卫生事件。该株目前无获批疫苗或抗体疗法，防控面临严重挑战。
暴力冲突、对当局的不信任以及薄弱的监测体系加剧了防疫难度。卫生部资料显示，医务人员单日仅能追踪约20%的接触者。特德罗斯指出，暴力导致民众和卫生、人道工作者逃离，使接触者追踪和早期识别治疗严重受阻。
当地社会紧张突出。美联社报道，5月24日，蒙布瓦卢一所收治埃博拉病人的医院因当局拒绝归还遗体而遭愤怒居民冲击；伊图里省此前也发生治疗帐篷被焚、病人逃离等事件。
谣言与误解广泛传播。援助行动（ActionAid）称，伊图里多达三分之一民众认为埃博拉不存在，该组织已开始开展宣教活动以纠正错误认知，强调当前既在对抗病毒，也在对抗恐惧与怀疑。
区域卫生官员此前在乌干达首都坎帕拉警示，疏松的边界、活跃的采矿通道和大量人口流动增加跨境传播风险。非洲疾控中心主任让·卡塞亚5月24日表示，因区域流动性和监测诊断能力差距，已有10个非洲国家被列为受疫情威胁国家。
乌干达已报告5例与此次疫情相关的确诊病例。一名在刚果照护埃博拉患者感染的美国人已被撤离至德国治疗，美国疾控中心称若干高风险接触者也被转移至德国和捷克接受监测与处理。
