AI演员担纲  中国仿真人短剧热度攀升
（北京讯）中国仿真人短剧热度攀升，由人工智能（AI）演员担纲的作品更具成本效益，也逐渐获观众认可，正成为短剧赛道新风口。
据关注中国文娱趋势的自媒体“文娱先声”报道，大数据平台DataEye数据显示，今年1月漫剧（指对网文、平面漫画或原创剧本等进行视频化再创作）百强榜中，飙升最快的类别是AI仿真人剧，百强榜占比从去年的7%上升至38%，总播放量达25.48亿。  
其中，末世题材的《末日寒潮我有移动堡垒我怕谁》凭着2.3亿播放增量，问鼎榜首。
报道分析，AI仿真人剧如今更容易被观众接受，关键在于AI演员真实感持续提升，过往的“恐怖谷效应”逐步减弱。“恐怖谷效应”是指角色与真人高度相似却不完全逼真时，容易引起观众反感。
目前，AI仿真人剧已呈现较清晰的垂直细分。例如，风水、玄幻修仙等题材多源自男频小说，常采用重生逆袭的叙事结构，结合末日、民国或古代时空背景。
报道指，在影视业寒冬的大环境下，AI仿真人剧成了“市场抓住的又一枚稻草”，可维持相对可控的时间和人工成本，被寄予营收增长的期待。 
不过，AI仿真人剧仍有明显短板，像是音画不同步、口型分离、AI演员演技差等问题未解决。合规层面上，关于数码版权的风险也浮现，例如AI演员的外貌与真人演员撞脸，引发后者粉丝不满。

中国能否维持长期较快增长，取决于是否推进市场化改革。前中央党校国际战略研究所副所长、现东北财经大学国家经济工程实验室主任周天勇警告称，若不采取行动，中国未来潜在增速可能降至约2.5%。
周天勇在2月1日的微信文章中指出，除非全要素生产率大幅回升并显著扩大居民消费，否则难以实现4%以上的增长。
“潜在增长”指衡量劳动、资本、创新和企业家等要素长期可持续产出的能力。他认为，供给侧压力增加、固定资产投资、居民消费和商品出口等需求端存在较大不确定性，导致潜在增速走低。
这一预估远低于中国去年公布的5%GDP增长，也低于北京测算的到2035年人均GDP翻番所需的4.17%年均增速。去年第四季度经济增速降至4.5%，创三年低点，受需求疲弱和地产长期低迷拖累。
周天勇表示，若深化市场化改革，激活被行政计划束缚的闲置劳动力、资本和土地，提高居民收入并释放农村、外来务工人员及低收入群体的消费潜力，中国仍有实现中高等增长的“可靠来源”。他强调，过去的增长多靠改革释放闲置资源，但改革红利会递减，需新一轮结构性改革来提振增长。
据报道，中国政府将于3月公布2026年GDP目标，南华早报称目标或改为4.5%–5%的区间，取代自2023年以来的“约5%”。
