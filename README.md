（华盛顿综合电）美国财政部长贝森特说，总统特朗普与中国国家主席习近平今年内可能会面多达四次，因美中关系已达至“极佳平衡状态”，双方分歧不太可能像去年那样演变为全面的经济冲突。

贝森特星期四（1月22日）接受美国“政治”（Politico）新闻网站采访时谈及美中关系的“平衡状态”。他补充说：“领导人一旦为整体关系定下基调，间中出现的任何小问题或摩擦，双方可立即通过电话沟通，局势即可迅速缓和。”
贝森特在采访中列举两国领导人今年会晤的多个机会，例如特朗普4月访问北京，习近平可能在夏季访问华盛顿或海湖庄园，以及在迈阿密举行的二十国集团（G20）峰会。
他还说，特朗普已表达意愿，准备出席在中国深圳举行的亚太经合组织（APEC）峰会。
“两国既然能有这一系列的会晤机会，我想双方都不会希望搞砸。” 
特朗普同日在美国总统专机空军一号上告诉记者，习近平会在今年年底访美。“我期待见到习主席，我一直与他保持着良好的关系。”
他表明，冠病疫情期间，美中关系一度紧张，但此后已明显改善。他还说，中国正在大量购买美国大豆，这对美国农民来说是件好事。
与特朗普第二任期初期相比，当前中美关系的基调似乎有了变化。特朗普去年重返白宫后宣布对中国加征高额关税，引发中方报复，双方随后陷入纠缠不清恶性循环的贸易关税战。
包括贝森特和中国国务院副总理何立峰在内的美中官员，先后在日内瓦、伦敦、斯德哥尔摩、马德里和吉隆坡等地展开贸易磋商，局势逐渐有所缓和。

中国AI芯片厂商摩尔线程预计2025年营收将较2024年翻三倍，达到14.5亿至15.2亿元人民币（约2.08亿美元），公司在周三向交易所披露的业绩预告中称。该数据为未经审计的预估，具体数字将以正式财报为准。
公司表示，营收同比增长幅度在231%至247%之间；净亏损预计收窄至9.5亿至11亿元，较上年同期的16亿元有所改善。摩尔线程由前英伟达中国区负责人张建中于2020年创立，上月在上海科创板上市，已实现连续四年营收增长并逐步缩小亏损。
公司将业绩提升主要归因于基于第四代GPU架构“平湖”的MTT S5000芯片，该芯片于2025年开始量产。摩尔线程称，基于该GPU的算力集群已能支持万亿参数级模型训练，其计算效率可与同规模的国外GPU集群相媲美。
摩尔线程是近期一批冲刺IPO的国内AI芯片企业代表之一，其业绩反映出在北京推动半导体自给自足、力图打造可与美国英伟达竞争的本土产业链背景下，厂商积极推动产品规模化应用的趋势。

马尼拉——前众议员、控告人迈克·德芬索指控总统费迪南德·马科斯二世、众议院议长及众议院秘书长办公室合谋阻止对总统的新弹劾案。德芬索称，众院秘书长切洛伊·加拉菲因出国领奖未能亲自接收，秘书长办公室遂在周四拒收两起新提交的弹劾控状。

司法委员会主席盖维尔·卢伊斯特罗表示，在最高法院尚未介入前，弹劾程序将依宪法与议事规则继续推进，并指出控告人可于周一返回秘书长办公室重新递件。
