新疆首家涉外审判机构成立一年受理43案
中国新疆首家涉外审判机构成立一年，处理了43起案件。

中国驻菲使馆早前公布信件片段，声称浅滩不属菲律宾领土且位于菲方200海里专属经济区内。洛佩兹指出这是选择性引用，误导性强，菲律宾对马辛洛克的主权“不可分割、长期且一贯”，依据包括《联合国海洋法公约》及2016年海牙仲裁裁决。 
他强调，只有菲律宾总统可就主权作正式声明，该信非条约或协议，不能作为削弱主权的依据；菲方并通过执法、海洋科研和军演等行动展示对该海域的实际控制。 
NMC表示，将待菲律宾海岸警卫队提交完整报告，作为对中国所谓“非法”或“骚扰与胁迫”行为提出外交抗议的正式依据。

马尼拉—众议院司法委员会主席、巴坦加斯第二区代表热维尔·卢伊斯特罗（Gerville Luistro）周四警告副总统莎拉·杜特尔特（Sara Duterte）的辩护团队，如果缺席3月25日委员会召开的“听证会”（她称为“小审判”），其弹劾案可能会被移送参议院审理。 
上周三，众议院小组认定对杜特尔特的弹劾控诉具备继续进入“正式审理”的充分理由。卢伊斯特罗对ANC表示，若辩方在众院委员面前提出反驳，仍有可能在是否有可疑理由（probable cause）的裁定中驳回起诉；但若杜特尔特及其律师下周缺席，“应预料此弹劾案确会被移送参议院”。
副总统的法律顾问之一迈克尔·波亚（Michael Poa）在另一次ANC访问中表示，无论结论如何，杜特尔特已准备好赴参议院受审——她此前的一起弹劾案曾因程序问题在参议院滞留，最终被最高法院以程序性理由作废。波亚称，他尚未就出席问题与杜特尔特商谈；若出席对辩方“毫无裨益”，他们可能会建议缺席。他并质疑将该听证称为“小审判”，理由是副总统或其律师恐无法交叉质询证人。
卢伊斯特罗则为“听证会”定名辩护，称其目的是让控辩双方提出证人与证据，以便在认定可疑理由时为司法委员提供依循。她说，依照众院规则，辩方可对证人进行交叉质询，但“直接询问与交叉询问须由司法委员执行”，律师的参与仅限于协助当事人。
新一轮弹劾书指控杜特尔特在任内涉贪污腐败，并曾对前盟友、现任总统费迪南德·马科斯 Jr. 发出死亡威胁。宪法规定，众院通过弹劾将触发参议院审判；若判有罪，被弹劾者可能终身被剥夺公职资格。 
—综合法新社报道

马尼拉讯 — 国家证人亨利·阿尔坎塔拉（Henry Alcantara）预计于3月24日在反贪法庭（Sandiganbayan）第六庭出庭作证，参与前Ako Bicol议员扎尔迪·科（Zaldy Co）及多名前DPWH‑Mimaropa官员因东方民都洛省一项防洪道路堤坝工程被控挪用公款的审理。 
此前，前DPWH副秘书长罗伯托·伯纳多称，他在2023至2025年间通过阿尔坎塔拉从科处收受约1亿披索所谓回扣；伯纳多还指称阿尔坎塔拉曾透露，他自被告之一、前DPWH‑Mimaropa区域主任杰拉尔德·帕卡南处收受的3000万披索亦来自科。阿尔坎塔拉曾在参议院蓝丝带委员会听证中点名涉案立法者，2026年1月被纳入司法部证人保护计划。反洗钱理事会代表则预计于3月26日出庭作证。 
本周四，前DPWH‑Mimaropa代理区域主任工程师埃迪莎·巴巴兰重回证人席，检方提交就堤坝受损的补充司法宣誓书、涵盖全段堤坝的损害报告及示范不同钢板桩长度对冲刷影响的视频。早期检查显示，工程实际安装钢板桩平均约3米，远低于设计要求的12米。辩方称上述材料属推测、提交时机不当，并反对巴巴兰就冲刷等技术问题提供专家意见，指出冲刷可能由气候变化等其他因素导致，且损害报告评估的是整段堤坝而非本案工程范围。检方回应称冲刷为基本工程概念，巴巴兰具备相关经验与学识，且应整体评估堤坝受损。由莎拉·费尔南德斯助理法官主持的合议庭允许巴巴兰继续证词。 
此前，多名DPWH‑Mimaropa官员包括杰拉尔德·帕卡南等因挪用指控被拒绝保释。
