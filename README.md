美国总统唐纳德·特朗普于4月25日在佛罗里达州棕榈滩的海湖庄园（Mar-a-Lago）接待了其第二届“表情包币”（meme coin）竞赛的获胜者，向顶级持币人提供与他的会面机会。此次活动吸引了已报名的297名最大持币者出席，前29名还参加了总统的“特别贵宾招待暨香槟祝酒”会，特朗普在会上发表主旨演讲。 
此次招待会正值外界对特朗普家族更广泛加密业务审查趋严、民主党人呼吁展开调查之际。白宫发言人安娜·凯利向路透表示，总统的资产由子女管理的信托持有，特朗普只为美公众利益行事，“不存在利益冲突”。
尽管特朗普为行业站台，$TRUMP 代币自2025年高点已暴跌超过95%。该币1月推出后曾冲至75美元高位，4月24日约为3美元，4月25日早盘曾跌至2.53美元，离开佛州后仍低于2.60美元。区块链分析公司Nansen估算，297名合格获胜者持有约2900万美元$TRUMP，远低于路透报道的2025年首届竞赛时的1.48亿美元。Nansen指出，2026年竞赛引发短暂交易，但未能复制2025年的持续买盘。 
竞赛排名不仅以$TRUMP持仓为准，还计入3月12日至4月14日期间购买的特朗普品牌商品（鞋、手表、香水等）。获奖者将获赠纪念海报、交易卡、手表和香水等品牌周边。 
路透此前调查称，特朗普家族通过加密资产销售已入账逾10亿美元（含2025年上半年与表情包币相关的至少3.36亿美元），并可能存在数十亿美元的未实现收益。与此同时，区块链数据指出，加密亿万富翁孙宇晨名下钱包位列$TRUMP前列，连续第二年在竞赛中夺魁；他于4月21日起诉World Liberty，称该公司冻结其持仓，World Liberty联合创始人兼CEO扎克·维特科夫称诉讼“毫无根据”，并指控孙的“失当行为”。孙未回应置评。

华盛顿 — 4月25日，应总统特朗普下令，工人在林肯纪念堂反射池底部施作“美旗蓝”泳池用涂层进行修复。特朗普4月23日对媒体表示，此项为首都为7月4日建国250周年庆典的整饰工程，预计耗资约150万美元，工期约三周。
反射池长约610米，建于1922—1923年，1963年马丁·路德·金在此发表著名演说《I Have a Dream》。特朗普称该池“年久失修、长期渗漏”，并拒绝了一项需3年、耗资3亿美元的石底更换方案；他联系以往承包商，改用工业级泳池表面材料，成本与工期均大幅缩短。承包方已清理池底并开始浇注，数日内可完工。特朗普还推动包括拆除白宫东翼并改建大型舞厅在内的其他整修工程。来源：法新社。
