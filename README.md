尼科西亚/5月23日 — 国际货币基金组织（IMF）在一份提交给欧盟财政部长的报告中警告称，未来15年欧盟各国将在国防、能源和养老金方面面临巨额开支，如果不采取行动，公共债务将走上不可持续道路。
报告指出，按现行政策不变，欧盟平均国家的债务规模到2040年将升至相当于GDP的130%，约为当前的两倍。为避免这一情形，IMF建议采取改革、财政整合与共同举债相结合的办法。
具体措施包括：提升劳动力在27国间的流动性并鼓励企业雇佣；一体化能源市场；促进居民储蓄跨境流向高回报投资；统一目前各国存在差异的法律；推进养老金改革并提高退休年龄；对低碳与抗气候风险项目提供政府担保以吸引私人资本。
IMF还认为，应把创新、能源与国防视为欧盟公共产品，并通过共同举债来融资。对此，欧盟内部存在明显分歧：西班牙、意大利、法国等支持共同债务，而德国及若干北欧国家强烈反对。欧元区财政部长会议主席Kyriakos Pierrakakis表示，这将是未来数月讨论的议题。
IMF警告称，即便实施改革，多数国家仍需财政整顿以使债务呈下降趋势；改革越有力，所需整顿越少。报告批评“边走边看”的做法已接近极限，碎片化、零敲碎打的调整难以应对不断上升的支出压力。来源：路透社。

核问题方面，巴卡伊表示不在初步框架内，将另行讨论。纽约时报援引两名美方官员称，建议协议的关键可能包括德黑兰放弃高浓缩铀库存，但伊朗法尔斯、塔斯尼姆通讯社否认已就核计划作出承诺，并称核相关议题将在签署后60天内谈判。
霍尔木兹海峡是谈判焦点之一。特朗普称海峡将被开放，但伊朗媒体称若达成谅解，伊朗对这一战略水道的“管控地位”不会恢复到战前状况，且据报海上封锁须在30天内全面解除。
关于资金与制裁，伊朗长期要求解冻在美方制裁下被冻结的资产，塔斯尼姆称伊朗坚持初步谅解应以至少部分取回资产为条件，并要求建立持续释放资金的保障机制。另有报道说，美方可能在谈判期内暂时放宽对伊朗石油、天然气与石化的制裁。
黎巴嫩问题方面，媒体称谅解备忘录将强调“所有战线停火”，包括黎巴嫩；巴卡伊表示，目前优先解决的紧迫问题是结束包括黎巴嫩在内的所有战线的战争。消息来源：AFP、纽约时报、法尔斯与塔斯尼姆通讯社。

耶路撒冷／路透社 — 以色列总理本杰明·内塔尼亚胡据称在5月23日与美国总统唐纳德·特朗普通话时表示，以色列将保留在各战线对威胁采取行动的自由，包括黎巴嫩，一位不愿具名的以色列消息人士24日告诉路透社。
特朗普称，美方与伊朗已“基本就一份谅解备忘录达成谈判”，该协议将重新开放自美以2月对伊朗开战以来实际关闭的霍尔木兹海峡。伊朗法尔斯通讯社称，草案规定美方及其盟友不攻击伊朗或其盟友，作为交换，伊朗承诺不对其发动先发制人的攻击。
以色列重要政界人士本尼·甘茨表示，若以色列为与伊朗达成协议而在黎巴嫩接受停火，将是重大战略错误。消息人士称，美方正向以色列通报与伊朗的谈判进展，特朗普强调其谈判立场——要求拆除伊朗核计划并将所有浓缩铀移出伊朗领土，且在这些条件未达成前不会签署最终协议。特朗普在社交平台上称，与内塔尼亚胡的通话“非常顺利”。

约翰内斯堡——南非克鲁格国家公园（Kruger National Park）保安措施加强，当局5月24日表示，原因为两名游客在园内被发现遇害。
两名遇害者均为南非籍，一名71岁男子及一名未公布年龄的女性。两人在5月20日晚未返回旅馆，引发搜寻行动。
其它游客于5月22日下午在一条河中发现两人遗体。警方称，男子有刀伤，女子身体多处瘀伤；他们的车辆失踪。
南非国家公园管理局（SANParks）表示，已在重点区域增派监控和护林员，并加强技术系统以提升监视与预警能力。
官员称，这是公园百年历史上首次发生游客被杀事件。克鲁格公园位于南非东北部，是该国最大的国家公园，面积约2万平方公里，与津巴布韦和莫桑比克接壤超过370公里。 来源：法新社
