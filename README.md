中国外交部发言人毛宁现身北京市两会
中国外交部发言人毛宁星期天（1月25日）以北京市人大代表的身份，出席北京市人大会议朝阳区代表团会议。
据中国《北京日报》报道，毛宁会上提了两个建议，第一个是关于促进消费的。“去年，团结湖猛火炒饭的视频火爆社交网络，我也凑热闹去看看，到了地方发现已经排起了长队，大家都是慕名而来，为了尝尝老板的手艺。”
据报道，后来朝阳区帮猛火炒饭安了“家”，不仅有了门店，还开了分店。这引起了毛宁的思考：人间烟火气，最抚凡人心。今年北京市的政府工作报告提出，坚持惠民生和促消费、投资于物和投资于人紧密结合，不断增强内需拉动经济增长主动力作用。
她建议，相关部门可以从“烟火气”入手，打造更多安全卫生有序的路边店、小饭馆、菜市场等业态，既满足了市民个性化、体验式的消费需求，也为消费市场增添特色与活力。
毛宁的第二个建议是关于花园城市建设的。她说，近年来北京越来越绿，建设了很多“无界公园”，打破公园和城市的界限，实现了“城园共融”。与此形成对比的是，一些老公园的管理理念还存在提高改进的空间。
她举例说，团结湖公园是周边市民放松休闲的好去处，但是规定了开放时间。有时她早上路过，看见很多晨练的老年人在排队等待入园。
毛宁说：“如果团结湖公园能在保证安全的前提下实现全天开放，这些老年人就免去了排队之苦。推而广之，全市有条件的公园都能做到这一点，一定会极大地提升市民的幸福感。”
#毛宁

中国官媒促网红依法纳税 称税务可精准掌握收入来源
中国税务部门多次曝光网络主播涉税案件，官媒发文指，对于追求“长红”的网络主播来说，依法纳税是保障事业长远发展的必修课。
中国官媒《经济日报》星期一（1月26日）发文称，近年来，税务部门对网红明星偷税逃税行为进行了系统整治。网络职业主播规模已达3880万人。行业的持续繁荣与高质量发展，离不开规范有序的环境。对于追求“长红”的网络主播来说，依法纳税是保障事业长远发展的必修课。
中国国家税务总局此前通报，2025年前11个月，税务部门共查处1818名包括明星网红在内的“双高”人员，查补税款高达15.23亿元人民币（2.78亿新元）。
《经济日报》文章称，2025年中国出台实施的《互联网平台企业涉税信息报送规定》明确要求，互联网平台企业按时、准确报送平台内经营者和从业人员的身份信息、收入信息，从而打通平台数据与税务监管的信息壁垒，让税务部门能够及时精准地掌握网红明星的收入来源，及时发现隐瞒收入、拆分收入等问题，对相关偷逃税行为进行依法有效查处。
文章指出，部分网络主播纳税意识薄弱、心存侥幸，采用多种手段偷逃税款。有的主播账号拥有超3000万粉丝，通过第三方平台个人账户收款，隐匿劳务报酬收入；有的主播刻意隐瞒直播带货收入，虚列成本、不依法申报，这些行为的本质是通过隐匿收入、虚假申报来逃避纳税义务，最终都未能逃过税务部门严密监管，落入法网。

台湾自制潜舰据报今首次进行下潜测试
台湾自制潜舰海鲲号据报在星期一（1月26日）进行首次下潜测试。
综合台湾《联合报》《自由时报》报道，海鲲号2024年2月27日从海昌厂房移至台船小坞，同年7月15日转至高雄港91号码头展开岸测或称码头测试（HAT）。若以2024年7月15日停靠码头的岸测时间点计算，海鲲号原型舰测试已超过一年半。
