【北京市卫健委主任：今年将落地一揽子生育支持政策】金十数据1月25日讯，北京市政协委员、北京市卫健委主任刘俊彩在接受专访时透露，北京已推动出台了67项生育支持措施，去年发放育儿补贴9.47亿元。在谈及北京疏解医疗资源的进展如何时，刘俊彩表示，2025年，实现清华长庚医院二期等3个项目开诊，首儿所通州院区等3个项目加快推进，宣武医院房山院区等6个项目开工。雄安宣武医院科室设置全面落地，协和医院国家医学中心（雄安院区）一期项目开工。今年我们将统筹推进育儿补贴、生育保险、生育假期等一揽子生育支持政策措施落地，同时配合市教委等研究放宽普惠托育收费标准、优化补贴政策。
