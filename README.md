日本将海啸警报降级为预警
日本当局已将东北沿海地区的海啸警报降级为海啸预警。
日本的海啸警报分为三个级别。海啸预警是最低级别的警报，适用于浪高低于一米的情况。在这种情况下，建议人们立即离开海域和沿海地区。
海啸警报是第二高的警报级别。这种警报适用于预计浪高可达三米的情况。
当发布海啸警报时，海啸警报区域内的人员必须立即从沿海地区和河岸地区撤离，前往地势较高的地方或避难建筑物。
最高级别的警报是特大海啸警报，预计浪高超过三米的海啸将反复袭击该地区。
日本星期一（4月20日）的地震最初估计为7.4级，随后上调至7.5级，最终修正为7.7级。地震于当地时间下午4时53分（新加坡时间下午3时53分）发生在岩手县北部近海的太平洋海域。

在这一背景下，库珀将在日本行程结束后于4月20日提前返回伦敦，已取消4月21日在东京的发言，并放弃原计划返回海湾的行程。外交部未就行程变更说明原因。 
库珀自上周起已在巴黎、安塔利亚、迪拜和东京会见盟友。 
外交部首席官员奥利·罗宾斯因被指推翻有关曼德尔森的审查建议，上周被解职，斯塔默与库珀对其失去信任。罗宾斯预计将于4月21日在议会委员会作证，部分盟友公开表示他受到苛刻对待。

麻疹在美国蔓延，感染水平数十年未见，威胁该国已达消除状态。美国疾病控制与预防中心截至2026年4月17日记录1748例感染，公共卫生专家称实际数字可能被严重低估。
曾持疫苗疑虑的家长开始转变：德州全职母亲凯蒂·詹宁斯在家乡出现第二例未接种儿童死于麻疹后，带6名子女接种MMR疫苗，并在TikTok发文劝导，视频播放逾42.2万次。
在若干疫区，这种变化已减缓疫情扩散：南卡罗来纳州1月免费诊所MMR接种较去年增长近170%，接近宣布疫情结束；德州2025年MMR接种上升15%，该州于8月宣布结束；犹他自2025年夏季以来也记录到接种回升。
反疫苗运动长期围绕罗伯特·F·肯尼迪及其“让美国重返健康”议程展开，但当麻疹在身边爆发时，部分支持者开始质疑其信息来源并转向接种。前线儿科医生采用不评判的沟通方式，耐心解答家长疑虑并强调接种隐私；同时说明麻疹可致“免疫遗忘”等长期风险以促成决策。
儿童免疫率在新冠疫情期间下滑——既有常规门诊中断，也有错误信息（如将MMR与自闭症错误关联）影响。部分曾为反疫苗者的个体已成立互助群体，帮助他人逐步重建对医疗体系的信任。专家指出，公众对麻疹缺乏集体记忆，使得卫生教育尤为重要。
（来源：彭博社）

莫斯科，4月20日——克里姆林宫周一表示，保加利亚政治人物鲁门·拉德夫在周日以压倒性优势获胜后，表示愿通过务实对话解决与俄罗斯的相关问题，这令俄方感到鼓舞。官方结果显示，拉德夫领导的“进步保加利亚”党大胜，挤压了长期主导的政党，可能使这一欧盟与北约成员国向莫斯科靠拢。拉德夫为前总统、前战斗机飞行员，反对对乌军事支持，并主张改善与俄关系、恢复俄油气向欧洲的自由流通。克里姆林宫发言人德米特里·佩斯科夫在被问及欧洲是否因其胜利而“恐慌”、以及拉德夫是否为俄方“特洛伊木马”时表示：“我们当然对拉德夫及某些欧洲领导人愿通过对话解决问题的表态感到鼓舞。”但佩斯科夫同时称，鉴于欧盟委员会在布鲁塞尔仍有他所称的反俄言论，现在就断言欧洲整体政治气候正在改变还为时过早。——路透
