美大豆对华出口回暖 美农民今春扩大种植面积
随着美国大豆对华出口回升，美国农民今春将增加大豆种植面积，同时缩减玉米种植规模。
据彭博社报道，美国农业部星期四（2月19日）在年度首份展望报告中预计，今年大豆种植面积将达8500万英亩，高于去年的8120万英亩。农业部指出，扩种主要因为大豆相较其他作物盈利能力更强，加上玉米带和三角洲地区进入轮作周期。
相比之下，玉米种植面积预计将从去年的9880万英亩降至9400万英亩；小麦面积也将由4530万英亩微降至4500万英亩。三大主要作物的总种植面积整体减少约1%，但农民销售价格预计将小幅上扬。
美国农业长期面临成本利润的双重挤压：农产品价格偏低，种子、化肥和农机成本却居高不下。去年，美国总统特朗普推出关税措施，一度扰乱农产品出口并推高部分进口成本，进一步加剧行业压力。中国在大部分生长季减少采购美国大豆，直至去年10月中美元首会晤后才恢复进口。
报道称，对华出口回暖有望改善美国农民的经营状况。同时，种植者也寄望提高以玉米为主要原料的高乙醇汽油在国内的使用量，以提振需求。

美国政府周三宣布撤销一项奖励电动汽车生产的规定，取消所谓“燃料含量系数”。分析人士称，这是联邦支持电动车的一系列回撤中的最新一项，可能导致美国在全球竞争中进一步落后于中国。
此举与特朗普重返白宫后削减税收激励、放松排放规则的做法相呼应。被撤销的规则允许制造商在CAFE（企业平均燃油经济性）标准下，将电动车计入人为提高的燃油经济值，从而在车队平均值计算中获得便利，目的是激励车企生产电动汽车。
部分分析认为，放宽燃油经济等监管会使美国市场更倾向于燃油车，抑制电动创新，给中国带来竞争优势；另有担忧则集中在产业转型期可能出现的不公平竞争环境。
美国环保署今年2月已撤销构成气候监管基础的“危害认定”，并废除了尾气排放标准。生物多样性中心安全气候运输行动主任丹·贝克表示：“美国家庭将承受长期伤害，而大型汽车和石油公司将获得短期利润。”
