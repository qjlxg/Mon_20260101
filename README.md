中国多项出口商品涨价 全球通胀阴影浮现
随着伊朗战争推高与石油相关的投入成本，中国出口商开始上调从泳衣到空调等各类商品的价格，显示全球消费品通胀可能加速。
彭博社对贸易数据监测机构（Trade Data Monitor）汇编的海关数据进行分析后发现，3月份有十多个消费品类别的价格同比大幅上涨，结束了过去几年持续下跌的趋势。报道称，此前中国出口价格下跌曾帮助抑制全球通胀。
报道引述上海一家医疗导管制造商的销售经理称，“3月份我尽可能拖着不涨价，但最后实在没办法。看着塑料成本几乎每天都在涨，我很慌。”
这种压力正波及多个行业。中国出口商也上调了泳衣、滑雪服和女裤等产品价格，涨幅为低至中个位数百分比。这些产品都依赖聚酯等合成纤维。3月份，它们的供应商甚至频繁到每天上调纤维价格。
其他依赖橡胶、塑料和石油衍生化学品的产品价格也出现飙升。医用注射器是受冲击最严重的产品之一，3月份价格涨幅高达20%。与此同时，家电价格面临双重挤压，制造商还要应对金属和半导体成本上升。
细分数据显示伊朗战争引发的能源冲击如何传导至中国，并进一步波及全球零售商。
近三年来，由于产能过剩和竞争激烈，中国出口商品价格一直下跌，帮助美国、欧洲等经济体抑制通胀。凯投宏观（Capital Economics）估计，近几年中国出口价格下跌使发达经济体的整体通胀率降低了约0.3至0.5个百分点。
直到今年2月，价格更低的中国商品仍在英国等经济体发挥抑制价格压力的作用。如今，随着中国制造商开始转嫁更高成本，这道反通胀缓冲正在减弱。
彭博经济研究估计，受能源成本飙升影响，欧元区、美国和英国2026年通胀率可能再度重新升至3%以上。 
成本升高也已使中国工业生产者出厂价格指数（PPI）经历了长达41个月的通缩后首次实现正增长。高盛预计，中国整体出口价格最早可能在3月份转为正增长。官方数据将在星期六（4月25日）左右公布。

中国前驻美大使李道豫逝世 曾与克林顿激烈交锋

白宫官方X账号星期四（4月23日）发布白宫科技政策办公室的备忘录，指出美国政府已掌握信息，来自中国的实体正在开展蓄意的、工业级规模的行动，对美国前沿AI系统进行蒸馏。
备忘录称，这些行动利用数以万计的代理账户来规避侦测，并使用“越狱”技术暴露专有信息。“这些协调一致的行动系统性地提取美国AI模型的能力，利用美国的专业知识和创新成果”。
备忘录还称，通过蒸馏开发的模型并不能复制原始模型的完整性能，但确实让他们以极低成本发布表现不错的产品。“AI蒸馏如果被合法用于生产更小、更轻量的模型，并服务于更先进的系统，是这一生态系统的重要组成部分。然而，系统性破坏美国研发、获取专有信息的工业化蒸馏活动，是不可接受的。“
这份备忘录由白宫科技政策办公室主任克拉齐奥斯（Michael Kratsios）发给各行政部门和机构负责人。
克拉齐奥斯表示，特朗普政府将：与美国AI公司共享有关外国行为者试图开展未经授权、工业级规模蒸馏的信息，并探索一系列措施，追究外国行为者从事工业级规模蒸馏活动的责任。
克拉齐奥斯转发白宫贴文表示，“我们将采取行动，保护美国创新。” 他没有在帖文中点名任何具体实体。
蒸馏是人工智能开发中的常见做法，企业常用这一方法，为自家模型开发成本更低、体量更小的版本。今年2月，美国人工智能开发商Anthropic指控三家中国公司深度求索（DeepSeek）、月之暗面（Moonshot AI）和MiniMax非法提取其Claude聊天机器人的能力，并称这是知识产权盗窃。
同月，ChatGPT开发商OpenAI致函美国国会议员，指控DeepSeek使用蒸馏技术，并称这发生在“持续搭便车、利用OpenAI和其他美国前沿实验室所开发能力的努力”之中。

华盛顿消息——美国司法部内部监察机构4月23日表示，将调查司法部是否遵守《爱泼斯坦档案透明法》（Epstein Files Transparency Act）对公开相关档案的规定。
该法案于2025年11月通过，要求司法部在30天内公开与性犯罪嫌疑人杰弗里·爱泼斯坦相关的全部文件。爱泼斯坦为亿万富豪，2019年在狱中等待性贩运审判时死亡。司法部在一系列文件公开行动中，于1月底释放了约300万页材料，但随之遭到强烈批评。
国会两党议员指出，公开文件中存在不当删节，同时司法部一度公布了此前未公开受害者的姓名与照片，引发愤怒，这类敏感内容随后被移除。该法要求对受害者的姓名及其他可识别信息进行涂黑，但规定不得以“窘境、名誉受损或政治敏感”为由为与爱泼斯坦关系密切的权势人物提供保护。
司法部监察长办公室表示，此次调查将评估司法部识别、涂黑与发布相关记录的流程，并在完成后发布最终报告。此前2月，获准在司法部查阅完整未删节档案的议员亦曾投诉，公开版本仍存在不当遮蔽。自1月30日大规模资料释放后，若干知名人士因与爱泼斯坦的关联受到震荡，引发欧洲等地的刑事调查、逮捕与辞职。
