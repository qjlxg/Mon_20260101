日本拟扩建成田机场 打造亚洲领先航空枢纽
日本首都圈拥有两座承担国际航班的主要机场——羽田机场和成田国际机场。随着日本入境旅游增长强劲，东京首都圈航空需求预计将进一步扩大。鉴于最靠近市中心的羽田机场扩建空间有限，日本当局计划强化位于近郊千叶县的成田机场功能，使两座机场形成互补联动，打造亚洲领先的航空枢纽，带动经济增长。　　
羽田机场毗邻东京市中心，近年来国际航班不断增加。根据2025年全球最繁忙机场榜单，羽田机场以9170万人次的客运量跃升至全球第三，仅次于哈茨菲尔德—杰克逊亚特兰大国际机场（1.063亿人次）和迪拜国际机场（9520万人次）。
据日本放送协会（NHK）报道，面对入境旅游业蓬勃发展带来的航空需求增长，日本当局决定扩大成田机场功能。国土交通省公布，计划将成田机场现有的三个客运航站楼整合为一个大型综合航站楼。
此外，当局也正与铁路公司磋商，改善成田机场通往东京市中心的铁路交通，并透露若开通连接成田机场与羽田机场的收费特快列车，将大幅缩短两个机场之间的通行时间。　　　
日本国土交通部星期五（7月10日）与千叶县政府召开会议，宣布将尽快落实成田机场新跑道建设计划。
成田机场管理公司说，目前规划中的扩建工程，包括将现有2500米长的B跑道延长1000米，并新建一条3500米长的C跑道。扩建完成后，机场年起降架次将增至50万架次，约为现有容量的1.5倍。　
据了解，B跑道延长计划进展顺利，但C跑道的工程因机场周边约105万平方米土地征用尚未完成而迟迟无法动工。千叶当局决定动用《土地征用法》，以公共利益为由推进土地征收。
成田机场在20世纪70年代兴建时，曾因依据《土地征用法》强制征地引发地主激烈反抗，并造成伤亡事件。此次扩建机场功能，千叶县以“第二次开港”为口号，获得机场周边市镇的理解与支持。
千叶县知事熊谷俊仁在会后说：“虽然这是一个艰难的决定，但征地制度是不可避免的。”
他也指出，韩国仁川国际机场、新加坡樟宜机场等亚洲主要枢纽机场都在持续升级功能。成田机场将通过全面推进跑道扩建、改善交通衔接，以及吸引更多客运和货运航线，进一步提升自身竞争力。

大摩：三大因素可能终结美股夏季涨势
格隆汇7月10日｜摩根士丹利全球固定收益研究主管安德鲁·希茨表示，该行正密切关注可能导致股市在夏季受挫的三大障碍；从历史上看，夏季通常是股市表现最强劲的季节。第一大风险是伊朗冲突重燃。美国战略石油储备降至历史最低水平，如果冲突再次升级，这可能会削弱其应对冲击的能力。第二大风险是美联储加息。美联储将在年底前维持利率不变的预期，是支撑当前股市牛市的关键支柱之一。 风险在于，这一假设可能是错误的，而且这种错误可能很快就会显现。当然有一种观点认为，如果美联储担心通胀，就不应拖延行动。第三是AI资本支出前景走弱。第二季度的财报可能显示出在支出方面更为谨慎的态度，这或许是因为部分大举投资AI的公司的股价近期表现不佳。鉴于当前的增长和盈利前景与AI高度相关，且投资者对AI概念股青睐有加，这种情况将带来风险。

高盛 ：抄底芯片股
高盛全球银行与市场部门ETF及定制篮子波动率交易负责人Shawn Tuteja在最新报告中指出，近期美股半导体及AI板块近20%的动量因子回撤，主要由技术性和结构性因素驱动。Shawn Tuteja认为，此轮动量因子超过20%的回撤，根本上反映的是市场定价效率的提升，而非AI交易逻辑的瓦解。从波动率调整后的回撤幅度来看，此次跌幅与此前动量因子在同等已实现波动率环境下的历次抛售轨迹高度吻合，只是市场到达这一水平的速度更快。据高盛TMT交易台的数据，目前客户情绪约为7.5至8分（满分10分），较一个月前的9.5至10分有所降温，但客户普遍认同本次回撤的核心驱动力是技术性的。高盛Prime Services数据显示，过去数周，基本面多空基金大幅削减了AI及动量敞口，当前总杠杆率已降至近一年的最低十分位。Shawn Tuteja认为，随着系统内杠杆降低、机构持仓更趋均衡，投资者具备在半导体及动量股多头方向“试探性建仓”的战术性机会。（券商中国）
