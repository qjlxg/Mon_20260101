各国释放储备难补缺口 今年石油料供不应求
（巴黎综合电）国际能源署说，由于中东战争造成了史无前例的供应中断，世界各国正以“破纪录的速度”动用石油库存与战略储备，但预计今年全球石油仍将供不应求。
根据国际能源署（IEA）周三（5月13日）发布的月度报告，全球石油库存继3月份减少1亿2900万桶后，4月份再减少1亿1700万桶。
由于霍尔木兹海峡油轮交通仍然受限，波斯湾生产商累计供应损失已超10亿桶。目前，每天有超过1400万桶石油被迫停产，这对石油供应造成了史无前例的冲击。
报告说：“随着全球石油库存以破纪录的速度下降，夏季需求高峰期来临前，油价可能会进一步波动。”
国际能源署成员国3月同意释放共4亿桶战略石油储备，其中约1亿6400万桶已投放到市场。这虽有助于缓解市场波动，但只能弥补部分产量损失，而各国储备正在不断减少。
报告预计，2026年全球每日石油供应量将比总需求量少178万桶。即使战争在6月初之前结束，到2026年第三季度末，市场供应仍将严重不足。
这颠覆了此前有关石油将过剩的预测。国际能源署去年12月预计，全球每日石油供应将比总需求高出近400万桶，上个月的预测则为每日过剩41万桶。
根据国际能源署的基本预测，霍尔木兹海峡航运将从第三季度起逐渐恢复，并在第四季度出现“适度过剩”，从而开始重建耗尽的库存。
与此同时，高油价、经济环境恶化，以及旨在降低需求的措施，预计将进一步抑制全球石油消费。今年全球日均石油需求料减少42万桶，高于此前预测的日均少8万桶。
美国能源信息署同日发布的数据显示，2026年第一季度，每天约有1460万桶原油和液化石油经霍尔木兹海峡运输，较去年同期的每日2040万桶减少近600万桶。
彭博社汇编的卫星图像显示，伊朗最大原油出口枢纽哈尔克岛（Kharg）已连续第四个周期未见远洋油轮的踪影。这是美伊冲突爆发以来，哈尔克岛码头闲置的最长时期。
分析员警告，伊朗石油业因美国对伊朗港口实施海上封锁，而面临越来越大的压力。随着出口持续受阻，伊朗储油设施填满的速度将加快，陆上储油设施或在六周内饱和，这将迫使它进一步减产。
