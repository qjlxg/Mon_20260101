据中新社报道，据中国科学院大连化学物理研究所（大连化物所）最新消息，该所科研团队与合作者明确上述现象在多相催化反应中的重要作用，并发现金属-载体三维体相参与催化过程的新机制。

大摩：若油价持续120美元 对消费拖累更显著
格隆汇4月15日｜中东战事未平息，油价上升影响亚洲经济。摩根士丹利亚洲首席经济学家Chetan Ahya表示，在地缘政治紧张局势升温之前，亚洲各经济体(包括中国)的消费增长正逐步改善，随着能源价格上升，短期而言会削弱可支配收入，但政策及储蓄或可作缓冲。\n他指出，若未来3至4个月油价实物交收价格回落至每桶90美元以下，工业周期有望重新加速，进而支持资本开支、就业及消费。不过，若油价持续于每桶120美元水平，随着燃料价格进一步上升，消费者将调整消费模式，对消费的拖累可能变得更为显著。\n大摩中国首席经济学家邢自强团队指出，自地缘政治局势升温以来，中国与“以旧换新”计划相关的板块消费增长持续放缓。团队解释，鉴于中国对能源进口的依赖相对较低、通胀起点较低，加上燃料定价受管制，能源价格上升对家庭消费的直接拖累可能有限。不过，投入成本上升可能压缩下游企业的利润率，进而拖累工资增长及就业，这反过来将打击消费，尤其是在工资增长起点相对疲弱的背景下。

霍斯顿消息——由于与伊朗的冲突导致经霍尔木兹海峡的中东供应大幅中断，上周美国距离成为二战以来首次原油净出口国仅一步之遥。
美国政府4月15日公布的周度数据显示，上周净原油进口仅为6.6万桶/日，为自2001年周度数据以来的最低值；同期出口升至520万桶/日，为七个月高点。按年度计算，美国上次实现原油净出口是在1943年。
伊朗冲突使约五分之一的石油天然气运输受阻，亚洲与欧洲炼厂纷纷向美国采购替代货源。船运跟踪公司Kpler称，上周约47%（约240万桶/日）的美国产油航向欧洲，37%（约149万桶/日）运往亚洲，买家包括荷兰、日本、法国、德国和韩国。数据还显示，一艘载50万桶的油轮正驶往土耳其，或为一年内首笔对土耳其的美国原油出口。
随着布伦特对西德州中质油的溢价在3月一度扩大到20.69美元/桶，且现货货价4月13日逼近150美元/桶，美国产轻质原油对欧亚炼厂更具吸引力。同时美国进口跌至530万桶/日左右，较前期减少逾100万桶/日，因为本土炼厂仍需较多中重硫质原油。
分析师警告，美国出口正接近管道和航运能力的上限：业内估计最大出口能力约600万桶/日，2023年纪录为560万桶/日。释放战略石油储备中的中硫油品或有助于更多轻质低硫原油投放出口，但船舶短缺与运费上升或抑制这一空间。Vortexa称，截至4月15日约有80艘空驶超级油轮正驶向墨西哥湾，预计在4、5月装载原油。

金門——距中國廈門最近處僅3公里的金門群島，既是兩岸交流的天然橋樑，也成為北京推動「統一」策略的試驗場。當地約13萬人因地緣與經濟原因，普遍對與廈門更緊密往來抱有期待；計程車導遊吳女士直言，金門本地缺乏發展，需與大陸合作才有出路。
北京近年以「軟硬兼施」方式在金門推進融合：提出金廈共生圈、橋梁與通往廈門翔安國際機場的連接、互認專業資格、水電供應等誘因，同時在海域頻繁採取灰色地帶手段，如海警常態巡航。學者張文勝稱若和平手段無效，便以壓力加碼；台大陳世民則指出，金門因經濟依賴更易受影響，但此路徑難以複製到主島，因台灣整體更倚重國際貿易且民意普遍反對「一國兩制」。
金門有深厚軍事史，曾是冷戰前線；2019年前後三通與2001年「小三通」推動兩岸人流，旅遊與貿易增加，卻在2019年後受兩岸關係緊張沖擊。2024年發生越界翻船致死事件後，中方在金門周邊海域的頻繁進出更引發主權與安全憂慮。學者提醒，北京可能利用金門作為法理與實務先例，逐步蠶食台灣管轄空間；台灣則被建議持續記錄並對外揭露中方行為、強化海巡防衛，防止主權被侵蝕。
對多數金門居民而言，經濟利益與和平穩定仍是首要訴求，但台灣對於維護自主與民主的決心，將決定北京這套「以小試大」策略能否成效。

解读：伊朗能否合法对霍尔木兹海峡收取通行费？
德黑兰近日与伊朗伊斯兰革命卫队合作，提出向过往船只收取“通行费”以换取安全通行，试图加强对霍尔木兹海峡的控制。该海峡连通波斯湾与阿曼湾，位于伊朗与阿曼领海内，是全球最重要的能源航道之一，约有20%的世界石油经由此通行。海峡全长约167公里，最窄处为两条各约2海里（约3.7公里）的进出航道，中间有约2海里的隔离带。
关于法律适用，1982年通过、1994年生效的《联合国海洋法公约》（UNCLOS）第38条确立了“过境通行”权，适用于包括霍尔木兹在内的100多个海峡。公约允许沿海国在其12海里领海内对通行实施管理，但须允许“无害通过”——即不危及沿海国的和平、治安与安全，禁止军事挑衅、严重污染、间谍活动和不当捕鱼等行为。1949年有关科孚海峡的国际法院判例对“无害通过”概念具有重要影响。
约170个国家和欧盟已批准公约，但伊朗与美国尚未加入。这引发争议：公约规则是否已成为国际习惯法以约束所有国家。学界普遍认为UNCLOS已接近或构成习惯国际法，尽管个别未批准国可主张“持续且一致”的反对。伊朗声称曾提出反对；美国则不承认伊朗征收通行费的权力。
执行与反制方面，公约本身缺乏强制执行机制，设立的汉堡国际海洋法法庭与海牙国际法院可裁决但无直接强制力。反制手段包括：由单一国家或联盟采取行动、联合国安理会通过决议反对、企业绕行或重新安排航路（业界已在实施），以及对被认为支付通行费的企业和金融交易实施或扩大制裁。
（路透）

华盛顿 — 众议院共和党人提议对那些通过不当“查询并复制”美方顶级AI模型结果来开发竞品的中国实体实施制裁，作为国会在全球AI竞赛中遏制中国的最新行动。 
由众议员比尔·惠曾加（Bill Huizenga）提出的法案草案要求政府识别在中国和俄罗斯使用该做法的机构，并考虑将其列入商务部实体清单或动用1977年总统紧急经济权力予以制裁。可能被点名的目标包括被指控提取美方能力的中国AI实验室DeepSeek、Moonshot和MiniMax等，尽管中国AI行业仍以阿里巴巴、字节跳动等巨头为主导。 
名为“遏制美国AI模型窃取法案”的提案将与十余项针对中国的出口管制议案一并于下周由众议院外交事务委员会审议，联合发起人为众院中国问题特别委员会主席约翰·穆伦纳尔（John Moolenaar）。惠曾加称，模型提取攻击是“中国经济胁迫与美国知识产权窃取的新前沿”。 
法案与中国问题委员会的报告均建议设立由美方牵头的信息共享中心，加强对“对抗性蒸馏”（adversarial distillation）的侦测，并将相关案件移交司法部审查起诉，同时建议商务部评估将涉事公司列入实体清单。 
OpenAI、Anthropic与Google已开始共享信息以发现此类行为。所谓“蒸馏”是用旧的“教师”模型通过查询训练出更廉价的“学生”模型；当用于未经授权复刻领先模型时，违反了厂商使用条款。相关厂商向国会称，DeepSeek等曾利用ChatGPT等结果构建缺乏安全防护的仿制品，且在厂商采取对策后仍有演化行为。 
国会报告并警示，蒸馏模型可能在敏感议题上受限（如天安门），无法阻止用户实施危险行为（如构建生物武器），并以几乎免费的开源权重模型侵蚀美方为数据中心与人才投入所需的收入。报告称，中国相关主体使用复杂的访问基础设施，通过数千个伪造账户和中介工具掩饰请求来源，并点名若干公开代码工具与中介运营者为中国企业与高校获取封闭源美国AI模型提供途径。 
（来源：彭博社）
