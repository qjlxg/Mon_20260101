美军核心弹药七周耗半：对伊行动虽足 应对同级别大国难
（华盛顿综合电）美军导弹储备亮起红灯。最新评估显示，美军短期内仍有足够弹药维持对伊行动，但部分核心导弹库存已大幅下滑，在西太平洋形成“脆弱窗口”。
专家警告，目前剩余关键弹药已不足以支撑与中国等近乎同等实力对手爆发大规模冲突，且补充缺口或需数年时间。
根据美国智库战略与国际研究中心（CSIS）的最新分析，在过去七周的美伊交战中，美军至少消耗45%的精确打击导弹、至少一半的萨德拦截导弹，以及近50%的爱国者防空导弹。三名熟悉五角大楼机密库存评估的人士说，这些数字与军方掌握的情况基本吻合。
CSIS专家指出，五角大楼今年初已签署一系列扩大导弹产能的合同，但即便产能提升，补回这些系统的交付时间仍需三至五年。
从短期看，即便美伊停火协议破裂，美军剩余弹药仍足以支撑对伊朗的后续行动。但评估警告，目前剩余的关键弹药已不足以应对像中国这样的“近乎同等实力”对手；同时，高强度消耗也在西太平洋形成了更脆弱的防卫窗口。
早在战争爆发前，美国军方便曾提醒总统特朗普，持续军事行动可能冲击美国武器库存，尤其是原本用于支援以色列和乌克兰的弹药与武器储备。
相比之下，伊朗仍有能力大量生产“见证者”（Shahed）无人机和中短程弹道导弹，而且现有库存规模庞大。
五角大楼发言人帕内尔回应称，美军拥有在总统选定的时间与地点执行任务所需的一切能力，并强调自特朗普上任以来，美军在多个战区展示了强大的实战与战备韧性。
五角大楼预算文件显示，美国海军预计明年将采购约785枚战斧巡航导弹，远高于今年的55枚；同时，美国陆军针对爱国者PAC-3 MSE导弹的大规模采购计划预计将持续至2031年，甚至更长时间。
另一方面，美国海军正推进把陆军使用的爱国者导弹整合进海军“宙斯盾”作战系统。
美国国防巨头洛克希德·马丁星期二（4月21日）宣布，已获得相关整合项目首阶段合同。这标志着爱国者导弹将首次部署到海上，可望增强美军舰艇拦截无人机和导弹攻击能力，并提升舰队整体拦截弹储备深度。

台军购案明协商 蓝白或同意预算增至8000亿新台币
台湾政府提出的1.25万亿元（新台币，下同，约504亿新元）军购特别条例草案持续卡关之际，立法院将在星期四（4月23日）再度召集朝野协商。民众党主席黄国昌透露，倾向将军购预算规模上调至8000亿元，“蓝白方向一致”。
综合台湾《上报》和TVBS新闻网报道，台湾立法院长韩国瑜上周已就行政院提出的1.25万亿元版本、国民党团主张的3800亿元+N版本，以及民众党团提出的4000亿元版本召开党团协商。不过，朝野在军购项目、采购经费及建置能力等方面仍存在分歧，最终未能就预算规模达成共识。
韩国瑜随后发出通知，将在星期四下午2时再次召集朝野协商军购特别条例。
此前，国民党主推3800亿元+N的预算版本，但据悉军购条例版本仍在整合中，且在美国关切及台湾内部政治攻防下，国民党内部立场已有所松动。
有国民党立委透露，党团总召傅崐萁日前在党团大会中提到，目前仍有多个版本待整合，尚未定案。
国民党此前也通过新闻稿表明立场，强调军购应“透明、官方、具单据”，并表示只要后续取得美国正式发价书，就会全力支持，且相关预算规模并不限于外界讨论的3800亿元，显示党内对军购案的态度仍在调整与讨论中。
在朝野协商前一天，国民党主席郑丽文星期三（4月22日）与美国在台协会（AIT）处长谷立言会面。郑丽文在社交平台发文称，双方都强调两岸关系应持续通过对话与和平发展推进。
至于双方会面是否触及军购议题，国民党文传会主委尹乃菁表示并不清楚，并称相关内容可参考美国在台协会对外发布的信息。
黄国昌星期三受访时则透露，条例可以先通过，民众党的立场是，军购部分可依据国防部私下说明的第二批军购金额，加上第一批金额，整体规模大约可上调至8000多亿元。“但会有个条件，就是要开始编预算时，一定要等到美国公布要卖什么，且蓝白一致的方向”。

伦敦地铁工人大罢工 上班族步行骑车通勤
英国伦敦地铁工作人员大罢工，数千名上班族被迫步行通勤。
法新社报道，英国铁路、海运和运输工会（Rail, Maritime and Transport union）成员于当地时间星期二（4月21日）中午开始为期24小时罢工，抗议伦敦交通局计划将每周工作日压缩为四天。
工会成员还会从星期四（23日）中午起再次罢工24小时，这意味着伦敦地铁网络在本周剩余时间里面临中断。
伦敦交通局说，这次罢工导致伦敦地铁部分线路停运，网络其他区域出现严重延误。
不过，运营商说，希思罗快线和盖特维克快线等机场接驳线路未受影响；开往法国、比利时和荷兰的欧洲之星列车也正常运行。
伦敦的上班族星期三不得不步行上班，也有人骑单车或乘巴士通勤。
工会称，司机在两次公投中，都反对每周工作日压缩到四天，但伦敦交通局仍决定推进这一规定。轮班时长和年假制度调整等问题，也引起工会成员争议。
伦敦交通局首席运营官克莱尔·曼恩则说：“这些调整都是自愿实行，合同工时不会减少，希望继续保持每周五天工作模式的员工，依然可以选择这么做。”

雅加达批以军 在加沙印尼医院挂军事“宣传”横幅
印度尼西亚指责以色列在加沙一家由印尼出资建造的医院废墟上，悬挂涉对伊朗军事行动的“宣传”横幅。
法新社引述印尼媒体报道，加沙北部、靠近以色列加固边境的印尼医院于2015年底启动，现已遭战争摧毁。废墟上悬挂一条横幅，暗指以色列去年针对伊朗展开的为期12天军事行动。
人道主义非政府组织“印尼医疗紧急救援委员会”此前为这家医院筹集了1260亿印尼盾（约930万新元）。

声明强调，此举是“在侮辱这座由印尼人民为巴勒斯坦人民团结共建的人道主义设施”。
印尼外交部说，根据国际人道主义法，所有医疗设施必须受到尊重和保护；雅加达呼吁以色列停止危及民用基础设施的行动。
印尼方面曾在2023年驳斥以色列关于这家医院位于哈马斯地下隧道网络之上的说法。

费城报道 — 特朗普与伊朗达成的“无限期”停火，究竟是通往和平的序曲，还是危险转折前的短暂平静？
美国总统特朗普于4月21日宣布与伊朗实现开放式停火，同时称伊朗政府“严重分裂”，并表示将继续维持海上封锁以施压。伊朗方面斥之为“毫无意义”，并对美方的海上封锁扬言将动用武力回应。
多位分析人士认为，尽管伊朗内部存在分歧，但此停火被视为伊朗的一次战略性“胜利”——它在避免直接冲突的同时保留谈判筹码。对美国而言，海上封锁构成实质性杠杆，但停火本身脆弱，任何战略或沟通失误仍可能引发新一轮敌对行动，尽管大规模战争的可能性已下降。

美伊在伊斯兰堡的第二轮和谈（原定4月21日）前景不明，能否达成持久停火堪忧，也令两国关系走向更加难测。
自2月28日美以对伊攻击以来，双边关系已遭到不可逆的结构性破坏，新一届伊朗领导层深知这一点，已不可能回到过去的“护栏”之下。
历任美政府虽将伊朗视为主要“恐怖输出国”，但多以克制手段应对。2019—2020年美军击杀苏莱曼尼后，双方仍曾有有限报复与谈判空间；但2025—2026年美以针对伊朗的规模性军事行动、以推翻政权为明确目标，正在摧毁伊朗长期依靠的“代理人网络”的遮掩作用。
尽管哈马斯与真主党自2023年后遭受重创、部分兵力与指挥层被削弱，二者仍保持军事行动能力。停火前夕，真主党在南黎巴嫩仍向以色列射击以示抵抗。
学者穆萨维安认为，美方反复用武力扼杀外交，使伊朗转向多重威慑：强化“抵抗轴心”、扩展代理人网络，甚至加快核能力讨论。伊朗被估有约440公斤高浓缩铀，若无法核查拆除，数年内研制核武的可能性令人忧虑。
西方亦担心伊朗在海外的“睡眠细胞”与同情者可能发动报复性袭击，欧盟已将伊朗革命卫队列为恐怖组织，美国亦破获刺杀特朗普的阴谋并有相关定罪。
东南亚方面，伊朗及其代理人曾在90年代至今对地区目标策划行动，并通过宣教与院校（如阿穆斯塔法大学海外分支）与社群联络，持续在马来西亚、新加坡等地培植同情者，近年新加坡亦对涉激进化者采取限制与拘留措施。
结论是：美以试图彻底消除伊朗及其代理人威胁的策略迄今未见成效，反而可能促使德黑兰采取更激进、全球化的报复与遏制手段，令地区与国际安全风险上升。
（作者：Kalicharan Veera Singam）

华盛顿 — 总统特朗普4月21日在CNBC采访中暗示，美国查获一艘疑载“来自中国的礼物”的船只，可能向伊朗运送致命军需，触及美方援助德黑兰的红线。他称“我们昨天查获了一艘船，上面有些东西，不太好——可能是中国送的礼物”，并表示曾与习近平有过某种谅解。白宫当日未回应置评请求。
中国外交部发言人郭佳昆在例行记者会上称，美方查获的是“外国集装箱船”，中方拒绝任何无端联想与猜测；中国驻美使馆则强调中国对军品出口谨慎负责，严格管控两用物项。
美国官员与情报显示（CNN报道称），中国或在数周内向伊朗交付新型防空系统，包括肩射防空导弹。特朗普此前称习近平在通信中保证中国不向伊朗提供武器。两国原定4月会晤因伊朗战争推迟，现计划5月中在北京会面。
分析人士指出，鉴于伊朗战事牵连与对中美关系的考量，特朗普不太可能因单次截获而与中国开展代价高昂的贸易对抗。美国为在会谈前施压德黑兰，已在霍尔木兹海峡拦截并掉头多艘船只——据美中央司令部称，已拦下一艘被制裁油轮和一艘伊朗货船，合计让28艘船返回。另有评论认为，中国对伊朗的低层次军事支持可能持续存在。彭博社报道，俄罗斯亦向伊朗提供情报与无人机相关支持，莫斯科与德黑兰近年军事合作加深。

贝鲁特 — 以色列入侵黎巴嫩南部、贝鲁特遭遇轰炸、逾120万人流离失所、教派对立加剧。对于经历1975—1990年内战的老兵与记者而言，2026年的冲突仿佛重演五十年前的惨况。
3月2日以色列与伊朗支持的黎巴嫩武装组织真主党爆发新一轮战事，进一步撕裂国内矛盾。短期停火旨在为黎以和平谈判创造条件，美国将主办第二轮会谈，但停火协议既未要求以军撤出，也未明确要求真主党解除武装，外交人士称这是“为国内对抗埋下的详细配方”。
曾在内战中并肩作战的68岁前战士齐亚德·萨布说，近期的轰炸让人瞬间回到内战场景，警告黎巴嫩人不要自相残杀。多名受访的什叶派民众将国家高层视为“背叛者”，指责政府未能保护他们；另有基督教前武装领袖拉菲克·巴泽尔吉担心未奠定良好基础的协议会重燃冲突，并指出年轻一代更易动员武装对抗。
内战遗留的分裂尚未和解，真主党自1982年成立以来一直保留武装，2000年以军撤出后势力扩张；2024年与以色列的战争使其受创，新政府曾承诺解除武装，兵力搜缴进展缓慢。分析人士警告，若裂痕继续激化，黎巴嫩可能重陷内斗泥沼。

自伊朗、以色列与美国爆发战争以来，首都德黑兰及其他主要城市出现了大量色彩鲜明、充满象征意味的壁画，主题多为抵抗与蔑视。
壁画和横幅长期构成伊朗城市景观的重要一部分，尤其在德黑兰中心广场，这类图像常与国家政治话语和对外政策相呼应，自1979年伊斯兰革命以来一直是传达意识形态与集体记忆的媒介。
近期作品包括：将美军航母画成载着覆盖美旗棺材的画面，周围有悬挂伊朗国旗的小艇和直升机，暗示海上优势——伊朗继续封锁战略性霍尔木兹海峡，该海峡和平时期输送全球五分之一的石油与液化天然气。
其他壁画有两臂缠国旗并比成心形的男子；伊朗国旗延展至墙面，其郁金香形徽章中嵌有导弹，旁有持横幅写着“我们都是为革命而来”的年轻女子，呼应1979年革命支持。
亦有作品描绘宗教领导系谱：从伊朗建国者鲁霍拉·霍梅尼起，继而为在2月28日美以首轮攻击中身亡的阿里·哈梅内伊，最终指向其继任者与子嗣莫杰塔巴·哈梅内伊。
另有壁画描绘伊朗无人机粉碎以色列旗帜上的大卫之星；一幅呈现示威者在自由广场集会、高呼“打倒美国”，有人举着阿里·哈梅内伊的画像。
在位于德黑兰市中心、现为“间谍巢穴”博物馆的前美国大使馆周边，长期存在显著的反美壁画——包括断臂的自由女神像碎片和用骷髅替代星星的美旗等图案。AFP

迪拜/巴黎 — 伊朗反对派领导人4月22日在欧洲议会批评欧洲国家对德黑兰近期处决行动沉默寡言，称未尽力制止这场所谓战时镇压言论自由的行动。
伊朗司法机关新闻机构Mizan当天报道，一名被控为以色列情报机构刺探并传递敏感情报的男子已被处决。
巴黎总部的伊朗全国抵抗委员会（NCRI）当选主席玛丽亚姆·拉贾维表示，“欧盟领导人与成员国在面对伊朗这一波政治性处决时保持沉默，难以辩解”。她称，这种沉默不仅助长当局继续处决，还向其发出软弱信号，促使其在核武发展与地区恐怖干预上持续作为。
拉贾维称，4月22日的处决之前一个月内已有约16名政治犯被处决，其中包括8名来自人民圣战者组织（PMOI）的成员。
报道并指出，1月反政府抗议后伊朗当局曾造成数千人死亡，为1979年伊斯兰革命以来最严重的国内动荡。美以在2月底对伊朗发动攻击时表示，部分目标是削弱当局以促成政府垮台。但在战争期间，国内组织性异议迹象有限，人权组织称当局对反对者大幅镇压。
总部在挪威的人权组织Iran Human Rights 4月21日称，自停火于4月8日开始以来，至少有3,646人被捕，至少报告了767起案件。国际特赦组织早前亦谴责，称在以美持续空袭、民众承受大规模伤亡之际，伊朗当局“将死刑武器化”以铲除异见并进一步制造恐惧。　来源：路透社

巴黎讯——天然气输出国论坛秘书长菲利普·姆谢尔比拉4月22日在巴黎表示，若伊朗战事持续，目前各国为应对危机而采取的天然气替代措施，可能从短期需求破坏演变为结构性变化。
他指出，自2月底中东危机爆发以来，Kpler数据显示全球被迫退出市场的原油与凝析油已超过5亿桶，为近代最大规模的能源供应中断。依赖海湾供应的国家因此转向燃煤并加速可再生能源部署，形成对天然气需求的即时冲击。
“如果冲突今天结束，世界将在6个月到1年内恢复；但若持续六个月，眼下这些应急性改变可能固化，”姆谢尔比拉说。他还表示，原本预计2026年将由供紧转向过剩的格局已被打乱，目前尚不清楚这是否仅是延迟，或是过剩永远不会到来。
姆谢尔比拉同时批评非洲天然气出口国未能抓住机会填补中东减产造成的空白。尽管部分国家在LNG和管道气上具备富余产能，但多数未满产，阿尔及利亚和利比亚对欧洲的出口管线并未满负荷，结果让北美生产者在欧洲和亚洲市场扩大份额。他总结道：储量在地下，但上游分量不足以填满现有基础设施。

伦敦 — 英国国家网络安全中心（NCSC）首席执行官理查德·霍恩4月22日在格拉斯哥举行的CYBERUK大会上警告，英国需为与敌对国家相关的网络攻击增多做好准备。
霍恩表示，NCSC平均每周处理约四起具有国家重要性的网络事件，最高影响力的攻击越来越多“直接或间接”与国家有关，包括中国、伊朗与俄罗斯，且这些活动正针对英国及其欧洲伙伴。尽管勒索软件等犯罪威胁仍是最常见风险，但最严重的事件多由国家行为者主导。
英国国内情报机构MI5称，自2022年以来已挫败逾20起与伊朗有关的阴谋。霍恩警告，若处于或临近冲突，英国可能面临大规模“黑客主义”行动，其破坏力可与重大勒索软件攻击相当，且无法通过付赎金恢复系统。
AXA XL的网络风险专家马修·库桑指出，随着美以与伊朗间冲突加剧，国家相关的网络行动可能上升。霍恩还称，人工智能的进步将加速攻击者发现漏洞的能力，但同样为防御提供工具。安全大臣丹·贾维斯呼吁领先AI公司与政府合作打造AI驱动的网络防御能力，邀请企业签署自愿的网络韧性承诺，并宣布未来三年追加9000万英镑用于加强网络安全、支持中小企业。

渥太华，4月22日 — 在上周赢得多数政府后，加拿大总理马克·卡尼面临的最大挑战是与特朗普政府重塑对美贸易关系。
加拿大、美国和墨西哥必须在7月1日前决定维持现有协议、重新谈判，或以年度审查方式维持至2036年到期。卡尼在美方压力要求让步的情况下，计划今年推动修订协议，重点解决针对加钢铝及汽车的关税问题。
卡尼在三场补选前强调需要议会多数以更有力应对美国的贸易战。政界与分析人士表示，卡尼必须兑现推动经济转型和与美方谈判的承诺。民调显示，加拿大人最关心经济与通胀，其次是对美关系。加拿大70%出口仍流向美国，卡尼曾承诺十年内将对其他市场的出口翻番，并在过去一年签署逾20项经贸与安全协议，正与印度谈判。
卡尼称与美关系的密切已成为“必须纠正的弱点”，并警告美方贸易政策已发生根本性变化、关税升至大萧条时期水平。美加尚未就北美自贸协定续约展开正式谈判。专家认为，多数席位使卡尼有资本在必要时作出不受欢迎的让步，如开放受保护的奶制品市场或说服省级政府调整政策。负责美加贸易的部长多米尼克·勒布朗表示，希望就行业关税与北美协议达成更大范围的协议；政府此前称奶制品供给管理不会成为谈判筹码。
反对党领袖皮耶尔·波利耶夫批评卡尼未能向美国关税“亮剑”。学者则敦促卡尼把重心转向国内事务，包括购置下一代战机、推动基础设施投资并应对不断攀升的生活成本。政府数据显示，加拿大在七国中失业率居第二、食品通胀最高。路透社报道。
