中国大使：新的一年中国期待和加国做相互尊重好伙伴
中国驻加拿大大使王镝大年初二在官媒《人民日报》撰文，形容这个春节对中加关系意义非凡，并称中国期待着新的一年和加拿大一道，坚定做相互尊重、彼此信任、相互协作的好伙伴。
王镝星期三（2月18日）在《人民日报》撰文称：“这个春节对中加关系意义非凡——顺应‘天时’，欢庆中加关系站在新的起点上；呼应‘地利’，推动春节文化在加拿大走入千家万户；彰显‘人和’，促进两国民心相通不断深化。”
王镝也说：“新的一年，中方期待着和加方一道，加快落实两国领导人重要共识，发扬跃马扬鞭的精神，汇聚万马奔腾的合力，坚定做相互尊重、共同发展、彼此信任、相互协作的好伙伴。”
加拿大总理卡尼上月访问中国，是自​​2017年以来加拿大总理首次访华。他寻求扩大加拿大的出口市场，以降低对美国的贸易依赖。

王毅当时也说，中国愿同加拿大一道努力，排除干扰，重启各领域交流合作，推动中加关系健康、稳定、可持续发展。
中国外交部也在官网公布，中国决定星期二（17日）起，对加拿大、英国持普通护照人员实施免签政策，两国持普通护照人员赴华经商、旅游观光、探亲访友、交流访问、过境不超过30天，可免办签证入境。上述免签政策实施至2026年12月31日。

美伊第二轮间接谈判取得进展 双方同意继续接触
美国和伊朗第二轮间接谈判星期二（2月17日）在瑞士日内瓦举行。美伊官员在谈判结束后说，尽管分歧仍存，但谈判较上一轮取得进展，双方同意继续接触。 
新华社报道，第二轮间接谈判在阿曼驻日内瓦外交代表机构内举行。美国总统特朗普的特使威特科夫和女婿库什纳作为美方主要代表出席谈判，伊朗外长阿拉格齐率领的外交和技术代表团代表伊方出席。阿曼外交部长巴德尔居中斡旋。
巴德尔在社交媒体发文说，伊朗与美国当天的间接谈判就“确定共同目标和相关技术问题”取得“良好进展”。巴德尔说，“还有许多工作要做”，各方明确了下次会议前的后续步骤。
阿拉格齐在谈判结束后说，本轮谈判较上一轮取得积极进展，双方就一系列“指导性原则”达成总体共识。目前双方立场仍存在差距，需要进一步沟通，但谈判前景较此前“更加清晰”。他说，双方将就潜在协议文本开展工作并交换意见，随后再商定第三轮谈判时间。
当天晚些时候，阿拉格齐在万国宫举行的裁军谈判会议上发言，谈及与美国的第二轮谈判，认为“一个新的机遇之窗已经开启”。他同时强调，美国应立即停止以动武威胁伊朗。伊朗始终全面准备好抵御任何威胁或侵略行为。任何针对伊朗的攻击，后果都不会局限于伊朗境内。
美国副总统万斯星期二说，此轮美伊谈判在部分议题上进展顺利，双方同意继续接触。美方向伊方划定一些“红线”，核心目标是“不希望伊朗拥有核武器，不希望核扩散”，但伊方“尚未真正认可并愿意协商解决”。他强调，特朗普在处理伊朗核问题上有多种选项，包括外交谈判和军事手段。
另据一名不愿公开姓名的美方官员向美国新闻网站Axios披露，伊朗方面提出，将在两周内带着“详细提议”重返谈判，以期弥合美伊双方“立场上存在的一些分歧”。这名官员说，美伊当天在日内瓦举行的谈判“取得进展”，但“仍有许多细节需要讨论”。

俄罗斯塔斯社引述消息人士的话报道，俄美乌三方首日会谈气氛紧张，其间还举行了俄美、俄乌会谈。

美国最繁忙海港负责人：对华出口前景黯淡
美国最繁忙的海港洛杉矶港执行董事塞罗卡（Gene Seroka）透露，洛杉矶港1月出口量同比下降8%，创近三年来单月最低水平，并称“对华出口形势看起来很糟糕”。
据路透社报道，塞罗卡星期二（2月17日）说，经洛杉矶港运往中国的大豆去年大减80%；即便两国代表在亚太经济合作组织峰会期间会谈后，去年11月和12月的贸易表现仍未见起色。
去年，美国总统特朗普强硬运用关税措施，冲击全球贸易；中国及其他国家随即实施报复性关税，也令美国出口商，尤其是农民承受沉重压力。
彼得森国际经济研究所高级研究员鲍恩（Chad Bown）指出，从牛肉、玉米到原油、煤炭，美国对华出口在2025年全面下滑，“如今美国对中国的出口已所剩无几”。
进口方面，洛杉矶港1月进口量为42万1594个标准箱，同比下降13%。塞罗卡说，2月进口量目前与去年同期大致持平，但受中国工厂农历新年停工影响，3月进口预计放缓。
不过，他仍预计今年第一季度港口总吞吐量同比降幅将低于10%。去年同期，美国进口商在特朗普威胁对包括中国在内的国家加征关税前集中备货，抬高了比较基数。
