中国4月原油进口创四年新低
随着霍尔木兹海峡关闭扰乱全球原油供应，中国4月原油进口量跌至近四年来最低水平。
中国海关总署星期六（5月9日）公布的数据显示，4月原油进口量同比下降20%，至3850万吨，为2022年7月以来最低水平。
作为全球最大石油进口国，中国约一半原油进口来自中东地区。随着霍尔木兹海峡关闭，运送原油及成品油的油轮数量大幅减少，影响全球能源运输。
中国官方当天公布的数据并未区分海运原油与通过管道进口原油。不过，船舶追踪公司Kpler的数据显示，中国4月海运原油进口量为每日803万桶，同样创下2022年7月以来新低。
尽管进口下降，船舶追踪机构Vortexa估计，中国4月原油库存仍增加1700万桶，但预计5月库存将回落。
此外，中东局势动荡也促使中国加强管控汽油和航空燃油等成品油出口，以保障国内市场供应。在相关政策影响下，中国4月成品油出口降至约310万吨，创近十年来最低水平，较3月减少约三分之一。
不过，这一数据可能仍高估中国对亚洲及其他地区的实际出口量，因为统计中包括了运往香港的成品油，而香港通常是中国成品油的重要目的地，并不受出口限制措施约束。
此外，中国4月天然气进口量同比下降13%，至842万吨，但官方数据同样未区分海运液化天然气（LNG）与陆路管道天然气。据悉，中国从中东海湾地区进口大量液化天然气。
尽管4月进口显著下滑，中国今年前四个月原油进口总量仍达1亿8530万吨，同比增加1.3%。

美国国际贸易法院于5月7日裁定，总统特朗普依据1974年贸易法第122条对全球进口商品征收的10%关税缺乏正当依据，但法院以2比1的意见仅对两家私人进口商和华盛顿州下达限制令，未对所有进口商普遍停止征税。 
裁决意味着在美国政府上诉期间，大多数进口商仍须缴纳临时关税，这些关税预计将于7月到期。法院认为第122条并不适用于特朗普在2月命令中所援引的贸易逆差情形，认定当局适用该条款不当；但另一名法官指出，现在对原告全面胜诉为时过早。 
两家私营企业（玩具商Basic Fun!与香料进口商Burlap & Barrel）以及华盛顿州提出诉讼，指控该措施是对最高法院此前以紧急权限法案撤销其先前关税后的规避。华盛顿州出具证据称通过华盛顿大学支付了关税。 
白宫指责法庭裁决并称将上诉，政府仍打算通过1974年贸易法第301条对主要贸易伙伴恢复关税，并有三项301调查将于7月完成。法律界预计此案将进入联邦巡回上诉法院乃至最高法院的进一步审理，是否能获得更广泛的退款令仍待观察。

专家普遍认为特朗普不太可能在短期内对台做出重大让步，但若出现暂停军售或模糊表态，都会在台海与台海民心中产生严重影响，而特朗普的不可预测性也让最终结果难以预判。
