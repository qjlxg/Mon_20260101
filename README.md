北京大学研究团队研发出一款模拟（analogue）AI芯片，研究者称该芯片已能处理真实世界数据任务，有望减少对高能耗数字处理器的依赖。研究成果周一发表于学术期刊Nature Communications，第一作者孙忠（Sun Zhong）等人表示，芯片在速度上较先进数字处理器提升约12倍，能效提高逾200倍。
论文指出，团队在推荐系统训练中的数据规模与Netflix、Yahoo可比；图像压缩测试显示，重建图像在视觉质量上几乎与全精度数字计算相当，同时存储需求减半。审稿人称，与传统数字芯片相比，实验结果在速度和能效方面实现了数量级的改善，显示出产业应用潜力。
研究团队采用非负矩阵分解（NMF）算法，基于阻性存储的类模拟计算芯片并重构核心电路，将算法中最耗时的部分在一次运算中完成，从而用更少计算单元实现同样结果，显著缩小芯片体积并降低能耗。孙忠在社交媒体上表示，该研究“将模拟计算的边界向前推进了一步”，并回顾他对1999年首次提出的NMF方法的长期关注。
文章也提到，随着材料、器件设计和算法进步，模拟硬件重新受到关注，因为AI系统中内存与计算之间的数据迁移已成为关键瓶颈。研究团队此前在去年报告中曾展示，模拟芯片在基础数学运算上比顶级数字处理器（如Nvidia H100）快上数百到上千倍并更省能。

科伦坡消息（2026年1月26日）——政府医务官员协会（GMOA）宣布，自今日（26日）上午8时起在全岛发动无限期工会行动。协会称此举系因卫生部未落实先前达成的相关解决方案。
GMOA新闻发言人Dr. Chamil Wijesinghe表示，本次行动将通过五项工会措施实施：
1. 不为门诊与诊所中院内无药可开具需到外部药房购买的处方开具处方。
2. 不为院内无法提供的化验项目开具须到外部化验所或私立医院检验的建议或处方。
3. 若医院系统内获批的新单位无法配足医生或未获审批，不予支持设立。
4. 不配合为政治目的或政治需求举办的部分诊所与义诊活动。
5. 如未安排协助医务人员协助查房和检查，医生将退出医院、诊所和门诊服务。
Wijesinghe强调，这些措施并非拒绝工作，医生仍将在可保障的条件下提供诊疗；若必要条件不被满足，免费医疗体系的崩溃责任将由卫生部、卫生部长及政府承担。GMOA并将于1月28日召开中央委员会会议，届时将决定是否扩大和加剧后续工会行动。
