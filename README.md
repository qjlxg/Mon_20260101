一个月内中国猛打十“虎”
中国官媒报道，官方在一个月内猛打十“虎”，其中包括中央军委副主席张又侠、中央军委联合参谋部参谋长刘振立。
据中新社报道，随着中共中央纪委国家监委上星期六（1月31日）通报应急管理部党委书记、部长王祥喜被查，今年以来中国官宣“落马”的中管干部已达八人。

武吉查卡设100个自动通关闸门 提升新柔地铁通关效率
马来西亚境内衔接新柔地铁系统的武吉查卡移民、关税、检疫综合大厦，将设有100个自动通关闸门，以简化新加坡和马来西亚的跨境通关手续。
《星报》报道，马来西亚捷运公司（MRT Corp）总裁莫哈末扎里夫说，电子闸门是“一体化隔离检疫”（ICQ）模式的核心组成部分，允许乘客在同一设施内完成两国的入境手续。
他说，高容量电子闸门系统旨在支持高效的乘客通行，并减少拥堵，尤其是出行高峰期。
莫哈末扎里夫对马新社说，除了100个自动通关闸门，武吉查卡移民、关税、检疫综合大厦也将设有十个安检通道进行人身检查，18台行李扫描器，以及为没有携带行李的乘客提供的专用通道，所有安排旨在最大程度地减少通关拥堵情况，确保乘客顺畅通关。
莫哈末扎里夫说，在马来西亚和新加坡移民局的密切合作下，自动通关闸门包括二维码通关设施，将减少当局对人工检查的依赖，并为乘客提供更可预测的跨境通关体验。

【沃什提名终结“美元贬值论” 大宗商品遭遇获利回吐转折点】金十数据2月2日讯，Maybank分析师表示，特朗普提名凯文·沃什领导美联储，为市场平仓极端头寸提供了所需的触发因素。他们在报告中写道：“这一提名缓解了市场对特朗普的人选可能是一名忠实拥趸的担忧，因为这类人选可能会以提高通胀风险为代价过度下调政策利率。”沃什过去一直是美联储的批评者，他认为美联储应该只承担物价稳定这一项职责。他意外获得提名促使市场参与者放弃了美元贬值的论调，导致金、银、铜在经历1月份的大幅上涨后悉数暴跌。但Maybank表示，价格走势暗示出现更剧烈逆转的可能性似乎仍然较小，预计阻力位在97.50。

【菲律宾南部沉船事故死亡人数升至43人】金十数据2月2日讯，菲律宾海岸警卫队发布消息称，2月2日，南部沉船事故的联合搜救队打捞出3具遗体，这使得该事故的死亡人数上升到43人。1月26日，菲律宾南部发生沉船事故，海警称事故船只共载有344人，有316人获救。但目前获救人数和遇难人数与此前报告的总人数不符。海警目前正在进行调查，以确认事故船只当时搭载的总人数。
