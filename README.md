工總CEO調查：企業界稱貪腐惡化，標案鎖規格與圍標最常見
泰國工業聯合會（FTI）第49次CEO調查（2016年3月，645位企業主管、涵蓋48個產業與76個省分會）顯示，85.7%受訪者認為當前貪腐較以往嚴重。受訪者認為主要成因為裙帶文化與利益衝突（70.9%）、法規缺口與執法不力（57.2%）、公務人員高裁量權易遭濫用（53.0%），及政治干預（49.6%）。
在貪腐形式上，最高比例為採購領域的鎖規格與圍標（81.4%），其次為賄賂與收受好處（72.6%）、政策性貪腐以利特定團體（69.8%）和買賣職位（40.2%）。超過55.5%受訪者估計，貪腐使營運成本增加逾20%；61.2%表示曾親身遭遇索賄或貪腐事件。
受訪者建議優先作法包括：改革與強化法規執行（64.5%）、推動數位政府與公開資料以減少裁量（60.2%）、擴大公私參與並保護檢舉者（55.5%）、加強政策/專案的前中後審查（54.9%）。FTI呼籲以系統性、持續的措施恢復信心，私部門表示願意協助反貪。

迪拜讯 — 阿联酋财政部自2026年4月1日起修订税务程序法规，以配合自2026年1月1日生效的相关税法修订，推出有关申报、退税与审计档案管理的新规。
修订明确了自愿申报的提交程序，并将退税程序适用于对纳税人有利的任何余额。法律还调整了向有权政府部门披露信息的机制，重申数据保密保护，并界定信息使用的范围与限制。
对于在诉讼时效届满前已提出退款申请且主管机关尚未作出裁定的税期，记录保存期限延长两年。新规还引入了为税务稽查或检查延长保全或查封文件、资产期限的可能性。
财政部表示，此次修订旨在提高透明度、促进纳税人合规，确保税务程序的准确性与完整性，同时保障纳税人权利。
（最后更新：2026-04-01；作者：Justin Varghese）

泰国数字经济与社会部下令，泰国邮政（Thailand Post）将把寄递费率维持至2026年4月15日，以减轻民众在宋干节期间的生活负担；因中东冲突推高能源与运输成本，自2026年4月16日起，将对部分商业服务暂时调涨价格，主要涉及EMS与电商经济型服务e‑CoPost。
受中东局势影响，国内外运输成本普遍上升，多家物流业者宣布调价，泰国邮政亦受波及。该部要求常务秘书密切关注事态，特别是对民众生活的影响，并指示在调整前优先考虑减轻民众负担。
据悉，价格冻结将持续至宋干节高峰期结束的4月15日；如自4月16日起调整，将为临时性措施，直至情势恢复正常。
将被调涨的项目为：快递EMS（仅针对部分重量区间）及e‑CoPost（电商经济型邮寄）；而基础邮政服务如信件、邮包/印刷品、挂号信及普通包裹则不在此次调价范围内。

迪拜 — 迪拜警方近日通过社交媒体发出提醒，敦促公众不要传播未经核实的信息，警告此类做法可能破坏社区安全与稳定。
警方指出，信息在当今传播速度极快，但并非所有网络内容都属实，一些信息甚至被有意散布以制造恐慌。警方强调“我们分享的内容很重要，依赖未经核实的来源可能引发不必要的恐慌，削弱社区的安全与稳定。”
为维护公共安全，警方呼吁居民在数字平台上谨慎行事，及时核实事实，避免转发谣言，并通过正规渠道确认消息。
阿联酋公诉机关也发出严正警告：转发或散播来源不明的谣言将承担法律责任；无论是否为原始发布者，分享未经核实的内容均可能面临追究。
当局重申，为保护自己和他人安全，公众应以官方渠道为准，谨慎传播信息。
