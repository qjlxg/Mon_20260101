鲁比奥称美国对台政策未改变 台外长表达感谢
在美国总统特朗普访华之际，台湾外交部长林佳龙感谢美国国务卿鲁比奥重申美国对台政策并未发生改变，并称台湾将持续强化自我防卫能力，维护台海和平稳定。

林佳龙说，台湾作为国际社会负责任的一员，将持续强化自我防卫能力，并坚定和美国等所有热爱自由民主的国家合作，共同维护台海与区域的和平、稳定与繁荣。
中国大陆国家主席习近平星期四（14日）在北京人民大会堂与特朗普举行会谈。鲁比奥在会后接受美国全国广播公司（NBC）采访时说：“截至今天，以及截至我们今天在这里举行会谈后，美国在台湾问题上的政策没有改变。”

警方同时提醒网民，网络不是法外之地，“不信谣、不传谣，不以博眼球、赚流量为目的编造散布虚假信息，共同营造清朗网络空间”。

新华社报道，阿根廷卫生部星期四（5月14日）确认，这支调查团队将在下周前往火地岛省首府乌斯怀亚，即暴发汉坦病毒疫情游轮洪迪厄斯号（MV Hondius）的出发地。

习近平向特朗普提普京 称“极少”在中南海接待外国领导人
美国总统特朗普在中南海同中国国家主席习近平散步交谈时，提问习近平是否接待过其他外国领导人到访此处。习近平回应说“很少”“极少”，并补充称“比如说，（俄罗斯总统）普京。”
习近平星期五（​​​​5月15日）上午在中南海同特朗普举行小范围会晤。两人边走边谈，期间驻足观赏园中古树。
视频画面显示，习近平和特朗普穿过云窦门后，对特朗普说：“你看看这棵树，这是一个两棵树连在一起了，也是100多年前的一棵树。”
特朗普通过华语翻译以“非常不一般”回应后，习近平继续说：“我告诉你，这边的树全部是在两三百年以上，那边还有一个400多年的树。”
特朗普随后询问“这些树真的能活这么久？”，习近平回应说：“是的，还有1000年的树，在其他的地方。”
听完习近平对古树树龄的介绍后，特朗普通过翻译提问：“那其他的国家来的元首、总统或者是总理什么的，您作为国家主席也都会到这边来接待他们吗？”
习近平摇头回应：“很少。这个地方我们首先以前外事活动都不在这里，后来有外事活动以后，极少。”当翻译人员向特朗普翻译完这句话后，习近平补充道：“比如说，普京。”特朗普随即回应：“很好，我喜欢。”
习近平随后继续边走边细数两旁树木的树龄，特朗普通过翻译说“很美的地方，我很喜欢，我喜欢这地儿，我这地儿能待的惯，我可能不想走了。”
作为毗邻故宫、戒备森严的中枢园区，中南海历来仅接待少数美国领导人。1972年，美国首位访华总统尼克松在访问期间曾在此与中共开国领袖毛泽东会面；前总统小布什分别于2002年和2008年两次到访；前总统奥巴马则于2014年访问。特朗普由此成为第四位进入中南海的美国总统。

根据独立研究机构IEEFA于5月13日发布的报告，2026年第一季度欧盟从俄罗斯进口的液化天然气（LNG）达到了自2022年俄乌战争以来的最高水平。
报告显示，欧盟在2026年前三个月自俄进口天然气约69亿立方米，较2025年同期增长16%；IEEFA并称4月份进口同比亦增长17%。尽管通过管道的俄罗斯天然气几近停止，但法国、西班牙和比利时等国仍通过油轮大量采购俄罗斯LNG。

国际货币基金组织（IMF）表示，欢迎美中两国高层展开建设性对话，并认为缓解两国紧张与不确定性有利于两国及全球经济。
IMF发言人朱莉·科扎克在华盛顿新闻发布会上就特朗普与习近平的初步会晤成果表示，双方在最高层交往十分重要，任何有助于降低贸易紧张和不确定性的举措，都将惠及两国并利好全球经济。
科扎克指出，由于中东冲突及霍尔木兹海峡局势导致原油价格维持在每桶100美元以上，IMF在4月《世界经济展望》中列出的三种情形正朝中间的“逆风情形”演进。该情形下，全球实际GDP增速今年将降至2.5%，而更温和的参考预测为3.1%，2025年预计为3.4%。
IMF称该逆风情形假定全年油价为每桶100美元，并伴随金融状况收紧与通胀预期上升。科扎克表示，尽管短期物价预期受推高，但中期通胀预期总体仍然锚定，全球金融环境仍较为宽松。
基金组织正与成员国讨论对受高能源和大宗商品价格冲击的国家提供可能的财政援助，但未披露具体国家或细节。此前IMF总裁格奥尔基耶娃在春季会议上曾估计，至少有12个国家可能需要国际货币基金组织与世界银行合计约200–500亿美元的援助规模。IMF同时建议各国避免大规模燃料补贴，以免耗尽财政资源并推高油价。

美国华盛顿 — 5月13日，美国参议院的共和党人阻止了由民主党推动、要求未经国会授权即结束对伊朗战争的最新决议，但随着第三名共和党参议员投赞成票，该决议仍更接近通过。参院以50比49投票不推进这项由俄勒冈州参议员杰夫·默克利提出的战争权力决议，投票几乎沿党派划分进行。肯塔基的兰德·保罗、缅因的苏珊·柯林斯和阿拉斯加的丽莎·墨科斯基三名共和党人加入了除宾夕法尼亚民主党人约翰·费特曼外的全部民主党人行列支持该议案。
这是今年参院第七次由共和党人阻止类似决议，也是自5月1日冲突触及《1973年战争权力法》规定的60天期限以来的首次投票。该法规定，总统在未获国会授权时只能发动60天军事行动，或在因“不可避免的军事必要性”撤军时请求延长30天。特朗普此前称停火“终止”了敌对行动，民主党人则反驳称冲突仍在继续，指出美方封锁伊朗港口并袭击伊朗船只，伊朗封锁霍尔木兹海峡并攻击美方船只和油轮。默克利称“双方仍在交战，60天时钟不应被中止”。
参院民主党人表示将于下周及以后每周再次提出相同决议，直到战争结束或特朗普向国会寻求授权。众院民主党人也提出了相关决议，但同样被共和党阻止。民主党强调宪法将宣战权赋予国会，警告总统可能将美国拖入长期冲突；共和党和白宫则称总统作为三军统帅有权下达有限军事行动。路透社
