阿姆斯特丹拟逐步调高旅游税至20%
（阿姆斯特丹综合电）为了应对过度旅游对城市环境和居民生活造成的影响，荷兰首都阿姆斯特丹拟在明年大幅提高旅游税至16%，并逐步提高至20%，以使用旅游税收打造一座更清洁、更安全、更宜居的城市。
阿姆斯特丹市政府星期四（6月4日）发布新的联合协议，划出2026至2030年的工作重点，其中包括调高过夜旅客旅游税、关闭邮轮码头、管控旅游商店，以及放弃搬迁红灯区等多项旅游业相关政策。
据荷兰新闻网报道，市政府提出，在明年将旅游税提高至16%，并在接下来以每年1个百分点的幅度增长，直到达到20%。这将使城市的旅游税收在明年达到6000万欧元（约8900万新元），并在2030年达到7500万欧元。 
市政府说，这笔钱将确保游客对城市的管理、维护、执法和投资作出公平的贡献。
阿姆斯特丹已是全球旅游税第四高的城市。自2024年起，市政府向过夜旅客征收12.5%的旅游税，一日游游客则需支付15欧元。
尽管如此，阿姆斯特丹每年仍有超过2000万人次过夜旅客，以及大量一日游游客。市政府强调，虽然旅游业为城市经济作出贡献，但也给公共空间、生活质量和市政设施带来巨大压力。

阿联酋称遭来自伊朗的弹道/巡航导弹与无人机攻击，富查伊拉一处能源设施受创，三名印度人受伤；阿曼布卡一处民宅也有两人受伤。阿联酋外交部警告这是危险升级，威胁国家安全。

油价应声上涨，7月交割布伦特原油一度涨逾5%。阿联酋宣布学校改为线上授课至本周末，以色列军方进入高度戒备状态。
阿联酋称四枚巡航导弹由伊朗发射，三枚被拦截；另称伊朗无人机袭向与阿布扎比國營石油公司有关的油轮。伊朗方面将事态归咎于美军在海峡的行动，但伊朗外长阿拉格奇呼吁政治解决，警告不要被好战势力拖入泥潭。
特朗普表示将通过“自由计划”护送中立船只并已派驱逐舰经过海峡，称已有两艘挂美旗商船驶出；但伊朗革命卫队否认近期有商船通过。韩国方面称一艘韩国船只在海峡发生爆炸起火。
此外，停火也在黎巴嫩战线承压；以色列轰炸与地面行动已致黎巴嫩大量伤亡，双方在南部继续交火，黎巴嫩总统要求在会见以色列总理前达成安全协议。

伦敦报道—总统唐纳德·特朗普威胁对被指未能遏制强迫劳动的贸易伙伴征收新关税，但专家、商界及部分人权组织认为此举难以遏制现代奴役，甚至可能适得其反。
美国贸易代表办公室提议对60国进口商品加征10%或12.5%关税，理由是这些国家未能阻止强迫劳动的商品贸易。新加坡贸易与工业部6月4日表示拒绝美方指控，并称三分之一对美出口或遭12.5%关税。
此举源于301条不公平贸易调查，旨在恢复今年2月被美国最高法院推翻的紧急关税。国际劳工组织最新估计显示，全球被强迫劳动者达2760万，比2016年增加约270万，近半数集中在制造、建筑、农业、渔业和采矿等出口相关行业。
批评者指出，美方措施更以贸易规模与地缘政治为导向，而非按剥削严重性精准制裁，未必能改善情况，反而可能激化部分国家的政治抵抗。欧盟反驳称其将于2027年生效的强迫劳动法规程序更为严格，覆盖范围广，且与美国在2025年达成的贸易协定将大多数欧盟商品对美关税上限定为15%。国际商会与人权组织也对关税的任意性与有限效果表示担忧。

伊斯坦布尔讯——过去二十年国家大力投入，使土耳其成长为无人机及军工出口大国。土耳其现向近40个国家，主要为海湾、非洲、亚洲及部分欧洲国家，提供被视为更便宜、交付更快且更易定制的武器装备。路透社统计显示，自2021年以来，土耳其防务出口已超过三倍，至2025年达100亿美元，约占该国出口总额的3.7%，对欧美出口近四倍增至56亿美元。 
本国成熟的军工体系包括Baykar、土航（Turkish Aerospace Industries）、Arca Defense与Kale等企业；持续的国家支持、灵活供应链与为买家定制系统的能力，使其能迅速填补西方供应短缺或冗长采购周期留下的市场空白。土方防务机构表示，目标在两年内将防务出口翻倍，以获取偿债与研发资金。 
土耳其地处乌克兰与伊朗之间，安全利害相关，其在防空、战机与坦克发动机等领域的不足亦可通过贸易与技术合作弥补。哈德逊研究所资深研究员Can Kasapoglu称，土耳其通过出口先进系统，尤其是空中无人机，实现了“重大跨越”。 
土耳其约占全球武装无人机使用量的65%，也是弹药重要出口国，正规划或生产护卫舰、航母、防空系统与装甲车。印尼2025年拟购48架仍在研制的土耳其战机。五月在伊斯坦布尔展示的一款洲际弹道导弹原型及其宣传片引发外界质疑。 
土耳其盼借7月7–8日在安卡拉举行的北约峰会（并拟接待美国总统特朗普与其他领导人）扩大在西方市场的军售与联合生产，但在欧盟面临成员国专属防务项目与政治阻力——包括对土外交分歧与民主倒退的担忧，令其在欧安保行动中多被排除。尽管如此，土耳其仍与波兰、西班牙、葡萄牙、罗马尼亚达成供货协议，Baykar收购意大利Piaggio Aerospace并与莱昂纳多合作生产。 
斯德哥尔摩国际和平研究所指出，全球军费在2021–2025年增长24%至近2.9万亿美元，欧洲增幅达75%。同期土方防务机构将资助的研发项目增至1400余项。防务分析师Arda Mevlutoglu表示，如能克服政治障碍，欧洲与土耳其的更深层合作具备条件，但需高层政治意愿推动。路透社报道。
