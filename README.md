美国商会：中国政策威胁七国集团6500亿美元工业产值
据美国最大商业游说团体发布的报告，中国的产业战略正威胁全球最发达经济体数千亿美元（1亿美元相当于1.27亿新元）的工业产值，并可能导致这些国家的工业能力被掏空。
据彭博社报道，美国商会星期一（5月11日）发布、由荣鼎集团撰写的报告称，中国的政策“正变得更加系统化且无孔不入”。报告指出：“北京正积极利用监管和经济胁迫手段，加强对价值链的控制。”
报告称，由于北京推行更大规模的产业政策，发达经济体面临制造业竞争力持续被削弱的风险，因此应加强协调，以应对这一威胁。报告还称，汽车、机械和化工等行业尤其容易受到中国竞争对手的冲击。
报告也提到，如果中国继续以目前的速度扩大市场份额，到2030年，七国集团（G7）约6500亿美元的制造业出口，即约占其制造业出口总额的12%，可能直接受到中国竞争的冲击。
北京的“中国制造2025”计划，已使中国在全球一些最尖端的产业中占据主导地位，同时加剧了与美国及其他贸易伙伴之间的紧张关系。中国已在太阳能电池板、高速铁路和锂电池等关键技术领域取得全球领先地位，并在制药和人工智能（AI）等其他领域迅速追赶。
美国商会呼吁发达国家加强协调，共同应对来自中国的威胁。报告写道：“尽管许多政府已开始通过贸易防御措施、产业政策以及供应链‘去风险化’等方式作出应对，但相关措施仍然零散且缺乏协调。”
报告作者警告说，如果缺乏更协调的行动，中国的产业政策可能继续重塑全球市场，进一步加深各国对中国的依赖，并削弱发达经济体和新兴经济体的产业竞争力。他们说，工业化国家可能面临的后果，包括“投资减少、创新生态系统被削弱，以及产业能力流失”。

中国官方公布平台发布短视频必选六类标签
中国官方公布，网站平台发布短视频必选六类标签。
中共中央网信办星期二（5月12日）在官网公布，针对部分短视频内容来源不清、真假难辨、混淆视听等突出问题，今年以来中央网信办全面部署推进规范短视频内容标注工作。
中央网信办说，3月指导抖音、快手、腾讯、小红书、哔哩哔哩、微博、淘宝、京东、拼多多、支付宝、美团、百度这12家平台，先行先试，完善内容标注标签，对内容标注功能进行优化和测试。
中央网信办续称，总结12家平台试行经验，指导各地各网站平台全面推进落实三项工作，其中一项是规范短视频内容标注标签，明确网站平台必须为用户提供六类“必选标签”，并可根据自身实际提供其他“可选标签”。
“必选标签”包括“含有虚构演绎内容”“含有AI生成内容”“含有营销信息”“内容为转载”“内容为个人观点”和“无需标注”。真实生活记录类短视频可选择“无需标注”标签，该标签不在短视频页面呈现。
其他事项还包括：将内容标注设为短视频发布必经环节，发布者必须从“必选标签”中选择一项，才能发布短视频；对新增短视频标注情况加强审核，对存量短视频进行分批回溯，对未规范标注的，进行补标或纠正，并对相关发布者进行教育警示，推动实现短视频内容应标尽标。
中央网信办也说，将加大对网站平台的指导督促和监督检查力度，对未按要求进行标注的账号和主体责任落实不力的网站平台，依法严惩并公开曝光。

泽连斯基称，乌方还将在无人机合作框架下与世界其他地区启动安全合作（未指明具体地区），并已通过这些协议开始获得所需燃料。他希望“无人机外交”能为乌克兰争取中东国家的能源供应和农产品市场。来源：路透社

布鲁塞尔 — 欧盟外长5月11日就新一轮制裁达成一致，针对在被占领的约旦河西岸对巴勒斯坦人施暴的以色列定居者以及若干哈马斯高层人物，欧盟外交与安全政策高级代表卡娅·卡拉斯（Kaja Kallas）宣布。制裁名单涵盖三名定居者和四个定居者组织，具体身份尚未公开。该方案此前因已经在四月大选中落败的匈牙利前政府阻挠数月。欧洲各国对约旦河西岸定居者暴力事件上升表示关切。卡拉斯在社交平台X上称，打破僵局、付诸行动“正当其时”，并警告“极端主义与暴力必将承担后果”。以色列外长吉迪恩·萨尔在X上反驳，称欧盟此举“任意且带有政治动机”，无根据地因政治观点对以色列公民和实体实施制裁，并指责将以色列公民与哈马斯恐怖分子相提并论是“完全扭曲的道德等同”。哈马斯尚未立即回应。路透社报道。

北京 — 白宫高级官员5月10日表示，美国总统特朗普本周将赴北京，与中国国家主席习近平就中国对伊朗战争的立场，以及拟设立的新贸易与投资“理事会”细节进行交涉。两国领导人定于5月14—15日会谈。 
官员称，峰会上将讨论中国对伊朗的石油收入、可能的武器输出等敏感问题。美国已对多家涉购伊朗石油或向伊朗提供卫星影像的中国企业实施制裁。财政部长斯科特·贝森特将于5月13日在首尔与中国副总理何立峰进行最后磋商。 
中国外交部发言人郭嘉坤表示，两国元首将就中美关系及世界和平与发展等重大议题交换意见，强调在平等、尊重和互利原则下扩大合作、妥善管控分歧。新华社称，中美联合行动于4月破获跨境贩毒网络并逮捕五名嫌疑人。 
台湾问题、人工智能安全与双方可能开辟的新沟通渠道也将在议程中讨论。此前Anthropic新模型暴露出的网络安全风险，已促使美英及亚洲监管机构加紧关注。习近平与前总统拜登曾就AI风险达成相关限制性承诺。 
白宫表示，特朗普还可能借此宣布北京拟购美国产品的计划，并就农业、航天与能源等领域达成若干安排。分析人士称，首尔的最后会谈暗示双方或有出乎预期的成果。 
行程方面，特朗普将于5月13日上午抵京，14日参加迎接仪式并会见习近平，下午参观天坛，晚间出席国宴；15日将共进茶宴与工作午餐后离京。
