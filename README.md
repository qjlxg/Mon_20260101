调查：超八成香港青年认同盛事经济重要性
一项调查显示，超过八成香港青年认同盛事经济对社会发展的重要性，接近八成认为盛事与自身息息相关。
据《星岛日报》报道，香港青协青年研究中心成立的青年创研库星期四（3月5日）公布“从青年视角看盛事发展”研究报告。
报告显示，86.2%受访者认同盛事对社会发展重要；79.4%认为盛事与自身息息相关；另有74.2%表示盛事有助提升对香港的归属感。
研究发现，82.9%受访者过去一年曾以观众或参观者身份参与盛事；42.3%受访者有意担任盛事义工。
青年创研库青年发展组召集人潘子锋说，香港盛事发展已有相当基础，但目前盛事范围广、资讯分散，欠缺统一识别，建议政府推出香港盛事官方标志，授权认可活动使用以作品牌认证。
青年创研库于去年12月10日至14日，以实地问卷调查方式，访问了520名15至34岁青年，得出上述报告。

斯里兰卡打捞出被美军击沉伊朗军舰84具遗体
斯里兰卡海军宣布，在该国海域以南、超出斯里兰卡海域管界的深海处发现并打捞出84名来自一艘遭美军攻击后沉没的伊朗舰艇的遗体。海军表示，搜救工作仍在继续。
遭袭沉没的舰艇为伊朗海军军舰IRIS Dena。海军称，该舰在遭到一艘美国潜艇攻击后沉没，事发海域位于斯里兰卡以南深水区。该舰据称共有约180名官兵，曾赴印度维萨卡帕特南参加国际舰队检阅和联合演习，返航途中发生此次事件。
美国国防部长皮特·赫格塞斯特（Pete Hegseth）已对外表示，美方在印度洋用鱼雷击沉了一艘“以为在国际水域安全”的伊朗军舰，称其为“安静的死亡”，并称这是二战以来首次用鱼雷击沉敌舰，但未透露被击沉舰名。
斯里兰卡海军早前确认，在接到IRIS Dena求救信号后救起32名幸存者，送往加勒国家医院治疗。海军发言人布迪卡·桑帕特（Commander Buddhika Sampath）在新闻发布会上说，救援首要目标是抢救生命；根据舰上文档估计约有180人，但具体失踪人数尚未确定。救援时未见完整舰体，仅发现海面油迹和漂浮救生筏。

台湾周边六天未见解放军战机，为近三年来最长停飞期，分析认为或为即将举行的习近平—特朗普会晤营造和缓氛围。 
据《南华早报》整理的台湾国防部数据，自上周五起至周三，解放军未派战机靠近该岛，系过去三年内最长的连续“空中空档”。同期台方侦测到的解放军舰艇数量仍维持在五至七艘，未见明显减少。 
国防部资料也显示，本月共计11天未出现解放军战机活动，其中包括农历新年期间2月16日至18日的三天，这是自2024年以来首次在除夕至正月初二连续三天无战机飞行。分析人士指出，此种克制不同于近年常态，可能与本月习近平与美国总统特朗普会面前的外交考虑有关。
