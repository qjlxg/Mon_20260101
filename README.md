陈文清：带着对人民群众的深厚感情推进信访工作法治化
中共中央政治局委员、中央政法委书记陈文清说，要带着对人民群众的深厚感情推进信访工作法治化，促进社会大局稳定。
据新华社报道，中国的中央信访工作联席会议全体会议星期四（1月29日）北京召开。陈文清出席会议并讲话。
陈文清在讲话中说，要带着对人民群众的深厚感情推进信访工作法治化，更好服务中共和国家工作大局、维护群众合法权益、促进社会大局稳定。
报道称，陈文清充分肯定了2025年推进信访工作法治化取得的成效。他强调，2026年要严格执行《信访工作条例》，推进预防、受理、办理、监督追责、维护秩序“五个法治化”，大力纠治“应受理而不受理、应办理而不办理、应追责而不追责、应查处而不查处”问题。
他也要求进一步打通信访法治化“路线图”，确保群众的每一项诉求都有人办理、群众的每一项诉求都依法推进，实现权责明、底数清、依法办、秩序好、群众满意；要充分发挥联席会议机制作用，健全工作制度规范，加强信息化建设，形成推进信访工作法治化的整体合力。
中国国务委员兼国务院秘书长吴政隆主持会议，强调要把思想和行动统一到中共中央决策部署上来，以推进信访工作法治化为主线，推动矛盾纠纷源头化解、多元化解、有序化解，更好履行为民解难、为中共分忧的政治责任，为实现“十五五”（2026年至2030年）良好开局作出新的贡献。

马尼拉 — 能源监管委员会（ERC）批准实行全国统一的最低生活电费补贴率。
根据该决议，全国最低生活补贴率定为每千瓦时P0.01。ERC表示，将各配电公司各自的补贴改为全国统一框架，可为所有菲律宾民众提供一致且可预期的折扣，不受地区或配电公司差异影响。最低生活电价面向无力全额支付电费的合格低收入用户。
决议并将全国统一的最低生活用电门槛定为0至50千瓦时，符合条件的弱势用户在此用电区间可享受适用电费的100%减免。对那些已获ERC批准、门槛高于50千瓦时的配电公司，现行更高用电区间的折扣措施将维持执行，直至委员会另行指示。
ERC指出，相关征收款项将汇入国家最低生活补贴基金，由电力部门资产与负债管理公司（PSALM）负责管理，用于支付合格受益人的补贴；这些款项被视为传递性费用，不计入配电公司营收，避免扭曲其运营成本或资费。
