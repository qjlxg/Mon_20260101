2025年10月13日凌晨3时许，成都天府大道南段发生一起道路交通事故，当事人邓姓男子（31岁）驾驶小米SU7 Ultra，行驶过程中与前方同方向、由李姓驾驶员的私家车发生碰撞，随后越过道路绿化带，起火燃烧。事故造成邓姓男子死亡，经检测，他涉嫌酒后驾驶。
