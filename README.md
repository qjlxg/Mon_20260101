当地居民理查兹说，她听到了“多次爆炸声”。

里奇曼说：“显然，我们担心这是对犹太社区的直接袭击。”他补充说，在纵火袭击发生前，并未收到任何威胁。
英国首相斯塔默此前曾因未能充分打击英国的反犹主义而面临批评，他称这次救护车纵火案是“一次令人深感震惊的反犹纵火袭击”。斯塔默说，反犹主义在英国社会“没有立足之地”。
在最近一次人口普查（2021年）中，英格兰和威尔士有近30万人自认为是犹太人。
根据监测此类事件的慈善机构社区安全信托（Community Security Trust, CST）的数据，英国2024年反犹事件的数量达到了创纪录的水平。

国际能源署：中东危机冲击全球经济 堪比过去三场危机总和
（悉尼综合电）国际能源署警告，中东战争引发的能源危机对全球经济构成的威胁超大，相当于1970年代的两次石油危机与俄乌战争造成的天然气危机的总和，任何国家都无法幸免。
国际能源署署长比罗尔星期一（3月23日）在澳大利亚国家新闻俱乐部发表讲话时说：“全球经济现在面临着一个非常、非常巨大的威胁，我非常希望这个问题能尽快解决。如果情况持续，没有国家能免受危机冲击。因此，这需要全球共同发力。”
他指出，这场危机的严重性如今已相当于两次石油危机与一场天然气危机加在一起。1973年和1979年的石油危机期间，全球每日损失约500万桶石油，而目前全球每日损失多达1100万桶石油。
俄罗斯入侵乌克兰后，天然气市场尤其是欧洲天然气市场，损失了750亿立方米的天然气。现在这场危机让我们损失了约1400亿立方米。“不只是油气，全球经济的其他命脉如化肥、石化产品、氦气、天然气等，它们的贸易也全受干扰，势必对全球经济带来严重后果。”
伊朗遭到美以联合袭击后，对能源运输要道霍尔木兹海峡实施形同封锁的严格管控，导致原油、天然气和燃料价格飙涨。
这场战争至今也对九个中东国家的至少40处能源设施，造成严重破坏甚至摧毁。比罗尔说，这意味着油田、炼油厂和输油管道需要好一段时间才能够恢复运作，即使冲突结束，全球供应链中断还可能延长。
比罗尔指出，亚太地区因高度依赖中东原油而处于冲击前沿。
被问及对中国限制燃料出口的看法时，他说，全球必须共同应对这场能源危机。“每个国家都会先考虑自身的国内利益，但在如今这样的情况下，没有正当理由地实施严格出口限制，可能不会得到国际社会的认可。”
国际能源署成员国本月早前同意释放总共4亿桶的战略石油储备，以缓解中东战争对石油市场的冲击。能源署正与亚欧国家的政府讨论，是否在必要时释放更多石油储备。
但比罗尔强调，唯有霍尔木兹海峡复航才能真正解决问题。他说：“我们释放石油储备只能帮助安抚市场，但这不是解决方案。这只能减轻经济痛苦。”
除了释放石油储备，国际能源署也建议各国政府采取措施来降低能源消耗，包括鼓励远程办公和拼车、降低高速公路限速等。

超过91%的多余热量被海洋吸收。海洋热含量在2025年创纪录，2005–2025年的增温速率较1960–2005年翻倍以上。海洋变暖导致海洋生态退化、生物多样性丧失与碳汇减弱，同时助长热带风暴并加剧极地海冰消退。南极和格陵兰冰盖明显减薄，2025年北极海冰年均范围为卫星时代最低或次低；自1993年卫星高度计记录开始，全球平均海平面约上升11厘米。海洋变暖与海平面上升预计将持续数百年。
WMO表示，全球气候仍受拉尼娜影响；2024年为最暖年（约较1850–1900年高1.55°C），并始于强厄尔尼诺。预测到2026年中期可能转为中性，年底或再次出现厄尔尼诺，或致2027年气温回升。WMO副首席称前景“严峻”。古特雷斯警示，战争与对化石燃料的依赖同时正在动摇气候与全球安全，‘延误致命’。

马尼拉 — 菲律宾大气地球物理和天文服务管理局（PAGASA）周四宣布，因“造成破坏的规模与程度异常”，已将7个2025年国内台风名退役，分别为Crising、Emong、Mirasol、Nando、Opong、Tino与Uwan。
PAGASA称，强热带风暴Crising与台风Emong共同增强西南季风，导致7月第三周吕宋及部分维萨亚斯、棉兰老广泛淹水。9月下半月，热带风暴Mirasol、超强台风Nando与台风Opong先后重创菲律宾北部与中部。
11月初，台风Tino带来暴雨，造成维萨亚斯历史性洪灾，逾250人遇难，主要集中在宿雾和内格罗斯；超强台风Uwan则袭击吕宋东部并引发巨浪。PAGASA指出，上述风灾合计造成373人死亡、约360亿披索（P36 billion）损失。
其中，Mirasol与Opong原为替代2021年台风Maring与Odette的名字，均于2025年首次且最后一次使用。自2029年起启用的新替代名单包括：Chico（果树）、Elias（菲律宾人名，《不可触碰》人物）、Magyawan（维萨亚海与死之女神）、Nilad（灌木类）、Omar（菲律宾人名）、Tala（星）、Upang（臭豆）。
PAGASA规定，若一台风直接造成至少300人死亡或对农业、基础设施等造成至少10亿披索损失，其国内名将被退役。2024年，该气象局曾退役8个台风名，为2001年现行命名制度以来最多。
