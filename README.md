中国科研团队取得锂电池技术新突破
中国科研团队取得锂电池技术新突破，打破长久以来锂电池电解液中氧配位的动力学束缚。
据中新社报道，国际顶尖学术期刊《自然》（Nature）星期四（2月26日）在线发表了这项突破性研究成果。
研究由南开大学化学学院研究员赵庆，中国科学院院士、南开大学常务副校长陈军，联合上海空间电源研究所研究员李永共同完成。
科研团队成功设计合成了系列基于氟代烃溶剂的新型电解液，实现了室温下700瓦时/公斤的超高能量密度，且在零下50摄氏度的极端环境下仍能稳定工作，展现出优异的宽温域适应性。
陈军说，基于该电解液的高比能电池在新能源汽车、具身智能机器人、低空经济以及极寒地区和航空航天等领域具有广阔的应用潜力。

日本首相高市早苗去年11月发表“台湾有事论”触怒北京，日中关系跌至谷底。中国从经贸、军事及人文交流等领域反制，要求高市撤回相关言论，但她并未撤回。

朝日新闻记者在星期四商务部新闻发布会上提问：“被禁止或者管制的两用物项指的是包括稀土在内的两用物项出口管制清单的所有物项，可以这样理解吗？”
何咏前回应时说，相关两用物项的内涵和范围详见《中国出口管制法》《中国两用物项出口管制条例》和《中国两用物项出口管制清单》。
