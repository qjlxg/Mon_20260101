中国第二艘国产大邮轮拟3月20日出坞
中国第二艘国产大邮轮拟3月20日出坞。
中国央视新闻客户端引述中船集团报道，中国第二艘国产大型邮轮“爱达·花城号”星期六（3月14日）完成了坞内注水，顺利起浮。
“爱达·花城号”完成坞内注水起浮后，平移靠泊至前坞室开展倾斜试验。倾斜试验是邮轮出坞的核心技术环节，可以测定船舶的重量和重心情况，与邮轮的稳性安全和游客的舒适度息息相关。
据介绍，这次出坞作业为期七天，后续还将相继开展救生艇脱钩试验及坞内巡游，对全船18艘超大救生艇和两艘救助小艇在正常工况和应急状态下的性能进行验证。
“爱达·花城号”并计划于3月20日正式出坞，全面转入系统调试和内装完工阶段。后续将在5月底开启试航，年底前交付。
“爱达·花城号”船名灵感取自“花城”广州，寓意如同一座富有生机的流动花城。
相较于首艘国产大型邮轮“爱达·魔都号”，“爱达·花城号”总吨位更大（14.19万吨）、舱房数增至2130间，满载游客量可达5232人。同时，舱内设计和布局得到优化，新增多种套房房型，中庭面积扩大一倍，内装设计深度加入了海上丝路、岭南风情等多元文化，并以“花卉”为主线。
“爱达·魔都号”2023年11月4日交付使用， 自2024年1月1日首航以来，已运营上海至日韩等航线，截至2025年3月累计完成100个航次，服务旅客超35万人次，单船载客率达95%以上。
“爱达·魔都号”今年2月8日在上海宝山区吴淞口国际邮轮港迎来爱达邮轮第100万名乘客。

驻迪拜总领事馆及DMW迪拜办事处保持开放，并提供热线，为受影响的菲律宾人提供资讯与协助。

菲律宾移民劳工部（DMW）表示，53名遭人口贩运者周五自柬埔寨金边被遣返，安全抵达马尼拉尼诺伊·阿基诺国际机场1号航站楼。
DMW称，这批人通过通讯软件Telegram被诱骗赴柬，承诺薪资600至1000美元并保证发放工作签证。机构表示，一名受害者透露遭到虐待，被迫做300个俯卧撑、挨饿，并因未达到每日26名“恋爱诈骗”客户的配额而被扣薪。
其中两人需医疗处理：一人肾功能衰竭，另一人因车祸锁骨骨折；当局已发药并将安排至公立医院免费治疗。受害者在陈述过程期间获住宿协助，随后将接受心理社会辅导、重返社会支持及返乡交通安排。
柬埔寨已成为网络诈骗“杀猪盘”与加密货币投资骗局的重灾区，诈骗团伙通过长期诱导让受害者投入假投资平台。联合国毒品与犯罪办公室今年2月报告称，全球网络诈骗产业规模估计可达约640亿美元/年。区域内从事诈骗的人中，有人被贩运并被关押，也有人是自愿参与。
柬埔寨政府反诈委员会资深部长Chhay Sinarith称，自去年7月展开整治以来，已关停约250处诈骗据点和91家赌场，逾20万人逃离诈骗现场并离境，约1万名外籍人士被遣返。报道并援引法新社资料。

沙特军方周五称拦截28架无人机。伊朗誓言打击中东油气设施并维持对霍尔木兹海峡的“封锁”，使国际油价飙升。以色列亦于周五凌晨遭伊朗发射的导弹袭击，以军表示防空系统正在拦截。 
国际能源署警告，冲突可能引发“史上最大规模的供应中断”。美国前总统特朗普在社交媒体上称，击败伊朗“邪恶帝国”比油价更重要，但政治压力持续升高。 
伊朗对邻国（包括驻美军设施所在国）发动多轮无人机和导弹打击。伊朗新任最高领袖莫吉塔巴·哈梅内伊通过国营电视称要复仇，并主张“必须使用封堵霍尔木兹的杠杆”。安全官员阿里·拉里贾尼批评美方并警告战争非“几条推特”可赢。 
以色列总理内塔尼亚胡称美以联军正在压制伊朗及其在黎巴嫩的代理武装真主党。伊朗副外长塔赫特-拉万奇称德为“自卫”，并表示愿以“将停火纳入总体方案”结束战争。 
海湾多国遭报复性打击：巴林燃料罐起浓烟，科威特机场和迪拜市区受损，产量下降、油轮滞留，基准油价自2月28日以来上涨约40%至50%。 
冲突亦导致重大人员伤亡和流离失所：伊朗卫生部称逾1200人死（未获独立核实），联合国称伊朗有300万流离失所者；黎巴嫩当局报道687人死；以色列死亡14人；海湾袭击造成24死，其中11名平民、7名美军。另有多架军机损失：美一架KC-135坠毁，三架F-15被友军误击，法国称在伊拉克库尔德区有6名士兵受伤。

密歇根一名不明袭击者周四驾车冲入底特律郊区一座犹太教堂后死亡，事件引发大规模警力与消防救援。
奥克兰县警长迈克尔·布沙德表示，袭击者驾皮卡冲破西布卢姆菲尔德Temple Israel教堂大门并驶入走廊，保安开火与其交火后该人死亡。警方称尚无法确定致其死亡的具体原因，车辆内疑为单人，警犬正检查有无爆炸物。教堂被火焰与浓烟笼罩，疏散时约30名执法人员因吸入浓烟被送医，一名保安被撞伤并在院治疗；教堂工作人员和儿童均未受伤。
联邦调查局底特律办事处负责人詹妮弗·鲁尼表示，FBI将此案作为针对犹太社区的有针对性暴力行为调查。尚未公布确切动机；美国媒体称袭击者疑为黎巴嫩裔。事件发生之际，美国在近两周来因美以对伊朗开战而全国安全警戒升高。
白宫与密歇根州长惠特默、以色列外交部长吉迪恩·萨尔等已就事件发表谴责。奥克兰县表示将继续在当地犹太场所加强安保。与此同时，弗吉尼亚诺福克旧道明大学当天也发生枪击，事后一名死者、两人受伤，联邦官员称该枪击正作为恐怖行为调查。

联邦调查局称将弗吉尼亚州一所大学的致命枪击案作为“恐怖主义”调查。 
周四，位于弗吉尼亚州诺福克、靠近华盛顿的老道明大学（Old Dominion University）校园内发生枪击，一名平民遇害，两人受伤；袭击者在现场也已死亡。学校发表声明称，警方和急救人员“立即回应”，袭击者已确认死亡。 
联邦调查局局长卡什·帕特尔在X上表示，开枪者被一群挺身而出的学生制服并已死亡，这些学生与迅速到场的执法人员的行动无疑挽救了生命。他并称，FBI现将此案作为恐怖主义行为调查，联邦联合作战反恐小组正与地方当局协作并提供资源，并呼吁为受害者、其家属及ODU社区祈祷。 
多家美媒包括NBC将嫌疑人辨认为36岁前国民警卫队成员Mohamed Bailor Jalloh。媒体称其曾在2016年承认试图支持“伊斯兰国”，被判11年监禁，并于2024年获释。 
在枪支数量超过人口、购买军用风格长枪等监管相对宽松的美国，校园枪击事件屡见不鲜。

伊朗誓言令美国为攻击“后悔”，并将继续对霍尔木兹海峡施压，推高国际油价。国际能源署警告，中东战争可能造成史上最大规模的石油供应中断；美国总统特朗普则表示击败伊朗比油价更重要，并对美方战事何时结束发出混合信号。
伊朗安全首长阿里·拉里贾尼表示不会罢休，新任最高领袖穆贾塔巴·哈梅内伊在其首次公开声明中呼吁复仇并主张“必须启用封锁霍尔木兹的手段”。该海峡通常承担全球海运石油约四分之一、液化天然气约五分之一。
以色列总理内塔尼亚胡称美以联手行动正在“击溃”伊朗及黎巴嫩真主党。伊朗副外长塔赫特‑拉万奇表示，伊朗是在自卫，愿将停火纳入结束战争的总体方案。
海湾国家遭多起报复性袭击：巴林燃料罐被击中、科威特机场与迪拜市区遭无人机袭扰，沙特拦截袭向油田和使馆区的无人机。自2月28日美以对伊攻击以来，基准油价上涨约40%至50%。
伊拉克北部库尔德区发生无人机攻击致六名法军受伤，美国一架KC‑135加油机在伊拉克西部坠毁。战事已严重扰乱伊朗民生，商铺大量关闭、银行挤兑、面包配给。伊朗官方称逾1200人死亡，联合国难民署称约300万人成为流离失所者；以色列自冲突开始有14人死亡，海湾地区袭击造成24人死亡（含11名平民、7名美军）。
