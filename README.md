伊拉克近海两艘油轮遭袭致一人死亡，搜救仍在进行
伊拉克官员和船运单位称，3月12日靠近伊拉克海域的两艘油轮遭到“破坏性”袭击，造成至少一名船员死亡、38人获救，另有人员失联仍在搜寻中。死亡者为印度籍，印度驻伊拉克大使馆表示共有15名印度船员已被撤离。
伊拉克国家石油营销组织（SOMO）证实两艘油轮遭袭，但未交代具体作案手法。国营电视台播出一艘起火船只的画面，巴士拉油码头一名员工称尚不清楚是无人机攻击还是装有爆炸物的小艇所为。
其中一艘悬挂马耳他旗的油轮ZEFYROS在准备驶入霍尔·祖贝尔港装载约3万吨液体石脑油时遭袭；另一艘名为SAFESEA VISHNU的船只挂马绍尔群岛旗，由伊拉克公司租用。
伊拉克政府称遭到“蓄意破坏”，石油部对海湾油轮事件表示高度关切，强调国际海上航道和能源供应线路不得卷入地区冲突。霍尔木兹海峡近来几乎对所有油轮关闭，伊朗曾威胁在与美以冲突持续期间阻断海湾石油出口。
美方方面，美总统称美军已打击多艘伊朗布雷船只；美国驻巴格达使馆亦警告，伊朗及其支持的伊拉克武装可能针对美资油设施发动攻击。
（来源：法新社）
