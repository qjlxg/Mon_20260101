港大研究发现骨质疏松药或有助降低患认知障碍症风险
香港大学医学院团队研究发现，广泛应用于治疗骨质疏松症的药物“含氮双膦酸盐”，或有助骨质疏松症或脆性骨折的老年患者降低罹患阿尔茨海默症及相关认知障碍症（简称“ADRD”）的风险。
据新华社报道，香港大学星期二（3月17日）公布最新研究成果，相关结果已发表于期刊《阿尔茨海默病与痴呆症》。
港大医学院研究团队早前已证实，骨质疏松症和骨折会增加日后出现认知障碍症的风险，而认知障碍症患者跌倒和骨折的风险也较高，反映两者之间存在相互影响的关系。研究团队指出，若能改善骨质疏松症的治疗，可能有助减轻认知障碍症的病情。
“含氮双膦酸盐”是目前治疗骨质疏松症的常用药物。港大医学院研究团队利用全港电子医疗记录进行大规模研究，分析2005年至2020年间超过12万名60岁以上、患有骨质疏松症或脆性骨折的长者。
结果显示，使用“含氮双膦酸盐”的患者罹患ADRD的风险，较未接受治疗的患者降低16%；若与使用其他骨质疏松症药物的患者相比，风险降低24%。
港大医学院药理及药剂学系副教授张正龙表示，这项研究揭示了“含氮双膦酸盐”为老年人提供双重益处，除强化骨骼和预防骨折之外，也可降低患ADRD的风险。

商务部要求对59项受管制商品实行价格管制，警告若趁机涨价、囤货或拒绝销售将严惩，最高可判7年监禁，并密切关注受中东局势影响的油价与运输成本。
内贸司长威塔卡恩·玛尼奈特表示，部门持续监测民生必需品价格，尤其是与油价和进口原料相关的品类。根据2025年商品与服务中央委员会公告，现有59项受管制商品，生产与销售者如欲调价须向部门申报，部门将全面审核成本结构后决定。目前尚无企业提出涨价申请，部门亦已呼吁业者维持原价，减轻民众负担。
部门评估出6大可能受冲击的品类：活鲜食品（如鸡蛋、猪肉、鸡肉）、主要农产品（稻米、棕榈油、水果）、日用消耗品（面纸、纸包装）、罐头食品（尤其是鱼罐头）、饮料及塑料包装品（饮用水、瓶装牛奶、食用油）和建材（水泥、钢筋、油漆、PVC管、瓷砖）。
在农资方面，化肥库存足够，进口来源已分散到未受冲突影响的国家（如马来西亚、文莱、阿曼），并准备由生产者直供农户，支援5月至8月播种季，重点保障尿素、磷肥、钾肥等三类主用肥供应，同时推行“绿旗”廉价化肥方案以降低农民成本。
监管上采用三层追踪机制并每月复核：Sensitive List 18项（如化肥、鸡蛋、猪肉、食用油、柴油、塑料粒）每日监控并派员查核；Priority Watch 4项（熟食、奶粉、农药、热轧钢板）每周两次；Watch List 197项（如肥皂、洗衣液）每15日检查一次。
当局称目前国内物资與價格仍可控，呼吁民众理性采购、不囤积，并节能减支。如发现未标价将处以上限1万铢罚款；若被查属人为哄抬、囤货或拒售，可处最高7年徒刑或不超过14万铢罚款，或并科之。民众可拨打内贸司举报专线1569提供线索。

曼谷 2569年3月16日 — 泰国三大制糖厂协会（TSMC）呼吁政府将以农产品为基础的替代能源提升为国家议程，推动把汽油掺乙醇比例由E10全面提升为E20为主燃料，以应对国际能源市场波动并减少对进口能源（占需60–70%）的依赖。
协会指出，若全面采用E20，全国对乙醇需求将由现行约3.5百万公升/日增至6.4百万公升/日，立即增加乙醇需量2.9百万公升/日（约1,058百万公升/年），可大幅减少原油进口支出并将数十亿铢資金留在国内。泰国糖料产业具备规模优势：种植面积逾1100万莱，年产甘蔗约9200万吨、糖约1000万吨，农业GDP约1.62兆铢，其中蔗糖业约1230亿铢。
同时，制糖厂具备将蔗叶、蔗渣等农业剩余物转为生物质发电的能力，合计可达650兆瓦，替代价格波动大的天然气并每年为农户因售叶增收约12亿铢，有助减少露天焚烧与PM2.5排放，改善环境并支援国家净零目标。
协会称，政府已显现支持信号：能源政策委员会自2016年3月14日起将生物柴油（B100）混兑比例由5%调升至7%，并拟将E20/E85燃料的價格差补贴由每公升2铢增至3铢。但协会仍要求长期、具法律效力的政策以促成投资与產能扩张，并愿与政府、農戶紧密合作，推动农业替代能源成为国家能源安全与绿色经济的核心路径。
