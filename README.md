高盛 警告全球石油供应过剩将再现 各国补充战略储备不足以改变局面
高盛集团表示，随着伊朗冲突的影响逐渐消退，霍尔木兹海峡的交通恢复，全球石油市场供应过剩的情况将再次出现。高盛全球大宗商品研究联席主管Samantha Dart在接受采访时表示，预计购买原油以补充战略储备会在一定程度上令全球市场供应收紧，但这只能部分抵消预期中的供应过剩。“一旦海峡流量恢复正常，预计我们将进入供应过剩，”Dart表示，预计明年原油过剩量平均每天将略高于300万桶。她补充说，“我们预计全球战略石油储备重建将为每天略高于100万桶，但即便如此，我们每天仍将有近200万桶的过剩量。”
