综合中时新闻网、《联合报》、《自由时报》等台媒报道，高雄市正义市场内经营40年的春卷摊，上星期六（4月4日）起陆续有消费者食用后出现上吐下泻的食物中毒现象。截至星期天（4月5日）下午，高雄市卫生局共接获九家医疗院所67人通报。
根据高雄市卫生局最新掌握的情况，星期一（4月6日）上午通报人数已经上升至134人。其中96人就医后返家休养、七人留院观察、两人就医治疗中，以及29人持续住院治疗。
卫生局调查，有关摊商在星期天接受稽查时极度不配合，不仅一度拒绝停业，更在交代春卷内容物时“挤牙膏”，即吞吞吐吐、不主动交代；同时，针对春卷所包的食材仅提供部分内容物。

卫生局指出，将依《食品安全卫生管理法》加重裁罚36万元（新台币、下同，1万4460新元），同时因食安风险情节重大而使就医人数持续增加，全案已同步移送地检署侦办。
长庚医院毒物科医师颜宗海说，医学上有个数字叫“危险温度”，介于7至60摄氏度之间；一般环境常见的危险物包括金黄色葡萄球菌、大肠杆菌、沙门氏菌，都容易在危险温度中滋生，造成食物中毒。
颜宗海也说，润饼或春卷的配料包括高丽菜、豆干、豆芽菜、萝卜、花生粉等，建议室温下别放太久，最好不超过一至两小时。
