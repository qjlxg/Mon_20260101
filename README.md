高市向靖国神社献祭品 中国已向日本提出严正交涉
日本首相高市早苗向靖国神社供奉供品后，中国外交部星期二（4月21）强硬表示，中国坚决反对、严厉谴责日本有关靖国神社的消极动向，并称已向日本提出严正交涉，强烈抗议。
中国外交部发言人郭嘉昆星期二主持例行记者会。他在回应有关高市早苗向靖国神社供奉供品的提问时称，中国坚决反对、严厉谴责日本有关靖国神社的消极动向，已向日本提出严正交涉，强烈抗议。
郭嘉昆指出，靖国神社是日本军国主义对外发动侵略战争的精神工具和象征，是事实上的“战犯神社”。
郭嘉昆进一步说，今年是东京审判开庭80周年，令人愤慨的是，80年后的今天，臭名昭著的靖国神社仍然供奉着对侵略战争负有直接罪责的日本二战甲级战犯，日本官员、政客多次前往参拜或供奉祭品、祭祀费。
郭嘉昆批评，日本在靖国神社问题上的消极动向，实质是对自身罪责的逃避、对历史公正的亵渎、对被侵略国家的挑衅，更是对二战胜利成果的挑战。
据日本共同社报道，高市早苗星期二配合东京九段北靖国神社开始的春季例行大祭，以“内阁总理大臣高市早苗”的名义供奉了被称为“真榊”的供品。
中国军方媒体《解放军报》旗下微博账号“钧正平工作室”也在星期二发文说，靖国神社供奉着沾满亚洲人民鲜血的二战甲级战犯，是军国主义招魂所，“日本政客屡行此事，本质就是为侵略历史翻案，为战争罪责洗白”。
文章称，在东京审判开庭80周年之际，“拜鬼常客”再次上演政治挑衅戏码，性质极为恶劣，并指此举公然破坏战后国际秩序，加剧地区紧张对立。
文章指出，历史不容亵渎，挑衅必遭反噬，“日本若执迷不悟，只会自陷道义孤立，给地区和平稳定埋下更大隐患”。
靖国神社位于东京千代田区，内供奉有包括东条英机等14名二战甲级战犯。长期以来，日本部分政客、国会议员坚持参拜靖国神社并供奉祭品，导致日本与中韩等亚洲国家关系紧张。
已故日本前首相安倍晋三2013年12月26日曾参拜靖国神社，导致中日关系一度跌入谷底。

香港高院7月8日处理港府申请充公黎智英罪行财产

据香港政府公报，港府4月2日入禀高院原讼法庭，申请充公黎智英的罪行相关财产。
港府发言人指出，港府向法庭提出充公令申请是有效打击严重罪行、保护公众利益的做法之一。就维护国家安全所需而作出的充公令，可以防止干犯危害国家安全罪行的人、其同谋或其代理人运用其罪行相关财产，继续从事危害国家安全的行为和活动，以切断危害国家安全罪行的资金链，并削弱其干犯危害国家安全罪行的能力。
综合网媒“香港01”和香港《星岛日报》等报道，根据香港律政司的入禀文件，港府要求充公黎智英的财产金额至少1.27亿港元（2100万新元）。
除了黎智英名下17家公司股份、银行账户等资产外，也包括黎智英助手马克·西蒙（Mark Simon）持有的公司。黎智英早前就科技园欺诈案上诉得直，获退还的200万港元罚款，以及他曾支付的1000万港元保释金，也同样被要求充公。
根据司法机构网页显示，香港高院将于7月8日处理上述充公申请，预计需时一小时。
黎智英的香港律师团队3月初确认，黎智英将不会为早前的定罪和判刑提出上诉。黎智英去年裁定两项“串谋勾结外国势力罪”和一项“发布煽动刊物罪”三项罪名成立，今年2月9日被判囚20年。
香港高院上诉庭2月裁决，黎智英涉嫌欺诈科技园案上诉得直，定罪和刑期获撤销，律政司3月表明，将不会就此案再采取进一步的法律行动。

受战事影响 中国3月自海湾六国进口原油同比降25%
受中东战事升级以及霍尔木兹海峡运输受阻影响，中国3月自沙特、伊拉克、阿联酋、阿曼、科威特和卡塔尔等海湾六国进口原油1.226亿桶，同比下降25%。
综合路透社和财新网报道，中国海关总署星期二（4月21日）公布的数据显示，3月原油进口量为4998万吨，约合每日1177万桶，同比下降2.8%。
数据显示，当月来自海湾六国的原油进口大幅下滑，其中自沙特、伊拉克、科威特和卡塔尔的进口降幅尤为显著，分别下降30.7%、46%、52.3%和64.5%。来自阿联酋的进口量变化不大，而自阿曼的进口量则大幅增长44%，在一定程度上弥补了前述四国进口萎缩带来的缺口，这主要是因为阿曼的出口终端位于霍尔木兹海峡之外。
船舶追踪公司Vortexa分析师指出，中国3月海运原油进口量同比持平，维持在每日1050万桶，库存则增加了3400万桶。由于3月进口的中东原油主要在1月和2月装船，因此当月进口尚未受到霍尔木兹海峡运输受阻的影响。
睿咨得能源副总裁林叶（Ye Lin）认为，中国4月可能面临更紧张的原油供应，进口量预计将比平均进口需求低约每日200万桶。
她指出，即使受利润疲弱影响，4月炼厂加工量预计将减少约每日100万桶；为维持成品油供应充足，中国仍可能需要动用库存。
中国上月下令禁止成品燃料出口，导致截至3月11日尚未完成清关的货物暂停出运。这一禁令不包括航空加油用途的航煤，预计将延续至4月，不过对于部分应地区国家请求援助的小规模出口，可能会给予豁免。
隆众资讯指出，在汽柴油出口计划被削减后，大型炼厂提高了汽油和柴油产量，而中国国内汽柴油消费仍相对疲弱，导致供应充足、国内库存上升。

国际特赦组织批美以俄领袖为“掠食者”
（伦敦法新电）国际特赦组织批评美国总统特朗普、俄罗斯总统普京和以色列总理内坦亚胡为“掠食者”，指他们试图构建一个“掠夺性的世界秩序”，而其他国家大多过于软弱，不敢阻止他们。
国际特赦组织秘书长卡拉马德周二（4月21日）在伦敦发布年度报告。报告称，美俄以领袖否定了二战以来的多边体系，推崇一个“愿景不受道德约束”、“由战争而非外交与规则治理”的世界秩序。
她警告，这份报告发布之际正值“充满挑战的时刻”，过去80年的建设成果可能被摧毁。她说：“2025年全年，特朗普、普京及内坦亚胡等人通过大规模国际破坏、压制和暴力，追求经济和政治霸权。”
她指出，这些领导人攻击普世人权的根基，而大多数政府，尤其是欧洲政府，选择绥靖而非对抗这些“掠夺者”。中东不断升级的冲突只是这种新的掠夺性世界秩序的最新例证，而这种秩序建立在通过种族主义意识形态而去人性化的世界观之上。
