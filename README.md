面对此轮国内销量下滑，比亚迪计划通过扩大出口（今年拟增24%至130万辆）来对冲国内市场疲软。机构预计今年内地车市将有所回落，德银与瑞银分别预测降幅约5%和2%。

美国财政部批准为期30天的许可，允许各国购买目前滞留海上的俄国原油和石油制品。财政部长斯科特·贝森特称此举是为稳定因伊朗战争而动荡的全球能源市场。
