耶路撒冷消息——以色列参谋长埃亚尔·扎米尔中将4月27日对部队发出严厉警告，抨击掠夺行为并警告在社交媒体上传播“有问题”信息已越过“红线”。
扎米尔在一份声明中称：“若存在掠夺现象，这是可耻的，可能玷污整个以军。若发生此类事件，我们将进行调查。”
此番表态是在以色列偏左的《哈雷茨》报援引前线官兵和指挥官证词报道，称在黎巴嫩南部有部队大量掠夺民用财物之后发出的。社交媒体上也出现士兵在疑似黎巴嫩南部民宅内破坏财物、相互拍摄打闹的画面；另有一张照片显示一名士兵用大锤砸碎掉落在十字架上的耶稣像。
以军上周表示，因相关事件已有两名士兵被处以30天军事拘留并取消战斗任务。扎米尔强调，征兵及预备役人员不得将社交网络作为传播争议信息或自我炒作的工具，越界者将面临纪律处分，并警告称类似行为的常态化与作战威胁同样危险。
军方在给法新社的声明中称，对任何损害平民财产和掠夺行为“予以最严肃对待并严格禁止”，对指控将“彻底审查”，一经查实将采取包括起诉在内的纪律和刑事措施；军警也对部队退出现役线经由的北部边境通道进行了检查，但未说明是否发现掠夺证据。
以色列维权组织“打破沉默”表示，虽未在黎巴嫩收集证词，但自2023年对加沙发动地面入侵以来，掠夺等行为已“变得十分普遍”。
（法新社）
