气象组织：厄尔尼诺现象或将于2026年中来袭
联合国指出，上次导致全球气温创下历史新高的厄尔尼诺现象（El Nino）预计将于今年年中再次出现，最早可能在5月至7月这三个月内出现。
总部位于日内瓦的世界气象组织（WMO）星期五（4月24日）发布的报告说，赤道太平洋地区的气候状况发生了明显变化。报告指出，海面温度正在迅速上升，“预示着厄尔尼诺现象最早可能在5月至7月再次出现”。
报告指出，早期迹象表明，此次厄尔尼诺现象强度较大。
厄尔尼诺是指赤道中东太平洋海水温度异常偏高从而影响大气环流的一种自然现象。厄尔尼诺往往引起太平洋周边多个地区气候异常，引起风、气压和降雨模式的变化。
厄尔尼诺现象和拉尼娜现象（La Nina）交替出现，两种现象之间的气候则处于中性状态。
厄尔尼诺现象通常每两年到七年发生一次，持续约九个月到12个月。
气象组织此前发布的新闻公报显示，2025年是有记录以来最热的三个年份之一。上次厄尔尼诺现象导致2023年成为有记录以来第二热的年份，而2024年则是有记录以来最热的年份。 
法新社引述气象组织气候预测主任奥基亚（Wilfran Moufouma Okia）说：“年初气候处于中性状态之后……我们高度确信厄尔尼诺现象即将到来，并会增强。
“模型显示，这可能是一次强烈的厄尔尼诺事件……4月之后的预测将会更为可靠。”
厄尔尼诺现象通常每两年到七年发生一次，持续约九个月到12个月。

金融机构加大核武投资 反核组织：或推高核军备竞赛
（日内瓦综合电）反核团体指出，金融机构正加大对核武器生产的投资，并警告在全球冲突日益增多之际，这一趋势可能进一步推高本已飙升的军费开支，助推新核军备竞赛。
美国和俄罗斯的《新削减战略武器条约》（New START）已在今年2月5日到期失效。专家一直警告，随着核武国家卷入欧洲、亚洲和中东冲突之际，而长期以来为裁军和防止核武器扩散所做的努力似乎正在瓦解，一场新的核军备竞赛正在酝酿之中。 
在星期五（4月24日）发布的一份新报告中，诺贝尔和平奖得主“国际废除核武器运动”（ICAN）和另一个反核武组织PAX指出，越来越多金融机构正在对参与核武器生产的公司进行投资，帮助全球九个拥核国扩充和现代化它们的核武库。
根据这份年度报告，截至2025年9月，共有301家银行、养老基金、保险公司和其他金融机构对参与核武器生产的公司进行融资或投资。与去年同期相比，这一数字增长15%，扭转多年来的下滑趋势。
ICAN项目主管斯奈德在声明中指出：“多年来，试图从军备竞赛中获利的投资者数量首次出现上升趋势。”
也是报告共同作者的斯奈德警告，这是一种短期且高风险的策略，只会加剧危险的局势升级，而“要从中获利，就不可能不助长军备竞赛”。
根据报告，俄罗斯、美国、中国、法国、英国、巴基斯坦、印度、以色列和朝鲜这九个拥核国目前正在对它们的核武库进行现代化改造，当中还有许多在扩充规模，“从而推动了对这些武器的需求”。
报告列出25家参与核武器生产的公司，其中霍尼韦尔国际（Honeywell International）、通用动力（General Dynamics）和诺斯洛普·格鲁曼（Northrop Grumman）获利最多。其他主要生产商包括英国航空航天系统公司（BAE）、贝克特尔（Bechtel）和洛克希德·马丁（Lockheed Martin）。
不过报告也强调，一些金融机构已证明，放弃与核武器相关的投资仍可获得可观利润。
反核组织说，截至2025年底，明确避免投资核武公司的机构所管理的资产总规模已超过4万亿美元（约5.1万亿新元）。

伊朗战事冲击亚洲聚酯供应链 或推高全球快时尚成本
（苏拉特路透电）自伊朗战争爆发以来，飙升的化石燃料价格不断挤压印度和孟加拉国的聚酯纤维（polyester）供应和服装制造业，可能推高Zara和H&M等快时尚零售商的成本。 
亚洲是服装供应链领的主导力量，但近来受中东战事冲击，印度、孟加拉和中国的聚酯纤维价格都在上涨，给生产商带来压力。
印度聚酯纱线生产巨头菲拉泰克斯（Filatex）的董事总经理巴格里亚说，由于中国供应商提价且中东供应中断，公司用于生产纱线的石油衍生原料——精对苯二甲酸（PTA）和单乙二醇（MEG）的采购成本已上涨近30%。
位于印度西部纺织业中心苏拉特的宾达尔丝绸厂（Bindal Silk Mills），向H&M、Zara母公司Inditex、塔吉特（Target）、沃尔玛和宜家等零售商提供染色和印花聚酯纤维面料，现在却难以满足全球订单需求。
苏拉特纺织商协会联合会主席哈基姆说，由于成本上涨，苏拉特的纺织印染厂现在被迫每周停工两天，此前仅为一天。“如果这种情况持续下去，原材料会出现短缺，工厂就不得不停产。”
由石油衍生品制成的聚酯纤维是纺织业的关键材料。霍尔木兹海峡封锁后，精炼石油产品供应紧缩，直接冲击供应链依赖聚酯纤维的零售商。
Zara和H&M等零售商已转而主要使用由废弃塑料瓶制成的再生聚酯纤维。这或能缓解部分由油价上涨带来的成本压力，但全球范围内，再生聚酯纤维仅占聚酯总产量的12%。
行业消息人士称，H&M预计，未来几周孟加拉供应商就会提价，不过目前还未发现生产出现重大中断或供应商因能源成本而要求调整订单的明显迹象。

法国建议设“信任俱乐部” 共享关键矿产信息
（悉尼讯）法国建议设立“信任俱乐部”，让志同道合的国家共享有关稀土和其他关键矿产的信息，以提高供应链的可预测性。
《日经亚洲》报道，在澳大利亚访问的法国对外贸易部长级代表弗里西耶星期四（4月23日）对媒体说，稀土不应被当成武器使用，因为这会引发剧烈冲击。“我们提议成立一个类似‘信任俱乐部’的组织，按矿物种类或类别划分，以便共享信息，甚至可能建立一些储备。”
法国今年是七国集团（G7）轮值主席，关键矿产是G7议程上的主要议题之一。
控制着全球70%稀土产量的中国正加强对稀土供应的控制。弗里西耶说：“所有志同道合的国家，未来都不应再像过去两年那样受制于人。”
法国拥有众多依赖关键矿产资源的重要工业如汽车、可再生能源和国防。为了确保关键矿产资源供应稳定，法国正积极扩大与其他国家在这方面的合作。
法国西南部目前在建造一座重稀土回收厂，这将是欧洲首个大型稀土回收设施。回收厂属于法日一个合作项目的一部分，每年将生产重稀土镝（dysprosium）和铽（terbium）共600公吨，占全球年产量的15%。
法国私募基金InfraVia上月宣布，向澳洲北领地一个锂矿项目注资5000万美元（约6388万新元），这个项目由澳洲矿业公司Core Lithium牵头。
法国已将“纠正失衡”列为G7议程的关键优先事项，其中包括对中国贸易顺差和工业产能过剩的担忧。
弗里西耶称中国电动车获得中国政府补贴，破坏市场公平竞争。他说：“我们不应再对中国经济产能过剩抱有天真的想法。”
他也呼吁对国际电商平台实施更严格管控。
法国和澳洲上月签署自由贸易协定，弗里西耶形容这是个双赢的协议。他说：“澳洲志在实现多元化，与法国等志同道合的国家建立更紧密的伙伴关系。”

4月24日，中国金融期货交易所发布通知，根据《合格境外机构投资者和人民币合格境外机构投资者境内证券期货投资管理办法》（证监会令第176号）《关于合格境外机构投资者和人民币合格境外机构投资者参与国债期货交易的公告》（证监会公告〔2026〕7号）等有关规定，现就合格境外机构投资者和人民币合格境外机构投资者（以下合称合格境外投资者）参与中国金融期货交易所（以下简称中金所）国债期货交易有关事项通知如下：一、合格境外投资者参与国债期货交易应当遵守法律、行政法规、规章、规范性文件、交易所业务规则等有关规定，参与国债期货交易的目的限于套期保值交易。二、接受合格境外投资者委托的会员，应当勤勉尽责，加强对合格境外投资者交易行为的合规性管理。三、自本通知发布之日起，中金所开始受理合格境外投资者参与国债期货交易的业务办理申请。

去年10月的表决中，共有57国（包括中国及利比里亚等主要船籍国）投票支持将碳价方案推迟，另有49国寻求达成协议。支持恢复谈判的国家包括欧盟成员国、巴西及一些受气候影响严重的小岛国家。 
由利比里亚、巴拿马、马绍尔群岛三大船籍国及包括沙特Bahri在内的油轮公司组成的联盟，呼吁IMO成员在下周会议上考虑原方案的替代方案，并称自去年会议以来对该框架的支持“持续被侵蚀”。 
欧盟内部此前曾因此出现分歧，希腊和塞浦路斯在表决中脱离欧方立场弃权。官员称，希腊、马耳他和意大利拒绝背书新的欧盟谈判立场，但该立场仍以强化多数通过。

俄罗斯总统普京的支持率连续第七周下滑，降至65.6%，为自2022年2月入侵乌克兰以来的最低水平，国家民调机构VTsIOM公布。 
普京自1999年起先后担任总理和总统掌权；若完成本届六年任期，他将超过斯大林，成为自叶卡捷琳娜大帝以来在位最久的俄国统治者。 
VTsIOM称，普京支持率已从3月的73.3%降至65.6%，对他的信任度也从77%以上回落至71%。目前并不清楚导致支持率下滑的确切原因。 
上周普京曾要求高层提出刺激经济的方案，此前2026年前两月经济出现收缩。近几个月来，对互联网、通讯软件和VPN的严管引发民众不满。普京4月23日表示，网络中断是出于安全需要，但执法部门要“想办法”保障重要服务运行。 
在战争与国家审查环境下，民意真实度存在争议：支持者引用多项民调显示普遍支持，反对者则质疑受访者在受控环境下的坦率程度。VTsIOM记载，入侵后普京支持率曾从64.3%飙升至接近80%，并在战争大部分时期维持在75%以上，2022年动员令后短暂下跌过。 
其他官员民调：总理米舒斯京信任度53.8%，前总统梅德韦杰夫36.8%，共产党领袖久加诺夫32.7%，“公正俄罗斯”领袖米罗诺夫29.8%。 
路透社报道称，距最迟9月底的议会选举还有数月，一些博主和政界人士公开警告称，若不进行改变，可能存在引发不安的风险。
