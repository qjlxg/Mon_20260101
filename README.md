【法国6月PMI高于初值，重返扩张区间】金十数据7月1日讯，周三的一项商业调查显示，尽管伊朗战争造成的运输中断给供应链带来持续压力，但法国6月份的制造业活动增速仍略高于最初预期。标普全球法国制造业采购经理人指数（PMI）6月升至51.2，高于5月的49.7，也高于初值50.7，显示该指数在4月至5月跌破50点水平后恢复增长。标普全球表示，法国制造商6月未完成订单量较5月上升，同时积压工作量也有所增加。标普全球市场情报经济学家乔·海耶斯表示：“法国制造商仍然承压，但积极信号是6月的下行趋势并未加剧。PMI价格指数较5月份下降也是个好消息，可能预示着制造业和更广泛的经济领域通胀率下降。”不过他也指出，供应链压力仍在持续拖累制造业，“交通运输不畅经常被提及，调查显示企业倾向于减少采购并消耗库存。供应链中断持续时间的长短，也可能决定法国制造业复苏的速度。”
