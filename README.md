英国高等教育统计署（HESA）最新数据显示，国际生流向出现显著变化：在英留学生人数下降的同时，英国在海外开办的教育项目报名强劲增长。 
HESA称，2024/25学年英国接待国际学生685,565人，比2023/24下降6%。英国内政部统计显示，截至2025年9月的一年内，共发放439,924份受资助学习签证，较前一年持平，但比2023年9月年终峰值低31%；其中主申办人419,558人，比截至2024年9月增长7%，家属签证仅20,366份，较上一年骤降57%。 
此轮学生签证减少与英国移民及留学政策收紧有关，尤其是2023年实施的留学生家属签证限制。英国拟自2028年8月对英格兰高校征收国际学生税，每名国际生每年固定收取925英镑，且每校仅对前220名学生给予免税额度。 
在各主要生源国中，国际研究生整体招生在2024/25学年下降10%；斯里兰卡研究生入学同比下降36%，为第二大跌幅，尼日利亚降幅最大，为39%。教育顾问认为，因家属难以随行，许多斯里兰卡学生放弃传统授课型研究生课程，改读研究型课程、推迟入学或转向加拿大、澳大利亚及斯堪的纳维亚等替代目的地。

华盛顿 — 2月11日（美联社）一队佛教僧侣周二走入华盛顿市中心，完成自德州出发、历时15周、约2300英里（3700公里）的和平行走。僧侣们身着赭色与藏红色僧袍，随行还有救援犬阿洛卡，行程途经美国东南部，在社交媒体和沿途城市引发关注与欢迎。
他们在华盛顿国家大教堂停留时，逾百位僧尼到场助阵。领队、性情温和的潘那卡拉比丘在教堂台阶上表示，这一刻将终生难忘，并与包括华盛顿圣公会主教玛丽安·巴德在内的多位跨宗教领袖同台倡议和平。
队伍单列行进时，围观民众多以肃静相待，不少人高呼“Sadhu”（善哉）以示赞许。近3500人在美国大学Bender体育馆迎接他们，场内观众为示尊重保持寂静。
行程也并非全无危险：11月在休斯顿外护送车辆遭货车撞击，造成两位僧侣受伤，乔治亚斯内尔维尔寺院住持玛哈·当·福玛桑尊者截肢。福玛桑尊者后来以轮椅回归并进场，令同行与观众动容。来自弗雷德里克斯堡的33岁轮椅使用者杰克逊·沃恩随队数日，称此次经历显示出人与人之间的连结与互助。
此次行走吸引了跨意识形态的支持，从阿拉巴马教堂到里士满市政厅均受欢迎。数百万网友在线关注，周二直播观众超过2万人，观看者遍及牙买加、德国、斯里兰卡与泰国等地。
19名僧侣于2025年10月26日从沃斯堡香道内观中心出发，来自多所上座部寺院，由潘那卡拉领导。寺方发言人龙思东表示，此次并非政治运动，而是“精神的奉献”，邀请人们以觉知的行动在日常中实践和平。部分路段僧侣赤足行走以体察当下；周二他们已连续徒步108天——佛教中象征圆满的数字。周三队伍计划穿越国会山并在林肯纪念堂举行闭幕仪式，随后在马里兰州议会露面并乘车返德州，回到沃斯堡后再步行约6英里回到出发寺院。

中国官方调查显示，南京博物院一起高调文物风波暴露出长期管理混乱与涉嫌腐败问题，多件国宝流入民间市场。 
事件始于去年12月，有报道称江苏南京博物院秘密出售捐赠画作。涉事为收藏家Pang Laichen于1959年向馆方捐赠的137件藏品中的5幅，后在其后代申请下的法院清查中被发现失踪。 
2025年初，仇英名画《江南春》在拍卖中现身，估价8800万元，引发其曾孙女Pang Shuling向主管部门举报并索要处理文书，该作随后撤拍。 
江苏省在文物局指导下展开数月调查，调取6.5万份档案并访谈1100余人，报告称部分涉案画作在数十年间被非法调出、出售或流失；90年代时任副院长Xu Huping被指违规批准将该批捐赠作品移交省级文物库出售。 
博物院承认存在“系统性问题”，并承诺改革；当局称此事已对文化和博物馆行业形象造成“严重”损害，国家与省级部门已介入调查。
