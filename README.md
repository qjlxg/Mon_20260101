中国据报已要求大型国有银行减少在银行间的拆借，以缓解市场融资成本过低，流动性过剩的问题。

知情人士说，此举旨在纠正市场对流动性获取的预期，维护政策利率的权威。中国人民银行未立即回应彭博社通过传真提出的置评请求。
央行的行动显示，监管层已认为当前货币市场利率已经过低，超过可以容忍的限度。
报道指出，在持续数月的流动性过剩推动融资成本降至多年低点、并助推债券上涨后，央行近几周已试图从银行体系中回收部分过剩资金。

报道指出，在全球能源危机波及中国之际，中国央行既要确保金融体系有足够流动性，以刺激经济增长，但另一方面，央行也一直警惕资金只是在金融体系内部循环，从而推高政府债券等资产泡沫风险。
而当前企业和家庭融资需求疲弱，银行营收承压，促使银行转而相互拆借，赚取短期利息，或增加债券购买，尤其是国债、政策性金融债等，获取相对稳定的收益。但是，大量资金没有进入实体经济，而是在金融体系里“空转”。
彭博社称，信贷增速已放缓至有记录以来最差水平，进一步加剧了体系内资金过剩。
