北京 — 中国国家主席习近平周二在会见来访的芬兰总理佩特里·奥尔波时表示，世界面临“多重风险和挑战”，大国应率先维护平等、法治、合作与廉洁。习近平强调“中欧是伙伴不是对手，合作多于竞争，共识大于分歧”，并希望芬兰为中欧关系健康稳定发展发挥建设性作用。
奥尔波此行为期四天，旨在在美方政策压力下深化双边关系，期间与中共中央政治局常委、全国人大常委会委员长赵乐际及国务院总理李强会谈。李强表示愿与芬兰对接发展战略，拓展投资、通信、清洁技术、专业造船、新型城镇化和健康等领域合作，并扩大旅游、语言与职业教育及青年交流。
两国见证了科技、建设、海关、文化旅游、贸易与能源等领域合作文件签署。奥尔波率领逾20家芬兰企业的商贸代表团来访。芬兰近年将中国视为重要贸易伙伴，去年1至10月对华进口占比9.4%，对华贸易逆差约30.7亿欧元。双方均表示将坚持多边主义、自由贸易并推动互利合作。

法院采纳的措施包括：每日听证不超过三小时、每小时休息、每周最多开庭四天并在周中安排休息。法院书记处被指示在2月23、24、26、27日的听证期间保障这些安排。
案件始于2025年3月7日的逮捕令（最初列为秘密，数日后公开），杜特尔特于3月12日被移送ICC并于3月14日首次出庭。预审合议庭由首席法官Iulia Antoanella Motoc主持，陪审法官为Reine Adélaïde Sophie Alapini-Gansou与María del Socorro Flores Liera。检方、辩方及受害人法律代表将在听证中就案情发表口头陈述。ICC将通过官网、Facebook和YouTube以30分钟延迟方式提供网络直播。
