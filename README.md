4月15日晚，美国伊利诺伊州New Lenox警方接报，对教皇列奥十四兄弟约翰·普雷沃斯特（John Prevost）位于芝加哥近郊的住所发出炸弹威胁警情。警方疏散附近住户并展开搜查，次日（4月16日）确认未发现爆炸物或危险物质，认定该威胁无根据。调查仍在进行，追查虚假报告来源。
此事发生在教皇列奥就伊朗战争发表批评之后。列奥为芝加哥人、领导约14亿信徒的教会，并于4月16日在喀麦隆发表强硬言论，抨击花巨资发动战争的统治者将世界蹂躏。此前，特朗普于4月12日抨击列奥“过于自由、对犯罪软弱”，但对教皇在佛罗里达的兄弟路易斯表示赞赏。约翰·普雷沃斯特为教皇的另一位年长兄弟。来源：路透社。

华盛顿 — 在一场党派对立的表决中，美国众议院于4月16日以214票对213票否决了一项由民主党推动、旨在在国会授权前停止对伊朗军事行动的决议。此前一天，参议院也以类似方式阻止了同类提案。
投票几乎完全按党派划分：除一名共和党议员外，其余共和党人都反对该决议，另有一人投了“出席”票；一名民主党人投了反对票。结果凸显，自2月28日美以对伊朗发动打击以来逾六周，共和党内部对总统特朗普的军事方针保持支持。
民主党人要求总统在使用武力前向国会寻求授权，援引宪法规定宣战权属国会。但历届总统常主张，此类限制不适用于短期行动或国家面临紧迫威胁的情形。白宫及几乎所有共和党国会议员则表示，特朗普以三军统帅身份下令有限军事行动是在其合法职权范围内，旨在保护美国安全。

在新加坡附近海域，远离霍尔木兹海峡的数千海里外，伊朗原油通过“船对船”（STS）转运持续进行，成为德黑兰规避制裁并继续对华贸易的重要渠道。
美国、欧盟和英国已针对近400艘与伊朗有关的油轮实施制裁。这些“幽灵舰队”以陈旧船舶、隐蔽所有权、挂旗伪装、无保险及篡改航位信息等手段低调运作，借助公海STS转运“洗白”货源。
位于马来西亚与新加坡东南约100公里的海域已成关键集散地。卫星影像分析显示，每周都有数十次转运发生。专家称该区域为“主要枢纽”，且形同“无序状态”。
航迹显示，伊朗油轮多在霍尔木兹经卡赫格岛出货，绕行印度次大陆经马六甲至新加坡海域抛锚，再与其他油轮对接转运。自3月1日起，至少37艘涉伊油轮在该区完成转运，量约6230万桶，终点多为中国山东、辽宁和江苏港口。
尽管美方曾在3月20日短暂允许对已装船原油出售，且授权将于4月19日结束，业内人士仍认为，在美方一面放松一面封锁伊朗港口的矛盾举措下，用影子船和STS继续出货比公开运输更安全。
追踪公司指出，STS转运常多次中转，且船舶常关闭AIS信号，需用航迹与吃水变化等算法辨别转运痕迹。分析人士认为，随着对伊朗港口的封锁以及海上“浮动储油”仍多，STS转运有望继续繁荣。

华盛顿，4月16日——路透社获悉的一份电报显示，特朗普政府已指示驻外外交官在联合国推动一项“以贸易取代援助”的宣言，敦促盟友支持这一倡议，作为重塑美国数十亿美元对外援助支出的更大努力的一部分。
电报署名日期为4月15日，指称美国国务卿马可·卢比奥要求驻各国外交人员在该国“适当的最高层面”传达此项信息，并在4月20日前取得签署。电报写道：“这是利用联合国体系推动‘美国优先’价值观并为美企创造商机的机会”，并将该宣言全文附后。华盛顿邮报最先报道了该文件。
这份指示符合特朗普政府外交政策的更广泛转向：减少将援助作为扩大美国影响力工具的做法，转而更强调符合所谓核心美国利益的强硬议程。上任初期，特朗普已开始对数十年形成的救援与发展援助分配体系进行冲击，称其存在“浪费和滥用”，并冻结或削减不符合“美国优先”价值的项目，影响粮食和医疗救援的交付并扰乱全球人道主义行动。
国务院副发言人汤米·皮戈特表示：“历史和事实证明，贸易与自由市场资本主义是致富的最可靠路径。”他称，美国仍是历史上最慷慨的国家，而主张“援助而非贸易”的声音实际上是在“填补腐败的非政府组织工业体系的口袋”。
特朗普政府主张以“互利商业关系”为核心、鼓励亲商改革的替代模式，此举招致部分人道主义界批评。洛克菲勒基金会副总裁、前白宫官员埃里克·佩洛夫斯基说，这种做法背离美国传统与价值，并损害国家安全。
电报为外交官提供了游说话术，并称该宣言将在四月底举行发布活动。美国驻联合国大使迈克·沃尔茨在谈及联合国改革时亦提到该倡议，称：“在发展方面，我们正在大力与私营部门接触。降低资本壁垒、推动外资、创造就业而非依赖，我们称之为‘贸易优于援助’。”
（路透社）

贝鲁特 — 美总统唐纳德·特朗普4月16日宣布，以色列与黎巴嫩已就为期10天的停火达成一致，标志着与伊朗战争并行的冲突出现暂时缓和。
特朗普在其Truth Social账号称，他与以黎两国领导人进行了“极佳的对话”，并指示副总统J.D. Vance、国务卿马可·鲁比奥和参谋长联席会议将军丹·凯恩与双方接触，争取实现“持久和平”。停火将于美国东部时间4月16日17时生效（新加坡时间4月17日05时）。他并表示将邀请黎巴嫩总统米歇尔·奥恩与以色列总理内塔尼亚胡到白宫会谈，称两国领导人约34年未直接通话。
冲突缘起于2月28日美以对伊朗的攻击，随后伊朗对海湾邻国还击，并在3月2日导致伊朗支持的真主党向以色列发射导弹，触发以色列对黎巴嫩的军事进攻。3月2日后黎以冲突导致黎巴嫩境内大规模破坏和人员伤亡。
此前美伊于4月8日达成为期两周的停火，但美以表示针对真主党的行动不在该停火范围内。巴基斯坦总理谢赫巴兹·谢里夫曾斡旋并称停火将涵盖黎巴嫩。黎巴嫩总统奥恩主张停火为直接谈判的前提，并强调以军撤出南黎巴嫩是关键步骤；政府已于3月2日禁止真主党军事行动。
战事仍在黎南部激烈进行，宾特杰贝勒等地为真主党据点。黎方统计称自3月2日以来已有逾2100人遇难，逾120万人流离失所；以方称真主党袭击已致2名以色列平民丧生、13名以军阵亡。
真主党资深议员哈桑·法德拉拉表示，停火取决于以色列停止一切敌对行动，并将停火前景归功于伊朗的外交努力。
与此同时，美伊谈判已从寻求全面和平转为争取一项临时谅解以防止冲突重燃，双方在霍尔木兹海峡等问题上有所缩小分歧。美国国防部长皮特·赫格塞斯在五角大楼警告，如伊朗不接受和平协议，美方可能打击其能源基础设施并维持海上封锁。路透社报道。

利马，4月16日 — 在秘鲁总统选举中，左翼国会议员罗伯托·桑切斯与极右翼前利马市长拉斐尔·洛佩斯·阿里亚加争夺与保守派候选人凯科·藤森进入6月决选的第二个名额，形势接近。
据秘鲁选举办公室（ONPE），在计票率达93%时，桑切斯以11.98%微弱领先于洛佩斯·阿里亚加的11.92%，两人差距不足1万票；藤森以17.07%暂居首位。第二轮投票定于6月7日举行。
本次投票计票延长至周一，部分选民周日无法投票后才补投，延迟引发洛佩斯·阿里亚加指控舞弊并要求暂停计票。他所在政党向选务人员提供2万索尔（约5830美元）悬赏，寻求“可能的不当行为或舞弊”线索，但截至目前未提交证据，欧盟观察员也未发现确凿舞弊证据。藤森呼吁保持冷静，表示在计票有显著进展前不发表评论。
民调显示大量选民投票意向不定或拒绝支持任一候选人，本次创纪录有34名候选人参选。桑切斯主张制订新宪法并加强国家对自然资源的控制，是被罢免入狱的前总统佩德罗·卡斯蒂略的盟友，其支持率上升令市场波动。
监票组织“透明度”负责人阿尔瓦罗·亨斯勒表示，全面计票可能还需数周，大部分未计票来自偏左的农村地区和海外选民，呼吁公众耐心等待。来源：路透社
