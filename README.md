布加勒斯特，4月27日电 — 罗马尼亚议会最大党社会党周一表示，将与极右反对党“团结罗马尼亚人联盟”（AUR）合作，试图罢免本月早些时候离开的亲欧盟执政联合体政府，此举可能危及该国的欧盟资金。 
本月社会党部长们已从自由党总理伊利·博洛扬领导的联合政府辞职，导致政府失去议会多数，影响获取欧盟超过100亿欧元的疫后恢复资金以及主权评级和债券收益率。博洛扬拒绝下台，称政府必须完成关键改革以在欧盟8月截止日前拿到资金。 
该联合政府在2024年12月大选后约十个月组建，旨在遏制极右势力扩张。但围绕将2024年逾9%的赤字降至今年6.2%的预算削减，两派多次发生冲突。 
社会党曾表示愿回归亲欧盟阵营，但前提是不再由博洛扬领军。自由党目前支持博洛扬，指责社会党违反十个月前签署的合作协议，排除与其再组阁的可能。 
AUR在民调中领先约35%，两党合计掌握议会464席中的约220席，若联合部分小型极右翼派系可达到罢免所需的233票。博洛扬也将努力巩固支持。罗马尼亚从未举行过提前大选，下一次大选原定于2028年。 
前副总理马里安·内亚克苏被问及是否可能与AUR组建执政联盟时只表示：“每一次旅程都始于第一步。”AUR领袖乔治·西蒙称，一旦收集到233名议员签名，将提交罢免动议，投票可能在5月5日进行，事后其党愿与所有政党对话。 
路透社
