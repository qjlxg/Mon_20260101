联合国难民署：中东危机正阻碍全球人道援助
联合国难民署警告，中东危机已产生远超地区范围的深远连锁效应，对全球人道主义供应链及援助交付的负面影响日益严重。
新华社报道，联合国难民署发言人沃尔夫星期五（5月1日）在日内瓦新闻发布会说，海湾主要航线及周边地区，特别是霍尔木兹海峡的安全局势日益恶化，导致海上运输受阻。
全球范围内燃料、食品、货运成本上涨，推高了物价，并延误了关键物资的交付，降低了援助机构及时提供援助的能力，冲击数百万难民和流离失所者等的生计。
沃尔夫说，自中东冲突爆发以来，主要物资供应国的货运价格上涨了近18%，部分货物的运输成本甚至增加了一倍多。

她警告，若中东局势持续动荡，成本上升、运输延误及运力受限，将进一步压缩人道主义行动空间。

RSF警告称，俄罗斯（第172位）利用反恐、分裂主义和极端主义法令限制新闻自由，截至2026年4月，仍关押48名记者。
跌幅最大的是由军政府掌控的尼日尔（第120位，下跌37位），RSF表示这反映了萨赫勒地区近年来在武装团体和军政府打压下，获取多元平衡信息权利的全面衰退。AFP报道。

为重启航运，美国宣布发起名为“海事自由构想”的国际行动，旨在恢复霍尔木兹海峡通行。法国外长让-诺埃尔·巴罗特表示，此项任务是对既有多国努力的补充，而非竞争。

阿布扎比消息（5月1日）—— 法国外长让-诺埃尔·巴罗表示，美国牵头重启霍尔木兹海峡航运的联盟将对英法主导的类似行动起到补充作用，而非竞争关系。
巴罗在结束对沙特和阿曼的地区访问后，于阿联酋首都会见海湾盟友时表示，他已就英法倡议向当地伙伴进行了通报，称该英法任务已进入“成熟”阶段。
路透社/法新社援引消息称，美方于4月30日确认启动名为“海上自由构想”（Maritime Freedom Construct）的多国联盟，旨在恢复这一关乎全球能源运输的重要航道。霍尔木兹海峡通常承载全球约五分之一的石油和液化天然气，因伊朗在中东冲突中实际封锁该海域，导致油价飙升并扰乱贸易网络。
巴罗称，美方行动“性质不同，是一种补充”，并“并非与我们发起的倡议竞争”。对外长是否会加入美国倡议，他表示目前无法置评。华尔街日报报道一份外交电报呼吁美使馆敦促他国参与。特朗普曾批评盟友不愿介入并敦促依赖石油国家承担责任；伊朗则声明只要美国封锁其港口就不会开放航道。本次封锁已使油价创近四年新高。AFP

特拉维夫——彩色鸽子图案为舞台背景，耶路撒冷青年合唱团以希伯来语和阿拉伯语演唱，为“人民和平峰会”主会场揭幕。第四月30日，第三届年会在特拉维夫举行，数百人出席。大会由“It’s Time”联盟主办，该联盟汇集80余个组织，旨在通过协议结束以巴冲突，保障两国人民的自决权。
会场气氛带着倔强的希望。与会者承认，临近十月大选、在以色列史上之一的极右翼联合政府下，左翼和平运动面临更大挑战；正如一位台上发言人所言，“和平已成一个污名”。
著名歌手亚希诺安·尼尼（艺名Noa）表示：“总有乐观的理由，也总有为和平努力的理由。”她并称，在对外多线作战的同时，以色列国内也在为能否保持自由民主而斗争，“临选只会更糟”。
大会虽规划为以巴联合活动，但与会者多为以色列和平阵营的老面孔。和平现在、并肩而立、丧亲者论坛等组织在主厅设展台。自2023年10月7日哈马斯袭击及随后加沙战争以来，约旦河西岸和加沙的活动人士因通行许可受阻未能到场，但仍有数十名青年运动成员及以色列的巴勒斯坦公民参加。
来自北以卡夫尔曼达的教师塔哈尼·阿卜杜·哈利姆说：“如果不团结所有和平运动，国家将难以为继。必须改变，两个民族要共存，政府不仅代表犹太人，也要代表阿拉伯人。”
在主会前的一个论坛上，法国外交部长让-诺埃尔·巴罗通过视频宣布，法国将在六月主办一场致力于“两国方案”的国际会议，距上次类似会议已有一年。
以色列媒体对本次峰会报道寥寥，反映出和平阵营在国内话语中的边缘化；议会中间派与右翼主流代表未见到场，仅有少数左翼与阿拉伯议员出席，其中远左哈达什党议员奥费尔·卡西夫表示：“我预计最坏，但希望最好。我们将坚持信念，为两国人民争取更好未来，终将胜出，只是时间问题。”（法新社）

土耳其警方5月1日在伊斯坦布尔对五一示威施放催泪瓦斯并逮捕数十人，全国多地则有数千人集会。人权律师协会CHD称，伊城近370人被拘，法新社记者见到防暴车向人群发射催泪弹。
反对频道HALK TV画面显示土耳其工人党主席厄尔坎·巴斯被胡椒喷雾笼罩。巴斯表示，掌权者一年365天都在发声，工人至少一天也该讲述他们的困境。
市区欧洲一侧有两支欲前往曾为多次反政府抗议现场的塔克西姆广场的队伍被特别盯防，广场前夜已被封锁。工会官员巴萨兰·阿克苏在批评封锁后被捕，他愤怒称，不能把广场对土耳其工人关闭，只有劳动者和穷人被拒之门外。
每年五一当局都会在塔克西姆周边重兵部署。2025年抗议转至卡德柯伊时逮捕逾400人；2026年逮捕人数似乎接近该规模。CHD在社交平台X表示，格林威治时间11:00拘押人数为370人。
伊斯坦布尔多处可见防暴警力和金属路障阻断通行。法新社在梅吉迪耶居看到警方向包括马克思主义政党HKP成员在内的人群施放催泪瓦斯，后者一边试图突破一边高呼口号。贝西克塔什街区的警察在示威者呼号时多次介入，有记者目击数人被推倒在地。
工会与民间团体以“面包·和平·自由”为口号发起示威。土耳其官方通胀率约30%，独立估计更接近40%。
首都安卡拉约100名进行为讨薪而绝食9天的煤矿工人在获呼声中加入游行；当日游行规模较大且年轻化，警方亦严密监控。本周早些时候，当局对62人发出逮捕与搜查令，认为其中46人（包括记者、工会人士和反对派人物）“可能实施袭击”。
