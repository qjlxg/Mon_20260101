台湾拟增加反舰导弹至逾1800枚 应对北京封锁威胁
随着北京对台军事压力持续升高，台湾计划在2029年初前将反舰导弹库存大幅增加至1800枚以上，以强化应对中国大陆封锁或武力攻台的能力。
据路透社依据军售数据、美国出口许可文件、防务分析师评估以及台湾官员访谈所作的统计，台湾正持续扩充可由战机、军舰及地面发射系统发射的反舰导弹。
报道称，这是台湾推动“不对称作战”战略的重要组成部分，即通过大量价格相对低廉但杀伤力强的武器，抵消北京在火力上的压倒性优势。

“祖辈是否加入过纳粹？” 德国纳粹成员档案公开 更多德国人或面对艰难事实
许多德国家庭长期回避一个不安的问题：祖辈在二战期间做过什么。如今，这个问题在网络上或许就能查到，但答案往往并不好面对。 
法新社报道，美国国家档案馆今年3月将约1200万张纳粹党党员登记卡数码扫描件公开上网。这批资料由美军在二战击败纳粹德国后缴获。随着档案数码化并全面公开，原本仅能通过微缩胶片查阅的记录，如今可直接检索。尘封数十年的信息由此重见天日。 
德国媒体《时代周报》和《明镜周刊》随后推出在线查询工具，协助公众搜索相关记录。“祖辈是否加入过纳粹？”也成为德国新闻报道中频繁出现的话题。
自此，大量德国人开始在数据库中查找家族成员姓名，即使可能面对难以接受的事实。
60岁的德国女士科琳娜发现，父亲在1935年加入国家社会主义德国工人党（简称国社党，通称纳粹党），当时距纳粹头子希特勒上台仅两年。这一信息来自她26岁女儿海伦娜在档案中检索到的记录。
科琳娜说：“当我女儿打电话告诉我这个消息，并发来截图时，我非常震惊。”
她过去只知道，父亲曾在法国和俄罗斯作战并受伤 ，且原以为父亲出身工人家庭、长期支持社会民主党。
历史学者施波尔（Johannes Spohr）指出，许多前纳粹党员不仅保持沉默，还会重塑叙事，把自己描绘为受害者或旁观者。
部分研究显示，战后德国社会中约11%至18%的人相信，祖辈曾帮助纳粹受害者，但历史证据表明真实比率远低于1%。
历史学者普尔姆（Felix Puelm）在母亲去世后，发现祖母于1940年加入纳粹党，当时年仅19岁。他认为，祖母当时已清楚战争在扩大，却仍选择加入。
普尔姆指出，入党时间具有重要意义：若在1920年代或1930年代初、希特勒上台前入党，通常意味着较强的意识形态认同；但在1933年后，更多人可能因职业或现实压力加入，例如公务员和教师群体中，党员比率较高。
他强调，当时虽存在社会压力，但并无强制入党制度。新开放的档案或将推动德国社会重新思考历史与现实的关系，甚至映照当下极右翼政治力量回潮。
他希望这些发现能促使更多家庭思考：人们为何会选择加入这样的政党。

旅客因英国电子入境许可（ETA）系统故障遭遇阻滞
伦敦——近期部分拟入境英国的旅客在申请电子入境许可时遇到系统瘫痪，导致无法在登机或登车前取得必须的数字授权。6月3日下午，ETA应用显示“系统繁忙，请稍后再试”，网站的数字排队页面也停滞并显示超过一小时的等待时间。
负责该项目的英国内政部表示已“注意到部分用户遇到延迟”，技术人员正“昼夜工作”修复问题。自2月起，大多数入境英国的免签旅客需申请ETA，费用为20英镑，有效期两年，适用于不超过六个月的短期访问，涵盖美国、加拿大及大多数欧洲国家。
此前审批通常很快，官方建议至少提前三天申请。但仍有旅客在出发当天才申请，近期因此滞留。例如：居住纽约的Zara Rahim与伴侣在6月2日晚到肯尼迪机场赶赴Delta航班赴伦敦参加婚礼，连续等待近六小时未获批；荷兰莱顿大学教授Jonathan London和巴黎文化策划人Ella Marder分别在乘坐Eurostar和从巴黎出发的当日遭遇未能及时获批、无法登车的情况。多位旅客反映车站内约有百余人陷入同样境地。
内政部未就故障何时开始或何时恢复作出具体说明。6月3日下午，有乘客陆续收到获批通知。

刚果（金）卫生部称，本周东部南基伍省发生针对埃博拉遗体安葬队的袭击，迫使救援人员弃置一具棺材，增加了病毒进一步传播的风险。袭击发生在6月1日卡塔纳镇，该地约距省会布卡武北约30公里，目前由AFC/M23叛军控制。
被袭的是“安全与体面下葬”专门队伍，按严格防疫规范处理高度传染性遗体。卫生部发布的6月3日情况报告称，棺材随后被社区成员处理，此类高风险做法可能引发新的传染链。卫生部与当地医院负责人未说明袭击动机。
当局表示，袭击凸显出持续的猜忌与抵抗，阻碍疫情防控。近期安葬队和医护人员多遭袭击，部分为质疑死因的遇难者家属所为。同日（6月1日）在伊图里省首府布尼亚也发生类似事件，一支应对队在公墓遭殴，至少4人受伤。
疫情方面，自5月15日宣布暴发以来，这是该国第17次爆发，确诊病例已达363例，死亡62例。卫生部最新通报新增19例确诊（含2例死亡），感染已扩散至伊图里省36个卫生区中的17个，并在北基伍省7个卫生区和南基伍省1个卫生区出现病例。
尽管形势严峻，卫生部也报告了积极进展：伊图里省Rwampara的32名密切接触者完成21天监测，未发现埃博拉感染；北基伍省戈马市一名康复患者6月3日准备出院，AFC/M23副协调员弗雷迪·卡尼基在社交平台X上表示该患者已与家人团聚。 （路透）
