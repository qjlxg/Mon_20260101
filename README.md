迪拜讯：伊朗议会专家委员会于2026年3月8日任命穆吉塔巴·哈梅内伊为最高领袖，标志着伊斯兰共和国首次出现父子继承。此举在美以空袭和地区冲突升级的背景下，凸显政权意在延续强硬路线。
穆吉塔巴，1969年9月8日生于马什哈德，现年56岁，是已故最高领袖阿里·哈梅内伊的次子。长期在“贝叶特”（领导办公室）隐身运作，未曾担任正式公职，但被视为掌控军事、情报与政治渠道的关键人物，尤其与伊斯兰革命卫队及巴西吉准军事组织关系密切。分析人士称，他长期充当父亲的“守门人”，在2009年总统选举危机及随后的镇压中颇具影响力，2019年曾被美国制裁。
任命发生在伊朗面临美国与以色列持续攻击之际，外界普遍认为这是战争时期体制内围拢熟悉人物的选择。德黑兰分析师称，穆吉塔巴熟悉安全与军事协调，因而是当前最务实的继承人；也有声音警告，此举可能激化国内对“世袭”质疑，并加剧与敌对方的对抗——以色列防长称若其指挥对以色列的攻击，可能成为“明确的消灭目标”。
伊朗领导层通过任命与现有权力结构关系密切的继任者，释放出坚持体制、生存优先的信息；但同时，新任最高领袖也将面临更大的外部威胁与政治风险，过去长期隐身的他已被推到全球聚光灯下。
