亚行启动融资计划 协助亚太开发关键矿物供应链
（撒马尔罕综合电）亚洲开发银行星期天启动一项新的融资计划，旨在协助亚洲和太平洋地区的国家开发用于清洁能源、电池、电动汽车和数码技术所需关键矿物的供应链。 
亚行声明说，这个计划将协助亚洲国家筹备项目，并为关键矿物的加工、制造和回收基础设施提供资金支持。

他指出：“这项机制关乎紧迫性和公平性：现在就建立负责任的供应链，这样我们的发展中成员国就能在先进制造业中竞争，并在国内创造机会。”

神田真人说：“电网和数码介入将决定亚太地区的未来……通过跨境连接电力网和数码网络，我们可以降低成本、拓展机遇，并为数亿人提供可靠的电力和数码接入服务。”

台学者：赖清德返台两途径 搭乘同架专机或民航经欧洲转机
台湾总统赖清德搭乘斯威士兰专机成功访非，外界关注他接下来的返台安排。有台湾学者认为，他可能原机返台，或搭乘民航客机经欧洲等地转机。
赖清德原定4月下旬出访非洲邦交国斯威士兰（台译史瓦蒂尼）的行程一度因专机航权取消而暂缓，但他上星期六（5月2日）以高度保密的方式飞抵当地首都。台媒称，他是搭乘斯威士兰副首相德拉德拉（台译札杜莉）4月30日赴台访问后的回程专机。
《自由时报》星期一（4日）引述台湾政治大学政治系兼任教授严震生分析指，赖清德回台湾主要有两种途径，除了原机返台，还可能采取台外交部长林佳龙上周访问斯威士兰的模式，搭乘民航客机，从欧洲等地转机返台。
另据《上报》报道，曾在台湾前总统马英九出访时担任专机机师、现任台北市议员的张志豪星期天（3日）分析，赖清德回程理论上也可能采取与去程相同的安排，即搭乘斯威士兰专机，“整体上会比较顺利”。
此外，严震生指出，台湾总统过去访问邦交国都不需如此保密，这次即使是成功出访也是特殊情况下的权宜之计，存在美中不足。
他进一步解释说，此番赖清德出访，虽是因行程提早被公布疑遭北京施压，后续才改用比较秘密的方式，但严震生仍认为，总统出访邦交国要保密，不算好事。
对于赖清德出访采取“抵达后通报”（Arrival then Announce，ATA）的模式，以及台湾外交部以美国总统特朗普、前总统拜登曾以这种模式出访过作为类比。
严震生认为，特朗普、拜登多是去战区劳军，行程若曝光恐有安全疑虑，但赖清德本次是出访没有安全疑虑的邦交国，因此不能相比。

华盛顿——随着5月3日进入距11月中期选举最后六个月，控制国会的争夺进入关键阶段，这次选举可能重塑特朗普第二任期并改变华盛顿权力平衡。
中期选举一向被视为总统的公投，民主党希望借经济不满和特朗普约40%的低支持率收复众议院与参议院，共和党则力争在逆风中保住席位。
掌控国会不仅关系立法权，更决定能否对总统展开调查、阻挠提名等，影响其任期走向。早期迹象显示共和党面临挑战，通胀与伊朗战争相关开支等经济问题正侵蚀民众信心。
民调显示民主党在总体投票意向上略占优势，部分调查也显示选民在经济问题上对民主党信任度上升。亲共和团体AFP Action警告，参议院多数党面临风险，称存在结构性逆风和选民冷漠。
本次中期选举将改选全部435个众议院席位、约三分之一的100席参议院以及多数州长职位。民主党若要掌控两院，需在众院净胜三席、参院净胜四席。
民调机构Impact Research的莫莉·墨菲指出，当总统支持率在或低于40%时，更容易在传统偏共和的地区看到席位变动。
选情仍然流动：共和党受益于相对有利的参议院版图和经过严重划分的选区，但多州正在进行年中重划，包括德州、加州、北卡、俄亥俄、佛罗里达、密苏里、犹他和弗吉尼亚，最高法院限制基于种族的重划也改变格局。
参议院领袖舒默指责共和党通过过度选举安全和移民执法措施削弱投票权，称这是“操纵制度”的企图。共和党则依靠资金优势并强调移民与国家安全议题，认为最终取决于投票率与选民对其执政成果的认可。众议院多数党领袖斯卡利斯表示，选情将取决于选民投票热情和对共和党“扭转局面”成果的评价。
