上海市政府宣布，自2月1日起在市区开放“免申报”无人机飞行空域，覆盖约46%的行政区域，以推动低空经济发展。新出台的民用无人机指引规定，经登记的消费级无人机（微型、轻型和小型）可在划定的“适航空域”内免事前申报飞行。

在部分受限空域，飞行仍需当地航空管理部门审批，操作者可通过“随申办”APP在线申请。该APP同时为无人机用户提供登记、航班申报、体验区预约等一站式服务，并在地图上标注免申报与受限空域。 
此举旨在配合一年多前启动的市级低空经济发展行动计划，进一步培育本地无人机市场与相关产业。
