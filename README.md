该官员称，马克龙与希腊政府还将就海上安全及霍尔木兹海峡局势进行磋商。法国与希腊作为长期北约盟友，曾于2021年签署一项安全与防务协议，约定遭受攻击时相互援助，并包括希腊购买三艘法制护卫舰和约24架达索“阵风”战机的采购计划。
目前背景是伊朗重申对霍尔木兹海峡的控制后，该海域航运一度停滞，恰逢与美国的脆弱停火即将到期。希腊虽领导欧盟在红海的护航行动，但反对在霍尔木兹海峡采取军事行动。路透社

德黑兰 — 在伊朗与美国的僵局中，霍尔木兹海峡4月19日仍被封锁，议长穆罕默德·巴格尔·盖利巴夫表示，尽管谈判出现“进展”，但最终协议“还很遥远”。
盖利巴夫在4月18日晚的电视讲话中称，双方在高层斡旋后仍存在“许多分歧和一些根本性问题”，并重申除非美国取消对伊朗港口的封锁，否则伊朗不会重新开放这一重要海上通道。
两周停火将于4月22日届满，除非续约。美国总统特朗普表示与伊朗有“很好的对话”，同时警告德黑兰不要试图“敲诈”。此前4月17日因停火短暂生效，海峡曾一度开放并导致油价大幅回落，但在特朗普坚持延续对伊朗港口封锁后，德黑兰改变立场再次封锁。
伊朗革命卫队警告称，未经许可擅自通过的船只“将被视为与敌人合作并成为攻击目标”。追踪数据显示，4月18日海峡短暂开放时仅少数油气油轮通过，次日入海口几乎停滞。
过去几天发生多起危险事件：英方海事安全机构称革命卫队向一艘油轮开火，情报公司Vanguard Tech报道卫队威胁要“摧毁”一艘空游轮，另有船只疑遭不明弹片击中导致货柜受损。印度外长称已召见伊朗大使就两艘印度旗船只遭射击提出抗议。
外交斡旋方面，参与调停的埃及与巴基斯坦表示有望“在数日内”达成最终协议。但一大障碍是伊朗约440公斤近武器级浓缩铀的去向。特朗普称伊朗同意交出该批铀并将用“挖掘机”取回，伊朗外交部与总统办公室则否认会将核物质移交美国，并强调伊朗拥有“合法的核权利”。
这场中东冲突始于2月28日美以对伊朗的大规模攻势，冲突迅速蔓延，伊朗对驻海湾美军基地所在国家实施打击，代理武装真主党也将黎巴嫩拖入对以色列的火箭袭击。

巴黎/迪拜——欧洲盟友担心，一支经验不足的美方谈判团队正推动与伊朗迅速达成一项“框架”协议，以博取总统川普的外交胜利，但此举可能固化而非解决复杂问题，多位曾与德黑兰打交道的外交官警告称。
他们担心华盛顿会先就伊朗核项目与制裁解除签署表面协议，随后在技术性后续谈判中陷入长期僵局。白宫发言人安娜·凯利反驳称，总统有取得好协议的记录，只会接受“把美国置于首位”的方案。
多名欧洲外交官指出，2013—2015年达成的伊核全面协议（JCPOA）由欧英法与美方共同促成，川普于2018年退出。近期在巴基斯坦伊斯兰堡恢复接触，仍以核限制换取经济缓解为核心。
争议焦点包括约440公斤、浓缩至60%的铀库存处理方案：首选在伊朗境内受国际原子能机构（IAEA）监督的“降级混合”，或把部分物质运出境外（曾提及土耳其或法国），但运送、核查与战损物资回收均需复杂谈判。
更深层分歧是伊朗是否享有铀浓缩权；川普阵营要求零浓缩，伊朗坚持民用权利并否认寻求核武。欧洲方面强调IAEA需有强有力的入侵式核查与不受限访问。
经济议题集中于取消制裁与解冻资产，但须循序推进并争取欧洲支持。地区国家亦有各自关切：海湾国盼限制导弹与代理人网络，以色列要求更严格约束；伊朗则将剩余导弹能力视为重要威慑。
欧方承认因推动恢复制裁和将伊朗革命卫队列为恐怖组织而被边缘化，但表示美方团队缺乏处理该议题的长期专业经验。白宫则称国家安全委员会、国务院和国防部官员已参与伊斯兰堡会谈并持续参与。

英国官员4月19日公开支持首相基尔·斯塔默，斯塔默因长期与杰弗里·爱泼斯坦有关联的人士彼得·曼德尔森引发的任命丑闻面临压力。
斯塔默将于4月20日在议会说明为何曼德尔森在2024年底被任命为驻美大使，尽管未通过安全审查。斯塔默称未被告知审查未通过，称此事“不可原谅”，并于4月16日解雇外交部首席文官奥利·罗宾斯，指责外交部高层在安全意见之外推进任命。
前公务员指责唐宁街将责任推给他人；反对党则要求斯塔默下台，指控包括无能或故意误导民众。科技大臣丽兹·肯德尔与副首相、大卫·拉米（任命时为外交大臣）均表示，如果知道曼德尔森未获适当安全许可，斯塔默不会任命他。肯德尔称斯塔默在与欧盟拉近关系及限制英国卷入伊朗冲突等重大问题上做出正确选择，应继续留任。
斯塔默此前因数项政策失误已广受批评，因其选中与爱泼斯坦关系为人所知的曼德尔森而被质疑判断力。斯塔默在2025年9月解除曼德尔森职务，之后有更多关联细节曝光。曼德尔森被指在15年前任工党部长时涉嫌失职，警方已对此展开调查，他于2月被捕后获释，尚未被起诉并否认违法。AFP

伦敦——英国首相基尔·斯塔默4月19日表示，将追查并绳之以法，针对近期伦敦多起针对犹太场所的纵火或未遂纵火事件。他在社交平台称对此“感到震惊”，并强调对犹太社群的袭击就是对英国的袭击，政府将不遗余力追捕肇事者。
伦敦警察厅反恐部门正调查过去一个月内多起在伦敦西北部发生的纵火或未遂纵火案件。社区安全信托（CST）称，4月18日哈罗区Kenton联合犹太教堂一间房间出现轻微烟损，巡逻警员发现窗户受损并有装有助燃剂的瓶子被投掷入内，所幸无人受伤。附近亨登区一处曾由慈善机构Jewish Futures使用的房产在4月17日晚遭到袭击。
警方已在犹太人聚居区加派巡逻，反恐警员主导调查。4月15日，警方因芬奇利一座犹太教堂纵火案拘捕两人；三月底北伦敦Golders Green有四辆社区救护车被焚毁。情报机构SITE称，一个名为Harakat Ashab al-Yamin al-Islamiya（HAYI）、与伊朗有关联的小组织声称对多起袭击负责，并在比利时和荷兰也曾发起类似行动。
英国首席拉比以弗拉因·米尔维斯警告，针对英犹太社区的“持续性暴力和恐吓”正在加剧，虽然目前未造成人员伤亡，但社会不应等到出现严重后果才意识到事态的危险性。

安哥拉基兰巴（路透）——教皇里奥4月19日在首都罗安达附近的一个土场为约10万名信众主持弥撒，呼吁该国放下历经数十年的血腥冲突所带来的分裂。
在其非洲四国行中规模最大的活动之一，教皇称安哥拉为“美丽却受伤的国家”，敦促国民“共同建设一个彻底克服旧有分裂、消除仇恨与暴力的国家”。弥撒结束时，他还谴责乌克兰战事近期升级，呼吁“停止武器，让对话取代冲突”，并称以色列与黎巴嫩之间为结束以军与伊朗支持的真主党之间战斗而达成的停火是“希望的迹象”。
信众冒着闷热早到基兰巴的广场等候，当弥撒开始，现场人潮已将场地挤满，教皇乘白色教皇专车经过时众人载歌载舞、欢呼欢迎。到场者基督娜·马藤德修女说：“教皇到来令人振奋，我们正处于诸多困难之中。”
4月18日，作为其非洲行第三站，教皇在对安哥拉政界发表讲话时抨击了非洲自然资源被掠夺的现象，批评那些“保证财富却未兑现承诺”的暴君与独裁者，导致民众受苦、丧生，并敦促领导人关切全体人民而非只顾企业利益。“历史终将证明你们是对的，即便短期内有人反对你们，”他说。
安哥拉是撒哈拉以南非洲主要产油国之一，人口约3660万，但据世界银行逾30%人均每日生活费不足2.15美元；全国逾半数为天主教徒。另有年轻信众安妮卡·卡利亚塔表示，教皇在非洲行中直言不讳的风格将促使民众反思并共同为和平努力。

日内瓦讯——奥地利警方18日晚通报，在一罐HiPP婴儿食品中检测出老鼠药成分，此前该批产品已因安全担忧在奥地利逾千家SPAR超市被召回。
布尔根兰州警方称，一名消费者上报的一罐190克“胡萝卜与土豆”婴儿食品样本检测出鼠药。HiPP18日表示，不排除有人向产品中投放危险物质，相关罐装可能遭人为篡改，食用罐内物质可能危及生命。
警方指出，涉事罐底部贴有红圈贴纸，罐盖已被打开、破损或缺失安全封条，或有异常气味。对捷克和斯洛伐克警方扣押的类似罐装样本初步检测也显示有毒物质存在。奥方还称，德国方面的调查曾对奥地利当局发出风险警示。
HiPP称这是针对SPAR奥地利流通渠道的外部刑事干预。SPAR表示召回为预防性措施，约影响奥地利1500家门店，未影响其他国家销售，消费者可退货全额退款。警方提醒，如接触过涉事罐装应彻底洗手。来源：路透社
