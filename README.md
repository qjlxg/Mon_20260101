“其次，我们重申支持国际海事组织（IMO）为确保海员和船只安全所做的重要工作。我们将继续与航运业合作，确保它们在条件允许的情况下尽快恢复运营，包括与航运公司、保险公司、行业机构进行沟通。”

利马，4月17日 — 随着总统选票计票进程缓慢并出现被指不规范的情况，要求撤换秘鲁选举机构国家选举办（ONPE）负责人皮耶罗·科尔韦托的呼声周五升温。由于尚未出现能在6月第二轮挑战保守派领跑者凯科·藤森的明确对手，局势更加不确定。 
ONPE数据显示，截至周五已计票93.3%，藤森以约17%领先；左翼的罗伯托·桑切斯以12.0%紧随其后，极右前利马市长拉斐尔·洛佩斯·阿里亚加以11.9%位居第三，二者相差约1.3万张选票。地方观察组织Transparencia表示，最终结果可能需要多达两周才能公布。 
ONPE称约5%的选票因投票站记录缺失或错误被标记须复查，这部分选票将由特别选举陪审团审定后并入最终计票。企业界和跨党派议员纷纷要求科尔韦托下台，商会CONFIEP主席豪尔赫·萨帕塔警告称，“如此严重的错误是有后果的”。 
科尔韦托本周承认在利马等地曾出现后勤延误，致使投票延长一天，但否认存在舞弊。洛佩斯·阿里亚加曾指控舞弊并要求暂停计票。国家选举法庭已向检方对科尔韦托提起刑事控告，指称包括侵犯投票权在内的违规行为。科尔韦托方面尚未回应路透社询问。 
警方表示，周四在利马一条公路上发现四个投票站的选举材料，目前正调查此事；ONPE称这些站点的票数已被记录。欧盟观察团本周未发现舞弊证据。 
市场方面，桑切斯上升为第二名曾引发对其拟改写宪法和加强国有资源控制的担忧，但对2026–2031届国会可能偏右的预期部分缓解了投资者顾虑。周五午间索尔币小幅波动，利马基准股指上涨0.4%。 
秘鲁过去十年内政局动荡，经历八任总统，但经济仍是拉美较为稳健的之一。来源：路透社
