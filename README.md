五年来最快增速 香港今年第一季度GDP增5.9%
香港经济出现近五年来最快增速。旅游和零售支出等传统增长引擎，缓冲了伊朗战争引发全球能源危机带来的冲击。
香港政府统计处星期二（5月5日）公布初步估算数据显示，今年第一季度GDP同比增长5.9%，高于彭博调查中所有经济学家的预测；调查预估中值为增长3.5%。
这已增幅也较2025年最后三个月4%的增速进一步加快。
香港政府发言人在声明中表示，经济提速主要受人工智能相关电子产品全球需求强劲带动。
发言人表示，展望未来，香港经济增长前景仍然正面。但他同时警告称，中东地区持续紧张局势仍是经济前景面临的下行风险。
彭博社指出，随着中东冲突扰乱市场和全球供应链，香港更加依赖旅游等传统经济支柱。这个亚洲金融中心几乎全部能源依赖进口，发电也主要依靠化石燃料，因此更容易受到这场战争引发的历史性油价冲击影响。

美国考虑审查新AI模型防止AI网络攻击带来政治冲击
（旧金山综合电）美国媒体报道，特朗普政府考虑要求对人工智能（AI）模型进行监管，之后才能向公众发布，这与此前政府不干预AI技术的态度形成鲜明对比。
《纽约时报》引述知情者透露，白宫正在讨论通过颁布一项行政命令，成立一个由科技高管和政府官员组成的工作组，负责研议新AI模型的潜在审查程序。据悉，白宫官员上周已与Anthropic、谷歌及OpenAI的高层会面，讨论部分计划内容。
美国政府官员希望避免毁灭性的AI网络攻击带来的政治冲击，并且也在评估先进模型是否能为五角大楼和情报机构提供实用价值。
特朗普此前一向主张让AI自由发展，他撤销了前总统拜登对AI制定的安全评估要求，认为对AI实施监管可能削弱美国与中国竞争的实力。拜登政府2023年发布行政命令，要求AI开发者向政府提交安全测试结果，并指示联邦机构针对AI制定标准，但特朗普去年初重新入主白宫后不久就撤销了这些措施。
不过，随着美国民众越来越担心AI对就业、能源成本、教育和心理健康造成冲击，特朗普政府对AI采取的不干预态度开始出现转变。这一转变可能与Anthropic的最新AI模型Mythos有关。Mythos能够自主识别主流软件系统中大量未被发现的安全漏洞，并开发出相应攻击手段，此先进模型一旦落入不法之徒手中，可能成为窃取数据或破坏关键基础设施的强大新武器。

世卫组织：游轮可能出现人传人的汉坦病毒
（日内瓦／阿姆斯特丹综合电）世界卫生组织指出，在暴发汉坦病毒疫情的荷兰游轮上，密切接触者之间可能存在人传人的情况。
世卫组织星期二（5月5日）说，在大西洋上航行的洪迪厄斯号（MV Hondius）游轮出现七起确诊或疑似汉坦病毒（hantavirus）感染病例。
其中，一对荷兰夫妇和一名德国公民已死亡，一名英国人则已从船上移送到南非接受重症监护。另三起疑似病例仍在船上，其中一人轻微发烧。
世卫组织初步推测是，这对荷兰夫妇在阿根廷登船后，可能是在船上进行观鸟等活动时感染。“在密切接触者之间，例如夫妻、同住一间舱房的人之间，可能存在人际传播。”
世卫组织也推断，这次的汉坦病毒可能来自安第斯病毒（Andes virus），这种病毒主要在阿根廷等南美洲地区传播，目前仍在进一步检测确认。这艘载有23个国家的147名乘客和船员，于3月从阿根廷南部的乌斯怀亚港出发。
目前的重点是撤离两名患病乘客，然后让这艘停泊在佛得角附近的游轮继续前往加那利群岛。不过，西班牙卫生部尚未决定是否接收游轮，仅称会先分析相关的流行病学数据，再作出判断。
船上人员正在接受严格的预防措施，包括隔离、遵守卫生规程和接受医疗监测。
不过，世卫组织强调，这个病毒目前对公众的风险仍然很低。
汉坦病毒感染是一种罕见、可能致命的疾病，主要通过接触受感染啮齿动物的尿液、粪便或唾液传播。目前没有针对这个病毒的疫苗或特效药物。

特朗普曾指责盟友对美在伊朗战争中的支持不足，甚至暗示美国或不再必须遵守北约第五条集体防御条款，并一度以掠夺格陵兰岛威胁丹麦，所幸荷兰首相吕特出面缓和。外交人士担心，在7月安卡拉峰会前，如果伊朗战争尚未结束，特朗普可能再次采取考验盟友的举动。“北约和欧洲盟友的长远棋局是熬过安卡拉，”一名欧洲外交官表示。
欧洲方面不得不尽力留住美国：多国在伊朗战事期间允许美军使用境内基地或领空（西班牙例外），包括英国、克罗地亚、法国、德国、希腊、意大利、黑山、葡萄牙和罗马尼亚等国。官员们还在向美方、国会议员及保守智库论证：坚持北约符合美国利益，并推动战后霍尔木兹海峡可能的护航任务。荷兰、比利时与德国表示将派舰艇参与，法国与英国主导相关方案。吕特称，欧洲已收到美国明确信息，正为可能的后方与后战争行动做预置。  REUTERS

霍尔木兹海峡航运几乎停滞，尽管美国总统特朗普日前承诺将推动恢复通航。4日并未出现通航回升迹象，MarineTraffic数据显示，进入阿曼湾的仅有一艘受制裁的小型液化石油气油轮、几艘货船和一艘敷设电缆船。未见油轮排队等候过航，德国航运集团赫伯罗特（Hapag-Lloyd）表示，由于缺乏关于安全通行的明确程序，其船舶仍无法通行。 
美国中央司令部（CENTCOM）称将于4日开始协助恢复海峡通航，同时继续对伊朗港口实施封锁。航运业尚未收到关于美方行动意图和指引的说明，国际海运经纪与顾问协会（BIMCO）警告称，若未获伊朗同意，尚不清楚是否能削弱或压制对商船的威胁；BIMCO为业界提供安全警示。国际海事组织表示，数百艘商船和最多2万名海员因伊朗冲突无法通行该水道。 
美方主导的联合海事情报中心将海峡海上安全威胁评为“危急”，建议船舶考虑绕行阿曼领海（位于交通分离带以南）。CENTCOM将其行动描述为“防御性”，并称会将外交努力与军事协调相结合。伊朗则警告美军远离海峡，要求商船与其军方协调通行，并公布新控制区地图。 
巴基斯坦表示，上月被美军登船扣押的伊朗籍集装箱船Touska的22名船员已撤离至巴基斯坦并将遣返，船舶经修复后返还船东，巴方称此举为“增进互信”的措施。自4月13日以来美方在伊朗港口实施的海上封锁亦已导致德黑兰的石油出口收缩。路透社
