“大杜”简介：从国防部副部长晋升为国防部长
2026年3月14日报道，原任国防部副部长的阿杜·布恩塔姆查洛恩（Adul Boonthamcharoen，绰号“南伊善战士/大杜”）被任命为第2届阿努廷内阁的国防部长。此前曾传言总理阿努廷将兼任该部，但最终由大杜出任。
人物背景：大杜出生于1964年6月2日，原籍武里南省，为军人家庭，配偶为苏匹斯，育有两子。毕业于陆军军官学校与国防学院（与总理同届）。
军旅生涯：长期驻守东南部边境，曾任第26泰国游击营及第23步兵团等要职，负责泰柬边境与普拉萨特·磅威哈等争议地区的驻守与情报管理；后升任第6步兵师长、素拉纳里作战司令、第二军区副司令，并于2023年（2566 BE）出任第二军区司令，强调缉毒与邻国关系并重，因常年在南伊善一带基層历练，获“南伊善战士”之称。
政治生涯：退役后被任命为首届阿努廷政府国防部副部长，负责边境安全与军方战略；在新一届内阁中获提名出任国防部长，外界认为其与总理及武里南地方势力关系密切。

中国全国人民代表大会通过一项促进民族团结的新法，人权组织警告该法可能进一步边缘化少数民族。
该法将普通话确立为官方通用语言，明确用于教育和公共事务。学校被要求以普通话授课，完成义务教育的青少年需对普通话“有基本掌握”。法律还规定可对境外行为适用，境外从事“破坏民族团结”或煽动“民族分裂”的人员可被追究法律责任。
法案核心还将“暴力恐怖活动、民族分裂活动、宗教极端活动”定为刑事打击对象。官方称此举是为应对社会的“前所未有变化”，以增强国家凝聚力。批评者指出，尽管法律未点名具体少数语言，但很可能影响维吾尔语、蒙古语、藏语等群体；西藏、内蒙古等地已有普通话作为强制教学语言的做法。
人权观察（Human Rights Watch）亚太区副主任Maya Wang表示，这部法律并非以实现平等为出发点，相关政策正被强制推行于西藏等地，真正包容的模式不应剥夺儿童掌握双语的权利。

简历：“นภินทร”（纳平他·斯里桑潘）——掌控“ศรีเมือง”市场的关键人物，再任总理办公厅兼任部长
3月14日，安努汀二届内阁名单确定，总理办公厅的三位兼任部长均由原有人选继续出任：帕拉顿·普利萨南嫩古尔（普吉党资深议员）、素帕玛斯·伊萨拉帕基（前多届部长、党内要角）以及长期稳坐该位的นภินทร·ศรีสรรพางค์（纳平他）。纳平他此前在安努汀一届内阁已任同职，此次继续留任。
纳平他生于1961年5月30日，家乡为叻差布里府班蓬区，早年就读当地公立学校，获拉蒙康大学法学学士与政治学硕士学位。事业上为บริษัทแอ็กโกรคอมเมอร์ส กรุ๊ป有限公司董事总经理，同时担任叻差布里省大型蔬果批发市场“ศรีเมือง”董事长，该市场为泰国及东南亚区域重要农产品物流与分销中心。
从政路起于地方，曾任市议员、省议员及省级参议员。2011年以为泰党党团名单入党籍并递补为国会议员；2019年正式转投ภูมิใจไทย（保民党），出任旅游体育部副部长，成为党内掌握叻差布里与下中部选票的关键人。2023年列入党团名单，后获任商务部副部长，曾随党退出、又重返政局变动，最终在安努汀一届获任总理办公厅兼任部长，并在安努汀二届继续连任。
