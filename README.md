中国官方提示使用“龙虾”撰写专利申请存在风险
中国国家知识产权局提醒，使用OpenClaw（俗称“龙虾”）撰写专利申请文件或诱发多重风险。
据央视新闻报道，中国国家知识产权局星期三（4月1日）发布风险提示称，OpenClaw等智能体工具被曝光默认安全配置脆弱，易引发严重安全风险。
国家知识产权局提示，OpenClaw等智能体存在权限过高、安全漏洞、插件投毒等隐患，使用其撰写申请文件，易造成技术交底书等核心信息外泄。一旦泄露，专利申请技术方案可能因丧失新颖性无法授权，甚至被他人抢先申请专利，给申请人带来重大损失，代理机构还需承担违约赔偿责任。
国家知识产权局提示，在利用该类智能体撰写申请文件时可能出现“AI幻觉”，导致申请文件存在内容逻辑矛盾、技术特征表述不清等问题，从而无法获得保护。
国家知识产权局还提示，通过智能体凭空生成、随机编造、内容拼凑等形成专利申请，属于违反诚实信用原则的不诚信申请专利行为。达到一定数量，申请人将面临警告、罚款等行政处罚；代理机构及代理师将面临吊销执业许可证、注销代理资格证等行政处罚，情节严重的将被列入严重违法失信名单。
为防范风险，维护好各方合法权益，国家知识产权局提示，申请人需提高风险防范意识，审慎选择合规专利代理服务，主动了解并确认代理机构是否使用智能体撰写申请文件。如发现代理机构擅自使用相关工具造成信息泄露或构成不诚信申请行为，可依法投诉举报并要求代理机构赔偿损失。
国家知识产权局称，专利代理机构及代理师务必高度警惕智能体使用风险，杜绝使用智能体开展不诚信专利申请行为，切实保障委托人合法权益。

全球经济正感受到油价飙升带来的冲击。中东冲突使霍尔木兹海峡实际受阻——该通道约承担全球石油供应五分之一——市场剧烈反应，燃料与能源成本上升，导致多国启动应急措施。
美国：汽油均价升至每加仑约4.018美元，较二月底不到3美元大幅上升，加州、夏威夷和华盛顿等州价格更高。美国总统称遇高油价的国家“去自己找油”。
欧洲：欧盟呼吁成员国尤其在交通领域削减油气消费，并举行能源部紧急磋商协调应对。
南亚：斯里兰卡电价上调近40%，本月已三次提高燃油价并实行四天工作制以节能；印度卢比一度跌至逾95兑美元；孟加拉要求公务员节约用电。
东南亚与东亚：印尼决定维持燃油价格稳定，但通过削减开支（包括缩减免费学校膳食）应对预算压力，预计节省约$2.3 billion；韩国推出首个原油互换计划，允许炼厂借用战略储备用于缓冲供应中断。
中东及其他：伊朗拟对经霍尔木兹过境船舶征收通行费，挪威则宣布临时下调燃油税。各国普遍通过补贴、减税和节能等组合措施应对冲击，凸显全球能源体系的高度联动性。

3月31日，人民党党首 ณัฐพงษ์ เรืองปัญญาวุฒิ（绰号“เท้ง”）在脸书发文称，针对当天反贪委员会（ป.ป.ช.）通过决议，同意向最高法院就与前进党44名前国会议员有关的案件提出诉请——即他们联署提出修订刑法第112条（侮辱君主罪）一事。 
他强调，提出修法是国会议员固有的立法权和重要职能，无论外界是否赞同，议员提出修法不应被视为可遭受严重道德或刑事指控的行为。 
他指出，此事显示掌权精英正动用国家机制维护其利益，通过法律战拖制政治对手；但无论对方如何试图冻结或阻挡，党内人士将继续尽职尽责，哪怕未来最高法院作出不同裁定。 
下一步，党将参加政府施政辩论，并在议会提出解决北部及相近地区PM2.5细颗粒污染的相关提案。 
他呼吁全民共同争取法治与司法公正，确保法律对所有人一视同仁、不被当作政治武器，并建立强有力的制衡与监督机制，让政治归于人民。
