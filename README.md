从星期一起，更多民众加入抗议集会，并高举反对虐待动物的海报，警方星期二（9日）下午带走部分抗议民众，入夜后仍有许多民众在李姓男子住所外围逗留。
当晚，大石坝街道办发布消息指，公安机关已对李姓男子立案调查，将根据调查结果依法处理；李姓男子家中的另外三只小狗也已被送往爱心动物医院和收容所进行救治与寄养。

美国消费者价格指数飙至三年新高 特朗普：我喜欢通胀
受伊朗战争推高能源价格影响，美国5月消费者价格指数（CPI）飙升至三年新高，但美国总统特朗普对此不以为意。
法新社报道，当局星期三（6月10日）公布的数据显示，消费者价格指数（CPI）同比上涨4.2%，高于上月的3.8%，创下自2023年4月以来的最高水平。
5月份的消费者通胀数据显示，能源价格较去年同期上涨23.5%，其中汽油价格上涨40.5%。
食品杂货价格也连续第二个月大幅上涨，较去年同期上涨2.7%。当月其他价格上涨的项目包括医疗保健、个人护理、机票和娱乐。
特朗普对此告诉记者说：“数据很棒……我喜欢通胀。”
众议院议长、共和党人约翰逊随后辩称，总统的言论被断章取义，但参议院民主党少数党领袖舒默指出，这表明特朗普对美国家庭面临的经济困境漠不关心。 
舒默说：“（他在）镜头前。全美国的人都能听到。他对你们的蔑视简直无以复加。”
民主党在社交媒体发布一则新的竞选广告，包含特朗普讲话的视频。“特朗普把不该说的话说了出来——他喜欢通货膨胀。每个美国人都应该看看。”
特朗普也重申他之前的预测，即冲突结束后通货膨胀将“像石头一样暴跌”。
不过，经济学家对这一说法提出质疑，预计油价需要数月时间才能恢复到战前水平，具体取决于敌对行动何时结束。
美国星期三公布的另一份将通胀数据与近期工资数据结合的报告显示，实际平均时薪同比下降0.7%，创三年多来最大降幅。
物价上涨连同工资增长乏力，正在进一步挤压家庭预算，而此时消费者信心已处于纪录低点。这一问题很可能成为11月中期选举的核心议题。由于美国民众对特朗普处理经济事务的方式愈发不满，特朗普的支持率已经下滑。

彭博：忧受恒大清盘波及 普华永道部分合伙人寻求保全资产
中国房地产开发商恒大集团的清盘程序，据报正引发公司前审计机构普华永道（PwC）中国区部分合伙人的担忧，他们担心这场巨额索赔将冲击个人的财务状况，并正寻求方法保全个人资产。
据彭博社报道，普华永道香港和中国大陆分支机构的多名现任合伙人，以及数名近年离职的香港前高管均坦言，正加紧探索保全个人资产的策略，以防公司面临的法律和监管风暴波及自身。
一位合伙人透露，甚至考虑通过离婚来隔离财富，还有人已着手削减子女的教育预算。
这股担忧源自恒大清盘人今年5月在法庭上提出的一项诉讼，指控普华永道在为恒大审计时存在“疏忽”和“失实陈述”，并向普华永道国际有限公司及分支机构索赔高达570亿元人民币（108亿新元）。此前，香港会计及财务汇报局曾指出，普华永道的审计存在“极为恶劣”的缺陷，助长了恒大粉饰利润与流动性。
这些合伙人说，自5月份的香港法庭听证会披露出清盘人的索赔规模后，他们便开始为最坏的局面做准备。
合伙人们说，尽管他们并未直接参与恒大审计项目，但部分人员仍担心清盘人最终可能会对他们个人提起诉讼，或被要求通过所持公司股权支付赔偿金。此外，公司业务遭受的财务冲击也将影响他们的薪酬。
彭博社指出，由于香港审计机构普遍注册为“无限责任合伙企业”，这意味着“权益审计合伙人（equity audit partners）”在分红的同时必须共担债务。根据香港《合伙条例》，这种潜在的法律责任即使在合伙人离职后仍可能存在。
在香港，清盘人将个人合伙人列为诉讼对象的情况虽罕见，但也并非没有先例。
虽然目前尚未有恒大清盘人起诉个人合伙人的消息，但今年4月，香港会计及财务汇报局（AFRC）已因恒大相关的“严重审计缺陷”，对两名前普华永道香港合伙人分别处以500万港元（82万新元）的处罚。
今年4月下旬，普华永道香港同意支付13亿港元的罚款和赔偿金，其中包括向符合条件的恒大小股东支付10亿港元。
普华永道中国发言人拒绝就正进行的诉讼发表评论，仅说：“我们的业务继续表现良好，我们将继续专注于为客户交付高质量的成果，支持员工，并为公司和中国审计行业的未来进行投资。”

台湾在野的中国国民党主席郑丽文于美东时间星期三（6月10日）拜访五位美国议员，并称国民党的真正立场不是反对美国对台军售。她也批评外界给她贴上“亲中”标签是断章取义和刻意丑化。

这段采访视频曝光后迅速在中国互联网上疯传，但当天便被微博、抖音等社交平台下架，平台方并未说明具体原因。
视频被下架后，受访女生的姓名、就读学校及家庭住址等个人信息遭网民“起底”曝光。部分社交平台上甚至出现了针对她的攻击言论，部分声音指责视频存在“摆拍”嫌疑。

加拿大提新法案 拟禁16岁以下青少年拥社媒账号
加拿大文化部长提出一项法案，拟禁止16岁以下青少年开设社交媒体账户，并要求人工智能聊天机器人服务限制有害内容传播。
法新社报道，这项名为《数码安全法案》的提案使加拿大成为近期一系列因担忧儿童受到伤害，对社交媒体平台进行监管的国家之一。
法案规定，16岁以下青少年不得开设社交媒体账户，但若社媒公司能够证明已采取充分的儿童安全保障措施，可获豁免。
未能遵守法案的公司可能面临全球营业额3%或最高1000万加元（约923万6500新元）的罚款，以较高者为准。
除了社交媒体禁令外，法案也将监管日益普及的人工智能（AI）聊天机器人，要求社媒公司“降低聊天机器人传播有害内容的风险”。
公司也将被要求在“危机情境下的报告门槛”方面提高透明度，例如当用户有意伤害自己或他人时。
文化部长米勒（Marc Miller）在宣布提案的声明中说：“我们已经目睹网络危害可能造成非常严重的后果……儿童安全不容忽视。”
他说：“这项立法将为加拿大青少年提供一个更安全的环境，并赋予他们更多能力，让他们能够面对面交流、建立友谊、专注于学业，并学习实用技能，从而茁壮成长。”
今年4月，加拿大小型矿业小镇图姆布勒里奇发生大规模枪击事件，造成包括枪手在内的九人死亡。此后，这个问题在加拿大变得尤为敏感。 
去年6月，OpenAI因枪手在ChatGPT发表令人不安的言论而封禁相关账户，但未向加拿大警方通报，因为当时未发现即将发动袭击的证据。事件随后引发批评。
去年12月，澳大利亚成为全球首个要求TikTok、YouTube、Snapchat和其他热门网站删除16岁以下用户账户，否则将面临巨额罚款的国家。印度尼西亚今年3月也开始实施针对16岁以下用户的社交媒体禁令，多个欧洲国家政府也宣布有意采取类似措施。
