台湾花莲凌晨发生5.7级及4.8级地震
台湾花莲星期天（4月5日）凌晨先后发生规模5.7地震及4.8级地震，几乎台湾全岛有感。中央气象署指出，深夜规模5.7级地震为独立事件，也是当地截至目前今年最大地震，提醒未来三天可能还会发生规模5至5.5级余震。
据民视新闻网、自由时报等报道，花莲星期天（4月5日）凌晨1时14分发生规模5.7地震，地震深度17.5公里，最大震度4级，清晨5时23分再发生规模4.8地震，地震深度23.7公里。

台湾气象署则提醒，虽然这次是花莲地区正常的能量释放，但提醒未来三天有可能发生规模5或5.5的余震，请民众多加防范。
