美国幼儿溺亡回升 专家：溺水往往悄无声息
为什么越来越多幼儿溺水？原因尚不完全清楚，但专家强调，大多数悲剧其实本可避免。
美国儿童溺亡人数在冠病疫情后回升，可能与游泳课程中断和公共泳池关闭有关。美国儿科学会最新报告指出，预防溺水不能只靠孩子会游泳，还须同时加强成人看护、泳池围栏、救生衣等多重防护。
根据报告，溺水目前是美国1岁至4岁儿童的头号死因，死亡人数甚至超过先天缺陷。对5岁至14岁儿童来说，在意外死亡原因中，溺水仅次于交通事故。
2024年，美国共有1075名未满20岁的儿童和青少年因意外溺水死亡，其中也包括与船只有关的事故。
美国有线电视新闻网（CNN）报道，专家指出，美国儿童溺亡率在多年下降后，自疫情后又开始上升，可能和游泳课中断、公共泳池关闭，以及孩子接受水上安全训练的机会减少有关。不过，很多事故其实原本可以避免。
不少人以为，溺水的人会大声呼救，或用力拍打水面，但真实情况是：当孩子在水里挣扎时，通常会把全部力气用来呼吸，根本没办法挥手或喊叫。他们可能直立在水中上下浮沉，看起来像在爬一把看不见的梯子，也可能在没有发出声音的情况下沉入水底。
专家因此建议，当孩子游泳时，必须有一名成年人专门负责看护，不能一边聊天、一边看手机。幼儿在水里或水边活动时，看护者应始终待在伸手就能碰到孩子的距离内。即使现场有好几个成年人，也要提前说清楚由谁负责看孩子，避免大家都以为“有人在看”。
美国儿科学会也建议，住宅泳池应安装四面围栏，把泳池与房屋和庭院完全隔开，并设置可自动关闭和上锁的门。儿童乘船，或在湖泊、河流等天然水域活动时，则应穿着尺寸合适、获认可的救生衣。
多数儿童最迟应在4岁开始学习游泳。正式游泳课程可明显降低幼儿溺水风险，但并不能完全消除危险，家长仍须持续看护。臂圈、泳圈和浮板等充气用品也只是玩具，不能代替救生衣或成人监督。
