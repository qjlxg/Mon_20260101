亚洲人进门脱鞋有道理 实验发现鞋底藏致病细菌
在许多亚洲家庭，进门脱鞋是再自然不过的事；但在欧美一些家庭，穿着鞋进屋也很常见。这个看似普通的生活习惯差异，最近被英国广播公司（BBC）拿到实验室做了检测。结果发现，鞋底可能带着致病细菌，让人忍不住感叹：“比我想象中脏多了。”
BBC 近日前往英国布莱顿大学的微生物实验室，请微生物学家皮特（Sarah Pitt）检测一双穿过的鞋。皮特用棉签分别在鞋面和鞋底取样，再把样本涂到琼脂培养皿中。24小时后，培养皿里长出了不少细菌，其中就包括金黄色葡萄球菌（Staphylococcus aureus）。
皮特说，金黄色葡萄球菌可能相当危险。即使是健康人，如果皮肤有抓伤，细菌进入伤口，也可能引发脓肿或化脓性感染；它还可能导致肺炎等严重感染，并在免疫功能较弱者身上造成血液感染。
据报，进屋该不该脱鞋的问题，在不同地区和互联网上都引发分歧。美国女子索菲亚和塞尔维亚丈夫佩贾此前就在社交媒体上分享过两人对此的不同看法。索菲亚觉得进屋脱鞋很奇怪，佩贾则说，自己从小在家就是要脱鞋的。
皮特指出，鞋上发现的细菌可能在家中存活一段时间，甚至长达数天，并有机会感染家中其他人；地毯这类安全、温暖的环境尤其有利于细菌停留。她也提醒，世界各地还可能存在不同细菌、真菌和寄生虫，因此建议不论身处何地，进屋后都应尽快脱鞋。

7月1日晚，黄金、白银直线拉升。现货黄金重回4000美元\/盎司上方。截至19:35，报4023.53美元\/盎司。现货白银报58.84美元\/盎司，日内涨0.48%。值得注意的是，暗盘金价同步上涨0.48%，但暗盘白银快速跳水，涨幅从4%迅速收窄至0.2%。 受美伊谈判不确定性增加、美元走强、美联储加息预期等因素影响，国际黄金价格6月下跌超过11%。今年第二季度黄金价格下跌约14%，创下2013年来最大季度跌幅。 据中国黄金网报道，高盛全球大宗商品研究联合主管萨曼莎·达特表示，过去4个月国际金价的下跌并不意味着全年上涨行情已经结束，全球央行需求将继续推动贵金属价格回升至接近5000美元\/盎司。（21财经）
