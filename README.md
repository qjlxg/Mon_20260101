2026年3月26日，清迈第8选区国会议员帕他蓬·利拉帕特（Pattharaphong Leelapath）带领“ส้มสู้ไฟ”灭火队投入森林灭火，目标是遏制北部PM2.5浓度飙升至600微克/立方米以上的危机。 
帕他蓬表示，队伍自2月26日起持续作业。该队自去年起成立，已具备装备、体能与实战经验，旨在为未来（若执政时）更合理地分配防火经费与装备、确定人员编制与福利，并在火情发生前提供及时支援；同时采用技术手段提升火情研判与调度效率。 
今年迄今已执行38次灭火任务，合计扑灭火线约107公里，覆盖清迈府的Mae Wang、San Pa Tong、Hang Dong、Doi Saket、San Kamphaeng、Mae On、San Sai等地区，以及邻近兰普恩府（Lamphun）的Mueang和Mae Tha县；并开展150余次无人机侦察与定位，累计飞行逾30小时。 
帕他蓬强调，尽管人民党并非执政党，但他们仍与官员和民众并肩全力以赴，既着眼紧急灭火与降低PM2.5，也推动从源头和社区参与着手的长期解决方案。他最后质问：政府在做什么？
