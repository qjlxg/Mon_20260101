在被英国政府以国家安全为由阻挡于苏格兰设厂后，中国风电巨企明阳智慧能源集团股份有限公司（简称明阳智能）考虑在西班牙设厂制造风力涡轮机。
据彭博社报道，明阳智能欧洲区首席执行官何瑞修（Horatio Evers）受访时说，公司近几个月已在欧洲开展招聘活动，以逐渐在当地建立存在，以便在这个未来数十年快速成长的市场保有订单。在欧洲开展制造将是公司在当地保有订单的关键。
何瑞修说：“我们跟欧洲市场很多项目开发商交流，当地具备额外竞争的需求。本土化将是推动公司在欧洲市场销售涡轮机的关键因素。”
欧洲海上风电市场主要由丹麦的维斯塔斯风力系统公司和德国的西门子能源股份公司掌控。明阳智能的入场将加大该行业的竞争力度，并将可能拉低电力成本。
不过，明阳智能也将可能突破由欧洲企业垄断的绿色能源市场。中国企业已在电池和太阳能板等科技领域占据主导地位。
何瑞修说，明阳智能西班牙工厂将制造海上风力发电机的叶片，以及装有涡轮机主要机械零部件的吊舱。公司正为工厂选址，但尚未确定具体地点。
西班牙首相桑切斯此前赴北京访问时，与明阳智能董事长张传卫会面，意味着西班牙欢迎明阳智能到当地投资。
不过，西班牙不是海上风电场的主要市场，并拥有大幅土地开展低成本的陆上风电项目。 
这意味着明阳智能必须在德国、荷兰和丹麦等其他欧洲国家寻找订单。目前上不清楚这些国家的政府如何看待明阳智能进军欧洲市场。
去年，一个德国开发商选择西门子能源公司的产品，而放弃与明阳智能合作，令明阳智能失去在欧洲市场拿下首份重大订单的机会。
中国外交部发言人3月在回应英国政府拒绝让明阳智能在英国设厂时，敦促英方为中企提供一个“公平、公正及非歧视的营商环境”，并不应将中英经贸绿色合作，泛政治化、泛安全化。
中国商务部发言人4月14日回应相同提问时说，英方以“国家安全”为由排除中国产品参与其风电项目，有悖于英方长期以来秉持的“开放、自由”的市场理念，不利于英国地方经济发展和民众福祉的提升，将对双边经贸务实合作产生负面影响，中方对此坚决反对。

日本众议院通过设立“国家情报局”相关法案
日本首相高市早苗内阁推动的设立“国家情报局”相关法案在众议院全体会议上经表决通过，并将移交参议院进行表决。相关法案在日本社会引发广泛担忧。
根据星期四（4月23日）通过的法案，日本拟构建以“国家情报会议”为核心、“国家情报局”为执行机构的情报体系，负责统筹安全保障、反恐等领域的情报活动以及涉外国间谍的“对外情报活动”。“国家情报局”还被赋予要求政府各省厅向其提供情报的综合协调权。
新华社报道，日本媒体普遍认为，尽管由自民党和日本维新会组成的执政联盟在参议院的议席未过半，但由于部分在野党也对法案持赞成态度，预计在参议院也将获得通过，因此可能在本届国会内完成立法程序。
据共同社报道，高市早苗一直有意强化日本的情报机能。报道称，待设立“国家情报局”的法案正式生效后，高市还将全面启动“防止间谍法”的立法工作以及设立独立对外情报机构的专项研讨工作。
当前在日本国内，围绕法案可能引发的隐私权及言论自由受侵犯的担忧持续发酵。庆应义塾大学名誉教授金子胜23日在社交媒体发文说：“高市政府试图通过修宪和军事化掩盖日本即将到来的经济崩溃，现在又限制民众言论自由、压制社会声音，这一切都是在重蹈覆辙。” 
日本社民党党首福岛瑞穗22日接受媒体采访时指出，政府虽称不会随意调查“普通市民”，但“普通市民”与“非普通市民”的界限如何划定仍不明确，这令人非常担忧，法案本身也存在缺陷。
日本媒体近日也普遍指出，政府有必要充分说明强化情报职能的必要性，并正视其可能带来的严重影响。

英法签三年协议 合作遏制难民搭小船渡海峡
（伦敦/巴黎综合电）英国与法国签署一份新的三年协议，英国将提供最高6亿6000万英镑（约11亿3700万新元）资金，协助遏制英吉利海峡非法偷渡；法国则会加强执法，并投入更多人员打击偷渡。
根据协议，英国将提供约5亿英镑提升海岸巡逻与执法力量，包括在法国北部海岸部署近1100名执法、情报及军方人员，扩充情报与司法警力。
法国则计划到2029年将沿岸执法人员增加至约1400人，并强化情报与技术手段，减少偷渡出发次数。根据海上国际法，一旦船只离岸，执法人员主要限于救援，这也增加了拦截难度。
此外，法国还将派遣一支由50名警员组成的新防暴警察分队，同时把一支旨在追踪人口走私者的18人情报小组扩充至30人。
若阻止难民的成效显著，英国还将额外支付1亿6000万英镑。双方每年将进行评估，若未达到预期成果，资金将转用于其他应对措施。
英国与法国长期就难民问题合作，但英方持续批评法国沿岸拦截力度不足。
数据显示，2025年约有4万1000人乘坐小船横渡英吉利海峡抵达英国，接近2022年最高纪录。与此同时，横渡海峡的风险持续上升，2025年至少有29名移民在途中死亡。 
新协议取代了2023年签署的金额为4亿8000万英镑的旧协议。过去数月，双方围绕协议续签展开激烈谈判，英国坚持要求对资金使用设置更严格条件，最终才促成新协议。
英国政府说，自2024年7月工党政府上台以来，英法合作已阻止超过4万2000次偷渡尝试。
英国首相斯塔默指出，达致新协议意味着英法两国可进一步加强情报、监控和地面警力部署，以保护英国的边境。

领土争端下 中国将进一步加强海岛建设
（北京综合讯）尽管和多国的海上领土争端僵持不下，中国官方表态将进一步推进海岛建设，这是北京增强海上力量、获取自然资源和巩固领土主张的长期战略的一部分。
中共中央机关报《人民日报》星期三（4月22日）发表中共自然资源部党组的署名文章《统筹推进海岛高质量发展和高水平保护》。
文章称，中国拥有逾1万1000个海岛，它们是壮大海洋经济、拓展发展空间的重要依托，是捍卫国家权益、保障国防安全的战略前沿，亟待统筹推进海岛高质量发展和高水平保护。
文章提到，虽然中国海岛发展硬实力持续加强，“南（中国）海岛礁建设积极推进”，但当前全球海洋发展格局深刻调整，世界主要国家竞相将发展重心转向海洋，中国海岛开发保护面临新挑战。

据路透社报道，2018年的官方统计数据显示，中国绝大多数的海岛位于距大陆海岸线100公里的范围内，近60%位于东中国海，其余海岛则位于南中国海。
多年来，中国在南中国海争议水域进行大规模填海造陆工程，建造人工岛屿、飞机跑道和军事设施。
华盛顿智库战略与国际研究中心东南亚专家波林说：“中国人工岛屿基地上的设施，使中国的执法、海军和民兵船只，能全年无休地在最远可达中国海岸线1000海里外的邻国水域巡逻。”

不做“夹心饼干” 特鲁多：中等力量可成全球稳压器
加拿大、亚细安和欧洲等“中等力量”并非只能在大国竞争中被动周旋，而是能通过深化协作、共同维护规则，成为支撑国际秩序韧性的重要力量。
加拿大前总理特鲁多星期四（4月23日）上午在星耀樟宜出席美国消费者新闻与商业频道（CNBC）Converge Live大会时，向主持人德鲁里（Mandy Drury）指出，以布雷顿森林体系（Bretton Woods System）为核心的传统国际秩序已步入深度转型期。 
这套秩序虽曾见证辉煌，却已难以有效应对如今的新兴挑战。特鲁多指出，2014年俄罗斯吞并克里米亚，很可能被后世视为规则基础国际秩序出现转折的重要节点之一。
当时，俄罗斯以武力改变边界却几乎没有付出真正代价，还能“全身而退”，这不仅削弱了既有规则的约束力，也释放出危险信号：支撑世界稳定的基本规范，已不再是不可逾越的底线。　
面对保护主义与孤立主义的双重夹击，特鲁多倡导通过“小多边主义”建立更灵活的伙伴网络。这种合作不设价值观的高门槛，核心在于围绕具体议题达成双赢的共同利益，共同制定和遵守规则。
他指出：“如果把欧洲、亚细安、加拿大以及部分南美国家的力量加总起来，其经济实力足以与任何大国相匹敌。”加拿大也正通过《跨太平洋伙伴全面进展协定》（CPTPP）等机制，与伙伴共同维持开放贸易与规则合作。
谈及人工智能（AI）时，特鲁多提出明确警示，AI带来的问题不只是效率提升，更关系到社会稳定与制度可持续性。
“AI已经到来，且不可逆转。”特鲁多直言，与其纠结是否推进技术发展，不如思考如何确保AI所创造的财富不会过度集中。他说，如果AI带来的优势最终“只集中在少数人手中，我们可能正走向社会分裂，甚至崩塌”。
因此，他强调，在AI时代必须更加重视人的自主性。个人真正需要争取的，不只是技术适应能力，更包括对数据、隐私以及掌控自身生活的能力。
在这一逻辑下，技术的发展方向应是增强人与社区之间的连接，让人更有掌控感，而不是被平台与算法进一步削弱主动权。
谈话的最后，当被问及“什么最让你夜不能寐”时，特鲁多说，最令他担忧的是“越来越多人被说服自己已经不再重要”。
他说，一旦人们失去对未来的掌控感，以及改变世界的信念，社会将更难凝聚力量，应对气候变化与AI带来的深层转型挑战。

美参院五度否决止战案  但特朗普对伊用兵60天大限将至
（纽约综合讯）美国共和党在参议院连续第五次否决了限制总统特朗普对伊朗发动战争的法案。不过，特朗普面临一个星期后到来的关键宪法期限，他如果无法获国会批准持续作战，或许必须停战。
由民主党人发起的止战议案星期三（4月22日）在参议院以46票赞成、51票反对，未获通过。
参议院民主党领袖舒默在投票前发表讲话说：“民主党人将继续每周强行推动对战争权力决议进行投票，直到共和党人将美国人民的利益置于特朗普之上，结束这场战争。”
美国1973年生效的《战争权力法》规定，总统如果未获国会批准而动用武装力量，须在60天内终止行动。这意味着对伊战争的关键期限就在5月1日，即从特朗普3月2日正式通知国会开战算起的第60天。
若特朗普到时不获国会批准继续作战，就必须从前线撤军。但根据宪法，特朗普之后还有30天撤军，理据是需要时间让军队安全撤出，但军队无权继续发动攻击。
美国国会自2002年授权对伊拉克动武之后，就未再投票支持开战。虽然大多数共和党人一致阻挡民主党法案通过，但要明确授权特朗普继续打仗，不见得能够保持团结。
犹他州共和党参议员柯蒂斯本月中旬发文表明：“不会支持未经国会批准而持续超过60天的军事行动。”
担任众议院外交事务委员会主席的佛罗里达州共和党众议员马斯特警告，如果冲突持续到5月，特朗普可能失去大批支持。
上周，共和党人在众议院勉强挡住民主党的止战动议后，马斯特就表明，60天期限之后的表决结果“可能有所不同”。
不过，《纽约时报》指出，特朗普也可能无视这个期限。美国历届政府一再辩称，宪法对总统兼任三军总司令赋予广泛的权力，而《战争权力法》对总统的限制是违宪的。
2011年，美国在利比亚的军事行动持续超过60天，时任总统奥巴马给出的理由是：“美国的行动不涉及与敌对势力持续战斗或激烈交火，也不涉及美国的地面部队。”奥巴马来自民主党。
一些美国议员推测，特朗普政府可能以类似论点辩解，继续对伊朗展开军事行动。

曼谷讯 — 联合国开发计划署署长亚历山大·德克鲁（Alexander De Croo）4月23日警告，伊朗战争引发的燃料和化肥供应中断，正值农户播种期，将使逾3000万人重返贫困。  
德克鲁对路透表示，霍尔木兹海峡对货轮的封堵加剧了化肥短缺，已降低农业生产力，并可能在2026年后期影响粮食产量。  
他还指出，能源短缺和侨汇下滑等连锁效应已令全球国内生产总值损失约0.5%至0.8%。“那些需要几十年才能建立的东西，八周的战争就能摧毁。”  
世界银行、国际货币基金组织和世界粮食计划署此前亦警告，战争将推高粮价，加重最脆弱人群负担。随着资助减少和需求上升，苏丹、加沙与乌克兰等地的人道救援面临压力，德克鲁坦言部分求助者可能被迫得不到援助，依赖援助者将更加脆弱。  
（路透）

4月23日——俄罗斯控制的南乌地区一名女子因购买支持乌克兰军队的“战争债券”被以叛国罪判处14年监禁，扎波罗热区法院周四公布了判决。
法院称，该女子通过乌方一款手机应用购买了价值270,080卢布（约3600美元）的债券，随后被俄联邦安全局（FSB）发现并抓获。法院仅以首字母B称呼被告，并称其为俄罗斯公民。被占领地区居民若要继续获得医疗、养老金和财产权，往往被迫接受俄罗斯国籍。 
人权组织“纪念会”（Memorial，俄罗斯本月被列为极端组织）披露，该女子为66岁的医生拉丽萨·别利亚耶娃，来自柳比莫夫卡村。乌克兰自2022年2月起发行战争债券用于国防筹资，相关债券被宣传为爱国且免税，赫夫里纳计价的年化收益可达17.45%，美元计价债券超过4%。

意大利政府表示，因经济与公共财政压力加剧及能源价格飙升，原定的国防开支增长计划可能无法如期推进。 
政府周四发布的财政多年度计划（DFP）指出，受中东局势与能源冲击影响，内阁已下调增长预期并上调赤字与债务预测，财政回旋空间有限，需优先援助家庭和企业应对能源冲击。财政部长詹卡洛·焦尔杰蒂在DFP引言中称，“因此必须重新界定优先次序，并重新安排包括国防在内的计划性支出增长”。 
意大利曾响应美国总统特朗普呼吁，承诺到2035年将防务与安全开支提高到占GDP的5%；近期预算则承诺到2028年增加约0.5个百分点（约120亿欧元），遭反对党质疑应优先用于公共服务。 
DFP并指出，意大利高度依赖进口能源——天然气占其能源供应约38%——对伊朗冲突等扰动尤为脆弱。欧盟允许在特殊情况下突破赤字上限，政府可能启用所谓“国家豁免条款”，但DFP表明这一条款更可能用于应对能源危机而非扩大国防支出。最坏情形下，经济在2027年可能萎缩0.2%。
（来源：路透）
