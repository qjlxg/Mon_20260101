中国加强加密货币监管，禁止在境内开展现实世界资产（RWA）代币化相关活动
中国人民银行牵头八部门周五发布通知，收紧对虚拟货币交易的监管，并将监管扩展至RWA代币化。通知规定，境内的RWA代币化活动及为证券发行、金融运作或融资提供的中介或技术服务一律禁止，特殊情况可经主管部门审批豁免。
通知强调，境外主体不得以任何形式为境内机构非法提供RWA代币化服务；境内机构或受其控制的境外实体，未经有关部门批准不得在境外发行虚拟货币。对基于境内权益在境外开展的代币化或准资产证券化活动，将按“同一业务、同一风险、同一规则”原则严监管，未经批准不得开展。
此外，涉事中国金融机构境外子公司及相关中介、技术服务方需加强合规、风险管控，落实适当性与反洗钱措施，并向监管机构报告或申请批准。通知还明确，未经批准不得发行与人民币挂钩的离岸稳定币。
