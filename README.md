中国拟实行统一税务行政处罚裁量基准
中国拟在全国范围内实行统一的税务行政处罚裁量基准。
据中新社报道，中国国家税务总局研究制定《税务行政处罚裁量基准（2026年版）》，星期三（7月15日）起向社会公开征求意见。
中国国家税务总局有关司局负责人介绍，近年来，东北、西南、西北、中南、华北、华东等区域先后发布区域内统一的税务行政处罚裁量基准，在规范税收执法、强化税款征收等方面发挥了积极作用。但也存在体例结构不统一、科学合理性不足、裁量因素不一、指引作用发挥不足等问题。
负责人表示，制定税务行政处罚裁量基准，统一全国税务行政处罚尺度，有利于压缩税务行政执法自由裁量空间，防止同案不同罚，平等维护各类经营主体合法权益。为了更好服务全国统一大市场建设，国家税务总局研究制定《税务行政处罚裁量基准（2026年版）》，在全国范围内实行统一的税务行政处罚裁量基准。
该负责人介绍，此次制定的裁量基准有三个特点：统一体例结构，逐项列明法定依据、裁量阶次、适用条件和具体标准，一目了然，便于执行；统一基准内容，遵循立法本意，“一把尺子量到底”，实现对违法行为的科学评价；防止“一刀切”。充分运用“定额＋幅度”方式设置处罚标准，为个案处理留有合理的裁量空间，防止因简单化“一刀切”导致执法不公。

粮农组织：健康饮食成本飙升 全球三分之一人口无力负担
联合国周三发布的数据显示，过去五年间，健康饮食的成本上涨了25%，导致全球三分之一的人口无力负担。
根据联合国粮食及农业组织星期三（7月15日）发布的《2026年世界粮食安全和营养状况》报告，食品成本的上涨意味着，人均每日维持健康饮食所需的费用（按购买力平价计算）已达到4.28美元（约5.51新元）。
粮农组织首席经济师库伦在纽约联合国总部举行的新闻发布会上说：“结果是，仍有26亿9000万人，相当于全球近三分之一的人口无法负担健康饮食。”
这份报告的完整版将于7月21日在粮农组织位于罗马的总部发布。
根据报告，各项成本的构成并不均衡：例如，豆类、谷物等主食仅占健康饮食成本的13%，而动物性食品占近30%，果蔬类则占16%。
库伦说：“因此，挑战不在于生产足够的热量，而在于让营养丰富的食品变得更可负担。”
库伦认为，政府应调整补贴方向，将资金更多地投向营养丰富的食品，而非大米和小麦等谷物。他也建议，投资本国基础设施和物流将显著降低成本，因为“健康饮食成本中70%至75%的部分是在食品离开农场后产生的”。
库伦指出，全球健康饮食成本最高的地区是拉丁美洲，特别是加勒比地区。这可能是因为该地区侧重于食品出口，而未能生产足够的食品以降低本地价格。
