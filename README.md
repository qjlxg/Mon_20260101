广西养蛇场被洪水冲垮  毒蛇出逃咬伤村民
中国广西发生洪水，有养蛇场被冲垮，大量包括毒蛇在内的蛇类出逃，咬伤村民。
综合红星新闻、《潇湘晨报》《新京报》等报道，有网友星期二（7月7日）发布截图称，横州市云表镇邓圩村一养蛇场被洪水冲毁，大量蛇类逃逸，提醒周边群众注意防范。
当地居民称，云表镇低洼地带和山坡上均有养蛇场。部分小型蛇场的蛇类已逃至周边区域，部分村民确实被蛇咬伤，因处于被困状态无法及时就医。
网络上陆续传出“养蛇场被洪水冲垮，蛇流入下游村庄伤人”“出逃蛇类多为毒蛇，有受灾村民被咬伤”等消息，同时有现场视频显示，当地村民手持竹竿开展驱蛇作业。
养殖户雷先生介绍，当地主要养殖眼镜蛇、王锦蛇和水律蛇三类，其中眼镜蛇具有毒性。雷先生推测低洼地带零散养殖户的蛇类可能已大量逃走，但当地养殖品种蛇生活场所多在山林，长时间泡水后大多会死亡。
“发生在7月6日上午，我们初步统计，从被冲垮养殖场出逃的蛇类约800条至900条。目前有一名村民被蛇咬伤，正在医院接受抢救。”7月7日中午，横州市云表镇邓圩村委负责人吴志告诉红星新闻记者，“出逃蛇类并非全部有毒，其中占比较大的是无毒水律蛇。”
吴志介绍，险情发生后，周边未受灾村屯的十余名村民，自发组建民间捕蛇队，深入受灾片区进村入户捕蛇，防范蛇类持续伤人。“队员主要借助捕鱼工具、电鱼设备开展捕蛇作业。”吴志补充道，“绝大多数蛇群已被洪水冲走，目前仅有少量蛇盘踞在洪水滞留的水面漂浮垃圾杂物上，现场捕获的蛇类，大多是无毒的水律蛇。”
横州市人民医院称，院内血清储备充足。
养蛇是横州及周边地区的一项产业。高峰时期，广西蛇类存栏量近2000万条，占全国总量七成。
受台风“美莎克”影响，自上星期六（7月4日）以来，广西多地持续出现强降雨，导致河流、水库水位迅速上涨，两座水库决堤，东斑江流域出现超过50年一遇的洪水，引发山洪暴发。防城港、贵港、南宁等地出现严重城市内涝、停水停电等情况，南宁横州南康村近1700人被困。
