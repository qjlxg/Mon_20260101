世卫：立百病毒从印度扩散风险低 不建议旅行贸易限制
（海得拉巴路透电）世界卫生组织称，高致命率的立百病毒从印度扩散的风险较低，因此不建议各国对印度采取旅行或贸易限制措施。 
印度政府日前披露境内发现两起立百病毒确诊病例后，香港、马来西亚、新加坡、泰国和越南等亚洲地区本周加强机场筛查措施，以防范病毒扩散。
世卫组织星期五（1月30日）在发给路透社的电邮中说，印度的两起确诊病例引发进一步传播的风险较低，而且印度当局具备遏制这类疫情的能力。此外，目前尚无证据显示人际传播有所增加，世卫组织已与印度卫生当局展开协调。
据了解，印度，尤其是在南部的喀拉拉邦，经常报告零星的立百病毒感染病例。这个地区被视为全球立百病毒风险最高的地区之一，自2018年首次出现疫情以来，已导致数十人死亡。
鉴于立百病毒在印度部分地区及邻国孟加拉国的蝙蝠种群中传播，世卫组织认为仍存在进一步感染立百病毒的潜在风险。
立百病毒属于副粘病毒科亨立百病毒属，是以核糖核酸（RNA）为遗传物质的病毒，可通过受感染的蝙蝠，或被蝙蝠污染的水果传播给人类，但人际传播并不容易，通常要与感染者长时间接触，感染风险才会明显增加。
世卫组织等机构介绍，立百病毒病从感染到出现症状的潜伏期一般为4到14天，最长可达45天。感染者死亡率则因各地疫情监测和治疗能力不同，介于40%至75%之间。
目前尚无针对立百病毒病的特效药和疫苗。为预防感染，应在食用新鲜水果前彻底清洗并削皮；处理患病动物及组织时，务必穿戴防护服和手套，避免无防护地密切接触感染者。

缅北徐发启犯罪集团案 徐发启判死缓 张龙龙判无期
中国官方通报，缅北徐发启犯罪集团案一审宣判，五名被告中，徐发启被判死缓，张龙龙被判无期，其他三人也被判刑。
重庆市第五中级法院星期五（1月30日）在微信公众号上通报，法院当天一审公开宣判徐发启犯罪集团案。对被告徐发启判处死刑，缓期二年执行；对被告张龙龙判处无期徒刑；对被告罗玉军、习康强、刘世兵判处25年至14年不等有期徒刑，并相应判处没收财产、罚金等附加刑。
法院审理查明，2019年以来，以徐发启为首要分子、罗玉军等武装头目为重要成员的犯罪集团，利用徐发启在缅甸果敢地区的影响力，与电诈犯罪集团合作，陆续修建开设多个诈骗园区实施电信网络诈骗，涉诈资金11余亿元人民币（2亿零100万新元），并在武装庇护过程中，伙同电诈犯罪集团故意伤害不服从管理的涉诈人员，致一人死亡、一人轻伤。徐发启还结伙实施了走私、贩卖毒品等犯罪。
法院认为，徐发启、张龙龙、罗玉军、习康强、刘世兵的行为分别构成诈骗罪、故意伤害罪、走私、贩卖毒品罪等六项罪名。法庭根据各被告人犯罪的事实、性质、情节和对社会的危害程度，依法作出上述判决。
现年54岁的徐发启又名徐老发，他和白所成、白应苍等六人都是缅北果敢电诈犯罪集团重要头目，2024年1月由缅甸警方移交给中方。
重庆市第五中级法院去年9月通报，审理徐发启等五名被告诈骗、故意伤害、非法拘禁、敲诈勒索、走私、贩卖毒品、偷越国（边）境案。庭审结束后，法庭宣布将择期宣判。
浙江省温州市中级法院星期四（29日）在微信公众号上通报，涉及缅北明家电诈集团案的11名罪犯被执行死刑。

参议院议长维森特“蒂托”·索托三世表示，鉴于最高法院裁定2025年对副总统莎拉·杜特尔特的弹劾案违宪，他愿意支持修宪（cha‑cha）以澄清弹劾程序规则。索托称，最高法院此次在弹劾程序上介入立法权，属于司法越权，裁决对宪法做出实质性“修正”。他表示：“我不会主导，但会支持……现在我支持修宪。”
最高法院认定，2025年2月移交参议院的第四起弹劾案违反宪法第XI条第3款第5项关于“一年内不得重复对同一官员启动弹劾”的规定，并裁定弹劾启动包括：经适当验讫并移交司法委员会、未在10个会期日内列入众议院议程或在列入后未在3个会期日内移交司法委员会，或在众议院休会（adjourn sine die）前将弹劾条款送交参议院。法院指出，弹劾须在现届国会任期内启动。
索托质疑裁决中若干事实认定（例如休会日期），并警告此裁定可能被滥用：任何人提交的“垃圾弹劾”均可用来阻止后续弹劾程序。他计划在二月上旬促成参众两院会谈，讨论应对之策。众议院副议长罗纳尔多·普诺欢迎参院配合，国民团结党曾提交提案建议召开制宪会议修宪。
同时，参议院临时议长潘菲洛·拉克森强调，尽管可有不同意见，最高法院的裁决仍应受到尊重，法院为宪法最终解释者。最高法院已驳回众议院的复议动议，维持弹劾案违宪的结论。

【墨西哥第四季度GDP超预期反弹 农业强劲表现成“救命稻草”】金十数据1月30日讯，墨西哥经济在去年年底反弹，成功避开技术性衰退。此前，由于受美国关税拖累及投资低迷影响，墨西哥经济在第三季度曾出现萎缩。初步数据显示，墨西哥10月至12月期间的国内生产总值环比增长0.8%，高于调查经济学家得出的0.6%的预期中值，而上一季度的读数为收缩0.3%。从2025年全年来看，墨西哥GDP增长了0.5%，这标志着该国经济增长连续第四年放缓。经济增长主要受到农业领域的强劲表现驱动，该部门较去年同期增长了5.3%；工业与制造业增长了0.4%，服务业则增长了2%。总统辛鲍姆总统近几周会见了私人经济学家和银行家，旨在寻找解决墨西哥投资不足的方案，并在经济形势复杂的年份探索加速增长的路径。
