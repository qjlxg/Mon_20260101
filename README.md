美对伊朗发动第六晚空袭 局势凸显美国掌控海峡局限
伊朗和美国交火升级，持续一周的空袭行动已基本使上月达成的停战协议化为乌有，而不断升级的局势凸显美国掌控霍尔木兹海峡方面面对局限。 
路透社报道，美国中央司令部说，星期四（7月16日）对伊朗发动了连续第六晚的空袭，目标是“进一步削弱伊朗的军事能力”。

伊朗则以导弹和无人机对邻国境内的美军基地进行反击，其中包括位于约旦、近期扩建的一处美国空军基地。德黑兰方面称，这个美军基地15日晚袭击了伊朗一家儿童癌症医院。

伊朗多家新闻媒体报道，美国16日晚袭击了伊朗沿海城市哈米尔港（Bandar Khamir）的三座桥梁和火车站，并向伊朗东南部的伊兰沙赫尔机场（Iranshahr Airport）发射了导弹。
路透社未能立即核实这些报道。
在袭击持续升级之际，特朗普15日仍对一名被伊朗拘留的美国公民获释表示欢迎，称这是德黑兰展现善意。这名美国公民为人权律师卡拉里（Dena Karari）。
然而伊朗国家媒体报道，伊朗司法部门16日驳斥特朗普的说法，称没有美国囚犯从伊朗监狱释放或进行交换。
另据彭博社报道，特朗普政府在控制霍尔木兹海峡局势方面面临局限。战争爆发五个月后，美国在霍尔木兹海峡这一全球关键的咽喉通道，面对一个态度顽强的对手，美国军事力量的极限日益凸显。
报道指伊朗的快艇、导弹、无人机、水雷依然有能力在霍尔木兹海峡大肆破坏，顶住了来自世界上最强大海军的每日打击。

路透社报道，特朗普定美国东部时间星期四晚上9时（新加坡时间星期五上午9时）发表全国讲话。
16日下午较迟时候，美国广播公司（ABC）新闻发言人说，ABC电视网将在流媒体平台ABC News Live和ABC新闻广播电台（ABC News Radio）转播特朗普的演讲，不会在ABC主电视频道播出。

2026年世界杯半决赛全部结束，决赛对阵正式出炉。
卫冕冠军阿根廷与欧洲劲旅西班牙会师决赛，英格兰与法国则将争夺季军。
北京时间7月15日，世界杯半决赛迎来焦点大战，阿根廷与英格兰展开强强对话。比赛中英格兰率先打破僵局，第55分钟取得领先，但阿根廷随后展开疯狂反扑。
第85分钟，恩佐·费尔南德斯帮助球队扳平比分，补时阶段劳塔罗·马丁内斯完成绝杀，最终阿根廷2比1逆转击败英格兰，连续闯入世界杯决赛。
另一场半决赛中，西班牙展现强大控制力，以2比0击败法国，成功拿到决赛门票。
西班牙凭借稳定的整体发挥，一路淘汰强敌，成为本届世界杯最大热门之一。
随着半决赛落幕，本届世界杯最后两场比赛确定：

一边是卫冕冠军追求连冠荣耀，一边是欧洲新王冲击世界之巅。世界杯最后的答案，即将在决赛夜揭晓。
