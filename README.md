中国官方公布数据显示，去年免签入境中国的外国人人次增长49.5%。
中国国家统计局星期六（2月28日）发布2025年国民经济和社会发展统计公报。在文化旅游方面，公报数据显示，2025年中国国内出游65.2亿人次，比上年增长16.2%。其中，城镇居民国内出游50亿人次，增长14.3%；农村居民国内出游15.3亿人次，增长22.6%。
中国国内游客出游总花费6.3万亿元（人民币，下同，约1.16万亿新元），增长9.5%。其中，城镇居民出游花费5.3万亿元，增长7.5%；农村居民出游花费1万亿元，增长21.4%。
入境游客15450万人次，增长17.1%，其中外国人3517万人次，香港、澳门和台湾游客11932万人次。入境游客总花费1311亿美元（约1658亿新元），增长39.2%。
公报也提到，通过免签入境的外国人3008万人次，增长49.5%。中国内地居民出境16792万人次，其中因私出境16166万人次，赴港澳台出境11271万人次。
