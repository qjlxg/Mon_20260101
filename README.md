据在线旅游平台数据显示，2026年春节假期，前往北京、上海、成都、广州等多条“反向过年”航线火热，机票预订量倍增。一些从中小机场飞往大都市的“反向过年”小众航线热度，机票预订量同比增长更显著。

中国国家发展改革委副主任李春临在今年1月29日的发布会上也提到，父母到子女家过年、“先回家再出游”等成为春运新趋势。

中央军委是什么？在反腐风暴中如何运作？
中央军委是领导人民解放军的最高决策与指挥机构，承担类似其他国家国防部的职能，负责战略、建设、装备、人员与经费等重大事项。名义上的国防部主要用于对外外交和公关，国防部长通常但并非必然是军委成员。
目前中央军委正面临领导层重组。习近平已连任中央军委主席进入第三任期，但原本任期至2027年的委员中，除习近平外仅有一人仍在职，其余五人均已不再活动。上月，中央军委第一副主席张又侠和中央军委联合参谋部参谋长刘振立因涉嫌腐败被调查。
中央军委由15个部门组成（包括联合参谋部、后勤保障部等），并领导解放军五大战区司令部、各军种及若干科研院所与院校。其最高决策层为一个七人委员会，由中共全国党代会每五年选举产生。
