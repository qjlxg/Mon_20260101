经济学家黄益平：勿对中国强力刺激消费政策期望过高
中国知名经济学家黄益平表示，尽管官方制定较温和的增长目标，以表明将更重视增长动力的再平衡，但中国经济向消费支出的转型仍需耗费不少时日。
中国人民银行货币政策委员会委员黄益平星期五（3月6日）告诉彭博电视台，投资人应下调对强力刺激政策的预期。

黄益平指出，尽管已从2010年的低谷中反弹，但中国的消费占比仍然过低。他表示，政府应着重提升居民收入和信心。
