马尼拉讯 — 国家灌溉管理局（NIA）局长爱德华多·吉伦表示，总统费迪南德·马科斯（Jr.）将于下周重返比科尔地区，视察卡马里内斯苏尔省的太阳能抽水灌溉项目。
吉伦称，马科斯将前往MASSBA太阳能灌溉项目（服务面积约1,200公顷）及位于布拉（Bula）的SARFC合作社项目（服务约600公顷）。政府正推动更多太阳能抽水系统，作为减少农业对燃油依赖、缓解国际油价波动影响的中期对策。
吉伦并指出，长期目标是在全国主要流域建坝以替代对燃料的依赖。他以伊洛伊洛的Jalaur水坝为例，造价约P10 billion；若以此估算，在18个主要流域建设水坝约需P180 billion，仍低于年度防洪预算。
目前全国约有3,000套太阳能抽水灌溉设备。吉伦还表示，虽然燃油补贴由农业部提供给受影响农户，NIA一直为疏浚渠道和维修灌溉系统所用的大型设备提供燃油补贴。

马尼拉 — 农业部长Francisco Tiu‑Laurel周三表示，中国已向菲律宾保证，尽管国际油价持续攀升，北京仍会维持对菲化肥出口。中国驻菲大使景泉（Jing Quan）否认了有关中国将停止出口化肥的报道，称问题在于价格而非供应。
Tiu‑Laurel称，菲律宾已采购今年84%的化肥需求，国内储备可支撑到5月，订单持续，但担心交付时程可能延误。政府正与印度、俄罗斯、白俄罗斯等产肥国磋商，并研究液体化肥、纳米技术等替代方案，拟派团赴华考察。报道指出，受中东空袭及霍尔木兹海峡受阻影响，化肥价已从每吨400美元翻倍至800美元。
在国会，众议院多数党领袖Sandro Marcos提交第905号决议，敦促银行、汇款中心及金融机构在中东冲突期间暂时免除或减少汇款与交易手续费，理由是四天工作制或居家办公将放大对无现金交易的依赖；目前跨行ATM取款费为18比索，在线交易费可达25比索。
众议院已通过将参议院第1982号法案作为第8418号法案的修正案，授予总统对燃油消费税行使紧急权力，该法案将送交总统费迪南德·马科斯（Ferdinand Marcos Jr.）签署。Sandro Marcos表示，希望尽快落实以减轻家庭与企业负担。
BH党团议员Robert Nazal呼吁采取紧急救济措施，包括暂免港务局（PPA）港口费及海事局（MARINA）吨税，指油价上涨使航运业承压并出现“定价扭曲”，散装船用燃油价格甚至高于零售油价，部分船公司已减少航次。他并提议召开国会调查，传唤能源部、MARINA与PPA，建立燃油价格指数组成自动票价調整機制并检视疑似价格扭曲。

马尼拉—菲律宾中央银行（BSP）周三表示，正在密切关注中东冲突升级，称这可能在4月23日政策会议前，对国内通胀与金融稳定构成风险。 
央行重申，维持物价稳定为首要任务，目前正评估油价飙升是否会通过化肥、运输费及一般消费品价格产生二次冲击。 
BSP同时关注冲突对经常账户的影响，尤其是贸易往来及派驻该地区的海外菲律宾劳工（OFW）汇款流动。 
针对近期汇率波动，央行重申采取弹性汇率制度，仅在出现“过度波动”时入市干预以维护市场秩序，而非守住比索的具体水平。彭博报道指，BSP本周早些时候在比索接近1美元兑60比索时曾出手干预——总统马科斯曾表示不希望比索突破该价位。 
央行表示“政策将以数据为依据”，暗示利率决定将取决于地缘政治对菲律宾经济的实际冲击。回顾货币政策：央行在2月将基准利率下调25个基点至4.25%，去年曾多次降息，但在2023年10月也曾一次性上调25个基点以抑制上升通胀与食品价格。行长伊莱·雷莫洛纳年初曾表示宽松周期接近尾声，而今年通胀在中东冲突推高油价前已呈回升迹象。 
此外，美联储在美以—伊紧张局势冲击油市与供应链、经济数据走弱之际，预计将维持利率不变。美联储为期两天的会议周二开幕，基准利率决定将于周三美东时间下午2点（菲律宾时间次日凌晨2点）公布。

马尼拉消息——大都会马尼拉发展局（MMDA）周一表示，正研究将政府部门四天工作制按周一与周五分开实施的可行性。MMDA主席罗曼多·阿尔特斯（Romando Artes）称，拟议方案为一半机关周一休假，另一半周五休假，周二至周四为各部门与地方政府通用的办公日，以免民众在多地办理事务时遇到混乱。
阿尔特斯指出，此方案为MMDA建议，旨取代選舉律師罗穆洛·马卡林塔尔（Romulo Macalintal）提出的将休假日分散到各不同工作日的建议，后者可能导致“混乱”。他表示，相关提案将提交下次大都会理事会与首都圈各市长讨论。
就交通影响，阿尔特斯认为此举可缓解周一车流量。他说，3月16日周一早高峰期间，沿EDSA和C5的车流明显较为顺畅，但MMDA仍以监控中心的影像为初步依据，待一周内数据出齐后才能更完整评估燃油价格上涨与四天工作制对都会交通的具体影响。
