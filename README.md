彭博：中国国有炼油企业寻求恢复燃料出口
在一度暂停出口后，中国国有炼油企业正申请政府许可，计划最快在今年5月恢复燃料出口，理由是国内库存充裕。
彭博社引述知情人士报道，包括中国石化和中国石油在内的多家大型国有油企，自上周起已向监管部门申请放行汽油和柴油出口。
截至目前，中国政府尚未就上述申请调整出口配额。

不过数周后，情况出现变化。知情人士称，中国国内燃料价格上升导致需求意外走弱，部分原因是电动车使用增加，从而缓解了供应压力；与此同时，出口利润空间则显著扩大。
根据Mysteel OilChem数据，中国国有炼厂的汽油和柴油库存，分别升至自2025年和2024年以来的最高水平。
作为全球最大原油进口国，中国多年来持续强化能源安全战略，包括拓展多元供应来源、加大国内产能投资以及建立储备体系，使它在此次历史性供应冲击中较周边经济体展现出更强韧性。此外，中国也在中东战事之处迅速收紧出口，优先保障国内市场供应，稳定本国消费者和工业用能需求。

参院拨款委员会主席、南卡罗来纳州共和党人林赛·格雷厄姆与两名共和党同僚起草了一项法案，拟主要以纳税人资金为已在进行的建设买单。格雷厄姆表示，希望尽快表决，以为总统及宾客提供“安全设施”，舞厅下方将设大量军事设施，包括特勤局附属空间；私人捐款可用于购置餐具等非安全开支。
该项目估计花费4亿美元（约5.09亿新元），其中3.32亿美元拟由纳税人通过进口关税筹措。格雷厄姆称，他从未感受过如今的威胁，并认为此举可避免总统为大型活动离开受控的白宫范围。4月25日晚的那场晚宴并非由白宫主办，约有2600名宾客，晚宴因有人在会场酒店被控制并疑似向一名美特勤局特工开枪而中止。枪击发生后，特朗普在白宫记者会上称，安全理由更应继续建造舞厅。
特朗普已拆除白宫东翼以启动这座可容约1000人的舞厅工程，曾表示愿以私人捐款支付约4亿美元。格雷厄姆希望就独立的舞厅拨款法案表决，若失败则考虑将资金并入一项可用简单多数（51票）通过的参院程序中的拨款草案；共和党在参院占53席。该草案主要用于为两家国土安全部执法机构提供经费至2029财政年。3月31日，联邦法官裁定这项9万平方英尺的工程仅在获得国会批准后方可继续推进。4月26日，新任参议员蒂姆·希伊表示将寻求参院快速通过相关立法，但此举需参院全体同意，往往难以在有争议的议题上实现。目前宾夕法尼亚参议员约翰·费特曼是参院中唯一公开支持该舞厅的民主党人；一些众院共和党人也在起草确保工程完成的法案。

华盛顿，4月27日——两名共和党众议员呼吁特朗普政府授权国民警卫队应对世界杯期间可能的无人机威胁，并在联邦层面形成统一安保态势。来自众议院国土安全委员会的迈克尔·麦考尔（Michael McCaul）和伊莱贾·克兰（Elijah Crane）周五致信国土安全部长马克韦恩·穆林（Markwayne Mullin）、代理司法部长托德·布兰奇（Todd Blanche）和五角大楼负责人皮特·海格塞斯（Pete Hegseth）。
信中要求立即开展跨部门行动，保障今夏在11个美国主办城市上空的空域安全。两人指出，人员短缺以及主办方与城市间复杂的管辖分歧可能导致安保碎片化，因而需要统一的联邦安保部署。信中还称，国民警卫队凭借快速部署、全国可扩展性及处置国内突发事件的经验，能够在反无人机（C-UAS）和安全协调方面有效协助联邦与州当局。
国土安全部、司法部和五角大楼尚未就此置评。该呼吁发出前一天，华盛顿白宫记者晚宴发生枪击事件，引发对今夏世界杯安保的新担忧。白宫周一表示，赛事将会安全，发言人戴维斯·英格（Davis Ingle）称总统正致力于确保比赛既精彩又“史上最安全”。 路透社

耶路撒冷消息——以色列参谋长埃亚尔·扎米尔中将4月27日对部队发出严厉警告，抨击掠夺行为并警告在社交媒体上传播“有问题”信息已越过“红线”。
扎米尔在一份声明中称：“若存在掠夺现象，这是可耻的，可能玷污整个以军。若发生此类事件，我们将进行调查。”
此番表态是在以色列偏左的《哈雷茨》报援引前线官兵和指挥官证词报道，称在黎巴嫩南部有部队大量掠夺民用财物之后发出的。社交媒体上也出现士兵在疑似黎巴嫩南部民宅内破坏财物、相互拍摄打闹的画面；另有一张照片显示一名士兵用大锤砸碎掉落在十字架上的耶稣像。
以军上周表示，因相关事件已有两名士兵被处以30天军事拘留并取消战斗任务。扎米尔强调，征兵及预备役人员不得将社交网络作为传播争议信息或自我炒作的工具，越界者将面临纪律处分，并警告称类似行为的常态化与作战威胁同样危险。
军方在给法新社的声明中称，对任何损害平民财产和掠夺行为“予以最严肃对待并严格禁止”，对指控将“彻底审查”，一经查实将采取包括起诉在内的纪律和刑事措施；军警也对部队退出现役线经由的北部边境通道进行了检查，但未说明是否发现掠夺证据。
以色列维权组织“打破沉默”表示，虽未在黎巴嫩收集证词，但自2023年对加沙发动地面入侵以来，掠夺等行为已“变得十分普遍”。
（法新社）
