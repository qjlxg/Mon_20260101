评估美高院裁决关税违宪 中国商务部：反对单边加征关税
美国联邦最高法院上星期五（2月20日）裁定美国特朗普政府的关税政策违宪，中国商务部对此回应时称，正全面评估相关裁决，但强调中国一向反对单边加征关税，并敦促美国取消相关措施。
中国商务部发言人星期一（23日）在官网就美国最高法院公布关税诉讼案裁决结果答记者问时说，中国正对相关内容和影响进行全面评估。
发言人指出，中国一贯反对各种形式的单边加征关税措施，反复强调贸易战没有赢家，保护主义没有出路。美国对等关税、芬太尼关税等单边措施既违反国际经贸规则，也违反美国内法，不符合各方利益。
发言人说，事实反复证明，中美双方合则两利，斗则俱伤。中国敦促美国取消对贸易伙伴加征的单边关税措施。
发言人称，中国也注意到美国正准备采取贸易调查等替代措施，以期维持对贸易伙伴加征的关税，北京将对此保持密切关注并坚定维护中方利益。
美国联邦最高法院星期五裁定大规模对等关税政策违宪后，总统特朗普当晚签署行政令，向全球贸易伙伴征收10%关税取而代之，隔天（21日）再宣布将税率上调至法律允许的15%上限。
分析认为，特朗普政府接下来寻求通过其他法律依据继续征收关税，恐怕会造成更多不确定性，让商业环境更不明朗，贸易伙伴与华府的谈判也可能更混乱。
最新宣布的15%税率，对各国的直接冲击喜忧参半。英国、澳大利亚等国，也包括新加坡，新税率实际上高于此前谈成的10％基准关税税率。对于此前面临美国高额关税的中国（20％）、越南（20％）、印度（18％）和巴西（50％）等国家而言，新税率则显著较低。

美高院关税裁决 中国印度等国成赢家
随着美国联邦最高法院上周裁定美国特朗普政府的关税政策违宪，那些曾遭关税冲击最严重的国家，包括中国、印度等国，如今反而成为最大的受益者。
据彭博社星期一（2月23日）报道，美国最高法院上星期五（20日）裁定特朗普援引1977年《国际紧急经济权力法》（IEEPA）加征关税逾越总统权限，构成违宪后，中国、印度和巴西等国家对美出口商品的关税税率随之下降。
特朗普之后在高院裁决后宣布10％新关税；不到24小时后，他在上星期六（21日）又宣布将税率上调至法律允许的15％上限，并且立即生效。但彭博经济研究测算，这将使平均实际关税税率约为12%，为4月推出“解放日”关税以来的最低水平。
摩根士丹利经济学家称，对亚洲而言，加权平均关税税率预计将从20%降至17%，其中针对中国商品的平均关税税率将从32%降至24%。不过，这一缓解可能只是短暂现象，因为特朗普政府正寻求通过对特定行业及特定经济体加征关税，重新构建其关税体系。
不过，由经济学家阿希亚（Chetan Ahya）领衔的摩根士丹利团队在一份报告中写道：“关税与贸易紧张局势的不确定性高峰已经过去”。
彭博社报道认为，这项新的全面征税措施实际上重塑了美国贸易伙伴之间的竞争环境。对中国等国家而言，此前法院已裁定以芬太尼问题为由加征的10%关税无效，使其出口所面临的关税压力有所减轻。
相较之下，英国和澳大利亚等经济体则成为受挫一方，它们此前在旧有的“互惠”关税框架下谈判争取到10%的较低税率。
美国高级官员正敦促包括欧盟和日本在内的伙伴履行此前谈判中作出的承诺。同时，美国也希望延续与中国达成的一年期“停火”安排，特朗普计划近期访问北京，与中国国家主席习近平会晤。
中国商务部发言人星期一（23日）在官网就美国最高法院公布关税诉讼案裁决结果答记者问时说，中国正对相关内容和影响进行全面评估。发言人指出，中国一贯反对各种形式的单边加征关税措施，反复强调贸易战没有赢家，保护主义没有出路。美国对等关税、芬太尼关税等单边措施既违反国际经贸规则，也违反美国内法，不符合各方利益。
发言人称，中国也注意到美国正准备采取贸易调查等替代措施，以期维持对贸易伙伴加征的关税，北京将对此保持密切关注并坚定维护中方利益。
美国贸易代表格里尔星期天（2月22日）接受福克斯新闻采访时说：“我们希望确保中国正在履行协议中的责任……这意味着他们（中国）必须继续购买承诺采购的产品。”
此前也因芬太尼问题面临加征关税的加拿大和墨西哥同样成为受益者，相关关税如今已不再适用；彭博经济研究分析师在一份报告中分析，若《美墨加贸易协定（USMCA）》下的豁免安排继续维持，两国将处于“非常有利的地位”。
彭博社报道也提到，新的15%关税安排使此前适用10%税率的国家反而处于不利位置，澳洲和英国正是如此。同时，那些此前出口商品适用15%税率水平的国家，例如日本，如今也失去了优势。
尽管法院裁决为局势增添了新的不确定性，但分析人士称，过去一年全球贸易展现出的韧性，以及整体平均关税水平变化幅度有限，意味着短期冲击可能相对有限。
