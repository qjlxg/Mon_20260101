执法队由商业发展局牵头，配合旅游局、旅游警察、特别调查局（DSI）、移民局及春武里商务厅，查验了4家以同一地址注册多家公司之会计事务所与律师事务所，发现一名泰籍股东名下竟登记逾100家公司，涉资合计超过3亿铢，疑为替外国人代持或提供协助、咨询。
当局已责令相关当事人紧急说明；对查证属实者将依法追究。另查获4家旅游公司涉嫌更改董事以规避法定资格，现场撤销其营业执照（บริษัท อะลิเทีย ทัวร์ จำกัด、บริษัท ยอร์ อินโด-ไทย กรุ๊ป จำกัด、บริษัท วาย เจ เอซ จำกัด、บริษัท ดิ วี-เอ็กท์ จำกัด）。
对涉嫌违规经营不动产的3家企业也在深查中；春武里府已发现146家外国籍企业涉经营禁业。商业发展局重申将持续全国稽查，并警告协助外国人违法的泰国人将面临与外国人相同的刑责：最高3年徒刑或罚款10万至100万铢，或并科；如违抗法院命令，还将被处以每日1万至5万铢罚款，直至停止违法。

泰国知识产权局宣布，将“ปลากะพงสองน้ำสมุทรปราการ”（素万那普府双水域海鲈）列为最新地理标志（GI），以提升地方渔业品牌、建立严格品控与可追溯体系，推动产品进入高端市场并带动社区收益。
该海鲈主产于邦博（Bang Bo）、Mueang Samut Prakan与Phra Samut Chedi等淡咸水交界养殖区。该品种体型健硕、肉厚呈白色、肉质紧实且天然甘甜无腥泥味，产品形态包括鲜鱼、冷冻及带天然油花的日晒鱼。区内逾63条河渠与泥地鱼塘让鱼只自由觅食、低压成长，品质优良。
目前当地经营者逾166家，年产量约4,467吨，预计到2025年可创造约5.36亿泰铢经济价值。此项GI为素万那普府第四件认定产品，之前已注册的包括：คุ้งบางกะเจ้าNam Dok Mai芒果、邦博干鲻鱼（ปลาสลิดบางบ่อ）及素万那普Nam Dok Mai芒果；四项GI合计带动全省年流通规模约8.5亿泰铢。
后续将推广GI标识与追溯系统防止滥用，并透过线上平台、现代贸易与国际展览拓展通路，同时结合美食与渔村观光，吸引游客直访产地体验与品尝。

“严重受损”：美以对伊战争正在改变海湾安全格局
卡塔尔拉斯拉凡（Ras Laffan）液化天然气枢纽——全球最大的LNG中心——在周四遭伊朗导弹袭击后受到严重破坏。
数十年来，美国的安全保护伞一直是海湾国家繁荣与稳定的基石。但如今，伊朗的导弹和无人机已使霍尔木兹海峡陷入瘫痪，并袭击了海湾各国的民用机场、港口、酒店和炼油厂。这些攻击发生在地面部署了数千名美军和先进拦截系统的背景下，暴露出防护能力的明显漏洞。
分析人士表示，随着对美国防务保障信心的动摇，海湾国家正权衡更为多元化的安全策略，以应对区域内日益严峻的安全现实。

中国表示愿与东南亚国家合作应对中东战争对全球能源供应的冲击。外交部发言人林剑在周四例行记者会上表示：“中国愿与东南亚国家加强合作与协调，共同应对能源安全问题。”他并呼吁各方尽快停止军事行动，防止地区动荡影响全球增长。
随着美以与伊朗冲突升级，中东对石油、天然气及化肥供应的影响加剧。霍尔木兹海峡受限，约占全球化肥贸易近三分之一。媒体称北京收紧化肥出口并要求部分企业暂停出货后，菲律宾周三表示已获中国保证将继续供应化肥。中国是菲律宾化肥进口的18%来源，也是全球主要出口国之一。
菲律宾农业部长弗朗西斯科·蒂奥·劳雷尔（Jr.）会见中国大使经权后称，大使否认会冻结化肥出口。林剑称，北京将在满足国内需求的同时继续向国际市场出口部分国产化肥。天然气为氮肥生产的关键成本（通常占比逾70%），战事使气价大幅上涨。
联合国警告，产量削减与航运受阻每月约减少300–400万吨化肥贸易，战争对粮食、油价和运输成本的影响可能将多达4500万人推入严重饥饿。冲突亦将能源基础设施作为打击目标：以色列袭击伊朗南帕尔斯气田，德黑兰随后打击卡塔尔一处大型液化天然气出口设施，造成严重损毁。美国总统特朗普在社交媒体称美方“毫不知情”，并威胁若伊朗再次袭击卡塔尔将“彻底炸毁南帕尔斯”。

北京 2026年3月20日电（記者 Amber Wang）
台灣主要在野黨國民黨與台灣民眾黨周三通過一項合作協議，將在今年11月舉行的“九合一”地方選舉中協調提名與參選安排。該選舉涵蓋包括縣市長、鄉鎮市長、村里長及各級議員在內的九類職務，涉及全台22個行政區。
國民黨主席Cheng Li-wun表示，這份協議是“藍白”陣營邁向更緊密合作的重要一步，期望不僅有助於11月選戰取得成果，也為2028年領導人選戰打下穩固基礎。台民黨主席Huang Kuo-chang則強調，黨內將以最大善意與誠意推動合作，並迅速展開下一階段協調工作。
學者指出，作為領導期中的四年一度地方選舉，九合一常被視為兩年後大選的試金石；但該選舉重點仍在地方治理，對兩岸等重大議題的直接影響相對有限。

重庆市市长胡衡华被中纪委宣布接受纪律审查，成为近十年来这座西南直辖市被查处的最高级别官员之一。
中央纪委国家监委周五发布通报称，胡衡华同时担任重庆市人民政府市长、市委副书记，因“严重违纪违法”正接受调查；该表述通常指涉腐败问题。
重庆为中国四个直辖市之一，与北京、上海、天津同级。胡衡华出生于湖南，长期在湘任职，曾任长沙市市长（2014—2016）及长沙市委书记（2017—2020）。2020年10月至2021年12月任陕西省委副书记，2022年1月在担任一个月代理市长后正式出任重庆市市长。
担任重庆市长期间，胡曾因与2022年长沙一居民楼倒塌（造成54人遇难）相关的责任问题受到党内处分。中纪委在2023年5月认定其任内未有效开展住房安全隐患排查，并给予严重警告。
