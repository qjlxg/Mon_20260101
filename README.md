机票因中东危机涨价 抢票热潮刺激美国航司营收
（纽约综合电）随着燃油成本上升推高票价，旅客纷纷抢购机票，美国各航空公司迎来业内史上最强劲的预订热潮。
燃油是航空业仅次于劳动力成本的第二大支出，通常占运营成本的五分之一到四分之一。随着2月底美伊冲突爆发，中东航运严重受限，燃油成本急剧上涨，航空业面临自冠病疫情以来的最大危机。
许多消费者担心票价会进一步提高，因此争相锁定机票，掀起抢票潮。
达美航空公司目前预计，截至3月份的销售额可实现接近两位数的增长；公司此前预计增长幅度为5%至7%。
美国航空公司说，虽然燃油价格上涨拉低盈利预期下限，但公司本季度营收仍会增长超过10%，创下历史新高。
达美航空股价在美股早盘交易中一度上涨4.8%；美航股价则飙升5.2%，有望创下近一个月来的最大涨幅。
尽管如此，燃油成本已经加重航司运营压力。达美航空首席执行官巴斯蒂安星期二（3月17日）在摩根大通工业会议上说，航空燃油价格的急剧上升，仅在3月份就使公司的成本增加多达4亿美元（约5亿新元）。
他说，航空业正在迅速行动，通过提高票价，将多出的成本转嫁给消费者。但他提醒，在消费者信心脆弱之际，航空公司在提价方面仍须谨慎。
一些航空公司开始征收燃油附加费，但这可能会侵蚀利润。

一份鄂州市公安局梁子湖区分局作出的行政处罚决定书，上周三（3月11日）在湖北公安执法公开平台公布。

中国邮电部已于1998年被撤销。

2026年3月18日，替代能源与节能署副署长帕差瑞·钟拉克在政府府举行记者会表示，政府正动用燃油基金稳定油价、减轻民众冲击，同时加速增加能源供给，优先依靠国内能源来源。 
针对“节能加倍”措施，署方呼吁各界配合，尤其是高耗能的工业与商业单位，在未来1—3个月内制定节能计划，例如调整设备开关机时段、优化空调温控等管理措施。 
政府依3月10日内阁决议，目标将能源使用减少10%，并透过推广远程办公与线上会议以减少汽油消耗。民众与企业亦可享相关优惠：安装屋顶太阳能可申报抵税，最高可抵减20万泰铢。 
对企业的支持包括以“共同付费（Co-pay）”方式补助高效设备更新，国家补助20%—30%，每案上限300万泰铢；运输业则鼓励改善设备与运输管理，预期可节能20%—30%，单一经营单位补助上限为200万泰铢。 
署方并与金融机构合作，提供公道贷款协助设备汰换与能效提升。公众如有咨询，可拨打节能诊所电话：063-826-9784。 
帕差瑞强调，各方须携手减能耗，共渡影响民生的能源挑战，强化国家能源安全。
