福建鞋厂火灾致28死 中国官方对查处挂牌督办
“中国鞋都”福建晋江一家鞋厂发生严重火灾造成28人死亡后，中国国务院安委会对这起事故查处挂牌督办。
位于泉州晋江的辉腾鞋业有限公司厂房星期四（7月9日）中午12时许发生火灾。涉事建筑共五层，为钢筋混凝土结构，起火部位为一层，是冲床、印线等工艺车间和原材料仓库。
官方披露，火灾发生时，厂房现场有237名本厂员工和两名外来人员，现场共疏散救出213人，其中两人送医不治身亡，26名失联人员确认遇难，遇难者大多数被困厂房楼顶部位。
据央视新闻星期六（7月11日）报道，中国国务院安委会决定对上述火灾查处挂牌督办，并派出国家消防救援局牵头的现场督办组，督促指导福建省开展调查处理工作，要求福建省依照有关法律法规及规章规定，抓紧组织开展调查，尽快查明起火原因和管理原因，精准确定火灾性质，查清责任，严肃提出处理意见和整改措施。

张雪机车最迟明年在台湾设门店
中国大陆摩托车品牌张雪机车创始人张雪受访时称，公司最迟明年在台湾开设门店。
张雪在台湾《联合报》星期六（7月11日）刊发的专访中说，他非常想在台湾开店，现在正在对接中，最晚不超过明年，“开幕时我一定要亲自到场”。
对于赴台开店，张雪说，“想要让门店真正站稳脚跟，车型的丰富度是关键，店里不能只有单一品类，得有复古车、跑车，也有踏板车，让不同喜好的车友都能找到自己的选择，这样门店的经营才能更稳。”
张雪承诺，未来台湾的车主将和大陆车主享受完全一样的售后保障服务，还会给大家配套赠送贴膜、车壳等周边福利，也随时邀请台湾的车友到大陆参观工厂。
另据中新社报道，张雪近日在重庆接受媒体访问时透露，将到台湾设门店的计划，并称跟有意向的台商对接代理事宜，各项筹备工作有序推进，初步计划“最晚不超过2027年”。
张雪机车于2026年3月在WSBK顶级赛事中夺冠后，知名度迅速破圈。兼具性能、配置、价格优势的张雪机车成为台湾车友的关注热点。
张雪介绍，两岸在机车产业上的合作早已开始。以张雪机车为例，部分车型轮胎是与台湾正新轮胎联合研发的成果，“我们一起攻关、优化测试、共同成长”，后期协同研发将持续深化。
张雪表示，若有机会赴台开展行业交流，正新轮胎总部将是他首站打卡地，同时也期待与台湾本土机车企业互通产业发展经验。

广西一动物园洪灾锁兽笼致狮子溺亡 PETA：不可接受
在台风“美莎克”引发严重洪涝期间，中国广西一家动物园为防止猛兽逃脱，将狮子等动物锁在笼中，最终导致部分动物被淹死。国际动物权益组织善待动物组织（PETA）批评这一做法“不可接受”，并呼吁动物园建立完善的灾害应对机制。
广西上星期六（7月4日）以来遭遇罕见的大范围强降雨，‌南宁、防城港、钦州、崇左、北海等地出现持续暴雨，境内多个水库发生险情。
据《红星新闻》报道，贵港动物园在洪水来袭前，为防止园内猛兽逃出伤人，将狮子、熊和狼等动物锁在笼中。动物园负责人殷飞飞星期三（7月8日）说：“洪水到来前，我们把猛兽区的笼门给锁死了。我们不能在洪水到来时再给国家添乱，让猛兽跑出去伤人。”
报道称，洪灾期间园区积水最深超过两米。洪水退去后，三只狮子死亡，另有100多只动物失踪。
根据动物园在社交媒体发布的寻物启事，失踪动物包括两匹斑马、三匹迷你马，以及鸵鸟、羊驼、浣熊和孔雀等。园方呼吁公众协助寻找和捕回被洪水冲走的动物。
中国官媒《环球时报》随后报道，其中一匹失踪斑马已被发现死亡；被锁在笼中的棕熊和狼虽然幸存，但因险些被洪水淹没，身体状况不佳。
PETA亚洲地区总裁贝克（Jason Baker）星期五（7月10日）在声明中说，贵港动物园的悲剧应为所有位于极端天气风险地区的动物园和野生动物圈养设施敲响警钟。
他说，在洪水上涨时让动物困在笼中“不可接受”；但另一方面，在灾害发生时直接放生圈养野生动物，同样是不负责任且危险的做法，既可能危及动物，也可能威胁公众安全。
贝克呼吁动物园建立完善的动物疏散和应急预案，并最终停止圈养野生动物。

张玉卓任中国工程院院长
今年5月转岗中国工程院的张玉卓，已任中国工程院院长。
据中新网报道，中国工程院第十八次院士大会星期六（7月11日）在北京闭幕。大会宣布，中国工程院产生新一届院领导班子，张玉卓任院长。
中国工程院第九届院领导班子组成如下：张玉卓任中国工程院党组书记、院长；张军、陈杰、李仲平、陈建峰、陈薇等五位院士任中国工程院党组成员、副院长。
公开资料显示，张玉卓今年64岁，十九届中央候补委员、二十届中央委员。他是一名洁净煤技术专家，主要从事煤制油化工技术研究，于2011年当选中国工程院院士。
张玉卓曾长期在能源领域工作，担任过神华集团有限责任公司董事长、党组书记兼董事长。2017年3月，张玉卓履新天津市委常委、滨海新区区委书记，2018年5月起兼任中国（天津）自由贸易试验区管理委员会主任。2020年1月，他调任中国石油化工集团有限公司董事长、党组书记。
2021年7月，张玉卓转任中国科协党组书记，后担任中国科协分管日常工作副主席、书记处第一书记职务。2022年12月履新国务院国资委党委书记，后获任主任。今年5月底接替李晓红，担任中国工程院中共党组书记。

中共就整治形式主义召开会议，要求深化拓展整治形式主义为基层减负工作，强调要深入纠治急功近利、盲目决策问题，动真碰硬纠治统计造假，防范和纠治政绩观偏差。

中共政治局常委、中央书记处书记蔡奇在会上讲话时强调，要深入学习贯彻中共总书记习近平“七一”讲话精神，学习贯彻习近平党建思想，把整治形式主义为基层减负“同实现新时代新征程党的使命任务联系起来，和保持党同人民群众的血肉联系联系起来”，结合开展树立和践行正确政绩观学习教育，持续深化拓展整治形式主义为基层减负工作，更好把减负成效转化为治理效能。
会议强调，要严格执行整治形式主义为基层减负若干规定，及时发现和纠正苗头倾向，紧盯突出问题深化整治工作。深入纠治急功近利、盲目决策问题，抓实新官不理旧账突出问题集中整治，动真碰硬纠治统计造假，有效防范和纠治政绩观偏差。
会议要求，持续解决基层反映强烈的难点堵点问题，推动乡镇（街道）履职事项清单执行，健全完善村（社区）工作事项准入制度，加快推进“一表通”工作，巩固拓展“指尖上的形式主义”整治成效；并切实做好发文开会、节庆展会论坛活动的源头管控和日常监管，落实制发文件与为基层减负一致性评估制度，持续整治节庆展会论坛活动乱象。
会议也提出，要加强对换届后各级领导干部的政绩观教育。
中国官方近年持续要求整治形式主义，今年3月的中共政治局会议提出，地方党委要树立和践行正确政绩观，坚持为人民出政绩、以实干出政绩；地方党委领导班子成员要坚定不移推进全面从严治党，严格贯彻中央八项规定及其实施细则精神，深入整治形式主义为基层减负，依法依规用权，守好廉洁底线。

高盛 ：抄底芯片股

机构：韩国领跌，6月新兴市场股市遭遇460亿美元大逃离
银行业贸易组织的数据显示，6月外国投资者从新兴市场股票投资组合中净流出 461亿美元。这一数据主要由对韩国科技股的大幅抛售推动，导致发展中经济体连续第二个月出现整体投资组合亏损。国际金融协会（IIF）周五发布的月度报告显示，外国投资者从韩国股市撤资 305亿美元，创下最近25年多来的最大流出规模。报告显示股市与债市资金流向出现明显分化：尽管整体投资组合资金流转为净流出178亿美元，但债券市场上月仍吸引了 283亿美元资金。国际金融协会（IIF）首席经济学家Jonathan Fortun表示：投资者仍愿意借钱给新兴市场，但他们不太愿意承担广泛的股市风险。全球贴现率上升、盈利信心疲软以及对科技和能源仓位的敏感性，导致投资者削减了股票配置。上半年的信息很明确，新兴市场总体上仍吸引了资本，但这仅仅是因为债务流入完全抵消了持续的股市清算。

摩根士丹利 警告称芯片制造商的定价能力面临更大压力
摩根士丹利的Lisa Shalett呼吁投资者对芯片股保持谨慎，她表示，更多迹象表明，芯片制造商的定价能力有限，意味着这些股票可能因为对人工智能支出的乐观情绪而上涨过头。摩根士丹利财富管理首席投资官Shalett周五表示：“我们看到AI数据中心技术栈正在被重新设计，以纳入成本更低的自研芯片，而许多超大规模云服务商如今在自行设计这些芯片。”Shalett发出上述警告之际，韩国芯片制造商SK海力士周五开始在纳斯达克交易。该公司此前通过美国上市募资265亿美元，创下外国企业在美国规模最大的首次公开募股纪录。与此同时，SK海力士在韩国上市的股票近期剧烈波动，自上月达峰以来已下跌26%。“总体而言，该交易领域仍有充足的资金可用，”Shalett表示。但她指出，市场正出现业内一种典型现象：“当供应链出现瓶颈，并且部分存储芯片厂商获取超额利润时，工程师们就会着手”寻找更便宜的替代方案。Shalett本周在一份投资者报告中称，半导体板块“明显超买”。她在Surveillance节目中表示，从芯片ETF到费城半导体指数，都可以看到支持这一判断的迹象。根据汇编的数据，费城半导体指数的市盈率自2022年以来已增长逾两倍。
