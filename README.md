中国十五五着眼建设生育友好型社会
中国政府星期四（3月5日）表示，未来五年将着眼建设“生育友好型社会”，并出台措施解决教育、就业、养老和收入等方面的问题。
中国总理李强星期四在全国人大会议开幕会上做政府工作报告，其中介绍“十五五”时期（2026年至2030年）的主要目标和重大任务。李强指出，在突出推进全体人民共同富裕这一“十五五”时期重大战略任务下，中国将着眼建设生育友好型社会，健全人口服务体系。
根据《中华人民共和国国民经济和社会发展第十五个五年规划纲要 （草案）》，政府将积极应对人口老龄化，并促进高质量、充分就业，完善收入分配制度，健全社会保障体系，着眼缩小区域差距和城乡差别，加快农业农村现代化等。
李强在报告也提出，政府将倡导积极的婚育观，并加强初婚初育家庭的住房保障，支持多子女家庭改善性住房需求，完善生育保险制度和生育休假制度等。
官方数据显示，中国人口在2025年连续第四年下降，出生率降至历史最低水平。
路透社报道指出，中国政府已将人口规划作为国家经济战略的重要组成部分。
路透社估计，今年中国在促进生育方面的潜在总成本约为1800亿元（人民币，333亿新元），主要支出包括去年首次推出的国家生育补贴，以及中国国家医保局此前所宣布、力争今年全国基本实现政策范围内分娩个人“无自付”等政策目标。

变革往往是渐进的，尤其在我们这里，很多变化不易察觉，甚至有时朝坏处发展。人们常以文化或个人喜好解释我们选出的领导人，但真正的问题在于体制与选择机制长期未改。我们需要一种“扰动”来打开改变的窗口，关键在于能否把这次扰动变成真正的制度性改革。
可以从是否真正拥有民主开始反思。四十年前的人民力量（EDSA）被称为“革命”，理应带来根本性重置，但两代人过去后，民生与政治结构并未根本改善。流行与名气越来越能左右选举，但国家更需要有能力、有资格的领导人，而非仅靠表演或形象当选。
回顾马科斯（老）时代，尽管其下台备受争议，但其政权在早期确实推动过重大改革，组建了一批技术型内阁成员，并尝试重新设计宪政架构（1971年制宪、1973年宪法、1976年第六号修正案）。若这些制度化的改革成功并延续，菲律宾或许能如韩国、台湾、新加坡般实现发展；可惜实施中出现严重问题，承诺未兑现，腐败与压迫最终引发民怨。
因此，1986年的人民力量既是对承诺破灭的回应，也反映出民众对真正变革的渴望。问题仍然是：那次运动是否真正完成了制度性的革命？答案需回到对历史、政治体制与人才培养机制的深刻反省，才能把一次“扰动”转为长期的政治革命与国家治理提升。
