澳门今年设政府引导基金 引领社会资本投资重点产业
澳门官方宣布，将在今年内设立政府引导基金，引领社会资本投资重点产业。
据澳门新闻局官网消息，澳门特区政府星期五（2月27日）召开新闻发布会介绍上述基金方案。
澳门政府计划将在今年内设立政府引导基金，以财政储备历年滚存收益向基金注资，希望借此引领社会资本投资重点产业和项目，推动经济多元发展、培育新兴产业、推动产业升级等，从而促进澳门就业与人才发展，惠及澳门企业和居民。
澳门经济财政司司长戴建业在会上介绍，为使基金能有效“发挥引领作用，共促产业多元”，政府经综合考虑，初步计划以财政储备历年滚存的收益向基金注资110亿澳门元（17.27亿新元），联同首批社会资本设立引导基金，预期引导基金规模为200亿澳门元。
据介绍，基金将引导不同类型的社会资本共同投向包括新兴产业、转型升级产业、科技成果转化项目及初创科技企业等领域；聚焦国家战略、产业升级、科技创新、民生保障等重点领域；符合“一中心、一平台、一基地”的发展定位，并充分发挥澳门独特优势的领域；支持澳门特区与合作区、大湾区联动发展的投资项目等。
基金将坚持“长期主义”及“耐心资本”的理念，着重培育对澳门经济有实际推动作用的新兴产业和未来产业。
“一中心、一平台、一基地”，是中国十五五规划（2026年至2030年）对澳门发展提出的部署。“一中心”，是将澳门建设成世界旅游休闲中心，有助于提升澳门经济社会活力。“一平台”，是打造中国与葡语国家商贸合作服务平台，有助于发挥澳门独特双语优势和自由港、单独关税区等政策优势。“一基地”，是建成以中华文化为主流、多元文化共存的交流合作基地，有助于传承和弘扬中华文化，促进东西方文化交流互鉴。

委员会中的民主党成员则呼吁应传唤同样与爱泼斯坦有联系的唐纳德·特朗普接受询问。民主党议员苏哈斯·苏布拉马尼亚姆表示：“说实话，我们问错了总统。”他同时称克林顿并未回避问题。
克林顿在声明中虽未点名特朗普，但强调“没有人高于法律，哪怕是总统——尤其是总统。”特朗普则对整个程序表示怀疑，并称他喜欢克林顿，不希望看到他被询问。
此前被司法部释放的相关文件中多次提及克林顿及特朗普，但二人均未被指控犯罪或接受正式调查。克林顿承认曾与爱泼斯坦有多次接触，并乘坐其私人飞机出差，但否认去过爱泼斯坦在加勒比海的私人岛屿。
克林顿夫妇最初拒绝传票，后在众院共和党人威胁以藐视国会后同意出庭。前国务卿希拉里·克林顿周四作证时强烈要求传唤特朗普，并称自己从未认识爱泼斯坦或到访其房产。
爱泼斯坦2008年因招募未成年女孩发生性行为被定罪，2019年在狱中死亡，官方判断为自杀，但围绕其案件仍有诸多阴谋论。此次闭门询问在克林顿家乡查帕夸进行，吸引众多记者和特勤局人员到场。

马尼拉 — 一只幼年亚洲棕榈狸（俗称alamid或musang）于2月27日（星期五）在马林杜奎省莫格波格被救出，免于遭到屠宰威胁。
马林杜奎省兽医办公室调查表示，该狸被居民用尼龙绳自制的陷阱捕获，原本可能被当作下酒菜。幸经一名关切市民提醒其为受保护物种、不得捕捉、屠杀、食用或买卖，才阻止了进一步伤害。
省动植物救援应急队迅速为其处理左后足伤口并进行治疗，以减少额外应激。
省兽医Dr. JM Victoria指出，棕榈狸极易受压、恢复缓慢，有时甚至出现自伤行为；因牙齿尚未全部萌出，仅以幼年、亚成体与成体作年龄分类，且部分个体呈雌雄同体，必须通过分子性别鉴定（PCR）确定优势性器官。
棕榈狸受2001年《野生动植物资源保护法》（第9147号）保护，在森林生态中担任重要的种子传播者角色。
Victoria表示，此次救援体现马林杜奎省政府在野生动物保护、栖息地保育及维护自然遗产方面的持续承诺。
待伤口痊愈并恢复体力后，省兽医团队将与环境与自然资源部（DENR）联络，安排其回归自然栖息地。

OpenRouter最新排名显示，中国开源人工智能模型在Token使用量上位居全球，结束了美国开发者一年的主导地位。该榜单反映出在一系列新模型发布后，国际市场对中国开源模型的需求显著上升。
上海MiniMax发布的M2.5（约两周前推出）本月在OpenRouter平台以4.55万亿Tokens成为使用量最高的模型；总部在北京的Moonshot上月推出的Kimi K2.5以4.02万亿Tokens位列第二。Token是模型处理输入数据的基本单位，包含整词、词片段、字符或标点。
OpenRouter本月前五名还包括Google DeepMind的Gemini 3 Flash Preview、杭州DeepSeek的V3.2与Anthropic的Claude Sonnet 4.5。OpenRouter表示，MiniMax、Moonshot与DeepSeek三家在前五名中合计约占总Token使用量的近三分之二。

美国AI公司Anthropic指控三家中国AI企业使用其系统“蒸馏”Claude模型一事，暴露了AI训练中的一处灰色地带并引发争论，分析人士指出。
Anthropic在周一的博客中称，DeepSeek、Moonshot AI和MiniMax AI通过约2.4万个虚假账号与其Claude模型进行了超过1600万次交互，Anthropic认为这些行为构成未经授权的“蒸馏”——即将大模型的能力迁移到更小模型的技术。
所谓“知识蒸馏”在AI开发中被广泛使用，旨在用大模型的输出训练更小、便于部署的模型。该方法由包括“AI教父”Geoffrey Hinton在内的研究者于2015年提出。
此事重新点燃了外界对中国开源模型快速提升能力背景下，蒸馏界限与合规问题的关注与质疑。
