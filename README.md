6月下旬热浪期间 欧洲录得1万多起超额死亡
（布鲁塞尔综合电）欧洲官方数据显示，6月底席卷欧洲西部并创下多项高温纪录的热浪，导致欧洲出现1万多起超额死亡个案，其中9000多起涉及65岁及以上人士。 
EuroMOMO发布的数据汇总了欧洲27个国家的全国死亡统计，涵盖6月22日至28日这一周所有原因造成的超额死亡，而不仅限于与高温直接相关的死亡。
负责运营EuroMOMO的丹麦国家血清研究所首席医师韦斯特加德指出，目前没有其他已知重大因素，足以解释死亡人数的激增。
他告诉路透社：“在一年中的这个时节出现如此高的超额死亡率实属罕见。这个数字确实很高。”
极端高温可能致命，原因在于它会引发中暑或加重心血管及呼吸系统疾病，而老年人属于最易受其危害的群体。
6月底的热浪席卷了法国、西班牙、英国等多个欧洲国家，造成电力供应中断，学校被迫停课，并刷新多项气温纪录。 
不过科学家说，若没有人类活动导致的气候变化，6月下旬的这波热浪“几乎不可能”发生；气候变化正使得热浪变得更加频繁且剧烈。
星期一（7月13日）发布的另一项科学研究则估计，单是英格兰和威尔士，就有2700人在5月和6月的热浪期间死于与高温相关的原因。其中42%的死亡是由于全球暖化导致高温加剧所引发的热浪造成的。
伦敦帝国理工学院研究员巴恩斯告诉彭博社：“热浪是最危险的极端天气。由于人类活动引发的气候变化，我们目前面临的这种极端高温对人类健康构成极大的威胁。”
经济师警告，今年可能出现的超级厄尔尼诺现象，可能对全球粮食价格造成严重冲击，影响或持续至2028年。 
随着伊朗战争推动全球粮食价格升至三年来最高水平，经济师指出，在全球暖化导致的极端天气影响下，全球供应链正同时面临双重冲击。 
高盛分析师预计，这次厄尔尼诺可能推动全球粮食大宗商品价格上涨15.8%。这将在全球范围产生连锁反应，波及包括欧洲在内的消费者，预计欧元区的食品价格可能因此上涨1.3%。
不过，由于气候变化造成的冲击传导至全球粮食供应链需要时间，其全面影响可能要到2028年下半年才会完全显现。

香港容许狗只进入获批食肆
香港特区政府上周四（7月9日）起容许狗只进入获批准的餐厅，被视为推动人宠共融的重要一步，但相关政策推出至今争议不断。有受访学者认为，当局应考虑推出奖赏计划，奖励表现出色的餐厅。
近年香港社会越来越重视宠物权益，港府上周四结束实施逾三十年的“禁狗令”，容许狗只进入“宠物友善食肆”，首阶段有940家餐厅获批。然而，食物环环境卫生署截至上星期日收到约20起投诉，社交平台上更涌现大量网民分享狗主的不当举止。有人目睹邻桌食客用餐厅餐具喂宠物车中的狗只；另有网民投诉，有宠物狗进入餐厅后随地小便，主人用纸巾简单清洁后，即把纸团留在餐桌。 
此外，连日来有餐厅陆续退回牌照。截至上周日晚上8时，食环署公布的获准餐厅名单上减至937家。其中，亚洲国际餐饮集团旗下的百份百餐厅锦英分店，于新措施实施首日便取消接待狗只。集团发表声明表示，经评估后，部分分店因店铺空间、座位布局、出入口设计及客流情况等因素，若容许顾客携同狗只进入用餐，或会影响其他顾客的用餐体验、现场秩序及安全，因此决定就个别分店主动申请取消相关许可。
港府食环署署长袁旭健周一指出，新政策推行的首月是适应期，署方已成立大约90人的专队，至今已巡查超过2300次，当中包括400家未获批准的餐厅；署方也接获约20宗涉及狗带长度或狗只位置等轻微违规的投诉，已跟进调查。
袁旭健认为，新政策的相关安排运作大致畅顺，包括分区安排、卫生管理或职员应对均有秩序，狗主普遍愿意配合规则。署方会采取审慎及循序渐进的策略，若第一阶段运作理想，预计最快在六个月后考虑展开第二阶段申请，届时会因应首阶段的评估，再决定及公布详情。
香港中文大学通识学部讲师蔡思行接受《联合早报》访问时指出，港人一向有比较强的“边界感”，顾客在餐厅用餐，理论上拥有用餐期间占用餐桌的使用权。但许多香港餐厅的座位距离狭窄，客人携带的宠物，很容易骚扰到邻桌食客，冲击了这种“边界感”，容易引起争执。
他认为，港府应加强指引，避免餐厅在未深思熟虑下加入计划后又迅速退出。长远而言，当局可以考虑推出奖赏计划，由宠物主人及食客投票选出“人宠共融食肆（餐厅）”，给予奖励和认证。

台风“巴威”继续影响中国多地 沈阳遭遇百年一遇极端降雨
（北京/长春/郑州综合讯）台风“巴威”登陆中国浙江省后进一步北上，东北多地出现持续性强降雨。其中，沈阳市遭遇百年一遇级别暴雨，城区多处被淹，全市星期一（7月13日）启动防汛应急一级响应。
综合央视新闻、《长春晚报》、《大河报》等媒体报道，受“巴威”影响，江苏、山东、安徽、甘肃、河北、辽宁、吉林等省份部分地区星期一出现大范围强降雨，其中辽宁多地雨势猛烈，省会城市沈阳遭遇特大暴雨袭击。
沈阳市气象台星期一发布气象预警最高级别的暴雨红色预警。沈阳市气象局人员称，本轮降雨强度，达到当地百年一遇级别的极端降雨。 
社交平台视频显示，沈阳多处低洼路段积水汹涌，雨水漫过路面绿化带，不少车辆浸泡在积水中，部分下穿桥洞完全被雨水灌满，水深直逼行人胸口，有人在水中划艇。
截至星期一上午，沈阳因降雨共产生73处积水点位全部封闭。除保障城市基本运行的机构，全市机关与企事业单位原则上居家办公，在建工地一律停工，全市景区景点暂停营业。
中国气象频道气象分析师信欣称，沈阳遭遇了降雨列车效应，“巴威”助力副热带高压持续向北传输水汽，且副热带高压位置相对稳定，因此水汽源源不断供应，令当地降雨既有爆发力，又有耐久力。
截至星期一，辽宁全省共转移17万1412人，其中抚顺15万6933人、沈阳2587人、鞍山908人等，沈阳、抚顺、锦州、铁岭、盘锦、辽阳等市星期一停课、停业、停工。
中国中央气象台星期一继续发布暴雨橙色预警和强对流天气蓝色预警，预计自星期一至星期二下午2时，辽宁、吉林、山东、江苏、安徽、江西、青海、甘肃、宁夏、广东、海南、台湾等地部分地区有大到暴雨。
目前遭遇强降雨的沈阳、鞍山、抚顺、本溪、锦州、营口等地，预计星期二（14日）仍有大暴雨，局部有特大暴雨。
