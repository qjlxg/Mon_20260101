湖北1500平方米楼房被指违规建设 村支部书记被免职

综合澎湃新闻和每日经济新闻报道，多位网友发视频反映，这座楼房位于襄阳市东津新区东津镇沈营村，占地面积明显超出国家规定的农村宅基地标准，涉嫌未批先建、超占土地、违规建房。房屋被举报为沈营村肖姓支部书记所有。
网友拍摄的视频显示，楼房有大面积院子和围墙，还有附属用房。多名网友称该楼房为“襄阳一二三大厦”，并到此打卡。

通报称，在2011年、2014年两次未办理相关手续违法建设房屋过程中，沈营村委会未派人前往制止，也未向上级主管部门报告。东津镇政府及相关部门巡查发现后，于2014年5月底对相关责任人进行了处罚，但未对肖姓父亲违法建设房屋进行处理。
管委会承诺，将对失管失察相关责任人进行追责问责，并将举一反三，加大违法建筑查处力度。

日澳签历史性协议 三菱重工将为澳建造11艘隐形护卫舰
（墨尔本／悉尼综合电）澳大利亚与日本政府正式签署合作协议，由日本三菱重工为澳洲海军建造11艘新一代隐形护卫舰。澳洲希望借此提升海上远程作战能力，应对中国在印太地区不断扩大的军事存在。
澳洲副总理兼国防部长马尔斯与日本防长小泉进次郎星期六（4月18日）在墨尔本的日本海上自卫队“最上级”熊野号护卫舰，出席澳洲采购首批护卫舰的启动仪式，并签署相关合作协议。
这是日本自2014年解除武器出口禁令以来，规模最重大的军售项目之一。小泉进次郎形容这项协议“具有历史意义”，并指日澳在深化双边安全关系方面“没有上限”。
澳洲政府早前披露，首三艘护卫舰合同预算介于150亿至200亿澳元（约136亿至182亿新元）之间。　　
根据规划，首批三艘升级版“最上级”护卫舰将由三菱重工在长崎造船厂建造，预计首舰于2029年交付，2030年投入服役。其余八艘则将在西澳珀斯附近的亨德森造船厂建造，由澳洲本土造船企业奥斯塔（Austal）承建。
澳洲计划在未来10年内，将主力水面舰艇规模从目前的11艘扩充至26艘，而通用护卫舰被视为未来水面舰队的主力。随着中国在印度洋、太平洋及澳洲北部海上通道的军事存在不断扩大，这批具备反潜、对海打击和防空能力的护卫舰将部署在关键海上航道，加强海上交通线的防护。
虽然这项合作具有高度战略价值，但也面临不小挑战。这是三菱重工首次在海外承担如此复杂的先进军舰建造任务。澳洲政府也表现出谨慎态度，希望在推进过程中避免过去大型军购案中常见的成本飙升或工程延误问题。

警方表示，虽未将本次纵火定性为恐怖袭击，但鉴于与早前案件的相似性，反恐部门负责侦查。目击者称，一名男子被看到走向一排商铺，携带一个塑料袋，内有三瓶液体，他将袋子放在建筑旁并点燃，瓶内物质未完全燃烧后该男子逃离现场。 
警方称目前尚无正式关联结论，动机仍在调查中。负责该地区警务的刑事总督察卢克·威廉姆斯表示，最新事件可能增加社区恐惧，警方将加强巡逻并增加武装警力。来源：路透社

德黑兰消息——伊朗军方中央指挥部4月18日宣布再次关闭霍尔木兹海峡。数小时前该海峡曾在黎巴嫩停火协议达成后对外开放，但因美国坚持对伊朗港口实施海上封锁，德黑兰威胁重新封锁，最终恢复“严格管理与控制”。
航运追踪显示，多艘商船在狭窄水道中折返，也有至少八艘油气轮在当日已通过海峡；另有多艘开始驶出波斯湾后掉头。英国海事安全机构UKMTO称，一艘在阿曼东北海域的油轮报告遭两艘伊朗革命卫队舰艇无无线电警告靠近并开火，船员安全，正在调查。
伊朗副外长赛义德·哈蒂卜扎德在土耳其外交论坛上表示，美国不能通过围困强加意志；伊朗最高领袖莫杰塔巴·哈梅内伊书面称伊朗海军“随时准备”击败美军。
停火仅余四天，但和平谈判仍在推进。美国总统特朗普称协议“非常接近”，并在社交媒体上称“伟大且辉煌”。巴基斯坦军方统帅阿西姆·穆尼尔结束对伊朗的三日斡旋访问，巴基斯坦将主办下一轮谈判。埃及外长亦表示希望“数日内”达成最终协议。
谈判两大难点为伊朗近武器级浓缩铀库存与霍尔木兹海峡的未来。特朗普声称伊朗同意交出约440公斤60%浓缩铀并称要“用许多挖掘机”取出，伊朗外交部发言人巴卡依否认将铀转移出境的说法。战争爆发以来，伊朗境内国际互联网仍部分中断，监测显示断网已达第50天。

英国外交大臣伊薇特·库珀4月18日在土耳其安塔利亚出席外交论坛时对路透表示，尽管针对伊朗的冲突已达成停火，霍尔木兹海峡尚未恢复正常通航，她敦促德黑兰允许全球航运全面恢复。
当日有油轮车队通过该海峡，这是自2月28日美以对伊发动行动以来该要道首次出现的大规模船只通行。
伊朗革命卫队称，已同意“有限数量油轮和商船的有序通行”，但警告若美国不确保往返伊朗船舶的完全航行自由，海峡将继续处于伊朗严格控制下。
库珀表示，美伊停火需发展为持久和平，尽快恢复航运对全球经济至关重要。她称已有50多个国家支持维护航行自由，十余国愿在冲突结束后提供包括扫雷与护航在内的海上支援，并敦促各方遵守停火承诺。路透社

安塔利亚 — 土耳其外长哈坎·菲丹4月18日在地中海度假城举行的安塔利亚外交论坛上指责以色列以“安全”为借口，制造国际“幻象”，实则图谋“夺取更多土地”。

菲丹17日曾在论坛间隙与巴基斯坦、沙特、埃及外长就中东局势举行会谈，呼吁地区国家成熟应对共同挑战。他还表示，土耳其针对俄乌战争的低调外交被与伊朗相关的紧张局势掩盖，待对伊紧张缓和后应重启乌克兰相关谈判。论坛期间，土耳其也分别接待了俄外长拉夫罗夫与乌外长安德里·西比哈出席的不同讨论。

马德里，4月18日——哥伦比亚总统古斯塔沃·佩特罗周六在接受西班牙《国民报》采访时警告，如果美国不重新审视对拉美的政策，对华盛顿的反弹可能演变为“叛乱”。
佩特罗曾为左翼游击组织M-19成员，长期与美方存在摩擦。美国财政部海外资产控制办公室（OFAC）将他列入制裁名单，指其涉入全球非法毒品贸易。佩特罗表示，OFAC制裁已被用作政治工具来胁迫“持不同政见者”，并称这是「类似几个世纪前西班牙王权的制度」，拉美的回应将是叛乱，除非美国重新考虑与拉美的关系。
他并指出，美国在一月为抓捕委内瑞拉总统马杜罗而对加拉加斯实施的轰炸，使许多拉美领导人感到恐惧。尽管如此，佩特罗称与特朗普的私人关系良好，最近两人通话打破了相互间的误传，他去华盛顿并非“下跪乞求”，而是以平等身份会面。佩特罗于2022年当选，因宪法限制不得连任，预计将于八月离任。路透社
