福建泉州六酒店民宿因农历新年非法涨价 被立案查处
中国福建泉州六家酒店民宿在农历新年期间非法涨价，被立案查处。
《南方都市报》报道，福建泉州市鲤城区市场监管局、丰泽区市场监管局日前发布，当地六家酒店民宿涉嫌在春节期间出现价格违法、虚假宣传侵害消费者权益情况，被立案查处。
记者发现，这些酒店民宿在泉州市中心，周边有众多知名景点，价格比非节假日期间翻了三至四倍。因价格超出其所作出的价格承诺，这些酒店民宿被立案查处。
通报称，经核查，一家位于开元寺附近的酒店“标准大床房”非节假日价格为202.52元（人民币，下同，37新元）/间/晚，二春节期间平台预订价格高达975.5元/间/晚，溢价3.8倍，涉嫌价格违法，违反《明码标价和禁止价格欺诈规定》，被予以立案查处。
此外，也有一家位于鲤城江南王宫附近的酒店，涉嫌对所提供商品或者服务作引人误解的虚假宣传。经核查，这家酒店在在线旅游平台发布的房型信息中，虚报房间占地面积，其双床房被宣传为占地28至35平方米，但实际面积约为24平方米，被当地市场监管局予以立案查处。
不过报道称，目前泉州主城区仍有房间在售的酒店民宿价格多在500元以上，多数涨幅相当于平日的二至三倍。
有行业机构在2月初监测统计显示，今年农历新年假期，广东（潮州、汕头）、福建（泉州、福州）、云南（西双版纳、普洱）的酒店价格水平领跑中国，较平日涨幅多在100%至250%区间。

俄罗斯为缓解劳动力危机转向印度、斯里兰卡等国
俄罗斯长期依赖中亚移民填补劳动力缺口，但人口结构恶化与对乌战争导致数十年来最严重的人手短缺。官方估计到本世纪末前，经济需再补充约1100万劳动力。普京去年12月访印时签署简化临时劳工程序的协议，推动与印度的务工合作。
数据显示，俄罗斯向印度公民发放的就业许可由2021年的约5000张增至2025年的逾5.6万张；2025年对外工作许可总数超过24万，为2017年以来最高。来自印度、孟加拉、斯里兰卡和中国的劳工开始承担清雪、市政、建筑和餐饮等岗位。
用工短缺与多重因素相关：人口老龄化（约四分之一已达退休年龄）、失业率仅约2%、约50万至80万适龄劳动者因反对战争、回避征召或其他原因外流，以及自2024年莫斯科郊区音乐会袭击后对免签流动的收紧。外企与国企均感受压力——诺里尔斯克镍业在西伯利亚曾短缺约1万员工，造船企业也缺1500至2000人。
除南亚外，俄方还在利用与朝鲜关系拓展劳力来源：2024年赴俄朝鲜留学生约9000人，开发商预估到2025年底在俄建筑工地的朝鲜工人或达5万人。中国劳工多在本国企业或中小企业体系内工作。
专家认为，这一劳动力匮乏并非短期现象，而是将长期影响俄罗斯经济结构与增长路径。来源：彭博社
