中国是全球最大的天然橡胶消费国与进口国。预计去年国内消费已超过700万吨，85%以上依赖进口，主要用于庞大的汽车产业。
杜仲（学名Eucommia ulmoides）既是传统中药材，其树皮长期用于中医药，也是中国独特的天然橡胶来源，被报道为全球第二大天然橡胶资源。
杜仲橡胶适用于军工材料：在高性能防刺轮胎中加入3%至5%可提高耐久性和耐磨性，亦可用于制造先进电磁屏蔽复合材料。
过去杜仲主要分布于长江流域等华中、华南地区，产量有限且提取工艺复杂。此次戈壁人工栽培取得进展，为拓展橡胶原料来源、缓解进口依赖提供了新路径。
