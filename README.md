澳大利亚向五名伊朗女足队员发放人道主义签证
2026年3月10日 — 澳大利亚政府周二为五名在澳参加亚洲杯的伊朗女足队员发放了人道主义签证，回应她们在国内恐遭迫害的求助并寻求庇护。
总理安东尼·阿尔巴尼斯在堪培拉表示，这些球员“现在安全，应感到这里像家一样”，并称对其他队员也提供帮助，但由她们自行决定是否接受。余下球员仍在黄金海岸的一家酒店。
事件曝光后，美国前总统特朗普称已与阿尔巴尼斯通话，表示已有五名队员“得到照顾”，其余人在处理当中；同时提到部分球员担心家人安全可能选择回国。阿尔巴尼斯称特朗普在当地时间凌晨致电，双方就过去48小时的处置交换了信息。
半官方的法尔斯通讯社称，五人曾在澳大利亚警方陪同下从队伍下榻酒店秘密离开。澳内政部长托尼·伯克在社交媒体发布与五名球员在不公开地点合影及签署文件的照片，并公布球员姓名。
全球球员工会FIFPRO称，此举是积极一步，但仍担忧其他队员及其家人在伊朗的安全与福祉，强调应确保球员知情并自主决策。
报道还指出，伊朗媒体称队伍原计划经迪拜回国遭拒，或改经马来西亚和土耳其返程；在对阵澳大利亚的第二场比赛中，队员曾唱国歌并敬礼，引发有人权界担忧她们可能被胁迫。此前澳大利亚在2021年也曾为阿富汗女板球队员发放紧急人道签证。
（来源：路透）

澳大利亚将向阿联酋提供先进中程空对空导弹，并派遣一架E-7A Wedgetail“楔尾”预警机赴中东执行空中监视任务，但不会在伊朗部署地面部队，澳大利亚总理安东尼·阿尔巴尼斯周二表示。
阿尔巴尼斯称，此举旨在帮助海湾国家防卫，抵御来自伊朗的“无端袭击”，并强调澳大利亚“不是主角”，其参与完全是防御性质，旨在保护驻地区的澳大利亚公民以及阿联酋等盟友。
他表示，预警机将由波音制造，初步派遣期限为四周；向阿联酋提供导弹的决定是在与阿联酋总统穆罕默德·本·扎耶德通话后作出的。
随着冲突升级，霍尔木兹海峡这一全球约五分之一石油和液化天然气运输的咽喉通道已基本关闭。冲突发生11天时，约有11.5万澳大利亚人在中东；其中约2600人已通过商业航班返回，当时多座海湾城市遭到伊朗轰炸。
（路透社）

“由我们决定何时结束战争，不是美国”：伊朗强硬回应特朗普言论
德黑兰对美国总统特朗普关于“战争很快会结束”的表态作出强硬反驳。伊朗革命卫队在声明中表示，“由我们来决定战争何时结束”，并称“地区的算式与未来格局掌握在我方武装力量手中；美军不会决定战争终结”。
特朗普曾表示对伊朗的军事行动可能是“短期的外出行动”，但警告若伊朗阻断霍尔木兹海峡的石油流通，“美国将以迄今为止二十倍的力度回击”，并称将摧毁“易被摧毁的目标”，使伊朗难以恢复为国家。同时他表示希望这种情况不要发生，并称这是对中国及依赖霍尔木兹海峡国家的“礼物”。
这场冲突已扰乱全球油气供应，油价一度升至2022年以来高点，此前伊朗新任最高领袖穆加塔巴·哈梅内伊的选举被视为局势趋硬的信号。随后油价回落、美国股市上涨，市场对冲突短期内缓和抱有希望。
普京与特朗普就此通话，俄方顾问称普京在与海湾和伊朗领导人会谈后提出若干“快速政治外交解决”的构想。
霍尔木兹海峡的封堵和对经过船只的袭击已严重冲击海湾能源基础设施，航运与石油贸易受挫。多艘靠近伊朗控制海域的船只遭袭，航运巨头MSC暂停部分海湾出口装运，巴林、卡塔尔、科威特等国能源企业宣布“不可抗力”，沙特称挫败一次针对东部油田的无人机攻击。
（来源：NDTV／通讯社）

霍尔木兹海峡约承载全球近20%的石油运输量。目前该地区冲突已显著减少海上通行，推高全球油价。

法国将派出约十余艘军舰，包括航母战斗群，赴地中海、红海并可能进入霍尔木兹海峡，以防护受中东冲突威胁的盟友。 
马克龙在塞浦路斯表示此举并在视察刚抵达东地中海的“戴高乐”号航母前，安抚上周有无人机接近的塞浦路斯。他在帕福斯会见塞浦路斯总统尼科斯·克里斯托杜利德斯和希腊总理基里亚科斯·米佐塔基斯后说：“当塞浦路斯遭到攻击，欧洲就遭到攻击。” 
随着美以伊冲突升级、海上航道受扰及油价重回百美元上方，欧洲正考虑如何保护自身利益。欧盟在该地区的主要行动是自2024年初启动、旨在保护船只免受支持哈马斯的胡塞武装袭击的“Aspides”（护盾）红海任务。米佐塔基斯表示将与欧洲伙伴一道实质性加强该行动。 
马克龙称法国将派出8艘军舰、航母编队和2艘直升机航母，行动为“纯防御、护航”性质，可在必要时包括霍尔木兹海峡，目标是保障航行自由与海上安全，并在冲突最激烈阶段过后恢复集装箱船和油轮通行。
