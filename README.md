中央纪委国家监委周二（3月24日）通报，国家金融监督管理总局副局长、前反腐负责人周良因涉嫌“严重违纪违法”正接受纪律审查和监察调查。通报所用的表述通常指向贪腐、受贿和滥用职权等问题，此案是近年来金融监管体系中最引人注目的查处之一。
周良，54岁，长期被认为与前副主席、前反腐主将王岐山关系密切。其早年曾在1990年代末任广东省政府办公厅，随后任国务院机构改革办公室、北京市政府和海南省政府等职。进入金融监管体系前，周曾任中央纪委组织部负责人，掌管反腐队伍的选拔与培训。2017年出任当时的中国银监会副主席，后随机构改革于2023年成为国家金融监督管理总局副局长。他从反腐调查员转为监管者，曾被视为推动银行业纪律制度化的关键人物。

商务部推出多项措施纾解稻农成本压力，因应中东局势推高全球生产成本。商务部内贸厅厅长维塔育•玛尼奈特表示，部长苏帕吉已指示全力援助受影响农民，重点在市场与减成本两方面着力。
启动“2569年稻谷收购吸纳计划”，目标吸纳稻谷100万吨，由仓储机构委托碾米厂和农会设点收购，收购价比市价高300铢/吨，4月1日起试点5省：那空沙旺、披集、阿育塔雅、甘烹碟、素可泰，确保丰收季有合理去化渠道。
同时铺开“稻谷集市”竞价收购，引入外地承购商直达产地，减运费、扩销路，收购价普遍高出市场200–400铢/吨，3月至5月间在8个场次举行（信武里、安东、披集、阿育塔雅、南邦等地）。内贸厅并派量器监管人员现场查秤重、含水率与杂质扣除，确保交易公平透明。
在减成本方面推“绿色旗帜Plus”肥料补助：化肥每袋补200铢、每户5袋共计1,000铢；持“良土卡”者额外200铢，另发200铢有机肥券，单户最高获1,400铢补助。内贸厅亦将紧急与能源部协调确保收割期柴油供给，减轻农机燃料负担。
农民与农会代表对政府措施表示肯定，称此举可缓解稻农担忧。商务部表示将持续跟进并维护农产品价格与农民收入稳定。

交通部预计，2569年宋干节（10天）期间出行将大幅增加，总流量约为公路与高速公路合计10.65万（应为百万）次、收费公路（高架/快速道）15.89万次，合计约26.5万（应为百万）次；公共运输人次预计达18.58万（应为百万）人次，其中曼谷及周边11.50万人次、跨省3.45万人次，分别较去年增约10%与14%。交通部设定目标：道路事故、伤亡与伤者较去年至少减少5%。
为应对中东战争对能源影响，交通部要求PTT在全国客运站把油罐加满，保证宋干期间不缺油，并承诺公共运输票价不会上调；同时推动B20柴油尽快上市，定价比B7低约每升5铢，目标日产量约1600万升以降低运输成本。
假期免收部分路段通行费：10–16日免收东部快速路与阐昭干纳披色（บางพลี–สุขสวัสดิ์）等7天；13–15日免收乌敦-拉特雅与贺莱玛哈纳空（บางสาย）等3天；并免收国道局管辖的M7（曼谷–芭堤雅）、M9环线部分与M81（บางใหญ่–甘加那布里）等三条高速公路通行费。国道局还将24小时开放M6（บางปะอิน–นครราชสีมา）全线196公里，按10–13日作为外返曼谷方向、14–19日作为进曼谷方向分段通行；M6工程进度99.01%，剩余两段预计于今年下半年与年底前完成，目标在2570年新年前全线通车。
同时将暂停泰国拉玛2路段施工并恢复路面（10–16日），恢复至来回各3车道；开放M82段（บางขุนเทียน–เอกชัย）10公里。交通警察将联合实时监控路况并发布信息，提醒民众加油备足。
航空方面，泰国航空导航公司同意对191班有票价优惠的国内航班于10–15日下调航行服务费30%，以鼓励增班，预计宋干期间全国航班约25,620班次（较去年增1.8%），但仍将密切关注战事影响。
