中国称日菲海域划界谈判构成国际不法行为
中国就日本与菲律宾即将举行的海域划界谈判向两国发出法律警告，指该谈判构成“国际不法行为”。此举凸显北京正加大力度向华盛顿的这两个关键区域盟友施压，以应对日菲试图抗衡中国在海上强势姿态的举措。
中国自然资源部下属机构“中国海洋发展战略研究所”星期四（7月2日）发布《关于日菲宣布启动双边海域划界的法律评论》。报告指出，日菲谈判未与中国协商，无视区域特定地理情况，违背国际法上主权平等、善意原则以及合作与克制义务，构成国际不法行为。
报告敦促日菲立即停止推进双边海域划界谈判，积极与中国展开协商。
报告强调，“台湾是中国领土不可分割的一部分，这牵涉中国依据包括《联合国海洋法公约》在内的国际法，在台湾岛以东主张的大面积专属经济区和大陆架。”
报告说，日菲不得借所谓“划界”挑战一个中国原则，不得以任何形式将台湾当局引入所谓“划界”进程，否则将是对一中原则的严重挑衅。
报告还警告各国不应为日菲提供协助，远离国家责任和政治风险。“相关国家应充分评估参与此类行动或承认其合法性的法律后果，不介入、不支持、不承认日菲非法划界进程。”
彭博社分析指出，此举表明，北京对马尼拉和东京决定就台湾以东海域展开海域边界谈判日益不满。自日菲宣布将启动海域划界谈判以来，中国大陆已多次派遣海警船赴该海域巡逻，并派出测量船进行海底测绘。
近期，中国大陆还多次派遣海警船在台湾以东、靠近日本与那国岛以南的海域宣示主权。对此，日本已多次通过外交渠道提出抗议。

综合中国天气网和海南省气象局消息，南中国海热带低压已于星期五（7月3日）凌晨加强为今年第10号台风“美莎克”（热带风暴级；英文名称：Maysak；名字来源：柬埔寨；名称意义：一种树）。
上午5时，“美莎克”中心位于海南省陵水县南偏东约155公里的南中国海中西部海面，中心附近最大风力有八级（每秒18米）。
“美莎克”预计将以每小时10至15公里的速度向西北方向移动，强度还将有所加强，将于星期五白天在海南三亚到琼海一带沿海登陆（每秒23至25米，九至10级，热带风暴级或强热带风暴级），穿过海南岛后移入北部湾，并将于星期六（4日）晚上在广西沿海和越南北部沿海交界附近再次登陆（每秒20至23米，八至九级，热带风暴级），以后强度逐渐减弱。
海南省气象局星期五上午6时30分提升台风四级预警为台风三级预警。
受“美莎克”影响，琼中、万宁、儋州、白沙、昌江、东方、乐东、五指山、保亭、陵水和三亚等市县星期五至六的累积雨量150至250毫米、局地350毫米以上，其余市县100至150毫米。此外，海南岛四周海面和北部湾海面风力逐渐增大，琼州海峡最大阵风九至10级，其余海区可达10至12级。

日本一名官员星期四（7月2日）告诉法新社，由于近期熊袭击事件激增，日本已开始在北部山区安装数百个摄像头。
法新社报道，据环境省统计，自4月1日以来，至少有5人被熊袭击身亡，全部发生在东北地区。此前，上一财年全国共发生13起致命的熊袭击事件，创下历史新高。 
近年来，熊的数量有所增加，科学家认为部分原因是农村人口减少。
环境省官员高桥悠告诉法新社，为了制定有效的应对措施，当局计划使用800多个摄像头来监测熊的活动，初期将重点关注东北地区的六个主要熊群栖息地，然后在未来四年内将调查范围扩大到全国。
环境省会在人头高度放置装有蜂蜜和葡萄酒混合物的罐子，以吸引熊。届时，摄像机将拍摄每只熊用后腿站立嗅闻诱饵时胸前独特的白色斑纹。
“此前，地方政府对辖区内的熊群数量进行调查的时间和方法各不相同。我们的计划是聚焦熊群，进行更精准的调查。”
近几个月来，随着熊结束冬眠，熊的目击事件激增，越来越多的熊闯入城市。
本周，当局表示正在调查一起可能为熊袭击致死的事件，一名男子的尸体在东北山区被发现，身上有熊咬痕。
当局建议民众避免独自前往山区，并在背包上系上铃铛，随身携带防熊喷雾。
今年6月，数十名警察、猎人和市政官员花了四天时间才将一头在东京以北的宇都宫市游荡的熊捕获，当地多所学校因此停课。 
在此之前，另一头被描述为“极其聪明”的、会打开窗户和水龙头的熊，在福岛两家工厂袭击四人，并在逃亡数日后再次出现。

官媒评泰山景区回应铁丝网争议：事实没摆全道理没讲清
中国最高人民检察院机关报《检察日报》评论泰山景区135公里铁丝网事件指出，景区给出的回应“事实没摆全，道理没讲清”，难以消除公众疑虑。
近日，山东泰山景区周边设置长达135公里的滚筒式刀片刺绳隔离铁丝网，将非游览区域层层围堵。景区回应称，相关设施位于非开放区域，与正常游览路线不交叉不重叠，并已预留生物迁徙和消防救援通道。
《检察日报》星期五（7月3日）发表题为《千年泰山插入135公里刀片刺绳，项目是否科学要经得起公众追问》的评论文章，指出泰山作为公共景观资源，对其管理决策的科学性论证不能只停留在文件里，而应该让公众看得见、信得过，经得起公众拷问、经得起时间检验。
文章指出，虽然景区对舆论质疑的回应不算慢，但仅凭其核心信息——铁丝网一是为了防火护林而建，二是不影响游客的正常游览，并不能证明这个工程是合理的。
135公里刀片刺绳、2000多万元（人民币，380万新元）资金投入，工程建设的必要性、科学性在回应中并未充分体现。症结没点透，公众的疑虑自然消不了，舆论风波也就无法平息。
文章续指，公众的质疑并非没有道理。泰山是世界文化与自然双重遗产，生态良好，兼具文化资源和人文价值。
文章接连发问：一道135公里的刀片刺绳网扎进山林，是保护大于破坏，还是破坏大于保护？动物迁徙遇到铁丝网，会不会被阻拦或受伤？一旦发生山火或者人员遇险的紧急情况，隔离设施会不会阻碍消防救援人员与设备通行？
此外，从视觉效果而论，生硬冰冷的刀片刺绳网与泰山的气质格格不入，让不少人觉得破坏了景观的整体风貌。所以，文章认为，公众的担忧并不是凭空想象，而是合理质疑。
对于景区回应，文章指出，如果当初的立项经过了充分的可行性研究、环境影响评估和安全风险研判，那么面对公众的追问，景区大可以用翔实的数据、专业的论证来讲明道理，让公众清楚地看到决策的科学性和合理性，眼见为实、心服口服。但目前的回应恰恰缺少这些内容。
文章进一步指出，公众对一项公共决策是否科学的追问，恰恰是与法治政府建设同向的。假如景区管理部门拿不出科学决策的充分依据，甚至被证明存在决策不科学、不合规的问题，就要依法依规承担相应的责任。
“这也表明，公共决策不是关起门来的事，花的是公共财政，就必须恪守法律法规要求，经得起公众的审视。”文章最后强调，用科学的论证说服人，用坦诚的态度打动人，这是管理者的本分，也是该有的担当。
