特朗普绕开“紧急权力”对全球征10%税 关税博弈步入更长不确定期
著名贸易专家、瑞士洛桑国际管理发展学院教授埃弗内特（Simon J. Evenett）在最新分析报告中指出，美国最高法院取消《国际紧急经济权力法》（IEEPA）关税后，特朗普立即宣布援引《1974年贸易法》第122条开征10%基准关税，并开启一系列针对性调查，将使得全球对美出口商所面临的关税不确定期延长。 
埃弗内特分析指出，取消IEEPA关税后，中国对美出口商实际关税率将从36.8%降至21.2%，降幅达15.6个百分点，巴西、印度、印度尼西亚和越南等主要贸易伙伴的税率降幅将超过10个百分点。
部分被单独施加高额IEEPA关税的小型经济体，包括缅甸、老挝、波斯尼亚和黑塞哥维那、摩尔多瓦、突尼斯及塞尔维亚等，降幅将达20个百分点以上。
埃弗内特指出，若中国输美商品的实际关税率确实大幅下调超过10个百分点，此前备受关注的“中国出口向美国以外市场转移”的势头，势必将显著减弱。
不过，特朗普周五（2月20日）宣布援引《1974年贸易法》第122条开征10%基准关税，实际上抹平了部分降幅。
埃弗内特说，对于此前关税率高于10%的国家，短期内负担有所减轻；但对于原本关税较低的国家，竞争优势将被削弱，例如在欧洲，英国出口商相对于欧洲大陆出口商享有的关税优势将被削弱。
尽管基于第122条开征的关税，有效期仅为150天，而且第122条关税也可能面临法律挑战，但在150天窗口期内，这类法律挑战极不可能提交到美国最高法院作出最终裁决。因此，各国出口商可合理预期，10%税率在这期间很大可能性将维持不变。
埃弗内特说，150天后的关税走向，将取决于特朗普政府正在启动的《1974年贸易法》第301条及其他调查的范围与结论。目前尚不清楚哪些国家或行业将成为优先调查对象，但这种不确定性本身就具有重大意义。 
埃弗内特分析认为，特朗普政府目前的策略是“以不确定性换取筹码”。这种不确定性被用来迫使外国政府作出让步，并鼓励企业考虑将生产迁往美国。这种“关税工具”短期内不太可能退出历史舞台。
埃弗内特预计，美国周五发布的一系列宣布所引发的不确定性，很可能持续到2026年年底。

香港大埔宏福苑去年11月发生五级大火，造成168人遇难，数千居民受灾。港财政司副司长黄伟纶星期六下午3时举行记者会，公布大埔宏福苑长远居住安排方案。房屋局局长何永贤和房屋局副秘书长章景星也出席。
据香港01、香港电台等港媒报道，黄伟纶公布，特区政府计划用现金或以楼换楼的方式收购宏福苑业主业权，这是最快的处理方案。
特区政府提出的收购业权平均尺价为8000港元（未补地价）及10500港元（已补地价），总收购成本约68亿港元。在收购业权完成后，特区政府计划将宏福苑7座受灾楼宇拆卸改建为公园等设施。
黄伟纶表示，早前收集受影响业主对不同长远居住安排的初步意愿，共有1975户提交意见，超过八成认为速度要快，另外四成七希望原区安置、亦有四成七希望照顾到全体宏福苑业主。
黄伟纶说，在1975户中，七成四、即1458户表示会考虑政府向他们收购业权，另外一成四表示未有决定、一成二表示不会接受。他又说，在愿意考虑接受政府收购业权的1458户业主中，四成一接受现金或楼换楼方式收购、三成九接受以楼换楼收购、两成只接受现金收购。
黄伟纶说，在这1458户业主中，有八成一表示，需要政府提供资助房屋作长远居所。

中国常驻联合国代表傅聪在改革安理会会议上发言时说，需要寻找方法改革安理会，提升发展中国家的代表权和发言权。
据中国常驻联合国代表团官网消息，傅聪当地时间星期五（2月20日）在第80届联大安理会改革政府间谈判会议上发言时，作出以上表述。
他说，当今世界，多极化潮流不可逆转，全球南方群体性崛起不可阻挡。各方的普遍共识是，要顺应历史大势，以实际行动维护安理会权威，推动对安理会进行必要、合理的改革，使安理会的决策更民主、高效，切实反映国际社会的集体利益，充分保障更多国家实质性参与。
