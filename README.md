字节跳动旗下AI聊天机器人农历新年假期斩获1亿用户
一项私人调查显示，字节跳动旗下的豆包聊天机器人，在中国农历新年假期期间吸引超过1亿名日活跃用户，在这场中国科技巨头斥资数十亿美元争夺用户的人工智能（AI）竞赛中胜出。
据路透社报道，农历新年是中国持续时间最长、最繁忙的假期，从2月15日正式开始，为期九天。这一时段也已成为中国科技公司集中展开营销攻势、争夺市场关注并推广消费级AI产品最关键的时期之一
路透社引述AI产品榜（AICPB.com）星期三（2月25日）公布的数据报道，豆包应用程序（App）在大年除夕2月16日的日活跃用户突破1亿，约为2月初水平的四倍。AI产品榜是一家追踪中国AI聊天机器人表现的网站。
豆包App的增长很可能受益于与中国央视春晚的合作，春晚是中国收视率最高的电视节目之一，在2月16日播出。根据字节跳动，豆宝在春晚播出期间处理了超过19亿条与AI相关的用户查询。
尽管阿里巴巴斥资30亿元（人民币，下同，5.51亿新元）推广旗下“千问”App，包括补贴用户通过App下单购买珍珠奶茶等商品，但调查显示，“千问”的日活跃用户数在除夕夜仅触及3000万，在此次调查的三款聊天机器人中排名垫底。数据显示，2月初“千问”的日活跃用户数仍不足1000万。
调查也显示，腾讯通过斥资10亿元推出优惠券赠送活动，推广旗下“元宝”聊天机器人；这推动了“元宝”App的日活跃用户数从2月初的2000万稳步攀升，并在2月16日触及5000万的峰值。
不过，这三款App在高峰过后日活跃用户数都出现回落，其中“元宝”在推广活动结束后跌幅最为明显。相比之下，“千问”的下滑幅度最小，截至2月21日仍维持约2200万用户。
《证券日报》引述中国电子商务专家服务中心副主任郭涛说，纯靠红包驱动的用户，7日留存率普遍低于20%，30日留存率甚至可能跌破5%。红包只能解决“用户为什么用”的问题，却无法回答“为什么持续用”。
网经社电子商务研究中心数字生活分析师陈礼腾也称，数据暴涨并非取得胜利，要看节后用户的留存质量与行为转化深度。农历新年期间的每一分投入，买的都不仅是流量，更是用户习惯的培育契机。
《证券日报》根据各AI厂商公开数据统计，多家互联网巨头在今年农历新年档合计投入或超过80亿元。

去年中国创下1.19万亿美元的贸易顺差纪录，引发外界担忧，认为这使得国际贸易难以为继，其他国家可向中国出口的商品变少，或不愿再出口。批评者指向中国在生产更好、更便宜商品方面的竞争优势。
对此，北京更多将其视为对长期制造业战略的肯定。扶持以技术为驱动的中小企业，成为增强中国全球制造地位的重要一环。自2011年提出定向支持中小企业以来，相关部门建立了分层识别与扶持体系，筛选优质企业并给予定向支持。
其中两类层级尤为重要：被称为“专精特新”（即“专、精、特、新”）的企业，以及其内部的““小巨人””企业。此类企业在细分领域依靠工艺经验、质量提升与创新取胜；“小巨人”则是技术能力、市场竞争力和成长潜力最强的领军者。
在中国第十四个五年规划期（2021—2025年）内，专精特新企业数量由不足4万增至14万余家，“小巨人”由约5,000家增至17,600家。2019—2022年间获评的“小巨人”中约75%涉足“中国制造2025”确定的核心领域。2024年，“小巨人”平均研发投入超过3,000万元人民币（约合434万美元），研发投入强度为7%。
