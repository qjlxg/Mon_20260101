法国通过法案 禁止15岁以下未成年人使用社媒
法国议会通过一项法案，自2027年1月起，禁止15岁以下未成年人使用社媒。法国因此成为首个禁止未成年人使用社交媒体的欧洲国家。
英国广播公司（BBC）报道，尽管遭到一些左翼人士的批评，法国参议院（上议院）和国民议会（下议院）星期二（7月21日）仍一致通过禁止15岁以下者使用社媒的法案。
法国总统马克龙对此表示欢迎，他此前承诺推出这项法案，作为他结束10年任期的标志性政策。
尽管有人对这项措施的可行性表示怀疑，但法国政府仍然坚信用于落实年龄核查的在线工具是安全和有效的。
新法将分两个阶段实施：从9月起，15岁以下未成年人将无法注册社媒账户，所有新注册账户必须进行年龄验证。
从2027年1月起，新规定将适用于所有现有的社媒账户。这表示，法国所有社媒用户都必须证明自己已年满15岁才能使用社媒。
法国是继澳大利亚之后，第二个实施此类禁令的国家。因担忧社媒危害儿童心理健康，英国和欧盟也在拟定相关的限制措施。
