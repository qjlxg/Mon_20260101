特朗普携英伟达CEO黄仁勋赴华，盼习近平“向美开放”
路透北京/纽约5月13日电 — 美国总统唐纳德·特朗普5月13日启程对华进行近十年来首位美国总统访问，并表示将敦促中国国家主席习近平“向美开放”以利美企在华发展。特朗普在社交平台表示，这是他此次行程的“首要请求”。
随行CEO主要来自有意解决在华经营问题的企业，英伟达因在中国获批销售其H200人工智能芯片遇阻。获悉内情者称，特朗普在阿拉斯加中途加油时临时邀请黄仁勋登机，白宫记者见其登上空军一号。
特朗普将于5月14日至15日与习近平会谈，日程包括人民大会堂招待、天坛参观与国宴。议题除贸易外还涵盖伊朗战争、核武与对台军售——一项价值约140亿美元的对台军售方案仍待特朗普批准，中国对此坚决反对。
此同时，首席贸易谈判代表斯科特·贝森特在韩国仁川机场与中国副总理何立峰举行约三小时会谈，旨在维持2025年10月达成的脆弱贸易休战（特朗普暂停对中高额关税，中方放弃限制稀土供应）。双方预计就便利贸易投资建立对话机制；美方力推波音、农产与能源出口，中方则希望放宽芯片制造设备与先进半导体出口管制。
分析称，法院已限制特朗普随意加征关税的余地，且伊朗冲突加剧美国通胀并增加共和党在中期选举面临的政治风险，因此美国方面对达成成果的需求更迫切。北京方面压力相对较小。中国官民对访问既抱希望亦存疑虑。来源：路透。

根据独立研究机构IEEFA于5月13日发布的报告，2026年第一季度欧盟从俄罗斯进口的液化天然气（LNG）达到了自2022年俄乌战争以来的最高水平。
报告显示，欧盟在2026年前三个月自俄进口天然气约69亿立方米，较2025年同期增长16%；IEEFA并称4月份进口同比亦增长17%。尽管通过管道的俄罗斯天然气几近停止，但法国、西班牙和比利时等国仍通过油轮大量采购俄罗斯LNG。
俄罗斯目前约占欧盟LNG供应的14%，为第二大供应国。与此同时，欧盟对美国LNG的依赖显著上升：2021至2025年间美国供应量增逾三倍，2026年一季度同比增长27%。IEEFA预测美国将在2026年成为欧盟最大气源，至2028年其LNG占比或高达80%，但也指出美国天然气对欧洲买家而言最为昂贵。
IEEFA欧洲首席分析师安娜·玛丽亚·雅勒-马卡列维奇表示，欧盟从管道气向LNG的转变本意在于保障供应与实现多元化，但中东战争造成的中断和对美LNG的过度依赖已暴露出此举的双重失败。欧盟曾承诺因俄乌战争在2027年底前停止进口俄罗斯天然气和石油。消息来源：IEEFA、法新社。
