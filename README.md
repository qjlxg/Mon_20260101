香港家办两年增长25%
香港官方公布数据显示，香港的单一家族办公室数目已经超过3380家，增长超过25%。
据香港政府新闻公报，香港财经事务及库务局与投资推广署星期二（2月10日）联合宣布，根据投资推广署委托德勤进行的《香港家族办公室市场研究》显示，截至2025年年底，香港的单一家办超过3380间，两年间增加约680间，升幅超过25%。
香港财库局局长许正宇表示：“香港是全球资产及财富管理的领先枢纽，家香港家办的数目持续增长，反映政府在政策推动和制度建设上的努力已取得实质成果”。
许正宇说，香港已计划于今年上半年提交立法建议，扩展有关基金及单一家办的优惠税制合资格投资范围，涵盖例如贵金属、贷款及私募债权投资和数字资产。“在各项政策措施支持下，我们有信心达到《行政长官2025年施政报告》所订下、在2026年至2028年年底前协助逾220间家办来港落户或扩展业务的新目标。”
投资推广署署长刘凯旋表示，家办专责团队过去在欧洲及东南亚等地进行推广时，了解到不少海外家办对香港的制度优势及税务优惠有浓厚兴趣。“香港提供高度灵活的投资环境，税务优惠制度对投资地点并不设限，家办可透过香港投资全球资产。同时，单一家办在港一般无需申请牌照，有助于维持高度的私隐性，这些都是海外家办尤其重视的关键考量。”

莱昂南的言论正值参议院议长文森特·“蒂托”·索托（Vicente Sotto III）批评最高法院一项裁决之际。该裁决认定针对副总统萨拉·杜特尔特（Sara Duterte）的弹劾案因宪法一年禁制及正当程序问题而不合宪，索托曾主张修宪可回应该裁决对立法机关的影响。
莱昂南并提醒检察官，公开发布的证据未必在法庭上站得住脚，过早公布只换来声名，真正的问责在于能否达到“排除合理怀疑”的定罪标准。
联合国毒品与犯罪问题办事处（UNODC）代表Kirbee Tibayan在会上指出，菲律宾亟需《举报人保护法》，现行改革（包括新《政府采购法》）仍存在法律空档。她说，大多数揭露腐败的信息来自举报人，但举报人保护不足，建议放宽银行保密、并通过涵盖所有政府部门的资讯自由法，以利调查与取证。
