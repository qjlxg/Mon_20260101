乌克兰总统泽连斯基在Telegram确认这是美方斡旋下的成果，称人道事务仍是优先事项，并在带有玩笑意味的命令中“允许”俄罗斯在5月9日举行红场阅兵，承诺不以乌克兰武器攻击红场。克里姆林宫助理乌沙科夫与莫斯科市长也表示已就此达成一致，后者称防空部门在当晚约七小时内拦截了飞向首都的乌克兰无人机。
尽管短期停火达成，双方此前互相指责对方违反各自宣布的停火，战斗仍以导弹、无人机和火炮继续，没有实现根本停战或谈判突破。克里姆林宫曾为胜利日单方面宣布两天停火，基辅则要求提前并无限期停火未获采纳。另据亲乌地图，俄方目前控制约19.4%的乌克兰领土，今年前四个月仅扩张约700平方公里。

莫斯科 — 5月9日，因担忧来自乌克兰的攻击，俄罗斯在红场举行了近年来最为简化的胜利日阅兵。该节日纪念苏联战胜纳粹德国并缅怀约2700万在二战中遇难者。
阅兵未见坦克或重型军车驶过广场，导弹、核潜艇、激光武器、苏-57战机、S-500防空系统及无人机和火炮等武器通过巨幅屏幕和国家电视播出，其中包括“亚尔斯”洲际弹道导弹和“阿尔汉格尔斯克”核潜艇。部分曾在乌克兰服役的官兵列队行进，朝鲜部队也出现在队伍中。普京与退伍老兵并坐列宁墓前，发表约八分钟讲话，称乌克兰战争终将取得胜利，并指责“整个北约集团”支持的侵略力量。
美国前总统特朗普宣布自5月9日至11日停火三天、并支持交换1000名俘虏，他表示希望大幅延长停火。此前俄乌互相指责打破各自单方面停火；停火期间未有双方报告违约。
为防扰乱阅兵，俄方曾警告若有企图将对基辅发动大规模导弹打击，并建议驻俄外交使团在攻击时撤离在基辅的工作人员。泽连斯基以带有戏谑意味的法令“准许”俄方阅兵进行，称乌克兰武器不会瞄准红场。莫斯科安保严密，市中心和周边道路被封闭。
在历史和现实的交叠下，莫斯科对战争走向深感焦虑：冲突已造成数十万死伤、广泛破坏并耗尽俄罗斯约3万亿美元经济实力，使俄欧关系陷入冷战以来的最低点。批评者如被监禁的民族主义者伊戈尔·吉尔金抨击克里姆林宫的战争指挥，克里姆林宫发言人佩斯科夫否认有关俄罗斯内部政变或刺杀普京的报道。安全会议秘书、被指为潜在政变人物的谢尔盖·绍伊古出席了5月8日的视频安全会议并在阅兵中公开露面。

华盛顿 — 一批高度拟真、外形光鲜的AI“网红”在社交平台上为特朗普造势，尤其在美国中期选举前夕大量出现。这些虚拟人物常穿比基尼或军装，高调称赞特朗普、重申“美国优先”等口号，并抨击其政敌，出现在TikTok、Instagram和Facebook等平台上。
相关研究机构与媒体近月追踪到数百个此类账号，普渡大学治理与负责任AI实验室（GRAIL）和非营利组织CivAI警告，这类细分、拟人化的AI内容可能被用来塑造舆论，且越来越难以识别。硅谷投资者安德森·霍洛维茨的AI合伙人贾斯汀·摩尔亦注意到大量男性用户追随明显为AI的“网红”。
今年早些时候，特朗普本人在Truth Social发布过一段铂金金发AI头像的视频，借此散布针对加州州长加文·纽瑟姆的未经证实指控。与之相对的还有反特朗普的AI账号，双方在选举周期常以虚假信息博取关注。
研究者指出，AI深度伪造技术已被用于针对全球领导人的宣传，可能影响投票率并扭曲地缘政治事件。部分亲特朗普的AI账号同时带有商业化目的，主页连结到售价数百美元的MAGA周边。普渡大学助理教授丹尼尔·希夫表示，随着选举临近，生成性AI驱动的政治信息将会增加，而“经济利益”往往是制造这些合成网红的主要动因。AFP

伊朗称已就美方提出的停火谈判方案向调解方巴基斯坦递交了回应，伊朗官方通讯社IRNA于5月10日报道称，回应目前将集中于结束战争，但未公布细节。当天，一艘卡塔尔LNG船“Al Kharaitiyat”经霍尔木兹海峡安全通行，正驶往巴基斯坦卡西姆港；这是自2月28日冲突爆发以来首艘经该海峡的卡塔尔液化气船。 
尽管停火已维持一个月，但5月10日多国侦测到敌对无人机，显示区域威胁未消。阿联酋称拦截两架来自伊朗的无人机，卡塔尔谴责一艘来自阿布扎比的货船在其水域遭到无人机袭击，科威特亦表示防空系统应对了进入领空的敌对无人机。英国海事贸易行动处报告，多哈东北一艘干散货船被不明弹体击中引发小火，无人员伤亡。 
德黑兰大体封锁了非伊朗船只通过该狭窄海峡，卡塔尔和巴基斯坦作为调停方获批过航以示信任。卡塔尔首相在迈阿密与美国国务卿通话并告知伊朗外长，勿将霍尔木兹海峡作为“施压工具”。 
此外报道提及美军曾击中两艘试图进入伊朗港口的涉伊船只；伊朗议员拟立法管控海峡通行，对“敌对国家”船只禁航。华盛顿方面关于封锁效果的情报评估引发争议，特朗普称尽管有冲突升级，停火总体仍在维持。北约盟友在无全面和约与国际授权的情况下拒绝单方面派舰开通海峡，英国则表示将向中东派遣军舰，为可能的多国行动做准备。

欧洲疾病预防控制中心（ECDC）表示，作为预防措施，因发生致命汉坦病毒疫情而受影响的邮轮上所有乘客在下船时都被视为高风险接触者。该邮轮预计于5月10日在西班牙特内里费岛外停泊，ECDC在5月9日的加急科学建议中称，无症状乘客将由各国通过专门安排的交通工具（而非普通商业航班）遣返并自行隔离。
各国正准备在格林威治标准时间（GMT）约6:30至7:00（新加坡时间14:30至15:00）开始撤离MV Hondius上的本国公民。世界卫生组织（WHO）5月8日通报，已有8人患病，其中3人死亡（包括一对荷兰夫妇和一名德国公民）；WHO称6例已确诊感染，另有2例为疑似病例。ECDC指出，虽在下船时均列为高风险，但返国后并不一定全部被认定为高风险个案。
在邮轮预计停靠的特内里费格拉纳迪利亚·德·阿博纳港已设立移动医疗点。ECDC敦促对有症状者在抵达时优先进行医学评估和检测，根据病情可能在岛上隔离或医疗转运回国。汉坦病毒通常由啮齿动物传播，极少情况下可人传人，卫生部门认为病毒进一步扩散的风险较低。
