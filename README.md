国际刑事法院检方反对杜特尔特放弃出庭请求
ABS-CBN记者报道，国际刑事法院（ICC）检察院在一份经涂黑的回应文件（2026年2月19日）中，正式反对菲律宾前总统罗德里戈·杜特尔特请求免于亲自出庭的申请。检方称，“没有合理理由”允许杜特尔特不亲自出席确认听证，并指出他有能力到庭。
针对杜特尔特自述“年老、疲惫、虚弱”的说法，检方引述专家小组结论，认为其“身体与精神上均适合、能胜任”出庭。独立医疗证据已在庭上审理，法庭也已采取安排以确保被告在听证时的舒适。
检方批评杜特尔特对自身健康的叙述不可靠，并指称其此前抱怨“十个月未在庭上露面”与现在突变为回避公开视野自相矛盾。检方强调，被告出庭不仅是程序性要求，也是受害者的重要里程碑，公开可见地为指控答辩很重要。
法庭关注杜特尔特甚至表示不会通过视像连线旁听，认为这更像是对法庭“不尊重”而非真实健康限制。审前庭将决定是否存在“充分理由”允许在被告缺席的情况下继续听证；若采纳检方意见，可能要求杜特尔特下周亲自到庭。
在围绕涂黑的医疗报告与“视觉空间能力”的争辩中，外界关注这位曾任菲律宾最高领导人的被告是否最终将在海牙法庭面对指控人。
