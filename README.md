随着台风“巴威”逐渐逼近中国浙闽沿海，上海两大机场星期六（7月11日）计划取消近400架次航班，杭州机场也将取消198架次航班，浙江多座机场则暂停当天所有航班。
综合央视新闻和第一财经报道，中国中央气象台星期六上午10时继续发布台风橙色预警，并预计“巴威”将于星期天（12日）凌晨在浙江三门至苍南一带沿海登陆，登陆后向西北转偏北方向移动，强度逐渐减弱。

此外，长三角铁路部门启动防台防汛应急响应，对多条线路列车采取临时停运措施。星期六至下星期二（14日），包括杭深、京沪、沪昆、合福等高铁及多条区域铁路，部分区段、部分时段列车临时停运。

针对中国男子篮球职业联赛（CBA）上海队外籍球员被检出兴奋剂事件，中国篮球协会发布声明，表示对任何形式的兴奋剂违规行为持“零容忍”态度，将对违规方进行严肃处理。
中国反兴奋剂中心星期五（7月10日）在网站公开，CBA上海队外籍球员哈桑·怀特塞德（Hassan Whiteside）在5月6日进行比赛中被检测出禁用物质氧雄龙代谢物和氯米芬，球员放弃B瓶检测。
中国篮球协会星期六（11日）在官方微博发布声明表示，协会对此高度重视，正按照相关程序，积极配合中国反兴奋剂中心开展调查。待中国反兴奋剂中心做出最终认定后，中国篮协将严格依据反兴奋剂规则及协会相关规定，对违规方进行严肃处理。
中国篮协在声明中强调，对任何形式的兴奋剂违规行为持“零容忍”态度。协会将以此为戒，持续强化各参赛单位及从业人员的红线意识与反兴奋剂教育管理工作，坚决维护公平公正、健康积极的竞赛环境。
据悉，怀特塞德被查出兴奋剂违规的比赛是2025-2026赛季CBA季后赛上海队主场对阵山东队的首场较量，而他此后还参加了上海队与山东队8进4系列赛的第二场比赛，以及与北京队的所有半决赛系列赛。总决赛期间，怀特塞德缺席上海队与广厦队的系列赛，最终上海队以总比分4:1战胜广厦队，夺得本赛季CBA联赛总冠军。
《北京日报》指出，一位兴奋剂检测违规的球员仍然能出场比赛，这种情况绝对是不公正、不合理的，对北京队、山东队都是不公平的。上海队进入总决赛后，怀特塞德没有再登场。对于处罚结果为何迟迟公布，以及怀特塞德被查出使用兴奋剂后为何还能参赛，其所参加的五场比赛是否有效，中国反兴奋剂中心、CBA公司目前都没有表态。

英国前国务部长家中遇害 警列谋杀案调查
英国前保守党议员维德科姆被发现倒毙在英格兰西南部家中，身上有严重伤势。警方已按谋杀案展开调查。
局英国广播公司（BBC）和彭博社星期六（7月11日）报道，警方星期四（9日）早上接到救护车服务部门报警后，前往维德科姆（Ann Widdecombe）位于德文郡海托尔的住家调查，在屋内发现维德科姆已经死亡。

维德科姆在遇害前一天曾接受媒体采访。
警方也说，现阶段没有迹象显示案件与恐怖主义或政治动机有关。内政部长马哈茂德敦促公众不要妄加猜测。

综合台湾三立新闻网、《中国时报》《联合报》等报道，台湾食用油龙头中联油脂公司生产的大豆沙拉油检出致癌物“苯并芘”超标四倍，冲击全台食安。

台中地方法院合议庭在评估涉案情节与相关证据保全状况后，于星期五晚上近11时决定准予余凌冲以2000万元重金保释，并要求他限制住居、限制出境、出海，向警方定期报到。

台中地检署认为，余凌冲有羁押的原因与必要，已当庭依法提起抗告。
台中地检署侦办上述案件，星期五依涉嫌违反《食品安全卫生管理法》将中联油脂、福寿实业和福懋油脂等四大油品公司董事长和公司高层共11人传唤到案。
复讯后，检方谕令中联董事长蔡清松以2000万元交保；福寿实业董事长洪尧昆、福懋油脂董事长吴星澄和泰山企业董事长刘伟龙各以1000万元交保，吴星澄因一时筹不出交保金，改以200万交保；其余被告以600万元至50万元交保。

国际能源署：美伊冲突再升温 重建石油库存进程或受扰
国际能源署（IEA）指出，美国与伊朗敌对行动再度升温，可能扰乱今年晚些时候全球重建石油库存的进程。这也再次凸显如果冲突持续升级，全球经济所面临的风险。
据彭博社报道，国际能源署星期五（7月10日）发布月度报告说，美国与伊朗于上个月达成临时和平协议后，全球石油供需关系一度开始恢复正常。IEA称，在阿联酋创纪录产量的带动下，波斯湾地区原油运输量已恢复至正常水平的约三分之二，全球原油库存也出现四个月来的首次增长。

苹果告OpenAI：窃取商业机密图利自身硬件研发
（硅谷／旧金山综合电）苹果起诉OpenAI和两名前员工盗取商业机密，获取了有益于它进军消费硬件市场的信息，掀开一场可能影响谁将主导未来人工智能（AI）器材市场的法律诉讼战。
苹果星期五（7月11日）在北加利福尼亚州联邦地区法院起诉OpenAI策动广泛的行动，系统性地通过苹果的前员工、聘雇方式和供应商关系，获取苹果的机密信息，协助推进自己的消费硬件业务。
被起诉的两名前员工当中，陈堂耀（Tang Yew Tan，音译）曾担任iPhone和苹果手表的产品设计副总裁，目前是OpenAI的硬件主管；刘昌（Chang Liu，音译）曾任苹果高级系统电子工程师，今年1月加入OpenAI。
诉状指陈堂耀在离任前将苹果供应商和内部工业摘要电邮给自己，“有条理地使用苹果的机密信息使OpenAI受益”。刘昌则没有将工作笔电归还公司，过后还使用一个授权漏洞进入苹果内部网络，下载了“数十个机密硬件相关文件”。
针对苹果提起诉讼，OpenAI表示“对其他公司的商业机密没有兴趣，公司继续专注打造可赋能所有人的创新科技”。
苹果的诉状还包括OpenAI属下的硬件公司io Products。这家公司由苹果前首席设计师艾维（Jony Ive）创立，OpenAI去年以65亿美元（约84亿新元）收购，以跨足硬件业务。目前，OpenAI有400多名苹果前员工。
业界分析人士相信，OpenAI是在研发一款结合AI功能的手机或其他器材；若取得成功，将摊薄iPhone的市场份额，因此激化了双方的竞争关系。
科技咨询公司PP Foresight的分析师佩斯卡托雷（Paolo Pescatore）说：“对苹果来说，OpenAI已逐渐从伙伴变成对手，OpenAI则试图降低对苹果的依赖，直接面向客户。就算诉讼不成立，也会拖慢OpenAI发展硬件的计划，让双方的关系更加脆弱。”

古巴本周第二次全国断电
由于美国实施石油封锁，加上发电系统老旧，古巴国家电网于星期五（7月10日）再次瘫痪。这是古巴本周第二次发生全国性停电，也是今年以来第四次。
据路透社报道，古巴能源部长莱维在社交媒体发文说：“我们正在努力恢复国家电力系统。在我们每天面临的诸多困难中，这是一个尤为复杂的问题。”
在星期五电网瘫痪之前，古巴星期一（6日）已发生全国性停电，导致岛上约1000万居民断电。包括圣地亚哥在内的大片地区后来仍处于停电状态，原因是燃料严重短缺。到星期二（7日）晚，当局已恢复全国大部分电网供电。
美国于1月3日推翻委内瑞拉原总统马杜罗的政权后，对古巴实施石油封锁。委内瑞拉原本是古巴主要燃料供应国。美国后来也向墨西哥施压，墨西哥被迫停止向古巴输送石油。
经济师和能源分析师说，多年来的投资不足和结构性缺陷，导致古巴电网日益脆弱。
间歇性停电、燃料短缺和电网反复故障如今已成为常态。外界也因此质疑，如果没有大规模投资和更稳定的燃料供应，古巴电力服务能否恢复正常。

法国艺术家上海雕塑被批“诡异” 媒体称翻车症结在忽视公共
上海一景区摆放48个全身会冒水的彩色人形雕像，被游客批评“诡异”、“吓人”。景区回应，属于专门设计的作品，具体情况需要进一步了解后再回复。中国媒体评论称，公共艺术翻车症结在忽视“公共”。
据大风新闻、澎湃新闻报道，有游客爆料称，上海市青浦区东方绿舟景区内的渔人码头附近，有48个彩色人形玻璃雕像，模样奇怪，有点儿吓人。景区回应称，这群雕塑的名字叫“池中人”。
据悉，这组雕塑在16年前就被安放在了该景点，创作者为法国艺术家法布里斯·伊贝尔（Fabrice Hyber），体现的是人与自然的对话。
《新京报》星期五（7月10日）发评论指出，48个玻璃人形吓到游客，公共艺术翻车症结在忽视“公共”。评论称，一些公共艺术最大的荒诞在于，作品放进了公共空间，公众却被排除在对话之外。私人空间里的创作可以百无禁忌，公共空间里的艺术品却必须面对一个事实：公众在看。“把48个人形玻璃像往空地上一摆，旁边竖一块“法兰西学院院士作品”的牌子，和“看不懂但很贵”的现代艺术笑话没什么区别。”
评论称，景区引进艺术品，不能只看名头不看效果，也不能只重宣发而轻视对话。要改变这种局面，景区应当配套必要的审美阐释机制，让观众在“吓一跳”之后还能“走进去”。多一块说明牌、多一场创作者对话、多一次公众意见征集，都比一句“人与自然的对话”更有诚意。公众的感受不是艺术的敌人，恰恰是公共艺术得以成立的基石。
