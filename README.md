战争快速消耗武器库 促美国重建稀土供应链
（纽约法新电）美国正加紧构建一个不依赖中国的稀土供应链，以补充在伊朗战争中迅速消耗的武器库。
据估计，自2月底开战以来，美国已向伊朗发射数千枚导弹，而制造一枚战斧（Tomahawk）或爱国者（Patriot）导弹需要数百公克稀土。相较之下，制造一台电视或笔记本电脑只需几克稀土。
华盛顿智库“银谷政策加速器”（Silverado Policy Accelerator）副总裁马纳兹·汗（Mahnaz Khan）指出：“中东冲突正实时揭示哪些矿产对军事任务至关重要，以及供应链的哪些环节可能会在压力下断裂。这可能给美国重建武器库的能力带来又一层压力。”
美国战略与国际问题研究中心（CSIS）说，美国需要多年时间，才能把武器库存恢复至理想水平。
过去数年，美国通过提供补贴和税收激励措施，已把在全球稀土产量的份额从3%提高到13%。
去年7月，位于怀俄明州的布鲁克矿启用，是70多年来美国首个新稀土矿。目前，怀俄明、蒙大拿和内布拉斯加还在开发其他类似项目。
与此同时，美国也通过回收利用来补充供应链，同时在海外扩展供应来源。
不过，稀土开采只是第一步，完整产业链还包括精炼、分离与加工阶段。截至去年，中国稀土分离总量占全球的91%，这是中国得以主导稀土产业的关键。
美国稀土业正设法摆脱对中国的依赖。美国公司MP Materials早前说，加州芒廷帕斯稀土矿很快就会启动分离作业。其他美企则计划在国内建造分离设施、与外企合作提升分离工艺等。
但专家指出，美国企业要发展到足够规模，从而显著降低中国市场份额还需时间。

根据独立研究机构IEEFA于5月13日发布的报告，2026年第一季度欧盟从俄罗斯进口的液化天然气（LNG）达到了自2022年俄乌战争以来的最高水平。
报告显示，欧盟在2026年前三个月自俄进口天然气约69亿立方米，较2025年同期增长16%；IEEFA并称4月份进口同比亦增长17%。尽管通过管道的俄罗斯天然气几近停止，但法国、西班牙和比利时等国仍通过油轮大量采购俄罗斯LNG。
俄罗斯目前约占欧盟LNG供应的14%，为第二大供应国。与此同时，欧盟对美国LNG的依赖显著上升：2021至2025年间美国供应量增逾三倍，2026年一季度同比增长27%。IEEFA预测美国将在2026年成为欧盟最大气源，至2028年其LNG占比或高达80%，但也指出美国天然气对欧洲买家而言最为昂贵。
IEEFA欧洲首席分析师安娜·玛丽亚·雅勒-马卡列维奇表示，欧盟从管道气向LNG的转变本意在于保障供应与实现多元化，但中东战争造成的中断和对美LNG的过度依赖已暴露出此举的双重失败。欧盟曾承诺因俄乌战争在2027年底前停止进口俄罗斯天然气和石油。消息来源：IEEFA、法新社。
