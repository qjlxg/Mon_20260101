## 来源: zaobaocn

天文学家首次在星际空间发现糖
天文学家首次在星际空间发现一种单糖，显示构成生命的重要有机分子，或早在恒星和行星形成前，就已能在太空中自然产生。
美国有线电视新闻网（CNN）报道，西班牙天体生物学中心领衔的研究团队，在银河系中心附近一团名为“G+0.693−0.027”的气体和尘埃云中，探测到赤藓酮糖。
赤藓酮糖（erythrulose）是一种含有四个碳原子的单糖，少量存在于覆盆子等水果中，也常用来制作美黑产品。
糖类在生命系统中扮演重要角色，不仅能提供能量和构成生物结构，也是核糖核酸（RNA）和脱氧核糖核酸（DNA）等遗传物质的组成部分。科学家此前已在陨石和小行星样本中发现核糖、葡萄糖等糖类，但从未在星际介质中直接探测到糖。
研究人员用西班牙两座大型望远镜，捕捉这团太空气体和尘埃发出的无线电信号。不同分子会留下不同的信号，就像人的指纹一样。研究团队发现，其中一组信号与实验室里的赤藓酮糖完全吻合，因此确认这种糖存在于星际空间。
研究团队原本想寻找的是结构更简单、含有三个碳原子的糖，但并没有发现；相反，他们发现了结构更复杂、含有四个碳原子的赤藓酮糖。
这让研究人员感到意外，因为过去天体化学界普遍认为，星际分子就像“搭积木”一样，会一个碳原子一个碳原子地增加，由简单逐步变复杂。
按照这种理解，三碳糖应该先形成，或者至少比四碳糖更容易被探测到；但这次观测结果正好相反。
这说明，星际糖类的形成可能并不只是靠逐步增加碳原子，还可能存在其他化学途径。研究人员认为，赤藓酮糖可能是由一些更简单的分子，在寒冷的星际尘埃表面结合形成的。
研究人员还说，这项发现证明，较复杂的糖可以在恒星和行星诞生前形成，并可能随着尘埃、彗星或小行星进入新形成的行星系统。地球早期所需的一部分有机物，因此可能并非全部在地球上产生，而是由太空中的物质带来。
不过，发现赤藓酮糖并不等于找到外星生命，也无法解释生命如何从这些分子中诞生。
这项研究的重要性在于，它扩大科学家对太空化学复杂程度的认识，也提高未来在星际空间发现核糖等与生命起源更直接相关分子的可能。
研究成果已刊登在《自然·天文学》期刊。
