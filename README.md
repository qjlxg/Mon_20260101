《香港铁路标准》下周面世
香港运输及物流局局长陈美宝在马年首篇网志写道，《香港铁路标准》将于下周面世，规管集体运输系统专营权的条例草案预计今年内提交立法会审议。
陈美宝星期六（2月21日）在官网发布的网志中说，运物局正规划和兴建超过10条重铁和四个智慧绿色集体运输系统，让香港的铁路网络更深入社区、覆盖面更广阔，当中包括与大陆合作推展的跨境铁路项目。在确保安全与质量的大前提下，港府致力引入创新思维、技术和工序，推动更高效、更具前瞻性、更具成本效益的铁路建设。
她指出，香港路政署紧联同相关政府部门以及铁路专家在过去半年全力制订的《香港铁路标准》，将在下周面世。
陈美宝介绍，《香港铁路标准》是将东西精髓紧密联系融合一起。这个体系主要以英国及欧洲标准为基础，再因应香港本地情况调整，已沿用数十年，并随着推展铁路项目更新。
她相信，《香港铁路标准》可推动业界采用标准化设计、引入创新建造技术及物料、完善供应链及优化工序。也正因新标准覆盖铁路项目的整个生命周期，对营运者纪录、管理及维修保养铁路资产，以及乘客的良好体验都有益处。
陈美宝强调，《香港铁路标准》不仅是香港的自家品牌，运物局的目标是让它成为大湾区标准，乃至新的世界标准之一。
她称，配合新推出的《香港铁路标准》，路政署将同步成立铁路审批组，整合屋宇测量、结构工程及土木工程等专业范畴的力量，一条龙处理新铁路项目的建筑图则审批工作，为安全和质量严格把关。铁路审批组同时设立电子中央审批平台，进一步加快提交文件及审批建筑图则的流程。
她续称，机电工程署（机电署）将继续坚守把关重任，按照现行法定制度和相关部门合作，在所有新铁路系统和主要设施投入营运前进行一系列严谨测试及检验程序，确保所有新建铁路系统安全稳妥才可启用。
陈美宝表示，运物局正积极筹备立法，引入一套技术中立、通用且标准化的全新监管框架，为规管日后引入的陆上集体运输系统的专营权及服务提供法律基础。
她说，新制度将有助缩短引进新系统及新营办商的筹备时间，让港府能灵活应对创新运输模式及市场变化，迅速把握技术突破带来的机遇。相关条例草案预计于今年内提交立法会审议。

印度重申：购买俄委石油立场未改
印度外交部重申，印度的石油购买策略始终以国家利益为导向，确保能源供给多样化。
