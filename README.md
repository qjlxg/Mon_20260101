深圳放宽核心区限购政策 提高贷款额度
深圳发布优化房地产新政，放宽福田、南山、宝安等核心区的限购，并提高公积金贷款额度。
深圳市住房和建设局星期三（4月29日）印发《关于进一步优化调整本市房地产相关政策的通知》，自2026年4月30日起施行。
《通知》明确，深圳市户籍居民家庭、自购房之日前在深圳连续缴纳社会保险或个人所得税满一年及以上的非本市户籍居民家庭，在执行现有住房限购政策基础上，可在福田区、南山区和宝安区新安街道范围内增购一套商品住房。
持有有效深圳经济特区居住证的非深圳户籍居民家庭，可在福田区、南山区和宝安区新安街道范围内购买一套商品住房，无需提供缴纳社会保险或个人所得税证明。
《通知》同步提高住房公积金贷款额度，以个人申请的，贷款额度从60万元（人民币，下同，11万2100新元）提高至70万元，缴存职工家庭申请的，贷款额度从110万元提高至130万元。
在深圳住房公积金贷款原本最高可贷额度的基础上，提高额度上浮比例。购买首套住房的，上浮比例从40%提高至60%；新增初婚初育居民家庭购买深圳住房上浮情形，上浮比例为50%，有两个及以上子女的居民家庭购买本市住房的，上浮比例从50%提高至70%。

摩根士丹利 ：为美股回调做好准备，恐慌顶峰正是买入良机
摩根士丹利投资管理公司高级投资组合经理安德鲁·斯利蒙表示，投资者应为美国股市回调做好准备，并在恐惧情绪达到顶峰时果断出击。当前企业的强劲盈利与累积风险相互碰撞。斯利蒙认为，大部分盈利乐观情绪已被市场消化。股市，尤其是所谓的“科技七巨头”，已经大幅上涨，这增加了即使盈利稳健也可能不足以继续推高股价的风险。斯利蒙表示：“如果出现意外的坏消息，你可能会看到某种回调或修正，因为仓位已变得非常激进。”他表示，这种回调有助于重置过于乐观的市场情绪。 他指出，股市恐慌情绪爆发的可能性将为投资者创造买入机会。斯利蒙建议投资者保持耐心，而不是在当前水平投入新资金。“如果有人有现金要投资，现在可能不是部署的最佳时机。”从长期来看，他仍持建设性态度。他认为，包括地缘政治紧张局势、油价上涨以及对AI颠覆的担忧等宏观风险不太可能破坏盈利轨迹。

德国计划从2028年对含糖饮料征收附加税——作为4月29日通过的医疗改革方案一部分，旨在遏制肥胖率上升，减轻医疗系统负担。
预计该附加税每年可带来约4.5亿欧元（约6.72亿新元）收入，用于疾病预防和健康促进项目；具体税率等细节仍在商议中。
德国糖业游说团体对此表示反对，称提高糖价未在任何国家降低超重比例。世卫组织曾建议在未来10年通过税收将含糖饮料、酒精和烟草价格提高50%。
今年2月Forsa调查显示约60%的德国人支持对含糖软饮征税。英国十年前实施类似税制，并于2025年将其扩展至预包装乳制饮料；世卫表示已有100余国对含糖饮料征税，英墨研究显示此类举措可减少糖摄入并有助预防糖尿病。
该提案需议会确认。提出该方案的石勒苏益格-荷尔斯泰因州首席部长丹尼尔·根特尔三月对路透社表示，“过多糖分使人患病”，并给医疗和经济带来压力。REUTERS

伦敦，4月29日 — 克里姆林宫周三表示，因乌克兰威胁升级，俄方将缩减每年5月9日的二战胜利日阅兵规模，红场今年不会展示通常亮相的武器装备。
俄国防部称，鉴于“当前作战态势”，今年阅兵不会出现军事装备。克里姆林宫发言人德米特里·佩斯科夫称，基辅一方面在战场上节节失利，另一方面“已发动全面恐怖活动”，因此需采取一切措施以降低风险。
近年阅兵常展示洲际弹道导弹等重型武器，普京也常借此激发民族情绪并为对乌战争造势。外界认为，此次不展示武器反映了俄军在战场上的压力以及担心成为乌方深远打击目标。乌克兰过去几年多次用无人机袭击莫斯科，并对俄后方设施造成重大破坏，俄方亦指控基辅与多起刺杀未遂有关。
对这一决定，批评者在社交媒体上嘲讽。前克里姆林宫演讲稿撰写人阿巴斯·加利亚莫夫质疑是否因军心不稳或装备在乌克兰损毁。前英国驻俄防务武官约翰·福尔曼指出，2025年阅兵曾有约1.1万名官兵和约150辆装备参与，此次删减既与战场形势相关，也为避免在大型阅兵中暴露目标。
目前美国因应对伊朗局势而牵扯调解，俄方宣称近期在战场上取得局部推进，而乌方对俄港口和炼油设施的打击也造成显著损失。阅兵缩减的消息在俄罗斯社交平台上引发热议，一些民众认为没有武器的阅兵“很奇怪”，也有人称真实庆祝应等到战争结束。

安卡拉，4月29日——土耳其总统埃尔多安周三在议会表示，旨在结束与库尔德武装长期冲突的和平进程正在“积极氛围”中推进，驳斥了亲库尔德议员的批评。埃尔多安称，随着议会委员会通过了一份将法律改革路线图与被禁库尔德工人党（PKK）解散挂钩的报告，进程已进入更为敏感的阶段。
此前，亲库尔德的民主党（DEM）发出其迄今最强烈的批评，指责政府在这次历史性和谈机会面前“犹豫、怯懦并拖延”。埃尔多安反驳称：“现在氛围积极，下一步要做的很清楚，进展符合预期。”他并指责悲观论调“基于幻觉而非事实”。
埃尔多安表示，委员会报告获批后已到需更谨慎应对的十字路口，执政的人民联盟将争取其他政党支持共同应对。土、欧、美均将PKK列为恐怖组织；PKK去年停止袭击，并在今年5月宣布决定解散、结束武装斗争，此前其在押领导人阿卜杜拉·奥贾兰于2025年2月发出号召。但围绕下一步仍有僵局，安卡拉主张在采取进一步法律或政治措施前须核实解除武装，库尔德政治力量则呼吁加快改革。自1984年冲突爆发以来，已造成逾4万人死亡，并波及伊拉克和叙利亚。— 路透

斯洛伐克最高法院周三维持对两年前向总理罗伯特·菲科开枪的男子判处21年有期徒刑的裁决，驳回其就恐怖主义指控提出的上诉，斯洛伐克媒体报道。
去年10月，法院认定现年73岁的尤拉伊·辛图拉（Juraj Cintula）在2024年5月于该国中部一城镇向菲科近距离连开五枪，构成恐怖主义罪。 
在线媒体Dennik N援引审判法官佩特尔·卡纳（Petr Kana）在判决理由中的话说，“对袭击的策划以及其明知是在袭击总理，表明他应当意识到会对政府职能产生负面影响。”
塔斯社称最高法院裁决为终审。法院未对置评请求作出回应。
2024年的枪击事件暴露出这个拥有540万人口的中欧国家内严重的政治分裂，菲科当时曾一度病情危重，但后来康复并重返工作岗位。
