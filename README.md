กรวีร์表示，党内尚无具体协商，但相信党会选出具备知识、能力与经验的人选。他希望议会更现代、透明，能有效连结人民、立法与行政，利用科技提高议员工作效率并拉近与民众的距离。

กรวีร์最后表示，作为国会议员要经得起选民检验，必须真正为人民发声、推动影响民生的法律，若党给予信任，他将以经验与专业全力以赴。

2026年3月9日，旅游体育部长阿达孔·西里拉特亚功（อรรถกร ศิริลัทธยากร）主持召开专题会议，跟进中东地区冲突对游客的影响并研究应对措施。会议在政府综合办公楼C座9楼举行，相关主管及单位参会，评估对泰国旅游业的冲击并部署应对方案。

与会单位并报告了因航班取消或改道造成的旅客问题。旅游体育部已与航空、移民、机场游客救助中心（TAC）及酒店等部门协同，提供持续协助，包括航司协调、在主要机场通过TAC提供信息与便利、移民部门对需延留游客的协助，以及建议酒店在游客无法如期返程时审酌调整房价，以维护游客权益并稳住旅游信心。

据3月9日报道，随着新组阁轮廓几近明朗——以ภูมิใจไทย为主导，联手为泰党、国民力量党、人民ชาติ党及小党组阁——关于下届众议院主席及三名副主席人选的流言四起。其中，属于为泰党配额的众议院第二副主席人选广传为曼普·查隆西（Dr. มนพร เจริญศรี，国会议员“เดือน”），为นครพนม第2选区议员。
曼普于3月4日在选管会报到时表示，对外界关于其将出任副议长的传闻感到惊讶，党内尚未讨论相关人事。现年60岁的曼普，毕业于รามคำแหง大学政治学本科与硕士，并在เกริก大学获政治传播博士学位，早年就读于นครพนม地区学校。
政治履历方面，曼普自1994年起曾两度当选省级议员、两度任副省长、并任省长一届。2011年起以为泰党身份当选国会议员，曾任多项国会委员会发言人、消费者保护委员会秘书、为泰党副秘书长及在野党党团秘书等职，并长期关注民生、农产品价格与就业问题，连任多届、深受选民支持。
2023年5月大选中，曼普在强劲对手挑战下仍以个人高票胜出，并于同年出任运输部副部长，创下府南县政治史纪录。最新一次选举（2026年2月8日）曼普再次以明显优势当选第2选区国会议员，外界因此推测其可能出任众议院第二副议长一职。

3月9日10时10分，党魁ณัฐพงษ์（Nattapong Ruengpanyawut）率人民党（พรรคประชาชน）119名议员乘三辆巴士到众议院报到，身着印有“ประชาชน”的白T、外着西装，支持者献上橙色鲜花鼓励。党内另有1名议员（สุริยา）此前已单独报到，党籍总数为120人。 
ณัฐพงษ์表示，118名议员今天集体报到，已准备履行作为第27届国会议员的职责；下午将举行议员会议部署议会工作。党将立即在选出议会主席后提交多项法案，包括经济法、反贪腐、预算体制改革、设立国会预算办公室，以及环保、人权与电价降低（推动能源自由化）等议题。 
他称，作为未入阁党，将与“กล้าธรรม”及民主党等共同担任在野党，维持既有反对派协作机制；对涉及44名议员的司法案并不恐慌，党已有应对预案，若被法院裁定部分议员须停止职务，将由党内程序决定替补人选并在议会会议中讨论，若牵涉党务高层则以党大会决议为准，预计四月召开党内会议。 
对于当天恰逢国家反贪机构向最高法院提请案件期限一事，他称属巧合，报到是等待选委会完成全部当选人认证后进行。对于素攀武里府第2选区的选举调查可能扩大，他表示将尊重程序，若发现异常将交由法律团队处理并已准备相关证据。 
他重申党接受选举结果并将履职，同时会继续并行调查选举不公，呼吁选委会加速查明并追查幕後主使，而非仅限基层执行者。

2026年3月9日，斯里沙凯选区国会议员、前总理府发言人西里蓬·昂格萨库利基特（สิริพงศ์ อังคสกุลเกียรติ）就有学者质疑其家族承接政府工程涉嫌贪污一事作出回应。西里蓬表示，政治人物为公众人物，理应接受监督，但学者的批评应基于无偏见、全面资料，而非断章取义。
他说明，名为“หจก.ไทยเจริญศรีสะเกษ”的合伙企业自2524年（即1981年）起营运，他当时仍是小学一年级学生；该合伙人为资深商人，十多年来承接政府工程并不稀奇。关于同姓或有亲属关系无法切割的问题，他称对方为尊重的长辈，平时仅在节日相见，双方家庭各自经营生计，要求亲属放弃生意以配合其从政并不合理。
西里蓬强调，指称“因姓氏即属不当”不公平，反指批评者也应审视对方阵营的类似情况；若指控属无根据，他将依法律程序处理，先交法律团队审查。他称政治监督必要，但含糊或恶意指控会造成误解，乃“令人厌恶”的行为，并表示会为能澄清之处说明真相，对造成损害的歪曲行为则会追究责任。
