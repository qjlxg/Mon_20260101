这是特朗普近期为淡化伊朗战争对经济影响并稳住11月中期选举支持所做的一系列公开活动之一。此前他在拉斯维加斯称部分通胀是“因燃料和能源”的“假性通胀”，并鼓吹对小费收入者的减税。
现场多数为铁杆支持者，向北约齐声嘘声、高呼“美国”，佩戴红色特朗普帽并举标语。Turning Point首席执行官Erika Kirk登台致辞并介绍特朗普。场外也有抗议者。多位与会者虽关切油价，但普遍信任特朗普能解决问题，有人称国家“破产了”，需要商人出身的总统；也有人前往山顶为总统祈祷，称其受主恩膏。来源：纽约时报

利马消息——秘鲁检方4月17日突查存放总统选举选票的仓库，欲查明组织投票的全国选举事务办公室（ONPE）是否履行了与选举材料相关的职责。
4月12至13日的投票仍在计票中，进展缓慢，几乎所有未完成的计票单都遭到质疑，须经选举法庭复核，约有100万张选票结果悬而未决。已计票的93.3%显示，右翼候选人Keiko Fujimori领先得票约17%。此前进入决选的两名候选人将角逐第二轮，但目前左翼的Roberto Sánchez约得12%、极保守派的Rafael López Aliaga约得11.9%，双方相差约1.3万票，竞争非常接近。
此轮选举还爆出多起争议：4月16日在利马垃圾桶发现装有1,200张选票的四箱纸箱；选举材料送达延误曾迫使延长投票时间；国家选举最高司法机构JNE主席Roberto Burneo在国会质询中指控ONPE在管理和执行上存在严重不规范。ONPE负责人Piero Corveto及另三名官员已被JNE移送调查，涉嫌侵害选举权。
本次共有创纪录的35名候选人参选。秘鲁政局长期不稳，过去八任总统中有四人被国会弹劾；本次竞选也以右翼关于打击敲诈与契约杀人等强硬主张为主要议题。AFP

布鲁塞尔 — 随着越来越多老款但仍受欢迎的游戏被发行商一夜关闭服务器、变为不可玩，欧洲玩家发起的公民请愿“停止摧毁电子游戏”已获得逾一百万签名，正敦促欧盟立法禁止此类做法。
请愿者指出，过去十年有数百款游戏因盈利或策略调整被停止服务，玩家在内购等方面的投入因此付诸东流，部分流行文化资产被抹除。2024年竞速游戏《The Crew》停运成为导火索，促成此次欧洲请愿。
请愿法国组织者布伦丹·福尔丹称，这种做法像“买了本书却发现内页被抹去”。代表们已于2月会见欧委会数字事务负责人海娜·维尔库宁和消费者保护负责人迈克尔·麦格拉斯，并于4月16日在欧洲议会听证会上陈述诉求，要求落实现有消费者保护并推动欧盟立法更新：在停服时应保留可继续游玩的方式（例如由志愿者运行的私服）或必须对玩家系统性退款。
业内反对私服方案，行业组织Video Games Europe称私服难以保障玩家数据并处理非法或不安全内容，可能使权利方承担责任。欧委会须在7月底前对请愿作出回应。

主办方“欧洲爱国者”党（欧盟议会第三大党团）号召支持者于格林威治时间13:00（新加坡时间21:00）在米兰大教堂前集结。组织者、意大利联盟党领袖兼副总理马泰奥·萨尔维尼称大教堂为“基督教象征”，本次活动名为“无畏——欧洲做自己家的主人”。

米兰当日亦将举行多场反法西斯等反对活动；同为执政盟友的意大利力量党在米兰也计划举办一场关于移民子女社会公民参与的活动。消息来源：法新社。

华盛顿——总统唐纳德·特朗普4月17日晚表示，如果无法与伊朗达成和平协议，美国将继续封锁伊朗港口；本轮美伊停火将于4月22日到期，他也可能不再续期。 
此前，因以黎停火协议，霍尔木兹海峡于4月17日再次开放，但伊朗威胁若美方持续封锁将再次关闭这一要道。 
特朗普在空军一号上称，和平协议“可能会达成”，并暗示有“关于伊朗的一些好消息”，还称伊朗同意上缴浓缩铀，言辞引发争议。伊朗则否认会交出铀储备，并警告若美军截拦来自伊朗港口的船只，海峡将被关闭。 
议长加利巴夫在社交平台写道，霍尔木兹通行需获伊朗许可；外交部发言人巴卡伊称，所谓海上封锁将遭到“适当回应”，并称封锁违反停火。 
受停火与通航恢复消息影响，市场与油价走高；美国同时放宽一项针对海上俄罗斯石油的豁免。 
黎巴嫩方面，十日停火使战区居民回返家园。以色列获美方限制进一步打击，但总理内塔尼亚胡警告战事尚未结束，真主党亦称将对任何越界行动作出回应。

里士满，弗吉尼亚 — 在州府里士满郊外一片林地，枪声不时响起。38岁的柯林正在学习使用半自动步枪，这是他拥有的第一把枪。柯林表示，他比周围公民更害怕政府，并指出明尼阿波利斯发生的两起死亡事件——公民Renee Good和Alex Pretti在一次大规模移民执法行动中被联邦特工枪杀——促使他买枪自保。
这类现象打破了美国传统对枪械拥有者的印象：很多左翼倾向的美国人因为对特朗普政府的担忧转而拥枪。支持持枪权的保守派把持枪视为宪法保障的个人自由；自由派则主张更严格的枪支管控。但包括曾遭枪击的前众议员加比·吉福兹和前副总统卡马拉·哈里斯在内的多位民主党人也公开承认持枪。
为适应这一变化，持枪教练克拉拉·艾略特表示，自2024年11月特朗普当选连任后，她的生意“翻倍”，面向女性、少数族裔和LGBTQ群体的课程常常爆满。课堂上多为从未接触过枪械的学员，他们把近期的移民抓捕、取消多元与包容政策以及社会极化列为上课动因。学员阿凯米说，她担心“右翼暴力”，对警方保护缺乏信心。
全国性组织自由派枪械俱乐部称，2026年前两个月申请枪支训练的请求达3000起，超过2025年全年。执行董事埃德·加德纳指出，重大政治事件或暴力事件后常见报名激增；不同年龄、城乡背景的人都在加入。北卡罗来纳州卫斯理福雷斯特大学社会学教授大卫·雅曼认为，这一变化反映出人们更多是出于对可能出现的专制或政府剥夺权利的担忧而购枪。
