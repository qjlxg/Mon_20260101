中国一网民买速效救心丸后收到注销驾照提醒短信 引隐私与安全争议
中国一名网民披露在网购速效救心丸后收到提示注销驾驶证的短信，引发关于隐私保护及监管是否“一刀切”的争议。交警部门对此回应称，若驾驶人确实患有相关疾病，继续驾驶存在安全隐患，此举旨在保障公共安全。
综合极目新闻和《湖北日报》等报道，涉事网民近期发布视频称，自己网购速效救心丸后，收到来自厦门市湖里交警大队的短信提醒。短信称，经大数据比对分析，“您可能存在器质性心脏病、癫痫病、美尼尔氏症、眩晕症、癔病、震颤麻痹、精神病、痴呆以及影响肢体活动的神经系统疾病等相关的就诊或购药记录”。
短信指出，根据相关规定，上述疾病属于妨碍安全驾驶的情形；患有此类疾病不得申请驾驶证，已取得驾驶证的，不得继续驾驶机动车，并应当向驾驶证核发地车辆管理所申请注销驾驶证。

迪拜 — 迪拜警方近日通过社交媒体发出提醒，敦促公众不要传播未经核实的信息，警告此类做法可能破坏社区安全与稳定。
