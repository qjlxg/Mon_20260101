广州立法保护早茶文化 规定24小时内现制才是传统制作点心
（广州/北京综合讯）中国预制菜行业呈现爆发式增长，广东省会广州市为“早茶传承保护”立法，规定以传统方式现场制作的早茶食品，从食品制成到提供食用时间不得超过24小时。 
综合《广州日报》与澎湃新闻报道，《广州早茶传承保护规定》星期三（4月1日）正式对外公布，将自5月1日起实施，为广州早茶习俗、制作技艺等非物质文化遗产的传承提供法治保障。 
茶点是“预制”还是“手工现做”，是民众在茶楼喝早茶时的关注点。新规提出，以传统方式现场制作的早茶食品，从食品制成到提供食用时间不超过24小时；如果因传统技艺制作需要超过24小时，由当地餐饮相关行业协会等予以确定。
法规提出，广州早茶经营者应当向消费者明示，所提供的早茶食品是以传统方式现场制作，还是以非传统方式制作。至于价格，同一种早茶食品，以传统方式现场制作和非传统方式制作，“可以实行不同价格”。
新规也提出，广州早茶经营者应当通过设立透明式、开放式、视频监控式厨房或者参观通道等形式，向消费者展示广式点心等食品加工制作过程。
广州这部地方法规公布后，不少网民对此表示欢迎，认为茶楼明示点心采用何种方式制作，可根据民众的不同倾向提供选择，保障了消费者的知情权与选择权，并指为纯手工制作的点心“消费者多付几元钱也值得”。
近年快速发展的预制菜产业满足了便捷性餐饮需求，但也被认为挑战传统饮食文化。传统广府点心的手工制作工序繁琐，不少茶楼近年转向使用速冻或半成品，以至有网民称，在茶楼要找到真正手工现做的虾饺与烧卖并不容易。

科普：关于“睡眠瘫痪”的误解
马尼拉讯——在菲律宾民间传说中，“Bangungot”（噩梦精）被认为是坐在胸口导致人无法动弹的恶魔。3月26日播出的《Klinika 630》节目中，Dr. Denis Ngo 与主播 Winnie Cordero 解释了睡眠瘫痪的科学成因，澄清相关迷思。
什么是睡眠瘫痪？
睡眠瘫痪是指人在入睡或醒来时短暂无法移动的状态，属“睡眠障碍行为”之一。发生于快速眼动（REM）睡眠阶段：大脑有清晰梦境，但身体处于肌肉失张力（atonia）以防演动梦境。若在此时意识苏醒，便会产生“被瘫痪”的错觉。
症状与诱因
约30%人口一生中至少经历一次，常伴无法说话、恐惧、幻觉、听见不存在的声音和出汗。诱因包括睡眠不足、作息不规律、高血压、焦虑、双相障碍、PTSD、睡眠呼吸暂停、发作性嗜睡、酗酒或药物滥用等。
常见误解与真相
1) “Bangungot 坐在胸口”——真实情况是胸部肌肉失张力，但膈肌仍工作，呼吸感觉不规则，引发恐惧感。 
2) “被巫术或附体”——多为无力移动加梦中幻觉交织所致。 
3) “会致死”——睡眠瘫痪本身不会致命，通常为短暂现象。 
4) “吃饱不能睡会导致”——饱寝可能引起胃酸逆流，但非睡瘫直接原因。 
5) “盖头会被闷死或瘫痪”——被褥限制呼吸可能不适，但与睡瘫无直接关联。
如何降低发生几率
坚持规律睡眠、改善睡眠环境、睡前远离电子产品、通过洗澡或阅读放松。反复发作者应就医，医生可能建议调整影响REM的药物、治疗精神或睡眠疾病，或寻求心理卫生专业支持。
