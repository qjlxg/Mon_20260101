专家指向晚婚（目前男性中位数30岁、女性29岁）、住房、医疗、教育成本上升及学贷负担等结构性因素。特朗普承认政府能力有限，但政治上也有考量：支持他的县域出生率普遍高于民主党支持区。媒体热传白宫新生儿照片，但专家认为这些高调的个案难以扭转全美低生育趋势。

美中在人工智能与芯片领域的竞争在特朗普与习近平5月13至15日北京峰会前进一步加剧。4月，美国国会审议的MATCH法案拟堵塞出口管制漏洞，进一步限制中国获取先进半导体制造与AI训练所需的技术与设备，可能严重制约中国的AI发展。
中国商务部4月25日表示将密切关注美方立法进程，并“坚决采取必要措施”，反对所谓“滥用出口管制”。这一动向发生在两国元首即将会晤之际，尽管预计双方会有礼节性友好互动，但难以消减技术竞争的实质矛盾。
美方智库人士指出，中国AI企业尤其是初创公司在算力规模、先进AI芯片与资金获取方面仍落后于美企，这凸显中国对最先进半导体及其制造工具的依赖与脆弱性。尽管特朗普在2025年底批准英伟达H200芯片对华销售，美国商务部长称迄今未有中国企业购买，部分原因是中国政府鼓励把投资集中于国内产业。
为应对外部压力，北京一方面加快国产替代，4月有初创企业推出适配华为芯片的软件和模型；另一方面通过法律手段拓展反制空间，4月7日出台新规以保障产业和供应链安全，授权对“歧视性限制”采取反制措施。新加坡学者胡欣悦指出，这些规定是中国首套产业与供应链安全行政规制，意在为反制措施提供法律依据，并可能将竞争延伸至经济法律层面。
市场方面，2025年中国国产AI芯片占本土市场近41%，而在2023年前英伟达在相关细分市场的份额曾超过90%。外界普遍认为，半导体与AI的地缘政治重要性短期内难以消减，双方较量仍将以防御性措施为主，关键看未来哪方在下一波技术突破中占先机。
