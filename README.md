明尼阿波利斯——3月28日，数万名示威者在明尼阿波利斯集会，参加全国性的“No Kings”运动，此前数月该市因联邦移民行动引发抗议，成为唐纳德·特朗普移民强硬政策的焦点。
今年1月的突袭导致两名美国人蕾妮·古德（Renee Good）和亚历克斯·普雷蒂（Alex Pretti）在联邦特工行动中遇难，使得当地局势更加紧张。
此次为2025年草根运动发起以来的第三次“No Kings”抗议，明尼阿波利斯承办旗舰活动，组织者估计有约20万人到场。
著名音乐人布鲁斯·斯普林斯汀在邻市圣保罗登台演出，并演唱了为悼念古德与普雷蒂所作的《明尼阿波利斯街道》，称“他们的勇气、牺牲和名字不会被遗忘”。
现场人群高呼口号：“No Kings，No ICE，No war！”明尼苏达州长、民主党人蒂姆·瓦尔茨在台上感谢民众站出来，称特朗普为“准独裁者”。
左翼政客伯尼·桑德斯也在集会上发言，批评特朗普是“病态的说谎者、掠夺者和自恋者，每天都在破坏宪法与法治”。
参与者帕梅拉·西尼斯（73岁）表示，她为平等权利而来，也为明尼苏达人民在暴力和对人的不尊重中受到的创伤感到痛心。另一名现场人士吉娜·比洛塔-拉塞利斯称特朗普“无能”，应像其他人一样遵守规则，她并预计因伊朗冲突、油价飙升及有争议的移民突袭，共和党在11月期中选举中会受挫。
（来源：法新社）
