位于泉州晋江的辉腾鞋业有限公司厂房上星期四（7月9日）中午12时许发生火灾，共造成‌28人遇难。中国国务院安委会对这起重大火灾查处挂牌督办，并派出督办组督促指导福建省开展调查处理工作。
央视新闻《焦点访谈》栏目星期六（11日）报道揭露，鞋厂整个楼里堆放的很多物品是易燃物，救援困难重重。据悉，街道两侧停满机动车、电动自行车，一包包废弃鞋类随意堆放在路边，可通行宽度受限，消防车进出困难。
参与救援的福建泉州消防救援支队灭火和特种灾害救援科科长郑铿说，到楼顶时因周边的可燃物全部过火燃烧，导致内攻小组没办法第一时间到达人员的被困的点。
此外，记者调查发现，起火厂房内部从一楼到五楼全是栅栏，车间里边也有很多栅栏，包括逃生通道也有安装。
对此，晋江市陈埭镇江头村驻村干部蔡荣烽说，铁栅栏“平时是执法部门在主抓”。村干部主要入场帮企业排查厂区里面的隐患。不过他坦言，铁栅栏确实在火灾中妨碍公众逃生。
此外，起火的生产车间也并未按照规定配备灭火的喷淋设施。
据了解，火灾发生前的两天，即7月7日，也就是发生火灾前两天，鞋厂刚经过镇里组织的消防检查，被警示物资、成品、半成品无序堆放问题较为严重，要求在7月13日之前整改。
据介绍，鞋厂易燃物应当有特殊的管理办法，但安全检查并没有严格执行。当地镇干部称，建筑是“历史遗留的”，不是规范化的，所以做到货物防火分区等“难度比较大”。
在检查发现鞋厂货物堆积时，这名干部称，鞋厂“没有办法每个时刻保证空间符合要求，他在赶货，日常管理疏忽，没有办法做到百分百”。
此外，当地驻村干部坦言，多次检查发现，厂内消防通道常常被堵塞。
报道得出结论指，火灾绝不是一时疏忽造成的，而是多种人为隐患堆积而成。这次事故暴露出劳动密集型厂房消防安全管理的诸多短板。

英国广播公司（BBC）报道，位于庆尚北道八公山的八世纪古刹桐华寺，日前迎来24名年轻男女参加活动。他们从约1600名报名者中脱颖而出，主办方通过问卷和自拍视频，评估报名者对婚姻和生育的态度。
活动主持人柳哲柱说：“每当国家陷入困境，佛教徒总是最先行动。如今，低出生率已成为国家危机，我们必须做点什么。”
28岁的金雅暻从首都圈搬到韩国东南部工作后，生活几乎只有公司和家两点一线，很少有机会认识异性。30岁的乳制品工厂职员权胜吾则说，约97%的同事是男性，过去参加过约10次亲友介绍的相亲，却觉得互动流于表面。
相亲营安排林间散步、赠送玫瑰、共进午餐、饭后洗碗及才艺表演等环节，让参加者在短时间内加深了解。

不过观察者指出，一系列的活动持续消耗着参与者“社交电量”。因此当僧人带头唱起国歌，疲惫的参加者只能低声跟唱。
韩国地方政府和民间团体自2000年代初起陆续举办相亲活动。与此同时，政府自2006年以来已投入约2500亿美元（约3200亿新元）推行鼓励生育政策。
韩国总和生育率2023年降至0.72的历史低点，2024年起开始小幅回升，预计2026年将由2025年的0.8升至1.0。
不过，官员认为，这可能与疫情推迟婚礼和生育，以及特定人口世代进入生育年龄有关，未必能直接归功于政府政策。
这场活动最终促成八对配对，其中两对由工作人员与参加者组成。未成功配对的参加者也结识了新朋友，并表示愿意再次参加。

原油价格回落燃油价格却反弹 消费者或迎更多冲击
（纽约彭博电）全球原油价格近期有所回落，汽油、柴油和航空燃油价格却出现反弹，这种罕见的分化正在推高旅游旺季的出行成本，并可能动摇美国总统特朗普在中期选举前遏制通货膨胀的承诺。 
伊朗战争爆发四个多月后，全球石油基准价格近期大幅回落，已基本抹去战争初期的飙升溢价。不过，美国和其他地区部分成品油与原油之间的差价，仍创下历史新高。 
分析师警告，鉴于俄乌战争引发的俄罗斯出口禁令，加之中东局势再度紧张导致供应吃紧，消费者须要做好承受更多冲击的准备。
全球石油贸易商巨头托里格拉集团柴油和航空煤油交易负责人托伦斯（Jamie Torrance）说：“炼油厂的开工率已经非常高，可库存仍在减少。”
欧洲洲际期货交易所（ICE Futures Europe）数据显示，美国本周对伊朗发起新一轮袭击之前，对冲基金已为油价持续上涨做好布局，对欧洲柴油的看涨押注达到3月下旬以来最高水平。

交易员称，巴西和土耳其等长期依赖俄罗斯供应的国家，近来正为非俄罗斯供应展开竞购战。
与此同时，霍尔木兹海峡周边战事再度升级，可能使问题进一步恶化。
这波供应趋紧正值北半球传统夏季度假旺季需求高峰期。美国汽车协会数据显示，截至7月10日，美国普通无铅汽油平均价格达到每加仑3.88美元（约5.02新元），是有记录以来同期第三高水平；柴油价格也创下有记录以来同期第二高。
白宫发言人罗杰斯则表明，特朗普和能源团队已预见短期市场动荡，并实施一项积极计划以减轻影响。“油价已大幅下跌，加油站的油价也应随之下降。”

国际能源署：欧洲未加快电气化进程是重大失误
（布鲁塞尔讯）国际能源署（IEA）署长比罗尔批评，欧洲自2022年能源危机爆发以来，未能加快摆脱对进口化石燃料的依赖，这是一个重大失误。
比罗尔接受《金融时报》采访时指出，欧洲的电气化率，即电力在欧盟能源消费总量中的占比，仅为23%左右，这一低水平正阻碍欧盟的竞争力与经济“主权”。
尽管欧盟高度依赖碳氢化合物进口，但其电气化率却与美国等主要产油国相近。他认为，欧洲应效仿中国、日本和韩国等电气化率超过30%的国家。
比罗尔说：“在我看来，这是欧洲的一个重大失误，我原本期望欧洲能更积极应对这场能源危机。”
欧盟能源专员约根森则在采访中说，2022年俄罗斯大幅削减管道天然气供应之际，欧盟已扩大部署可再生能源、提高能效，并将天然气消耗量降低20%，但欧洲的供暖、交通和工业部门依然依赖化石燃料。
他坦言，这种对进口的持续依赖意味着，欧洲仍深受中东冲突引发的全球油气供应中断的冲击。
欧盟也将提出2040年电气化长期目标；欧盟早前承诺，到2030年将电气化率提高至32%。
约根森强调：“过去10年，我们的电气化率停滞不前，我们需要加快步伐来推进电气化。”
欧盟委员会下周将公布新计划，要求成员国降低电力税，并支持家庭采用热泵、电动汽车以及其他绿色技术。
根据报道引述的有关草案，欧盟计划通过推行激励措施来推动电气化进程，包括强制规定电力税率得低于化石燃料，争取到2030年，家庭用电成本不超过天然气价格的2.5倍，工业用电价格不超过天然气价格的两倍。
不过，这些措施可能对高度依赖电费税收的成员国带来高昂代价，而希腊、意大利、匈牙利和爱尔兰等国电价相对气价偏高。

银行业贸易组织的数据显示，6月外国投资者从新兴市场股票投资组合中净流出 461亿美元。这一数据主要由对韩国科技股的大幅抛售推动，导致发展中经济体连续第二个月出现整体投资组合亏损。国际金融协会（IIF）周五发布的月度报告显示，外国投资者从韩国股市撤资 305亿美元，创下最近25年多来的最大流出规模。报告显示股市与债市资金流向出现明显分化：尽管整体投资组合资金流转为净流出178亿美元，但债券市场上月仍吸引了 283亿美元资金。国际金融协会（IIF）首席经济学家Jonathan Fortun表示：投资者仍愿意借钱给新兴市场，但他们不太愿意承担广泛的股市风险。全球贴现率上升、盈利信心疲软以及对科技和能源仓位的敏感性，导致投资者削减了股票配置。上半年的信息很明确，新兴市场总体上仍吸引了资本，但这仅仅是因为债务流入完全抵消了持续的股市清算。
