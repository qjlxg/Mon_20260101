中国社交媒体和视频游戏巨头腾讯公布2025年反腐报告，首次出现利用人工智能（AI）反舞弊的表述，有90余人因违规被解雇，其中20余人涉嫌犯罪移送警方处理。

中国海警与海军参与南海救援：17名菲律宾船员被救
中国海军和海警周五表示，他们在南沙黄岩岛（国际通称斯卡伯勒浅滩）附近救起了一艘新加坡籍货船倾覆事故中的17名菲律宾船员。

北京大学研究团队研发出一种模拟运算（analogue）AI芯片，能够处理真实世界数据任务，或改变当前对高能耗数字处理器的依赖。团队在Nature Communications发表论文，第一作者孙中表示，该芯片在推荐系统训练中比先进数字处理器快约12倍，能效提升超过200倍。相关测试使用了与Netflix、Yahoo规模相当的数据集。
在图像压缩实验中，系统以近似全精度数字计算的视觉效果重建图像，同时将存储需求减少约一半。审稿人指出，与传统数字芯片相比，这些“数量级的速度和能效提升”显示出产业化应用潜力。
该芯片基于电阻式存储器实现片上（in‑memory）模拟计算，研究团队采用非负矩阵分解（NMF）方法并重新设计核心电路，将算法中最耗费的部分压缩为单步完成，从而用更少计算单元实现同样结果，显著缩小体积并降低功耗。孙中提到，团队此前曾在基础算术测试中展示出对顶级数字处理器高达千倍的加速，但此次工作将模拟芯片的应用扩展到个性化推荐与图像处理等更复杂任务，并将NMF这一自1999年提出的技术引入模拟片上计算领域。

中国商用航天2025年实现“50次发射”突破，占全国发射总量的54%。
国家航天局周二公布数据称，商用航天部门记录发射50次，其中25次由商业运载火箭完成。海南首个商用发射场于2024年底投运，去年承担9次发射。全年共有311颗商业卫星入轨，占中国卫星入轨总量的84%。
机构认为，商用航天正由政策孵化期向产业化扩张转变，动力来自星座部署、可回收运载技术进展及来自以SpaceX为首的美方竞争。航天局已设立商用航天司，并发布2025—2027年高质量安全发展行动计划。
行业规模方面，超过600家企业，2025年估值约2.5—2.8万亿元，复合年增长率超过20%。可回收火箭方面，北京澜天（LandSpace）研制的朱雀三号于12月3日首飞入轨，但一级垂直回收未完成，原因仍在调查中。
资本市场方面，约10家航天企业正筹备上市，监管层允许尚未盈利的高技术企业在商业航天、人工智能等领域融资。
同时，商用航天也遭遇挫折：一枚长征三号乙（搭载“实践”32号）发射失败，数小时后格拉宙能（Galactic Energy）的商用火箭Ceres-2在酒泉发射场任务亦告失败。
摩根士丹利指出，美国在轨发射仍领先，商业化发射占比最大，并预计2026年星座扩张、直连终端通信与地缘竞争将继续推动行业发展。

首次接触：为探月寻水，中国团队警告触碰月球冰层有丢失风险
中国“嫦娥七号”任务今年将首次尝试采集并直接测量月球水，但科研团队在新发表的论文中提醒，哪怕轻微触碰也可能导致冰被丢失。
嫦娥七号预计在月球南极沙克尔顿陨坑边缘着陆，部署月球车和短飞行器（hopper）寻找冰层。哈尔滨工业大学与中国科学院的研制团队指出，月壤中的水并非像地球上那样以自由水形式存在，而是被锁定在冻结的土壤中，不与空气接触，仅靠极低温与真空保持稳定。
当月球车机械臂的取样器刮入含冰土壤时，接触和摩擦产生的哪怕微弱升温，都可能解吸水分子并使水丢失。论文已被《中国空间科学》接受，研究提醒：正确保留并分析月球样品要比检测存在更为困难，但这对于将来利用月球水资源（如饮用水、制氧和火箭燃料）至关重要。
