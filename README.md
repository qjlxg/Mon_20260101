挪威前首相因爱泼斯坦档案受到反腐调查
挪威警方已对挪威前首相亚格兰启动“严重贪腐”调查，力求查明他与美国已故性罪犯爱泼斯坦的关联。亚格兰也曾担任欧洲委员会秘书长。
法新社报道，挪威经济和环境犯罪调查起诉局（Okokrim）星期四（2月5日）发表声明说，调查源于美国日前公布爱泼斯坦案相关档案所披露的信息。调查重点在于核实亚格兰（Thorbjorn Jagland）担任挪威诺贝尔委员会主席和欧洲委员会秘书长期间，是否收受美国富商爱泼斯坦提供的礼物、旅行资助或贷款等不当利益。 
Okokrim局长隆塞特说：“亚格兰在已公布文件所涵盖的时期担任诺贝尔委员会主席和欧洲委员会秘书长，因此我们有充分理由对他进行调查。”
挪威外长艾德说，鉴于亚格兰曾任欧洲委员会秘书长并享有相关公务行为的豁免权，挪威政府将向欧洲委员会部长委员会提出建议，请求解除他的豁免权，以便调查顺利推进。
挪威首相斯特勒当天对媒体说，针对亚格兰的调查凸显这起案件的严重性。他强调，外交豁免权不能成为警方执法的障碍，挪威政府支持通过调查还原真相。
亚格兰的律师告诉法新社：“我们平静等待调查结果。”
他说：“对亚格兰而言，从Okokrim那里得到一个权威性的澄清，那是好事，总比整个媒体界各自开展小规模调查要好得多。”

中国就预制菜国标征求意见 提出保质期不超一年
中国就预制菜国标发布征求意见稿，明确预制菜定义，并要求尽量缩短预制菜的产品保质期，最长不应超过12个月。
中国国家卫生健康委星期五（2月6日）发布《食品安全国家标准 预制菜》（征求意见稿），面向社会公开征求意见。
意见稿明确，预制菜是以一种或多种食用农产品及其制品为原料，使用或不使用调味料等辅料，不添加防腐剂，经工业化预加工制成，配以或不配以调味料包，加热或熟制后方可食用的预包装菜肴产品。
标准要求，预制菜不包括主食类食品、净菜类食品、即食食品和中央厨房制作的菜肴，而且应尽量缩短预制菜的产品保质期，最长不应超过12个月。
标准也要求，预制菜不得使用防腐剂，尽可能减少食品添加剂的使用。
去年9月，中国连锁餐厅西贝与网红创业者罗永浩就预制菜问题发生争执，引发公众关于预制菜标准的讨论，以及餐厅是否应该明示使用了与预制菜。
在上述意见稿发布的同一天，据央视新闻报道，中国国务院食安办、市场监管总局、商务部也已联合起草了推广餐饮环节自主明示的公告公开征求意见稿。
意见稿提出，鼓励餐饮服务提供者根据自身经营实际，自主明示菜品加工制作方式。使用预制菜、中央厨房成品或半成品、预包装食品的，明示内容应真实准确。
意见稿说，餐饮服务提供者可以通过菜单或点餐小程序标注、企业官方网站或公众号介绍、门店显著位置展示、应消费者询问后回应等多种方式进行明示。

【澳新银行：美伊双方仍存在很大分歧 地缘政治风险溢价保持不变】金十数据2月6日讯，在欧盘交易中，油价小幅上涨，但预计本周将出现数周以来的首次周度下跌，因为市场正关注美国和伊朗即将举行的会谈。布伦特原油和西德克萨斯中质原油价格均上涨1.1%，分别达到每桶68.28美元和63.13美元。然而，这两项基准价格本周累计下跌超过1%，因为华盛顿和德黑兰之间的谈判缓解了对美国可能采取军事行动从而威胁该地区石油基础设施的担忧。不过，紧张局势仍处于高位。澳新银行研究公司的分析师表示：“双方仍存在很大分歧。这应该会使地缘政治风险溢价保持不变。”

【凯投宏观：油价在2026年底前跌向每桶50美元水平】金十数据2月6日讯，周五，油价在经历了前一日的暴跌后反弹超过1%，但仍有望录得近两个月来的首次周线下跌，原因是供应担忧有所缓解，投资者将注意力集中在当天晚些时候举行的美伊会谈结果上。布伦特原油期货一度上涨1.2%，至每桶68.33美元。伊朗和美国在阿曼举行的会晤议程缺乏共识，令投资者对地缘政治风险感到担忧。但凯投宏观指出，地缘政治担忧终将让位于疲软的基本面。哈萨克斯坦的石油产量在经历了1月份的严重中断后有望恢复。这将有助于推动油价在2026年底前回落至每桶50美元左右。
