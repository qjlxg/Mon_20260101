临近两会 部分中国钢企被要求减产
分析人士和咨询机构称，为确保中国“两会”（全国人大、政协年会）期间的空气质量，中国北方的部分钢铁企业须从3月4日起减产至少30%。
据路透社报道，在两会期间，中国钢铁厂尤其是靠近首都北京的北方地区的钢铁厂，通常会被要求控制产量，以改善空气质量。两会将于下周在北京召开。
咨询公司Mysteel星期三（2月25日）在报告中称，中国北方多家钢铁厂已接到通知，自愿启动限产措施，在3月4日至11日期间将高炉产量至少削减30%。
咨询机构兰格钢铁的副总经理辛戈（译名，Xin Ge）说，限产措施有望放缓钢材库存的季节性积累，从而对钢铁价格形成支撑。
辛戈说，尽管高炉产量下降意味着对原料需求减少，但钢价走高，加上市场预期两会期间可能出台刺激政策，将促使钢铁厂在两会后补充原料库存，为后续扩大生产做准备。
作为最大的钢铁生产国和消费国，中国的钢铁需求通常会在3月回升，天气转暖带动建筑活动恢复。
另据北京市公安局交通管理局网站星期三（2月25日）消息，北京市公安局交通管理局发布通告称，为保证北京市道路交通安全与畅通，根据道路交通安全法律法规有关规定，决定自3月2日0时至3月13日24时，全天禁止运载危险化学品（含剧毒化学品）车辆在北京市行政区域内道路行驶。

台北市推动育儿减少工时1小时
为促进生育，台北市推动育儿减少工时计划。
据中时新闻网报道，台北市长蒋万安星期三（2月25日）宣布，台北市自3月1日起将推动育儿减少工时计划。
计划针对设籍台北、家中有12岁以下孩子且需亲自接送的爸妈，只要企业不减薪且让家长每天减少1小时工作，且符合资格者，台北市府将提供企业八成补助，每人最高1万5000元（新台币，下同，约606新元）、每家企业总补助10万元。
计划旨在让家长重新拿回生活主导权，有弹性地安排职业生涯和育儿。
台北市劳动局长王秋冬表示，许多家长在育儿阶段常面临工作与家庭难以兼顾的压力，尤其是在孩子仍需接送与陪伴成长的时期，更需要制度支持。
王秋冬说，推动这项计划，正是希望协助企业体贴员工的实际需求，打造更友善的育儿职场环境，协助家长在孩子成长关键期多一点陪伴，也让企业透过弹性制度提升员工向心力与稳定度。

英国内政部宣布，自2026年2月25日起，来自包含新加坡在内的85个国家的访客必须在出行前获得电子旅行许可（ETA），否则将被禁止登机。该措施适用于不需签证入境英国的旅客，网上申请费用为16英镑（约21.6美元）。
ETA计划自2023年推出，并于去年4月扩展至欧洲旅客，但此前执行并不严格。内政部表示，从本周三起，持有ETA将成为强制要求，航空公司将拒绝无ETA、电子签证或其他有效入境证件的乘客登机。英国与爱尔兰公民（含双重国籍者）及拥有英国居留权者免于申请。
移民事务大臣迈克·塔普表示，ETA是加强英国边境安全、提供更高效现代化服务的重要举措。报道并指出，欧盟自去年10月开始对入境的英国国民实施入出境电子系统（Entry/Exit），该系统虽取消护照人工盖章并建立数字记录，却在部分机场引发延误，业界担忧全面实施可能在复活节期间造成更大混乱。
