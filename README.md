倡导者警告，倘若实施，普通用户将承担不便与风险，而有心作恶者往往能规避监管，措施未必能达致宣称的安全效果。

中国加强加密货币监管，扩大打击范围至现实世界资产（RWA）代币化。
周五，中国人民银行牵头八部门发布通知，禁止境内开展的RWA代币化活动及为证券发行、资金融通或财务运作提供相关中介或技术服务；经有关部门批准的个别事项可例外处理。
通知指出，任何境外机构或个人不得以任何形式非法向境内主体提供RWA代币化相关服务。基于境内权益在境外开展RWA代币化或准资产证券化的，由“同业同风险同规则”原则严格监管，未经批准不得开展。
此外，中国金融机构的境外子公司以及参与此类跨境业务的中介和技术服务商须加强合规与风险控制，落实客户适当性和反洗钱措施，并向监管机构报备或申请批准。
通知还明确，境内主体及其控制的境外实体未经有关部门批准不得在境外发行虚拟货币；任何中外机构发行与人民币挂钩的离岸稳定币，亦须经批准。
