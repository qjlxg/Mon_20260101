日产美洲区董事长：欢迎中国车企进军拉美 竞争让行业更强
日本汽车巨头日产美洲区董事长穆尼耶说，公司欢迎中国汽车制造商进军拉丁美洲及其他市场，认为竞争将使行业更强大，并让消费者受益。
据彭博社报道，穆尼耶（Christian Meunier）在圣保罗受访时说：“竞争不是坏事。我们的责任是为客户提供比中国品牌更好的产品、服务和整体体验。只要做到这一点，就没有问题。”
近年来，中国电动车制造商加速布局拉丁美洲市场，重点拓展巴西和墨西哥，并以更具竞争力的定价策略消化中国国内德过剩产能。大量中国汽车涌入，也促使巴西和墨西哥德上调关税，以缓冲进口激增带来的冲击。
在关税压力下，部分中国车企开始在当地设厂生产，以巩固不断扩大的市场份额。不过，即便转向本地化布局，仍需面对成熟竞争对手在品牌、渠道和服务网络方面的优势。
穆尼耶指出，中国品牌的海外扩张在一定程度上受益于政府补贴，但普遍缺乏对长期发展至关重要的经销与售后服务体系。“有些成功，有些则不那么成功。我不知道其中有多少能够存活下来。”
他还说，日产目前没有计划与中国企业在巴西开展合作。
日产位于巴西的工厂在2023年获得28亿雷亚尔（6.7亿新元）投资，目前生产发动机，以及Kait紧凑型跨界车和Kicks小型SUV车型。

会议提出，要针对江苏东海、湖北宜城烟花爆竹爆燃事故暴露出的违规行为，加强事故警示并层层传导到乡镇街道和每一个烟花爆竹零售点。要举一反三、深刻吸取事故教训，把“严禁‘下店上宅’经营烟花爆竹，严禁超量储存，严禁将产品摆放至店外，严禁在零售店周边100米内燃放试放，严禁在禁放区及人员密集场所燃放烟花爆竹；务必专人现场盯守，务必严控进店人数，务必张贴‘五个严禁’警示标语”的八项硬措施落地落实落到位。
会议提出，要坚决防止把隐患排查异化成形式主义动作，切实提升发现问题隐患的强烈意愿和能力水平，提高排查针对性、专业性、具体化。要盯住烟花爆竹零售店铺等重点部位和周边，盯住初五、元宵节等重点时段，把各项措施落实到具体场所、具体点位、具体人员。

中国科研团队研发出AI看病系统   可诊断罕见病
中国科研团队研发出一个人工智能（AI）看病系统，可有根有据地诊断罕见病。
据央视新闻报道，上海交通大学医学院附属新华医院孙锟教授、余永国教授领衔团队与上海交通大学张娅教授、谢伟迪副教授领衔团队联合研发出一款能看罕见病的人工智能系统，相关成果星期四（2月19日）在国际顶尖期刊《自然》上发表。
这套名为DeepRare的系统，被称作全球首个“推理过程可追溯”的罕见病智能医生。
长期以来，罕见病很难确诊，病人常常跑多家医院也查不出是什么病。以往的医疗AI虽然能给出诊断结果，但就像一个只给结论、不给依据的医生，医生们不敢轻易相信。这套新系统最大的不同，是它每下一个诊断，都会附带完整的“证据链”——就像老医生查房时，一步一步讲清楚为什么这么诊断。
系统的“脑子”里装满了全球顶尖的医学知识和真实病例。它不只是简单查资料，而是会像人类医生一样思考：先提出假设，再找证据验证，发现不对还会自我纠正，反复推敲后才下结论。
测试数据显示，就算没有基因检测结果，只靠病人的临床症状，这套系统也能判断，第一次诊断的准确率达到57.18%，比原来的国际最好模型提高了近24个百分点。这意味着，在缺少基因检测设备的基层医院，医生也能用它给罕见病患者做初步筛查。如果加上基因数据，系统的诊断准确率能超过70%。
孙锟教授介绍，研发团队下一步计划发起一个全球性的罕见病诊疗联盟，并在未来半年内，用两万个真实病例对这套系统进一步验证，让它能帮到更多患者。

马尼拉讯——国家测绘与资源信息局（Namria）周三发布了更新后的菲律宾地图，明确标示“西菲律宾海”，并称该命名自2012年起即纳入官方地图。
2012年，时任总统贝尼尼奥·阿基诺三世签署第29号行政命令，将菲律宾群岛西侧海域命名为西菲律宾海，并指示Namria制作和公布反映该命名的海图与地图。
此举系回应参议员罗丹特·马尔科莱塔对“西菲律宾海”存在性的质疑——他称该海域无明确坐标，并认为卡拉延群岛（Kalayaan Island Group）位于国家专属经济区（EEZ）之外。
Namria解释称，根据国际法，专属经济区自群岛基线向外延伸至200海里，形成“连续的弧形边界”，而非由直线构成的多边形，因此用少量坐标无法准确描述海域界限。该边界通过依据《第9522号共和国法》（即基线法）确定的精确地理空间数据来表示，以便航行系统和制图软件正确判定界线。
2016年常设仲裁法院裁决支持菲律宾在西菲律宾海的主权权利，认定中国在南海的广泛主张不具国际法依据。
Namria并指出，其航海图遵循国际水道测绘标准，供菲律宾及外国船舶使用，并提交给包括国际水道测绘组织在内的相关国际技术机构，通过既有技术程序将菲律宾地理信息纳入全球海事记录，支持安全航行与负责任的海上活动。
菲律宾大学海事与海洋法研究所所长律师杰伊·巴通巴卡尔周四证实，Namria发布的是最新版地图，且西菲律宾海自2012至2013年间已出现在该机构地图上。他并指出，各国可将地图提交给联合国秘书长办公室以便传播，联合国在领土主张上予以承认但保持中立。
Namria表示将继续履行其维护国家主权与海上安全的职责。
