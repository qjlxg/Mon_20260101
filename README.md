酒驾丈夫撞上醉驾妻子 四川夫妻双双被罚
四川官方通报，当地一对夫妻酒驾引发连环撞，双双被罚。
综合华西都市报和大象新闻报道，四川省乐山市夹江县公安局星期天（5月10日）晚发布通报称，该局交通管理大队在处理一起追尾事故时，查获一对夫妻分别酒后驾车的违法行为。
事发当晚，夹江江姓男子和罗姓妻子聚餐饮酒后心存侥幸，认为“时间晚不会被查”，分别驾驶轻型多功能用途货车和小型轿车，沿国道245线由峨眉方向往夹江城区方向行驶。当两人驾车行驶至界牌909红绿灯路口时，江男驾驶的车辆与罗女的车辆发生碰撞，撞击后，罗女的车辆失控与前方等候交通信号灯的小型轿车发生追尾，事故造成三车不同程度受损。
事故发生后，民警迅速赶赴现场处置。民警在调查中得知江男、罗女为夫妻，且在沟通过程中，民警闻到双方身上散发浓烈酒味，当即对两人开展呼气式酒精测试，结果均涉嫌酒后驾驶机动车。
民警于是依法将两人带至医院抽血检测，最终江男血液中酒精含量为48.7mg/100mL；罗女血液中酒精含量高达151.4mg/100mL，已达到醉酒驾驶机动车标准，涉嫌危险驾驶罪。
夹江县公安局交通管理大队目前依法对江男饮酒后驾驶机动车违法行为，作出罚款1600元人民币（299新元）、驾驶证记12分、暂扣六个月的处罚；对罗女醉酒后驾驶机动车违法行为，作出吊销驾驶证、五年内不得重新申领，并移送检察机关审查起诉，依法追究刑事责任。

欧洲疾病预防控制中心（ECDC）表示，作为预防措施，因发生致命汉坦病毒疫情而受影响的邮轮上所有乘客在下船时都被视为高风险接触者。该邮轮预计于5月10日在西班牙特内里费岛外停泊，ECDC在5月9日的加急科学建议中称，无症状乘客将由各国通过专门安排的交通工具（而非普通商业航班）遣返并自行隔离。
各国正准备在格林威治标准时间（GMT）约6:30至7:00（新加坡时间14:30至15:00）开始撤离MV Hondius上的本国公民。世界卫生组织（WHO）5月8日通报，已有8人患病，其中3人死亡（包括一对荷兰夫妇和一名德国公民）；WHO称6例已确诊感染，另有2例为疑似病例。ECDC指出，虽在下船时均列为高风险，但返国后并不一定全部被认定为高风险个案。
在邮轮预计停靠的特内里费格拉纳迪利亚·德·阿博纳港已设立移动医疗点。ECDC敦促对有症状者在抵达时优先进行医学评估和检测，根据病情可能在岛上隔离或医疗转运回国。汉坦病毒通常由啮齿动物传播，极少情况下可人传人，卫生部门认为病毒进一步扩散的风险较低。

特朗普政府内部出现“婴儿潮”，但美国整体生育率仍在下滑。
5月1日，白宫新闻秘书卡罗琳·利维特在华盛顿诞下女婴维维安娜，成为首位在任内分娩的白宫新闻秘书。副总统J·D·范斯的妻子乌莎预计7月分娩，将迎来第四个孩子；总统办公厅副主任斯蒂芬·米勒之妻也将迎来第四个孩子。此前，维韦克·拉马斯瓦米在竞选俄亥俄州州长期间于2月迎来第三个孩子。参议员马可·鲁比奥育有四名子女，国防部长皮特·赫格塞斯有七名子女。特朗普本人是五子之父、11名孙辈的祖父，曾自称“助孕总统”，并推动所谓的“婴儿潮”。
2025年10月，白宫提出扩大体外受精（IVF）可及性、降低相关药物成本并鼓励雇主报销等措施，但批评者认为覆盖仍不足。同时提出为任期内新生儿每人提供1000美元并投资于低费率指数基金，以及将Title X计划从避孕服务向生育与家庭政策倾斜。保守派智库提出削减高等教育经费等更具争议的建议。
不过人口数据并不乐观：2025年新生儿为3,606,400人，较2024年下降1%；育龄妇女出生率降至每千名15至44岁妇女53.1例，总和生育率降至1.6，远低于维持人口更替所需的2.1水平。自2007年以来出生率已下降逾25%；30岁以下女性生育率暴跌，30岁以上回升不足以弥补缺口。高校及高中学历女性生育分别下滑约20%与50%；近半数30岁女性无子。
专家指向晚婚（目前男性中位数30岁、女性29岁）、住房、医疗、教育成本上升及学贷负担等结构性因素。特朗普承认政府能力有限，但政治上也有考量：支持他的县域出生率普遍高于民主党支持区。媒体热传白宫新生儿照片，但专家认为这些高调的个案难以扭转全美低生育趋势。

行程方面，特朗普将于5月13日上午抵京，14日参加迎接仪式并会见习近平，下午参观天坛，晚间出席国宴；15日将共进茶宴与工作午餐后离京。
