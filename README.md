【世界杯前瞻:梅西萨拉赫巨星对决,埃及轻装上阵挑战阿根廷】
        北京时间7月8日凌晨0点,2026年美加墨世界杯1/8决赛一场焦点战,阿根廷对阵埃及。
        本场比赛将在美国的亚特兰大体育场进行,主裁判是法国名哨莱特西耶。
        埃及将是阿根廷队本届世界杯面对的第3支非洲球队。在世界杯历史上,阿根廷对阵非洲球队的战绩是8胜1负,1990年揭幕战0-1小负喀麦隆后,阿根廷在世界杯对阵非洲球队已经取得8连胜。
        这场比赛的焦点人物是梅西与萨拉赫,两位超级巨星注定有一人要告别世界杯。赛前,在被问到世界杯的最后一舞想面对谁时,萨拉赫毫不犹豫给出了自己的选择:梅西。
        埃及走到现在已经没有压力,他们在本届取得了队史世界杯首胜,又在淘汰赛更靠前了一步,面对阿根廷,轻装上阵也许会让他们有着不错的发挥。
        阿根廷与埃及历史交锋两次,第一次在1928年奥运会,阿根廷6-0击败埃及。第二次是2008年在一场友谊赛,阿根廷凭借阿圭罗与布尔迪索的进球2-0取胜,梅西没有出战那场比赛。

【世预赛欧洲区24支晋级球队确定,第二阶段分组出炉】
        世预赛欧洲区小组赛第一阶段全部结束,爱沙尼亚、拉脱维亚、斯洛文尼亚和瑞典成为最后四支锁定第二阶段参赛资格的球队。至此,参加第二阶段的24支球队全部确定。
        卫冕冠军德国领衔晋级名单,共有21支球队在上一届预选赛中也曾晋级第二轮。克罗地亚和波兰在错失2023年世界杯后重返赛场,而葡萄牙队则是历史上首次晋级第二轮。
        获得参赛资格的球队名单如下:波黑、克罗地亚、爱沙尼亚、芬兰、法国、格鲁吉亚、德国、希腊、匈牙利、冰岛、以色列、意大利、拉脱维亚、立陶宛、黑山、荷兰、波兰、葡萄牙、塞尔维亚、斯洛文尼亚、西班牙、瑞典、土耳其和乌克兰。
        第二阶段预选赛将以四组六队的形式继续进行,第一轮的所有比赛结果都将进入下一阶段。具体为:A组和B组的队伍将组成I组,C组和D组将合并为J组,E组和F组将组成K组,而G组和H组的队伍将组成L组。
        第二轮比赛将于2026年8月、2026年11月和2027年2月三个窗口期进行,四个小组中每个小组的前三名球队将晋级世界杯,欧洲共有12支国家队参加世界杯。
