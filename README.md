能源部加强稽查油品短缺，追溯炼厂与油库日报资料，盼1–2周内得出结论
针对近期加油站燃油短缺引发的供应链透明度质疑，能源商业署（กรมธุรกิจพลังงาน）已要求全国炼厂与油库提交自2月1日起的每日库存与出货纪录，以与3月的销售数据比对分析。自3月20日起，业者须逐升上报数据，未报者将依法处理。
当局检查并比对运输单据，预计逐步汇整结果于1–2周内公布。对外流往邻国（如寮国）的监测未见异常。为掌握实况，已设专责中心并联络各省能源办公室通报缺油点，协调业者按第7条紧急供货。
截至目前已巡查2,649处加油站：247处关门（9.1%）、1,912处部分缺油或接近售罄（72.2%）、496处供应充足（18.7%）。主要成因为需求暴增（45.7%）、业者配给减少（40%）、运输车不足（7.4%）及油库缺货（6.8%）。
3月21–23日与相关单位检查7个油库未发现囤货异常；3月25日在沙拉武里县查获3处非法储油约40,000升，已扣押并将依法追究，非法者最高可处2年以下有期徒刑或罚款不超过20万铢，若涉牟利或扰乱市场另追其他罪名。
物价与补贴：泰国柴油零售价约38.94铢/升，在区域内仍属偏低；马来西亚柴油调涨后为45.59铢/升（无同等税制比较）。国家油价基金每日补贴柴油约19.12铢，负担近17亿铢/日，危机以来累计补贴逾380亿铢。能源部正与财政部研议下调特种税及其他救助措施，且加油站缺油状况已有缓解，持续密切监控。
