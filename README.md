医保局明确要求湖南、河南省医保局指导和督促辖区各级医保部门对此次被约谈单位的后续整改情况进行监督，及时报送整改情况。

白宫的通稿未提及上述“新定位”，更侧重具体成果，称双方讨论扩大美企进入中国市场、增加中国对美投资和采购美农产品，并就霍尔木兹海峡保持畅通达成一致。白宫还称习近平有意多购美油以减少对霍尔木兹海峡的依赖（约45%—50%的中国原油经该海域运输）。
白宫通稿未提及台湾问题；习近平此前称台湾是中美关系中“最重要的问题”。中方通稿表示双方就中东局势、乌克兰战争、朝鲜半岛等重大国际和地区问题交换了看法；白宫则强调习近平在伊朗问题上总体与美方立场一致，反对伊朗在霍尔木兹设卡并认为伊朗绝不可获得核武器。
两国领导人在公开致辞中语调积极：习近平呼吁“做伙伴、非对手”，称2026年可望成为中美关系“历史性、里程碑式”的一年；特朗普称两人“相处一直很好”，并在国宴上邀请习近平9月24日访美。两人还共同出席欢迎仪式并参观天坛。北京在访问前允许数百家美屠宰场恢复对华牛肉出口，作为善意举措。
观察人士认为，此次峰会总体“积极但克制”，目标并非彻底重置关系，而是在长期竞争背景下建立更稳定、可预测的框架。特朗普于5月13日晚抵京，由国家副主席韩正迎接，5月15日结束行程；随行有多位美企高管。

美国华盛顿 — 5月13日，美国参议院的共和党人阻止了由民主党推动、要求未经国会授权即结束对伊朗战争的最新决议，但随着第三名共和党参议员投赞成票，该决议仍更接近通过。参院以50比49投票不推进这项由俄勒冈州参议员杰夫·默克利提出的战争权力决议，投票几乎沿党派划分进行。肯塔基的兰德·保罗、缅因的苏珊·柯林斯和阿拉斯加的丽莎·墨科斯基三名共和党人加入了除宾夕法尼亚民主党人约翰·费特曼外的全部民主党人行列支持该议案。
这是今年参院第七次由共和党人阻止类似决议，也是自5月1日冲突触及《1973年战争权力法》规定的60天期限以来的首次投票。该法规定，总统在未获国会授权时只能发动60天军事行动，或在因“不可避免的军事必要性”撤军时请求延长30天。特朗普此前称停火“终止”了敌对行动，民主党人则反驳称冲突仍在继续，指出美方封锁伊朗港口并袭击伊朗船只，伊朗封锁霍尔木兹海峡并攻击美方船只和油轮。默克利称“双方仍在交战，60天时钟不应被中止”。
参院民主党人表示将于下周及以后每周再次提出相同决议，直到战争结束或特朗普向国会寻求授权。众院民主党人也提出了相关决议，但同样被共和党阻止。民主党强调宪法将宣战权赋予国会，警告总统可能将美国拖入长期冲突；共和党和白宫则称总统作为三军统帅有权下达有限军事行动。路透社

新德里 5月15日 — 伊朗外交部长阿拉格奇15日在新德里表示，伊朗对美国“毫无信任”，只有在美方真正表现出诚意时才会与其谈判。该言论是在他出席印度举办的金砖国家外长会期间发表的。
阿拉格奇称，只要与伊朗海军协调，除与德黑兰“交战”的船只外，其余船只可通航霍尔木兹海峡，但他同时指出该海区形势“非常复杂”。
自今年2月伊朗与美以冲突爆发后，霍尔木兹海峡已被伊朗在很大程度上封堵——该海域此前约承担全球约五分之一的石油与天然气运输。4月虽达成停火，但围绕永久和平的谈判因双方互相拒绝对方方案而中断，巴基斯坦居中调停被形容为“遇到困难”。
阿拉格奇称，伊朗希望维持停火以给外交留出空间，但也已做好恢复战斗的准备。阻碍谈判的主要问题包括伊朗的核问题与对霍尔木兹的控制。他并表示，欢迎中国等有能力国家的斡旋，称伊中关系良好，并希望通过谈判实现海峡安全与航运恢复。来源：路透社

基辅消息：据《南华早报》报道，俄罗斯总统普京将于5月20日访华，此行时间仅在中国国家主席习近平与美国总统特朗普在北京会晤数日之后。

彭博社援引消息称，4月习近平在会见俄外长拉夫罗夫时强调双边关系稳定并承诺深化协调；5月初普京表示，两国“原则上”在推进油气领域合作上达成共识，并希望与习近平就核能问题进一步磋商。

哈佛教授格雷厄姆·艾利森认为，特朗普与习近平正努力摆脱“修昔底德陷阱”。（费城）——尽管外界对峰会起初期望不高，艾利森仍看到重大且多面的积极信号。首日成果之一是三场更多元的领导人会晤提上日程：特朗普公开邀请习近平于9月24日访美，习近平尚未正式回应；若持续对话成行，峰会可在分歧依旧的情况下收尾于一线希望，难题包括台湾与出口管制等。
随后两场面对面会谈可能分别在11月深圳亚太经合组织会议及迈阿密二十国集团峰会举行。习近平首日致辞强调两国应遏制对抗、平等相处，并引用艾利森2017年著作中关于“修昔底德陷阱”的概念，提出“中美能否克服修昔底德陷阱、建立大国关系新范式？”这一问题。特朗普则在Truth Social上将世界格局问题归咎于其前任乔·拜登。

对习近平而言，此次峰会有利于维持中国继续崛起的宽松环境；对特朗普而言，推动强劲经济对其11月3日中期选举及树立“和平特使”历史定位至关重要。报道：海峡时报美国首席记者Bhagyashree Garekar。

基辅消息——乌克兰总统泽连斯基5月15日表示，俄方正试图将白俄罗斯进一步卷入对乌战争，并正在权衡从白俄罗斯境内对乌克兰北部或直接对北约国家发动攻击的方案。
