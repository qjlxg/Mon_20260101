【壳牌：尽管伊朗局势动荡，但液化天然气需求仍将持续增长】金十数据3月16日讯，壳牌公司认为，尽管中东地区的冲突导致天然气价格出现波动，但全球对液化天然气的长期需求仍将持续增长，因为这种能源具有灵活性和可靠性。该公司周一表示，到2050年，这种燃料的消费量将较2025年的水平增长至少45%。该公司预计，到本世纪中叶，其年需求量将在6.1亿至7.8亿之间。该报告发布之际，全球液化天然气市场因中东地区的战争而遭受重创，通过霍尔木兹海峡（这是燃料运输的重要通道）的运输量已大幅减少，几乎降至零。虽然壳牌公司未对这场冲突的影响发表评论，但该公司已宣布因卡塔尔最大出口工厂的停产而触发不可抗力条款，这一条款允许公司中断对某些客户的发货。
