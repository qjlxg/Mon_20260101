韩联社星期四（6月4日）报道，投票结束后，韩国三大电视台联合发布的出口民调显示，预计吴世勋得票率为46%，落后于51.4%的郑愿伍，且差距超出误差范围。开票初期，吴世勋也一度明显落后于郑愿伍，败势较为明显。但午夜过后，他迅速缩小差距，并在凌晨时段实现反超，最终逆转出口民调结果，于周四上午锁定胜局。
至此，吴世勋成为首尔市长选举历史上首位五度当选的市长。
吴世勋于2006年首次当选首尔市长，并于2010年成功连任。但2011年8月，围绕学校免费供餐问题举行的居民投票因投票率未达法定门槛而无效，吴世勋随后引咎辞职。此后，他先后参加2016年和2020年两届国会议员选举，但均未能当选。
经历政治低潮后，吴世勋因时任首尔市长朴元淳去世而于2021年补选中胜出，重返首尔市政舞台。随后，他在2022年地方选举中成功连任，成为四届首尔市长；此次再度胜选，创下“五度当选”首尔市长的历史纪录。
吴世勋的主要竞选承诺包括到2031年新增31万套住房供应、推进城市更新第二阶段项目、加快江北及首尔西南地区开发、完善交通网络、增加深夜及凌晨公交运力，以及打造“10分钟运动生活圈”等民生项目。
随着此次胜选，吴世勋不仅确保了首尔市施政路线的延续，也进一步提升了他在韩国保守阵营中的政治影响力，巩固了他作为下届总统潜在竞逐者的地位。

韩中七年来首次新增航权
韩国国土交通星期四（6月4日）通报，韩中同意去年首次新增航权，两国每周往返航班将有所增加。
综合路透社和韩国国际广播电台报道，这项协议在5月27日至28日于首尔举行的韩中航空会谈中达成。韩国国土交通部称，每周客运航班数量将由608趟次增至664趟次，货运航班数量则由每周54趟次增至68趟次。
上述会谈商定新增70个往返韩中两国航班班次的航权额度。
这是继2019年之后，韩中两国再新增航权。韩国国土部介绍，今年首季度韩中航班承运旅客约439万人次，高于冠病疫情暴发前的414万人次。
具体来看，从仁川飞往北京、上海、广州、大连、成都、哈尔滨的航班每周均新增七班，总计新增42班。
从釜山、清州等韩国地方城市飞往广州、重庆、深圳等中国10座主要城市的地方机场航班每周新增14班。
货运航班方面，往返中国四大货运机场（天津、郑州、鄂州、合肥）与韩国机场的航班每周新增14班。
韩国国土部表示，将在今年下半年内迅速向韩国航司分配新增航权，并为航班增设提供支持。

金正恩视察新投产核物质厂 强调增强核武力
朝中社报道，朝鲜领袖金正恩周三视察了新投产的核物质厂，强调必须加快提升核战争遏制力。
报道称，金正恩指出，鉴于朝鲜革命面临与最凶恶敌人长期较量的特殊环境，以及现实威胁和潜在威胁持续加剧、长期危机难以预测的形势，必须持续从质量和数量两方面加快提升核战争遏制力。
金正恩还说，作为维护国家安全利益和发展权的根本保障及强有力安全屏障，这一历史使命的紧迫性和责任感进一步凸显。
朝中社指，有关核武力强化的重要协商会议周三同时召开。金正恩在会上作出重要结论，明确相关实践方针。“我们今天更新核活动中的一些重要数据，全面考虑核遏制力建设方面的战术及战略需求问题。我们确定了今后推动国家核力量呈几何级数增长的宏伟计划及其保障措施，这是在核能力提升方面树立变革性里程碑的历史事件。”
金正恩重申，朝鲜将坚定维护宪法和国家主权，切实保障国家安全这一意志将更加坚定、更加果断。
韩联社报道指，朝中社未提及金正恩视察工厂的具体位置及生产能力等信息。目前已知的朝鲜铀浓缩设施所在地包括平安北道宁边、南浦市降仙和平安北道龟城等三处。因此，不排除朝鲜此次公开的是新建于“第四地点”的高浓缩铀制造设施。
此外，从朝中社发布的照片来看，厂房内密集排列着一排排圆筒状离心机，这类离心机是铀高浓缩过程中的关键装置。
