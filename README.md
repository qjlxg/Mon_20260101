美国海关：对等关税退款系统45天内准备就绪
美国海关总署将在45天内建立一套系统，用于处理被最高法院裁定非法的“对等”关税退款。 
路透社报道，美国海关与边境保护局（CBP）执行主任洛德星期五（3月6日）在一份法庭文件中说，CBP无法遵守一项法院命令，即停止处理包括总统特朗普援引《国际紧急经济权力法》实施的进口关税。 
洛德向法官提交的宣誓书中说，CBP正“尽一切可能”创建一个自动化的关税退款系统，这个系统“将比现有流程更简单、更高效”，并可能在45天内投入使用。
洛德发布声明之际，正值美国政府律师与联邦贸易法官试图敲定一项全面的和解方案，以向约33万进口商退还1660亿美元（约2118亿新元）的关税款项。
美国最高法院两周前裁定，总统特朗普援引1977年《国际紧急经济权力法》对全球贸易伙伴产品加征对等关税的做法违反宪法。但特朗普随即援引《1974年贸易法》第122条款，宣布对全球征收10%的新关税，为期150天。美国财长贝森特预计这项新关税将在本周提高到15%。 
美国俄勒冈州3月5日联合另外23个州向美国国际贸易法院提起诉讼，要求阻止特朗普政府近期宣布实施的新关税措施。

据香港星岛头条网报道，香港有组织罪案及三合会调查科警司卢嘉晋星期五（3月6日）说，警方早前接获线报，得知有犯罪集团计划进行抢劫。经部署，警方星期五早上9时在上环地铁站内采取行动，以“串谋行劫”罪拘捕三名年龄介于19至31岁的男子。
调查显示，案中两名受害人分别为澳门及香港居民，为亲戚关系。两名受害人星期五一早携带总值约港币1000万元的外币，包括美元和人民币，前往上环鿋利街一间找换店兑换港币后，在上环地铁站内被两名匪徒持刀行劫。在场埋伏的警员当场制服两名匪徒，受害人没有财物损失。

三名被捕男子持有香港身份证，其中一人持有菲律宾护照，现正被警方扣留调查，案件交由有组织罪案及三合会调查科继续跟进。
卢嘉晋续称，警方注意到近来先后有三宗找换店相关劫案，警方高度重视，因此作出情报主导行动，配合锐眼计划闭路电视系统，收集到相关证据，侦破所有案件。目前没有证据显示受害人与被捕人有关联，警方会继续调查涉案人员如何得知当天有人会进行大额兑换，以及涉案人员或其他人事前是否到场踩线等。

中国科研团队在空间激光通信领域实现新突破：实验在距地面4万多公里（2.5万英里）高空的卫星与地面之间，长时间维持了高速激光链路。
该项目由中国科学院光电研究所牵头，团队在云南用一台1.8米口径望远镜在4秒内完成对一颗地球同步轨道卫星的锁定。实验持续超过三小时，双向数据传输稳定维持在1吉比特每秒（1Gbps），全过程未中断，相关情况由《四川观察》报道。
研究团队称这是在高轨道实现长时实时通信的“领先突破”，因为高轨通信极易被微小的指向误差或大气扰动中断。首席研究员刘超指出，以往高轨卫星与地面之间的通信常常不稳定、时长很短，有时只有几分钟；而卫星向地面的数据速率虽快，地向卫星的回传通常较慢，导致难以实现实时互动。“就像你发十条消息只收到一条回复，很难进行高效对话。”

布鲁克林联邦法院3月6日裁定，一名与伊朗有关联的巴基斯坦男子阿西夫·拉扎·梅钱特（Asif Raza Merchant）犯有密谋在美国刺杀时任总统唐纳德·特朗普或其他美方官员的罪名。检方称该阴谋系为报复美军2020年在巴格达无人机袭击中击毙伊朗革命卫队指挥官卡西姆·苏莱曼尼。
检方在审理中表示，梅钱特试图雇佣刺客在美国对政治人物或政府官员实施暗杀。梅钱特在3月4日作证称，他为保护在德黑兰的家人被迫卷入此案，并预计在有人被杀之前就会被捕；他称并未被下令杀害特定人员，但其伊朗联络人曾提到特朗普、前总统乔·拜登和前联合国大使妮基·黑利三人。
检方发言人向法新社表示，梅钱特在两项指控——跨国恐怖主义和雇凶谋杀——上被定罪，量刑日期未定，可能面临终身监禁。美国官员此前称梅钱特与伊朗“关系密切”，并形容该阴谋“与伊朗政权的做法相符”。梅钱特于2024年7月12日被捕，当时正计划离境。

华盛顿消息——3月6日，美国总统唐纳德·特朗普在接受CNN电话采访时表示，继伊朗之后，古巴将成为他的下一个议程，并预测这个共产党执政的小岛“很快会垮台”。他称古巴“非常渴望达成协议”，并表示国务卿马可·卢比奥将负责古巴事务。
特朗普与卢比奥——后者为古巴移民之子——不讳言希望在哈瓦那实现政权更替，此前两人已将目标投向曾被华盛顿视为眼中钉的前委内瑞拉领导人尼古拉斯·马杜罗。
美国驻哈瓦那大使馆3月6日在社交平台X上称，国务院古巴事务协调员罗布·艾利森近期在哈瓦那与天主教高层会面，出席者包括枢机主教胡安·德·拉·克拉里达及主教会议主席阿图罗·冈萨雷斯，双方讨论了美国对古巴的人道援助以及“改善古巴局势的必要性”。
为促成政权更替，特朗普对古巴实施能源封锁：自1月9日以来古巴未接收石油船，导致航班停降、燃料和药品短缺，加剧长期经济危机。3月4日，岛上最大电厂安东尼奥·吉特拉斯发生故障，约三分之二地区一度停电，随后大部分家庭恢复供电。
教廷长期在美古之间发挥斡旋作用：2015年教宗方济各促成奥巴马时期恢复邦交，拜登时期也促成哈瓦那释放数百名政治犯并获从美国“支持恐怖主义国家”名单中移除。特朗普于2025年1月上任当天将古巴重新列入该名单。

美国总统唐纳德·特朗普表示，白宫会后美国最大的军工企业已同意提高部分武器产量，以应对对伊战争对五角大楼弹药库存的消耗担忧。
特朗普3月6日在社交媒体称，会中讨论了“生产与生产计划”，公司同意把所谓“Exquisite class”（指高端、难产导弹）产量提高至四倍，目标是尽快达到最大规模。他并称与BAE、波音、霍尼韦尔、L3Harris、洛克希德·马丁、诺斯罗普·格鲁曼和雷神等公司CEO会面，并拟在两个月后再开会。
目前尚不清楚企业是否在当日签署新承诺。参会公司表示支持加速交付，但未给出具体新计划；诺斯罗普称将继续扩大产能，洛马称已同意“将关键弹药产量提高四倍”，RTX表示按2月与五角大楼达成的协议加速五类弹药产量。五角大楼将具体问题转给白宫回答。
特朗普同时淡化军备紧张，称中等及中上等级弹药库存“几乎充足”，并已增加相关订单。但任何额外经费需国会批准，可能耗时数周到数月。会谈发生在他呼吁伊朗“无条件投降”数小时后，冲突进入第二周、未见终止迹象，美以对伊朗发动打击，德黑兰则以无人机和导弹还击在邻国驻军的美军目标。
白宫新闻秘书称，美军弹药足以继续“毁灭伊朗政权”和完成“史诗之怒”行动。尽管五角大楼此前签署了扩大爱国者、THAAD、战斧等武器生产的框架协议，并推动B-21与固体火箭电机相关投资，但这些协议短期内难以显著增加当前对伊朗冲突的供应。分析人士称，洛克希德将PAC-3产量从约600枚提高到每年2000枚可能需三年时间。来源：彭博社。
