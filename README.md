据CNN和CBS报道称，美军最早本周末即可准备对伊朗发起打击，尽管特朗普尚未做出最终决定。战略与国际研究中心高级顾问苏珊·齐亚德指出：“地区集聚的火力本身就会产生一股势头，有时很难突然踩刹车。”
舰艇方面，美国目前在中东部署13艘作战舰艇：1艘航母（“亚伯拉罕·林肯”号）、9艘驱逐舰和3艘近岸战舰，且仍有增援在途。刚被特朗普部署前往中东的全球最大航母“杰拉尔德·R·福特”正在大西洋航行，随行3艘驱逐舰。两艘航母同时出现在中东属罕见情形；去年6月美国也曾在该区域部署两艘航母并针对伊朗三个核设施采取行动。
空中力量方面，开源航迹数据显示，多架F-22、F-15、F-16战机及KC-135加油机已抵近或在中东空域活动，另有E-3预警机与运输机参与行动。
背景上，特朗普在伊朗镇压最初由经济不满演变的大规模抗议时下令调遣“林肯”号。他曾多次警告若德黑兰杀害示威者，美方将干预，并鼓励伊朗民众接管国有机构。上月他曾因伊朗暂停逾800起处决而暂缓下令打击，但随后再度威胁采取军事行动。
美伊周二在日内瓦举行会谈以避免美方军事干预，伊朗称已就“指导原则”达成共识。白宫新闻秘书卡罗琳·里维特称虽有“小幅进展”，双方在若干问题上仍“相距甚远”，并表示“有许多理由可支持对伊朗动武，伊朗若达成协议将很明智。”

零售大亨莱斯·韦克斯纳周三在众议院监督委员会的传票下，通过位于俄亥俄新奥尔巴尼的家中作闭门证词，称自己被杰弗里·爱泼斯坦“欺骗”，否认知悉或参与其犯罪。
这位88岁的L Brands（维多利亚的秘密前母公司）创始人在一份准备好的声明中表示：“我天真、愚蠢且易受骗，把信任交给了爱泼斯坦。他是个骗子……我被欺骗，但我没有做错事，也没有可隐瞒的。”
民主党议员称韦克斯纳淡化了双方关系、未提供新细节，并多次表示记不清关键事件。但加州众议员罗伯特·加西亚表示，此次证词有助厘清爱泼斯坦如何积累用于犯罪的财富，认为若无韦克斯纳的支持，就不会有爱泼斯坦岛、专机和贩运资金。
公开资料显示，两人1980年代相识，韦克斯纳曾授予爱泼斯坦财务委托权，允许其管理投资与重大地产交易，长期将其作为主要客户并助其在富豪圈建立声誉。韦克斯纳称，他在2007年得知爱泼斯坦挪用家族巨额资金并被控诱供未成年人后切断来往。

此番证词紧随数千页政府文件公布后，对爱泼斯坦权势网络的审视再度升温。国会仍在传唤其他人物，前总统与前国务卿克林顿夫妇预计二月下旬到国会作证。幸存者与维权组织也加紧施压，与韦克斯纳有关的机构（包括俄亥俄州立大学）被要求撤除其名字。监督委员会表示将继续追查爱泼斯坦的资金与关联者，民主党人誓言追究所有协助或实施这些罪行的人。

教皇利奥周三（2月18日）在罗马阿文廷山主持传统圣灰星期三游行，正式开启为期40天、以忏悔反省为主的四旬期。
他从圣安瑟伦教堂步行至附近的圣萨比纳大殿，主礼祈祷并为信众施灰。教皇在这座五世纪大殿的讲道中表示，施于额上的灰象征“一个正在燃烧的世界”，并指向被战争摧毁的城市、国际法准则的侵蚀、受损的生态系统与破裂的社会和谐。教皇说，人们越来越难以感到自己是一个共同体，但许多年轻人“明显看得很清楚”，“我们必须从力所能及之处、与愿意的人一起开始”。
圣灰星期三标志着四旬期开始，这是为迎接复活节而进行的祈祷、禁食与施舍期。阿文廷的这一仪式是罗马最古老的四旬期传统之一，教皇经主门进入圣萨比纳的做法可追溯到基督教早期。
在马尼拉，俗称奎阿波教堂的耶稣受难主小礼拜堂周三清晨人潮涌入，信众排长队接受额上施灰。来自黎萨尔省比南戈南的艾斯·卡斯特罗说，自幼由祖母带来参加仪式，已成习惯；对他而言，圣灰标志着四旬期的开始，是向主表敬意的传统，因此从小就让额头沾灰。来自帕塞市的阿尔德林·奥比斯指出，按教会教导，人出自尘土，终将归于尘土。信众也在为期40天的祈祷和禁食做准备，奥比斯表示将多参加弥撒、念玫瑰经并遵守戒肉等禁忌，以示奉献。
为應付大量信徒，奎阿波教堂自上午5时起每小时有弥撒至11时；下午场次在12:15、15:00、16:00、17:00，最后两场在18:00与19:00举行。
