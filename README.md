基辅——乌克兰总统弗拉基米尔·泽连斯基5月6日宣布，匈牙利已将今年3月从国有奥沙德银行（Oschadbank）职员处扣押的现金和黄金归还。此举被视为基辅与布达佩斯关系进一步缓和的标志。 
泽连斯基在社交媒体上称：“这是与匈牙利关系上的重要一步——今年3月被匈牙利特别机构扣押的奥沙德银行资金和贵重物品已被归还。”他对匈方的“建设性做法和文明举措”表示感谢。 
今年3月，7名奥沙德银行员工在经匈牙利中转时被扣押，随身携带约4000万美元、3500万欧元及约9公斤金条被没收。匈方次日释放了职员，但现金和黄金一直被扣押至5月6日才归还。基辅指控这是“人质外交”，而匈方称当时按洗钱调查采取行动。 
两国关系因匈牙利亲俄立场长期紧张，但在反对派彼得·马贾尔当选并计划会见泽连斯基后出现缓和迹象。此前执政16年的欧尔班曾多次利用匈牙利在欧盟内的否决权阻碍对乌援助与对俄制裁。（法新社）

英国选民于5月7日参加地方与地区选举，普遍预期将对首相基尔·斯塔默领导的工党造成重创，并再度引发其执政能力的质疑。此次投票包含英格兰近5,000个议会席位以及苏格兰和威尔士议会，若选民转向民粹和民族主义政党，可能动摇英美长期的两党格局。
民粹派改革英国党（领袖奈杰尔·法拉奇）民调上升，或扩大在英格兰地方议会的影响，并可能在苏格兰与威尔士成为主要反对党。绿党在伦敦等城市亦有望蚕食工党的铁票区。
若工党大量失守英格兰议会席位、在威尔士议会丧失主导地位，或在苏格兰仅列第三，斯塔默将面临更大辞职压力，甚至被要求公布离任时间表。近期市场亦因担忧其可能被更左倾的领导人取代而推高英国借贷成本。
斯塔默重申将继续执政，承诺应对民生危机并推动“积极干预型政府”。他此前因任命工党资深人物彼得·曼德尔森为驻美大使而遭遇下台声浪，该任命牵出曼德尔森与已故被定罪的美国性罪犯杰弗里·爱泼斯坦及与俄中商业往来的关联。曼德尔森于2025年9月被罢免，2026年2月因涉嫌公职不当被捕，但未被起诉且不涉性行为指控。
党内基层与议员反映选举时常遭遇选民愤怒，若选情不佳，可能出现敦促斯塔默公布离任时间的书面要求。但要替换他并不容易，潜在接替人选如曼彻斯特市长安迪·伯纳姆和前副首相安吉拉·雷纳目前尚不足以发起挑战。前工党副领袖汤姆·沃森则劝告同僚勿重蹈2006年草率求仕的覆辙，警告内斗只会让选民侧目。 路透社

英国地方选举初报显示，5月8日工党在多地遭遇重创，凸显选民对首相基尔·斯塔默的不满，并在其两年前大胜后再度引发对其前途的质疑。
早期计票显示，工党在中北英前工业重镇及伦敦部分选区失血严重。脱欧活动家奈杰尔·法拉奇领导的民粹派改革英国党成为最大受益者，在英格兰已增幅逾300个议席（早报计335席），有望在苏格兰和威尔士成为主要反对力量。
民调权威约翰·柯蒂斯表示，局面比多数人预期更糟。此次选举涵盖英格兰136个地方议会及苏威两地议会，是2029年大选前最重要的民意测试。
局部结果显示，曼彻斯特大区的坦姆赛德议会近50年来首次失守；威根、萨尔福德等老矿区也大量失利。早报统计：改革党+335，工党-247，保守党-127。多数选票及苏威选举结果将于当日晚些时候公布。
党内有人称若苏格兰、威尔士失守且英格兰数千席流失，斯塔默将面临下台压力；但其盟友迅速表态反对更替，防止领导权争夺。斯塔默称将带领工党打下次大选，党内历史上未曾废黜在任首相。同时，政策反复、人事更替及任命争议（包括曼德尔森因与已故罪犯埃普斯坦关联被解职）亦被认为削弱了其权威。
分析人士认为，此次选举反映英国产党格局正由两党制向多党化演变，绿党与民族主义政党亦在局部得益。能源大臣艾德·米利班团队否认曾劝斯塔默公布离任时间表的报道。

美国国际贸易法院于5月7日裁定，总统特朗普依据1974年贸易法第122条对全球进口商品征收的10%关税缺乏正当依据，但法院以2比1的意见仅对两家私人进口商和华盛顿州下达限制令，未对所有进口商普遍停止征税。 
裁决意味着在美国政府上诉期间，大多数进口商仍须缴纳临时关税，这些关税预计将于7月到期。法院认为第122条并不适用于特朗普在2月命令中所援引的贸易逆差情形，认定当局适用该条款不当；但另一名法官指出，现在对原告全面胜诉为时过早。 
两家私营企业（玩具商Basic Fun!与香料进口商Burlap & Barrel）以及华盛顿州提出诉讼，指控该措施是对最高法院此前以紧急权限法案撤销其先前关税后的规避。华盛顿州出具证据称通过华盛顿大学支付了关税。 
白宫指责法庭裁决并称将上诉，政府仍打算通过1974年贸易法第301条对主要贸易伙伴恢复关税，并有三项301调查将于7月完成。法律界预计此案将进入联邦巡回上诉法院乃至最高法院的进一步审理，是否能获得更广泛的退款令仍待观察。

在美中领导人预计于5月中旬会晤前，北京已提前为议程定调，将台湾称为双边关系“最大风险”，并暗示华盛顿在台湾问题上的态度将决定两国关系能否继续回稳。
4月30日，中国外交部长王毅在与美国国务卿马可·鲁比奥通话中称，台湾事关中国“核心利益”，是中美关系的“最大风险”，并强调双方应“维护来之不易的稳定”，为高层互动做好准备。美方未就通话发布正式声明，但鲁比奥表示特朗普访华（5月14日至15日）期间将讨论台湾，并称不希望出现“任何破坏稳定的事件”。
分析人士认为，北京此举意在提高台湾议题的谈判权重，确保其成为峰会可交付的议题，而非被贸易、科技等议题边缘化。北京更倾向以交易性方式处理——对台让步或克制可换取更广泛的双边进展。
背景上，中国视台湾为领土并不排除武力统一；美方虽无正式邦交，但依法向台提供防卫所需。2025年12月华府批准有史以来最大规模对台军售（111亿美元），激怒北京。特朗普在对台军售、关税与防务开支等问题上曾发出混合信号，其2025年国家安全战略在表述上也未重申“反对台独”文字，引发外界关注是否为未来调整留空间。
专家普遍认为特朗普不太可能在短期内对台做出重大让步，但若出现暂停军售或模糊表态，都会在台海与台海民心中产生严重影响，而特朗普的不可预测性也让最终结果难以预判。
