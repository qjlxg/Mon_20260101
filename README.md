中国2月汽车销量降幅创两年最大
受季节性因素影响，中国2月汽车销量同比下降15.4%，包括出口在内的整体销量创下自2024年2月以来的最大降幅。
综合路透社、《每日经济新闻》、IT之家报道，中国汽车工业协会星期三（3月11日）公布的数据显示，上月国内汽车销量暴跌34.2%至95万辆，削弱了出口增长58%至59万辆带来的支撑。
今年前两月，中国国内的汽车销量和出口量下降了10.7%。
对于今年前两月销量下滑的原因，中国汽车工业协会副秘书长陈士华解释，主要是汽车产销受政策切换调整、需求前置释放、春节假期错位、消费意愿不足、同期高基数等多重因素叠加影响。
数据也显示，今年2月，汽车产销分别完成167.2万辆和180.5万辆，环比分别下降31.7%和23.1%，同比分别下降20.5%和15.2％。今年1至2月，汽车产销分别完成412.2万辆和415.2万辆，同比分别下降9.5%和8.8%。
