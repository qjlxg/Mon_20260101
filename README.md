Anthropic警告前沿AI发展或失控 吁各国业者共设“刹车”
（纽约综合讯）当人工智能可以实现自我进化时，人类的监管可能跟不上技术的发展速度，诸多威胁将随之而至。美国人工智能公司Anthropic呼吁各国的前沿AI公司，合作设置一个“刹车”，在必要时一致放缓或暂停尖端AI的发展。
Anthropic星期四（6月4日）在官网发长文，探讨人工智能（AI）惊人的发展速度、它们的递归自我改进（Recursive Self-Improvement）能力以及这可能带来的社会影响。 
递归自我改进是指AI系统能够自主设计和开发下一代系统，完成自我进化。文章指出，这项重大的技术突破有望在科学、医疗保健等诸多领域为世界带来巨大福祉，但同时也可能使AI系统失控的风险提高。 
“如果系统能够完全自主进化，那么我们如何保障它安全、如何监控它们以及如何引导它们的行为，都会变得更为重要。” 
Anthropic因此提出，前沿AI开发者应构建一个经过一致协调且可验证的程序，以备AI系统开始以超过社会风险管控能力的速度进行自我进化时，能够放缓或暂停AI开发进程。
文章强调，这须要以中美为首的多国主要AI公司，同意同步放缓或停止研发，并且彼此监督；因为如果只有一些公司暂停，其他竞争对手就会赶超，这无益于控制技术发展。
Anthropic联合创始人克拉克（Jack Clark）接受英国广播公司（BBC）采访时说：“我们需要能够选择踩油门或者踩刹车。可是目前，AI行业像是有油门踏板，却没有刹车踏板。” 
法新社报道，Anthropic的这个提议遭到一些硅谷业者和白宫官员反对，指称它夸大风险，试图以安全隐忧为借口，拖慢竞争对手的前进步伐。
他们也认为，美国的AI发展只要有丝毫放缓，都可能使中国在这场世纪技术竞争中获得决定性的战略优势。
Anthropic开发的Claude Mythos前沿AI模型，已证实可轻易识破许多常见应用的大量漏洞，近期引起业界和各国政府关注与警惕。 
美国总统特朗普本星期二（2日）刚签署一项行政令，让AI开发商在公开最新的先进模型前，自愿向联邦政府开放为期30天的测试窗口，让模型接受网络安全测试。

旅客因英国电子入境许可（ETA）系统故障遭遇阻滞
伦敦——近期部分拟入境英国的旅客在申请电子入境许可时遇到系统瘫痪，导致无法在登机或登车前取得必须的数字授权。6月3日下午，ETA应用显示“系统繁忙，请稍后再试”，网站的数字排队页面也停滞并显示超过一小时的等待时间。
负责该项目的英国内政部表示已“注意到部分用户遇到延迟”，技术人员正“昼夜工作”修复问题。自2月起，大多数入境英国的免签旅客需申请ETA，费用为20英镑，有效期两年，适用于不超过六个月的短期访问，涵盖美国、加拿大及大多数欧洲国家。
此前审批通常很快，官方建议至少提前三天申请。但仍有旅客在出发当天才申请，近期因此滞留。例如：居住纽约的Zara Rahim与伴侣在6月2日晚到肯尼迪机场赶赴Delta航班赴伦敦参加婚礼，连续等待近六小时未获批；荷兰莱顿大学教授Jonathan London和巴黎文化策划人Ella Marder分别在乘坐Eurostar和从巴黎出发的当日遭遇未能及时获批、无法登车的情况。多位旅客反映车站内约有百余人陷入同样境地。
内政部未就故障何时开始或何时恢复作出具体说明。6月3日下午，有乘客陆续收到获批通知。

刚果（金）卫生部称，本周东部南基伍省发生针对埃博拉遗体安葬队的袭击，迫使救援人员弃置一具棺材，增加了病毒进一步传播的风险。袭击发生在6月1日卡塔纳镇，该地约距省会布卡武北约30公里，目前由AFC/M23叛军控制。
被袭的是“安全与体面下葬”专门队伍，按严格防疫规范处理高度传染性遗体。卫生部发布的6月3日情况报告称，棺材随后被社区成员处理，此类高风险做法可能引发新的传染链。卫生部与当地医院负责人未说明袭击动机。
当局表示，袭击凸显出持续的猜忌与抵抗，阻碍疫情防控。近期安葬队和医护人员多遭袭击，部分为质疑死因的遇难者家属所为。同日（6月1日）在伊图里省首府布尼亚也发生类似事件，一支应对队在公墓遭殴，至少4人受伤。
疫情方面，自5月15日宣布暴发以来，这是该国第17次爆发，确诊病例已达363例，死亡62例。卫生部最新通报新增19例确诊（含2例死亡），感染已扩散至伊图里省36个卫生区中的17个，并在北基伍省7个卫生区和南基伍省1个卫生区出现病例。
尽管形势严峻，卫生部也报告了积极进展：伊图里省Rwampara的32名密切接触者完成21天监测，未发现埃博拉感染；北基伍省戈马市一名康复患者6月3日准备出院，AFC/M23副协调员弗雷迪·卡尼基在社交平台X上表示该患者已与家人团聚。 （路透）
