中国重启大豆拍卖 或为从美采购腾出仓储空间
在与美国达成贸易休战、持续加大美国产大豆采购之际，中国在暂停三周后重启大豆拍卖，外界认为此举意在腾出仓储空间。
据彭博社报道，中国国家粮食交易中心网站上星期五（1月9日）深夜发布公告称，中储粮将在星期二（13日）拍卖113万吨大豆。去年12月，中储粮曾多次举行大豆拍卖，这是自中美元首去年10月底在韩国会晤并达成协议以来，首次进行此类销售。
华盛顿官员透露，中国已加大对美大豆的采购力度，并承诺在今年2月底前至少购买1200万吨。中国国有企业自10月下旬起恢复从美国采购，并密集预订多批货物，使中国这一全球最大大豆进口国有望实现相关目标。
报道称，此次拍卖规模异常庞大。彭博社根据中国大宗商品咨询机构Mysteel的数据测算，在此前三轮拍卖中，饲料生产商和压榨企业共购入约90万吨大豆，而投放拍卖的总量接近158万吨。
在美国总统特朗普去年10月底与中国国家主席习近平会谈后，白宫方面曾表示，中国承诺在去年年底前购买至少1200万吨美国大豆。美国官员随后澄清，相关采购的最后期限延至今年2月。

马国追回300亿令吉一马资金
马来西亚政府透过反贪污委员会等执法机构，已将一个马来西亚发展有限公司（简称一马公司）流失的约420亿令吉资金追回约300亿令吉。
反贪会主席阿占巴基在《每日新闻星期刊》的专访中说，这并非简单任务，因为追回资产是复杂的工作。
阿占巴基说：“以一马公司为例，我们已从流失的420亿令吉（约133亿零337万新元）中追回约300亿令吉，接下来可能还将追回更多。”
他说：“当我公布追回的金额时，民众往往就追问还能再追回多少。实际上这并非易事，因为我们是从罪犯手中追回资产。”
阿占巴基希望尽可能追回被挪用的一马资金。“假设流失10亿令吉，能追回6亿或7亿令吉就算成功。若要衡量成效，不应只看金额，我并没有设定关键绩效指标（KPI）。”
他说，反贪会的职责是专注于追查及将涉贪者控上法庭。“这样的追讨成果并非反贪会单独能完成，其他单位如警方、国内贸易及生活费部、内陆税收局及关税局，都提供了专业协助。”
阿占巴基也说，反贪会在一马公司案中也学到许多宝贵经验。例如马国与一些国家过去在司法合作方面的关系并不密切，但一马案使新加坡、美国、英国及巴巴多斯等国愿意配合马国并提供证据。
他透露：“我们运用鉴证技术与财务分析向法庭展示资金流向，这在马国过往任何检案件都未曾出现。”
他说：“一马公司并非简单案件，它涉及复杂的财务分析与银行术语等。我们拥有具关专业能力的官员，在法庭上清楚说明资金如何从海外流入，最终进入国内某个帐户。”
在司法互助协议（MLA）方面，阿占巴基指出，连以往难以合作的瑞士，目前都打开大门提供协助。
他说：“至于涉及与马国签有引渡条约的国家，是否合作则取决于个国家的法律。例如澳大利亚，若马国有某些刑罚涉及鞭刑，但澳洲并无此刑罚，双方自然无法合作。若罪犯逃往澳洲，他们便不会与我们合作。”

智库预测马国2026年经济增长放缓至4.5%
马来西亚著名智库社会经济研究中心（SERC）预测，马国在2026年的经济增长率预计将放缓至4.5%。
《东方日报》报道，SERC执行董事李兴裕星期一（1月12日）指出，尽管2026年经济增长呈现放缓趋势，但马国经济基本面仍展现韧性，这得益于内需支撑、稳定的劳动力市场、较宽松的货币政策和具有支持性的财政措施。
李兴裕星期一在“2025年10月至12月季度经济追踪报告”记者会上，分析马国2026年经济前景时指出，若一些发达经济体推出更强劲的刺激措施、或美联储进一步降息，都将使全球市场和经济受益，成为马国经济的上行风险。
但他也说，若关税影响全面发酵、国内项目延误实施，以及出口回报期比预期剧烈，都将使马国的经济面临下行压力。 
“若全球持续存在不确定性，或政府犯下政策错误，导致企业经营成本增加，都将削弱投资信心，进而影响国家经济前景。”
在谈及令吉的表现时，李兴裕认为，随着政府持续推动结构性改革、美元走弱，以及市场信心回温使资本逐步回流，促使令吉持续呈现走强态势。
他指出，根据国家银行的数据，截至1月9日，令吉兑美元的升值幅度达10%，次于欧元的12%。他强调，未来马国必须维持良好的投资前景，确保继续推进对国家有利的改革，才能巩固令吉的表现。
