在努力重振欧洲最大汽车市场销量之际，德国将把规模30亿欧元（约45亿新元）的电动车补贴计划向所有制造商开放，包括中国品牌。
据彭博社报道，这项激励措施于星期一（1月19日）公布，它是在2023年底补贴结束、需求大幅下滑之后，德国为推动电动车普及、扶持陷入困境的汽车产业而采取的新一轮举措之一。
尽管德国政府的目标是支持本土车企，但也不会按产地设限。
“我对欧洲和德国品牌的质量充满信心，”环境部长卡斯滕·施奈德（Carsten Schneider）周一在新闻发布会上表示，“我既没有在数字中，也没有在道路上看到所谓中国汽车制造商大量涌入德国的证据 —— 因此我们选择直面竞争，而不是设置任何限制。”
这一决定对比亚迪等中国品牌是一大利好。尽管中国车企已面临欧盟对进口电动车征收的关税，但凭借本土较低的制造成本，它们在欧洲市场仍能实现盈利。欧盟方面正在考虑以最低价格机制取代进口关税。
德国向中国品牌开绿灯的做法与其他国家形成反差。英国去年推出的补贴计划实际上将中国电动车排除在外，该计划强制要求满足包括电池及组装过程低碳排放在内的环境标准。法国的计划也包含类似条件。

马科斯在视察新建的稻米加工系统（RPS）并发放农渔业援助时宣布该计划。该中心配备现代化烘干、碾米设施、农资、机械及太阳能设备，可自动化烘干和碾米，提高稻米品质并减轻农民劳力。
他指出，政府已建成152处类似设施，但仍要扩大规模，“今年我們要再建300多套烘干系统，不要再看到有人在路边晒稻谷被过路卡车碾过，太可惜了。”
马科斯表示，此举可加强收获后基础设施、降低损失；政府也在通过提供机械、作物保险和必要投入品推动农业现代化，以减轻天灾带来的风险。
他并主持发放运输卡车与渔船，并由作物保险公司向受台风损害的农户发放救助。
汉姆蒂克RPS由汉姆蒂克综合合作社管理，11月开始营运，目前日均产约260袋稻米，供给安蒂克各地机构。

今年或成为中国电动汽车（EV）厂商打开西方市场的关键一年，随着欧盟和加拿大等地放松贸易壁垒，高性能国产车型迎来更多机遇。 
加拿大在国事访问后宣布取消此前对中国纯电动车征收的100%惩罚性关税，同时设定年进口配额为49,000辆，但仍对续航多于500公里、配备复杂车载娱乐系统的中国电动车征收6.1%关税。 
欧盟与北京也达成共识，以价格承诺替代此前7.8%至35.3%的关税，为中国车企提高海外利润率留出空间。吉利称将密切关注，而比亚迪海外出货占比约20%，加拿大关税取消将提高其盈利空间。 
中国仍是全球最大电动车市场，协会数据显示大陆去年共出货约260万辆（含插混），同比增长104%；乘联会称中国买家占全球新增电动车约70%，约1300万辆。业内认为，5万辆的年配额虽不足以改变总体出口规模，但若能赢得当地消费者信任，将是重要开端。

马尼拉消息——菲律宾科学技术部—菲律宾核研究所（DOST‑PNRI）研究人员确认，西菲律宾海检测到“异常偏高”的放射性碘‑129含量。该长期研究由Angel Bautista VII博士主导，始于2017–2018年，旨在追踪2011年福岛核事故的污染扩散。 
研究采集巴勒（太平洋一侧）和西菲律宾海帕罗拉岛的珊瑚样本，结果显示西菲律宾海的碘‑129水平比太平洋一侧高出约50%–70%，出乎研究团队预期。研究认为此类升高多与核试验、燃料再处理或核事故有关，但并不等同于立刻的健康或环境危害。 
研究推测碘‑129很可能来自黄海一带（中国、韩朝沿海）的核活动，并随沿岸洋流向南输送；福岛事故产生的放射性物质据珊瑚数据约于2014年抵达菲律宾海域。 
分析依赖与菲国立大学海洋科学研究所合作采样，检测则需赴东京大学使用加速器质谱仪，因本国及东盟缺乏相应设备。报告同时指出菲律宾在放射性物质常规监测与政策上存在缺口，呼吁加强科研投入与建设本土分析能力。 
该项目获DOST、农业與水產資源研究發展理事會（PCAARRD，800萬披索）及國家研究委員會（NRCP，500萬披索）資助。研究强调：碘‑129半衰期极长（约1600万年）但放射性低，目前风险可视为可控，建议持续监测与政策改善以强化海域安全与环境应对能力。

抗议者随后前往奎松市东大道的LTFRB中央办事处，集体提交临时营运许可（PA）申请与恢復五年特许权的请愿，之后又到LTFRB–NCR办事处重复递交相关文件。弗罗里达强调，PA不应成为长期公共运输许可，五年特许权为业者长期所有权，理应恢复。他并表示，不论政府给予何种形式的特许，都是集体行动的成果。

特朗普第二任期一年后，移民执法在美国乡村部分地区重塑着生活，乔治亚州靠近佛罗里达边界的小镇弗尔克斯顿尤为明显。由私营监狱运营商GEO Group管理的弗尔克斯顿ICE处理中心近年来扩建，正成为全国最大的移民拘留设施之一。支持者称带来就业，但当地居民与宗教界人士警告其带来的道德与人权代价。
在距该中心约两小时车程的蒂夫顿，一位长期从事移民正义工作的圣公会牧师李安·卡尔布里斯（Leeann Culbreath）常到拘留所探视被拘者。她回忆近十年前曾被大批被拘者围住求助，称“所谓只针对最坏分子的驱逐只是幌子，实质是大规模羁押”。她担忧扩建只会使无法出庭、得不到医疗和基本照护等问题加剧。
亚裔美国人权组织（Asian Americans Advancing Justice）诉讼主管梅雷迪思·尹（Meredith Yoon）警告，弗尔克斯顿长期存在权利侵犯：缺乏医疗、看守虐待、食物质量差、与ICE沟通不畅、环境肮脏等问题可能随扩建恶化。
Mount Carmel传教浸礼会牧师安特万·尼克松（Antwon Nixon）表示，小镇不应以拘留所闻名，并呼吁民众发声、参政以应对行政政策。当地约三分之一居民生活在贫困线以下，牧师们担忧经济脆弱使社区易受拘留经济利用。
路透向GEO Group就扩建事宜征询评论，但未获回复。
