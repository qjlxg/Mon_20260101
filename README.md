印度首列氢动力列车启用 推进铁路运输脱碳
印度推出首列氢动力列车（hydrogen-powered train），旨在实现铁路运输脱碳，减少对化石燃料的依赖。
印度总理莫迪星期五（7月17日）为列车主持发车仪式，他在社交媒体上发文说：“对于印度而言，今天是朝着自力更生和可持续发展方向迈出的重要一步。”
法新社报道，这列由10节车厢组成的列车运行于印度北部哈里亚纳邦金德（Jind）和索尼帕特（Sonipat）之间，全长89公里（55英里），由一套1200千瓦的氢燃料电池推进系统提供动力。
氢动力列车利用燃料电池将氢气和氧气结合发电，排放物仅为水和蒸汽。新列车投入服务让印度跻身少数几个正在测试这个动力技术的国家行列。官员称，这个系统是目前世界上动力最强的列车系统。
