香港警务处将扩展“锐眼计划”，保安局局长邓炳强说，增加摄像头并非剥夺隐私，而是为了降低犯罪。
据网媒“香港01”报道，香港立法会保安事务委员会会议星期二（2月24日）召开，讨论用于扩展“锐眼计划”的约40.6亿（港元，约6.6亿新元）非经常性开支。
邓炳强说，锐眼计划首阶段已于去年底完成，主要在港岛及九龙区安装约5000支镜头，并接入约6000支属于其他政府部门及机构的镜头；该计划已协助警队侦破899案，包括谋杀和抢劫案。
保安局向立法会提交的文件显示，警方将推展第二阶段计划，每年增加超过2万支镜头 ，预计于2028年将总共设有约6万支镜头，覆盖全港。第三阶段将于2028至2031年进行 ，额外增加约6500支镜头。
“锐眼计划”将接入更多政府部门及公共机构的闭路电视系统，例如隧道、公共屋邨的公用地方、康体场所、机场及边境管制站等。警方正研究与私人商业机构合作，将合适的闭路电视接入系统。
邓炳强称，香港市民开始明白闭路电视并非剥夺隐私，而是有助降低犯罪率。
据《星岛日报》此前报道，香港警务处处长周一鸣2月15日说，正研究为全港闭路电视系统引入人脸识别功能，期望最快今年落实，同时希望香港可以像中国大陆一样“每走两步就有一个闭路电视”。
