共和党重调中期竞选策略：借用特朗普的动员力，但避免把竞选变成对不受欢迎总统的公投。
鉴于汽油价格上涨、特朗普支持率下滑及伊朗战争持续，党内正在重新设计竞选蓝图。四名知情人士称，白宫幕僚长苏西·威尔斯（Susie Wiles）、政治主管詹姆斯·布莱尔（James Blair）与资深民调员托尼·法布里齐奥（Tony Fabrizio）在4月20日一次闭门会中，建议候选人突出共和党减税与抗通胀政策，同时避免把特朗普本人作为竞选焦点。
共和党人担心特朗普的政治运势下滑会拖累摇摆选区，国会两院控局面临压力。美国汽油均价接近每加仑4美元，可能抵消特朗普政府“伟大美好法案”的成效。路透/益普索民调显示仅36%受访者认可其施政，不少人对其性格与精神状态存疑。
会中气氛曾乐观，宾客签署保密协议并预言弗吉尼亚重划选区会有利共和党，结果却被民主党绘制的新地图通过。消息人士称，党内正转向强调地方议题，而非“把特朗普放上选票”；但共和党全国委员会仍称特朗普是中期选举中最重要的保守派动员者。
在对伊政策上，特朗普被指在与伊朗的军事与外交对抗中未能取得决定性成果；4月21日将原定两周的停火无限期延长，被视为退让。卡内基国际和平基金会中东问题专家亚伦·大卫·米勒（Aaron David Miller）认为，伊朗认为自己有耐心，可以等特朗普让步。  （路透）

罗马——2月，意大利议会以137票对117票否决了一项将产假与陪产假同等化、且不可转移并全额带薪的提案。反对者以财政负担和对公共部门及小企业影响为由拒绝通过此议案。现行制度下，意大利女性可享约5个月产假，而父亲仅有10天陪产假。
尽管立法未果，一批“爸爸网红”正通过社交媒体重塑父亲角色。45岁的迪亚戈·迪弗兰科在米兰郊区纪录自己做主要照护者、接孩子放学、做晚饭等日常，他在Instagram有5万多粉丝，约85%为女性关注者。迪弗兰科表示，自己的出现让妻子得以全职发展为高级经理人。
统计显示，2024年意大利女性就业率仅53%，为欧盟最低之一（欧盟平均70.8%）。女性约占自愿辞职者的70%，产后离职普遍，被视为明显的“生育惩罚”。反对党称该法本可引发文化变革；学者引用西班牙经验指出，强制且不可转让的带薪陪产可显著提高男性请假率并缩小性别薪酬差距。智库Tortuga研究也表明，私人企业若提供更长陪产，父亲的请假率可升至71%。
迪弗兰科以自身经历乐观估计：他第一胎时幼儿园几乎没有其他父亲，六年后已有三四位，他认为变化正在发生。

美国同意修改对委内瑞拉的制裁，允许委政府支付尼古拉斯·马杜罗的辩护律师费用，撤销了此前可能阻碍其毒品走私案审理的限制，法庭文件显示（4月24日）。
马杜罗（63岁）及其妻子西莉亚·弗洛雷斯（69岁）于1月3日在加拉加斯被美国特种部队拘捕，并押赴纽约面临包括“毒品恐怖主义共谋”等刑事指控。两人均表示无罪，目前在布鲁克林羁押候审。
辩护律师巴里·波拉克在2月向曼哈顿联邦法官艾尔文·赫勒斯坦申请撤诉，称制裁阻止委政府支付律师费，侵犯马杜罗依宪法选择律师的权利。辩方指出，两人无法自付，且委政府愿意承担费用。
赫勒斯坦在3月26日的听证会上表示不打算撤案，但对政府阻止付款的正当性持怀疑。检方律师凯尔·威什巴则称，相关制裁基于正当的国家安全与外交政策考量，司法部门无权命令财政部修改制裁措施。
法官指出，自马杜罗被罢免以来，美国已放宽对委制裁，德尔西·罗德里格斯临时执政后两国关系有所改善，并强调被告享有宪法保障的辩护权。
此前特朗普政府曾以腐败和破坏民主为由加大对委制裁，并指称马杜罗2018年连任存在舞弊；马杜罗否认这些指控，称其为美国意图夺取该国石油资源的借口。　REUTERS
