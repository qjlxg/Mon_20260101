柬埔寨位于危机中心：私人债务占GDP比从2010年的24.2%飙升至2023年的134.5%。柬埔寨信贷局称，截至2025年12月，平均每借款人未偿个人贷款约为6500美元；成衣业最低月薪为208美元。边境冲突与美方新贸易限制令楼市走弱，进一步加剧压力。
泰国2025年家庭债务占GDP为86.8%；马来西亚到2025年中达84.3%。债务结构不同：马来西亚以房贷和车贷为主，泰国则以消费性個人貸款占比高，信用卡和个贷构成大部分不良账户，部分泰国借款人每月超过半收入用于还债。
微型金融商业化被指推高欠债。报告显示，柬埔寨约380万户持有超过310万笔小额贷款，金额逾180亿美元；研究还发现大量家庭为医疗借款。去年与泰国冲突后，约90万柬埔寨外出务工者返乡，仅21%已再就业，汇款从上一年29.5亿美元降至18.6亿美元。
分析认为，解决之道既包含更严格的银行与小额信贷监管，也需深化社会保障与公共服务，减少居民因病就学和住房而被迫举债的必要性。

马尼拉 — 多名三轮车司机表示，政府发放的补贴已不足以抵消持续攀升的油价，补助大多被迫用于燃油与日常家用。

伊朗向距伊朗约3,800–4,000公里的美英军事基地迪戈加西亚发射导弹未中，引发其真实射程的质疑。

分析认为，这可能反映未公开的更远射程能力、现有系统的改装，或对中程/远程技术的早期测试；也可能是战略模糊的一部分，用以增加对手防御负担。
