中国科学家绘制免疫多组学图谱，发现某基因变异可显著提高哮喘风险
由BGI-Research基因组与多组学技术国家重点实验室、山西医科大学、复旦大学附属瑞金医院（注：原文为上海交大附属瑞金医院）与中国科学院大学等机构的研究团队在Science发表论文，构建“中华免疫多组学图谱”。研究对428名20至77岁成人的外周血样本中逾1000万免疫细胞进行测序，鉴定出73种免疫细胞类型，包括占比低于0.1%的罕见细胞。
研究揭示，某一特定基因变异通过调控基因表达影响调节性T细胞功能，导致其失调并增加哮喘易感性；该变异还与一种在接触过敏原后促发气道炎症的蛋白水平升高相关。
第一作者尹建华表示，研究将疾病风险精确到细胞类型，为基于细胞靶向的免疫治疗和精准医疗提供方向，但需更大规模的哮喘人群数据进一步分层验证。联合第一作者黄卓立指出，团队还发现与类风湿关节炎等疾病相关的风险变异，可作为新的治疗靶点，并为通过外周血研究其他免疫相关疾病提供蓝图。

自学出身的高惠军当选为国际电子工程界重要学会IEEE工业电子学会（1951年成立）的新任主席，成为首位来自中国大陆的该会主席。IEEE在160多个国家拥有逾40万名会员。
高惠军为哈尔滨工业大学航天学院教授，黑龙江籍，15岁赴陕西读中专学机械制造。1995年毕业后在工厂做磨床工，边工作边自学取得工业企业管理大专，1997年完成本科学业。

中国嫦娥六号在月球背面采回的岩石样品中，首次在自然环境中发现了由单层原子构成的极细管状结构——碳纳米管。
这一发现来自2024年嫦娥六号任务，该探测器是首个在月球背面着陆并带回样品的任务。吉林大学研究团队利用高分辨率电子显微镜，在样品中检测到如“稻草”般的超薄管状结构。
研究人员指出，此类仅一层原子厚的碳纳米管此前仅在实验室人工合成中见过，此次证据表明这种长期被认为需要精密工程技术的材料，也可在自然条件下形成。

中国北方工业集团（简称“北方工业”）首席科学家蔡毅在本月接受《现代兵器》采访时表示，乌克兰利用无人机投放改装小型与微型空炸弹的做法，为我国研究无人机战法提供了有价值的借鉴。
蔡毅指出，参与乌俄战争的无人机主要分为自杀式和用于侦察打击两类。乌军在侦察打击型无人机上配备了多种小口径空炸弹，既有北约标准弹药，也有多种简易改装，实战效果显著。
他将10–100公斤称为“小型”炸弹，10公斤以下称为“微型”。小型化弹药将成为扩展无人机打击能力的重要方向。乌军把手榴弹等现成弹药改装为空投炸弹，例如基于RKG-3的RKG-1600、以RGD-5加羽毛球头作尾翼的小型炸弹，且有视频显示可袭击俄军坦克顶部；还用啤酒瓶制作莫洛托夫火瓶投放燃烧弹，并把PG-7V反坦克火箭改装为穿透坦克上装甲的小型空炸弹。
蔡毅建议，可设计一系列可与无人机配套的模块化小/微型空炸弹，以及简单的改装包和制导套件，便于一线官兵在战场上快速将现成弹药转化为简易空炸弹。
另据报道，去年8月时任乌克兰數字化轉型部長費多羅夫称，超過80%的俄軍戰場損失由無人機攻擊造成。
