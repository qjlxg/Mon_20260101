马尼拉——菲律宾公共工程与公路部（DPWH）周六表示，将在复活节周（3月）出行高峰前，对马哈尔里卡公路奎松省至卡马里内斯苏尔段严重损坏路段采取临时修补措施，以缓解行车压力。
公共工程部长文斯·迪宗（Vince Dizon）说，整条从北吕宋到棉兰老的马哈尔里卡公路修复总体规划仍在制定中；因此已下令卡马里内斯苏尔工区工程师加快修复受2024年热带风暴“克里斯廷”及后续台风冲击而塌陷的外车道。
迪宗指出，通行车道减少难免造成拥堵，必须在圣周前完成路面重铺和临时补强，副部长拉拉·埃斯基比尔（Lara Esquibil）将负责寻找短期改善方案，确保行车较为顺畅。他强调，短期措施为应急，长期且全面的整治由DPWH中央办公室主导研究与实施。
迪宗此前实地沿路考察，见证坑洞和严重拥堵。他并指出，改善公路的关键之一是建设更大、更完善的排水系统，避免像布基东—达沃（BuDa）公路因横向排水不足、水积地下造成路基塌陷的情况，并计划与日本顾问共同制定长期方案。

马尼拉讯 — 针对假期期间有居民反映部分路段垃圾堆积的报道，卡卢坎（Caloocan）市政府强调本市垃圾收运仍维持常规。 
第174区（Barangay 174）早前有居民称垃圾数日未被清走，但该区主席吉尔伯特·里维拉（Gilbert Rivera）否认此说，称本区每日两次收运，市府收运车并有专人引导，主干道收运时段为清晨至早上8点及夜间9—10点。 
里维拉表示，圣诞与新年假期垃圾量增加、收运时程会暂时调整，但通常在第2或第3周恢复正常，并呼吁居民在收运车未到前勿随意将垃圾置于路旁，强调纪律与配合的重要性。 
市环境管理局（CEMD）称，南北卡卢坎主干道每日早晚收运，里区收运为每周两次。部分居民反映，有外来者将垃圾倾倒于第174区，影响居住环境，已向区公所反映；另有居民则称并非“一周未收”，早晚确有收走。 
市府并援引第0753号市令，非法倾倒可处500至5,000比索罚款。卡卢坎市政府上周也承认部分区域收运确有延误，已向承包服务商提出并要求尽速整改，市务管理员奥罗拉·C·西耶戈表示将认真处理民众关切。

【NASA将把SLS火箭运至发射台 4月前冲刺载人绕月】金十数据1月17日讯，周六，美国航空航天局（NASA）将把一枚造价高昂的火箭运往佛州肯尼迪航天中心的发射台。根据NASA发布的新闻稿，将波音制造的太空发射系统火箭（SLS）和洛克希德·马丁公司制造的“猎户座”飞船从装配大楼转移至发射场，这段4英里的行程需要花费长达12个小时的时间。火箭和飞船抵达发射场后，NASA将开始执行飞行的其他准备工作，包括连接地面支持设备、测试硬件组件以及检查地面基础设施系统。SLS火箭已研发约15年，目前超出预算且进度滞后，迄今仅发射过一次（2022年执行的绕月无人测试飞行）。此次转运是“阿尔忒弥斯II号”任务的第一步，该任务计划将四名宇航员送往月球轨道并于十天内返回地球。此次飞行原定于2024年底进行，现计划不晚于4月发射。NASA完成转运和发射台检查后，将于1月底进行模拟发射演练。这一关键环节须顺利完成，后续步骤才能推进。
