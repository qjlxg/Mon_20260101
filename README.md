中国科研人员提出超长伽马暴成因新解
伽马暴是宇宙中最剧烈的爆发现象之一，中国科研人员提出超长伽马暴成因新解，即“超巨星自调控坍缩星模型”。
据新华社报道，中国科学院高能物理研究所粒子天体物理全国重点实验室团队利用自主研发的新分析工具，将慧眼、极目等卫星对这例伽马暴发生前后30天内的观测数据进行了全面检索，发现其在伽马射线波段持续长达29小时，打破伽马暴持续时间记录。
此外，团队还发现其伴随的X射线辐射也具有独特的时变特征。
根据这些发现，科研团队提出了全新的“超巨星自调控坍缩星模型”。
这一模型提出，这例伽马暴的前身星是一颗质量远超太阳的超巨星。与普通伽马暴的前身星不同，超巨星的坍缩可长达几十天。当超巨星耗尽燃料时，其核心首先坍缩形成黑洞，黑洞会快速吸积超巨星内层物质，进而产生接近光速的喷流，这正是伽马暴爆发的来源。而在后续的吸积过程中，产生了速度稍慢的喷流，发出X射线辐射。
相关成果已发表于国际学术期刊《天体物理学杂志快报》。

前最高法院助理大法官安东尼奥·卡皮奥在周四于菲律宾大学迪利曼分校举办的论坛上表示，最高法院今年发布的决议与2025年7月对副总统萨拉·杜特尔特弹劾条款的裁定“非常不一致”。他指出，差异主要涉及全院表决所需程序及宪法上的问题。
卡皮奥称，法院已撤回此前增加的若干要件，因此在“第二种模式”上已不存在问题，所指即为是否须由众议院三分之一议员背书的争议。
菲律宾大学法学院院长格温·格雷西亚·德维拉认为，最高法院在处理此次弹劾案时的部分做法存在“问题”，并且似乎背离了1986年宪法起草委员会关于采取较宽松弹劾方式的记录。
出席本次论坛发言的还有1986年宪法委员会成员、退休助理大法官阿道夫·阿斯库纳，以及前远东大学法学院院长梅尔·圣玛利亚等人。
最高法院上月维持其裁定，认定对杜特尔特的弹劾条款违宪，参议院未取得审理该弹劾案的管辖权；在1月28日的全院裁决中，参与先前判决的法官一致最终驳回了众议院的再审动议。

馬尼拉——選舉委員會（Comelec）已結束對參議員羅丹特·馬科萊塔（Rodante Marcoleta）選舉捐款與支出報表（SOCE）涉異常情況的事實查證，Comelec 主席喬治·加西亞（George Garcia）周四證實。
加西亞以簡訊回覆 ABS-CBN 記者，確認政治與財務事務部（PFAD）已完成事實查核。馬科萊塔的辯解是：款項在他成為候選人之前已給予。加西亞並援引最高法院 2009 年「Peñera 準則」，指出候選人僅自競選期開始時才對選舉行為負責。
質疑者 Kontra Daya 去年指出，馬科萊塔在 SOCE 中申報逾1.12 億披索的競選支出卻未列捐款來源；其資產負債表顯示淨值僅約5,200 萬披索，僅相當於申報支出的約46%。馬科萊塔曾在電視採訪中稱，2025 年中期選舉期間有友人捐款但要求匿名，故未申報。
加西亞表示，事實查核是為釐清是否觸犯《綜合選舉法》相關條文。該法第98 條規定出資人應以真實姓名向候選人或其代理人出資並記錄在案，第99 條則要求出資人向選委會遞交宣誓報告，列明每筆捐款金額、受款人及日期。若 SOCE 有不實陳述，除違反 SOCE 法外，亦可能構成偽證罪。
加西亞未說明完整調查結果何時公布；若發現違規，Comelec 將啟動初步調查。他並表示自我回避，不介入相關討論。ABS-CBN 已向馬科萊塔陣營徵詢回應，但截至發稿仍未收到回覆。

马尼拉 — 周四拂晓前，数百名穆斯林信徒在奎阿波的马尼拉金清真寺（Masjid Al-Dhahab）聚集，迎来斋月首日。约凌晨5:15，信众参加晨礼（法吉尔），斋戒自此正式开始。祈祷前，家庭和礼拜者共享黎明前餐（suhoor）。
清真寺内男女分区礼拜，男区较大，并设有供净礼（wudu）用的洗净处。白天信众照常礼拜并禁食至日落；清真寺日程约为午礼12:30、晌礼15:30、日落礼（麦格里布）约18:30，至此开斋（iftar）。晚间则有夜礼（伊沙）及约19:30的塔拉维托（拉玛丹特别夜间会众礼）。
会众鼓励集体礼拜以获更多功德。金清真寺建于1976年，是大马尼拉重要伊斯兰中心，斋月活动的城市焦点。斋戒强调纪律、耐心、慈善与道德，戒绝说谎、暴力、傲慢、贪婪、偷窃、不检点及不合清真规的经商行为。
今晨晨礼由伊玛目Abdul Samad领祷。清真寺与文化中心管理员Hadji Muhammad Erzad “Bebot” Malli表示，斋月提醒人类共通的价值——“又到斋月，这是最神圣的月份，全球约20亿穆斯林同庆。希望大家以爱与尊重相待，彼此体谅。斋月的精髓是耐心、自律与宽恕”。
斋月结束时将迎来开斋节（Eid al-Fitr），菲律宾通常将其定为国定假日。对许多信徒来说，第一天的晨礼不仅标志禁食开始，更代表对信仰、仁爱与社区的新承诺。

达沃市讯（Hernel Tocmo 报道）—— 菲律宾海岸警卫队东南棉兰老区（CGDSEM）周三宣布，已终止对失踪渔船MBCA Amejara的搜救与打捞行动。 
CGDSEM称，行动先后进行了15天的强化搜救，随后又延长15天进行搜寻与打捞，搜索范围扩大至巴西蘭海域并延伸至印尼部分海域。 
目前仍有9人下落不明，6人遗体已被打捞上岸，1人获救生还。涉事小型柴油船在1月18日自达沃市Sta. Ana码头驶往达沃东方省Governor Generoso时遭遇恶劣海况，该航线并未在船只营运证照（franchise）许可范围内。 
当局调查还发现，该船以休闲垂钓为目的出海，未向相关政府机构申领所需许可。 
在打捞到6具遗体后，搜救行动陆续回收了船只残骸、救生圈和救生衣等物证，但近日报告未再有新发现。 
“经全面评估并完成既定与延长搜寻任务后，CGDSEM于2月18日正式结束实地行动。”该区表示，并向所有参检机构、人员、志愿者及相关单位致谢。 
CGDSEM对仍在等待亲人的家属表示深切慰问与支持，并强调已尽力以慎重、同情与尊重的态度动用一切可用资源。 
菲律宾海岸警卫队重申，呼吁所有船舶营运者及海事界人士严守海上安全法令，遵循航行计划程序并密切关注主管机关发布的气象警报。

首尔法院将于2月19日就韩国前总统尹锡悦的暴动罪作出裁决，审议是否因其在2024年12月试图实施戒严而判处死刑。
尹锡悦当时在电视讲话中突然宣布戒严，称须以非常手段肃清“反国家势力”。现年65岁的这位强硬保守派随后被弹劾、逮捕，并面临包括暴动、妨碍司法等多项指控。
检方在一月庭审中对暴动罪求处最严厉的刑罚。根据韩法，暴动罪的可判罚为无期徒刑或死刑；不过韩国自1997年后实际停用死刑，判处死刑实质上等同于终身监禁。
法院的当庭宣判将在首尔时间下午3点（格林尼治时间06:00）现场直播，预计将引发全国关注与广泛反应。尹锡悦目前被单独关押，正在应对多起刑事审判，他持续否认不当行为，称其是为“维护自由”和恢复宪政秩序而行。
检方指控他以“独裁与长期统治的权力欲”为驱动，发动暴动。法律界观察人士认为，法院较可能判处无期徒刑：尹未认罪亦未表示悔悟，减刑空间有限。
尹已因较轻罪名被判五年徒刑，多名高层官员也面临重刑。12月3日的戒严宣告在六小时后被解除，当时国会改革派议员冲向议场并以办公家具阻挡武装部队，事件引发街头抗议、股市恐慌，并使美等盟国措手不及。
尹夫人金建希另于一月因就任第一夫人期间涉嫌收受贿赂被判处20个月监禁。
（法新社）
