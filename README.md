国际能源署：受伊朗战争影响 石油库存以创纪录速度下降
国际能源署（IEA）说，随着伊朗战争导致的中东供应中断加剧，全球石油库存正以创纪录的速度下降，且未来数月将继续下降。
综合法新社和彭博社报道，国际能源署星期三（5月13日）发布月度报告说，继3月份减少了1亿2900万桶，4月份再减少1亿1700万桶。
月度报告显示，3月和4月全球观测到的石油库存以每天约400万桶的速度下降。国际能源署正协调美国、日本、德国等主要经济体释放应急燃料储备。报告称，即使冲突下个月结束，市场仍将持续“严重供不应求”直至10月。
随着冲突持续，有效封锁连接波斯湾与国际市场的霍尔木兹海峡，出口受挫程度正在加深。报告显示，上月全球供应量进一步减少180万桶/日，自2月以来的累计减产量已达1280万桶/日。
总部位于巴黎的国际能源署说：“随着全球石油库存已以创纪录的速度下降，在夏季需求高峰期来临之前，油价可能进一步波动。”
这场危机对需求造成的冲击也愈发深重，促使国际能源署自美以伊战争爆发以来，连续第三个月下调预测。受产品供应中断和油价飙升影响，本季度全球石油消费量预计势将骤降245万桶/日，为自2020年冠病疫情以来最剧烈的跌幅。
美国和以色列2月28日向伊朗发动空袭后，伊朗做出反击，且实际上封锁了具有战略意义的霍尔木兹海峡，阻止海湾地区的石油和天然气出口，导致油价飙升，迫使各国争相寻找替代供应来源。
随着北半球夏季旅游旺季临近，人们对石油短缺的担忧日益加剧。多家航空公司已发出警告，如果供应中断持续下去，几周内就可能出现航空燃油短缺。
油价飙升也对需求前景构成压力，因为石化和重型制造业等终端用户正在减少用量。国际能源署目前预计，第二季度全球石油需求每日将减少240万桶，低于中东战争爆发前预测的每日350万桶。
国际能源署报告称，“油价上涨、经济环境恶化以及抑制需求的措施将进一步抑制全球石油消费。”

美中在人工智能与芯片领域的竞争在特朗普与习近平5月13至15日北京峰会前进一步加剧。4月，美国国会审议的MATCH法案拟堵塞出口管制漏洞，进一步限制中国获取先进半导体制造与AI训练所需的技术与设备，可能严重制约中国的AI发展。

美方智库人士指出，中国AI企业尤其是初创公司在算力规模、先进AI芯片与资金获取方面仍落后于美企，这凸显中国对最先进半导体及其制造工具的依赖与脆弱性。尽管特朗普在2025年底批准英伟达H200芯片对华销售，美国商务部长称迄今未有中国企业购买，部分原因是中国政府鼓励把投资集中于国内产业。
为应对外部压力，北京一方面加快国产替代，4月有初创企业推出适配华为芯片的软件和模型；另一方面通过法律手段拓展反制空间，4月7日出台新规以保障产业和供应链安全，授权对“歧视性限制”采取反制措施。新加坡学者胡欣悦指出，这些规定是中国首套产业与供应链安全行政规制，意在为反制措施提供法律依据，并可能将竞争延伸至经济法律层面。
市场方面，2025年中国国产AI芯片占本土市场近41%，而在2023年前英伟达在相关细分市场的份额曾超过90%。外界普遍认为，半导体与AI的地缘政治重要性短期内难以消减，双方较量仍将以防御性措施为主，关键看未来哪方在下一波技术突破中占先机。
