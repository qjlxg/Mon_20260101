世界气象组织：2025年是有记录以来最热的三个年份之一
世界气象组织星期三发布公报指出，2025年是有记录以来最热的三个年份之一，延续了全球气温屡创新高的趋势。过去11年均为有记录以来最热的11年，且海洋升温仍有增无减。
新华社消息，世界气象组织（WMO）对八个数据集的综合分析显示，2025年全球平均表面温度比1850至1900年即工业化前的平均值高出1.44摄氏度。其中两个数据集将2025年列为176年记录中第二热年份，另外六个将其列为第三热年份。
在所有八个数据集中，过去三年是最热的三年，三年期综合平均气温比工业化前高出了1.48摄氏度。在所有八个数据集中，从2015年至2025年的过去11年均是最热的11年。
气象组织秘书长绍洛在公报中说，2025年以具有降温作用的拉尼娜现象开始和结束，但因大气中吸热温室气体的累积，它仍是全球有记录以来最热年份之一。陆地和海洋高温助长了热浪、强降雨和强热带气旋等极端天气，表明对早期预警系统的迫切需求。
拉尼娜现象（La Nina）是指赤道太平洋中东部海水大范围异常变冷，引发全球气候异常的自然现象。

该定性评估基于对22个重灾区85个分区、510名地方官员、商界及社区代表的访谈，95%的受访者报告当地出现住宅、交通、工业或社区基础设施损毁，住房和道路受损最为普遍；93%反映生计受影响，出现作物与畜牧损失、失业与企业关停，尤其冲击了占劳动力大部份的非正式部门。  
迪特瓦风暴于2025年11月26日登陆，为该国自2004年海啸以来最致命灾害。UNDP此前的遥感分析估算约20%国土被洪水淹没、约230万人受暴露。当前社区普遍缺乏重建所需的材料、劳动力与资金。  
UNDP驻斯里兰卡代表浅田（Azusa Kubota）赞赏政府提出对中小微企业提供950亿斯里兰卡卢比支持、含3%贴息贷款的计划，但强调必须覆盖大量在非正规体系中运作但支撑地方经济的群体。评估指出，微型企业支持、低息贷款与营运资本对恢复至关重要。  
受访者还提到建材短缺、熟练工匠不足、资金受限、审批拖延及机构协调不力等阻碍重建问题；地方官员并报告水源污染、土壤侵蚀与生态破坏等环境与公共卫生风险。  
评估呼吁在从紧急救援向恢复与重建过渡过程中，持续为基础设施和生计提供支持，并加强企业与产业的气候与灾害韧性，特别关注非正规劳动者与小微企业，避免弱势群体再次陷入贫困循环。
