综合法国国际广播电台中文网与路透社报道，上述星期三（5月6日）公布的研究，由国际咨询机构毕马威受中国欧盟商会委托完成，研究显示，欧盟相关措施一旦实施，德国将成为损失最严重的国家，预计损失超过1700亿欧元，占总额近一半；法国和意大利的损失分别约为463亿欧元和365亿欧元。
报告所估算的损失不仅涉及电信运营商，还涵盖整个经济体系，包括更换现有设备所产生的额外支出、转向价格更高且可能效率较低替代方案的成本，以及行政、法律费用和因强制过渡带来的延误。
能源和电信将是受影响最严重的两个行业，而这两个行业恰恰是欧盟数字化和绿色转型计划的支柱。
欧盟委员会今年1月提议，在修订《欧盟网络安全法》过程中，允许欧盟范围内禁用被认定对网络安全构成风险的外国电信设备供应商，包括中国企业华为与中兴通讯。 
中国驻欧盟使团4月29日在布鲁塞尔一场会议上警告，如果欧盟坚持推进这一提案，“中方将不得不采取对等反制措施，对欧盟及企业作出回应”。
欧盟委员会一名发言人星期一（5月4日）称，已建议成员国将华为和中兴的设备排除在当地电信运营商的连接基础设施之外。 
欧洲多个国家及英国目前已限制在电信网络关键环节使用华为设备，理由是华为与中国政府的关系可能构成安全风险。

英国地方选举将于5月7日举行，民调显示执政工党面临沉重失利，可能加剧对首相基尔·斯塔默的党内不满，并推动左右民粹势力抬头。此次投票是斯塔默自2024年7月大选大胜以来的最大一次选民检验，涉及苏格兰、英格兰和威尔士数百万选民。民调普遍预测对工党不利，可能引发领导权质疑，但党内尚无明显接班人。

民调显示工党可能在威尔士失去执政权，若成真将是威尔士自27年前设立议会以来首次失守，卡迪夫大学政治学者劳拉·麦卡利斯特称其后果“灾难性”。在苏格兰，民族党（SNP）预计将延续掌权，民调甚至显示改革党可能挤下工党位居第二。
英格兰方面，民调员罗伯特·海沃德预计工党约失去1,850个其防守的地方议席，全国约有5,000席位竞选；改革党或从工党与保守党手中夺得约1,550席，绿党则可能在伦敦等地凭借亲加沙立场获得数百席位。
斯塔默自结束保守党14年执政以来屡遭政策争议，近期又陷入被解职驻美大使彼得·曼德尔森相关丑闻，经济提振乏力，但在对美伊问题上有抵抗美国的正面评价。媒体有报道称若选举失利，部分前内阁成员或会推动罢免，但任何挑战者需获得20%议员提名；政府部长史蒂夫·里德斥该类说法“无稽”。

伊朗表示正在审议美方新提议，并将通过巴基斯坦尽快回复，巴方曾主持本轮唯一和平会谈并一直充当双方主要联络人。 
有巴基斯坦消息人士及介入斡旋的知情人证实，美国媒体Axios报道称，美伊正在接近一份14点、一页纸的备忘录，拟形式上结束海湾战争，随后再就霍尔木兹海峡通航、解除美国制裁及伊朗核问题展开详细谈判。巴方人士称：“我们很快会结束，这事儿接近了。” 
特朗普当天在社交媒体上未披露具体文本，只称“如果伊朗同意已达成的内容，战争就可结束”，并警告若不同意将升级轰炸。他随后对《纽约邮报》表示，目前尚不宜举行签署协议的面对面会晤。 
白宫、国务院及伊朗方面未回应路透问询。以色列也未公开评论，并据称未获消息，仍在为冲突升级做准备。 
受相关报道影响，国际油价暴跌，布伦特原油一度下挫约11%，至约98美元/桶，全球股市上扬、债券收益率下行。 
据称，美方谈判由特朗普特使史蒂夫·W及女婿贾里德·库什纳主导。若双方接受初步备忘录，将开启为期30天的详细谈判，力图达成包含解除制裁、释放被冻结伊朗资金、互相解除对霍尔木兹海峡封锁及对伊核计划进行限制的全面协议，目标包括暂停未来铀浓缩。 
报道显示，该一页备忘录并未包含美国此前向伊朗提出但被拒绝的若干关键要求，如限制导弹项目、停止支持代理武装或交出已存在的约400公斤接近武器级的浓缩铀。伊朗议员、议会外委会发言人伊布拉欣·雷扎伊将该文本称为“美国的愿望清单”。伊朗外长阿巴斯·阿拉格奇在访华时表示，德黑兰寻求“公正全面”的协议。 
此前特朗普宣布暂停名为“自由计划”的护航行动。该行动并未显著恢复海峡通航，反而引发伊朗对船只及邻国目标的新一波打击。法方一家航运公司5月6日报告称，其一艘集装箱船日前在霍尔木兹海峡遭击中，受伤船员已被撤离。

一艘遭遇罕见汉塔病毒疫情的豪华邮轮MV Hondius在佛得角海域滞留后，于5月6日启程前往西班牙。船上近150人，此前已疏散三人（其中两人病情严重）。西班牙卫生部长莫妮卡·加西亚表示，邮轮预计三天内抵达加那利群岛的特内里费，船上未出现症状者若到港仍健康，将被遣返；14名西班牙乘客将集中在马德里一所军医院隔离，隔离期将视其可能接触时间而定（报告称潜伏期可达45天）。
世卫组织称共有8人疑似感染、3例实验室确认为阳性。瑞士政府表示，一名曾乘船回国的男子在苏黎世接受治疗，未构成对公众的危险。此前一对荷兰夫妇及一名德国乘客已死亡，一名英国乘客在南非重症监护。两架空中救护飞机于5月6日从佛得角起飞，运送三名疑似感染者前往荷兰，预计约当地时间19:30降落。世卫组织总干事特德罗斯认为，此次疫情与新冠疫情爆发并不相似。
汉塔病毒通常通过受感染的啮齿动物及其排泄物传播，人传人极为罕见，但在南美流行的安第斯株曾在密切接触者中出现有限传播。南非国家传染病研究所检测显示，导致约翰内斯堡死亡的荷兰女乘客和仍在院治的英国男乘客均为安第斯株。南非卫生部已开展62名接触者追踪（包括机组与医护人员），目前无确诊病例。佛得角拒绝让乘客上岸，世卫组织与欧盟请求西班牙接收邮轮，中央政府最终同意靠泊特内里费；加那利地方领导人对此表示反对并要求与首相会商。

基辅——乌克兰总统弗拉基米尔·泽连斯基5月6日宣布，匈牙利已将今年3月从国有奥沙德银行（Oschadbank）职员处扣押的现金和黄金归还。此举被视为基辅与布达佩斯关系进一步缓和的标志。 
泽连斯基在社交媒体上称：“这是与匈牙利关系上的重要一步——今年3月被匈牙利特别机构扣押的奥沙德银行资金和贵重物品已被归还。”他对匈方的“建设性做法和文明举措”表示感谢。 
今年3月，7名奥沙德银行员工在经匈牙利中转时被扣押，随身携带约4000万美元、3500万欧元及约9公斤金条被没收。匈方次日释放了职员，但现金和黄金一直被扣押至5月6日才归还。基辅指控这是“人质外交”，而匈方称当时按洗钱调查采取行动。 
两国关系因匈牙利亲俄立场长期紧张，但在反对派彼得·马贾尔当选并计划会见泽连斯基后出现缓和迹象。此前执政16年的欧尔班曾多次利用匈牙利在欧盟内的否决权阻碍对乌援助与对俄制裁。（法新社）
