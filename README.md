中国连续三年开展招生专项行动 严禁中小学设重点班快慢班
（北京／成都综合讯）中国教育部连续第三年开展中小学阳光招生专项行动，要求义务教育学校严禁设立或变相设立重点班、实验班、快慢班，以促进教育公平。
综合新华社与红星新闻消息，中国教育部星期五（4月3日）发布通知，启动2026年中小学阳光招生专项行动。这一行动自2024年开始，至今已实施了三年。
今年的专项行动重点，在严格规范特定类型招生、全面推进义务教育均衡编班、保障重点群体公平入学等。其中，要求中小学严禁违规提前招生、超计划招生、跨区域招生，严禁通过“意向登记”“预录取协议”等名义变相提前招生。
与前两年相比，今年行动的覆盖范围从“义务教育”拓展到“普通高中”，包括要求严禁部属高校附中、省属高中、设区的城市高中违规面向县域掐尖招生。
规范招生行为是行动重点之一，在中国教育部连续三年开展的这一阳光招生专项行动中，均提及中小学不得设立或变相设立重点班、快慢班。
事实上，早在中国2006年修订的《义务教育法》中，便已规定不得将学校分为重点学校和非重点学校，学校不得分设重点班和非重点。
不过，21世纪教育研究院院长熊丙奇也指出，从办学现实看，不少地方仍存在以实验班、创新班为名设立重点班和非重点班的问题。他也认为，不设重点班，首先要扭转基础教育存在的“唯分数”应试化倾向。
中国教育部今年的专项行动中，也包括教育领域的廉政风险防控。通知指出，对招生问题频发、情节严重、社会反映强烈的地区和学校，要严肃追究相关行政部门及学校负责人责任。

农资“忽悠团”近年兴起 给有关部门打击违法犯罪带来困难
中国近年兴起以“忽悠团”形式销售伪劣农用物资的新型犯罪形式，因其具有跨地域流动特点，而给农民维权和相关部门打击违法犯罪带来困难。
综合《新京报》、澎湃新闻等报道，中国最高人民法院、最高人民检察院星期五（4月3日）联合发布“农资打假”刑事典型案例。有案例显示，一“忽悠团”在山西省多地，以免费就餐、赠送礼品为诱饵，夸大虚构产品功效，利用限量促销、买赠等活动向农民销售伪劣农资，售出假化肥197万元（人民币，36.8万新元），主犯获刑11年。
案情显示，2022年2月，张姓化肥厂销售员将与“增值底肥1号伴侣”包装袋标识成分不符的208吨化肥销售给另一张姓被告人，后者明知上述化肥是伪劣化肥，仍对外销售。
张姓被告人将化肥部分销售给他人，剩余部分则由其自行组织人员，先后在山西省长治市上党区、潞城区、武乡县、沁源县，以免费就餐、赠送礼品、送货上门为手段吸引农户前往宾馆、酒店听课，夸大宣传化肥效果，销售给当地农户。经检测，上述化肥均为不合格产品。
山西省武乡县人民检察院分别以销售伪劣产品罪和生产、销售伪劣产品罪，对数名涉案人员提起公诉。武乡县人民法院经审理，以销售伪劣产品罪判处张姓被告人有期徒刑11年，并处罚金人民币100万元。
据介绍，这类典型的农资“忽悠团”的犯罪方式中，不法分子通常以赠送礼品、免费就餐等小利为诱饵，吸引农民参加所谓的“讲座”“推介会”或者“培训会”。
在会议现场，团伙成员常会扮演农业专家或者大学教授等角色，夸大甚至虚构产品功效，利用限量促销、买赠活动等话术诱骗参会农民现场下单。
中国最高法、最高检联合提醒农民，一定要选择正规渠道、证照齐全、信誉良好的门店购买农资产品，对走村串户、来历不明的流动商贩要高度警惕，谨慎购买价格异常低廉并宣传“全能特效”的产品。另外，采购农资后要保留发票、收据和包装、样品，以便为后续维权提供证据。

国家公彩局（GLO）迈入第87年，宣称在2568年度向国家缴纳逾5.421万百万泰铢，并持续推进乐透票创新与永续发展。4月3日局方在นนทบุรี机场น้ำ办公处举行庆祝活动，迎接4月5日成立87周年，局长与高层并邀政府及民间单位出席祝贺。 
局方表示，过去一年在“以创新为本、推动经济与社会”的愿景下，获颁2568年度国营事業永续发展奖（SOE Award），并在印刷、销售、开奖与兑奖四大任务上推进现代化：提升防伪印刷、推出六位数（L6）与三位数（N3）实体与数字彩券、扶持逾75,000名经销商、并依ISO 9001:2015规范公开透明开奖。兑奖方面扩展至กรุงไทย、ธนาคารออมสิน、ธ.ก.ส.等银行、办公处自助车道及“เป๋าตัง”G-wallet等渠道。

局方感谢彩民、经销商与各界支持，表示将持续以创新与廉洁治理，推动组织永续与社会共享价值。
