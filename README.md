国际金融研究所（IIF）在最新《全球债务观察》报告中称，2025年底全球债务创纪录达到348万亿美元，全年增加近29万亿美元，为自疫情以来增长最快的一年。此轮增长主要由政府支出推动，政府债务贡献逾10万亿美元，其中美国、中国和欧元区约占三分之二至四分之三的增幅。
按主体看，政府债务升至约106.7万亿美元，非金融企业债务约100.6万亿美元，居民负债温和上升至64.6万亿美元。成熟市场总体债务约231.7万亿美元，新兴市场约116.6万亿美元；以GDP比重计，全球债务占比小幅降至约308%，但新兴市场负债率创纪录突破235%。
