中国航协党委书记、理事长王昌顺主持会议并讲话。他说，协会接下来突出目标导向，确保任务落实，持续扩大第四届中航协（CATA）航空大会影响力，切实加强无人机操控员培训机构管理，抓好教育培训转型，不断加强研究咨询业务。

中国科技部部长阴和俊上星期一（6日）在接受人民日报采访时明确表示，要“加快新能源、新材料、航空航天、低空经济等战略性新兴产业集群发展”。
另据上观新闻报道，以“驭低空新势，启经济新篇”为主题的2026国际低空经济博览会，将于7月22日至25日在上海国家会展中心举办。
