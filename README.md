德利马周五在声明中说，最高法院周四发布、维持2025年7月判决的决定，不仅是驳回或确认原判，而是通过重新定义“启动”弹劾的含义并将立法不作为转化为具宪法效力的行为，越过了解释权，制定了国会并未规定的新规则、时限与后果。
参议院议长文森特·“蒂托”·索托三世也在周四发表类似批评，称法院在弹劾程序上“擅自引入国会须遵循的规则”。
依据1987年宪法，弹劾的启动与审理权在于国会，适用对象除总统与副总统外，还包括最高法院大法官及选务、审计等宪制机关成员。
最高法院在2025年7月裁定，众议院依据第四项并已送交参议院审理的弹劾条款，违反了同一官员在同一年不得发起多次弹劾的禁令；法院并指出程序上有正当程序疑虑，要求众院须向被控者及全体代表提供弹劾条文及证据，确保充分审议与回应机会。
德利马警告，此项判决“规训”了原本由国会决定的程序，可能把弹劾这一政治性保障，转变为由司法操控的程序，背离宪法设计与精神。
