他说：“在最理想的情况下，他们希望台湾通过某种投票或公投，同意并入中国。我认为这是他们更倾向的方式。”

鲁比奥并称，美国长期以来基本维持这一立场，而某种程度上的模糊性也一直定义着美国处理台湾问题的方式。
他说，美国之所以采取这一模糊性策略，是因为“不希望看到冲突，不希望出现破坏稳定的局面，因为这将对世界，以及对中美两国都造成巨大冲击”。

根据独立研究机构IEEFA于5月13日发布的报告，2026年第一季度欧盟从俄罗斯进口的液化天然气（LNG）达到了自2022年俄乌战争以来的最高水平。
报告显示，欧盟在2026年前三个月自俄进口天然气约69亿立方米，较2025年同期增长16%；IEEFA并称4月份进口同比亦增长17%。尽管通过管道的俄罗斯天然气几近停止，但法国、西班牙和比利时等国仍通过油轮大量采购俄罗斯LNG。
俄罗斯目前约占欧盟LNG供应的14%，为第二大供应国。与此同时，欧盟对美国LNG的依赖显著上升：2021至2025年间美国供应量增逾三倍，2026年一季度同比增长27%。IEEFA预测美国将在2026年成为欧盟最大气源，至2028年其LNG占比或高达80%，但也指出美国天然气对欧洲买家而言最为昂贵。
IEEFA欧洲首席分析师安娜·玛丽亚·雅勒-马卡列维奇表示，欧盟从管道气向LNG的转变本意在于保障供应与实现多元化，但中东战争造成的中断和对美LNG的过度依赖已暴露出此举的双重失败。欧盟曾承诺因俄乌战争在2027年底前停止进口俄罗斯天然气和石油。消息来源：IEEFA、法新社。
