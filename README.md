港前民政局长批有偿新闻将毁掉公信力
香港前民政事务局长、《大公报》前总编曾德成批评一些传媒机构在营运压力下热衷“有偿新闻”，有关人员分不清楚新闻与“鳝稿”（软广），甚至连别人免费提供的新闻线索“竟以无进账而不受”，这无异于饮鸩止渴，终将毁掉新闻机构赖以生存的公信力。

曾德成指，当初协会初创立时，不可能预见传媒行业出现如此激变，当年的同业前辈也难应对当今的经营困局。他坦言，传媒是最受信息科技革命冲击的行业。数码化、网络化、智能化迅速发展，社媒成为新世代的第一信息来源，每个人又都可发布图文、视频甚至虚拟场景，载体越多，传播越快，大众传媒碎片化，人工智能聚焦小众。网上讯息浩如烟海，争夺注意力，换取广告，先吸睛后吸钱。
但有谓“真相花钱，谣言免费”。曾德成提醒，协会成立的初衷，是维护新闻工作尊严，坚守真实、客观、公正的原则，应是始终不渝、恒久宝贵的。他说，不仅是协会同仁的共同价值，更是新闻事业走出新天的根本出路。

与此同时，战略要道霍尔木兹海峡几近关闭。美方称4月17日已重开通航，但仍维持对伊朗船只的海军封锁，激怒德黑兰并导致海峡再次封闭。伊朗指责美方封锁违反停火、构成“对人民的集体惩罚”，并称为战争罪与反人类罪。英国海事监控机构报告称革命卫队炮艇曾向一艘油轮开火并击中一艘集装箱船，两船均挂印度旗，印度召见伊朗大使抗议；另有报道称两艘油轮被迫改航。 
特朗普同时威胁，如海峡不重开或停火不延长，将“摧毁伊朗所有发电厂和桥梁”。美国驻联合国大使迈克·沃尔茨表示，打击民用与军用混合基础设施仍在考虑之列并辩称为合法目标。上周万斯在伊斯兰堡主持的首次高层会谈未获突破，特朗普称伊朗在核问题上已基本让步，伊朗坚决否认。巴方尚未证实本周是否继续主办会谈，但4月19日晚伊斯兰堡已实施安保封锁，并增派数千警力和特种部队。 
此番僵局亦冲淡了中东近期的一丝希望：在美方斡旋下，以色列已同意对黎巴嫩真主党停火10天，伊朗则将黎巴嫩战事停止视为与美以谈判的前提。

华盛顿消息——美方称，4月19日一艘美国驱逐舰在阿曼湾对一艘试图规避美军海上封锁的伊朗籍货船“Touska”开火并将其扣押。 
总统特朗普在Truth Social上说，该船无视停船警告后，导弹驱逐舰USS Spruance“在机舱上打出一个洞”，迫使其停航，并称美国海军陆战队已控制该船并正在检查船上物品。 
美国中央司令部（CENTCOM）称，“Touska”正试图驶向伊朗班达尔阿巴斯港。Spruance在下令“撤离机舱”后，用5英寸舰炮多次射击以破坏其推进装置，并在X上发布了伴有远距离射击画面的视频。自封锁开始以来，美军已指示25艘商船掉头或返回伊朗港口。 
特朗普并指出，“Touska”因“过去的非法活动”被列入美国财政部制裁名单；美国财政部海外资产控制办公室的资料将其标注为受制裁的伊朗挂旗集装箱船。MarineTraffic数据显示，在特朗普声明前约6小时，该船位于伊朗南部近恰巴哈尔海域约45公里处；船舶监测网站Tanker Tracker称该船从马来西亚启航。 
此次事件发生在霍尔木兹海峡局势紧张之际——自七周前美以与伊朗爆发战争以来，该海峡作为全球石油和液化天然气通道几乎处于封闭状态。AFP报道。

基辅—乌克兰总统泽连斯基4月19日表示，欧洲应建立自己的反弹道导弹防御体系，乌克兰已就此与数国进行会谈并在推动相关工作。
他指出，防御俄罗斯弹道导弹是乌克兰目前面临的最大挑战之一，只有美制“爱国者”系统中的部分导弹型号能拦截此类弹道弹。俄军常以弹道导弹打击乌方能源基础设施，摧毁热电和输电系统。
“我认为也主张，欧洲应有自己的反弹道防御系统。我们需要在一年内建成，任务很难但可实现，”泽连斯基在乌国营电视台表示，并称已与若干欧洲主要国家讨论此事，但未点名。
此外，制造乌军“火烈鸟”巡航导弹的Fire Point公司向路透表示，正与欧洲企业洽谈，计划到2027年推出一套新的低成本防空体系，作为日益难以获得的“爱国者”系统的替代方案。当前“爱国者”弹药因大量部署于海湾地区应对伊朗方向的威胁而愈发紧缺，意法联合研制的SAMP/T反弹道系统产量也较少。

汉诺威，4月19日 — 在汉诺威工业博览会开幕式上，德国总理弗里德里希·梅尔茨与巴西总统路易斯·伊纳西奥·卢拉·达席尔瓦（卢拉）呼吁加强欧洲与南美大国之间的合作。
两人对欧盟—南方共同市场（Mercosur）自由贸易协定将于5月1日生效表示欢迎。梅尔茨称，此举表明双方坚持多边秩序、希望建立以规则为基础的经贸体系，并力求降低关税，最好实现零关税。
卢拉指出，作为今年汉诺威博览会的主宾国，巴西是可靠且重要的原材料供应方。他称巴西仅勘探了约30%的矿产潜力，却已拥有全球最大的铌储量、第二大石墨和稀土储量、第三大镍储量。卢拉强调，这些资源应推动经济与社会发展，呼吁加大技术转移并在巴西扩大加工能力，并表示愿与梅尔茨就人工智能、关键矿产和数据中心等广泛经贸议题交换意见。
