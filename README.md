以色列拟重开拉法口岸 维持“必要安全限制”
以色列国防部说，加沙地带拉法口岸将于（星期三）3月18日起重新开放，允许“有限”人员双向通行。
新华社报道，以色列国防部负责协调对巴勒斯坦加沙地带人道援助事宜的机构星期天（15日）发表声明说，以方经安全评估和审查恢复运营条件，决定重开拉法口岸，与此同时，仍将对这一口岸维持“必要安全限制”。
这一名为“领土内政府活动协调办公室”的机构称，口岸通行规则将参照关闭前的机制，人员经口岸出入加沙地带需与埃及协调，并需事先获得以方批准，欧盟代表团将监督口岸运行。此外，以方还将对通过口岸的人员实施“进一步的筛查和身份识别程序”。
拉法口岸位于加沙地带南部与埃及交界处，是加沙地带居民最主要的对外通道，也是国际人道援助物资进入加沙的重要通道。自2024年5月以军控制巴方一侧以来，拉法口岸基本处于关闭状态。
拉法口岸于今年2月2日正式重新开放，供数量有限的人员双向通行。以色列与美国2月28日联手对伊朗发起大规模军事行动首日，以方以安全为由关闭拉法口岸。

战争冲击供应 中国最大炼油企业开工率下调10%
由于中东战争不断扩大，以及霍尔木兹海峡运输困难导致原油供应受阻，中国最大炼油企业中国石化已下调开工率。
彭博社星期一（3月16日）引述知情人士报道，这家国有企业本月的原油加工量将较原计划减少约10%。知情人士说，此次削减不包括正在进行的计划性检修造成的产量损失。
中国石化炼油厂每天加工约520万桶原油，约占中国炼油总加工量的三分之一。报道称，削减10%意味着，中国石化3月的炼油加工量可能减少超过每日50万桶。3月通常是炼油旺季，因为春夏季需求在此之前开始上升。中国石化超过一半的原油进口来自中东。
另一方面，彭博社与路透社3月5日引述知情人士称，中国国家发展和改革委员会官员与国内最大几家炼油企业高管会面，口头要求他们立即暂停成品油出口，暂停签署新出口合同，并就已达成的出口协议进行谈判。
不过，这项要求不包括为国际航班提供的航空燃油加注、保税仓库中储存的航空煤油和船用燃料油，以及对香港和澳门的供油。

日本启动释放创纪录石油储备
日本政府星期一（3月16日）启动释放石油储备，释放总量约合8000万桶，相当于日本国内约45天的石油用量，规模创下历史新高。
新华社报道，本次释放采取民间库存先行、国家储备跟进的方式。日本石油储备法规定企业有义务储备石油，并规定它的库存量。
日本政府16日发布通告称，将石油储备法规定的企业储备义务从70天用量下调至55天，允许企业将相当于15天用量的库存用于市场供应。在此基础上，日本政府计划自3月下旬起释放约一个月用量的国家储备。
根据日本经济产业部数据，截至2025年底，日本国内石油储备总量相当于日本254天的用量。其中，国家储备相当于146天用量，民间储备相当于101天用量，产油国共同储备相当于七天用量。

3月15日，国会大楼——ภูมิใจไทย党党首安努廷（อนุทิน ชาญวีรกูล）表示，众议院已完成议长及副议长选举，接下来将办理相关文件并呈请国王御准（ทูลเกล้าฯ）。待议长获御准并就任后，会尽快召集会议，不会拖延。
对于19日可能召集国会投票选总理的传闻，安努廷表示不清楚，也不能事先确定日期，因为议长及副议长等职位必须先获御准。
他称，内阁名单要等总理依程序完全就任后才会提出，届时会按程序办理，不会延误。联席党派的席位配额将于总理选出并获御准后召集协商，随后各党将把部长人选送交内阁秘书处（สลค.）审查。
对于是否会严格把关内阁人选，安努廷说将依宪法法院及有关指引审视，各党须事先在党内经高层把关，ภูมิใจไทย也会筛选，若有人选有问题，将与其他党党魁协商调整，目标是提交无可指责的人选，满足民众期待并遵循伦理标准。
谈及是否已与ธรรมนัส พรหมเผ่า会面，安努廷称已与多人接触，但因人多尚未见到ธรรมนัส，只见到同属一党的อัครา พรหมเผ่า（社会发展与人类安全部长）。他笑言，朋友的关系“永不过期”。

穆斯林长老理事会（主席：阿尔‑艾资哈尔大伊玛目艾哈迈德·艾尔‑塔耶卜博士）在3月15日国际反伊斯兰恐惧症日发表声明，坚决反对一切形式的极端主义、仇恨、歧视及针对穆斯林的暴力煽动，警示伊斯兰恐惧症日益抬头，对共存、社会安定与全球稳定构成严重威胁。 
理事会呼吁发起一场全面的全球性运动，遏制极端、仇恨及宗教歧视性言论，推动明确立法和政策以打击仇恨与宗教歧视，弘扬相互尊重及文化、宗教多元，提升公众对伊斯兰作为仁慈、公义与和平宗教的正确认识，维护人人安居权利。 
声明指出，伊斯兰恐惧症并非孤立个体或极端话语所致，而是因无知与扭曲刻板印象加剧，并被极右势力利用以服务狭隘政治利益，助长分裂与仇恨。应由各国政府、宗教机构、教育界与媒体协同应对，强化公民意识、多元包容与接纳。 
理事会并持续通过东西方对话巡回、国际和平车队、新兴和平缔造者论坛、学生对话项目与斋月宣教等倡议，推广温和理性信念。声明还援引2019年在阿布扎比由艾尔‑塔耶卜博士与已故教皇方济各共同签署的《人类博爱文件》，重申信仰与差异自由为人权，反对以任何形式强迫他人信仰或强加他者无法接受的文明模式。

知名网红“เจ๊เอ๋”（娜塔琳·卡塞姆萨潘）就“请托办理子弹证”一事回击内政部长兼总理安努廷。此前เจ๊เอ๋在节目中称曾“打了一次电话”，随后其从事射击运动的儿子迅速获颁ป.3（枪械许可），此前县长曾拒签。安努廷则回应称与她并不熟识、只见过一次，未曾留过电话，也并未介入枪证事务。
3月15日，เจ๊เอ๋在社交媒体上反驳称：“以后我们不生气了，因为我们互不相识，从未交谈……口罩一盒1800（泰铢）也能应付，洪水有办法，这次燃油也够民众使用。如果我们揭开（真相），你别生气，因为我们不认识。”她并补充指出，“有无职务，是两种完全不同的人。”

谢赫·哈姆丹·本·穆罕默德·本·拉希德·阿勒马克图姆（迪拜王储、副总理、国防部长、迪拜执行委员会主席）在主持迪拜发展与公民事务高级委员会2026年首次会议时，批准了一揽子旨在强化家庭作为社会基石的战略举措，推动迪拜社会议程33目标的落实。
获批举措包括：“谢赫哈茵德家庭计划”第二阶段以支持阿联酋家庭增长与稳固；社区邻里议事厅（Majalis）项目第三阶段并新增数处议事厅，以深化居住区社会联系；“充满活力的迪拜”健康计划，提升政府雇员体质与心理状态；“巴尔扎特迪拜”项目为社区提供交往空间。会议还通过迪拜亲家庭战略，并成立专项小组督导谢赫哈茵德计划各项举措。
迄今迪拜婚礼项目已组织1,077场婚礼，占本地公民婚礼的43.5%，同比增长44%。“充满活力的迪拜”由体检、体能评估与心理准备三部分组成。社区邻里议事厅第二阶段在Nad Al Sheba、Al Awir、Al Warqa和Al Barsha South开设四处；第三阶段将于2026年第二季度启动（预算1.6亿迪拉姆），预计2027年第三季度完工，并新增Nad Al Hamar、Jumeirah等五处议事厅。
另批准在Al Khawaneej古兰经公园开设Thukher俱乐部第二分部；社区发展局与迪拜市政厅将协调落实“巴尔扎特迪拜”并研究在各酋长国推广的可能性。

阿布扎比捐赠与未成年人基金管理局（Awqaf Abu Dhabi）与MAIR集团签署谅解备忘录，拟在阿布扎比的捐赠土地上开发总值2亿迪拉姆的项目。 
此类开发将通过房地产及其他资产投资增强捐赠收益的可持续性，产生稳定回报以支持阿联酋孤儿的关怀与福利项目。该举措支持在穆罕默德·本·扎耶德·阿勒纳哈扬赞助下发起的“国家之母孤儿捐赠”运动，旨在为孤儿关怀提供可持续资金来源并改善其社会经济稳定。 
根据协议，Awqaf Abu Dhabi提供捐赠土地，MAIR集团旗下的Makani Real Estate负责项目开发、管理和实施。 
Awqaf总干事Fahd Abdulqader Al Qassim表示，这一合作是将捐赠资产纳入可持续投资框架、提升资产效率并放大社会经济影响的先进模式。MAIR集团董事总经理兼集团首席执行官Nehayan Hamad Alameri称，公司将按照运营标准推进项目，力求创造可持续回报，支持国家级社会价值目标。 
Awqaf Abu Dhabi指出，该倡议是酋长国通过投资合作强化捐赠生态、确保资产保值增值并扩大代际社会影响的更广泛努力的一部分。
