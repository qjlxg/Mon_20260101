【法国文化部长：今后不会资助“用人工智能取代创作者”的影视作品】金十数据5月17日讯，法国文化部长卡特琳·佩加尔16日在第79届戛纳电影节期间表示，国家电影和动画中心计划修改对创作项目的资助规则，以保护创作免受人工智能的冲击。佩加尔当日在戛纳出席活动并向电影从业者发表讲话，其间谈及人工智能给行业带来的挑战。她表示，国家电影和动画中心“将很快修改相关资助规则”，以确保遵循“支持人类创作”的原则。佩加尔强调，创作行为是人类的本质特征。她说，人工智能是一种工具，可以加快、简化、丰富电影制作过程；但是，一部作品是一种视角、意图和情感，任何算法都无法做到这点。佩加尔明确表示，国家电影和动画中心“今后不会资助没有作者的作品”。她同时强调，这并不意味着“受到资助的作品中禁止使用人工智能”，而是不会资助那些“用人工智能取代创作者的作品”。

伦敦警方为5月16日可能迎来的大规模极右翼集会与对抗示威做足准备。大都会警察厅称将部署约4,000名警力，并动用警马、警犬、无人机和直升机，应对汤米·罗宾逊（真名史蒂芬·雅克斯利-伦农）组织的游行、与之合并的亲巴勒斯坦示威以及足总杯决赛。警方对两场集会的路线和时间设限，力图分隔对立人群，并表示此次行动预估耗资4.5百万英镑，将采取“零容忍”政策。首次要求组织者对受邀演讲者遵守仇恨言论法律承担法律责任，且将首次在示威中使用实时面部识别技术。
首相基尔·斯塔默在警务指挥中心视察后警告称，任何在街头制造混乱、威胁或恐吓他人的人都将面临法律制裁，并指责5月16日的极右翼集会在“传播仇恨与分裂”。警方估计罗宾逊的支持者约5万人，纪念1948年巴勒斯坦被迫迁徙的纳克巴日对立集会约3万人，反法西斯组织Stand Up to Racism已将其游行并入纳克巴日活动。

纽约长岛铁路（LIRR）约3500名工人于2026年5月16日罢工，为32年来首次，导致美国最繁忙的通勤铁路全面停运。LIRR日均运送约30万乘客，罢工发生在阵亡将士纪念日长周末前夕，可能造成严重出行干扰。
国际卡车司机兄弟会表示，此次罢工由五个工会发起，称在谈判期间已有三年未获加薪。机车工程师与列车员兄弟会主席马克·华莱士表示，如果MTA接受政府多次建议的合理条件，罢工本可避免；运输通讯工会全国副主席尼克·佩卢索称双方工资差距约1个百分点。
大都会运输署（MTA）在官网确认停运并称将继续谈判，敦促乘客尽量在家工作，警告停运将导致严重拥堵与延误，并在工作日为必须上班者提供有限接驳巴士。纽约州长凯西·霍楚尔呼吁各方重返谈判桌，称MTA已本着诚意提出多次包含实质性加薪的方案。总统唐纳德·特朗普在社交平台指责霍楚尔应对此负责并提出可协助解决。MTA首席执行官贾诺·利伯表示不能接受会使预算无法承受的协议，但称最后一份被拒的提案已在薪酬方面满足工会诉求。来源：路透社

美国国务院5月16日宣布，以色列和黎巴嫩同意将停火再延长45天，并就政治解决方案展开更广泛会谈。两国政府代表在华盛顿会晤两天后表示，原定于5月17日到期的停火将“延长45天以便取得进一步进展”，美国国务院发言人汤米·皮戈特（Tommy Pigott）透露。美国方面计划于6月2至3日就长期政治协议进行谈判，五角大楼将于5月29日召集双方军方代表会谈。
黎巴嫩代表团表示，延长停火并启动军事对话可为民众争取“关键喘息机会”，目标是实现“持久稳定”。黎巴嫩总理纳瓦夫·萨拉姆在贝鲁特一场活动上隐约指责真主党，称“我们受够了为外国项目或利益服务的鲁莽冒险”。
尽管有停火，冲突并未停止。停火续签宣布后，以色列在南部汉努夫的一处与真主党有关联的伊斯兰健康委员会中心发动袭击，黎巴嫩卫生部称造成6人死亡，包括3名医护人员。提尔城也遭遇空袭，造成至少37人受伤，其中有医院工作人员、妇女和儿童。
以色列称一名士兵在黎南部阵亡，自3月初与真主党冲突以来阵亡以军士兵增至19人，另有一名民间承包商遇难。以方称上周击毙逾220名真主党武装分子并打击数百个目标。联合国驻黎人道事务协调员伊姆兰·里扎警告称，黎巴嫩局势“极为令人担忧”，空袭与夷毁造成不可接受的平民与基础设施损失。
自4月17日停火以来，据黎巴嫩当局统计，已有逾400人在停火期间丧生，自3月冲突爆发以来全境死亡人数已超过2900人。
