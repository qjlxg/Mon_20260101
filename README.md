叙利亚列库尔德语为国家语言 库尔德人：未达期待
叙利亚把库尔德语（Kurdish）列为叙国家语言之一，并宣布将保护叙库尔德人的语言、文化，以及其作为叙公民的合法权利。库尔德人说，这一举措仍未达到他们的期望。
根据叙利亚过渡政府总统艾哈迈德·沙拉（Ahmed al-Sharaa）星期五（1月16日）签署的法令，叙库尔德公民为叙利亚人民不可分割的组成部分，同时其文化和语言认同是叙统一而多元的民族认同中不可分割的组成部分。过渡政府将保障库尔德公民在国家主权框架内传承其传统文化、艺术和语言的权利。
法新社报道，这项法令是自1946年叙利亚独立以来，对库尔德民族权利的首次正式承认。法令淮许库尔德人集中地的公立学校开设库尔德语课程。
过去数十年来，库尔德人在叙利亚历任统治者手下遭到边缘化和压迫，在近期冲突后，这一举措被认为是叙利亚当局对库尔德少数民族释出的善意。
新华社引述叙当地媒体称，叙境内目前居住着上百万库尔德人，他们大多生活在叙利亚北部和东部地区。
叙利亚北部与东北部的库尔德地方管理当局发表声明说，这项法令是大马士革当局“迈出的第一步，但并未满足叙利亚全体人民的愿望与期盼。”

此前应丹麦邀请，挪威、瑞典、法国、德国、英国、荷兰、芬兰及比利时等国宣布向格陵兰岛派兵，参加在岛上举行的联合军事演习。

声明也说，此前丹麦协调盟国计划在格陵兰岛举行军事演习，旨在加强北极地区安全，不针对任何一方。欧盟将全力支持丹麦和格陵兰岛人民。
特朗普要占有格陵兰岛的理由是要防止未来中俄占据这个世界最大岛。欧洲国家于是提议加强北约在北极地区的防卫，希望借此打消特朗普对格陵兰的念头。
然而，特朗普近日仍公开质疑丹麦保卫格陵兰岛的能力，还批评丹麦对北极地区安全的投入太少。他还声称，格陵兰岛“到处是俄罗斯和中国船只”。
丹麦和格陵兰民间召集在当地时间星期六举行大规模示威，抗议特朗普觊觎格陵兰。

美国对部分高性能半导体加征25%关税，分析人士称这将提高中国企业部署美制先进AI芯片的成本。
美国政府周三宣布，新关税适用于英伟达H200、超威（AMD）MI325X及其同类产品和衍生品。白宫称，依赖海外半导体供应链构成国家安全和经济风险，新增关税旨在鼓励本土产能并减少对台湾等制造枢纽的依赖——意味着代工厂（如台积电）生产并运往中国的芯片可能须先经美国中转。
商务部下属工业与安全局（BIS）日前对英伟达在华销售H200给予有条件批准，要求每批H200在出口前须在美接受第三方独立检测，并将对华出货量限定为不超过对美客户出货量的50%，且不得因中国需求挤占全球代工产能。关税依据为1962年《贸易扩展法》232条款，但对“支持美国技术供应链建设和增强本土制造能力”的芯片可给予豁免，商务部长有广泛裁量权，白宫称后续可能扩大相关征税范围。
经济学人智库高级分析师Chim Lee指出，此举可能是特朗普政府口中“对H200和MI325X征收25%分成”的落实路径，也可能利用法律绕开美国宪法对出口税的限制。Lee并称，北京已在政治上不鼓励私企过度依赖进口AI芯片，但中国开发者具有较强适应性。日经亚洲报道则称，北京正在制定对从英伟达等外国供应商购买先进AI芯片数量的限制规则。

关于欧洲科技未来的讨论既及时又必要。平台技术、高端半导体、超大规模云与人工智能的重心确实向美中转移，但将此视为欧洲不可避免的技术边缘化是错误的。欧洲拥有一项常被低估的战略优势：将复杂创新转化为合规、可融资且受信任的跨国落地系统。在多极化世界，这种可靠部署能力与发明本身同样重要。
未来的科技竞争不只是看谁训练出最大的AI模型，而是看谁能在经济、医疗、能源电网和高铁等现实系统中将前沿技术安全规模化应用。影响力往往由最后一公里决定：认证、复杂系统集成、网络安全与长期运维，这决定了一个国家是否真正从AI与自动化中获益。
西门子、博世、爱立信与空中客车等欧洲企业长期在多种监管环境下运营，擅长安全关键系统集成与长寿命资产管理，赋予欧洲在大规模执行上的比较优势。这是硅谷速度或北京规模难以复制的能力。
实现战略自主不需要技术孤立或选边站队，而应占据“部署中立”的位置：帮助各国尤其是全球南方在避免供应商锁定与管理风险的前提下采用先进技术。可操作的五大领域包括：一是提供认证与合规服务（欧盟AI法已提出安全伦理标准）；二是推动跨厂商系统集成与互操作标准；三是构建全球服务与运维网络，保障可靠性；四是配套出口信用与绿色金融以支持大型基础设施；五是把部署产生的运行数据形成反馈，指导安全关键软件与机器人等研究。
这不是放弃创新，而是务实策略：欧盟应把部署视为关键基础设施，投资共享测试床与基准体系，呈现为能提供选择而非依赖的中立可信伙伴。在分裂的世界里，促进来自各方的最佳技术融合不仅合乎经济利益，也有助于地缘政治稳定。欧洲已有全球足迹、监管信誉与运营专长，挑战在于将这些优势组织成一致战略，把部署能力作为多极世界技术主权的核心支柱。

【伊朗瓦解一间谍组织 指其参与骚乱并纵火袭警】金十数据1月18日讯，当地时间1月17日获悉，伊朗情报部门近日在行动中瓦解了一个由32人组成的间谍组织网络，该组织成员近期积极参与骚乱和破坏活动，其中12名成员被逮捕，另有13名成员被传唤。根据伊朗情报部门掌握的信息和被捕人员的供述，该组织成员在骚乱期间曾纵火焚烧清真寺、袭击政府大楼以及煽动和唆使暴徒袭击警察局和军事部门等。据调查显示，该组织的一个主要成员曾多次前往以色列。目前，针对该组织的后续调查工作仍在进行中。

【荷兰国际银行：欧洲国家不应对特朗普的关税威胁做出仓促反应】金十数据1月18日讯，荷兰国际银行全球宏观主管Carsten Brzeski认为，欧洲国家不应对特朗普的关税威胁做出仓促反应。Brzeski表示：“不要理会，静观其变。欧洲已经表明不会全盘接受，因此与军事入侵的威胁相比，关税实际上已经是向前迈出了一步。”
