综合港媒《明报》《香港经济日报》和“星岛头条”等报道，香港美国商会星期二（4月28日）举行研讨会，崔建春与伊珠丽（Julie Eadeh）皆出席并先后致辞。

此外，中美目前就特朗普访华事宜保持沟通。崔建春表示，有信心在两国元首指引下，推动稳定及可持续的关系，进一步增强在港美国商界信心，确保其在大陆及香港的长期运营，加深中美互惠双赢合作。
