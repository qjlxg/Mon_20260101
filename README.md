中国整治短视频标注乱象 六平台处置违规账号逾3400个
针对虚构摆拍、人工智能（AI）生成等短视频误导乱象，中共中央网信办要求全面规范短视频内容标注，抖音等六大平台已清理违规短视频3.7万余条，处置账号3400余个。
据“网信中国”微信公众号消息，目前各平台在短视频内容标注上的标准与尺度不一，一些含有虚构演绎、摆拍营销、AI生成等内容的视频未作明确标注，严重误导公众认知，扰乱社会秩序，影响网络生态。
针对上述问题，网信办拟从三方面推进规范工作。一是对短视频内容标注使用的标签进行规范，明确必须设置的标签种类。二是将内容标注设为短视频发布的必经环节，引导用户对发布的短视频内容进行标注。三是对存量短视频内容进行分批回溯和补充标注，推动实现应标尽标。
对此，抖音、快手、腾讯、小红书、哔哩哔哩、微博等六家重点平台近一个月以来，深入清理虚假摆拍等违规短视频3.7万余条，处置违规账号3400余个，补充标注短视频60余万条，并累计发布治理公告18期，曝光一大批典型案例。
在此基础上，各重点平台先行先试，对短视频内容标注功能进行优化，将内容标注入口提至短视频发布一级页面，方便用户进行标注。
网信办指出，接下来将明确具体工作要求和工作进度安排，在全国范围内统一部署推进规范短视频内容标注工作，并同步开展巡查取证，严惩未按要求进行标注的账号和主体责任落实不力的网站平台，公开进行曝光。

“储备油可用101天”是能源部与中东战区局势指挥中心（ศบก.）在镜头前反复宣称的口号，后增为104天。现实却是成公里的油桶队列、皮卡与大货车司机彻夜排队、各地油站高挂“柴油售罄”告示，连曼谷也受波及。公众质问：油若未缺，究竟去哪了？若政府坚持没有“蒙面囤货者”，只是民众恐慌，这究竟是事实澄清，还是推卸责任、挑战公信力？
首相安努廷19日会后说产能77百万升/日、需求84百万升/日，缺口7百万升归咎于民众囤油；但副首相兼ศบก.负责人披露，部分油站配额由15,000升骤减到5,000升，千升油“不见”是否流向更高利的渠道仍成谜，海运走私以赚取每升逾10铢价差的疑虑未被正面澄清。
问题核心并非无油，而是扭曲的价格结构：政府将泵价钉在30–33铢，油基金每日赤字增约20亿铢，批发与零售间出现每升11–18铢的价差，工业与大车争抢本应留给民众的油料，致分配体系崩坏。尽管已要求OR试点统一对jober价格，并放行储备冲刷与货车全天运行等短期缓解措施，但“谁在说谎”“油到底去哪”仍无明确答案。
影响已扩散至基层：灌溉泵无柴油、化肥涨破千铢/包、运输延误、SME受创，甚至部分县区火化场因无油暂停服务。若政府继续只信室内数据而非路面现实，信任危机恐远超油价本身。政府应以透明、可检验的措施回应民众质疑，公众的声音需要被认真聆听。

商务部紧盯全泰商品物价，强调供应充足、发现侵害消费者立即依法处理
商务部常务秘书处会同全国76个府的商务厅，自3月5日至18日密切监控中东局势可能对物价和民生的影响，并于3月18日完成现场抽查。此次共检查1,005处，包括538家加油站、97个市场、126家批发零售商及244家肥料店。
总体看，民生消费品供给充足，未发现囤货或拒售现象；商务部要求商品价格与成本相符并加强监管。抽查期间发现5起未张贴价格标示的违规案，分布于春蓬、沙缴、色军那空、乌汶及清莱5府，均已依法处理。
商务部表示将在非常态期间持续主动监管，若查获疑似侵害消费者的行为将严肃起诉。民众若发现疑虑可拨打内贸厅热线1569或向各府商务厅投诉，相关人员将前往稽查，查实即依法处置。

泰国政府彩票办公室提醒，近期有不法分子以伪造或篡改号码的彩票骗取奖金，呼吁中奖收购者提高警惕，仔细辨认六位数纸本彩票以免受骗。

如怀疑被篡改，可用放大镜检查号码区或期号处是否有贴补、覆写、刮擦或移花接木等痕迹。办公室强调，六位数纸本彩票的印制依ISO 9001:2015质量体系进行，难以伪造，但仍提醒收购者谨慎核验，相关检验方法可见官网 www.glo.or.th。
