日本热浪致东京453人送医 创2010年以来单日最高纪录
日本正遭受严重热浪侵袭，东京消防部门周四说，东京周三因高温送医的人数创下2010年有记录以来的单日最高纪录。此次高温天气已造成一人死亡，另有四人情况危急。
东京消防厅指出，周三（7月22日）共有453人被送入医，创下2010年有记录以来的单日最高纪录。同时，紧急救援服务的出动次数也达到至少63年来的最高水平。
消防部门发言人告诉法新社：“我们呼吁民众保持水分充足并使用空调。患者人数似乎随着气温升高而增加。特别是在昨天和前天，我们确实观察到了显著的增长。”
发言人补充说，消防部门周三共出动救援3612次，创下1963年有相关记录以来的最高纪录。
他指出，人口约1400万人的东京都已有一人死亡，另有四人情况危急，18人病情严重。

日本今年引入“酷热日”（kokushobi）一词，用来形容气温超过40摄氏度的高温天气。这个新词旨在提高公众对高温预警的关注度，但不会触发全国范围的官方措施。
周三至少还有六个地区的气温达到了39.8摄氏度。

香港住宅租金料创10年来最大暑期涨幅
随着香港房地产市场复苏势头增强，当地住宅租金预计将创下10年来最大暑期涨幅。
据彭博社报道，中原地产数据显示，今年上半年，香港住宅租金较去年下半年上涨近4%。房地产中介预计，随着中国大陆专业人士和学生持续赴港，加上外籍人士的租房需求增加，香港住宅租金将出现2016年夏季以来的最大涨幅。
网上租赁平台Spacious.hk首席运营官费舍尔（James Fisher）说：“学生和大陆人士仍是租赁市场炽热的主要推动力。与过去几年夏季租金的季节性上涨相比，今年最大的不同无疑是外籍人士市场需求强劲。”
报道称，租金飙升反映香港楼市正从持续多年的低迷中稳步复苏。截至今年5月，香港房价已连续12个月上涨，创2018年以来最长连涨纪录。此前，大批外籍人士在冠病疫情后离开香港，楼市一度承压。
楼市回暖有利于在上环和山顶等热门地区持有住宅单位的个人业主。尽管香港房地产发展商持有的出租物业不多，但租赁市场蓬勃可提高投资者需求，进而带动住宅销售。
不过，租金上涨也加重学生和新抵港人士的负担。按每平方英尺计算，香港部分住宅的租金甚至高于纽约和伦敦。
过去几年，香港推出多项新工作签证计划，吸引中国大陆专业人士赴港，为当地租赁市场带来支持。港府推动大学招收更多非本地学生，也进一步推高租房需求和租金价格。
在邻近商业区、以中产阶级居民为主的上环，一套面积约500平方英尺的典型两居室公寓，平均月租约3万4000港元（5591新元），与纽约曼哈顿同等面积的公寓相若。
香港住宅一向以面积较小、价格较高著称。尽管纽约租金近年来持续上涨，缩小了两地差距，但香港住宅通常仍比其他金融中心更狭小、昂贵。
Spacious.hk数据显示，山顶、清水湾、苏豪区和浅水湾等外籍人士聚居区的租金涨幅高于平均水平。与此同时，来自海外、浏览该平台出租房源的流量同比增加9.4%，其中来自美国、新加坡、英国、澳大利亚和日本的搜索量均录得双位数增长。
