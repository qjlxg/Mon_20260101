2026年3月6日，司法与国家融合部长哈沙纳·纳纳亚卡拉在议会表示，太平绅士（Justice of the Peace，简称JP）的任命将不再以政治为依据。该职务今后将设立投诉机制，并建立全国性JP名录，便于在公证或盖章时核验身份。 
部长称，正准备一套包含全国JP数据的系统，目前正在录入，完成后可即时核查是否为合法JP。针对违规JP，已开通WhatsApp、电子邮件和电话举报渠道，但对既有任命尚未采取全面取消措施。 
他并指出，未来任命将要求每两年更新个人资料，并须经Grama Niladhari与区级秘书处认证；同时，任职最高年龄将限制为75岁。负责起草的委员会认为，受任者必须具备阅读和理解复杂文件的身心能力，部长表示目前未见改变年龄限制的充分理由。

美国财政部长斯科特·贝森特周五宣布，允许印度在30天内购买俄罗斯原油，以保障能源供应。美国财政部外国资产控制办公室（OFAC）发布一项与俄罗斯相关的许可，授权交付并出售截至2026年3月5日已装船、来源于俄罗斯联邦的原油和石油产品至印度，相关交易获准至2026年4月3日止。该许可也涵盖因各种制裁而被滞留海上的油轮货物。 
贝森特称印度为“美国的重要伙伴”，并表示这项有意短期化的豁免旨在在中东紧张局势下稳定全球能源市场。他强调，许可仅针对已滞留海上的原油交易，不会为俄罗斯政府带来显著财政利益，同时预计印度会增加对美国产油的采购，称此项临时措施可缓解因伊朗行动造成的能源压力。 
去年11月，为对俄侵略乌克兰施压，特朗普政府对俄油企卢克石油和俄罗斯石油实施制裁。数据显示，印度1月自俄进口原油约110万桶/日，为2022年11月以来最低，市场份额降至21.2%，但2月回升至约30%。 
此外，海湾重要油田近期遭袭，伊朗封锁霍尔木兹海峡使供应受挫，沙特塔努拉炼厂和伊拉克鲁迈拉油田等曾遭攻击，布伦特油价今晨上涨至83.07美元/桶。尽管国际油价上扬，消息来源称印度不会提高汽油和柴油零售价。
