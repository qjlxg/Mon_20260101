【特朗普政府拟再废除702项联邦法规 加码放松监管】金十数据7月5日讯，特朗普政府正加大力度削减联邦政府监管负担，公布了一项监管改革计划，拟废除702项现行行政法规。这份于周五发布的改革方案，创下白宫半年度监管议程中拟推进放松监管措施数量的新纪录。这一数字约为特朗普首个总统任期内任何时期相关改革规模的两倍。此外，自联邦政府本财年（始于2025年10月1日）开始以来，政府已最终完成或落实752项放松监管措施。根据计划，政府拟在9月底本财年结束前废除的措施包括：针对能源项目的环境审查要求、能源效率标准，以及推动多元化、公平与包容（DEI）的相关规定。特朗普政府预计，到9月底完成所有计划中的监管废除措施后，将为美国经济节省1.5万亿美元成本。

马未都回应“被盗佛像”疑云：尺寸细节存多处差异
中国海南一尊被盗佛像疑似现身知名收藏家马未都创办的观复博物馆。马未都回应相关争议时说，丢失佛像的照片与馆内佛像间存在身高、腰间束带、左腿裂纹等多处不同。
近期有网民发现，观复博物馆（北京）展览的一尊佛像与海口市五公祠2000年丢失的一尊宋末元初铸铜佛像相似，并晒出相关图片。观复博物馆星期四（7月2日）称，已向国家文物局、北京市文物局等上级主管部门上报申请鉴定。
马未都星期六（4日）晚在社媒发布视频，站在馆内引发热议的佛像前，首度回应相关争议。
马未都说，2005年在厦门白鹭洲古玩城征集文物时发现这尊佛像，并与古玩城的合法商户“细致商讨”，以了解其来源。对方告知佛像来路清晰后，马未都团队“仍不放心”，进行了近半年的全方位调查，“没有发现任何报案记录，也没有看到网络及新闻媒体报道丢失的新闻”。
进行充分调查后，团队在2005年底斥巨资买入佛像，四个人押车运回了北京。那之后便在观复博物馆出口展出，至今已有20年。
马未都同时指出，海口五公祠丢失佛像是在2021年6月1日才上传至中国被盗（丢失）文物信息发布平台，距离丢失时间已有20年。
在将平台报案信息以及海口方面提供的信息和观复博物馆内佛像作对比后，马未都称，丢失佛像照片与馆内佛像间存在身高、腰间束带、左腿裂纹等多处不同。
他举例指出，报案信息显示丢失佛像高130厘米，而馆所藏佛像为146厘米，坐姿差16厘米。此外，从已知照片来看，袈裟环装饰、腹部裂纹、袈裟边缘花纹、僧裙柱孔、双手手印及握姿等多处细节也存在明显不同。
马未都说：“我们依然接受最终的鉴定结果。无论是什么结局，最终都是一个圆满的结局。”他也感谢发现问题的网民，称其有助于博物馆完善制度。

中国二手燃油豪车价格暴跌 保时捷仅卖15万人民币
中国二手车交易市场的豪华燃油汽车近期出现一轮密集降价，其中德国跑车品牌保时捷一款入门车型仅卖15万元（人民币，2.86万新元），报价较去年腰斩。

入门级保时捷Macan车型的二手报价则低至15万元。而新车指导价曾接近90万元的德国大众途锐，二手实际成交价已跌至6.8万元。

豪华燃油车的“保值神话”或走向消散，中国汽车流通协会历年保值率报告显示，保时捷三年保值率从2022年度的92%降至2025年的67%，跌幅超25%。去年数据显示，三大德系豪华品牌（马赛地、宝马、奥迪）三年保值率均退至50%至60%区间。
