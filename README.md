中国发布一项新方案，要求各城市将青年发展融入城市规划、住房、医疗、教育和公共服务体系，包括拓展房源为高校毕业生异地求职提供便利、深入实施育儿补贴制度等，进一步提升城市对青年、儿童及家庭的支持力度。
中国共青团中央、中央网信办、国家发展改革委等15部门星期三（4月22日）联合印发《关于深化青年发展型城市建设 助力建设现代化人民城市的意见》，围绕激发青年创新活力、优化青年宜居环境、引导青年参与基层治理等方面，推出18项具体举措。
《意见》提出，要强化城市规划的适青导向，结合15分钟生活圈建设，鼓励“青春小店”等发展；在青年就业集中区域、文教医机构周边及轨道交通沿线，因地制宜建设宿舍型、小户型青年公寓，降低通勤成本，促进职住平衡。
在婚育支持方面，《意见》提出完善婚恋和社会服务体系，扩大托育补贴覆盖范围，推动公共场所母婴室更广泛覆盖，改善妇产和儿科医疗服务，加强课后及假期托管服务，并推动进城务工人员子女享有更公平的入学机会。
在就业保障方面，《意见》提出推广“青年驿站”，稳定和拓展房源，兼顾免费住宿和低偿住宿，升级配套服务，为高校毕业生异地求职提供便利；同时，针对青年不同阶段的安居需求，构建更完善的服务支持体系。
《意见》明确，到2030年，青年发展型城市理念将广泛普及；到2035年，中国将形成比较成熟完善的青年发展制度体制。
这项政策出台前，北京已于今年3月提出，将在2026年至2030年间建设“生育友好型社会”。中国去年也发布高质量城市发展方案，目标是在2035年前建设更加宜居的城市。

意大利监管机构正在调查国际旅游预订平台Booking.com，理由是它涉嫌向支付更高佣金的住宿商家提供更突出的展示位置。
法新社报道，意大利竞争管理局星期三（4月22日）在一份声明中说，管理局官员与意大利财政警察成员于前一天对Booking.com意大利分公司展开现场检查。
声明说，这家平台疑似在搜索结果中，为加入“优选合作伙伴计划”的住宿供应商，提供更高的曝光度，包括突出商家质量和性价比。
管理局认为，这个“优选合作伙伴计划”的入选标准，似乎主要取决于商家向平台支付的佣金多少，而非基于服务质量高低。
“因此，这些商家的展示方式，加上平台强调质量的宣传，可能会让消费者误以为在其他条件相同的情况下，这些商家的整体性价比高于未加入计划的商家。”
管理局指出，这种情况甚至可能导致消费者以高于平均水平的价格预订住宿。

不超过17岁者终身禁止买烟  英国立法培养“无烟一代”
（伦敦法新电）英国国会两院通过一项法案，限定现年不超过17岁的未成年人终身禁止购买香烟。
这项《烟草和电子烟法案》的目的是要有效遏制2009年1月1日之后出生（如今年龄不超过17岁）的人开始吸烟。这也意味着，香烟销售对象，将严格止于17岁以上的群体，目前不超过17岁的一整代人以及之后的一代代人，都将终身禁止买烟。
法案在获得御准后将正式生效，并赋予政府更大监管权力，将室内禁烟令扩展至儿童游乐场、学校和医院周边等室外场所。同时，政府也可限制电子烟口味和包装，并把电子烟纳入禁烟场所管理。 
这项措施也为了加强预防健康措施，缓解英国国家医疗服务体系长期面临的压力。公共卫生组织“行动控烟”（Action on Smoking and Health）认为，这是公共健康领域发展的重要一步。 
英国卫生大臣韦斯·斯特里廷（Wes Streeting）阐明，新法案通过，是英国公共健康的重要转捩点，有望打造“无烟一代”，彻底杜绝人们长期吸烟成瘾与随之而来的健康风险。
英国政府去年已宣布禁止销售一次性电子烟，原因是价格低廉、包装吸引年轻人，使用率不断上升。
