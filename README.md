韩国扩大燃油税下调幅度 李在明呼吁节约用电
韩国政府决定再延长燃油税下调措施期限，并进一步扩大下调幅度，以应对中东局势紧张给民生经济带来的冲击。
韩联社报道，韩国财政部长具润哲星期四（3月26日）在中央政府首尔办公楼召开跨部门联合记者会，并发布包括上述内容的“紧急经济应对方案”。
具体来看，燃油税下调措施期限将再延长一个月至5月底，3月27日起汽油和柴油燃油税下调幅度将由现行的7%和10%进一步扩至15%和25%。
届时，汽油燃油税每升698韩元（约0.60新元）和柴油燃油税为每升436韩元，将分别降价65韩元和87韩元。夜间运行的营运车辆和线路公交车将免收高速公路通行费一个月。
考虑到中东局势升级，导致国际尿素价格飙升，韩国政府星期五（27日）起将实施“关于禁止囤积尿素溶液和尿素行为”的告示。尿素主要用于农业作为高效氮肥促进植物生长，同时也广泛应用于工业和化工领域。
物价方面，政府将重点监控品目由现有的23项增至43项，包括工业产品、加工食品、快递费和外餐服务费等，并在上半年内冻结公用事业费。
另外，韩国总统李在明就中东局势引发的能源危机，呼吁全民积极节约用电、减少能源浪费，以防政府财政负担加重。
