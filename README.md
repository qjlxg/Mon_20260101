英国《金融时报》星期二（4月28日）引述贸易消息人士报道，中国准备恢复航空燃油、汽油和柴油的出口，中国国内大型国有石油公司已申请出口许可，以便在5月发运。
根据路透社，《金融时报》引述一名了解中国出口意图的人士称，北京希望主要向目前航空燃油库存处于危险低位的亚洲国家供应航空燃油。
另据彭博社引述知情人士报道，包括中国石化和中国石油在内的多家大型国有油企，自上周起已向监管部门申请放行汽油和柴油出口。
伊朗战争初期，北京曾为保障国内供应而限制燃料出口。当时，霍尔木兹海峡关闭引发全球主要原油及成品油进口国恐慌，推动中国优先确保本土能源安全，以应对潜在供应风险。
不过数周后，情况出现变化。知情人士称，中国国内燃料价格上升导致需求意外走弱，部分原因是电动车使用增加，从而缓解了供应压力；与此同时，出口利润空间则显著扩大。
作为全球最大原油进口国，中国多年来持续强化能源安全战略，包括拓展多元供应来源、加大国内产能投资以及建立储备体系，使它在此次供应冲击中较周边经济体展现出更强韧性。此外，中国也在中东战事之初迅速收紧出口，优先保障国内市场供应，稳定本国消费者和工业用能需求。

美国要求多家晶片设备公司停止向华虹出货部分设备
路透社引述两名知情者报道，美国商务部上周下令多家晶片设备公司，停止向中国第二大晶片制造商华虹出货部分设备，这是美国为减缓中国先进晶片发展采取的最新举措。
知情者说，美国商务部已向至少数家公司发出信函，告知它们对运往华虹工厂的设备及其他材料实施新的限制。美国官员认为，这些工厂可能生产中国最先进的晶片。
知情者补充称，美国顶尖晶片设备公司泛林集团（Lam Research）、应用材料公司（Applied Materials）和科磊（KLA Corporation）等，据信都已收到相关信函，这些公司对华供货业务占比较大。
此前，路透社3月16日引述四名知情人士报道称，华虹旗下的华力微电子，正在上海厂筹备投入生产7纳米晶片。
中国最大的晶片制造商中芯国际，目前是中国唯一一家能够生产7纳米晶片的代工厂。报道称，华虹取得的突破，是北京推动晶片自给自足的一项重大进展。
知情者称，美国商务部的这些信函也旨在阻止相关设备和材料运往华力。
在路透社报道美国商务部发出相关信函后，科磊、泛林集团和应用材料公司的股价下跌4%至6%，华虹股价也在星期二（4月28日）下跌3.5%。
近年来，美国商务部以国家安全为由，限制美国企业向生产先进晶片的中国工厂出货设备，以维护美国在人工智能及其他先进晶片领域的技术领先地位。
路透社报道认为，最新发出的信函延续这一政策，但可能在美国总统特朗普计划于5月在北京与中国国家主席习近平会晤前，加剧美中紧张关系。
知情者还提到，美国晶片设备公司及其他供应商可能损失数十亿美元销售额，尤其是在向在建晶圆厂或正升级以生产更先进晶片的工厂供货时。相关限制或将放缓中国推进本土晶片制造的步伐，不过华虹可能可改用来自海外或中国厂商的设备替代。

消息人士：中国加强化肥出口海关检查
随着霍尔木兹海峡关闭引发贸易中断，推动价格飙升并扩大内外价差，中国正加强海关检查，以执行新的化肥出口管制措施。
路透社星期二（4月28日）引述三名不具名化肥贸易商称，按出口量计为中国最大化肥出口品种之一的硫酸铵（ammonium sulphate），此前未被纳入3月出台的出口限制措施，如今其出口也被纳入海关检查范围。
两名化肥贸易商说，此次行动始于中国东部港口城市青岛，当地海关官员发现有出口商将受限出口的尿素和钾肥谎报为硫酸铵。
一名从事该行业的贸易商说：“正因如此，我们的硫酸铵出口近期面临很高的查验率。”
中国是全球主要化肥出口国之一，去年出口额超过130亿美元（166.03亿新元），但为保障农民利益，对化肥出口实行严格管制。上月，北京在春播前限制了大部分化肥出口，仅少数品种被排除在外，其中以硫酸铵最为突出。
这些禁令进一步推高了国际化肥价格。伊朗战争已扰乱霍尔木兹海峡的运输，而全球约三分之一的尿素贸易经由该海峡。
据路透社报道，受出口限制和以煤为主的生产体系支撑，中国国内尿素价格远低于国际水平，形成较大价差，一旦放开出口将带来可观利润。
尿素出口受配额制度控制。北京通常会在5月观察是否出现供应盈余，再评估可出口规模。
StoneX数据显示，中国去年尿素出口量为490万吨，略低于500万至550万吨的历史常态水平；这一水平通常约占全球出口量的10%。

联合声明称，这些国家正密切关注中国针对巴拿马的经济施压，以及近期影响悬挂巴拿马旗帜船只的相关行动。

美国官方3月底指控中国，在巴拿马废除长和的港口运营合同后，骚扰并扣押了悬挂巴拿马国旗的船只，且中国境内悬挂巴拿马国旗船只被扣留的数量激增。
中国否认扣押巴拿马船只，称美国这一言论“无中生有，颠倒黑白”。
