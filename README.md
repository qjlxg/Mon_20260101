报告：加沙10年重建需714亿美元

目前加沙地带约190万人流离失所，占当地人口的绝大多数，超过六成人失去住所。约37万间房屋被毁或受损，逾半数医院无法运作，几乎所有学校遭破坏。
和平委员会加沙事务首席特使姆拉德诺夫接受路透社访问时说，对推动哈马斯解除武装的方案“相当乐观”，但也承认进展仍需时间。他说，过去几周已与哈马斯进行非常认真的讨论，但谈判过程复杂。

台湾监察院星期二（21日）在官网发布文告，通报上述消息。
根据文告，台湾地方公职人员选举的直辖市长、县市长、议员、乡镇市区长等拟参选人，自星期六起（乡镇市区民代表、村里长等拟参选人自8月20日起）经监察院许可设立政治献金专户后，开始收受政治献金，至选举投票日前一日（11月27日）。
台监察院也在文告中公布上一届（2022年）地方选举参选人申报政治献金收入的查核结果。
经监察院廉政委员会审查决议，共计裁处129件、裁罚金额2501万7115元（新台币，下同，101万1942新元），其中捐赠超过限额48件，裁罚金额863万4500元、境外资金捐赠40件，裁罚金额646万9515元、受赠者未尽查证义务九件，裁罚及没入金额合计102万元；另有开立专户未经监察院许可而收受政治献金者，涉有违法情事因属刑责，而移请检察机关侦办五件。

伊朗战争重创德国景气 投资者信心跌至三年多新低
美以伊战争正在侵蚀欧洲最大经济体的复苏势头，德国投资者信心跌至三年多来最低点。
分析人士警告，这场冲突对德国经济的负面影响已不止于推高通货膨胀，企业对长期能源供应短缺的担忧正削弱投资。
彭博社报道，德国智库欧洲经济研究中心（ZEW）星期二（4月21日）公布的调查显示，德国4月经济预期指数从3月的负0.5降至负17.2，创下2022年以来最差表现，衡量当前经济状况的指标也同步下滑。
ZEW总裁万巴赫（Achim Wambach）指出，伊朗战争的经济后果远不止物价上涨，企业对能源长期短缺的担忧正在抑制投资，并削弱政府刺激措施的效果。
德国主要研究机构已将今年经济增长预期下调至不足几个月前预测的一半；在经历连续两年萎缩后，德国2025年GDP仅微增0.2%，基础设施和国防领域的公共投资则提供了一定支撑。
面对能源危机，德国政府已推出16亿欧元（约24亿新元）燃油价格减免措施，并指若局势继续升级，将准备进一步应对方案；欧洲央行则预计在4月30日维持利率不变，以评估战争造成的损害。

全球最大零食店长沙开业 三天被“挤爆”停售
（长沙综合讯）中国头部零食量贩企业鸣鸣很忙在湖南省会长沙开设全球面积最大的零食店“零食王国”，吸引大批民众前往消费打卡，人气远超预期，开业仅三天就宣布从星期一（4月20日）起暂停销售三天。
据新华网、《长沙晚报》等报道，零食王国上个星期五（17日）在长沙芙蓉广场正式开业。它的营业空间达1.2万平方米，面积约相等于30个篮球场大小，以逾6500个全球品牌、逾3.5万款零食，现场获得“全球最大零食店”健力士世界纪录认证。
有别于传统零售陈列，零食王国打造沉浸式体验场景，店内设置10多个主题区域，十分吸睛。消费者入店后，首先穿过琳琅满目的“零食长廊”，可在由7.6万颗棒棒糖拼成的周杰伦《范特西》封面墙完成首站打卡。
据报道，店内的“泡面之城”汇集多达3500种口味。“全球零食站”则聚集近20个国家的可乐与薯片，除常规品类外，还有“盐可乐”等特色款式。
集团负责人指出，店内零食涵盖全球70个国家和地区，中国产品牌超5000家、产品2万余款；进口品牌超1500家、产品1万余款。若每天品尝一款，试遍全部零食需约96年。
不过，零食王国开业仅三天即公告，从星期一起暂停销售至星期三（22日）。
公告显示，由于现场激增的客流远超预期，在商品补货、收银效率、系统承载等各个方面承受着不小压力，导致现场体验不佳，因此决定对所有问题、卡点进行优化调整。停售期间，零食王国仍正常开放参观，消费者可继续参观主题区、拍照打卡。

北京曾主动靠近：2025年9月24日向托斯卡纳普拉托首席检察官卢卡·特斯卡罗利（Luca Tescaroli）发函，11月25日派高级代表团到访，提供了2024年未遂谋杀案主谋名单并提出合作。特斯卡罗利称此举“具有里程碑意义”，但后续受意大利不同执法机构抵制未能展开。国家反黑反恐检察院不赞成地方检察机关与中方警方直接通道；警方也持谨慎态度，称需相互试探，中方并非欧盟盟友可比。
此外，网络安全事件加剧戒心：意大利称1月起自中国境外的网络攻击曾试图入侵内政部，定位中国异议人士及侦查中方犯罪小组的办案人员，虽被挫败。中方否认并反对网络攻击。
目前通过欧盟机构的协作有所尝试，Eurojust称正与中意就至少一案合作。另有情形中，中方曾通过欧洲渠道请求派员赴佛罗伦萨并查阅反黑案卷，但遭拒，理由为案情受侦查保密保护。迄今最具体的双边直接协助仍集中在普拉托：中方提供了口译服务，填补了长期缺乏翻译的问题。特斯卡罗利表示，中方提供合作不可盲从，但也不应全盘拒绝这一机会。

伦敦消息（路透）— 英国通信监管机构Ofcom于2026年4月21日启动对即时通讯应用Telegram的调查，因有证据显示该平台上存在和传播儿童性虐待相关内容。
Ofcom表示，已收到加拿大儿童保护中心提供的证据，并进行了自身评估，因而决定查明Telegram是否未能履行防范非法内容的义务。此次调查是英国打击儿童在线受害无明确问责机制努力的一部分。英国2023年通过的《在线安全法》已对Facebook、YouTube和TikTok等社交平台提出更高标准，首相基尔·斯塔默亦要求平台承担更多责任，政府正就是否禁止16岁以下使用社交媒体进行磋商，斯塔默上周也会见了多家社交媒体高管。
Telegram对Ofcom的指控“予以断然否认”，称自2018年起已通过检测算法几乎消除平台上公开传播的儿童性虐待材料，并对该调查表示惊讶与关切，称担忧可能针对维护言论自由与隐私的在线平台展开更广泛攻击。该公司此前因在防止儿童虐待和暴力极端主义内容方面回应迟缓，曾于今年2月被澳大利亚监管机构罚款。
Ofcom同时宣布，对Teen Chat和Chat Avenue展开调查，认为在与这些公司沟通后，仍不满意其在保护英国儿童免受诱骗风险方面的做法。Ofcom执法总监Suzanne Cater警告称，这些公司若不加强保护，将面临《在线安全法》下的严厉后果。

教皇利奥十四于4月21日在前往赤道几内亚的班机上追思其前任方济各教皇，纪念其逝世一周年。利奥十四称，作为首位来自美洲的教宗，方济各用言行为14亿天主教徒“贡献良多”，并在12年任内促进了对人人平等的真实尊重，强调“我们都是兄弟姐妹”。
利奥十四在这次非洲四国行的最后一程中表示：“我们祈愿他已享受主的怜悯，并感谢主赐予教会及全世界方济各生命的伟大礼物。”
方济各（本名豪尔赫·马里奥·贝尔戈里奥）于2025年4月21日清晨因中风去世，享年88岁；此前曾患重症双肺炎并康复。任内他推动教会改革，常因被认为在某些教义问题上较为宽容而遭保守派批评。2016年他曾就美国建墙政策批评时任总统唐纳德·特朗普，后者回击称教宗“可耻”。
方济各以朴素著称，未入驻华丽教宗寓所而住在梵蒂冈酒店；他于2025年4月26日安葬，请求以镀锌木棺简单下葬，成为百余年来首位葬于梵蒂冈外的教宗，墓地在罗马圣母大殿，墓碑以拉丁文刻名“Franciscus”。圣母大殿将于4月21日举行纪念弥撒。路透社
