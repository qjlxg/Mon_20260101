斯德哥尔摩——斯德哥尔摩国际和平研究所（SIPRI）4月27日发布报告称，2025年全球军费开支接近2.9万亿美元，已连续第11年上升。美国、中国和俄罗斯三国合计开支约1.48万亿美元，占全球总额逾半。

以色列与伊朗支持的真主党互相指责违反脆弱停火。以色列总理内塔尼亚胡在内阁会上称，军方正“积极”打击真主党；真主党则表示将持续回应“违规行为”。以军表示，一名士兵在黎南部“战斗中”阵亡，另有六人受伤，其中四人伤势严重。
国家通讯社NNA报道，4月26日多地遭以军空袭，既包括官方发出撤离警告的区域，也有其他村庄。以军向南部七个城镇和村庄下达撤离令，随后传出卡夫尔·蒂布尼特被击中造成人员伤亡，扎乌塔尔·谢尔基耶一处清真寺及宗教建筑被摧毁。法新社记者在纳巴蒂耶·法乌卡等地目击浓烟升起。
以色列军方称针对“火箭发射细胞和武器库”发动打击，并早前对真主党在所谓“黄线”以北的军事目标实施炮击和空袭；事发后又拦截三架飞向以色列领土的无人机。以军此前宣布在沿边境大约10公里深的黎巴嫩地带设立“黄线”，并警告当地居民不要返回。
自冲突爆发以来，黎巴嫩卫生部统计死亡超过2500人、受伤逾7700人。联合国驻黎巴嫩维和部队（UNIFIL）在贝鲁特为一名于4月24日身亡的印尼维和人员举行悼念活动，初步联合国调查认定爆炸由以色列坦克炮弹引发。
