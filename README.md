特朗普政府多管齐下平息民怨 拉抬中期选举情势
（纽约／费城综合电）美国中期选举距今已不到四个月，特朗普政府正加紧通过“特朗普账户”儿童基金，以及施压业者降低汽油和食品等零售价格来减缓民众对生活费高涨的不满，拉抬选情以求保住共和党在参众两院的微弱优势。 
特朗普星期一（7月6日）在白宫亲自推介以自己命名的“特朗普账户”，目标是为10岁及以下孩童补贴资金，用于投资华尔街股票基金和政府项目。每个在特朗普第二任期出生的孩童都可获得1000美元（约1290新元），父母和雇主还可每年补贴高达5000美元。 
根据标普500指数年均10%的历史涨幅推算，这1000美元到了18岁可增长至6000美元，若每年都获最高5000美元的补贴，18岁时将可累积超过27万美元。受益人年满18岁将可提取存款用于高等教育、买房头期或投资小生意。
共和党人希望这个由“大而美法案”拨出税款资助的“特朗普账户”，将提供他们在11月3日中期选举的胜算，声称账户让每名孩童都能“从美国梦分得一杯羹”。 
不过，民主党人批评这个账户的推出，只是为了掩饰特朗普政府在同一项法案大幅削减了医疗保险和粮食补贴等社会安全措施，且富裕家庭更有能力为孩子填补基金，也意味着将来可能导致孩童的财富更加不均衡。
中期选举距今时日无多，多项最新民调都显示特朗普的支持度只有40%左右，美国民众尤其不认同他管理经济的表现。
若民主党在选举中翻盘取得国会多数优势，将可能迅速启动弹劾总统的程序。特朗普近期更加如火如荼，多番出手抑制让美国人最为头痛的汽油和日常用品价格上涨。
《华尔街日报》引述消息报道，美国农业部长罗林斯的幕僚长贝内特（Tate Bennett）上周在7月4日独立日假期前，曾致电主要零售集团包括沃尔玛、克罗格和艾伯森，要求他们降低牛肉价格。
据知情者称，沃尔玛当时向贝内特说，已计划在夏天降低多个产品的价格，包括牛肉。
农业部于是将沃尔玛的计划上报白宫，这让特朗普有机会向民众邀功，星期一在社媒贴文称，沃尔玛是在“我的政府要求下”决定降价。
特朗普也在社媒抱怨汽油价格调降太慢，要求业者应当将价格降至每加仑2.25至2.50美元，并已下令司法部介入调查石油公司是否故意抬高价格。
不过，由于美伊谈判局势不明， 专家并不排除油价下来又回弹至之前的高点。特朗普的多番努力能否扭转选情颓势，看来同样局势不明。

广西洪灾大量蛇出逃致人死伤 官方：全力防范蛇伤险情
中国广西首府南宁横州市的洪灾导致大量人工养殖的蛇出逃，已出现人员死伤情况。当地官方称正全力防范洪水蛇伤险情，已特邀医疗专家支援提升蛇伤重症病例处置能力。
据新华社报道，横州市云表镇一家养蛇场被洪水冲毁，导致场内养殖的眼镜蛇、大王蛇、水律蛇被冲入洪水中。大王蛇、水律蛇均为无毒蛇，但眼镜蛇毒性较强。据红星新闻报道，云表镇邓圩村委负责人吴志告称：“初步统计，从被冲垮养殖场出逃的蛇类约800条至900条。”
有现场救援人员称，救援期间多次接到群众遭蛇咬求助，疑似为眼镜蛇咬伤。《新京报》星期三（7月8日）上午从多方核实获悉，云表镇一名被蛇咬伤的女子不幸离世。参与救助的村民称，因道路不通，这名女子辗转多次才送至医院治疗，他看到时女子已无意识。接诊医院证实确有一名被蛇咬伤者不幸离世。
官方的南宁市融媒体中心随后在星期三下午通报，横州市正全力防范云表镇洪水蛇伤险情。
通报称，横州市已强化救治资源储备，紧急扩充医院蛇毒血清库存，目前血清储备量可满足救治需求。同时，特邀医疗专家驻点支援，全程指导蛇伤救治工作，提升重症病例处置能力。
根据通报，在一线防控救治环节，当地持续加密搜救巡查频次，在云表镇布设多处现场医疗点位，就近为民众提供应急诊疗服务。此外，当地也开通蛇咬伤急救绿色通道，精简急诊就诊、检查、救治环节，“实现蛇伤患者快速接诊、快速处置、快速救治，最大限度缩短救治时间、降低伤情风险”。
通报称，横州市也同步开展一线医务人员专项急救培训，聚焦蛇伤初步清创、伤口规范处理、伤情快速预判、院前应急处置等核心技能，提升基层医护人员蛇伤应急处置水平。
在隐患预警防范方面，通报介绍，卫健部门已联合云表镇乡镇政府开展风险预警工作，依托村屯广播、村民微信群、入户上门通知等方式，推送防蛇避险提示，提醒村民尽量减少夜间外出，远离田间杂草、沟渠、低洼积水等蛇类易出没区域，严格做好居家、户外双重防蛇防护。

        而在世界杯进球榜单上，梅西以21粒进球的成绩锁定历史射手王的位置。如今随着助攻纪录的解锁，梅西同时占据世界杯历史进球榜与世界杯历史助攻榜榜首，用无可争议的表现再次刷新了足坛的历史高度。
