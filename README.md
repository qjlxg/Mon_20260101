世卫：不安全食品每年致全球150万人死亡
世界卫生组织周四警告称，每年全球有150万人死于受有害细菌、病毒、寄生虫或化学物质污染的食物，其中幼儿受影响最大。
联合国卫生机构分析了2000年至2021年间194个国家的数据后发现，每年有8.86亿人因食用不安全食品而患病，其中五岁以下儿童的风险几乎是其他人群的三倍。
世卫组织总干事谭德塞说：“食品安全并非抽象的概念，它与每一餐、每一个家庭、每一天都息息相关。”
尽管自2000年以来，因食用危险食物而导致的疾病总体呈下降趋势，但区域间的巨大不平等依然存在。仅非洲和东南亚就占全球食源性疾病病例总数的近四分之三，以及全球60%的死亡病例。
2021年，全球约有8.6亿起食源性疾病，其中绝大多数是由细菌和病毒等生物危害造成的。但摄入化学物质也导致了不成比例的死亡，砷和铅中毒是主要的非生物致病因素。
除了对健康的影响外，研究估计，2021年食源性疾病造成的全球经济生产力损失高达6470亿美元（约8305亿新元）。
