澳大利亚蒙纳士大学的科学家与巴西圣保罗大学和澳洲迪肯大学合作进行的研究，对2100多名未患痴呆症的澳洲成年人进行了调查。
研究员在评估研究对象饮食习惯的同时，也对他们的认知能力进行测试，特别是注意力与信息处理速度。结果显示，饮食中的超加工食品比例增加，研究对象在注意力相关任务中的表现也会受影响。
研究报告的主要作者卡多索博士说，超加工食品摄入量若增加10%，约相当于每天多吃一包薯片，专注力表现就会出现下降。

这项研究并未直接把超加工食品与记忆力丧失或失智症联系起来，但研究确实发现，这些食物与导致失智症风险增加的疾病存在关联，包括肥胖和高血压。

美国军方称，4月24日在东太平洋击中一艘船只，造成两人死亡。美国南方司令部表示，遭击中的船只由其所谓的“指定恐怖组织”运营，但未透露具体身份；美方称无美军伤亡，并将遇难者描述为“男性贩毒恐怖分子”，未提供更多细节。
南方司令部在社交媒体发布了一段16秒视频片段，显示该船只在海上遭到打击；其称情报确认该船沿已知毒品走私航线并从事贩毒活动。
近几周来，美军在东太平洋已多次对类似船只发动致命打击。特朗普政府表示此类行动旨在打击所谓的“贩毒恐怖分子”，自去年9月以来相关打击已造成超过170人死亡。
美国及国际专家与人权组织质疑这些行动的合法性。人权观察和国际特赦称此类打击构成“非法的法外处决”，美国公民自由联盟则称政府对目标的指控“缺乏证据、散布恐惧”。（路透）
