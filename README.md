中国市监总局要求三大外卖平台尽快自查整改
中国市场监管总局召开外卖平台企业行政指导会，要求美团、淘宝闪购、京东三大平台加强食品安全措施，尽快开展自查整改。
据微信公众号“市说新语”消息，市场监管总局星期四（4月2日）在北京召开行政指导会，指导上述三家平台企业严格落实即将实施的《网络餐饮服务经营者落实食品安全主体责任监督管理规定》，强化企业食品安全主体责任，采取切实有效措施，保障消费者饮食安全。
会议指出，食品安全风险不容忽视。市监总局出台《规定》，明确外卖平台企业务必将落实食品安全主体责任摆在企业经营管理的核心位置，保证餐饮食品安全，让消费者放心消费、安全饮食。
会议要求，外卖平台企业必须严格执行法律法规要求，逐条对标对表《规定》，从落实资质审查责任、落实管理保障责任、落实社会公开责任、落实应急管理责任、落实协同监管责任等五个方面，全面落实平台食品安全主体责任。
会议强调，外卖平台企业要以《规定》正式实施的6月1日为时间节点，倒排工期，对照《规定》各项要求，梳理现有制度、流程、人员配备、技术保障等环节，尽快开展自查整改；严把审查关、管理关、配送关，强化技术赋能管理，主动融入政府监管体系，变“被动应付”为“主动对接”，通过推动“互联网+明厨亮灶”、鼓励外卖骑手参与食品安全监督等方式，共建食品安全社会共治新格局。

据新华社报道，何立峰星期五（4月3日）在北京与商鹏飞（François-Philippe Champagne）会面时说，中加应共同落实好两国领导人重要共识，深化经贸、金融等领域合作，推动中加经济关系稳定健康发展。双方也同意在今年下半年举行经济财金战略对话。
综合路透社与彭博社报道，商鹏飞会后指出，会谈以金融服务为重点，同时涵盖能源、猪肉贸易及公平劳动规范等议题。他强调，加拿大高度重视供应链完整性，双边贸易应遵循国际标准。
商鹏飞还提到，加中双方讨论了地缘政治紧张局势对油气市场的影响，以及中国如何看待加拿大作为稳定能源供应方，并称“我们正成为首选合作伙伴”。

中国是加拿大第二大贸易伙伴，双边贸易额约1200亿加元（）。加拿大计划到2030年将对华出口总额提升50%。今年1月，加拿大总理卡尼与中国国家主席习近平会面，渥太华正推动贸易多元化，以降低对美国的依赖。
尽管卡尼访华后，中国已暂停对部分加拿大农产品的关税，但对加拿大猪肉仍维持25%的进口关税。商鹏飞表示，他已在会谈中提出相关问题，并称双方需推动解决这些贸易摩擦，“此行已为此奠定基础”。
此外，商鹏飞亦提出扩大加拿大在华金融服务业务，认为这对实现2030年出口增长目标至关重要。他指出，拓展贸易离不开配套的金融服务，以支持企业深耕中国市场。
商鹏飞此次访华为期五天，除会晤何立峰外，还与中国财政部长蓝佛安及中国人民银行行长潘功胜举行会谈。

2026年4月3日，泰国监察专员会主席ทรงศัก สายเชื้อ（宋萨克·赛切）赴沙密沙叻（Samut Songkhram）与拉差汶里（Ratchaburi）两省，调查“香水椰子”价格暴跌及涉嫌掺假、外资垄断等投诉。
监察组会见了内贸署、商业发展署、竞争委员会、省代表及受影响农户。来自Amphawa区的农户反映，大椰子收购价仅3铢、小椰子0.5铢，而种植成本约7铢，导致无法购肥、保养，质量不达标被取消出口订单，情绪强烈质疑政府救助成效。
农民代表并对官方抽检存疑：食品药品监管局赴9家厂抽样未查出掺假，农户怀疑检验前已被通知、难获真实数据。卫生部门回应，抽检未发现漂白剂或氯超标，但发现部分厂用防腐剂仍属标准内；医学科研机构正拟定“100%椰子水”检验标准以供后续监管。
商业发展署则称，抽查显示部分厂商涉嫌“傀儡公司”与外资持股问题，已与中央侦查局合作追查并拟推法律与行政措施。农户呼吁政府尽快设立中央收购点、拓展除中国外的新出口市场（尤其欧盟），并协助取得有机认证以提升议价能力。

3日（泰历2569年）——民主党副党魁กรณ์·查蒂卡瓦尼ช向新任财政部长兼油价成本审查委员会主席เอกนิติ致意，并呼吁重新检视成品油定价的成本结构，指出过去一个月民众单方面承受了过高油价。
กรณ์提出须公开且透明查证的7项要点：一、炼油厂实际购入原油成本为多少；二、市场价上涨后炼厂实际购油价与用于计算加工费的参考价差异；三、关联交易商（trader）对炼厂的买卖是否存在转售利润；四、所谓从新加坡计入的“假想”运输费数额；五、炼油的真实加工成本；六、本月异常加工费产生的炼厂利润与正常期比较；七、每日进出油库的库存数据。
若查证发现炼厂获利异常，กรณ์建议两项主要对策：一、修订加工费计算公式，使之反映合理成本；二、征收“风fall税”（Windfall Tax），追回超额利润以直接冲减油基金债务。同时他主张仿效他国，将燃油特别消费税下调6铢并显著下调增值税，且追溯三个月以缓解民众负担。此类措施参考国别包括已降税的印尼、澳大利亚；采用风fall税的印度与英国；以及正在审议的菲律宾、韩国与欧盟国家。
กรณ์强调，全面公开相关数据是确保此次危机能被透明、可检验地解决的关键。

“ถอดรหัสกติกาประชามติ 69”座谈会：修订规则以便利公民投票
4月3日13:00，泰国法政大学（Thammasat University）法学院在塔帕差尼中心211室举办题为“ถอดรหัสกติกาประชามติ เพื่อแก้ไขการลงประชามติในครั้งต่อไป”的座谈会。与会者包括国会议员 ภัณฑิล น่วมเจิม（พรรคประชาชน）、Dr. อุษณีย์ ปฐพีศรีกิจ、iLaw 活动官 เกวลิน ถนอมทอง、以及法学院副教授 รศ. ปริญญา เทวานฤมิตรกุล 等。
会议由เกวลิน列举了影响公投投票的八大问题：一是仅允许在“外县”（นอกเขตจังหวัด）投票，不能在同一省内跨区投票；二是公投媒体宣传少于选举；三是示范选票印刷遗漏导致废票；四是选举委员会（กกต.）禁止政党引导，但无相关法律依据；五是现场官员有引导选民投“不赞成”的情况；六是误导选民在“赞成”栏打勾；七是部分地区存在买票诱导投“不赞成”；八是假消息鼓动投“不赞成”。
Dr. อุษณีย์提到曾尝试网上登记未果并截图，但法院以其未尽充分努力驳回延长登记请求，引发社会呼吁选举委员会行使权限延长登记期。iLaw执行董事นายยิ่งชีพ指出，选举委员会提供的不利信息可能误导选民，未来公投争议将更多。รศ. ปริญญา强调，提前投票登记与异地公投登记时间不对等（前者17天，后者仅3天），选举委员会可依规放宽或延长登记，他主张开通邮寄或线上投票，并呼吁公民社会与政党共同修法，优先考虑公民便利，只修必要条文。
