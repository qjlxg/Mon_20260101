据路透社报道，威斯康星州民主党参议员鲍德温（Tammy Baldwin）、亚利桑那州民主党参议员凯利（Mark Kelly），以及南卡罗来纳州共和党参议员斯科特（Tim Scott）和印第安纳州共和党参议员托德·扬（Todd Young）星期一（5月11日）致函特朗普称，中国数十年来试图“摧毁美国造船业”，美国必须充分运用贸易措施加以应对。
特朗普与习近平去年10月在韩国会晤时，同意暂停对彼此船只征收报复性费用一年。此举预计可让驶往美国港口的大型中国建造船舶，每年减少约32亿美元（40.62亿新元）的相关费用。除非双方进一步同意延长暂停措施，否则美国将于11月10日恢复征收这些费用。
美国在去年4月首次宣布征收港口费用，旨在削弱中国对全球海运产业的主导地位。此前，美国一项调查认定，中国在海运、物流和造船领域的主导地位，是通过不公平做法形成的。
特朗普将于星期四（5月14日）至星期五（15日）在中国与习近平举行峰会。伊朗战争预计将成为此次峰会的核心议题，而这场战争也进一步加剧美中紧张关系。尽管特朗普政府持续施压，中国仍是伊朗石油的最大买家。
据路透社报道，这些参议员在信中写道：“美国正处于一个关键转折点，不能再向中华人民共和国进一步让步。”
信中也提到：“我们敦促你（特朗普）在这些谈判中保持强硬立场，与国会共同推进贸易补救措施，并推动《美国船舶法案》（SHIPS for America Act），以创造更公平的竞争环境。”
这项法案去年已分别提交美国参议院和众议院审议。法案内容包括为投资美国本土造船厂及相关生产提供税收抵免，并授权在未来十年拨款25亿美元，用于支持国内造船项目等措施。
