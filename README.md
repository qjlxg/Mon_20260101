中国湖南一家医院的院长儿子被曝涉嫌吃空饷。医院所属的湖南中医药大学展开调查后，要求院长儿子退还重复领取的绩效奖金。
贵州广播电视台公共频道“百姓关注”报道，湖南中医药大学第一附属医院一名陈姓女职工星期二（4月28日）公开举报指院长朱镇华的儿子长期吃空饷，涉及金额可能超过百万元人民币。
院纪委此前调查认定院长儿子“重复拿钱”2万元（人民币，下同，3746新元）并已退款，院长则否认吃空饷说法。陈姓职工对调查结果不予认可，呼吁提级调查。
陈姓职工说，院长儿子2019年7月本科毕业于湖南中医药大学，同年9月入职该校直属附属医院，后被安排至健康管理科，“但实际从未在该科室上班”。她指院长儿子从未在健康管理科履职，实际就职于耳鼻喉科、重症医学科，却多次领取健康管理科内部发放的福利。
报道称，在向媒体举报之前，陈女士已多次向医院举报。
另据新华社报道，湖南中医药大学已经介入调查。初步调查发现，院长儿子2019年7月毕业于湖南中医药大学，之后进入湖南中医药大学第一附属医院健康管理科工作，2022年1月被借调到医院重症医学科工作。
借调期间，医院健康管理科给院长儿子重复发放绩效奖金共计8万4000元，湖南中医药大学已经要求院长儿子退还重复领取的8万4000元绩效奖金。湖南中医药大学也要求医院进一步规范内部管理，堵住管理漏洞。
湖南中医药大学官网资料显示，湖南中医药大学第一附属医院创建于1963年，1994年成为湖南省首家三级甲等中医医院、获评全国省级示范中医医院，2008年入选国家中医临床研究基地，2021-2023年连续三年在国家三级公立中医医院绩效监测中获A+等级。
湖南中医药大学是湖南省重点建设本科院校，全国首批设立国家级重点学科的中医药院校。

新加坡 — 美属马里兰大学与全球森林观察（Global Forest Watch）发布的年度卫星分析显示，2025年全球热带雨林损失面积下降36%，至430万公顷，低于2024年创纪录的670万公顷。报告由华盛顿世界资源研究所与马里兰大学联合编制。
研究指出，2025年平均每分钟约有相当于11个足球场的雨林消失。巴西原始林损失下降42%，至160万公顷，部分归因于更强的环境政策；印尼原始林损失上升14%，接近30万公顷，主要由采矿、种植园、洪水及政府的粮食与能源基地计划推动；马来西亚下降5%，约6.5万公顷。
原始森林对固碳与水源调节至关重要。研究者警告，2025年损失减少反映的是在2024年极端火灾后的短暂缓和，未来仍受厄尔尼诺带来更高气温和干旱引发火灾的威胁。全球森林观察联合主任伊丽莎白·戈德曼表示，“火灾与气候变化相互影响，若2026年出现厄尔尼诺，预防与应对投入将至关重要。”
全球总树冠覆盖损失亦下降14%，至2550万公顷，但火灾仍占42%。加拿大2025年620万公顷的树冠损失中有85%由火灾造成，使其成为仅次于2023年的第二严重火灾季。欧洲南部夏季高温与干旱导致法国创下历史上因火灾的最大树冠损失；俄国则创十年最低。
报告强调，热带地区因农业、棕榈油、木浆等需求仍是森林砍伐主战场。过去50年巴西亚马逊约失去20%森林，印尼逾40%。雅加达智库Auriga Nusantara指出，印尼2025年森林损失激增66%，为八年新高，近五分之一与政府的粮食与能源基地项目有关，且未来可能在南巴布亚形成面积达200万公顷的大型粮食种植区。
全球森林观察警示，签署了2030年停止并扭转森林流失承诺的140多个国家仍未接近目标，“到2025年我们仍比达标线高出70%，在气候风险与对食物、燃料和原料需求持续增长的背景下，实现目标难度很大。”

新加坡讯（美國馬里蘭大學／全球森林观察报告）——在经历了2024年创纪录的热带火灾后，2025年全球热带雨林净损失明显回落。研究显示，2025年热带雨林损失降至430万公顷，较2024年的670万公顷减少36％。研究由马里兰大学基于卫星树冠覆盖数据编制，报告由世界资源研究所与全球森林观察发布。
报告指出，2025年平均每分钟相当于约11个足球场的雨林消失。数据分析把树冠变化归因于人为砍伐、火灾或洪水等自然扰动。
巴西原始森林损失约160万公顷，较上年下降42％，部分归功于更严格的环保政策。印尼原始森林损失上升14％，近30万公顷，主要由采矿、种植园、洪灾及政府的粮食与能源地项目推动；马来西亚下降5％至约6.5万公顷。
研究者警告，2025年下降更多反映的是2024年极端火灾后的缓和期，而非长期趋势。科学家预计厄尔尼诺可能在未来数月形成，通常会推高气温并在东南亚及南美引发干旱与火灾。全球森林观察联合主任伊丽莎白·戈德曼（Elizabeth Goldman）表示，“火灾与气候变化相互助长，随着2026年可能出现厄尔尼诺，对预防与应急的投资至关重要”。
就全球总体树冠损失而言，2025年下降14％至2550万公顷，但火灾仍占42％。在加拿大，2025年620万公顷的树冠损失中有85％因火灾，成为该国史上第二严重的火灾季。南欧因夏季高温与干旱发生大面积野火，法国因火灾的树冠损失创纪录；俄罗斯则为十年内最低。
热带地区长期受农业扩张（大豆、玉米、牛肉、棕榈油）、木材与纸浆需求驱动。尽管多国企业承诺遏制并逆转森林损失，亚马孙50年已损失约20％，印尼超过40％。雅加达智库Auriga Nusantara称，印尼2025年总体森林损失激增66％，近八年新高，近五分之一与政府粮食与能源地项目有关，南巴布亚的大规模稻米与甘蔗基地或将占地数百万公顷。
报告提醒，2021年逾140国签署的到2030年停止并扭转森林损失承诺距离目标尚远，“到2025年，我们仍比达标水平高出约70％，在需求不断增长和气候威胁加剧的情况下，要实现目标将非常艰难。”
