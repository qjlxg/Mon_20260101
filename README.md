新华社报道，克普勒公司（Kpler）说，这一数据包括商业和非商业船舶，商业船舶数量略高于非商业船舶。“美国与伊朗军事对抗再度升级，削弱了市场对外交努力能够在近期带来局势稳定的信心。”
美国和伊朗达成谅解备忘录后，霍尔木兹海峡主要有两条通行航道，分别为伊朗管控的北部航道和靠近阿曼一侧的南部航道，美军在南部航道提供通航协助。
克普勒公司数据显示，在阿曼附近海域近期发生袭船事件影响下，航运运营商继续倾向选择伊朗一侧航道，阿曼一侧航道9日仅有一艘船舶通过。
另据《纽约时报》10日报道，三名要求匿名的美国高级官员当天说，霍尔木兹海峡将很快全面恢复通航，并指伊朗谈判代表将近期袭船事件归咎于擅自行动的军事单位。
他们预计伊朗将在未来几天发表公开声明，确认霍尔木兹海峡所有航道保持开放，伊朗军队将停止向通过这条狭窄水道的船只开火。他们还说，如果伊朗不发表这项声明并予以遵守，“就不会有好下场”。
7月7日和8日，美军以回应伊朗袭击霍尔木兹海峡的三艘商船为由，连续两天对伊朗发动军事打击。作为回应，伊朗军方对中东地区多个美军目标发动打击。
美国总统特朗普10日说，与伊朗达成的14项条款停火协议已经失效，但应伊朗的要求，美国同意继续进行谈判。
