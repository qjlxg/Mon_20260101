张文宏反对把AI引入医院病历系统
中国著名医学专家、复旦大学教授张文宏说，他拒绝把人工智能（AI）引入医院病历系统，因为医生需要训练专业诊断能力。
据澎湃新闻报道，上星期六（1月10日），张文宏在香港高山书院与全球科学组织联合主办的高山论坛上发言。
张文宏说，他自己也用AI，因为接触的病人数量太大、太复杂，并且短时间内要处理很多病例。但他强调，自己能看出AI哪里是错的，不会被它误导。能否作出正确判断，取决于医生自身能力有没有强过AI。
张文宏提到，他拒绝把AI引入医院病历系统中，因为这会影响对医生专业诊断能力的训练。医生没有经过系统训练，就没有能力去判断AI给出的诊断结果是否有误。
张文宏强调，年轻医生必须掌握两大关键能力：一是判断AI诊断的准确性，二是诊治AI无法应对的复杂疑难病症。他认为，这是技术便捷背后不可或缺的专业根基。
