传国共论坛延至2月初举办 转型为智库交流
台湾媒体报道，原拟于1月底重启的国共论坛将延期至2月初举行，且论坛定位不再为政治层面，而是聚焦人工智能（AI）等非政治领域，并被定调为“智库交流”，论坛名称也可能随之调整。
综合台湾《自由时报》和ETtoday新闻云报道，国民党副主席萧旭岑原计划于1月27日至29日赴北京，重启停摆九年的国共论坛，并为国民党主席郑丽文3月访问中国大陆铺路。
此前，郑丽文与前总统马英九会面时说，国民党必须负起历史重责大任，让两岸融冰，因此希望积极推动两岸交流与对话，并将这次交流定调为“智库交流”。她还说，在相关细节确定后，将第一时间召开发布会说明。
国民党内人士指出，作为郑丽文上任党主席后首场国共恢复交流的活动，上述论坛已明确定位为“智库交流”，因此名称可能不再沿用“国共论坛”，议题也设定在AI、防灾及低碳永续产业发展等非政治领域，以降低敏感性。
相关人士还透露，交流活动已确定延期，不会在1月底举行，拟改至2月初，由萧旭岑率团赴陆，成员以智库人士为主，涵盖产业界、学界以及马英九政府时期官员约30人，连同随行人员在内，共约40人。
报道称，由于郑丽文此次不会出席交流活动，按照惯例，代表中国大陆方面出面会谈的高层人士可能为全国政协主席王沪宁。

【民调显示多数美国人不满ICE执法 驱逐移民政策引发严重分歧】金十数据1月24日讯，据美国国会山报报道，一项最新民调显示，大多数受访者对美国移民及海关执法局（ICE）的工作表现持否定态度。《纽约时报》与锡耶纳学院周五发布的一项民调发现，63%的登记选民不赞成ICE处理工作的方式，仅36%表示认可。其中，表示“强烈反对”的受访者占55%，而“强烈支持”的仅为24%。此次民调是在明尼阿波利斯发生ICE执法人员枪杀37岁的Renee Good后进行的，该事件引发了该市数周的动荡。作为特朗普政府激进移民镇压行动的一部分，美国国土安全部（DHS）向明尼苏达州派遣了移民官员。然而，当地官员表示，公民及其他合法居留者在这些行动中也遭到了不公平的对待，抗议者与联邦官员之间的紧张关系依然高度紧张。。

【美国邀意大利加入加沙稳定部队 梅洛尼权衡外交平衡策略】金十数据1月24日讯，据知情人士透露，美国已邀请意大利作为创始成员国加入“加沙国际稳定部队”（ISF），此举正值特朗普政府试图增强该倡议的可信度之际。知情人士表示，外交官本周已向总理梅洛尼的办公室及意大利外交部发出了这一提议。他们补充称，是否加入的决定权目前掌握在梅洛尼手中，目前尚未做出最终定夺。根据该提议，意大利无需向ISF派遣军队。相反，意大利此前关于培训加沙未来警察部队的承诺将足以满足条件，其主要贡献将体现在其对阿拉伯国家、以色列和巴勒斯坦的政治影响力上。知情人士称，意大利方面对于是否以及如何加入ISF存有疑虑，尽管其参与加沙整体和平努力的政治意愿是明确的。梅洛尼此前辩称，未签署“和平委员会”章程是因为其与意大利宪法相冲突，但她强调自己对章程的修改持开放态度。
