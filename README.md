英国将群岛主权移交毛里求斯 特朗普：软弱行为

路透社报道，特朗普星期二（1月20日）在社媒发文说：“令人震惊的是，我们这个‘聪明’的北约盟友——英国，目前正计划把迪戈加西亚岛（Diego Garcia），一个关键的美军基地所在地，无缘无故地交给毛里求斯。”
特朗普说，毫无疑问，“中国和俄罗斯已经注意到英国的这一行为”。
不过，美国政府曾支持这项协议，并在协议于去年签署时表示“欢迎”，同时赞扬英国在确保联合军事基地长期运作的努力。
迪戈加西亚岛位于印度洋，是查戈斯群岛（Chagos Islands）最大的岛屿，设有一座战略重要的美英空军基地。英国与毛里求斯去年达成协议，将整个查戈斯群岛的主权移交给毛里求斯，但允许英国通过长期租约继续控制这个空军基地。
英国首相首席秘书、政府间关系事务部长高级部长琼斯星期二说，这项协议是保障军事基地持续运作的最佳方案，可确保基地在未来100年继续运行。
琼斯在接受泰晤士广播电台（Times Radio）采访时说：“条约已经签署，现在无法更改。”
一名英国政府发言人也发声明说，英国绝不会在国家安全问题上妥协。“我们采取行动是因为法院的裁决削弱了我们的立场，并可能阻止迪戈加西亚岛的军事基地未来按预期运作。”
发言人也说，英国与毛里求斯的这项协议确保美英联合基地在迪戈加西亚岛的运作能够持续，并制定强有力的条款，以保持独特的作战能力，同时阻止敌对势力的干扰。　
发言人强调，这份协议“已得到美国、澳大利亚和所有其他‘五眼联盟’的公开欢迎”。“五眼联盟”指的是二战后由美国、英国、加拿大、澳大利亚和新西兰组成的情报网络。
毛里求斯是英国的前殖民地，于1968年获得独立，一直在争取查戈斯群岛的主权。在日益增长的国际压力下，英国于2024年10月同意将群岛主权移交给毛里求斯，并于去年签署协议。
去年5月，英国同意每年向毛里求斯支付1亿零100万英镑（约1亿7500万新元），预计在协议有效期内总计将支付34亿英镑，以确保迪戈加西亚军事基地的未来，租期为99年。

“斩杀线”（Kill Line）原是电玩游戏术语，指角色的生命值跌至临界点，极易被对手一击毙命。这个词近期在社交媒体走红，被用来形容部分美国人的生活现状：即便收入看似不俗，但扣除房租房贷、保险等开支后，银行户头所剩无几。一旦发生突发状况，就可能迅速陷入贫困，甚至面临生存危机。 
在网上带动“斩杀线”风潮的，是一名网名“牢A”的“00后”中国留学生。他在西雅图修读生物医学，为了赚取生活费，经教授推荐，到当地警署担任法医助理，主要负责处理死于街头流浪汉的遗体。“牢A”见到了许多曾是中产阶级的流浪者，他通过视频揭示了美国社会存在的各种问题。

台湾在美台经贸协议实施细节上遭遇反弹，反对党质疑所谓关税让步背后真实代价。
内阁周二公布更多协议内容，确认台方可取得15%互惠关税率，低于先前20%的提议，也远低于美方今年4月曾提出的32%。该安排通过1月15日与美国商务部签署的投资合作备忘录（MOU）落实，未来数周还将与美方贸易代表办公室签署互惠贸易协定。

该型推动解放军登陆战术现代化，具备“空中突击”（直升机投送特战与速袭）、视距外气垫艇登陆（运送坦克与重装备）和正面两栖装甲登岸三种协同方式。但分析认为在高强度冲突中，075及更大舰艇易成为反舰导弹重点打击目标，生存性受限。
中国同时推进076型研发（被指可能作为无人机母舰），并强调075在撤侨、人道救援、海上安全与海军外交等非作战任务中的作用。075与已有的071型两栖舰配合，显著提升长程投送与远海持续能力，但在台海情景中的实际效能仍存在争议。

军事刊物指出，中国最新国产航母“福建号”存在严重设计缺陷，显著降低舰载机弹射效率；未来要想从根本上解决这些问题，可能只能转向核动力。 
福建号是中国首艘自主设计的航母，去年11月入列，排水量逾8万吨，是目前世界上最大的常规动力军舰，也是首艘装备先进电磁弹射器的非美舰艇。 
该刊分析称，受常规动力与舰体能量供应限制，当前电磁弹射器难以实现高频率、持续的出动节奏，影响舰载机战斗持续力与整体作战效率。媒体与专家认为，若要显著提升弹射效率、扩大作战半径并延长持续作战能力，未来的新型航母在设计上可能需优先采用核动力平台。
