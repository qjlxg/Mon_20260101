俄媒：朝鲜劳工“身价暴涨” 俄地方政府因工资过高放弃引进
俄乌战争持续导致俄罗斯劳动力短缺加剧，朝鲜劳工在当地已不再是“廉价劳动力”。随着薪资水平大幅上涨，部分俄罗斯地方政府和企业被迫放弃引进朝鲜劳工。 
俄独立媒体《梅杜扎》（Meduza）星期一（7月13日）报道，俄罗斯南部奥伦堡市原计划引进朝鲜劳工，从事市政清洁及维护等公共服务工作，但双方因薪资分歧最终未能达成协议。
奥伦堡市长尤马迪洛夫说，市政府去年曾提出每名工人月薪5万5000卢布（约900新元）的条件，但朝方没有接受。“据我了解，朝鲜工人不会为了5万5000卢布来工作，他们现在要求的工资是这一水平的两三倍。”
按此推算，朝方期望的月薪约为11万至16万5000卢布（约1800至2700新元）。
尽管如此，尤马迪洛夫仍高度评价朝鲜劳工，称他们“像机器人一样工作”，纪律严明、效率极高，但以当地财政预算难以承担如此高的工资，最终未能签约。
他还说，俄部分地区已成功引进朝鲜劳工，但奥伦堡州未被列入合作名单。最终，当地改为引进来自西非塞内加尔的劳工，目前已有31人在公共服务部门工作。
韩国统一研究院今年发布的报告指出，根据俄企业者等受访时透露，俄方为朝鲜劳工支付的人均人工成本通常约为每月1000美元（约1290新元）。
分析指出，尤马迪洛夫所说的金额并非单纯工资，而是引进一名朝鲜劳工的整体成本。截至2025年底，包括机票、签证、保险及规避联合国对朝制裁所需的中介费用在内，单名劳工的用工成本已超过2300美元。
专家也对俄地方政府披露的薪资数字持保留态度。朝鲜经济专家世宗研究所沃德（Peter Ward）认为， 普通朝鲜劳工的平均工资绝不可能达到1400美元，更接近1400韩元（约1.21新元）。”
他认为，朝方往往不是根据劳工实际价值定价，而是依据对方的支付能力来开价。
研究俄罗斯经济的东西大学教授蒙代（Chris Monday）则认为，朝鲜政府及中介机构可能从海外劳工工资中大幅抽成，也不排除朝方利用俄劳动力严重短缺的机会，提高对俄方的报价。他形容，这更像是朝鲜中介商借机向俄罗斯抬价。

中国6月新增贷款低于预期 信贷需求疲弱
受经济增长放缓、房地产市场长期低迷以及企业投资意愿不足影响，中国6月银行新增贷款低于市场预期，显示信贷需求仍偏弱。
路透社根据中国央行星期三（7月15日）公布的数据测算，中国银行业6月新增人民币贷款1.61万亿元（人民币，下同，3100亿新元），较5月的5200亿元大幅上升，但低于市场预期的2万亿元，也不及去年同期的2.24万亿元。
由于中国央行不再公布单月贷款数据，路透社根据央行公布的上半年累计数据与前五个月数据推算出6月份数据。
数据显示，今年上半年新增人民币贷款总额为10.72万亿元，低于去年同期的12.92万亿元，反映中国信贷增长持续放缓。
对于信贷增速下滑，中国央行货币政策司司长谢广奇在新闻发布会上说，仅看贷款数据已难以全面反映融资状况，市场还应关注债券融资及融资结构等指标。
这一说法与中国央行行长潘功胜上月的表态一致。潘功胜当时指出，信贷增速放缓是中国经济结构转型的一部分，融资渠道正趋于多元化。
路透社此前报道称，由于贷款增长持续疲弱，中国央行近月已通过“窗口指导”等方式，敦促银行加大信贷投放力度。
