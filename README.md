随着中国对日本实施的出口管制日益升级，日本驻华大使馆发文提醒在华日企警惕中国的管制措施。
日本驻华大使馆星期二（7月7日）在官网刊载的“注意事项”公告指出，中国商务部在今年1月、2月和6月相继宣布对日出口的管制措施。今年7月，中国还针对涉嫌非法出口两用物项（军民两用品）的行为设立举报渠道，并鼓励对典型案例进行举报，显示中国当局正持续强化相关的审查与执法力度。
使馆提醒，若违反中国出口管制相关法令，涉事者不仅会被处以罚款，还可能面临刑事处罚。使馆透露，尽管日本政府已就相关管制措施向中国提出强烈抗议并要求撤回，但实际上，近期已发生日本公民因涉嫌触犯“走私国家禁止进出口货物罪”而被中国当局拘留的案件。
使馆呼吁在华日企在从中国向日本出口两用物项时务必注意；若遭遇中国当局的任何质疑或调查，应及时联系使馆的企业支援窗口咨询。这份公告仅以日文发布，使馆中文版网页并未显示相关内容。
今年6月，有两名日本富士电机集团的员工因涉嫌内置稀土磁体的马达等出口违规问题被中国当局拘押。
日本共同社分析，随着中国的出口管制升级，中国限制关键矿产出口所引发的供应短缺，已开始波及日本整体经济，正迫使日本政府和企业加速寻求替代方案。
日本经济团体联合会（经团联）会长筒井义信在星期一（7月6日）的例行记者会上，针对中国对日本实体追加实施两用物项出口管制一事直言：“这极其令人遗憾，我们要求撤回。”
他指出，自今年1月以来不断强化的一系列管控措施，负面影响已扩大至经团联的会员企业。
随着中国商务部6月29日宣布将包括三菱电机子公司在内的20家企业和团体列入管控名单，受影响的日本实体总数已增至40家。对此，筒井充满危机感地形容：“（管制的）水位正在一点点上升。”
筒井同时提及由经团联、日中经济协会以及日本商工会议所三大经济团体负责人计划率团访华一事。他坦言目前“具体安排未定”。据悉，自原定于今年1月的访华行程被推迟以来，相关访问的重启至今尚无眉目。
