整治殡葬腐败乱象 中国严禁医疗卫生机构开展或外包殡仪服务
中国着手整治殡葬领域腐败乱象一年多后出台新规，规范医疗机构对亡故患者的服务流程，包括严禁提供殡仪服务，并严厉打击倒卖死亡证明、泄露逝者家属信息等行为。
中国国家卫生健康委员会星期一（1月12日）在官网公布，联合五部门发布《医疗卫生机构亡故患者全流程服务管理规定》，在多个行为上划下“红线”，包括明确医疗卫生机构不得开展殡仪服务。
规定严禁在医疗机构内陈列、展示、售卖殡仪服务用品；严禁外包或引入第三方运营太平间；严禁接收和存放院外来源的遗体；严禁使用院前急救车辆、非急救医疗转运车辆转运遗体等。亡故患者遗体在医疗机构临时停放时间原则上不得超过24小时。
规定也禁止医疗卫生机构代替丧属，与殡仪馆等联系殡仪服务事宜，也不得推荐、诱导、强制丧属到指定殡葬服务机构接受服务。
规定要求医疗机构完善患者及家属个人信息保护技术手段，防范并及时制止社会人员“蹲点”“扫楼”。
对于死因明确的正常死亡，医疗机构需在死亡发生或家属申报后一日内签发死亡证明。规定明确，严格防范、惩处违规开具、倒卖《死亡证明》的行为，并积极推动《死亡证明》电子化管理。
中国国家卫健委星期一在网发布政策解读称，全国自2024年10月发起整治殡葬领域腐败乱象专项行动，加强行业监管。上述规定的出台，是为进一步巩固专项行动成果，形成制度化、长效化政策措施。
中国在2024年8月曝出大规模尸体盗卖案，4000多具尸体直接在火化场内被剔除尸肉，取骨后交给山西奥瑞生物材料公司，用以制造骨材卖给各地医疗机构。案情引发舆论震惊，不过相关话题很快被官方噤声。
两个月后，中国官方开启专项行动，整治殡葬领域腐败乱象。

韩国成立军警特别工作组 调查无人机事件
韩国星期一（1月12日）成立军警联合调查特别工作组，以查明日前发生的无人机事件真相。
韩联社报道，韩国警察厅国家调查本部12日向媒体通报说，针对此次无人机事件，已组建一个规模约30人的特别工作组，并展开调查。特别工作组由警方约20人和军方约10人组成，安保调查局局长任组长。
国家调查本部说，特别工作组将迅速展开调查，明确查清事件真相。
同一天，韩国统一部一名高官就朝鲜劳动党中央委员会副部长金与正涉“韩国无人机侵朝”谈话强调，急需修复韩朝间的通信联络渠道。
朝鲜人民军总参谋部1月10日通过朝中社发声明称，军方1月4日在仁川江华郡松海面上空发现一个朝北飞行的空中目标，动用电子战装备进行干扰后将它击落。朝方说，被击落的是一架无人机，搭载着监视装置，内存约七分钟的影像资料。
韩国国防部随即回应称，经核查，相关日期并无军方操作无人机的情况，韩方无意对朝进行挑衅或刺激。次日，金与正发表谈话主张，韩国无人机侵犯朝鲜领空，并要求韩方对此事件作出具体说明。
韩国总统李在明10日指示组建军方和警方联合调查组，迅速对可能有民间无人机“侵入朝鲜”一事展开调查，并表示若此事属实，将是威胁朝鲜半岛和平与国家安全的重大犯罪。

香港立法会“二把手”指的是内务委员会主席，“班长”则是建制派召集人，在立法会担任协调人角色。上届“班长”是弃选本届立法会的廖长江和本届立法会主席李慧琼。

据了解，陈振英将成为内务委员会主席，出任立法会“二把手”。由于立法会主席李慧琼要每隔两个月到北京出席人大常委会会议，每年约有四次与立法会大会重叠，预料陈振英将暂代主持会议。经民联党魁吴永嘉则将出任财务委员会主席。
在挑选建制派召集人方面，其中一个考量是希望以旧带新，预料陈振英将获选出任“班长”。除他之外，同样获得中央信任的港区人大代表吴秋北，也将受委建制派召集人，与陈振英一起负责收集意见和传达信息。
