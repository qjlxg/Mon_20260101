研究：中国改碳排放算法 压缩近五年一半排放增幅
一项研究发现，中国可能改变了碳排放的计算方式，使得其从2020年至2025年期间原本报告的排放增长被“抹去”了一半。
路透社报道，独立研究机构能源与清洁空气研究中心（CREA） 发布的报告称，中国在“十五五”（2026年至2030年）规划中公布的碳强度数据，若换算为绝对碳排放，显示2020年至2025年间排放增长约为7%。
而此前基于碳强度年度数据的推算则显示，上述期间排放将增长约14%。
分析指出，这说明中国的二氧化碳排放量每年被下调约7亿吨，相当于德国或韩国的年度排放量。
这一调整意味着，即使绝对碳排放量继续上升，中国仍可能实现其2030年气候承诺目标。
中国国家发展和改革委员会以及生态环境部未立即回应路透社的置评请求。
碳强度是中国长期用于衡量气候目标进展的重要指标，指每单位经济产出（如GDP）所对应的二氧化碳排放量。
中国未公开详细说明其如何计算碳强度，但CREA研究人员利用国内生产总值（GDP）数据以及基于化石燃料使用的排放估算，复现了相关数据。
模型显示，从“十五五”规划开始，其算法可能开始剔除化石燃料的非能源用途，如用于化工生产的石油和煤炭，而这类用途近年来有所增加。
研究人员称，新算法同时开始纳入工业过程排放。水泥行业是工业过程排放的重要来源之一，但由于房地产行业疲软，中国水泥产量近年来有所下降。
而有数据表明，化工行业的排放可能存在低估等统计缺口。
这些变化综合起来，会使排放数据在统计上显得偏低。
报告称，碳强度定义的变化使中国气候目标的约束力有所减弱，并增加了追踪中国减排进展的不确定性。
中国承诺在2030年前，单位国内生产总值（GDP）二氧化碳排放比2005年下降65% 以上。

欧盟正式批准与美国贸易协议 移交议会表决
欧盟27个成员国正式批准与欧洲议会达成的一项妥协方案，以落实与美国的贸易协议。该协议目前仅需欧洲议会最终批准即可生效。
法新社报道，星期三（5月27日）在布鲁塞尔举行的外交官会议正式批准这个方案。相关立法文本预计将提交6月中旬举行的欧洲议会全会进行最终表决，随后由欧盟理事会批准。
欧盟与美国于去年7月达成贸易协议，核心内容包括欧盟暂停对所有美国工业品的关税，并对大量进入欧盟市场的美国农食产品建立关税配额制度，以换取美国对大多数欧盟输美商品征收15%的关税。
由于美国索要丹麦自治领地格陵兰岛引发欧美关系紧张，加上美国最高法院裁定特朗普政府大规模关税政策违法，欧盟对欧美贸易协议的批准经历数月延迟。
不过，上周欧盟各国和议员们达成一项妥协方案，使欧盟有望在7月4日特朗普设定的最后期限前落实该协议，否则将面临惩罚性的新关税。
这项妥协方案为取消欧盟关税附加了保障措施，以防美国违背承诺。但这些保障措施较最初的要求有所缩减，以避免与白宫再次产生紧张关系。
协议中一项所谓的“日落条款”被显著推迟至2029年底，即特朗普任期结束后。
协议文本也给予美国在年底前取消对钢铁零部件征收的15%以上额外关税的宽限期，而非像议会希望的那样将这个作为先决条件。
一项所谓的“日出条款”也被删除。这项条款规定，只有在美国履行承诺后，欧盟方面的协议才会生效。

专家：家门之内藏隐患 防假莫忘修信任
如果防范了大半天的虚假信息，竟然九成都来自“自己家里”，那真正需要修补的究竟是信息漏洞，还是信任裂痕？
意大利驻新加坡大使布兰迪（Dante Brandi）星期三（5月27日）出席意大利驻新大使馆与新加坡全国青年成就奖（National Youth Achievement Awards，简称NYAA）理事会联合举办“在错误信息时代寻找真相”炉边对话。
布兰迪引用调查指出，新加坡与意大利民众对假信息威胁的警觉程度相近，分别为77%和74%；但两国公众对政府应对能力的信心差距明显。70%的新加坡受访者相信本国主管机构有能力应对假信息行动，意大利仅36%；在应对混合威胁方面，新加坡为82%，意大利为42%。
布兰迪受访时解释，新加坡社会的凝聚力较强，能够在尊重不同族群、语言和宗教的前提下，建立起一种共同的身份认同；相较之下，意大利的公共辩论长期受政治极化、地区差异和身份认同分裂影响，舆论更为碎片化，也削弱了民众对政府在应对假信息、混合威胁和网络安全挑战方面的信心。
主讲者之一、普林斯顿大学访问教授兼意大利路易斯大学数据实验室主任里奥塔（Gianni Riotta）则提醒，公众信任不能被视为理所当然。
里奥塔受访时说：“不要以为民众10年前信任你、今天信任你，明天就一定还会信任你。”随着社交媒体削弱传统权威的影响力，社会必须持续维护公众信任。
这种信任一旦松动，假信息便不只是外部势力制造的威胁，也可能从社会内部的分歧和不满中滋生。
里奥塔以意大利天主教会数年前委托的一项研究为例说，研究原本要查明是谁在散播针对前任教宗方济各的虚假信息，结果发现，在意大利约95%的相关内容来自天主教圈内部，而非外部敌对者。
“当我们把结果带回去给那位主教时，我其实很尴尬。”里奥塔说，因为这些信息主要来自反对教宗路线的保守派和传统派天主教徒，显示问题“在自己家里”。
因此，应对假信息不能只停留在辟谣和监管层面，更要弄清楚人们为何愿意相信这些错误叙事。
里奥塔指出，随着社会愈发两极化，许多人相信假信息，并非因为缺少事实，而是因为不再信任主流机构。真正的难点不在于指出“什么是假的”，而在于重建公众对可靠资讯的信任基础——这将是一个漫长过程，也不是新闻界或政府能单独完成的任务。

世界卫生组织总干事：刚果（金）疑似埃博拉死亡达220人，疫情蔓延速度超过应对能力
日内瓦讯 — 刚果民主共和国（DRC）东部冲突区当前爆发的罕见Bundibugyo型埃博拉已造成重大影响。卫生部5月24日数据称，三省11个卫生区已报告逾900例疑似病例；世卫组织总干事特德罗斯·阿达诺姆·盖布雷耶苏斯5月25日表示，目前疑似死亡达220人，因病例发现延误，救援工作正在“追赶疫情”。他并敦促与刚果接壤国家立即采取行动，并于5月26日启程前往刚果。
世卫组织已将此次Bundibugyo株疫情列为国际关注的突发公共卫生事件。该株目前无获批疫苗或抗体疗法，防控面临严重挑战。
暴力冲突、对当局的不信任以及薄弱的监测体系加剧了防疫难度。卫生部资料显示，医务人员单日仅能追踪约20%的接触者。特德罗斯指出，暴力导致民众和卫生、人道工作者逃离，使接触者追踪和早期识别治疗严重受阻。
当地社会紧张突出。美联社报道，5月24日，蒙布瓦卢一所收治埃博拉病人的医院因当局拒绝归还遗体而遭愤怒居民冲击；伊图里省此前也发生治疗帐篷被焚、病人逃离等事件。
谣言与误解广泛传播。援助行动（ActionAid）称，伊图里多达三分之一民众认为埃博拉不存在，该组织已开始开展宣教活动以纠正错误认知，强调当前既在对抗病毒，也在对抗恐惧与怀疑。
区域卫生官员此前在乌干达首都坎帕拉警示，疏松的边界、活跃的采矿通道和大量人口流动增加跨境传播风险。非洲疾控中心主任让·卡塞亚5月24日表示，因区域流动性和监测诊断能力差距，已有10个非洲国家被列为受疫情威胁国家。
乌干达已报告5例与此次疫情相关的确诊病例。一名在刚果照护埃博拉患者感染的美国人已被撤离至德国治疗，美国疾控中心称若干高风险接触者也被转移至德国和捷克接受监测与处理。

梵蒂冈—5月25日，教皇里奥十四在其首份重要通谕《Magnifica Humanitas》（壮美人性）中呼吁各国放慢人工智能发展，警示AI助长错误信息、优先化冲突并可能将世界推向无休止的战争。该近4.3万字文件自他一年多前当选以来酝酿完成，并在梵蒂冈发布会亮相，出席者包括人工智能公司Anthropic联合创始人克里斯·奥拉赫。
里奥指出，部分自主武器“几乎超出人类治理能力”，主张AI数据所有权不可全由私营部门掌控，政策制定者应保护劳动者权益、保障儿童安全并遏制AI企业间的竞赛。他呼吁“更积极的政治介入”，建立健全法律框架、独立监管与知情用户制度，政治体系不得放弃责任。
通谕同时谴责当今多起残酷战争、多边机制弱化及军火利润推动冲突，明确否定“正义战争”理论，警告领导人可能以战争转移国内问题。在劳工与伦理方面，他批评AI供应链存在“新型奴役”，描述儿童在危险环境中开采稀土、工人身心受损的现状，并就教会曾迟迟未谴责跨大西洋奴隶制表示道歉。里奥以巴别塔寓言警示人类勿以傲慢追求“通天”计划，号召各方为共同利益携手行动，强调人人有责。（路透）

美国对伊朗南部目标的有限空袭并非意在恢复全面战争，而是向德黑兰发出政治警告。五角大楼称此次“自卫”打击针对伊朗导弹阵地及据称在霍尔木兹海峡布雷的船只，华盛顿传达的明确信号是一次性行动。
但此举可能进一步动摇脆弱的停火局面——伊朗高级谈判代表正赴邻国卡塔尔进行结束战争的高风险会谈。该事件也凸显了特朗普在对伊政策上的政治平衡：一方面寻求尽快结束军事行动并推动谈判，另一方面需回应国内对妥协的强烈批评。
多年冲突已造成双方互不信任：伊朗指责每次与美方谈判都会遭空袭，而伊朗本身的行为亦常难以预测。尽管自4月8日以来双方大体遵守停火，实际仍在加强军事部署——美国增派舰只并运送弹药，伊朗则加强沿岸防御与布雷以维持对霍尔木兹的影响。
美国此次打击部分目的在于削弱伊朗巩固海峡控制的努力，同时也有助于缓解对特朗普的国内政治压力。此前特朗普为促成迅速协议已在多项要求上作出让步，但拟议协议遭到包括共和党鹰派在内的强烈反对，迫使白宫开始回撤并重申一些旧前提。
白宫仍表示愿谈判，但若伊朗报复，冲突恢复的风险将大增。即便停火持续，特朗普力图达成一项他能宣称优于先前政府成果的协议，仍可能成为谈判实现持久结束的主要障碍。
（作者：乔纳森·埃亚尔）

新加坡 — 推动“Pax Silica”倡议的美国官员雅各布·赫尔伯格表示，该计划并非要求成员在美中之间二选一，而是让签署国分享切实利益。赫尔伯格在接受《海峡时报》采访时称，加入国家看重的是“更多投资、更高附加值活动、更多人才以及在未来十年AI经济扩张中更大的话语权”，成员并非出于义务，而是被实际利益所吸引。 
赫尔伯格指出，包括阿联酋、卡塔尔在内的多国既是Pax Silica成员，也是中国主导的一带一路参与国，表明参与并非要取代或对抗另一方。他同时强调，美方会有针对性地与那些技术先进、嵌入高附加值产业链的国家合作，但在有具体可交付成果的前提下也欢迎扩大成员。 
特朗普政府于2025年12月发起Pax Silica，旨在确保关键矿产、半导体与人工智能供应链的韧性。菲律宾今年4月正式加入，拟与美国共建产业枢纽以稳固关键产业供应链。赫尔伯格称，新加坡凭借约占全球晶圆产量10%的地位，可在晶圆、封装、材料、资本与人才等多层面发挥重要作用。 
赫尔伯格于5月17至21日访新访菲，并在新加坡召集首届东盟AI部长级圆桌会。除新加坡与菲律宾外，澳大利亚、日本、以色列、韩国、瑞典、卡塔尔、挪威、英国、芬兰、阿联酋、印度与希腊亦已签署Pax Silica，台湾为非签署参与方。赫尔伯格总结称：“未来十年增长的故事将贯穿AI供应链，Pax Silica是伙伴们与我们共同乘势而上的方式。”

华盛顿 — 美国司法部民权司于5月26日对加州大学洛杉矶分校（UCLA）提起诉讼，指控该校容忍对犹太人和以色列学生的敌对教育环境。
起诉书称，UCLA对校园内“普遍存在的反犹主义”表现出故意忽视，违反了第六章（Title VI），即联邦法律禁止在接受联邦资助的项目或活动中基于种族、肤色或民族出身的歧视。诉状指出，校园内于2024年4月出现的帐篷营地为非法搭建，并指控有犹太学生遭到袭击。
中区加州联邦检察官第一助理比尔·埃赛利（Bill Essayli）表示：“大学有义务为所有学生维护安全、包容的校园环境。对反犹行为反复不加防护的高校将被追究责任。”
UCLA尚未对本案置评。司法部在提起诉讼前不久，一联邦上诉法院于5月26日部分维持一项禁令，要求恢复加州大学在2025年被特朗普政府以支持“多样性、公平与包容”（DEI）为由撤销的拨款。
这是特朗普政府自2023年10月7日哈马斯袭击以色列后，针对美国高校所谓反犹问题采取的一系列法律行动之一。2023年袭击导致逾1200人死亡，随后引发以色列—加沙冲突，迄今已造成数万人伤亡。冲突爆发后，美国及欧洲多地数百名学生举行支持停火的示威，部分人士占领校园建筑。 路透社
