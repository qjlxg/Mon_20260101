中消协：春节“抢票难”“民宿违约”等舆情突出
今年中国春节，“抢票难”“民宿违约”等消费维权舆情突出。
据中新社报道，中国消费者协会星期二（3月3日）发布的2026年春节消费维权舆情分析报告显示，春节假日消费维权舆情相对集中，主要涉及交通出行、景区管理、住宿履约、用餐服务等方面的维权诉求。
报告指出，“抢票难”导致第三方票务平台业务需求高涨，有消费者网购全程卧铺票却仅获半程出票，暴露了第三方票务平台审核不严、失信成本过低的问题。与此同时，部分列车因人员超载导致列车无法正常启动、旅客行程受阻，进一步加剧节日出行困难。
报告称，从南京桔子酒店因自身履约问题，对合理退订的消费者发送恶意恐吓信息，到三亚民宿单方取消大额预订，此类事件虽表现形式各异，但均反映出部分商家在服务履约中存在漠视契约精神，只求“一锤子买卖”问题。从平台方面看，其在商家准入审核、日常运营监管等环节的机制不完善，对违约行为的惩戒力度不足，致消费者维权成本高、维权难度大。
春节景区客流激增，安全管控与设备运维的短板暴露。报告指出，云南楚雄莲溪湾度假村涉客船舶倾覆，船上26人落水，造成四人死亡，凸显景区水上项目安全监管缺位；辽宁辽阳弓长岭温泉滑雪场索道发生故障出现倒行，虽未造成人员受伤，但涉事索道停运、设备运维不到位，也暴露了景区游乐设施管理的松懈。
报告称，烤羊缺斤短两事件，以及网红蛋糕店使用发霉草莓等侵权行为隐蔽性强，消费者仅凭外观、宣传难以辨别，易陷入信息不对称的消费陷阱，面临货不对板、食品安全风险。餐饮行业的部分不当经营行为破坏了消费公平，背后折射出商家经营导向偏差、行业自律不足以及监管存在薄弱环节等问题。

中国敦促各方立即停止军事行动 维护霍尔木兹海峡航道安全
中东战火升温之际，中国外交部星期二（3月3日）称，敦促各方立即停止军事行动，避免紧张事态升级，维护霍尔木兹海峡的航道安全。
受冲突影响，全球能源命脉霍尔木兹海峡的石油运输几乎停摆。中国约三分之一的原油供应经由这条水道运输。

克林顿称，特朗普当时显然知道克林顿曾乘坐爱泼斯坦的飞机；特朗普据称说两人“这些年一起度过了一些美好时光，但因为一桩房地产交易闹翻了”。特朗普方面曾表示，分歧始于爱泼斯坦雇用了在其海湖庄园工作的年轻女性。克林顿说，这段对话并未让他认为特朗普涉入爱泼斯坦的不当行为。 
两人在爱泼斯坦2008年认罪前均曾与其交往，但双方都否认有卖淫或性交易等违法行为，且未被当局控以与爱泼斯坦相关的刑事罪名。爱泼斯坦2019年再度被捕，面临联邦人口贩运指控，在狱中死亡，官方裁定为自杀。 
克林顿在证词中说，他是由前财长拉里·萨默斯引介给爱泼斯坦，曾乘其飞机赴亚洲、非洲、欧洲等地出访，2003年以后转而依赖其他捐赠者。他否认与爱泼斯坦或其合作者吉斯莱恩·马克斯韦尔介绍的任何人发生性接触，但承认曾在飞机上接受一名后被认定为受害人的空姐的颈部按摩，并表示不知爱泼斯坦曾招募并性侵被当作按摩师的女孩。克林顿说他从未去过爱泼斯坦的加勒比岛，也不知爱泼斯坦在其任内曾17次出入白宫。 
由共和党控制的委员会亦传唤了前国务卿希拉里·克林顿，她上周作证称不记得曾见过爱泼斯坦。路透社

联合国驻阿富汗援助团（UNAMA）3日表示，自2月26日至3月2日，阿富汗与巴基斯坦交火已造成至少42名平民死亡、104人受伤。冲突进入第六天，局势仍然紧张。 
阿方称已在坎大哈地区再夺一处巴方哨所，战斗“仍在进行中”。UNAMA指出，平民伤亡包括越境交火的间接火力及空袭所致，相关数字为初步统计。 
这场数年来最严重的冲突始于上周，塔利班称其对巴基斯坦设施的报复性打击是回应巴方在阿境内打击武装分子。阿方指责巴方袭击平民，伊斯兰堡予以否认。 
伊斯兰堡近一周对塔利班军事目标发动空对地导弹袭击，并首次以塔利班政府为目标，称其庇护对巴发动袭击的武装分子。巴方安全消息称，3日已在阿富汗楠格哈尔省摧毁一处军事基地。双方均称已重创对方部队与设施，路透无法独立核实。 
巴基斯坦总统阿西夫·阿里·扎尔达里2日在国会联合会议上重申，不会允许邻近领土被用来对巴发动袭击，并称巴基斯坦领土神圣不可侵犯。 
UNAMA呼吁停止战斗，警告冲突已致约1.64万户流离失所，加剧了仍在从去年8、9月造成1400余人死亡地震中恢复的民众困境，边境地区行动受限也削弱了人道救援能力。

特朗普称“很难过看到”英美关系不如从前
美国总统唐纳德·特朗普表示，看到与英国的关系“明显不如从前”，对此感到“很难过”。在3月3日接受英国媒体采访时，特朗普批评首相基尔·斯塔默在对伊朗的打击行动中最初未提供军事支持。
特朗普称，法国等国反而更为支持，自己从未料到这段曾被视为“最为牢固”的盟友关系会出现这种变化。他还表示，美国在中东不依赖英国参战，但斯塔默“本应”给予帮助。
英国高级部长达伦·琼斯回应称，英美关系仍然至关重要，但英国从2003年伊拉克战争中吸取了教训：参与此类行动时应与国际伙伴保持一致并有明确的法律依据。斯塔默已于3月1日表示，会允许美军使用英国基地进行防御性打击，并在3月2日于议会重申，他是依据国家利益作出决定并坚持该立场。　路透社

巴黎 — 法国外长让-诺埃尔·巴洛特（Jean-Noel Barrot）3月2日与中国国务委员兼外长王毅通话，双方就伊朗战争局势交换意见，并同意共同努力实现降级与缓和，巴洛特办公室在声明中表示。
声明称，两国外长承诺寻求一项既能保障集体安全又考虑伊朗人民诉求的政治解决方案。巴洛特重申，伊朗政权应对当前局势升级负责，称其已不当袭击本地区数国，并批评伊朗拒绝遵守联合国安理会有关核计划、弹道活动及对准军事组织支持的决议，且拒绝以诚意参与多边谈判。
巴洛特还重申，法国与美以等国的相关行动无关也并不事先知情，并强调各国应优先通过国际机构解决争端，必要时可诉诸武力。两国外长同意就此保持对话渠道畅通。
与此同时，中国外交部发言人在例行记者会上敦促各方停止在伊朗的军事行动，防止战争进一步蔓延，呼吁有关各方回到对话轨道，并表示中国对战争波及其他国家深感关切，将继续在争取和平方面发挥建设性作用。 路透社

迪拜等中东购物中心多家商店因地区冲突关闭或仅以精简人手运营，企业与出行受扰。3月2日，随着美以对伊朗的空袭升级、德黑兰以导弹和无人机回击并波及海湾国家，局势暂无缓解迹象。
代理多品牌门店的Chalhoub集团表示，巴林门店已关闭，阿联酋、沙特和约旦等地保持营业但员工“自愿”出勤；公司高层3月2日赴迪拜商场探望员工。亚马逊暂停阿布扎比履单中心、停发区域内配送并指示沙特与约旦员工留在室内。开云集团称在阿联酋、科威特、巴林和卡塔尔的门店暂时关闭，并停止前往中东的差旅。
奢侈品股3月2日下午普遍下跌4%至5.7%。分析师称，中东在全球奢侈品消费占比仅5%至10%，但在2025年表现突出；若机场停航、旅游骤减并持续数周，可能使价值数亿至十亿美元的免税与旅游零售收入面临风险。
卡地亚、路易威登等刚在当地举行展览或开业活动；快时尚与消费品企业亦受影响：Primark原计划在迪拜和海湾开店的计划受关注，苹果迪拜门店关闭至3月5日，H&M在巴林与以色列的门店关闭，消费品公司Reckitt要求中东员工居家办公、暂时关闭巴林厂并暂停该地区差旅。路透社报道。
