调查：战争对美经济冲击低于预期 但通胀更顽固降息空间收窄
（纽约讯）经济师认为，伊朗战争对美国经济造成的冲击没有预期严重，但与此同时，居高不下的通货膨胀也变得更加顽固，使美联储失去了降息空间。 
根据《华尔街日报》本月进行的调查，受访经济师预计，今年美国经济将增长2.1%，略高于4月调查时的2%。他们对美国在未来12个月内陷入衰退的平均概率预期，已从4月份的33%降至25%，为2025年初以来的最低水平。
不过，在增长前景改善之际，通胀担忧却在加剧。预料，在截至今年12月的一年里，美国消费者价格指数（CPI）将上涨3.4%，高于4月调查预测的3.2%。
战争造成的价格上行压力已蔓延到能源以外领域。剔除食品和能源价格后的个人消费支出价格指数今年料上涨3.2%，而4月份的预测为2.9%。
经济师此前担心，霍尔木兹海峡因美伊战争关闭，导致能源价格飙涨，将给消费者带来沉重打击。然而，油价上涨带来的冲击，要低于许多分析师的预期。
《华尔街日报》指出，与过去数十年相比，如今经济活动对石油的依赖程度较低，股市上涨也有助于维持消费支出的稳定。
美国和伊朗上个月签署谅解备忘录后，油价一度跌至每桶68.55美元（约88.6新元）。近日双方再次交火，导致油价在上周五（7月10日）反弹至每桶71.41美元，但仍远低于4月份112.95美元的峰值。
