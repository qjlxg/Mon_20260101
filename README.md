中国今年军费预算为19095.61亿元人民币，同比增长7%。过去三年，军费增幅一直维持在7.2%。
这是自2022年以来，中国军费预算增幅连续五年超过7%。中国财政部在今天（3月5日）公布的政府预算草案报告中披露今年军费预算。中国军费预算自2016年结束两位数增长后，保持个位数增长至今，但都高过当年的GDP目标增速。

李家超：将带领港府全速推进香港五年规划编制工作

李家超也说，他将带领特区政府全速推进香港五年规划的编制工作，全方位对接国家“十五五”规划。香港会主动融入和服务国家发展大局，贡献粤港澳大湾区建设和“一带一路”倡议等。

美参院否决限制特朗普战争权力议案
美国参议院共和党人周三投票支持特朗普总统对伊朗的军事行动，否决了一项旨在停止空袭并要求任何针对伊朗的敌对行动都必须经国会授权的两党决议。
参议院星期三（3月4日）以47票赞成、53票反对的投票结果否决了该决议，投票结果基本按照党派路线划分。除一名共和党人外，所有共和党人都投票反对这项程序性动议；除一名民主党人外，所有民主党人都投了赞成票。
依据美国宪法，只有国会能授权发动战争。多名民主党人曾发声，称特朗普政府对伊朗的军事行动未经国会授权，当属违法。
美国参议院外交委员会主席里施（Jim Risch）在反对该决议的演讲中说：“这并非一场永无止境的战争，实际上远非如此。这场战争很快就会结束。”
该决议的支持者则表明，他们不会放弃。甚至一些投票反对该决议的共和党人也说，他们将敦促特朗普的助手就政府的伊朗战略公开作证，尤其是如果冲突像特朗普预测的那样将持续数周。
关于特朗普在中东地区增兵，以及美国和以色列对伊朗发动空袭的争论，主要集中在特朗普是否正在将美国卷入另一场像伊拉克和阿富汗那样旷日持久的“永无止境的战争”。
该决议的共同发起人、参议院少数党领袖舒默（Chuck Schumer）说：“今天，参议员们面临着一个抉择：要么与厌倦了中东战争的美国人民站在一起，要么站在特朗普一边，支持他把美国拖入另一场大多数美国人强烈反对的战争。”
路透社报道指，鉴于国会控制权可能在11月的中期选举中落入民主党手中，旷日持久的伊朗战争可能会令选民感到担忧。路透社/益普索周二（3日）公布的一项民调显示，只有四分之一的美国人赞成美国对伊朗发动打击，约半数美国人认为特朗普过于热衷于动用武力。

美国参议院周三以53票对47票否决了一项旨在限制总统特朗普对伊朗发动军事行动权力的决议。该两党提案由民主党参议员蒂姆·凯恩和共和党参议员兰德·保罗提出，要求在国会未授权的情况下撤回参战美军。
共和党在参院占53席，本次表决结果正好反映了党派分野。投票发生在冲突爆发第五天——据报道，伊朗最高领袖哈梅内伊及多名高级官员丧生，同时美军在科威特基地遭伊朗袭击也有伤亡。

若要通过，该决议至少需四名共和党人倒戈；但宾夕法尼亚州中间派民主党人约翰·费特曼反对。即便参众两院通过，总统仍可否决，且国会要想推翻否决需在两院分别获得近三分之二多数，几乎不可能。
各国政府正撤离中东受冲突波及的公民，迪拜和利雅得等原被视为相对安全的城市也已受牵连。政府在机密简报中称，这场行动可能持续数周，需额外拨款；国防部可能很快申请紧急资金补充弹药与物资。
该决议援引了1973年《战争权力法》，该法允许国会对未经授权的军事行动进行表决并将持续时限限定为60天。尽管成功几率不大，民主党仍强调让国会议员公开表态的重要性。

白宫：美方正制定确保霍尔木兹海峡通航计划
白宫新闻秘书卡罗琳·利维特周三（3月4日）表示，美国正积极制定保障霍尔木兹海峡及油轮安全通行的方案。国防部与能源部正在紧密协调并定期向总统唐纳德·特朗普汇报，利维特未给出时间表，但称必要时美海军可为油轮提供护航。
利维特在记者会上否认部署美军地面部队进入伊朗是当前行动计划的一部分，但强调总统不会将军事选项“排除在外”，一切取决于事态发展。
她还驳斥了美方欲武装库尔德势力以煽动伊朗国内起义的报道，称此类说法“完全错误”。她确认总统已与包括伊拉克北部库尔德领导人在内的地区伙伴通话，但否认就此类策略达成任何协议。
霍尔木兹海峡为重要油运通道，2月28日美国与以色列对伊朗的攻击使该航道重新成为关注焦点，并可能导致海峡短期关闭。海峡最窄处约21英里（33公里），航道在两侧各仅约2英里（3公里）。
分析公司Vortexa数据显示，约五分之一的全球石油通过此海峡，去年日均逾2000万桶原油、凝析油及燃料通过该处。
随着德黑兰对船只和能源设施的攻击导致航行受阻、从卡塔尔到伊拉克的产量停摆并中断出口，油价周三继续走高。半官方的法尔斯通讯援引伊朗革命卫队称海峡处于其“完全控制”之下，并警告船只勿通行；其称已打击10艘无视警告的油轮，但这一说法尚无法独立核实。
若能源价格持续上升，特朗普在11月国会中期选举前可能面临政治压力。
