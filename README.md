中国国内航线燃油附加费将再上调 单程最高170人民币
中国国内航线燃油附加费将在星期六（5月16日）再次上调，单程最高收费为170元（人民币，下同，31.8新元）。
综合澎湃新闻和《证券时报》报道，九元航空星期二（5月12日）发布关于调整国内航线燃油附加征收标准的通知。
自5月16日零时起，上调国内航线旅客运输燃油附加费征收标准。800公里（含）以下航线，每位旅客燃油附加征收标准调整为90元。800公里以上航线，每位旅客燃油附加征收标准调整为170元。
这意味着，国内航线旅客运输燃油附加费将分别上调30元和50元。
九元航空公告指出，婴儿旅客免收燃油附加。儿童、按民航局规定享受成人公布普通票价50%优惠的革命伤残军人和因公致残的人民警察实行减半收取，800公里（含）以下航线每位旅客收取40元，800公里以上航线每位旅客收取80元。
公告还指出，燃油附加的征收以原始出票日期为准，客票改期时，燃油附加费不退不补。
目前中国各航司国内航线燃油附加费都是统一价格，这意味着将实现普涨。国内航线燃油附加费上一次调整为今年4月5日，上涨了五倍，800公里（含）以下航段向每位成人旅客收取燃油费60元，800公里以上航段向每位成人旅客收取燃油费120元。

5月1日，白宫新闻秘书卡罗琳·利维特在华盛顿诞下女婴维维安娜，成为首位在任内分娩的白宫新闻秘书。副总统J·D·范斯的妻子乌莎预计7月分娩，将迎来第四个孩子；总统办公厅副主任斯蒂芬·米勒之妻也将迎来第四个孩子。此前，维韦克·拉马斯瓦米在竞选俄亥俄州州长期间于2月迎来第三个孩子。参议员马可·鲁比奥育有四名子女，国防部长皮特·赫格塞斯有七名子女。特朗普本人是五子之父、11名孙辈的祖父，曾自称“助孕总统”，并推动所谓的“婴儿潮”。

专家指向晚婚（目前男性中位数30岁、女性29岁）、住房、医疗、教育成本上升及学贷负担等结构性因素。特朗普承认政府能力有限，但政治上也有考量：支持他的县域出生率普遍高于民主党支持区。媒体热传白宫新生儿照片，但专家认为这些高调的个案难以扭转全美低生育趋势。

泽连斯基称，乌方还将在无人机合作框架下与世界其他地区启动安全合作（未指明具体地区），并已通过这些协议开始获得所需燃料。他希望“无人机外交”能为乌克兰争取中东国家的能源供应和农产品市场。来源：路透社

布鲁塞尔 — 欧盟外长5月11日就新一轮制裁达成一致，针对在被占领的约旦河西岸对巴勒斯坦人施暴的以色列定居者以及若干哈马斯高层人物，欧盟外交与安全政策高级代表卡娅·卡拉斯（Kaja Kallas）宣布。制裁名单涵盖三名定居者和四个定居者组织，具体身份尚未公开。该方案此前因已经在四月大选中落败的匈牙利前政府阻挠数月。欧洲各国对约旦河西岸定居者暴力事件上升表示关切。卡拉斯在社交平台X上称，打破僵局、付诸行动“正当其时”，并警告“极端主义与暴力必将承担后果”。以色列外长吉迪恩·萨尔在X上反驳，称欧盟此举“任意且带有政治动机”，无根据地因政治观点对以色列公民和实体实施制裁，并指责将以色列公民与哈马斯恐怖分子相提并论是“完全扭曲的道德等同”。哈马斯尚未立即回应。路透社报道。

北京 — 白宫高级官员5月10日表示，美国总统特朗普本周将赴北京，与中国国家主席习近平就中国对伊朗战争的立场，以及拟设立的新贸易与投资“理事会”细节进行交涉。两国领导人定于5月14—15日会谈。 
官员称，峰会上将讨论中国对伊朗的石油收入、可能的武器输出等敏感问题。美国已对多家涉购伊朗石油或向伊朗提供卫星影像的中国企业实施制裁。财政部长斯科特·贝森特将于5月13日在首尔与中国副总理何立峰进行最后磋商。 
中国外交部发言人郭嘉坤表示，两国元首将就中美关系及世界和平与发展等重大议题交换意见，强调在平等、尊重和互利原则下扩大合作、妥善管控分歧。新华社称，中美联合行动于4月破获跨境贩毒网络并逮捕五名嫌疑人。 
台湾问题、人工智能安全与双方可能开辟的新沟通渠道也将在议程中讨论。此前Anthropic新模型暴露出的网络安全风险，已促使美英及亚洲监管机构加紧关注。习近平与前总统拜登曾就AI风险达成相关限制性承诺。 
白宫表示，特朗普还可能借此宣布北京拟购美国产品的计划，并就农业、航天与能源等领域达成若干安排。分析人士称，首尔的最后会谈暗示双方或有出乎预期的成果。 
行程方面，特朗普将于5月13日上午抵京，14日参加迎接仪式并会见习近平，下午参观天坛，晚间出席国宴；15日将共进茶宴与工作午餐后离京。
