距离中美领导人峰会仅剩几周，中国完全有理由置身事外，将自身塑造成稳定力量，而非冲突的直接参与者。
报道认为，北京重视与德黑兰的关系，但有关重要性远不及与俄罗斯的关系。中国与伊朗的关系似乎更多建立在务实基础之上，而非深层战略一致。
伊朗加入金砖国家和上海合作组织等由中国主导的机制，有助于北京稀释美国影响力，并强化其作为全球南方重要声音的地位。

但长期以来，中国在中东奉行分析人士所说的双轨策略（dual-track approach），在与伊朗的关系与同美国及其地区盟友的往来之间保持平衡。彭博社引述北京清华大学国际关系学系副教授佘纲正说，对伊朗提供军事支持“并非中国在该地区的一贯做法”。
北京与莫斯科的关系则处于完全不同的层级。俄罗斯是中国抗衡美国主导地位的重要支点，也是关键能源供应国，并在多个多边机制中与中国密切合作。中国国家主席习近平与俄罗斯总统普京之间的密切个人关系，进一步强化了这种联系。

美伊冲突扰乱航运 多家保险公司取消战争险
由于不断升级的伊朗冲突扰乱了航运，导致油轮受损或搁浅，并造成至少两人死亡，保险公司正在取消对波斯湾船只的战争风险保险。
霍尔木兹海峡连接波斯湾和阿曼湾，是沙特阿拉伯、伊拉克、卡塔尔、阿联酋等中东产油国的原油出口必经之路，通过这一海峡运输的石油约占全球石油运输总量的五分之一。
美国和以色列上星期六（2月28日）袭击伊朗后，伊朗革命卫队宣布封锁霍尔木兹海峡。
路透社报道，随着波斯湾地区的船只在伊朗报复美以袭击的行动中遭到打击，上述航道的运输几乎陷入停滞。  
截至星期一，波斯湾有一艘油轮起火，另有至少四艘油轮受损。
有关影响以及对霍尔木兹海峡长期封锁的担忧，已推高了石油和欧洲天然气价格。随着冲突在中东引发多处油气停产，布伦特原油期货一度上涨多达13%。
航运数据显示，截至上星期天（3月1日），包括油轮和液化天然气运输船在内的至少150艘船舶已在霍尔木兹海峡及周边水域抛锚停泊。
受此事件影响，海运保险公司正取消船舶战争风险保险，油品运输运费料将进一步飙升。
据报道，多家保险公司星期天宣布，从3月5日起，凡驶入波斯湾、特定毗邻水域或伊朗水域的船舶，战争险保障将被取消。这些保险公司包括：Gard、Skuld、NorthStandard、伦敦保赔协会等。
这也意味着，在有关地区拥有船舶的航运公司需以更高费率寻求新保险才能维持保单。
保险经纪公司McGill and Partners的海运业务负责人史密斯说：“局势瞬息万变，各承保商普遍上调费率，部分针对霍尔木兹海峡过境船舶的承保甚至已暂停。”
行业消息人士周一透露，过去48小时内战争风险保费已从上周约0.2%飙升至船只价值的1%，这意味着每批货物成本将增加数十万美元。

开罗消息——伊朗革命卫队一名高层3月2日表示，霍尔木兹海峡已被关闭，伊朗将向任何试图通行的船只开火，国内媒体报道。

伊朗称此举是对2月28日美以对伊朗领导层发动袭击及美国总统提出协助推翻伊朗最高领袖的回应。作为反击，德黑兰向驻有美军基地的邻国发射了多波导弹，包括卡塔尔、科威特和巴林，并对阿联酋、沙特和阿曼发射了导弹。
此外，全球航运此前已因也门亲伊朗的胡塞武装在红海和亚丁湾发起的无人机与导弹袭击而受到干扰，自2023年加沙战争爆发以来该组织多次攻击过往船只。
