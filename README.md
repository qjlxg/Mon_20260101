台行政院副院长：台美贸易协议不包括采购美农产品内容
台湾行政院副院长郑丽君在台美关税谈判说明记者会上说，台湾对美国农产品的采购将不包括在台美贸易协议里。
郑丽君一行此前赴美进行新一轮台美关税谈判，然后在上周签署了投资合作备忘录，星期一（1月19日）返台，星期二（20日）召开记者会对台美关税谈判进行说明。
综合中时新闻网、TVBS新闻网和《自由时报》等台媒报道，郑丽君在会上说明，台美贸易协定将在数周后签署，先前提及的台湾对自美采购农产品，这属于农业法规管理下的民间采购行为，不适合列入政府间协定，因此已删除。
她称，台方也将向美方争取超过1000项货物免关税，目前仍在确认中。
对于台美避免双重课税协定（ADTA）进度，郑丽君说，历次谈判都曾向美方提出希望尽快洽签ADTA，在这次总结会议时跟美国贸易代表署再次提出台方期待，星期一收到最新消息，相关单位希望启动双方就ADTA后续加速程序的会谈。　
郑丽君重申，以台湾模式支持境内产业进行有利的国际布局，不是“供应链外移”，而是支持台湾高科技产业以加法、乘法延伸产业的实力，扩大在美国有利的国际布局，期待未来透过台美强强联手，打造高科技产业供应链。
与郑丽君一同赴美参与谈判的台湾行政院经贸谈判办公室总谈判代表杨珍妮在会上，针对美制汽车关税是否调降的提问时说，美方要求台方全面市场开放，希望可以对齐邻近东南亚国家和地区全面市场开放情形，并参考过去台方与他国签署自由贸易协定的开放程度，汽车关税将纳入这一范畴与美方进行协商。

美国洛克希德·马丁周四发布的长程反舰弹（LRASM）动画显示，包括F‑35在内的战机对一艘酷似055型大驱的目标投放导弹；另一段片中LRASM瞄准的目标被指类似中国首艘航母辽宁号。
