卡尼吁加澳携手合作 加强对抗霸权
（堪培拉综合电）加拿大总理卡尼星期四提出，加拿大与澳大利亚应成为战略伙伴而非竞争对手，两国携手合作将加强与包括美国在内的超级大国谈判的实力。
卡尼星期四（3月5日）访问澳洲期间向澳洲国会发表演讲，这是自2007年以来加拿大领导人首次在澳洲国会发言。
卡尼说，他此次来访是为了在全球架构因接连不断的危机而分崩离析之际，重申渥太华与堪培拉的同盟关系。他指出，加澳虽然在地理上相隔遥远，但理应成为“战略表亲”。
他说：“在旧世界……我们倾向于把彼此视为竞争对手。在新世界，我们应该成为战略合作伙伴，在提升战略自主性的同时，加大投资、加快技术合作，增强供应链韧性，以及扩大国内加工能力。”
卡尼说，在大国竞争的世界中，中等强国面临的选择是：要么争取大国青睐，要么联合起来增强实力。他认为，中等强国的当务之急是制定规范，并参与制定新规则，以共同应对日益被霸权大国主导的世界格局。
他强调：“中等强国不能仅仅筑起更高的墙，并退到墙后。我们必须共同合作。”
澳洲总理阿尔巴尼斯星期四在国会致辞时也说，卡尼来访反映了加澳在世界舞台上发挥更大作用的共同抱负。
阿尔巴尼斯说：“澳洲和加拿大是正在变化的世界中的中等强国。我们无法逆转变革，但可以支持自己与我们的公民，同时互相支持。”
卡尼指出，两国可以合作发展战略优势的领域包括关键矿物、人工智能（AI）、贸易，以及无人机和卫星等防御设备。
他也宣布，加澳当天早些时候签订了一系列与关键矿产有关的新协议，包括澳洲加入七国集团（G7）矿产联盟。
这个联盟由加拿大主导，是全球拥有主要矿产储备的民主国家组成的最大集团，旨在促进全球关键矿产的多样化，并保障生产和供应。加拿大和澳洲共生产全球约三分之一的锂和铀，以及超过40%的铁矿石。
澳洲是卡尼亚太之行的第二站。他此前已访问印度，星期五（6日）离开澳洲后将续程前往日本。

新加坡驻华大使陈海川在北京表示，面对全球格局深刻变化及贸易摩擦带来的不确定性，新中两国在维护以规则为基础的国际秩序方面具有“共同利益”。他在中国人民大学崇阳金融研究院主办的活动上指出，自去年以来，多边主义、自由贸易与开放环境面临巨大压力。 
陈海川强调：“小国需要稳定，大国需要可预见的发展。开放、互联与合作符合我们的共同利益。”他并称，新加坡与中国正在共同推动构建更加一体化、繁荣的亚洲。 
回顾两国关系，陈海川表示，建交35年来，双方形成了“独特、牢固且富有活力”的双边关系，基于主权平等与相互尊重，而非单纯的文化或族裔联系（新加坡常被视为除中国外唯一有多数华人群体的主权国家）。 
在美中战略竞争日益激烈的背景下，陈海川指出，新加坡一方面将美国视为最大外资来源与重要安全合作伙伴，另一方面中国是其最大的货物贸易伙伴；自2013年以来，新加坡亦是中国新增外资的重要来源。作为高度依赖贸易的国家，新加坡密切关注美中经贸竞争，并将继续捍卫多边、开放与自由贸易秩序。他期待今年11月在深圳举行的APEC会议能发挥重要作用。 
同场的中国外文局所属中国国际问题研究院副院长刘青则呼吁，中方与新加坡应共同反对集团政治与“小圈子”做法，共同维护地区和平与繁荣，抵制国际舞台上“丛林法则”的蔓延。

宋平，这位经历了党和国家五代领导集体的老一辈革命家，于108岁去世。新华社报道，他于周三下午3时36分在北京逝世，被誉为“忠诚的共产主义战士”和“杰出的国家与党领导人”。
宋平出生于1917年4月，山东莒县人，1937年19岁时加入中国共产党，早在新中国成立前的1930年代就开始从事革命工作，曾任未来总理周恩来的政治秘书。
此后数十年间，他历任要职，1989年进入中共中央政治局常务委员会，成为党内最高决策层成员，1992年以75岁高龄退休。
报道并配发的照片显示，宋平在2022年10月中共全国代表大会开幕式上曾与前国家副主席曾庆红同框。

北京在军事力量和政治决心上已有所强化，但在对台威慑的“可信度”投射方面仍不足。 
北京大学台湾研究院院长、全国人大代表李毅虎表示，未来五年两岸统一进程将进入“加速阶段”，大陆需要更清楚地向外界传达这一进程的必然性及相关后果。李指出：“目前我们在构建能力和展现使用（军事）威慑的决心上做得很好，但还需努力让明面和潜在的对手充分理解威慑带来的后果以及其中的得失。”

伊朗导弹袭击后，阿联酋楼市面临首个实质考验。针对迪拜与阿布扎比的机场、港口和居民区的打击，打破了海湾长期作为避风港的形象，震动投资者，也暴露出两城对离岸资金维持建筑热潮的高度依赖。
3月4日，阿布扎比最大上市开发商Aldar和迪拜掌控市中心及哈利法塔的Emaar股价均下跌约5%，主要开发商债券价格亦大幅回落，债券市场对新发债几近关闭、利差普遍扩大。
据Betterhomes，2025年迪拜交易中有65%为期房，外资需求将成为能否消化这些未建项目的关键。部分开发商如Dar Global仍对基本面乐观，但有投行人士称已搁置本周的筹资计划，称投资者短期内不愿进场，风险溢价上升，若冲突持续，国际放贷收紧或迫使出售资产。
回顾过去十余年，免税、放宽签证与改革吸引富裕外来人口，疫情后楼市加速：Fitch称迪拜房价自2022年至2025年一季度涨约60%，CBRE称2025年四季度住宅同比仍涨约13%，阿布扎比同期涨幅近32%。然而摩根大通曾警告，至2028年新增30万–40万套供应可能超出人口吸纳能力。阿布扎比商业银行分析指出，冲突后外国买家兴趣将至关重要；行业高管与分析师则认为，房地产真正的影响要看冲突结束后需求能否恢复。来源：AFP/路透等。
