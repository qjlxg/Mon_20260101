3月31日，人民党党首 ณัฐพงษ์ เรืองปัญญาวุฒิ（绰号“เท้ง”）在脸书发文称，针对当天反贪委员会（ป.ป.ช.）通过决议，同意向最高法院就与前进党44名前国会议员有关的案件提出诉请——即他们联署提出修订刑法第112条（侮辱君主罪）一事。 
他强调，提出修法是国会议员固有的立法权和重要职能，无论外界是否赞同，议员提出修法不应被视为可遭受严重道德或刑事指控的行为。 
他指出，此事显示掌权精英正动用国家机制维护其利益，通过法律战拖制政治对手；但无论对方如何试图冻结或阻挡，党内人士将继续尽职尽责，哪怕未来最高法院作出不同裁定。 
下一步，党将参加政府施政辩论，并在议会提出解决北部及相近地区PM2.5细颗粒污染的相关提案。 
他呼吁全民共同争取法治与司法公正，确保法律对所有人一视同仁、不被当作政治武器，并建立强有力的制衡与监督机制，让政治归于人民。

印度外交部称，随着西亚冲突升级，已有8名印度公民在多起事件中遇难，1人失踪。外交部海湾事务副秘书Aseem R. Mahajan在新德里一次跨部简报会上于周一通报了最新数字。他还提到，近日科威特发生的一起袭击造成一名印度人死亡；沙特方面也确认一名印度人在3月18日利雅得的事件中遇难。此前3月20日外交部曾报告6死1失踪，此次为最新更新。
外交部表示，自2月28日以来约55万名旅客已从该地区返回印度。发言人Randhir Jaiswal称，印度持续呼吁各方克制、通过对话和外交尽快降温。总理莫迪与沙特王储兼首相穆罕默德·本·萨勒曼通话，双方讨论冲突并一致强调确保该地区航行自由的重要性；莫迪重申印度谴责针对地区能源基础设施的攻击。
