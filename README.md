中东不停火油供不足 日本温泉区忧泡不了热汤
中东局势一天不平息，燃油供应不足给日本各行各业带来的影响扩大。日本地方旅游区也为之焦虑，除担心油价高，开车旅游的人减少外，各地温泉旅馆还因油供不足，无法为温泉保温，游客恐怕泡不了热汤。
由4月29日起，为期10天的黄金周假期是日本旅游业最繁忙时期。但偏远的温泉乡客房预约量不如往年，可能是油价上涨，计划出游的人数减少所致。
此外，受中东战事影响，日本3月煤油价格就明显上涨约10%，4月又上涨20%，日本各地温泉乡正在节约为温泉水加热的燃料。一些温泉旅馆从4月开始，都以计时器进行限量的加热运作。兵库县的濡森之里温泉还因油价过高，成为首个宣布暂时停业的温泉设施。
在寒冷的日本地区，旅馆设施为确保泉水达到一定热度，重油是作为温泉水加热的锅炉燃料。在泉水不够时，也得另外烧开水，确保浴池中有足够的热水。富士山山梨县的庆山温泉旅馆有122间客房。因为规模大，除了使用重油为露天温泉保温，整栋建筑物的暖气设备也使用重油燃料。
旅馆老板千须和昌接受《联合早报》采访时说：“旅馆拥有巨大的室内浴池和使用自家温泉水的露天岩浴池，因此必须用巨大的锅炉加热设备。上个月，因为中东局势导致重油涨价，旅馆多了120万日元（约9598新元）的额外开销。”
他也担心，基于燃料原材料供应不足，温泉旅馆里给客人做小火锅的加热小型固体燃料也断货。“我们旅馆每天得消耗近900个，如果没了货源，恐怕这道料理就得取消。”
日本超过90%的原油进口来自中东，其中大部分原油需经由霍尔木兹海峡运输。自霍尔木兹海峡被封锁后，日本政府已在3月19日开始动用相当于15天的私人储备和30天的国家储备。据估计，这些储备足以满足日本国内消费需求至4月底。
日本首相高市早苗星期五（4月10日）宣布，将在5月初释放约20天的国家石油储备。她说：“我们制定了确保直至年底的石油供应计划。目前，由于供应不平衡和分销渠道受阻，部分地区无法获得充分的燃油供应。为了确保一切井然有序，决定从5月初开始追加发放第二批国储石油。”
《日经新闻》报道，尽管当局正在努力以不通过霍尔木兹海峡的替代路线运输燃油，但中东局势动荡，将导致燃油供应不稳定的情况持续。

阿里巴巴集团在CEO Eddie Wu Yongming 领导下成立高层技术委员会，以加速人工智能研发并应对国内模型与智能代理领域日趋激烈的竞争。

周三，中国人工智能公司智谱AI开源其最新旗舰模型GLM-5.1，并同步将API价格上调10%，显示在与美国对手竞争加剧的背景下，开始加强对高级模型的商业化变现。此次为公司今年第二次涨价：2月对编程订阅的调整曾上调逾30%，本次幅度较小但覆盖包括云端输入token在内的多项计费类别。消息发布后，智谱港股盘中上涨约11.5%。
定价方面，智谱对输入token每百万收费约1.40美元、输出约4.40美元；而Anthropic的Claude Opus 4.6截至2026年2月分别为每百万输入5美元、输出25美元，差距仍在。GLM-5.1上月末先作为付费编程订阅产品推出，此举引发其从开源策略向更强调变现的猜测。
基准测试机构Artificial Analysis认为，按综合智能指标计算，GLM-5.1为全球最强开源模型，领先国内竞争者如MiniMax，但仍落后于OpenAI与Anthropic的顶级模型。智谱在X上称，GLM-5.1可在复杂工作流中持续运行约8小时，能从零构建可用的Linux桌面，反映对可长时自治运行的agent型AI工具的需求。
智谱定位为中国硬件生态中的重要软件合作方，并称已与多家国产芯片企业合作适配其GLM模型，包括华为、摩尔线程与寒武纪。摩尔线程表示其MTT S5000服务器已实现对GLM-5.1的“Day-0”兼容性。
智谱CEO张鹏在3月中关村论坛表示，依赖低价竞争不利行业长期发展，提价旨在建立“更健康的商业闭环”，支持高质量AI服务的长期交付；小米AI模型负责人罗福立也表达了类似看法。

新华社科伦坡4月9日电 中国公安部周四表示，斯里兰卡警方已于3月29日将此前多次收网行动中抓获的125名涉嫌电信诈骗人员遣返回中国，遣返工作得到中国驻斯里兰卡使馆和中国警方协助。
公安部称，此次联合行动是中斯两国深化执法合作、加大打击跨境电信诈骗力度的成果。自今年年初以来，双方针对部分涉诈人员及相关地下产业由他国转移至斯里兰卡的态势加强了协作。
公安部透露，斯里兰卡警方近期多次开展清查行动，摧毁数个诈骗窝点并抓获多名涉案人员。参与此次遣返工作的中国湖北省公安机关正在对相关案件进行侦查。
公安部一名官员表示，将继续加强与更多国家的执法合作，共同构建打击跨国犯罪的务实高效新机制。来源：新华社

深圳报道 — 中国国务院总理李强在周五举行的高层座谈会上，要求加快在制造业推广工业人工智能，推动制造业的数字化与智能化升级。李强在与新兴独角兽Moonshot AI创始人杨志林等企业家和业界专家座谈时指出，要更深入落实“AI+”战略，加快把人工智能嵌入制造环节，强化先进制造与现代服务的融合，增强产业体系整体实力。 
李强并表示，尽管外部环境今年更为复杂多变，但在及时出台积极的宏观政策支持下，经济已开局良好。他还强调要抓住新兴产业发展机遇，扩大就业、提升收入、拉动内需，并鼓励企业家在创新和带动就业方面发挥更大作用。 
受邀参加座谈的还有广发证券首席经济学家郭磊、中国人民大学金融学教授马广荣以及招商局港口控股董事长冯伯明等。

美国联邦通信委员会（FCC）周四表示，正考虑更广泛的措施，禁止中国电信运营商在美运营数据中心，并进一步限制其接入美国网络与基础设施。草案同时建议禁止美国运营商与这些被认为存在“重大国家安全风险”的公司互联互通。中国移动、中国电信和中国联通均被列入受限实体名单，可能被要求出售、转让或关闭在美的数据中心和网络节点，甚至被迫退出美国市场。
分析人士称，此举标志着华盛顿对中国科技企业多年打击的升级，尽管短期内对相关公司财务影响有限。塔霍塔律师事务所合伙人王宇祥表示，草案将管制从禁止向公众直接提供服务和限制硬件，进一步延伸到对底层互联网基础设施和互联协议的更深层控制。
