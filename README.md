中东局势升级 本周油价料再攀升
迪拜报道：受中东地缘政治紧张和供应风险扩散影响，国际油价本周预计进一步上扬。布伦特原油一度高于112美元/桶，WTI接近100美元，较冲突前约70美元水平已有大幅回升。
冲突进入第29天，事态出现区域溢出：伊朗发射导弹与无人机袭击多处目标，包括阿布扎比Khalifa经济区（KEZAD）；胡塞武装首次向以色列发射导弹，以军加强对黎巴嫩和贝鲁特的打击，阿曼等地港口与基础设施亦有干扰报道，增加航运与物流枢纽安全忧虑。
外交斡旋短期未能抑制涨势：特朗普将霍尔木兹海峡通行最后期限延至4月6日，虽曾短暂缓和市场，但因交战持续且美伊口径不一，油价很快回升。分析人士指出，市场越来越不信口头声明，只有伊朗方面确认实质进展才可能影响价格。
多家机构认为，油价守住关键心理关口反映基本面偏强；对航运、工业区和港口的攻击，以及缺乏可信的降级路径，可能在短期内继续推高油价，除非地区出现明确且可信的缓和迹象。

卡塔尔教育和高等教育部宣布，公私立学校已做好全面准备，将于周日恢复线下授课，近41.7万名学生返校。部门称已完成各项必要安排，确保返校顺利且安全。
总务处对校舍和设施进行了预防性维护，并核查运行与技术系统，督促服务供应商达到既定安全标准，为师生与行政人员创造稳定环境。公共关系与传播部主任玛丽亚姆·阿卜杜拉·阿尔·莫汉纳迪表示，此次准备体现了多部门协调和与国家有关当局的持续合作，目的是增强校内信心并为学生顺利返校提供保障。高层监控将持续，以便快速应对任何事态发展，维护教育秩序的安全与稳定。
为提升应急能力，民防总局在军官俱乐部为约1,200名校长、副校长及安全官员举办了为期四天的应急演练，内容涵盖疏散、避难和国家预警系统响应等。培训与发展中心上周四举办“Irtiqa 3”论坛，约9,700名学校领导、教师与协调员在线视频参加，聚焦领导力、心理韧性与运作组织。
卫生与安全部下发了更新的应急与突发事件安全指引，标准化保护措施。学校与学生事务部与行政副职及心理专家会商，制定迎接学生返校的心理支持与适应策略。私立学校与幼儿园也落实了安全、人员稳定与教学连续性措施，依据国家指引保障正常教学。
教育部强调，家长是确保学生出勤、遵守校车与校内安全规范以及推广防护意识的重要合作者，这对提升全校安全水平和顺利开学至关重要。

阿曼一名苏丹籍教师在伊布里（Wilayat of Ibri）瓦迪·阿尔艾因（Wadi Al Ain）内一栋建筑在洪水中倒塌后遇难。此次由山谷（wadi）泛滥引发的天气低压已使全国死亡人数上升至10人。警方航空部门在搜救队从现场救出遇难者后将遗体运走。 
此前，在达赫利耶省伊兹基（Izki）一处瓦迪，一名儿童被急流冲走后经搜救队发现遇难。搜救部门表示，紧急小组已被部署并完成相关打捞与搜救工作。

Response Plus Holding（RPM）公布2025年业绩：收入增长13%至5.16亿迪拉姆，较2024年的4.55亿迪拉姆上升。年内净利润为4,996万迪拉姆，总资产由3.759亿升至3.8722亿迪拉姆，股东权益增至2.458亿迪拉姆，每股收益为0.25迪拉姆。
公司表示，业绩增长主要受地理与业务拓展推动。2025年RPM在挪威奥斯陆、沙特塔布克与吉达开设新办事处，并在巴哈马、哈布山和拉斯海玛等地扩大业务，同时参与巴哈马机场应急医疗服务基础设施建设。
董事长Omran Al Khoori称，成绩印证了集团扩张策略的成效，公司正稳固在紧急医疗服务领域的地位并扩大国际布局。首席执行官Dr Rohil Raghavan表示，公司将继续多元化收入来源、强化急救服务、培训与咨询能力，并计划于2026年推出一体化医疗空运撤离及与ICATT Air Ambulance合作的商业航班医疗护送等新业务。
关键进展还包括将航空应急业务拓展至巴哈马、与Falcon Aviation签署在阿联酋运营空中救护协议；2025年哈季期间，公司在沙特铁路站部署350名医务人员和125辆救护车。旗下主要子公司Response Plus Medical在油气等行业运营逾420家诊所，业务覆盖阿联酋、沙特、阿曼、约旦、印度、英国、瑞士、挪威和巴哈马等市场，未来将继续扩大国际版图并发展新服务以应对医疗需求增长。
