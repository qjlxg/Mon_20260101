香港录得74年来最大贸易逆差 或是中东战争和AI热潮影响
香港因进口激增，录得至少74年来最大的贸易逆差，这很可能是中东战争引发的干扰，以及全球人工智能（AI）投资热潮推动科技产品需求所带来的外溢效应。
综合香港文汇网、《香港经济日报》、彭博社报道，香港特区政府统计处星期二（4月28日）发表的对外商品贸易统计数字显示，香港今年3月商品整体出口货值为6184亿元（港元，下同，1008.43亿新元），较2025年同月上升35.8%。同时，今年3月商品进口货值为7075亿元，较去年同月上升41.2%。这使贸易录得891亿元的逆差，为自1952年有记录以来的最高水平。
香港这一转运枢纽是许多运往中国大陆货物的门户。由于企业囤积半导体及其他物资，大陆3月高科技产品进口大幅增加。香港几乎全部能源依赖进口，而且发电主要依靠化石燃料，因此容易受到伊朗战争引发的油价冲击影响。
