中国证监会主席：坚决打击操纵市场行为
中国证监会主席吴清指出，将持续强化交易监管机制，坚决打击操纵市场等行为，并呼吁业者为投资者创造更可持续的中长期收益，不要走回“冲规模、赚快钱”的老路。

他还围绕加快建设一流投资机构提出，要坚持规范为先，不断夯实合规风控内在基础；坚持客户为本，始终注重投资者利益优先；坚持功能为要，更好发挥培育新质生产力作用；坚持创新为基，持续增强行业高质量发展内生动力。

吴清认为，注重权益投资是行业服务投资者的关键所在，要下更大力气加强投研专业能力建设，强化权益投资的行业影响力和竞争力；要加强供需适配，推出更多适配中长期资金的投资产品，更好对接不同群体的差异化需求，同时加快推进向投资者回报导向的买方投顾转型，不断提升投资者满意度。
中国国务院办公厅星期五（6月5日）发布关于加强监管防范风险、促进私募投资基金高质量发展的指导意见，在加强政府投资基金和国有企业投资基金管理等方面进行具体安排。

伊波拉防控计划启动 刚果（金）仍现社区传播
（日内瓦综合电）世界卫生组织和非洲疾病预防控制中心周五（6月5日）宣布，在未来六个月投入5亿1800万美元（约6亿6716万新元），以应对刚果（金）和邻国暴发的伊波拉疫情。
这项疫情防控计划将从6月执行至11月，重点包括应急协调、监测、实验室检测、感染预防和控制，以及临床护理和社区参与。
世卫总干事谭德赛说，这项计划的目标很明确：阻止疫情在现有地区蔓延、支持正在采取应对措施的国家，同时确保邻国做好准备，能在出现病例时迅速采取行动。
刚果（金）卫生部周五说，当地确诊伊波拉病例增至452起，当中82人已病亡，目前疫情仍在社区内快速且持续传播。乌干达同日通报三起新病例，使累积确诊病例增至19起。
非洲疾控中心主任卡塞亚说，现有的援助承诺已能满足疫情防控计划的大部分需求，但这些承诺尚未转化为实际资金。
另一方面，美国周五宣布追加拨款3800万美元，用于应对伊波拉疫情。这使美国针对这波疫情的直接资助增至逾2亿美元。
美国疾病控制与预防中心（CDC）进行的情景模拟显示，如果没有强有力的公共卫生干预措施，刚果（金）的疫情可能会与2014至2016年的西非疫情规模相当，甚至更加严重。那次疫情造成超过2万8000人确诊感染，1万1000多人死亡。
CDC强调，这不是对疫情走势的预测，而仅是一个规划工具；如果能识别、隔离和治疗更多患者，就能避免最糟糕的后果。

业内：相关政策研究和制定要精细化
近段时间以来，加大购房补贴力度成为多地楼市政策的新着力点。中指研究院的数据显示，今年前5个月全国各省市（县）出台政策超430条。从政策方向上看，涉及优化限制性政策、调整公积金贷款、发放购房补贴、完善城市更新相关配套政策等内容。多位购房者表示，期待更多降低购房压力的新政策，包括房贷利率和交易税费能够进一步下调、发放购房补贴等。广东省住房政策研究中心首席研究员李宇嘉提出，购房补贴政策可能存在一些问题。例如补贴可能导致后续的需求提前释放，而不是产生增量需求，导致补贴一停，销售态势就恢复到常态。如果补贴一直延续下去，暂不论地方财力支持。即便有财政实力，但对于市场来说，就认为是新常态，自然也不会因为有补贴而加速购房。此外，补贴还要选择时机，也要精细化研究，哪些人群对补贴政策比较敏感，有针对性设计补贴发放范围。“总之，政策研究和制定要精细化，将宝贵的财政资金用在刀刃上。”（证券时报）

此前审批通常很快，官方建议至少提前三天申请。但仍有旅客在出发当天才申请，近期因此滞留。例如：居住纽约的Zara Rahim与伴侣在6月2日晚到肯尼迪机场赶赴Delta航班赴伦敦参加婚礼，连续等待近六小时未获批；荷兰莱顿大学教授Jonathan London和巴黎文化策划人Ella Marder分别在乘坐Eurostar和从巴黎出发的当日遭遇未能及时获批、无法登车的情况。多位旅客反映车站内约有百余人陷入同样境地。
内政部未就故障何时开始或何时恢复作出具体说明。6月3日下午，有乘客陆续收到获批通知。

刚果（金）卫生部称，本周东部南基伍省发生针对埃博拉遗体安葬队的袭击，迫使救援人员弃置一具棺材，增加了病毒进一步传播的风险。袭击发生在6月1日卡塔纳镇，该地约距省会布卡武北约30公里，目前由AFC/M23叛军控制。
被袭的是“安全与体面下葬”专门队伍，按严格防疫规范处理高度传染性遗体。卫生部发布的6月3日情况报告称，棺材随后被社区成员处理，此类高风险做法可能引发新的传染链。卫生部与当地医院负责人未说明袭击动机。
当局表示，袭击凸显出持续的猜忌与抵抗，阻碍疫情防控。近期安葬队和医护人员多遭袭击，部分为质疑死因的遇难者家属所为。同日（6月1日）在伊图里省首府布尼亚也发生类似事件，一支应对队在公墓遭殴，至少4人受伤。
疫情方面，自5月15日宣布暴发以来，这是该国第17次爆发，确诊病例已达363例，死亡62例。卫生部最新通报新增19例确诊（含2例死亡），感染已扩散至伊图里省36个卫生区中的17个，并在北基伍省7个卫生区和南基伍省1个卫生区出现病例。
尽管形势严峻，卫生部也报告了积极进展：伊图里省Rwampara的32名密切接触者完成21天监测，未发现埃博拉感染；北基伍省戈马市一名康复患者6月3日准备出院，AFC/M23副协调员弗雷迪·卡尼基在社交平台X上表示该患者已与家人团聚。 （路透）

停火岌岌可危：海湾关键航道霍尔木兹海峡再起冲突，阿联酋称遭伊朗袭击
德黑兰/阿布扎比 — 5月4日，伊朗与美国在霍尔木兹海峡再次交火，几乎一个月来的停火面临瓦解；阿联酋报告自停火以来首次遭到导弹和无人机攻击。
美国总统特朗普此前宣布一项“自由计划”以护送受困船只通过海峡，并在社交平台发表强硬言论；美国一名海军将领称美军击沉六艘小型伊朗船只。伊朗否认军舰受创，称美军袭击造成五名平民丧生，要求追责。
阿联酋称遭来自伊朗的弹道/巡航导弹与无人机攻击，富查伊拉一处能源设施受创，三名印度人受伤；阿曼布卡一处民宅也有两人受伤。阿联酋外交部警告这是危险升级，威胁国家安全。
美以两国国内也出现呼吁加大对伊朗打击之声。美中央司令部司令和美国驻联国大使均回避是否认定停火被违反的问题。
油价应声上涨，7月交割布伦特原油一度涨逾5%。阿联酋宣布学校改为线上授课至本周末，以色列军方进入高度戒备状态。
阿联酋称四枚巡航导弹由伊朗发射，三枚被拦截；另称伊朗无人机袭向与阿布扎比國營石油公司有关的油轮。伊朗方面将事态归咎于美军在海峡的行动，但伊朗外长阿拉格奇呼吁政治解决，警告不要被好战势力拖入泥潭。
特朗普表示将通过“自由计划”护送中立船只并已派驱逐舰经过海峡，称已有两艘挂美旗商船驶出；但伊朗革命卫队否认近期有商船通过。韩国方面称一艘韩国船只在海峡发生爆炸起火。
此外，停火也在黎巴嫩战线承压；以色列轰炸与地面行动已致黎巴嫩大量伤亡，双方在南部继续交火，黎巴嫩总统要求在会见以色列总理前达成安全协议。

伦敦报道—总统唐纳德·特朗普威胁对被指未能遏制强迫劳动的贸易伙伴征收新关税，但专家、商界及部分人权组织认为此举难以遏制现代奴役，甚至可能适得其反。
美国贸易代表办公室提议对60国进口商品加征10%或12.5%关税，理由是这些国家未能阻止强迫劳动的商品贸易。新加坡贸易与工业部6月4日表示拒绝美方指控，并称三分之一对美出口或遭12.5%关税。
此举源于301条不公平贸易调查，旨在恢复今年2月被美国最高法院推翻的紧急关税。国际劳工组织最新估计显示，全球被强迫劳动者达2760万，比2016年增加约270万，近半数集中在制造、建筑、农业、渔业和采矿等出口相关行业。
批评者指出，美方措施更以贸易规模与地缘政治为导向，而非按剥削严重性精准制裁，未必能改善情况，反而可能激化部分国家的政治抵抗。欧盟反驳称其将于2027年生效的强迫劳动法规程序更为严格，覆盖范围广，且与美国在2025年达成的贸易协定将大多数欧盟商品对美关税上限定为15%。国际商会与人权组织也对关税的任意性与有限效果表示担忧。
