中国拟推动发展太空旅游 与美国竞逐
中国主要航天承包商计划在未来五年推进太空旅游发展，这意味着在与美国展开激烈技术竞赛的背景下，北京正加快布局商业航天和深空探索。
据央视新闻报道，中国航天科技集团星期四（1月29日）指出，“十五五”时期（2026年至2030年）将统筹推进太空旅游、太空数智基础设施、太空资源开发，以及太空交通管理等新兴领域。
在太空旅游方面，中国航天科技集团称，将加快迭代形成亚轨道和轨道太空旅游飞行器产品，完成相关无人或有人飞行验证，建立完善的太空旅游运营体系，实现亚轨道太空旅游航班化运营，并逐步发展轨道太空旅游。
随着中美竞相将太空探索打造为类似民用航空的可商业化产业，并争夺由此带来的军事和战略优势，双方在太空领域的竞争日益升温。中国航天科技集团此前曾表示，计划到2045年将中国全面建成航天强国。
目前，北京面临的关键瓶颈之一是可重复使用火箭尚未完成验证，而这一技术被普遍视为降低火箭发射成本、压缩卫星入轨费用的核心环节。
在这一领域，美国太空探索技术公司SpaceX已明显领先；公司的“猎鹰9号”可重复使用火箭已广泛投入商业运行，使旗下星链（Starlink）在近地轨道卫星领域几乎形成垄断，并被用于轨道太空旅游任务。
SpaceX还曾披露，计划使用正在研发的星舰，将乘客送往地月空间甚至更远区域，体验飞越月球并观赏“地出”等深空景象。
面对SpaceX在近地轨道领域迅速扩张的态势，中国正加快推进自主卫星星座建设，目标是在未来数十年内形成规模达数万颗的卫星网络。
去年12月底，中国相关机构向国际电信联盟提交文件，规划在未来14年内将约20万颗卫星送入轨道。其中，两大“超级星座”占据绝大部分，也被视为北京为自身长期航天发展战略性锁定轨位和频段的重要举措。

美媒：中国收紧互认基金跨境投资计划
美国媒体报道，中国已收紧互认基金跨境投资计划，限制全球管理公司为中国大陆客户分配资金的地点，此举可能会抑制资金流入美国等热门市场。
据彭博社报道，上述限制措施针对的是规模达860亿美元（1085亿新元）的互认基金计划，而这项计划允许中国大陆投资者投资香港基金，反之亦然。
不愿具名的知情人士称，此举旨在保护自去年以来涌入这项计划的散户投资者，而且只有新成立的基金将受到影响。
彭博社引述文件内容报道，中国证监会在向香港基金项目参与者发布的2025年下半年指引中说，将加强相关要求，确保投资组合根据散户投资者的风险承受能力，在资产类别和地域上实现多元化。知情人士称，此前并无此规定。
知情人士称，中国证监会也通过口头沟通告诉申请人，任何单一国家或地区的基础基金占比，均不得超过总额的50%，但香港不受到影响。知情人士补充称，基金中高收益证券的比例也有限制。这些规定不适用于现有产品。
彭博社认为，此举暗示中国大陆现在正寻求对这类投资者在海外的敞口有更多控制权，而这可能会减缓全球资产管理公司参与这项计划的速度。
中国证监会在2015年宣布，与香港证监会决定开展中国大陆与香港基金互认工作，并发布《香港互认基金管理暂行规定》。
中国证监会之后修订发布并在2025年1月实施《香港互认基金管理规定》，将香港互认基金客地销售比例限制由50%放宽至80%。彭博社报道，这导致大陆投资者希望投资在香港产品的需求激增。
