【伊朗官员：已准备好与美谈判方案 但对结果不乐观】金十数据2月15日讯，当地时间15日，伊朗议会国家安全与外交政策委员会发言人易卜拉欣·雷扎伊表示，在当前的谈判中，没有讨论伊朗停止或放弃铀浓缩活动，也没有讨论将伊朗的核材料运离该国，这些议题美国方面已经接受。雷扎伊表示，日内瓦谈判与导弹问题或地区问题无关。伊朗认为，以色列才是该地区的主要问题。以色列的威胁应该被排除在日内瓦谈判之外，并以其他形式与该地区各国讨论，以确保地区国家的发展、和平和共同利益。雷扎伊强调，伊朗谈判团队已准备好谈判方案，“但考虑到美国的过往记录，我们对谈判结果并不十分乐观。”

【春节假期第一天全国道路交通平稳有序】金十数据2月15日讯，今天是春节假期第一天，也是节前最后一次返乡小高峰，主干公路交通流量环比明显上升。截至18时，除个别路段因交通大流量导致缓行外，全国道路交通总体平稳有序，未接报严重交通拥堵和较大以上道路交通事故。
