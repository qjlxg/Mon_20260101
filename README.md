中国严管侮辱企业家言论 热搜不得呈现企业家“口水战”
中国持续以优化营商网络环境的名义整治涉企负面信息，官方指导网络平台制定自律公约，明确严管侮辱谩骂、造谣诽谤企业家的言论，热搜热榜不得呈现人工智能（AI）生成的涉企负面信息和企业家“口水战”。
中国网信办官网星期五（6月12日）公布，在网信办指导下，中国重点网站平台共同制定《整治涉企侵权信息优化营商网络环境自律公约》。
公约共计10条，在清除侵犯个人权益信息方面，明确及时清理泄露企业家个人隐私的信息，严管侮辱谩骂、造谣诽谤企业家的言论，也提出统一同类信息处置标准，主动清理已核实的涉企虚假不实信息。
在“严厉打击各类网络水军”方面，公约要求清理机器人账号涉企评论信息，坚决处置发布统一话术、同质化涉企信息的机器人账号、水军账号。在“加强榜单涉企话题管理”上，公约提出加大热搜热榜管理，不得呈现AI生成的涉企负面信息，不得呈现企业家“口水战”、企业家私生活以及未经核实的涉企负面热点事件等话题。
公约也提到“持续优化算法推荐机制”，要求不得推荐集纳企业、行业和企业家负面信息的自媒体账号，对AI生成的涉企负面信息予以限流。
公约还鼓励相关主体对外发声，采纳企业、企业家在超大平台上的官方账号发布的公开声明，及时清理与声明不符的信息。
在“完善平台流量投放机制”方面，公约提出强化投流审核管控措施，坚决杜绝涉企负面信息“投流”行为。此外，在“从严管理非法牟利行为”方面，要关闭假冒仿冒新闻单位账号，坚决处置向企业发送“负面信息求证函”的自媒体账号，以及以各种名义收取费用并开展所谓涉企监督的账号。
公约还涉及“取消相关账号营利权限”和“加强涉事账号处置力度”，明确要取消经常性发布涉企负面信息自媒体账号的营利权限，包括但不限于流量分成、广告收益、推广商品和直播打赏；坚持“信息、账号一同处置”原则，坚决处置发布传播侵权信息的账号，对账号所属网红经济公司实施联动处置。
中国国家网信办有关负责人称，《公约》的制定发布有利于强化行业自律，加强涉企侵权信息治理，进一步优化营商网络环境，保护企业和企业家网络合法权益。

高盛集团将其2027年布伦特原油价格预测下调5美元至每桶80美元，理由是供应增加和需求减少。高盛分析师称，鉴于阿联酋退出OPEC，该行预计明年来自该国以及美洲的供应将会增加。霍尔木兹海峡“重新开放后，需求很可能大幅回升”，但该行预计，“随着中国加速转向替代能源（如电动汽车），需求疲软将持续，占比略高于10%。”高盛目前预计波斯湾产油国的石油出口将在8月 底前正常化，而此前预计为6月。该报告预计今年第二季度全球原油供应缺口将达到500万至600万桶\/日，如果没有冲突，预计需求将减少约500万桶\/日 ，而供应过剩将超过400万桶\/日。

北京/巴黎，6月11日 — 中国国务院副总理张国清将出席由法国总统马克龙周四主持、围绕全球经济失衡展开的视频会议，此举距七国集团（G7）在法国召开峰会仅数日。此次G7峰会将讨论如何应对大量低价中国产品冲击欧洲市场。
马克龙在会议开场表示，目标是“让全球经济重回更强劲的增长轨道”，实现这一目标既需国内政策，也需有效的国际合作；协调至关重要，否则失衡可能以无序方式释放，带来突发的经济金融调整。欧盟领导人将在6月17日G7峰会后立即开会，届时中国将是议题焦点。
中国外交部发言人宣布，张国清将参加被称为“全球增长汇聚”的视频会。这在中国与由法、英、加、德、意、日、美及欧盟组成的G7之间属罕见互动。长期以来，北京对G7讨论全球事务的代表性提出批评。
欧洲对中国长期贸易顺差及其在电动汽车、锂电池等高端领域的崛起日益担忧，分析称这可能构成继2000年代低端产业冲击后的“第二次中国冲击”。中方则为其产业政策辩护，否认出口商靠国家补贴获不当利益，并指责个别国家以单边关税破坏全球贸易规则。
自去年12月以来，北京已先后接待五个G7国家领导人。马克龙在当月会见中国国家主席习近平时曾敦促中方合作实现经贸再平衡，否则欧洲将不得不采取更具保护性的措施。欧盟内部对如何应对中国仍存在分歧，德国虽传统上对在主要出口市场征关税持谨慎态度，但随着中国车企与德国制造商竞争加剧，部分德国议员开始呼吁采取更强硬立场。 路透
