湖北省委省政府联合调查组星期五（2月13日）通报，襄阳襄雅精神病医院等10家精神卫生医疗机构涉嫌存在违规诊疗骗取医保基金；襄阳安康精神病医院等六家医院存在以减免费用诱导招揽患者；宜昌夷陵康宁精神病医院存在患者假出院、连续住院；襄阳宏安精神病医院等三家精神卫生医疗机构存在少数工作人员殴打精神障碍患者等违法违规问题。
根据发布在“湖北发布”微信公众号的通报，调查还发现存在属地管理、行业管理、部门监管失职失责问题；个别公职人员存在权钱交易、利益输送等违纪违法问题。
湖北省卫健委也组织省级专家组，对襄阳、宜昌共51家精神卫生医疗机构进行了全面排查，没有发现收治非精神障碍患者入院情况，但存在临床治愈后继续住院、空挂床位等问题。
襄阳、宜昌公安机关已对14人以虚构医药服务项目、虚开医疗服务费用、伪造医学文书等方式骗取医保基金，涉嫌诈骗罪，立案侦查并采取刑事拘留措施；对一人以涉嫌隐匿会计凭证罪立案侦查并采取刑事拘留措施；对殴打他人的四人作出行政处罚；对违法招用保安的宜昌夷陵康宁精神病医院，责令限期改正并处罚款。
调查组也说，卫生健康、医保行政部门组织对10家涉案精神卫生医疗机构严肃处置。指导督导两地妥善安置在院患者，确保患者得到规范治疗、权益得到有效保护。
调查发现，襄阳、宜昌两地卫生健康、医保等行政部门个别工作人员利用职务上的便利，为襄阳阳一光精神病医院、枝江康宁精神病医院等精神卫生医疗机构在行政审批、医保结算等方面提供帮助，并非法收受财物、违规接受宴请，涉嫌严重违纪违法。纪检监察机关已对九名公职人员涉嫌受贿、滥用职权等问题立案调查并采取监察强制措施，对四名党员干部涉嫌违反中央八项规定精神问题立案审查调查。
