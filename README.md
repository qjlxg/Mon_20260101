台美达成贸易协议削减台湾产品关税
美国和台湾星期四（1月15日）达成贸易协议，削减针对多个台湾产品的关税，引导新投资流向美国科技产业。
路透社报道，根据协议，对于扩大在美生产规模的台湾晶片制造商，如台积电，进口到美国的半导体产品将获得较低税率。美国也将把适用于大多数其他台湾对美出口产品的关税税率，从20%降至15%。
美国商务部说，仿制药、飞机零部件和稀缺自然资源，将享有零百分比关税。
台积电等台湾科技公司，则将在美国进行投资，总值至少为2500亿美元（3220亿新元），用于提升半导体、能源和人工智能领域的产能，作为交换条件。美国商务部长卢特尼克说，这其中包括台积电在2025年承诺的1000亿美元投资事宜，未来还有更多投资事宜。
台湾行政院在官网上说，由行政院副院长郑丽君、行政院经贸谈判办公室总谈判代表杨珍妮所率领的台湾谈判团队，于美东时间1月15日与卢特尼克、美国贸易代表格里尔率领的美国谈判团队进行总结会议，达成包括台湾对等关税调降为15%且不叠加MFN（最惠国待遇）、半导体及半导体衍生品等232关税取得最优惠待遇、扩大供应链投资合作、深化台美人工智能（AI）战略伙伴关系等多项谈判预定目标，并共同见证驻美国台北经济文化代表处、美国在台协会在美国商务部共同签署投资合作备忘录。
行政院引述台美经贸工作小组的说法称，由于台湾是美国第六大贸易逆差地区，逆差中有高达九成来自半导体、资通讯产品、电子零组件等，涉及美国232条款调查，因此台湾在谈判中聚焦对等关税并同232关税与美国贸易代表署及商务部进行多轮磋商，并在这次顺利达成谈判预定的四项目标。
行政院续称，其中一项为对等关税调降为15%，且不叠加原MFN税率，获得美国主要逆差国中“最优惠盟国待遇”，与日、韩、欧盟齐平。
至于台美间涉及关税、非关税贸易障碍、贸易便捷化、经济安全、保障劳动权益及环境保护、扩大采购等重要内容的台美贸易协议，工作小组说，因双方仍在进行法律检视中，台湾将另择期与美国贸易代表署进行文件签署，待协议签署后将详细对外说明，并于未来依法定程序将完整协议文本送交议会审议。

美国六大银行去年裁员逾万人 写近10年新高
美国大型银行去年合计裁减逾万名员工，裁员人数写近10年来新高。面对成本压力，管理层开始从人力成本这一主要支出项下手。
彭博社报道，数据显示，美国六大银行，摩根大通、美国银行、花旗集团、富国银行、高盛集团以及摩根士丹利，在截至12月底的员工总数约为109万人，较一年前减少约1万零600人，降至2021年以来最低水平。上一次出现如此大规模裁员是在2016年，当时员工人数同比减少约2万2000人。
眼下，华尔街的关键词是“效率”。在冠病情期间，华尔街的并购交易十分火爆，银行大幅扩招；但自2022年起交易活动骤降，裁员也随之而来。近期，讨论的焦点进一步演变为：人工智能（AI）将如何取代部分人力。
去年裁员幅度最大的银行是富国银行。首席执行长沙夫持续推进公司重整。截至年末，富国银行员工数较一年前减少逾1万2000人，降至约20万人，为2008年金融危机期间收购Wachovia之前以来的最低水平。
