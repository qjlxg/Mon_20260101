她强调，透过这次供应链合作磋商，国际社会更清楚看见台湾的实力与价值。政府未来将持续扩大内部投资、强化产业根基，同时支持产业进行有利的国际布局，让台湾以更坚实的自信走向世界。

中国一组织吁日本早日返还唐鸿胪井碑
中国一个组织呼吁日本政府，早日返还被指非法运至日本的唐鸿胪井碑。
据新华社报道，由上海大学中国海外文物研究中心等编纂的新书《唐鸿胪井碑档案文献总汇》，星期五（1月16日）在上海发布。新书系统梳理了迄今存世的所有唐鸿胪井碑在旅顺原址及日本的图片、唐鸿胪井碑题刻拓片、唐鸿胪井碑自诞生至被掠入日本期间的相关档案文献资料，以及20世纪以来中外学者对于唐鸿胪井碑的研究著述。
报道称，唐鸿胪井碑立于公元714年，是唐朝中央政府特使奉命册封中国东北地区少数民族靺鞨族地方政权首领，确立该地区隶属于唐王朝的重要历史物证。1908年4月底前，日军驻旅顺镇守府将唐鸿胪井碑及1896年增建的用于保护唐鸿胪井碑的碑亭拆解装船，非法运至日本。

上海大学中共党委副书记，中国海外文物研究中心主任，《唐鸿胪井碑档案文献总汇》主编之一段勇，星期天（18日）接受澎湃新闻访问时说，他十年前即称唐鸿胪井碑为头号流失国宝，“自然也应该成为头号追索目标”。

中国对欧“魅力攻势”：绕开布鲁塞尔争取缓和贸易压力
本周，中国外交部副部长华春莹在布拉格将一只装饰瓷盘赠予捷克资深议员，笑容可掬的画面被解读为中捷关系可能出现缓和迹象，也引发欧洲舆论关注。此类外交细节被视为北京对欧洲采取更积极接触的一部分，旨在拉近与个别首都的关系。
在与欧盟整体关系持续趋冷的背景下，北京在年初数周内对若干欧盟成员国展开拉拢：对法国出现飞机订单、恢复对爱尔兰和西班牙的牛猪肉市场准入、以及在签证和象征性礼遇上的让步等，都显示出明显的双边外交优先策略。
分析人士认为，中国此举意在绕过布鲁塞尔层面，直接与成员国建立更密切的经贸与政治联系，以在欧盟委员会即将采取的更强硬对华贸易行动之前，削弱其统一立场并争取分化空间。

肯尼亚与中国达成一项“早收成”初步贸易框架，给予咖啡、茶、切花等主要商品零关税进入中国市场。肯尼亚贸工部周四宣布，该框架覆盖肯尼亚98.2%的出口，谈判于12月19日结束后不到一个月达成。官方称双方将继续谈判，争取最终实现100%的市场准入并签署全面经贸伙伴协定。
肯尼亚贸易部长李·金延久表示，此举“具有里程碑意义”，将为以农业为主的出口商释放巨大潜力。外交部常务秘书科里尔·辛戈伊否认该协议因美方施压而推迟，称两国已交换关税表并就原产地规则达成一致，早收成框架为最终协定提供过渡性优惠。
分析指出，肯尼亚自2014年晋升为中等收入国后未能享受中国对部分非洲国家的免税待遇，导致与乌干达、坦桑尼亚、卢旺达等邻国的竞争劣势。2023年肯尼亚对华出口约2.25亿美元，进口约59.7亿美元，贸易逆差明显。此协议有望推动咖啡、茶、鳄梨、夏威夷果、海产和鲜切花等农产品对华出口。
另有报道说，美国曾以恢复《非洲增长与机会法案》（AGOA）优惠为条件对肯尼亚施压。美国众议院已投票将AGOA延长三年至2028年底，法案现待参议院与总统批准。
