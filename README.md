美国2025年吊销逾10万份非移民签证
美国国务院说，自总统特朗普去年1月上任以来，美国已吊销超过10万份非移民签证，刷新纪录。 
路透社报道，美国国务院星期一（1月12日）在社媒X平台说，2025年被吊销签证包括约8000份学生签证和2500份技术人员签证。被吊销签证的学生和技术人员都在执法部门有犯罪记录。 
国务院称，这是特朗普政府推行强硬移民政策的又一例证，并指出将继续“驱逐暴徒以保障美国的安全”。
国务院副发言人皮戈特说，签证吊销的四大原因是逾期居留、酒驾、袭击、盗窃。与2024年相比，签证吊销数量大幅增加150%。
皮戈特说，国务院还成立一个持续审查中心，旨在确保“所有在美国境内的外国公民遵守我们的法律，并迅速吊销那些对美国公民构成威胁者的签证”。
2025年8月，特朗普政府宣布对所有持有效美国签证的外国人展开审查，一旦查出存在违规行为，将采取吊销签证或驱逐出境措施。

日官方：中国海警船巡航常态化 令尖阁诸岛局势严峻
日本第11管区海上保安总部（那霸）总部长坂本诚志郎称，中国海警船在有主权争议的尖阁诸岛（中国称钓鱼岛）周边海域巡航已呈常态化，令当地局势“极其严峻”。
坂本诚志郎星期一（1月12日）接受日本共同社访问时，透露上述信息。
他说，日本海上保安厅巡逻船的重装化和强硬应对，将“给中方行动升级的机会，无益于尖阁诸岛维持稳定的管理”。
日本海上安保厅统计，中国海警船2025年在尖阁诸岛周边航行天数达357天，创日本政府2012年“国有化”尖阁诸岛以来的历年新高。
据悉，只要不碰上影响航行的恶劣天气，中国船只就持续在尖阁诸岛周边海域活动。
坂本诚志郎分析，自2018年归入中央军委指挥的人民武装警察部队旗下以来，中国海警局的“操船技术有所提升，管理也比过去有所加强”，并称“日本巡逻船每天都（尖阁诸岛现场）与中方船只对峙”。
去年5月，一架直升机从进入日本领海的中国海警船起飞，并进入日本领空。坂本诚志郎称，“日本巡逻船在现场通过无线电警告直升机离开，但在现有框架下，日本海上保安厅能做的仅止于此”。
据悉即使在恶劣天气时海警船进行躲避时，巡逻船仍持续警戒。坂本表示“职员非常辛苦，但绝不能（给中方）可乘之机”，强调了海保任务的重要性。

美国佛罗里达州共和党籍众议员法恩提出一项法案，旨在授权总统特朗普采取一切必要措施吞并格陵兰岛，并最终使格陵兰岛“正式成为美国的一个州”。 
美国《国会山报》报道，法恩（Randy Fine）星期一（1月12日）在一份声明中声称，提案“旨在确保美国在北极的国家安全战略利益”。“格陵兰岛并非我们可以忽视的遥远前哨——它是至关重要的国家安全资产……谁控制了格陵兰岛，谁就控制了北极关键航道和保护美国的安全架构。”
这项法案名为《格陵兰岛吞并和建州法案》，授权特朗普总统采取一切必要措施，包括与丹麦谈判达成协议，使格陵兰岛成为美国领土。法案还要求政府吞并格陵兰岛后，向国会报告“建州”所需的任何措施以加快国会批准。

格陵兰岛自治政府同日表示，将加大力度确保北极地区的防御工作在北约框架下进行，并再次拒绝特朗普吞并格陵兰岛的企图。
格陵兰岛自治政府发布声明说，格陵兰岛是丹麦王国的一部分，也是北约的一部分，格陵兰岛的防务应通过北约来实现。格陵兰岛自治政府绝不接受美国接管格陵兰岛。
美国白宫新闻秘书莱维特12日说，美国接管格陵兰岛没有“时间表”。
格陵兰岛位于北美洲东北方，是世界第一大岛，也是丹麦自治领地，有高度自治权，国防和外交事务由丹麦政府掌管。美国目前在格陵兰岛设有一处军事基地。特朗普2025年上任以来多次扬言要得到格陵兰岛，并声称不排除动用武力的可能性。

中国机电商会：中欧电动车关税谈判达成协议鼓励企业争取权益
中欧星期一（1月12日）就电动汽车关税谈判达成协议，将以最低售价替代惩罚性关税，中国机电产品进出口商会对此回应时称，中欧经过多轮磋商，成功推动该案实现“软着陆”，也鼓励企业争取自身对欧出口权益。
中国机电产品进出口商会（简称中国机电商会）星期一通过微信公众号发布声明成，欧委会发布《关于提交价格承诺申请的指导文件》（简称指导文件），中国电动汽车企业可依据指导文件内容提交价格承诺申请。
欧委会承诺，将本着非歧视原则，采用统一标准对中国企业提交的申请进行客观、公正审查，符合条件的企业可用价格承诺替代反补贴征税。
中国机电商会认为，妥善解决欧盟对华电动汽车反补贴案，是中欧电动汽车上下游业界的普遍期待，有助于促进中欧相关产业链供应链的安全稳定，维护中欧经贸合作大局及以规则为基础的国际贸易秩序。
中国机电商会指出，中欧双方本着相互尊重和对话精神，经过多轮磋商，成功推动该案实现“软着陆”。
中国机电商会称，相关成果将鼓励和支持涉案企业充分利用磋商成果，以价格承诺申请方式，努力争取自身对欧出口权益，促进中欧相关产业合作健康发展。
中欧电动车关税谈判打破僵局。布鲁塞尔星期一宣布针对出口到欧盟的中国电动车实行价格承诺，将以最低售价来替代惩罚性关税；北京则表示，这体现中欧双方的对话精神和磋商成果。
分析认为，价格承诺为后续处理中欧经贸摩擦提供了积极示范，有助缓和双边紧张经贸关系，并为中国电动车企提升利润空间创造条件。

他说：“现在，司法部的独立性和公信力受到了质疑。”
另一名共和党参议员、来自阿拉斯加州的穆尔科斯基（Lisa Murkowski）支持蒂利斯的行动，她称针对鲍威尔的调查“就是一种胁迫手段”。

2026年1月13日 科伦坡 — “重建斯里兰卡”全国计划启动仪式今日在科伦坡班达拉奈克国际会议中心（BMICH）莲花厅举行，得到总统阿努拉·库马拉·迪桑纳亚克（Anura Kumara Dissanayake）的支持。该计划旨在统筹应对飓风Ditwah造成的全国性破坏，救助受灾民众并恢复公私财产与基础设施。
筹资采取三大策略：调整在建项目目标以重新利用资源；依《公共财政管理法》重新拨款，并拟为2026年准备5000亿卢比的补充预算；通过“重建斯里兰卡”基金动员国内外、公私部门与捐助者支持。
总统已成立由25名部长和高级官员组成的总统工作组，主席为总理哈里尼·阿马拉苏里亚（Dr. Harini Amarasuriya），负责战略领导、跨部门协调与关键资金和重建决策。为落实与监督，工作组设立八个由相关部长牵头的专门委员会：
1. 灾后需求评估委员会：收集、分析灾害的物质、经济与社会影响数据。
2. 公共基础设施恢复委员会：统筹交通、电力、供水、灌溉、医疗、教育与通信等公共设施的恢复与重建。
3. 受灾社区住房恢复委员会：按恢复计划修复或重建住房。
4. 地方经济与生计振兴委员会：稳定生计与企业、恢复关键供应链、改善金融服务并拓展经济机会。
5. 社会服务恢复委员会：恢复教育、卫生、水与卫生设施及福利援助等基本社会服务。
6. 财务与资金委员会：监管募资、预算规划、捐助者协调及基金的透明使用。
7. 数据与信息系统委员会：优先排序恢复需求，建立包含分析工具与预测模型的信息与监测系统。
8. 公共沟通委员会：统筹对外沟通、公众宣传与利益相关方参与。
