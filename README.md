创奥运新猷 宁忠岩为中国夺冬奥会第三金
宁忠岩沉着应战力克强敌，在本届米兰—科尔蒂纳冬季奥运会以创造大会新猷夺冠，为中国赢得这次赛会的第三枚金牌。
综合路透社和央视新闻客户端等报道，26岁的宁忠岩当地时间星期四（2月19日）在米兰冬奥会男子1500米速度滑冰决赛中，战胜夺冠热门、本赛季保持全胜纪录的美国21岁小将斯托尔兹和寻求三连冠的荷兰36岁老将努伊斯，以打破奥运会纪录的1分41秒98佳绩夺标。
这是宁忠岩在本届冬奥会收获的第三枚奖牌，也是中国代表团在这个项目赢得首枚冬奥会金牌。宁忠岩此前在男子1000米速度滑冰项目和男子团体追逐赛中进账两枚铜牌。
宁忠岩这枚金牌的含金量十足，不仅因为他打破了奥运会纪录，还终结了欧美选手长达102年的垄断。自1924年冬奥会以来一共产生了26枚1500米速度滑冰金牌，均由欧美选手获得。
斯托尔兹此前已在本届冬奥会收获男子速度滑冰500米和1000米速度滑冰项目金牌，距离“三金王”仅一步之遥，但最终未能滑出优于宁忠岩的成绩，获得亚军。将在本届冬奥会后退役的努伊斯收下铜牌。
在确认夺冠后，宁忠岩激动掩面流泪，披上国旗绕场庆祝。
宁忠岩赛后受访时说：“今天是我最完美的一场比赛，在大年初三为中国队获得第三块金牌，是我的新年大礼也是送给全国人民的新年礼物。我这一刻太开心了，四年的辛苦和汗水在这一刻都值得了。”
不过，宁忠岩坦承，没想过自己能战胜全赛季状态火热的斯托尔兹。他说： “我整个赛季从未打败过乔丹（斯托尔兹）。即便今天胜出，仍难以置信。在他完成比赛后，我感到震惊，没想到竟然打败了他，感觉终于登上了山顶。”
宁忠岩说：“我之前看到有人留言说‘我是总拿银牌的人’。但我依旧坚持努力争夺金牌。今天大家都滑得很好，只不过我的表现稍好些。”
中国冬奥会代表团的首两枚金牌，由苏翊鸣和徐梦桃分别在男子单板滑雪男子坡面障碍技巧项目和女子自由式滑雪空中技巧项目中夺得。

扮灾民骗大埔火灾救援金 港男获准保释候讯
一名香港无业男子假扮灾民骗取大埔宏福苑救援金，被控欺诈等五项罪名。香港法院裁定，男子获准保释候讯。
综合网媒“香港01”和《星岛日报》报道，香港大埔宏福苑去年11月发生五级大火，造成168人遇难，数千居民受灾。一名36岁香港无业男子冯俊杰灾后涉嫌以他人身份证明副本登记成为灾民，取得灾民证后向三家社福机构骗取合共1万8000港元（2924新元）援助金，被控欺诈等五项罪名。
此案星期五（2月20日）在粉岭裁判法院再讯，署理主任裁判官香淑娴应控方申请，将案件押后至4月24日再讯，以待警方进一步调查，包括向香港社会福利署职员录取口供、进行文件指纹校对及向银行调查。

【中国再次成为德国最大贸易伙伴】金十数据2月20日讯，根据德国联邦统计局当地时间20日发布的最新数据，2025年中国以约2518亿欧元的进出口总额再次成为德国最重要贸易伙伴。数据显示，2025年德国与中国的货物进出口总额同比增长2.1%。统计局指出，自2015年以来，中国一直是德国最重要的进口来源国。2025年德国自中国进口商品总额达1706亿欧元，出口为813亿欧元。同期，美国关税政策及贸易争端对德美贸易造成一定影响，2025年德美贸易总额为2405亿欧元，同比下降5.0%。德国对美出口额为1462亿欧元，同比下降9.4%。从整体看，2025年德国货物出口总额同比增长0.9%至15630亿欧元，进口总额同比增长4.3%至13625亿欧元。全年贸易顺差为2005亿欧元，较2024年减少424亿欧元。
