去年比亚迪大陆以外销量占比约23%，当年总交付460万辆，同比增长7.7%，海外占比较2024年的10%已大幅提升。比亚迪旗下有宋（Dynasty）、海洋（Ocean）、腾势、仿城堡（Fang Cheng Bao）与仰望（Yangwang）等五个品牌，海外经销商约2000家。
执行副总裁Stella Li此前在慕尼黑车展表示，公司计划到2026年将欧洲展厅数量翻倍至2000家，并在欧洲建立完整本地供应链。李云飞补充，海外工厂产能今年将提速，匈牙利工厂年产15万辆，预计3或4月投产；比亚迪亦在泰国和巴西设厂。
市场方面，德意志银行与瑞银预计2026年中国大陆乘用车销量将分别下滑约5%和2%。北京市正逐步取消购车税优惠，新能源车购置税从去年免征降为5%，并将在2028年恢复至10%。
摩根大通亚太区汽车研究负责人Nick Lai指出，中国厂商大陆单车平均净利约5000元，若更多出口到价格更高的海外市场，净利可望增至约2万元。独立分析师高申表示，更多中国车企，尤其是新能源车企，会加大海外投入以提升盈利能力。捷豹支援的中国电动汽车厂商零跑（Leapmotor）运营总监徐军也预计其2026年海外交付占比将从2025年的5%升至10%。
