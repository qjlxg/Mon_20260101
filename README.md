澳洲炼油厂发生大火 或影响汽油生产
澳大利亚维多利亚州吉朗市（Geelong）一家炼油厂星期三（4月15日）深夜发生火灾。截至星期四上午，大火仍在燃烧，恐影响汽油生产。
失火的炼油厂属于维瓦能源集团（Viva Energy），火灾发生在车用汽油生产区域，调查人员初步怀疑，失火原因是发生了泄漏。目前尚无人员伤亡报告。
据报，维瓦能源炼油厂是澳洲仅有的两家炼油厂之一，每天可以处理12万桶原油，产能满足全国10%的成品油需求。当前原油和成品油供应受中东战事影响，这次火灾进一步加剧澳洲民众对本国油品供应的担忧。
澳洲能源部长鲍恩告诉第九频道：“火势尚未得到控制……出于安全考虑，炼油厂已降低航空燃油和柴油的生产水平……影响主要体现在汽油生产上，但显然，现在下结论还为时尚早。”
大火产生的有毒气体一度威胁到吉朗市较大区域，消防与救援局提醒受影响区域的居民紧闭门窗、关闭空调系统。后来由于风向变化，受影响区域缩小，火情目前对公众不构成威胁。

法国全票通过法案 圆明园文物有望被归还中国
法国国会通过一项法案，简化其在殖民时期掠夺的文物归还程序。中国媒体报道，法国将不义之财归还中国的一天终于到来。
综合法新社、法国《世界报》等报道，法国国民议会星期一（4月13日）全票通过一项法案，将简化法国在殖民时期掠夺所得文物的归还程序。
法国目前仍拥有数以万计掠夺所得的艺术品及珍贵文物。该法案主要针对1815年至1972年间获取的文物。
中国媒体引述议员热雷米·帕特里耶-莱图斯发言，提到了法国文豪雨果的梦想。他说：“雨果早在1861年，就北京圆明园被洗劫一事致信巴特勒上尉：‘总有一天，法兰西能脱胎换骨，将这不义之财归还被抢掠的中国’”。
这名议员表示，“这一天终于到来”，且这不仅仅关乎一件文物或一个国家，这部法案要将归还原则纳入法国的法律体系。
《文汇报》报道，在西方博物馆中，法国枫丹白露宫收藏的圆明园珍宝最多，宫中的中国馆可以说是圆明园在西方的再现。
据悉，欧洲前殖民大国正逐步推动归还殖民时期掠夺所得艺术品。不过法国现行法律规定，收藏中的每一件文物被归还前，都须提交议会表决，为归还程序构成主要障碍。

大摩：若油价持续120美元 对消费拖累更显著
格隆汇4月15日｜中东战事未平息，油价上升影响亚洲经济。摩根士丹利亚洲首席经济学家Chetan Ahya表示，在地缘政治紧张局势升温之前，亚洲各经济体(包括中国)的消费增长正逐步改善，随着能源价格上升，短期而言会削弱可支配收入，但政策及储蓄或可作缓冲。\n他指出，若未来3至4个月油价实物交收价格回落至每桶90美元以下，工业周期有望重新加速，进而支持资本开支、就业及消费。不过，若油价持续于每桶120美元水平，随着燃料价格进一步上升，消费者将调整消费模式，对消费的拖累可能变得更为显著。\n大摩中国首席经济学家邢自强团队指出，自地缘政治局势升温以来，中国与“以旧换新”计划相关的板块消费增长持续放缓。团队解释，鉴于中国对能源进口的依赖相对较低、通胀起点较低，加上燃料定价受管制，能源价格上升对家庭消费的直接拖累可能有限。不过，投入成本上升可能压缩下游企业的利润率，进而拖累工资增长及就业，这反过来将打击消费，尤其是在工资增长起点相对疲弱的背景下。

公安部称，此次联合行动是中斯两国深化执法合作、加大打击跨境电信诈骗力度的成果。自今年年初以来，双方针对部分涉诈人员及相关地下产业由他国转移至斯里兰卡的态势加强了协作。
公安部透露，斯里兰卡警方近期多次开展清查行动，摧毁数个诈骗窝点并抓获多名涉案人员。参与此次遣返工作的中国湖北省公安机关正在对相关案件进行侦查。
公安部一名官员表示，将继续加强与更多国家的执法合作，共同构建打击跨国犯罪的务实高效新机制。来源：新华社

深圳报道 — 中国国务院总理李强在周五举行的高层座谈会上，要求加快在制造业推广工业人工智能，推动制造业的数字化与智能化升级。李强在与新兴独角兽Moonshot AI创始人杨志林等企业家和业界专家座谈时指出，要更深入落实“AI+”战略，加快把人工智能嵌入制造环节，强化先进制造与现代服务的融合，增强产业体系整体实力。 
李强并表示，尽管外部环境今年更为复杂多变，但在及时出台积极的宏观政策支持下，经济已开局良好。他还强调要抓住新兴产业发展机遇，扩大就业、提升收入、拉动内需，并鼓励企业家在创新和带动就业方面发挥更大作用。 
受邀参加座谈的还有广发证券首席经济学家郭磊、中国人民大学金融学教授马广荣以及招商局港口控股董事长冯伯明等。

美国联邦通信委员会（FCC）周四表示，正考虑更广泛的措施，禁止中国电信运营商在美运营数据中心，并进一步限制其接入美国网络与基础设施。草案同时建议禁止美国运营商与这些被认为存在“重大国家安全风险”的公司互联互通。中国移动、中国电信和中国联通均被列入受限实体名单，可能被要求出售、转让或关闭在美的数据中心和网络节点，甚至被迫退出美国市场。
分析人士称，此举标志着华盛顿对中国科技企业多年打击的升级，尽管短期内对相关公司财务影响有限。塔霍塔律师事务所合伙人王宇祥表示，草案将管制从禁止向公众直接提供服务和限制硬件，进一步延伸到对底层互联网基础设施和互联协议的更深层控制。

美伊在伊斯兰堡举行的首轮面对面谈判首阶段已结束，双方专家团队继续就关键问题交换书面文本。伊朗国家广播电视台（IRIB）援引接近谈判团队的消息称，谈判可能在今晚晚些时候或周日继续。
周六的会晤被认为是华盛顿与德黑兰数十年来最为实质性的直接接触。白宫此前表示，美国、巴基斯坦和伊朗当天举行了三方面对面会谈，凸显伊斯兰堡在将本周初斡旋达成的为期两周脆弱停火转化为更广泛政治解决中的核心斡旋角色。
巴基斯坦主办的“伊斯兰堡会谈”被视为自1979年伊朗革命以来最重要的谈判之一，旨在在停火框架下推动结束更广泛的地区冲突。两代表团抵达后，还在首都分别与巴方官员会面以“修补分歧”。
美国代表团由副总统J·D·万斯率领，随行包括特使史蒂夫·维特科夫和贾里德·库什纳。伊朗代表团由议会议长巴格尔·加利巴夫率领，成员有外长阿巴斯·阿拉格奇和国家安全委员会副手阿里·巴格里·卡尼等。巴基斯坦由外长伊沙克·达尔代表，会谈中有陆军参谋长阿西姆·穆尼尔与内政部长莫辛·纳克维协助。—— 各通讯社

华盛顿 2026年4月12日 — 美国总统唐纳德·特朗普警告称，如果中国向伊朗转移武器，将面临50%的高额关税，但他同时表示怀疑北京会这样做。

2026年4月13日 22:17 — 面对日益增长的争议，美国总统唐纳德·特朗普似已删除他在社交平台Truth Social上发布的一则争议帖子。 
该帖配发一张由AI生成的图像，显示特朗普身穿白袍、以发光的手触摸病床上男子额头，画面被指类比耶稣医治病人的宗教画作，并在背景加入自由女神像、飘扬的美旗、战机、鹰以及一名护士、祈祷者与疑似军人等元素。 
这张图在美国政治光谱两端都引发强烈批评，部分特朗普支持者亦表态反对。该帖发布数小时之前，特朗普曾发表长文批评教宗列奥十四（Pope Leo XIV），后者对美以在伊朗行动持批评态度。 
基于争议，死党内外有人呼吁立即删除此帖，其中基督教活动人士肖恩·福赫特（Sean Feucht）明确要求撤下。BBC已就贴文被删除一事向白宫求证。

记者在科威特因发布美军战机坠毁视频被拘留
据CNN报道，曾在VICE、HuffPost和半岛电视台等媒体工作的记者艾哈迈德·希哈布-埃尔丁（Ahmed Shihab-Eldin）在科威特被拘，时间约为他在科威特城西部发布一段美空军战机坠毁视频六周之后。
希哈布-埃尔丁的朋友与支持者周二将此事公开，期望舆论能推动其获释。保护记者委员会称，当局已以“散布虚假信息、危害国家安全和滥用手机”等指控起诉他，批评这些模糊且过于宽泛的罪名常被用来压制独立新闻工作者。
自伊朗战争爆发以来，科威特及其他海湾国家加强了对发布伊朗导弹袭击等敏感内容者的打压。海湾人权中心在四月初报告指出，战争被用作限制言论、剥夺公民公共自由的借口，并针对与政府立场不符的记者、博主和网络活动家实施任意拘押与以反恐、网络犯罪法为由的捏造指控与不公审判。
希哈布-埃尔丁为科威特裔美国人，事发时为三月初回国探亲。CNN已向科威特政府求证。
来源：CNN / 通讯社
