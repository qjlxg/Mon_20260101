新竹科学园变电所爆炸起火 致一死二伤
台湾新竹市科学园区一座超高压变电所星期一（3月16日）下午爆炸起火，造成一死二伤。由于变电所内储存大量绝缘油，火势持续延烧至当晚。
据联合新闻网、自由时报等报道，新竹科学园区力行路当天下午3时许发生火警，位于该处的台湾电力公司竹园超高压变电所突发爆炸起火，现场伴随惊人火光与漆黑浓烟，遮蔽园区半边天空。
消防局出动大批人员和消防车赶往现场灭火，三名伤者送医治疗，其中一人伤势严重，救出时已无生命迹象。
新竹市消防局指出，星期一下午3时27分接获多通民众报案，指东区力行路18号“竹园超高压变电所”传出爆炸巨响，现场随即窜出大量烟火，情况十分危急。
消防局第一时间派遣多个车组赶赴现场，3时33分首波人员抵达现场后见火势猛烈，随后再加派人手及多辆水库车支援，总共出动16车、41名消防人员。
据现场目击画面，变电所上方不断窜出漆黑浓烟，数公里外清晰可见，引发园区各厂区员工高度关注。有民众在社群平台上表示，当时听到一声巨大的爆炸巨响，接着看到大量浓烟，担心电力供应受影响。
事发处为竹科供电核心，与新竹县峨眉超高压变电所相通，负责稳定园区及新竹县市电力。因此，消防人员立刻布线压制火势，严防波及其他高压电气设备造成大规模断电。
消防人员随后在现场陆续救出三名受难者，并于下午4时许陆续送医。据了解，现场有三名伤者伤势严重，分别为一名腿部封闭性骨折，另一名身上有明显烧烫伤。最严重的一名伤者全身多处烧烫伤，救出时已失去生命迹象。

美方顾问哈塞特称五角大楼预计行动可能需时达六周。油价方面，布伦特最高触及约106.5美元，回落至约104美元，西德克萨斯中质油突破100美元；同时日本在国际能源署指引下开始释放战略石油储备。IEA成员上周同意创纪录释放4亿桶库存以缓解价格冲击。
在经济面，市场担忧能源危机冲击全球增长，东京、上海、悉尼、台北、曼谷等亚洲主要市场多数下跌，香港、首尔、新加坡上涨；欧美开盘多走高。美国第四季度GDP修正为增长0.7%，联储偏好的通胀指标1月降至2.8%（能源上涨前的数据）。本周包括美联储、英格兰银行和欧洲央行在内的七大央行将举行政策会议，市场将密切关注各行对冲突影响的评述。
