马尼拉讯 — 参众两院议员、选举委员会（Comelec）及选举监督团体周一在参议院选举改革与人民参与委员会举行听证，会中推动修改党团名单（party-list）制度，并审议相关改革法案。
Comelec主委乔治·欧文·加西亚（George Erwin Garcia）呼吁立法明确替补候选人的界限，指出现行制度允许在选前乃至选后（因死亡、被取消资格或撤回等）替换名单，导致当选席次可能被非原始弱势代表占据。他并建议明确列明认证资格，质疑当选代表是否应来自贫困与边缘群体，或是否可住在富人区如Forbes Park、Dasmariñas Village。
委员会主席兼参议院副多数党领袖里萨·洪蒂韦罗斯（Risa Hontiveros）认为应重审现行的2%门槛及席次上限。根据第7941号法案，获党团名单有效票2%以上者得一席，逾2%者可按票数再获额外席次，但每团体不得超过三席。
参议院议长文森特·索托三世（Vicente Sotto III）对党团名单制度被曲解与滥用深表关切，主张恢复党团原本属“边缘化团体”的精神，并重申边缘部门范围包括劳工、佃农、农渔民、城市贫困者、原住民、老年人、身障者、妇女与青年。他主张仅允许达到2%门槛的團體入席国会。
FPJ Panday Bayanihan党团众议员布莱恩·波伊（Brian Poe）则对严格2%并取消席次上限表示保留，警示可能促成若干党团在国会内部形成“小型政党”。洪蒂韦罗斯表示，委员会将在拟写报告时综合各方意见。

中国东部浙江在“共同富裕”实验中既见成效也面临挑战。官方草案显示，浙江城镇与农村人均可支配收入比从2021年的1.96缩小到2025年的1.81，省府提出到2030年在前五年基础上“持续缩小”差距，并把人均GDP与收入提高到接近发达经济体水平。浙江自2021年被列为共同富裕示范区，同时也是阿里巴巴、DeepSeek、Unitree等科技企业聚集地，AI、机器人、游戏与脑机接口等产业发展迅速，催生多位新富豪。 
当局担忧的是能否避免硅谷式的不平等。研究显示，硅谷0.1%人群掌控71%财富，少数家庭占据大量区域财富，而大量家庭几乎无资产。东华理工大学教授熊万胜指出，资本密集的技术革命正在替代大量岗位，“就业自然分配财富”的观念遭到冲击，中国社会保障在养老、失业和家庭政策等方面仍较薄弱。 
浙江草案提出发展全民增收计划、鼓励企业推行灵活股权激励和员工持股、盘活闲置农房，并强调兼顾发展与安全、现实与可能。浙江大学何文炯建议评估科技对就业的影响、加强职业教育与培训、保障公共服务均等并探索普惠性基本收入以维护民生底线。

“铁饭碗”遇上算法：中国经济为何更能抵御AI冲击
在通常被视为中国职工最快乐时刻——除夕夜全国欢聚的春晚舞台上，类人机器人跳霹雳舞、翻跟头、舞剑并表演小品，国内AI品牌大量抢占黄金广告位。约6.77亿观众观赏节目后，不少人并非为科技进步欣喜，反而在社交媒体上表达对被替代的恐惧：“感觉离失业又近了一步”“它要抢走努力人的饭碗”“舞蹈演员会先被替代”。
这种担忧并非中国独有：春晚后一周，美国小公司Citrini Research的一份报告警示AI将导致大规模失业并引发经济衰退，曾在华尔街引发抛售。业内人士指出，尽管短期存在置换效应，AI同时也创造大量新职位；且中国庞大的公共部门和就业保障体系在一定程度上能缓冲冲击，但仍需更多政府支持与有序的政策安排。能否将AI变为助力而非威胁，取决于劳动者、企业与政策制定者如何共同调整与应对。
