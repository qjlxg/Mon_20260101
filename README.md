台湾8月起停止对柬埔寨公民免签
台湾官方公告，将从8月起停止对柬埔寨公民实行的免签措施。
台湾外交部星期五（7月17日）在官网发布的新闻稿中指出，台湾自2018年8月起开放柬埔寨国民适用“东南亚国家人民赴台先行上网查核”（TAC）有条件式免签，以及“观宏专案”等签证便利措施入境台湾，盼借由增加两地民间往来，促进柬埔寨政府与人民对台湾了解。
然而，柬国政府长期并未以平等互惠方式回应台湾善意，更屡次公开配合中国大陆发表贬抑台湾主权甚至隐含支持大陆武统的声明，损害台湾尊严及人民情感。另外，考量近期诸多跨国重大诈骗案件也与柬埔寨有关，相关负面发展引起台湾各界及民众高度重视与严重关切。外交部因此决定自8月1日起停止对柬埔寨国民适用的上述签证便利措施。
外交部强调，未来拟赴台的柬国民众均须向台驻外馆处申办适当签证。至于何时恢复相关签证便利措施，将视柬国政府对台政策再予研议调整。　
同日，台外交部也公告，自今年8月1日起，延长试办泰国、文莱及菲律宾国民赴台免签证措施一年至明年7月31日；另对印度、印尼、越南、缅甸、老挝等五国国民续行“观宏专案”一年至明年12月31日，并依现行作法持续办理TAC有条件式免签证入境措施。
