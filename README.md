一张配图显示，一段乐高风格的AI生成战争动画出现在美国总统唐纳德·特朗普与美国国防部长皮特·赫格塞斯（Pete Hegseth）照片前。

德国政府尽力淡化影响，国防部长皮斯托留斯称撤军“可预见”，外长瓦德富尔称正与欧洲伙伴“以信任精神密切磋商”。北约方面仅表示正与美方沟通细节。
欧洲内部担忧此举连同此前从罗马尼亚撤走4,500名部队、以及取消在德部署中程导弹，可能削弱对俄威慑并侵蚀跨大西洋安全保障。美国在欧军力约7万人，德国驻美军约4万，拟撤离的作战部队可能是驻巴伐利亚维尔塞克的第2骑兵团。
两党国会关键共和党军委主席罗杰·维克尔和众院军委主席迈克·罗杰斯已联名致函表达关切，国会在军费与持续支援伊朗冲突上仍有制衡力量。记者：乔纳森·伊亚尔（伦敦、布鲁塞尔）

西棕榈滩/迪拜 5月2日—美国总统唐纳德·特朗普表示，他已被告知一项与伊朗有关的“方案构想”，但正在等待具体措辞，同时警告若德黑兰“表现不当”，美方可能会恢复对伊朗的打击。
一名不愿具名的伊朗高级官员称，伊朗向美方经中间人递交的一份正式提案（已被特朗普拒绝）建议先开放霍尔木兹海峡并解除美国封锁，将有关伊朗核问题的谈判推迟到后期。
特朗普在登机前回应称：“他们告诉我方案的概念，现在要把确切措辞给我。”他并在社交媒体上表示，难以想象这些提议会被接受，伊朗为其所作所为付出的代价还不够。被问及是否会重启对伊朗的打击时，特朗普称无法对记者详细说明，但“如果他们表现不当，这是有可能发生的”。
伊朗外交部长则表示，若美方改变立场，德黑兰愿意通过外交解决。伊朗媒体披露的14点建议还包括：美军撤离伊朗周边地区、解除封锁、解冻伊朗资产、赔偿并解除制裁、结束包括黎巴嫩在内的各前线战争，以及为霍尔木兹海峡建立新监管机制。
美以四周前暂停对伊朗的轰炸行动，但双方仍未就结束这场已严重扰乱全球能源供应的冲突达成协议。伊朗已封锁波斯湾大部分航运两月余，美国4月对伊朗港口船只实施了封锁。美国称不会在未阻止伊朗获得核武的条件下结束战争；伊朗则坚持其核计划为和平用途。
据路透社报道，伊朗官员称，将核问题留到最后谈判是为促成协议创造更有利氛围：在一项框架下，战事结束并获得美以不再攻击的保证，伊朗开放海峡、美国解除封锁，随后就核项目限制与制裁解除进行谈判，伊朗要求即便暂停核计划也保留和平浓缩铀的权利。

贝鲁特消息——黎巴嫩卫生部表示，以色列3日对南黎巴嫩发动空袭，至少造成1人死亡，多人受伤。卫生部称，阿拉布萨利姆（Arabsalim）遭袭致1人死亡、3人受伤（包括一名儿童）；斯里法（Srifa）一处空袭在其附近击中一救援中心，造成5人受伤，其中4人为与真主党关系密切的伊斯兰卫生委员会的救援人员。
卫生部对持续的袭击表示谴责，并援引《日内瓦公约》第19条强调必须确保医疗机构在冲突区免受攻击威胁。
以色列军方当天对南黎巴嫩发布新一轮撤离警告，涉及10多个村镇，包括纳巴泰区（位于利塔尼河以北）在内的数处社区；阿拉布萨利姆和斯里法均在警告范围内。黎国营新闻社报道，南黎巴嫩多地遭到袭击，部分被袭城镇并未出现在撤离警告名单中。
自4月17日以来，以色列与真主党之间维持一项脆弱停火。以军参谋长4月29日曾威胁要对“黄线以北”的目标采取行动；停火文本亦赋予以色列对“计划中、迫在眉睫或正在进行的攻击”采取行动的权利。过去一周，数起无人机袭击已致两名士兵及一名军方承包商死亡，多人受伤。
真主党近来使用以光纤控制的廉价无人机，抗干扰能力强、航程数十公里，构成对驻黎以军及北部以色列社区的威胁。真主党曾于4月27日表示，已用无人机袭击贝亚达（Bayada）以色列军队，作为对停火“违规”的回应。
美国呼吁黎以双方直接和谈，但黎巴嫩总统朱瑟夫·阿ウン要求以色列先全面履行停火。真主党坚决反对直接谈判，其议员哈桑·法德拉拉3日称，该组织有能力挫败这些谈判的目标，并加剧国内及国家内部的分裂。新华社/法新社报道

华盛顿——随着5月3日进入距11月中期选举最后六个月，控制国会的争夺进入关键阶段，这次选举可能重塑特朗普第二任期并改变华盛顿权力平衡。
中期选举一向被视为总统的公投，民主党希望借经济不满和特朗普约40%的低支持率收复众议院与参议院，共和党则力争在逆风中保住席位。
掌控国会不仅关系立法权，更决定能否对总统展开调查、阻挠提名等，影响其任期走向。早期迹象显示共和党面临挑战，通胀与伊朗战争相关开支等经济问题正侵蚀民众信心。
民调显示民主党在总体投票意向上略占优势，部分调查也显示选民在经济问题上对民主党信任度上升。亲共和团体AFP Action警告，参议院多数党面临风险，称存在结构性逆风和选民冷漠。
本次中期选举将改选全部435个众议院席位、约三分之一的100席参议院以及多数州长职位。民主党若要掌控两院，需在众院净胜三席、参院净胜四席。
民调机构Impact Research的莫莉·墨菲指出，当总统支持率在或低于40%时，更容易在传统偏共和的地区看到席位变动。
选情仍然流动：共和党受益于相对有利的参议院版图和经过严重划分的选区，但多州正在进行年中重划，包括德州、加州、北卡、俄亥俄、佛罗里达、密苏里、犹他和弗吉尼亚，最高法院限制基于种族的重划也改变格局。
参议院领袖舒默指责共和党通过过度选举安全和移民执法措施削弱投票权，称这是“操纵制度”的企图。共和党则依靠资金优势并强调移民与国家安全议题，认为最终取决于投票率与选民对其执政成果的认可。众议院多数党领袖斯卡利斯表示，选情将取决于选民投票热情和对共和党“扭转局面”成果的评价。
