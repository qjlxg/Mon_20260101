中国广东省广州市32岁程序员高广辉在过度劳累后猝死，引发广泛关注。据中国媒体报道，高光辉死亡情形已被认定为工伤。
去年11月29日，广州视源电子科技股份有限公司高广辉周末突然在家中晕倒，送医抢救无效。他被曝出抢救过程中还被拉入工作群，死亡后还收到工作消息，引发社会关注。
高广辉妻子杨华思12月29日通过丈夫所在公司，向广州市黄埔区人力资源和社会保障局提交工伤认定申请。
澎湃新闻星期五（3月13日）从家属处获悉，今年2月，广州黄埔区人力资源和社会保障局作出认定：高广辉死亡的情形视同为工伤。
