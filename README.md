台自制潜舰海鲲号完成潜航测试
台湾自制潜舰海鲲号星期四（1月29日）完成首次潜航测试。负责制造的台船公司说，海鲲号将按计划执行后续海上测试。
台船在官网公布，海鲲号当天首次执行浅水潜航测试，依海测测试程序书，已顺利完成首次下潜及本次计划测试项目。
公司说：“潜舰是具备吓阻能力的关键战力，受限于国际环境及中共打压因素，国造潜舰自始至今，遭遇各种困难与挑战，造舰团队秉持面对问题，解决问题的态度，共同齐心合力逐步克服万难，完成首次‘潜航下潜’里程碑。”

台湾军事观察学者纪东昀说，如果这次潜航测试顺利，或再调整后，海鲲号将可能赴小琉球以南海域，进行深水域潜航测试，下潜200公尺。

大陆移民局：去年下半年台湾“首来族”增40%
中国大陆移民局表示，为首次赴陆的台湾人免证件费办理台胞证的政策受到普遍欢迎，去年下半年，入境大陆的台湾“首来族”同比增加40.9%。
大陆移民管理局新闻发言人、综合司副司长吕宁星期三（1月28日）在例行新闻发布会上介绍，台湾“首来族”办证人数及来往大陆的人数均实现大幅增长。
大陆移民管理局自去年7月1日起，为台湾“首来族”办理台胞证减免证件费。
吕宁表示，据统计，政策实施后至去年年底，台湾“首来族”办证人次环比上升25.9%，同比上升57.2%。台湾人办证数量也实现持续增长，同期申请办理台胞证人次环比上升5.2%，同比上升11.9%，台湾人前往大陆交流交往的意愿明显增强。
此外，“首来族”来往大陆人次持续增加。去年下半年，台湾“首来族”入境人次环比上升32.9%，同比上升40.9%。同期，台湾入境大陆人次环比上升14.3%，同比上升24.4%，越来越多的台湾人用实际行动坚定支持和积极参与两岸交流。

在向碳中和出行转型过程中，欧洲和中国呈现出截然不同的政策路径。欧洲原先提出到2035年停售新燃油车的硬性禁令，近期却拟改为减少90%二氧化碳排放的灵活目标，这一变化反映出从理想主义向受限务实的回调。早在2023年初政策推出时就已显裂痕：在德国等工业大国压力下，对合成燃料（e-fuels）出现豁免。  
欧洲之所以退让，源于理想与现实冲突的三重难题：维护社会公平、弥补基础设施短板和应付政治分歧。对普通民众而言，转型带来高昂购车成本、不可靠的充电网络和续航焦虑；更深层的问题是分散且尚不成熟的充电生态与不适应大规模电气化的电网。政治上，制造业国家主张“技术中立”，绿色国家坚持纯电路线，最终酿成折中而模糊的政策。  
相比之下，中国作为全球电动汽车领先者，采取的是“有引导的务实”策略：国家导向但尊重市场，避免设定夸大的固定禁令，采取分阶段、以激励为主的政策体系。其“双积分”政策对高排放车型施加惩罚，同时奖励新能源汽车生产，既推动市场化转型又避免企业断崖式冲击。  
更关键的是，中国同步建设配套生态：全国统筹推进充电基础设施建设，公共充电桩达到数百万级别，创造了大规模普及的条件；消费者补贴在前期降低购买门槛，随后逐步退坡。欧洲在借鉴中国经验时，可考虑以更灵活和配套完善的路径推进绿色出行。

印度出现两例致命的尼帕病毒病例后，泰国和马来西亚已加强机场筛查以防扩散。以下是关于尼帕病毒的要点说明。
尼帕病毒是一种罕见的动物源性病毒，主要来自果蝠。世界卫生组织指出，病死率高达40%至75%，但病例也可能无症状。尽管可发生人与人传播，但传播并不容易，暴发通常较小且相对可控。现有多种疫苗候选正在研发中，但尚无获批产品。
尼帕首次于1999年在马来西亚发现，此后多次小规模暴发，多见于孟加拉国，印度偶有散发。流行病学追踪机构CEPI统计，截至去年12月累计约750例，死亡415例。
传播途径上，早期多与感染猪直接接触有关，随后主要与果蝠有关。世卫组织指出，食用被感染果蝠尿液或唾液污染的水果或果汁（如生棕榈汁）是常见感染来源；人与人多在病患家属或护理者的密切接触后发生传播。
临床表现初期为发热、头痛、肌肉痛等非特异性症状，随后可出现急性脑炎的神经症状和严重呼吸道问题。重症可发生惊厥并在数日内进入昏迷，部分康复者存在长期神经后遗症。
专家认为，尽管尼帕致死率高，但目前尚无证据表明其在人群间传播能力增强或即将全球流行；但在流行地区仍属重大公共卫生威胁，且可导致易感家畜（如猪）大规模扑杀。由于潜伏期较长，机场筛查效果有限。
目前无获批的特异性疫苗或治疗药物。包括牛津大学团队在内的候选疫苗正在试验，其中一款使用与新冠疫苗相同技术的疫苗已于去年12月在孟加拉开始Ⅱ期试验，合作方为国际腹泻病研究中心并获CEPI资助。

【琼州海峡预计春运期间将运送旅客超466万人次】金十数据1月30日讯，从今天交通运输部举行的新闻发布会上了解到，作为海南自由贸易港封关运作后的首个春运，今年琼州海峡客滚运输需求尤为旺盛，交通运输部全力做好各种准备保障措施。交通运输部数据显示，琼州海峡预计春运期间将运送旅客超466万人次、车辆超117万台次，同比分别增长12%和10%。其中新能源汽车过海需求激增，预计运输量将突破20万台次，占比超17%。春运期间，琼州海峡客滚运输整体发班能力可达单日350航次，船舶按照“人休船不休”要求配备足量船员，确保运力连续稳定。针对新能源汽车运输需求增长，增配两艘新能源汽车专用运输甲板驳。每日可设置50个新能源车运输专班，并根据船票预售情况及时增加专班数量，新能源车运输保障能力由去年的日均3500台次提升至5000台次。
