香港特区政府计划立法保障外卖员和送货员工伤和死亡补偿权益，希望能在今年底将条例草案提交立法会审议。
综合香港《星岛日报》和香港电台等报道，香港劳工及福利局星期五（7月10日）向立法会人力事务委员会提交立法框架，拟为平台工作者设立工伤和死亡补偿机制， 先涵盖外卖员和送货员。
劳福局局长孙玉菡星期六（11日）在一档电视节目上受访时说，根据现行条例，外卖员在工作期间死亡的最低死亡赔偿额，倾向参考《雇员补偿条例》下的相关赔偿金额51万港元（8万4000新元）。雇员若因工受伤休假，雇主必须支付原本收入约八成的补偿金，最多两年。
孙玉菡指出，平台工作者与传统雇员的性质存在差异，因此平台方支付的工伤补偿需采用多层次计算方法。这包括考虑相关人士从事平台工作的时间、厘定按收入计算病假津贴的比例，以及获补偿的最长期限。
在罚则方面，孙玉菡称，工伤补偿机制要求平台公司在指定时间内呈报平台工作者受伤，法定要求与现行《雇员补偿条例》非常类似。因此，政府在制定罚则时会参考现行条例下逾期未呈报工伤的最高罚款金额5万港元。
孙玉菡透露，政府未来一两个月将重点讨论具体细节，希望在暑假内就工伤补偿计算有较为清晰的安排，随即开展法例草拟，争取在今年底将条例草案提交立法会审议。
他补充，若死亡和工伤补偿等核心方案能达成平衡，他倾向一次过推行工伤和死亡补偿机制，但如果若部分条文需要更多时间斟酌，也可分开执行。
孙玉菡最后说，政府下一步将研究如何把网约车等平台工作纳入上述机制。

大摩：三大因素可能终结美股夏季涨势
格隆汇7月10日｜摩根士丹利全球固定收益研究主管安德鲁·希茨表示，该行正密切关注可能导致股市在夏季受挫的三大障碍；从历史上看，夏季通常是股市表现最强劲的季节。第一大风险是伊朗冲突重燃。美国战略石油储备降至历史最低水平，如果冲突再次升级，这可能会削弱其应对冲击的能力。第二大风险是美联储加息。美联储将在年底前维持利率不变的预期，是支撑当前股市牛市的关键支柱之一。 风险在于，这一假设可能是错误的，而且这种错误可能很快就会显现。当然有一种观点认为，如果美联储担心通胀，就不应拖延行动。第三是AI资本支出前景走弱。第二季度的财报可能显示出在支出方面更为谨慎的态度，这或许是因为部分大举投资AI的公司的股价近期表现不佳。鉴于当前的增长和盈利前景与AI高度相关，且投资者对AI概念股青睐有加，这种情况将带来风险。

高盛 ：抄底芯片股
高盛全球银行与市场部门ETF及定制篮子波动率交易负责人Shawn Tuteja在最新报告中指出，近期美股半导体及AI板块近20%的动量因子回撤，主要由技术性和结构性因素驱动。Shawn Tuteja认为，此轮动量因子超过20%的回撤，根本上反映的是市场定价效率的提升，而非AI交易逻辑的瓦解。从波动率调整后的回撤幅度来看，此次跌幅与此前动量因子在同等已实现波动率环境下的历次抛售轨迹高度吻合，只是市场到达这一水平的速度更快。据高盛TMT交易台的数据，目前客户情绪约为7.5至8分（满分10分），较一个月前的9.5至10分有所降温，但客户普遍认同本次回撤的核心驱动力是技术性的。高盛Prime Services数据显示，过去数周，基本面多空基金大幅削减了AI及动量敞口，当前总杠杆率已降至近一年的最低十分位。Shawn Tuteja认为，随着系统内杠杆降低、机构持仓更趋均衡，投资者具备在半导体及动量股多头方向“试探性建仓”的战术性机会。（券商中国）

机构：韩国领跌，6月新兴市场股市遭遇460亿美元大逃离
银行业贸易组织的数据显示，6月外国投资者从新兴市场股票投资组合中净流出 461亿美元。这一数据主要由对韩国科技股的大幅抛售推动，导致发展中经济体连续第二个月出现整体投资组合亏损。国际金融协会（IIF）周五发布的月度报告显示，外国投资者从韩国股市撤资 305亿美元，创下最近25年多来的最大流出规模。报告显示股市与债市资金流向出现明显分化：尽管整体投资组合资金流转为净流出178亿美元，但债券市场上月仍吸引了 283亿美元资金。国际金融协会（IIF）首席经济学家Jonathan Fortun表示：投资者仍愿意借钱给新兴市场，但他们不太愿意承担广泛的股市风险。全球贴现率上升、盈利信心疲软以及对科技和能源仓位的敏感性，导致投资者削减了股票配置。上半年的信息很明确，新兴市场总体上仍吸引了资本，但这仅仅是因为债务流入完全抵消了持续的股市清算。

摩根士丹利 警告称芯片制造商的定价能力面临更大压力
摩根士丹利的Lisa Shalett呼吁投资者对芯片股保持谨慎，她表示，更多迹象表明，芯片制造商的定价能力有限，意味着这些股票可能因为对人工智能支出的乐观情绪而上涨过头。摩根士丹利财富管理首席投资官Shalett周五表示：“我们看到AI数据中心技术栈正在被重新设计，以纳入成本更低的自研芯片，而许多超大规模云服务商如今在自行设计这些芯片。”Shalett发出上述警告之际，韩国芯片制造商SK海力士周五开始在纳斯达克交易。该公司此前通过美国上市募资265亿美元，创下外国企业在美国规模最大的首次公开募股纪录。与此同时，SK海力士在韩国上市的股票近期剧烈波动，自上月达峰以来已下跌26%。“总体而言，该交易领域仍有充足的资金可用，”Shalett表示。但她指出，市场正出现业内一种典型现象：“当供应链出现瓶颈，并且部分存储芯片厂商获取超额利润时，工程师们就会着手”寻找更便宜的替代方案。Shalett本周在一份投资者报告中称，半导体板块“明显超买”。她在Surveillance节目中表示，从芯片ETF到费城半导体指数，都可以看到支持这一判断的迹象。根据汇编的数据，费城半导体指数的市盈率自2022年以来已增长逾两倍。
