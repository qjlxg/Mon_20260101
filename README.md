民进党中国事务部列出四种认知作战的典型手法，包括短期无法证实的疑美论、疑赖（台湾总统赖清德）论和“弃台论”等负面论述，以及把局部事件夸大成全面碾压，如亲陆网红经常称颂大陆的人工智能（AI）与武器科技已经全面碾压美国，全世界都追不上，台湾防御已经没有意义等。
