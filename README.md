他说，此次“特习会”应还是聚焦在双方的贸易问题，例如要求中国大陆履行过去谈判中答应的购买美国农产品、处理芬太尼及进口美制飞机等。

华盛顿——随着5月3日进入距11月中期选举最后六个月，控制国会的争夺进入关键阶段，这次选举可能重塑特朗普第二任期并改变华盛顿权力平衡。
中期选举一向被视为总统的公投，民主党希望借经济不满和特朗普约40%的低支持率收复众议院与参议院，共和党则力争在逆风中保住席位。
掌控国会不仅关系立法权，更决定能否对总统展开调查、阻挠提名等，影响其任期走向。早期迹象显示共和党面临挑战，通胀与伊朗战争相关开支等经济问题正侵蚀民众信心。

民调机构Impact Research的莫莉·墨菲指出，当总统支持率在或低于40%时，更容易在传统偏共和的地区看到席位变动。
选情仍然流动：共和党受益于相对有利的参议院版图和经过严重划分的选区，但多州正在进行年中重划，包括德州、加州、北卡、俄亥俄、佛罗里达、密苏里、犹他和弗吉尼亚，最高法院限制基于种族的重划也改变格局。
参议院领袖舒默指责共和党通过过度选举安全和移民执法措施削弱投票权，称这是“操纵制度”的企图。共和党则依靠资金优势并强调移民与国家安全议题，认为最终取决于投票率与选民对其执政成果的认可。众议院多数党领袖斯卡利斯表示，选情将取决于选民投票热情和对共和党“扭转局面”成果的评价。
