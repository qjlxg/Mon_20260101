国际能源署说，在成员国一致决定动用4亿桶战略石油储备后，相关储备已开始向市场投放。
新华社报道，国际能源署（IEA）星期四（3月19日）发布新闻公报说，此次投放的石油储备总体将以原油为主，在欧洲地区则主要是成品油。与此同时，美洲国家将通过提高产量补充市场供应。
公报还说，随着各国进一步细化各自承担的份额，原油与成品油的具体分配比例以及来自公共储备的供应规模等仍可能调整。
受中东局势升级影响，伦敦布伦特原油期货价格星期四开盘后一度升至每桶116美元（约148新元）以上，欧洲天然气和石油价格当天早盘也明显上涨。
国际能源署11日宣布，32个成员国一致同意释放4亿桶战略石油储备，以应对因美国和以色列军事打击伊朗导致全球石油供应紧张的局面。这是国际能源署迄今协调规模最大的一次石油储备释放。

长和称公司持续与有关方商讨港口出售事宜
香港长江和记实业有限公司（简称长和）称，公司持续与有关方商讨港口出售事宜。
综合路透社和香港《南华早报》报道，长和联合董事总经理陆法兰（Frank Sixt）星期四（3月19日）在公司2025年度全年业绩报告记者会上，透露上述消息。
美国总统特朗普去年初重返执政以来，反对中国掌控巴拿马运河，令负责当地港口运营的香港长和集团陷入中美博弈漩涡。
去年，长和集团同意以230亿美元（294亿新元）向由美国资产管理巨头贝莱德牵头的财团脱售旗下全球23个国家和地区的43个港口，包括位于巴拿马运河两端的克里斯托瓦尔港和巴尔博亚港。
此举引发北京不满，长和过后称，港口出售磋商也将包括一个“重大战略投资者”，据报是中国航运巨企中远海运。
不过，巴拿马最高法院今年1月底裁定，长和旗下的巴拿马港口公司（PPC）运营巴拿马运河两个港口的合约违宪，之后长和公告，PPC已根据适用特许经营合约及国际商会的仲裁规则，积极果断地对巴拿马启动仲裁。
陆法兰星期四在记者会上说：“我们还是按计划与原本的买方财团，以及来自中国的重大金融战略投资者就出售港口事宜进行磋商。”
陆法兰也说，长和与PPC将持续与法律顾问不懈努力，通过全国和国际程序寻求所有适用的法律救济措施，以化解巴拿马两个港口合约被当局接管的困境。
巴拿马最高法院裁决后，巴拿马当局对PPC进行突击搜查，并接管PPC所运营的两个港口，将运营权暂交航运巨头马士基和地中海航运负责。上周，巴拿马政府呼吁此前宣布暂停运营巴尔博亚港的中远海运考虑重启当地货物航运业务。
截至去年12月底的2025年度财报，长和录得净利润118亿港元（19.28亿新元），低于分析师预估的217亿港元。业绩受非现金损失影响较大，全年营收为5073亿港元，上年同期为4767亿港元。

涉向中国走私AI技术 超微电脑高层被美司法部起诉
美国超微电脑（Super Micro Computer）高层被控将受出口管制的英伟达晶片等尖端人工智能技术非法转移至中国，并从中获利数十亿美元。
综合路透社 、法新社等媒体报道，美国司法部星期四（3月19日）指控，超微电脑联合创始人廖义贤（Yih-Shyan Liaw，音译） ，以及员工张瑞增（Ruei-Tsang Chang，音译）、孙廷伟（Ting-Wei Sun，音译） 串谋，将内置高性能英伟达（Nvidia）GPU的计算机服务器走私至中国。
美国纽约南区联邦检察官克莱顿（Jay Clayton）在一份声明中表示，被告参与了一项系统性计划，将大量美国人工智能技术转移给中国客户，并以谎言掩饰和隐瞒此行为。
根据起诉书，这三人自约两年前起与他人共谋，通过一家位于东南亚的“中转”公司，向中国销售至少价值25亿美元（32亿新元）的服务器。而美国法律禁止未经许可向中国销售此类设备。
总部位于加利福尼亚州圣何塞的超威电脑表示，已于星期四收到起诉的通知。公司本身未被列为被告。公司已将廖义贤和张瑞增停职，并终止与合同工孙廷伟的合作。
美国司法部表示，美国籍的廖义贤和台湾人孙廷伟已被逮捕，台湾人张瑞增仍在逃。

中国大陆学生意向留学目的地 香港超越美国
中国教育机构新东方发布的最新报告显示，在中国大陆留学生意向留学目的地方面，香港今年首次超越美国。
综合《新京报》《中国青年报》报道，新东方星期二（3月17日）发布的《中国学生出国留学发展报告》显示，英国连续七年稳居意向留学目的地榜首，香港近12年来排名持续上升，美国则受政策波动影响滑至第三。
新东方教育科技集团助理副总裁、新东方前途出国副总裁俞仲秋说，香港特区政府不断推出利好政策。“比如港八大的非本地生配额，已经从20%上升到40%-50%，每年差不多可以有7500人。而非本地生中超过60%都来自中国内地。”
报告也显示，2026年中国意向留学家庭的平均预算突破60万元（人民币，11万新元），创12年新高，主要原因是受全球通胀影响。这一变化直接影响了留学生的申请策略，超过半数的意向留学生计划同时申请两至三个国家和地区的高校，降低单一国家政策波动、录取不确定性等风险，并拓宽择校的弹性空间。
受全球经济环境影响，学费跃升为第二大择校因素，奖学金政策也成为重要考量维度。此外，学校排名、适配专业与毕业后就业率依然是关注度高的因素。这一数据说明留学生和家长择校越来越理性，不再盲目追求名校，形成了更务实、更看重性价比的择校标准。
此外，意向留学生的申请和留学考试准备越来越前置，提前两年及以上开始准备的学生占比接近半数。
这也是新东方连续第12年发布该报告。本次问卷调研范围广泛覆盖中国33个省级行政区域及部分海外地区和国家，受访对象涵盖有留学意向的学生和家长，以及有留学经验的学生、参与单位招聘的社会人士，共收回有效样本6904份。

能源危机若持续到4月 沙特预测油价恐升至180美元
如果霍尔木兹海峡危机对能源的冲击持续到4月以后，沙特阿拉伯预计油价恐飙升至每桶180美元（约230新元）。如此高的油价可能引发经济衰退或消费者行为改变，从而导致对石油的需求骤减。
《华尔街日报》星期四（3月19日）报道，沙特石油官员正在紧急评估，如果伊朗战争以及这场战争对能源供应的干扰不尽快结束，油价可能飙升到多高——他们对目前形势深感不安。
这个波斯湾最大产油国的几名石油官员说，他们的基本预测是，如果供应中断持续到4月底，油价可能会飙升至每桶180美元以上。这对于一个仍然严重依赖石油收入的国家来说似乎是一笔意外之财，但实际上却令人堪忧。
如此高的油价可能会促使消费者养成大幅削减石油消费的习惯，而且这种习惯可能长期持续下去；甚至引发经济衰退，进一步打击需求。
沙特在这场并非由自己挑起的战争中，还可能被视为趁火打劫者。
费萨尔国王学术及伊斯兰研究中心的沙特外交政策和地缘政治分析师卡里姆（Umer Karim）说：“沙特通常不喜欢油价上涨过快，因为那会造成长期的市场不稳定。”
他说：“对沙特来说，油价适度上涨，同时确保他们的市场份额保持稳定才是理想情况。”
负责沙特石油生产、销售、定价的沙特国家石油公司沙特阿美（Saudi Aramco），拒绝对《华尔街日报》报道置评。
伊朗过去一周针对中东邻国能源设施的袭击推高了油价。为报复以色列18日对伊朗南帕尔斯天然气田的袭击，德黑兰袭击了卡塔尔拉斯拉凡能源枢纽的设施，还攻击其他波斯湾地区的基础设施，包括沙特延布的Samref炼油厂。
沙特西部港口延布是沙特红海沿岸的原油管道终点站，这条原油管道可以绕过霍尔木兹海峡输送原油。美以伊战争爆发后，石油航运要道霍尔木兹海峡被伊朗实际封锁，引发石油市场历史上最严重的供应中断，促使全球油价飙升。沙特紧急调整石油运输路线，改为经由一条长达1200公里的管道，将原油输送到延布，同时迅速集结一支庞大的油轮舰队，驶向红海装载原油。

中国将道路运输从业人员年龄上限放宽至63岁
中国决定将道路运输从业人员年龄上限，放宽至63岁。
据中国央视新闻报道，根据交通运输部修订《道路运输从业人员管理规定》的决定，星期五（3月20日）起，经营性道路客货运输驾驶员、道路危险货物运输从业人员等岗位从业年龄上限，由60周岁放宽至63周岁，同步调整从业资格证注销条件。
另据澎湃新闻报道，为积极回应道路运输从业人员关切，交通运输部修改了《道路运输从业人员管理规定》等五个部门规章，将经营性道路客货运输驾驶员、道路危险货物运输从业人员、驾驶操作教练员等相关道路运输从业人员年龄要求统一调整为“年龄不超过63周岁”，并于星期五起正式施行。
为确保新旧规定平稳过渡，交通运输部部署了一系列便民服务措施，其中各省级交通运输主管部门将对因年龄超过60周岁但未满63周岁而被注销从业资格证的人员，统一在系统层面恢复证件数据，并将证件有效期延长至63周岁。澎湃新闻也报道，相关从业人员无需重新申领或换证，避免增加群众负担。
