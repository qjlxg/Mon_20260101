中国社科院报告：中企投资拉美中东等地区风险较高
一项报告显示，拉丁美洲、中东等地区政治稳定性相对不足，中国企业投资风险较高。
据中新网报道，《中国海外投资国家风险评级报告(2026)》暨主要地区地缘政治风险前瞻研讨会星期三（4月8日）在中国社会科学院世界经济与政治研究所举行。
本年度报告涵盖120个样本国家，构建经济基础、偿债压力、社会弹性、政治风险和对华关系五大指标共42个子指标的评级体系，量化评估中国企业海外投资所面临的国别风险。
结果显示，发达经济体的整体风险较低，在20个低风险国家中占据19席。澳大利亚、德国、瑞士等国的投资风险相对较低。乌兹别克斯坦、巴基斯坦、几内亚、尼日利亚等国的投资风险相对下降。拉丁美洲、中东等地区政治稳定性相对不足，投资风险较高。
与会专家还对美国、欧洲、非洲、东南亚、中东等地的投资风险进行研讨。中国社会科学院世界经济与政治研究所副研究员潘圆圆认为，美国当前对外国投资的需求主要表现为：需外部资金持续流入为赤字融资，期待实现投资资金与技术转移的分离，降低融资成本、提高资产收益等。
中国社会科学院欧洲研究所研究员赵晨认为，欧洲投资环境受伊朗局势与通胀冲击、地缘政治收紧等约束，整体呈“谨慎结构性机会、合规优先、区域赛道分化”格局。宜关注新能源、AI等赛道和南欧国家政策包容的机遇，关注欧洲整体滞胀压力与财政状况。
中国社会科学院西亚非洲研究所副研究员朱泉钢认为，面对伊朗局势带来的短期、长期风险，中企需构建“底线思维”与“机遇意识”并重的策略框架，以“短期控风险、中期转模式、长期抓机遇”思路加以应对。

## 来源: WorldSpotNews

前OpenAI市场拓展负责人扎克·卡斯（Zack Kass）表示，传统企业文化和森严的组织层级正在放慢中国企业对人工智能的采纳步伐，使企业端落后于美国同行；与此同时，中国消费者对AI的接受速度非常快。
卡斯将中美差异概括为“中国是以消费者为中心的技术文化（techno-centric consumer），美国则更偏向以企业为中心的技术文化（techno-centric enterprise）”。他指出，中国消费者普遍比美国更热衷新技术，这与科技在中国推动脱贫和壮大中产阶级的作用有关。
卡斯在2021至2023年间任职OpenAI并参与了2022年11月ChatGPT的推出，目前经营自己的咨询公司。他还称，美国消费者对大科技更为怀疑，担忧社交媒体对儿童的影响及公众信任下降，这也解释了为何一些AI应用在中美两地的热度差异。
以OpenClaw为例，上月约有近千人在腾讯深圳总部排队申请免费安装，本地社交平台上也出现收取数百元安装费的服务，反映出中国市场对AI工具的强烈需求。

Meta宣布聘请中国资深高管许锐（Xu Rui）出任旗下超智能单元的AI硬件团队负责人，负责开发独立于Reality Labs的AI设备团队。该消息由Business Insider披露，南华早报亦对其背景进行了梳理。
许锐毕业于哈尔滨工业大学与首尔国立大学，历任多家中国大型科技公司的关键岗位。2015至2017年，他在小米担任全球拓展总监，推动公司智能电视海外布局并与谷歌达成将首款Android设备引入美国的合作。
2020至2022年，许锐在字节跳动旗下硬件部门New Stone Lab任总经理，期间推出包括Dali智能台灯在内的多款消费产品；2022至2024年曾在腾讯的扩展现实（XR）设备团队任职。2024年他加盟帕洛阿尔托的人形机器人初创公司K-Scale Labs，该公司于11月宣布破产，随后他加入AI代理初创公司Dreamer，后者于今年3月被Meta收购。
业内认为，Meta此番任命反映其在AI硬件领域的持续投入与竞争布局。

港股上市、吉利入资的曹操出行押注重资产、车队优先战略，力争在中国机器人出租车赛道中脱颖而出，目标到2030年部署10万辆自动驾驶车辆。 
公司CEO龚欣在接受南华早报采访时表示，机器人出租车的未来取决于以车辆制造、自动驾驶技术和车队运营构成的闭环“金三角”资产管理模式。曹操正在完善其L4级自动驾驶系统，并于2025年底在杭州先行投放了100辆机器人出租车；尽管中国多数车辆仍需安全员看守，曹操计划今年实现无人化运营。 
4月1日，公司获批在杭州开展无人路测，成为该市首家获批企业。面对Pony.ai、WeRide等本土竞争者，曹操强调其“完全为机器人出租车打造”的车型是核心竞争力：该车过去两年从头设计、软硬件高度集成，预计今年亮相并于2027年上半年进入量产。

在全球算力与AI代币供给紧张的背景下，中国AI厂商就美国初创公司Anthropic将其Claude模型从开源代理工具OpenClaw中撤出，展开公开交锋。周日Anthropic宣布，Claude订阅将不再覆盖OpenClaw等第三方工具，称此举是为“优先保障现有客户”。
小米与上海公司MiniMax迅速呼吁用户转投自家代币订阅方案。MiniMax在X上指责Anthropic的限制会扼杀实验性创新，认为“更多好点子来自实验室之外”。
观察者指出，AI代理带动了对AI代币的大量需求，全球算力收紧令行业能否长期满足这一需求成为疑问。报道：记者 Vincent Chow。

中国为在人工智能领域取得突破、应对与美国加剧的科技竞争，正加紧从海外吸纳半导体人才；对此，台湾当局也强化对大陆疑似挖角行为的调查。
台湾法务部调查局（MJIB）上周一表示，已对11家大陆企业展开调查，控诉这些企业以隐瞒大陆背景、设立空壳公司并未经核准在台开展业务等方式，非法挖角半导体及其他高科技人才。MJIB称，自2020年以来已处理约100起相关案件；去年也曾调查大陆最大晶圆代工厂中芯国际（SMIC）涉嫌挖角案。
加拿大顾问机构The Geopolitical Business地缘战略师阿比舒尔·普拉卡什（Abishur Prakash）称，这是一场“低调的科技战”，与美中围绕出口管制和资本等议题的公开对峙不同，大陆更着眼于推动下一波AI创新所需的人才。台湾当局对此高度警觉，调查行动反映出大陆企业对具备经验的人才在半导体与AI领域的强烈需求，即便面临美国对高端技术的限制，这类人才流动仍在加剧。

多家中国企业包括阿里云和知谱科技近期选择不将部分最新人工智能模型开源，转而通过官方付费渠道获取使用价值。 
本周阿里发布三款仅能在其云平台或聊天机器人网站访问的专有模型，其中包括增强编程能力的Qwen3.6-Plus与可处理文本、音频、图像和视频的多模态模型Qwen3.5-Omni。 
今年9月发布的上一代多模态模型Qwen3-Omni为开源版本，但阿里表示新版不会开源，理由是基于开源平台Hugging Face的下载数据，Omni系列在开发者中的受欢迎程度较低。 
业内认为，顶级模型规模不断增大、难以在本地硬件上部署，使厂商更倾向于通过集中托管和商业化服务变现，同时并未明确放弃开源策略。 
阿里云为阿里巴巴集团旗下的人工智能与云计算业务单元，阿里巴巴集团为《南华早报》的所有者。

伊朗民众在全国多个桥梁与发电站周边自发组成“人链”，以防这些设施成为攻击目标，国家媒体周二报道。
Press TV发布的画面显示，大批民众举旗集结在发电厂和桥梁外，数千人围成疑似“人盾”。此举发生在美国总统特朗普警告称，若德黑兰不接受其要求结束战争，美国将对包括桥梁和发电厂在内的关键基础设施发动打击之后。
伊朗政府此前呼吁平民、青年、艺术家和运动员参与。最高青年与青少年委员会秘书阿里雷扎·拉希米表示，组成人链的想法由青年提出，活动名为“伊朗青年为光明未来的人链”，旨在展示团结与决心。
特朗普在社交平台Truth Social上再度发出警告，称“一个文明今晚将消失”（“a whole civilisation will die tonight”），并提及“全面彻底的政权更替”等言论，令局势更趋紧张。
-- 通讯社

阿里巴巴集团在CEO Eddie Wu Yongming 领导下成立高层技术委员会，以加速人工智能研发并应对国内模型与智能代理领域日趋激烈的竞争。 
据吴在周三发给员工的内部信，杭州总部设立的阿里集团技术委员会将汇聚多位高层，统筹集团AI战略与技术路线。委员会成员包括前阿里云CTO Zhou Jingren，后者将出任首席AI架构师；集团CTO Wu Zeming 将聚焦业务技术与AI推理平台，此前曾负责外卖业务。 
同时，Li Feifei 被任命为阿里云技术与AI基础设施负责人，并兼任云CTO，任命自周三生效。此次重组还延续了此前成立的Alibaba Token Hub (ATH) 业务群，以整合公司核心AI团队与产品。通义实验室（Qwen系列大模型开发者）亦升级为通义大模型业务单元，由Zhou领导，正式固化其在模型研发中的核心地位。

周三，中国人工智能公司智谱AI开源其最新旗舰模型GLM-5.1，并同步将API价格上调10%，显示在与美国对手竞争加剧的背景下，开始加强对高级模型的商业化变现。此次为公司今年第二次涨价：2月对编程订阅的调整曾上调逾30%，本次幅度较小但覆盖包括云端输入token在内的多项计费类别。消息发布后，智谱港股盘中上涨约11.5%。
定价方面，智谱对输入token每百万收费约1.40美元、输出约4.40美元；而Anthropic的Claude Opus 4.6截至2026年2月分别为每百万输入5美元、输出25美元，差距仍在。GLM-5.1上月末先作为付费编程订阅产品推出，此举引发其从开源策略向更强调变现的猜测。
基准测试机构Artificial Analysis认为，按综合智能指标计算，GLM-5.1为全球最强开源模型，领先国内竞争者如MiniMax，但仍落后于OpenAI与Anthropic的顶级模型。智谱在X上称，GLM-5.1可在复杂工作流中持续运行约8小时，能从零构建可用的Linux桌面，反映对可长时自治运行的agent型AI工具的需求。
智谱定位为中国硬件生态中的重要软件合作方，并称已与多家国产芯片企业合作适配其GLM模型，包括华为、摩尔线程与寒武纪。摩尔线程表示其MTT S5000服务器已实现对GLM-5.1的“Day-0”兼容性。
智谱CEO张鹏在3月中关村论坛表示，依赖低价竞争不利行业长期发展，提价旨在建立“更健康的商业闭环”，支持高质量AI服务的长期交付；小米AI模型负责人罗福立也表达了类似看法。
