研究与学界人士称，两巨头的监管执行“不均衡且不透明”，搜索与推荐机制可能放大有争议用途。开发者示例包括曾在谷歌商店上架、下载量逾百万的“Video Face Swap AI: DeepFace”，其内含可将人脸贴到带性化动作的模板上。涉事开发商称未提供“去衣”功能，已删除部分用户上传内容。
随着监管压力上升，多国正推动更严格措施：美国于2025年通过《下架法案》（Take It Down Act），将非自愿性内容传播定为犯罪；英国今年4月亦拟立法，追究未删除相关图片的科技公司高管责任。来源：彭博社

算完了吗？特朗普将伊朗战争描绘为“已成定局”却遭现实碰撞
华盛顿 2026年4月16日 12:00（更新 12:11）
总统唐纳德·特朗普试图将对伊朗的军事行动定性为一场已告一段落的胜利。白宫方面强调行动成果，但多名分析人士警告，事态发展可能并非如官方所述——战争反而可能提升了伊朗革命卫队在该地区的影响力。

新加坡讯 — 尽管美国自4月13日对到访伊朗港口的船只实施封锁，航运数据显示第二艘遭美方制裁的超大型油轮仍通过霍尔木兹海峡进入波斯湾。
美国总统唐纳德·特朗普在4月12日美伊在伊斯兰堡举行的会谈未达成协议后宣布封锁。美中央司令部在X上称，自封锁开始以来已有10艘船只被迫掉头，未有船只突破封锁。与此同时，伊朗法尔斯通讯社4月15日报道称，一艘受美方制裁的伊朗超大型油轮在封锁下向伊玛目霍梅尼港方向穿越海峡，未披露具体船名或航次细节。
路透援引LSEG与Kpler数据称，空载超大型原油运输船（VLCC）RHN于4月15日进入波斯湾，该船可载约200万桶原油，但目前航向不明。此前一天，另一艘受美制裁的VLCC Alicia已通过霍尔木兹海峡，Kpler显示其正驶往伊拉克。两船近年均有装运伊朗原油记录。被迫折返的船只包括受制裁的Rich Starry，该船在离开一天后于4月15日返回波斯湾。
华盛顿曾警告可能对购买伊朗原油者实施二级制裁，以在后续谈判中争取筹码——此举发生在美国数周前放松对部分伊朗能源制裁执法之后。有熟悉德黑兰立场的消息人士称，若能达成防止冲突重燃的协议，伊朗可考虑允许船舶自由通过霍尔木兹海峡的阿曼一侧以免遭袭。
分析师认为，美国封锁将压缩伊朗原油出口，但产油国可通过陆上储罐维持约350万桶/日的产量数周。Kpler数据称，伊朗3月原油出口约184万桶/日，4月迄今约171万桶/日，2025年平均为168万桶/日。 路透社

华盛顿 — 美国军方4月15日表示，在对一艘疑似贩毒船只的打击中有三人遇害；这是连续五天发生的第五起致命攻击。
美南方司令部在X平台发文称，对“由指定恐怖组织运营的船只”实施了“致命的动能打击”，但未点名具体组织，并称“行动中三名男性‘毒贩恐怖分子’被击毙”。
法新社汇总的统计显示，此轮打击使死亡人数至少达到177人。
特朗普政府坚持认为正与在拉美活动的“毒贩恐怖分子”处于实质战争状态，但未提供确凿证据证明所击中船只涉毒，这引发了针对行动合法性的激烈争论。国际法律专家与人权组织认为，这些打击可能构成法外处决，因为目标似乎为不对美国构成直接威胁的平民。——法新社

休斯顿消息：受伊朗战争导致霍尔木兹海峡通行受限影响，上周美国几乎成为自二战以来首次原油净出口国。  
美国能源部于4月15日公布的周度数据显示，上周原油净进口（进口减出口）缩窄至每日6.6万桶，为自2001年有周度数据以来的最低值；同期出口攀升至每日520万桶，为七个月来高点。  
数据显示，按年计算，美国上次成为原油净出口国要追溯到1943年。分析人士称，伊朗与以色列的冲突造成约五分之一的全球石油与天然气贸易受阻，迫使亚洲与欧洲炼厂广泛寻求替代货源，从而大幅推高对美国原油的需求。  
船运跟踪公司Kpler称，上周约240万桶/日（占美国出口的47%）流向欧洲，约149万桶/日（占37%）流向亚洲，主要 买家包括荷兰、法国、德国、日本与韩国；一艘载50万桶的油轮正在驶往土耳其，为近一年首批对该国的美国原油出口。  
与此同时，美国进口下降超过100万桶/日至530万桶/日，因美国炼厂多以加工重硫原油为主，而本国多产轻甜原油。布伦特与WTI价差扩大（3月一度达每桶20.69美元），令美国原油对欧洲和亚洲炼厂更具吸引力；现货货轮运往欧洲的原油价在4月13日接近每桶150美元。  
但分析师警告，美国出口正接近运输与管道能力上限。贸易商与研究机构估算最大出口能力约600万桶/日，且受船舶短缺与运费上升制约。释放战略石油储备中的中硫原油或增加可供出口的轻质低硫原油，但船运瓶颈可能抑制后续出口增长。
