不过，由于历届阿根廷政府遗留下来的财务与技术问题，这一项目自2023年底起陷入停滞。据估计，塞佩尔尼克水电站工程已完成超过40%，而配套建设的基什内尔（Néstor Kirchner）水电站约完成五分之一。

全球升温步入“快车道” 1.5摄氏度恐在2030年前失守
（马德里彭博电）一项国际研究显示，地球过去10年变暖的速度明显加快，而且超过有记录以来的任何一个10年。
这项星期五（3月6日）发表在《地球物理研究通讯》期刊的研究指出，2016年至2025年的10年间，全球升温约0.35摄氏度；1970年至2015年的平均升温速度仅为每10年略低于0.2摄氏度。研究报告作者说，这是首次在统计上明确显示全球变暖正在加速。
过去三年是有记录以来最热的三年，2024年的平均升温幅度已超出《巴黎协定》设定的1.5摄氏度下限。由于大自然中存在厄尔尼诺现象、火山喷发和太阳辐射等现象对气候的影响，因此科学家很难断定近年来的极端高温是短期波动还是长期趋势。
研究报告合著者、美国统计专家福斯特（Grant Foster）说，研究小组通过统计学方法过滤了这些“杂音”，使潜在的长期变暖信号更加清晰。结果证实，异常炎热的2023年和2024年并非孤立现象，而是自2015年以来，全球气温便已脱离了此前较慢的升温趋势。
波茨坦气候影响研究所研究员拉姆斯托夫说：“如果过去10年的变暖速度持续下去，全球长期升温幅度将在2030年之前突破《巴黎协定》设定的1.5摄氏度限制。 
“地球变暖的速度最终取决于我们以多快的速度，把化石燃料产生的二氧化碳排放量降至零。”

布伦特油价飙升至92.62美元/桶，WTI触及90.72美元/桶，市场担忧源自伊朗相关冲突导致航运受阻和产量中断。更新时间：2026年3月7日，编辑：Jay Hilotin。
自1月以来，油价已上涨约40–42%：1月布伦特均价约64–67美元/桶，至3月初接近92美元/桶。Murban原油升至102.20美元/桶（周五上涨6.39%），天然气报3.116美元/百万英热单位，美国汽油价约3.15美元/加仑，达特朗普任内高位。
路透/AFP分析显示，霍尔木兹海峡通行严重受阻——自本周一以来仅记录9艘油轮、货船通过，且有船只遭袭后通航几近中断。该海峡通常承载全球近20%的原油及液化天然气运输。供应担忧推动市场戒备，持续高位恐助长通胀并压制经济增长。
分析师警告，若霍尔木兹长期关闭，布伦特可能上探100–108美元/桶。缓解措施包括通过谈判迅速平息冲突、美国海军护航（特朗普3月3日下令）、提供保险补贴、提高非OPEC产量或释放战略储备。短期仍将高度波动，但供应多元化可减缓长期冲击。

马尼拉——进步派“马卡巴彦”议员团驳斥副总统萨拉·杜特尔特将弹劾程序称为“阴谋”的说法，呼吁她正面回应众议院司法委员会关于涉嫌滥用机密经费的调查。
马卡巴彦由ACT教师党众议员安东尼奥·蒂尼奥、加布里埃拉妇女党众议员萨拉·伊拉戈和青年党众议员蕾妮·寇组成，批评杜特尔特把国会监督贬为“钓鱼行动”，称这是“老调重弹”“想以受害者姿态逃避问责”。他们强调机密经费属公款，数额巨大且审批仓促、保密，公众有权追问用途与防弊机制。
议员团指出，杜特尔特反复援引审计署（COA）对副总统办公室出具的无保留意见并不能证明没有不当行为，尤其是在支出分类限制披露的情况下。
马卡巴彦还批评杜特尔特以“红色标签”攻击议员，敦促其“别再谈什么阴谋，正面面对程序”。
司法委员会已就两起被认定“实质足够”的弹劾案向杜特尔特下达通知，要求她在10天内答辩。两起控诉除涉嫌滥用机密经费外，还指控贿赂教育部官员、未解释资产来源及财产申报遗漏，并指称曾雇凶谋杀总统、第一夫人及前众议院议长等严重指控。
去年2月众议院曾对杜特尔特提出弹劾，但最高法院以违反一年时效限制为由裁定相关弹劾条款违宪。

泰创泰党发言人Pariyes Angoorakit于3月7日表示，关切因เตรียมอุดมศึกษา入学考试引发的“把贵重物品与手机寄放在考场外”争议，对学生、教师及考务人员的心理影响。尽管已设置寄物点，但因考生人数众多、由学校单独负责场地管理，仍造成混乱与巨大压力。

2026年3月7日，前商务部及前能源部长披猜·纳里帕塔潘（Pichai Naripthaphan）就中东冲突对全球油价影响表示关切。他称，美以联合攻击伊朗并导致霍尔木兹海峡封闭，使国际原油价格快速上涨至约每桶100美元，政府需密切应对。
披猜支持总理安努廷将柴油价控在29.94铢的做法，指出柴油是运输成本核心，若任由上涨将带动各项商品涨价，民众受苦。他建议商务部加强查货与市场管制，防止囤积与趁机涨价；若冲突持续，可考虑将柴油特约消费税下调每升6.92铢以缓解民众负担，事后再恢复征税。
他并提出三项可把危机转为机遇的政策：一、推进“粮食安全”计划，利用泰国作为全球重要粮食供给国的优势，扩大储备与出口（并已获新加坡、阿联酋、卡塔尔等国兴趣）；二、发展面向中东的国际数据中心与AI产业，吸引大量投资；三、建设南部陆桥并扩大石油储备，以规避霍尔木兹航道风险。披猜认为，政府应尽快整合资源，抓住这三项机会，既稳住民生也推动国家长期发展。
