中国欧盟商会：欧盟企业因出口管制重新评估对华稀土依赖
（北京讯）中国欧盟商会星期二（4月14日）发布报告称，北京对稀土元素的出口管制正引发巨大不确定性，迫使欧洲企业重新评估其全球供应链对中国的暴露程度。
报告指出，出口管制在2025年成为中欧关系的关键议题。在中美贸易战持续升级的背景下，中国决定对部分稀土元素实施新的出口管制，导致多家欧洲企业遭遇供应中断，营收受损。
中国去年10月宣布加强稀土相关物项与技术的出口管制，并首次对境外相关稀土物项实施出口管制。 
报告称， 40%的受访欧洲企业透露，中国出口许可审批流程令去年的交付周期延长了两个月以上。虽然情况今年以来有所改善，“但在许多情况下，许可流程仍存在效率低下、不可预测、缺乏协调及透明度不足等问题”。
36%的受访企业计划积极与中国以外的供应商合作，以扩大产能，另一途径是限制材料的需求量，但报告也指出，由于中国稀土元素的成本优势，多数企业仍难以摆脱对中国的依赖。
中国欧盟商会警告，这类管制措施一旦正式生效，或将对全球供应链造成冲击，迫使其他国家及企业采取适当应对举措，长此以往，将“进一步对中欧双方造成不利影响”。
中国欧盟商会主席彦辞称，在国际环境日益动荡的局势下，中国应依托自身优势，向包括欧盟在内的贸易伙伴表明：中国是值得各方信赖的关键商品供应来源地。
他呼吁北京对出口管制实施精准把控，严防两用物项用于军事用途，维护民用贸易秩序，这将“更有利于达成互利共赢的局面”。

韩国启动对伊谈判营救26艘船只 特朗普“逆封锁”增添变数
韩国政府正加紧展开外交协调，力图营救被困霍尔木兹海峡的26艘韩籍船只。韩国已向伊朗提供船舶详细信息，并着手启动正式谈判机制。韩国拟于本周内敲定对中东追加人道援助方案。然而，美国总统特朗普提出的“逆封锁”措施为局势增添变数，相关外交斡旋面临更大挑战。
韩国《中央日报》14日报道，目前滞留在霍尔木兹海峡的韩国籍船只共26艘，包括在外国船只上工作的韩国船员在内，共有173名韩国人员受影响。为协助船只安全撤离，韩国政府近期已向伊朗方面提供包括船名、国籍、航运信息及国际海事组织（IMO）注册编号等在内的详细资料。
分析认为，在无人机袭击及扣押风险仍存在的情况下，此举有助于降低误判与冲突风险。
据了解，在冲突初期，韩国对是否提供船舶信息一直持谨慎态度。3月23日韩伊外长通话时，伊朗方面曾明确提出相关要求，并希望韩国协助改善人道主义状况。鉴于难以确保通航安全且存在信息被滥用的可能，韩方当时未积极回应。
不过，本月7日美伊达成停战两周安排，随着部分航道逐步恢复通行，韩国政府开始调整策略。在坚持“保障所有船舶通航自由”的原则下，韩方将外交重点转向优先确保本国船舶安全撤离，并以提供船舶信息作为启动实质性协商的必要前提。
韩国政府同步加强外交行动，派遣特使赴中东，与伊朗高层就地区局势及韩国国民、船舶和船员安全问题展开磋商。外交人士称，当前重点在于通过直接沟通，尽快协助韩方船只脱离高风险海域。
然而，特朗普政府近期提出限制伊朗相关航运的“逆封锁”措施，使局势再趋紧张。为降低潜在风险，外交部相关人士称，双边协商与多边合作并行推进，“在考虑谈判破裂等多种可能性的同时，与有关国家保持密切沟通”。
包括韩国在内的40多个国家计划在星期三（15日）通过副部长级视频会议，就霍尔木兹海峡通航问题展开协调。
韩国政府相关人士指出，韩国被困的26艘船只都不涉及伊朗原油运输，原则上不属于美方限制措施的适用范围，但随着局势变化，整体环境趋于紧张，“局面确实更加僵化”。
