机场方面，圣周期间旅客流量也将上升。马尼拉国际机场管理局预计NAIA各航站量将达139.3万人次，高于去年同期132.6万。为优化服务，自3月29日起将有5家航空由一号航站调至三号航站，2家由三号调回一号，另有一次4月1日的调动。新NAIA基础设施公司表示，此次航站调整为提升机场运作的一部分。

马尼拉——菲律宾大气地球物理与天文服务管理局（PAGASA）表示，周日科塔巴托市（Cotabato City）录得最高体感温度41°C，为全国当日最高。
大马尼拉地区体感温度则在36至37°C之间。
PAGASA补充指出，周一（3月30日）全国最高体感温度预计可达约40°C，可能出现在达古潘市（Dagupan City）、西民都洛省的圣何塞（San Jose）、内格罗斯西部的La Granja及三宝颜市（Zamboanga City）。
预计未来几天天气仍以晴朗并伴有多云变化为主。
