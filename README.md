韩国地方选举投票率疑遭篡改 调查人员搜查选委会总部
（首尔法新电）韩国调查人员星期四搜查选举委员会办公室，调查选举官员涉嫌在6月的地方选举投票率数据造假的案件。
韩国6月3日举行地方选举，当时数十个投票站发生前所未见的选票短缺，至少有90名选民未能投票。 
这起争议导致中央选举管理委员会委员长卢泰岳辞职，并引发要求重新选举的示威。韩国总统李在明下令彻查此事。

调查组说，两名选举官员已因涉嫌伪造官方电子记录等罪名接受调查，并将进一步查明有关操纵行为是否涉及选举高层官员。

美众院微差通过国防授权法案 拨款1.15万亿美元历来最高
（华盛顿综合电）美国众议院星期三（7月22日）以216票赞成、212票反对的微差通过2027财年的国防授权法案，批准历来最高的1.15万亿美元（约1.5万亿新元）国防拨款。
此次投票基本依循党派路线，共和党除了七名议员投反对票，其余均投赞成票；民主党则有六名议员投赞成票，其余投了反对票。
美国国会已连续65年通过这个年度法案，不过今年两党分歧严重，民主党参议员上周已阻挡参议院版本的法案通过。他们反对特朗普和共和党人在大幅削减社会支出的当儿，还批准如此大笔国防拨款，同时也对伊朗战争至今花费高达375亿美元深感不满。
支持法案的共和党人说，这笔万亿美元的国防拨款至关重要，将支持美军在伊朗的军事行动，资助防御系统研发与军备采购，并改善军人待遇，包括加薪、建造营房和家属住房。
法案还将扩大美国和以色列在军事技术领域的合作。然而不少美国人认为，以军袭击在加沙地带已造成大量巴勒斯坦平民伤亡，美国不应加大对以色列的军事支持。
按照立法程序，参众两院须分别通过各自版本的国防授权法案，随后由两院军事委员会协商达成一个折中版本，再交由两院重新表决。法案的最终版本通过后，才送交白宫由总统决定要签署生效或是否决。
众议院同天也以216票对214票的微差，批准了一笔950亿美元的预算，当中730亿美元用于伊朗战争，120亿美元用于援助被特朗普关税战影响的美国农民，其余100亿美元用于选举相关的计划。法案接下来将交由参议院审议和表决。
众议院议长约翰逊称，这笔预算将协助共和党实现两个目标，即保障美国选举安全和巩固本土安全。民主党众议员德劳罗则反对法案，称这将允许特朗普继续为伊朗战争提供资金。
这是共和党在11月中期选举前推动的最后一项重大法案，中期选举结果将决定共和党能否继续保住国会多数党地位。
