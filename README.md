非洲国家向中国偿还的资金超过新增借款

吉林退出中国地方债务重点省份
中国吉林成功退出地方债务重点省份。
据第一财经报道，吉林省人民政府新闻办公室星期二（1月27日）披露的2026年吉林省政府工作报告（图解版）内容显示，当地退出地方债务重点省份。
报道称，此前中国为了防范地方债务风险，将12个省份纳入债务高风险地区名单。
中国中央财经大学教授温来成去年向《每日经济新闻》说，在政府投资领域，此前被纳入12个高风险名单的地区，一般情况下不得新增政府投资项目，即便有确实需要的项目，审批流程也大幅收紧；原本由省政府即可批准的项目，需上报国家发展改革委乃至国务院审批，难度显著增加。而退出高风险名单后，审批难度将大幅降低。
为了化解地方债务风险，中国2024年10月推出了一揽子总额12万亿元人民币（2.19万亿新元）化债方案。受此影响，近年来各省份地方政府隐性债务快速削减。
