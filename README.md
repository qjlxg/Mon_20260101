斯德哥尔摩——瑞典政府一项委员会于6月2日建议禁止15岁以下儿童使用社交媒体，加入各国应对互联网对未成年人危害的行列。公共卫生部长雅各布·福斯梅德在新闻发布会上警告：“我们正逐渐失去整整一代人给无尽的滚动浏览。”
该委员会去年秋天受政府委托，评估设定社交媒体最低使用年龄，涉平台包括Snapchat、Instagram、Facebook、TikTok和YouTube。委员会负责人丽莎·恩格隆·克拉夫特建议将年龄下限定为15岁，并提议该规定自2028年1月1日生效。
社交媒体公司将被要求验证用户年龄，但像Roblox、Minecraft和Fortnite等受儿童欢迎的视频游戏则被排除在外。
澳大利亚在2025年12月率先实施类似禁令，旨在保护儿童心理健康，迄今成效褒贬不一。巴西和印度尼西亚等国已相继跟进，许多欧洲国家也在考虑类似措施。
