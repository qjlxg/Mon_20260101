他指出，这也预示北京未来将持续加强在印太地区，特别是针对“第一岛链”的军事演训与灰色地带行动，试图通过军事施压改变台海及区域现状。

在两岸交流方面，沈有忠呼吁台湾任何团体、个人或政党在进行两岸交流时，应看清北京加大对台统战与分化的意图，并称北京坚持的“九二共识”是“消灭中华民国的主张”，任何政党或个人都不应配合北京的政治操作。

为什么过去五年中国的军购暴跌72%？
