美内政部长：中国主导美太阳能供应链 须加强国安审查
美国内政部长伯古姆指出，中国在美国太阳能项目供应链中占据关键地位，因此有必要对相关项目展开更严格审查，以排查任何可能威胁国家安全的风险。
据彭博社报道，伯古姆在美国众议院拨款委员会作证时说：“许多太阳能项目几乎完全使用中国制造的太阳能板建成。因此，这就是一个国家安全问题。”
在被问及太阳能领域是否比其他能源领域面临更大风险时，伯古姆提到，美国基于国家安全考量，事实上已对中国科技巨头华为的大多数产品实施禁令。
美国内政部去年宣布，将对风能和太阳能项目实施更严格的审查程序，要求从运营计划到土地使用权等多个环节，均须获得伯古姆本人签字批准。可再生能源行业认为，此举实际上已令联邦土地上的清洁能源开发陷入停滞。
此外，美国内政部此前也曾以国家安全风险为由，对五个总值数十亿美元（10亿美元约等于12.7亿新元）、正在建设中的海上风电项目下达停工令，称这些设施可能干扰雷达系统，并增加美国遭受无人机攻击的风险。不过，相关停工令随后被法院推翻。

日本贸促会据报拟6月访华 盼与中国领导层会面
日本媒体报道，日本国际贸易促进协会计划6月访华，并期待在访问期间与中国领导层会面。
日本共同社星期一（4月20日）引述多名知情人士称，日本众议院前议长河野洋平担任会长的日本国际贸易促进协会代表团，拟于6月21日至24日访问中国首都北京，并已就此展开协调。代表团提出与中国领导层会面的意愿。
如果成行，这将是日本首相高市早苗去年11月在国会答辩发表“台湾有事”言论后，日本经贸代表团首次访华。
代表团计划配合6月22日至26日在北京举行的中国国际供应链促进博览会访华。预计将再次确认日中经济合作的重要性，并要求改善中方对日出口管制等悬而未决的议题。
日本贸促会是对华展开贸易与经济交流的窗口，往年都会组织访华团并与中方最高领导层会面。贸促会上次访华是在去年6月，会长河野洋平当时与中国国务院总理李强举行了会谈。在美中关税摩擦加剧的背景下，李强当时提出希望与日本加强经济合作，并回应称“重视”日方提出的租借大熊猫请求。
自高市早苗去年国会涉台言论以来，日中两国关系持续恶化，影响了经济交流。日中经济协会、日本经济团体联合会、日本商工会议所组成的经贸代表团原本计划今年1月访问北京，但直至去年12月未获中方答复，最终延期，至今尚无实施的眉目。
在日中紧张关系不见缓解的情况下，日本关西经济联合会会长松本正义原本考虑今年3月访华，但已在2月决定放弃行程。
不过，日本经济同友会计划5月小规模访华，日中经济协会也计划5月赴上海等地进行企业考察，显示日方探索恢复经贸交流的动向已出现。

中国指欧盟《网络安全法》修订草案将经贸问题政治化
针对欧盟《网络安全法》修订草案提出的意见，中国商务部回应时称，此草案内容存在多方面问题，将经贸问题政治化、泛安全化。
中国商务部发言人星期一（4月20日）在官网就上述问题答记者问时说，商务部上星期五（17日）向欧委会正式提交了对欧盟《网络安全法》修订草案的评论意见，表达了中方严正关切和正式立场。
发言人称，中方认为，草案以网络安全和供应链安全为名，引入极具主观随意性的“非技术风险”因素，特别是在草案中认定“网络安全关切国家”和“高风险供应商”名单，并在能源、交通、ICT等18个行业将列单的国家和供应商，全链条排除出欧盟相关供应链之外，这是经贸问题政治化、泛安全化的典型做法。
发言人指出，中方认为，草案内容存在多方面问题：一是涉嫌违反最惠国待遇、国民待遇等世贸组织基本原则，既违反《1994年关税与贸易总协定》《服务贸易总协定》《补贴与反补贴措施协定》《技术性贸易壁垒协定》等多项协定规则，也违背欧盟服务贸易减让承诺。
二是涉嫌超出欧盟法律授权，侵蚀成员国管理国家安全事务的专属权限。三是将对中欧经贸关系造成实质性伤害，对全球产供链造成严重冲击，也必将拖累欧盟自身的数字化、绿色化转型进程。
发言人提到，中方建议欧方删除草案中关于“网络安全关切国家”和“非技术风险”的规定，删除或实质修改“高风险供应商”认定标准及相关限制措施。希望欧方高度重视并认真考虑中方提交的评论意见和修改建议，严格遵守世贸组织规则，避免采取歧视性限制措施，维护中欧及全球产供链稳定畅通。
发言人强调，中方将密切关注草案修订进展，愿与欧方就此进行对话沟通。如欧方执意据此成法，并歧视性对待中国企业，中方将不得不采取相应反制措施。希望欧方不要低估中方维护国家利益和企业合法权益，并阻止中欧经贸关系倒退的坚定决心。
2026年1月20日，欧盟委员会今年1月20日正式发布《网络安全法》修订草案。此草案是欧盟强化网络安全防御能力、提升战略自主性的关键举措，旨在从技术合规治理转向涵盖地缘政治考量的供应链风险管理，如在能源、交通、ICT服务管理等关键行业排除“高风险供应商”，以系统应对地缘政治、供应链依赖等新兴网络风险。

查尔斯表示，尽管许多当下事态或许会使母亲忧心，但他相信“善终会胜利，黎明终将来临”，并重申年轻时伊丽莎白在14岁首次公开广播中呼吁“为使明天的世界更美好、更幸福而尽一份力”的信念。
为纪念百年，王室本周安排多项活动。4月20日，查尔斯与王后卡米拉参观了一场以女王服饰为主题的新展；4月21日，王室成员将在大英博物馆观摩纪念区的比例模型。首相基尔·斯塔默称，国家将以此纪念她非凡的在位并为后世提供沉思之地。
纪念碑由雕塑家马丁·詹宁斯为女王塑一尊年轻形象的铜像，身着礼服，灵感源自1955年意大利画家皮耶特罗·安尼戈尼为28岁女王所作的名画。前私人秘书、纪念委员会主席罗宾·詹夫林称该肖像“优雅而具标志性”。铜像连基座高约7.3米，置于圣詹姆士公园靠近白金汉宫的纪念入口区。园区将由一系列花园与曲径组成，并架设一座受女王婚礼冠冕启发的半透明玻璃“团结桥”；园内还将立一尊身着海军制服、仰望妻子的菲利普王子小塑像。
背景方面：伊丽莎白生于1926年4月21日，因舅父爱德华八世1936年为与两次离异的美国女子沃利斯·辛普森结婚而退位，历史巧合使她意外成为继承人。其父乔治六世去世后，她在1952年以25岁之龄登基，开启创纪录的君主生涯。
伊丽莎白逝世后，王室经历疾病、家庭疏离与丑闻考验：查尔斯与长子夫人凯瑟琳曾与癌症抗争；次子哈里与妻子梅根退位后出版揭露性回忆录；查尔斯之弟安德鲁因与已故美籍性罪犯杰弗里·爱泼斯坦的关联而面临警方调查。资料来源：法新社、路透社。

华盛顿 — 据法新社获得的书面发言稿，特朗普提名的美联储主席候选人凯文·沃什将在4月21日参议院银行委员会的确认听证会上强调货币政策的独立性。

华盛顿4月20日电——美国空军部长特洛伊·梅因克在社交媒体上宣布，将把A-10“疣猪”攻击机的服役期限延长至2030年，撤回此前计划于2026年退役的安排。梅因克表示，此举“在国防工业基础提高战机产能期间，保全作战能力”。
A-10自1976年首飞以来一直备受争议，但在对伊朗的当前冲突中发挥了作用。美国中央司令部和多方报道指出，该机曾在霍尔木兹海峡对伊朗舰艇使用其机鼻火炮。
支持者认为，A-10对地面部队的近距空中支援不可或缺；反对者则称其机龄已久、速度慢且维护成本高，退役可将经费用于高超音速等现代化优先项目。空军官员也警告，维持全量机队会挤占维修人员，影响新型战机的保障。
A-10难以被淘汰还因其强大的政治影响力——机队多数驻扎在亚利桑那州图森的戴维斯-蒙森空军基地，带动当地经济；亚利桑那为关键摇摆州，参议员马克·凯利在2021年曾成功推动国防立法阻止大规模退役。
分析认为，最新延寿决定显示，短期内保全作战能力优先于部队结构调整。来源：路透社

美国总统特朗普援引《国防生产法》为一系列能源项目提供联邦资金，以应对不断攀升的石油、汽油和电力成本。4月20日，他签署了五项总统裁定，重点涉及国内燃煤发电、液化天然气（LNG）、国内石油及电网基础设施，称这些领域的不足威胁国家防务。
该决定允许能源部动用去年税收与支出方案中拨付的资金，通过采购、财政支持等手段解决项目延误、融资短缺、监管阻碍及市场壁垒。可获支持的项目包括燃煤电厂、炼油厂及制造燃气轮机和变压器等电力设备的设施。
白宫发言人泰勒·罗杰斯称，此举有助于“全面释放美国能源主导地位，保护经济与国家安全”，并强调将加强电网基础设施、释放可靠、可负担且安全的能源。
白宫忧虑能源价格对选情的影响——伊朗战争与人工智能带动的用电增长均可能推高电费，危及共和党在年内中期选举中的国会控制权。特朗普还表示，燃煤发电、LNG和炼油产能对保障国防部署、工业扩张与盟友能源安全至关重要。
《国防生产法》赋予总统在国家防务需求下指令私营部门扩产的权力。拜登政府此前亦以该法推进太阳能板、变压器等能源技术的国内生产。彭博社
