美国24州联合起诉特朗普政府新关税政策
美国俄勒冈州联合另外23个州向美国国际贸易法院提起诉讼，要求阻止特朗普政府近期宣布实施的新关税措施。 
俄勒冈州司法部门星期四（3月5日）发布的声明说，一年多来，特朗普政府在没有法律授权的情况下征收关税，给美国经济带来混乱。
美国最高法院两周前裁定，特朗普政府此前依据《国际紧急经济权力法》对多国产品加征关税的做法违法。在裁决后，总统特朗普并未接受败诉结果，转而援引《1974年贸易法》第122条款，宣布对全球大多数产品征收15%的关税。

华盛顿——3月5日，美国众议院以219票对212票否决了一项要求总统对伊朗的任何军事行动须经国会授权的决议，实际上在冲突第六天继续支持特朗普的对伊军事行动。投票大体沿党派划分，两名共和党人投赞成，四名民主党人投反对。
反对者指责民主党人把议题推上表决仅因反对总统，增加了美国人的风险。众院情报委员会共和党主席里克·克劳福德称，如果总统不是唐纳德·特朗普，就不会有今天的表决。
决议发起人称此举是收回宪法赋予国会的宣战权。2月28日美以对伊发动攻击以来，冲突已致逾1000人死亡，其中至少包括6名美军，并在中东造成破坏与动荡。
支持者表示，该决议若通过将迫使总统向国会与公众说明交战理由及结束方式。外交事务委员会资深民主党人格雷戈里·米克斯称，这是行政当局未经授权、无明晰目标和结束方案的“选择性战争”。
即便众院通过，该决议也不会立即终止空袭：还须参院通过，且若遭总统否决需两院以三分之二多数推翻。参院也在3月5日投票阻止了类似两党决议。
1973年《战争权力决议》规定，总统只有在国会宣战、国会授权或对袭击作出反应时方可将军队卷入武装冲突；另对未经授权的军事行动要求60日内终止，这为特朗普政府在四月底前寻求国会批准设定了最后期限。路透社报道。

阿联酋：以备战与透明维护安全与稳定
在应对伊朗发射的导弹与无人机袭击中，阿联酋展现出卓越的应变与防卫能力，成功管控事态并维护社会稳定。官方将其平稳处置称为国家安全的重要资产。
官方通报显示，在美以对伊朗开展开头四天内，阿联酋防空部队应对逾1000次导弹与无人机攻击：对186枚弹道导弹作战，摧毁172枚，13枚坠海，1枚落入境内；拦截812架无人机，摧毁755架，57架落入境内；另有8枚巡航导弹全部被拦截摧毁。
分析认为，伊朗常用“饱和”战术——多波次、混合型导弹与无人机攻势，意在耗尽拦截弹并突破防线。但阿联酋凭借多层次防空体系、充足弹药储备和完善的监测、指挥与协同机制，经受住了高强度、多样化打击，仅造成可控的物质损失，未发生重大人员或战略资产损失。
长期战略规划与领导层对国防的重视被视为关键因素。阿联酋多年的防务投入、分层拦截架构与应急准备确保了持续威慑与防护能力。
