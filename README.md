香港专上学生联会表决通过开展解散程序
香港专上学生联会（学联）星期四（2月5日）在常委会会议上，表决通过开展解散程序。
学联当天在社交平台Instagram发布声明，宣布上述消息。
根据声明，过去68年，香港学联一直是团结香港各大专院校的力量，关心社会，争取改革。学联多年来在各重大政治、社会事件中，从不缺席。在悠长的岁月中，学联见证香港社会变迁，一直在不同议题上身体力行，表达学生及市民的诉求，以香港前途为依归。
香港学联表示，为多年来与市民同行、共同奋斗感到荣幸，也对中国和香港未来的由衷关怀，始终如一。
学联称，随着环境变化，学生各成员及其同行者近年面临日益严峻的压力。在考虑各方面因素后，“学联决定在今天划上句号”。希望大家继续为未来的前途，加倍努力，不忘初心。
公开资料显示，香港学联1958年正式成立，目的是代表香港大专院校学生，借此推动全港学生活动及增加同学对社会的参与，成为香港对外最具代表性的学生团体。
学联去年6月发声明澄清有人假冒学联之名发出新闻稿，称该会从即日起解散的消息。
综合网媒“香港01”和香港《星岛日报》报道，学联过往参与多场社会运动，包括在2012年反对国民教育科罢课；2014年就政改方案发起罢课、支持“占中”等。
2015年，学联多个成员发起退联公投，最终香港大学、香港浸会大学、香港城市大学和香港理工大学的学生会退出学联，目前成员是香港中文大学、岭南大学、香港树仁大学和香港科技大学的学生会。

被罢全国人大代表后 罗琦名字从中国工程院院士名单删除
中核集团原总工程师罗琦被罢免中国全国人大代表职务后，他的名字已经从中国工程院官网的院士名单中删除。
据中国工程院官网的全体院士名单页面，2019年当选为中国工程院院士的罗琦，名字之前位列能源与矿业工程学部的144人名单。星期四（2月5日）的检索结果显示，他的名字已经被删去，能源与矿业工程学部院士数量已变为143人。
第一财经报道，多名与罗琦有过接触的核电业内人士透露，去年以来，罗琦已有很长时间没有出现在公众场合。记者曾多次拨打罗琦的电话，但一直无人接听。

加西亞指出，此次補選是為填補因眾議員羅密歐·阿科普（Romeo Acop）過世而產生的席次空缺，選舉應為各級政府之優先事項。
Comelec在1月12日致馬拉卡南的函件中說明，9800萬中的約三分之一（約3470萬）將用於印刷與刊登，3180萬用於專業服務，另至少1600萬用於人事、物資與材料，因補選將以人工投票進行。預算還將涵蓋選務委員會與特別選務委員會及其支援人員的酬金、訓練與部署、後勤支援及必要的營運費用。
加西亞表示，Comelec會尋找資金來源，無法向眾議院請款以免牽涉另一次程序，而向行政部門申請則因有待命基金可用且過去常獲支援。他並提到，行政部門在2023年鄉鎮（barangay）選舉重排後曾追加250萬披索資助。
欲參選安蒂波洛第2選區眾議員者，候選人登記（COC）時間為2月5日至7日。
