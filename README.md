2020至2022年，许锐在字节跳动旗下硬件部门New Stone Lab任总经理，期间推出包括Dali智能台灯在内的多款消费产品；2022至2024年曾在腾讯的扩展现实（XR）设备团队任职。2024年他加盟帕洛阿尔托的人形机器人初创公司K-Scale Labs，该公司于11月宣布破产，随后他加入AI代理初创公司Dreamer，后者于今年3月被Meta收购。
业内认为，Meta此番任命反映其在AI硬件领域的持续投入与竞争布局。

港股上市、吉利入资的曹操出行押注重资产、车队优先战略，力争在中国机器人出租车赛道中脱颖而出，目标到2030年部署10万辆自动驾驶车辆。 
公司CEO龚欣在接受南华早报采访时表示，机器人出租车的未来取决于以车辆制造、自动驾驶技术和车队运营构成的闭环“金三角”资产管理模式。曹操正在完善其L4级自动驾驶系统，并于2025年底在杭州先行投放了100辆机器人出租车；尽管中国多数车辆仍需安全员看守，曹操计划今年实现无人化运营。 
4月1日，公司获批在杭州开展无人路测，成为该市首家获批企业。面对Pony.ai、WeRide等本土竞争者，曹操强调其“完全为机器人出租车打造”的车型是核心竞争力：该车过去两年从头设计、软硬件高度集成，预计今年亮相并于2027年上半年进入量产。

中国为在人工智能领域取得突破、应对与美国加剧的科技竞争，正加紧从海外吸纳半导体人才；对此，台湾当局也强化对大陆疑似挖角行为的调查。
台湾法务部调查局（MJIB）上周一表示，已对11家大陆企业展开调查，控诉这些企业以隐瞒大陆背景、设立空壳公司并未经核准在台开展业务等方式，非法挖角半导体及其他高科技人才。MJIB称，自2020年以来已处理约100起相关案件；去年也曾调查大陆最大晶圆代工厂中芯国际（SMIC）涉嫌挖角案。
加拿大顾问机构The Geopolitical Business地缘战略师阿比舒尔·普拉卡什（Abishur Prakash）称，这是一场“低调的科技战”，与美中围绕出口管制和资本等议题的公开对峙不同，大陆更着眼于推动下一波AI创新所需的人才。台湾当局对此高度警觉，调查行动反映出大陆企业对具备经验的人才在半导体与AI领域的强烈需求，即便面临美国对高端技术的限制，这类人才流动仍在加剧。

多家中国企业包括阿里云和知谱科技近期选择不将部分最新人工智能模型开源，转而通过官方付费渠道获取使用价值。 
本周阿里发布三款仅能在其云平台或聊天机器人网站访问的专有模型，其中包括增强编程能力的Qwen3.6-Plus与可处理文本、音频、图像和视频的多模态模型Qwen3.5-Omni。 
今年9月发布的上一代多模态模型Qwen3-Omni为开源版本，但阿里表示新版不会开源，理由是基于开源平台Hugging Face的下载数据，Omni系列在开发者中的受欢迎程度较低。 
业内认为，顶级模型规模不断增大、难以在本地硬件上部署，使厂商更倾向于通过集中托管和商业化服务变现，同时并未明确放弃开源策略。 
阿里云为阿里巴巴集团旗下的人工智能与云计算业务单元，阿里巴巴集团为《南华早报》的所有者。
