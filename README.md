解套海马斯火箭头款 台行政院拼月底前送特别预算案
为解决海马斯火箭头款问题，台湾行政院争取在本月底前向立法院送出特别预算案。
综合台湾《联合报》和ETtoday新闻云报道，台湾立法院上周三读通过国民党（蓝）和民众党（白）共推的7800亿新台币（315亿新元）国防特别预算条例草案。但对美采购的海马斯火箭（台称海马士火箭）5月底就要付头期款，因此行政院将力拼在本月底前向立法院送出特别预算案。
台国防部表示，海马斯火箭军购案须在5月31日交付头款，否则案子就要作废重来，但国防部仍在草拟军购特别预算案，尚未送至行政院。
国防部新闻处长乔福骏证实，特别预算相关内容必须配合条例所规定的品项与金额，才能拟定预算书，战规司周末正在加班，务必最快速度完成预算书送交行政院。
根据军购条例三读条文，明定行政院须于一个月内提出相关专案报告，经立法院同意后始得筹编预算案。但立法院长韩国瑜5月17日才自欧洲返台，距离月底交付头款期期限只剩两周。
民进党立委庄瑞雄说，预算法第八十四条是因应重大急迫政务需求设计的机制，允许先行支付。海马斯首期款5月31日到期，立院理应支持国防部依法处理。
国民党立委赖士葆说，除预算法第八十四条，国防部仍有先前已编列、但尚未付出的军购款项可以调度，筹钱方式不会成为政治问题。
针对外界关注是否再提第二波国防特别预算条例，台湾国防官员表示，国防部不会再提第二个特别条例，未通过项目将改以追加预算或纳入下年度国防预算处理。

中国手机品牌OPPO一则母亲节活动文案引起舆论批评，品牌方四天内第二次致歉，并对中国区高级副总裁段要辉处以降两级处分。
OPPO星期一（5月11日）在官方微博再发致歉声明，称对在母亲节营销文案中存在的问题，再向社会公众致以最深刻的歉意。
声明说，这次事件严重地伤害了公众的感情，也违背了公司一贯坚持的文化价值观。经过深刻反思，公司已经对分管中国市场业务的高级副总裁段要辉以及营销相关管理者作出了从严处罚。
据第一财经报道，OPPO内部星期一正式发布问责通告，就因母亲节不当文案事件对相关责任人进行了定级处罚。其中，OPPO中国区业务负责人段要辉负最终管理责任，职级直降两级，其余相关管理人员也均做出处分决定。

欧洲疾病预防控制中心（ECDC）表示，作为预防措施，因发生致命汉坦病毒疫情而受影响的邮轮上所有乘客在下船时都被视为高风险接触者。该邮轮预计于5月10日在西班牙特内里费岛外停泊，ECDC在5月9日的加急科学建议中称，无症状乘客将由各国通过专门安排的交通工具（而非普通商业航班）遣返并自行隔离。
各国正准备在格林威治标准时间（GMT）约6:30至7:00（新加坡时间14:30至15:00）开始撤离MV Hondius上的本国公民。世界卫生组织（WHO）5月8日通报，已有8人患病，其中3人死亡（包括一对荷兰夫妇和一名德国公民）；WHO称6例已确诊感染，另有2例为疑似病例。ECDC指出，虽在下船时均列为高风险，但返国后并不一定全部被认定为高风险个案。
在邮轮预计停靠的特内里费格拉纳迪利亚·德·阿博纳港已设立移动医疗点。ECDC敦促对有症状者在抵达时优先进行医学评估和检测，根据病情可能在岛上隔离或医疗转运回国。汉坦病毒通常由啮齿动物传播，极少情况下可人传人，卫生部门认为病毒进一步扩散的风险较低。
