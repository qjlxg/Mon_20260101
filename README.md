赖清德吁立法院跨越党派藩篱审议台美关税协议
台湾总统赖清德星期二（2月2日）在总统府举行的台美经济繁荣伙伴对话记者会上，呼吁立法院跨越党派藩篱审议台美关税协议。
综合ETtoday新闻云和中时新闻网等台媒报道，台美经济繁荣伙伴对话（EPPD）会议1月27日在华府落幕。总统赖清德在记者会上说，对话主要聚焦在供应链安全的战略对接、关键矿物合作、第三国合作、以及台美合作四大支柱，从中归纳出台美未来合作的三大战略方向。
他说，台湾与美国签署了硅盛世宣言及台美经济安全合作联合声明，具体展现台美携手强化经济安全的共同承诺。双方未来将准备成立工作小组，持续针对双方关切重要议题，保持密切沟通、加强互动，一起打造更安全、更有韧性的供应链。
他指出，台美未来将共同打造创新经济，针对供应链安全、无人机系统认证、消除投资税务障碍，推进实质性计划，进一步确保台美伙伴关系，跟上科技重塑全球产业版图的速度，开创下一世代的繁荣。
赖清德强调，台美合作不再局限于单一产业，而是全方位、跨领域的深度链结。台湾与美国正是彼此繁荣未来不可或缺的关键伙伴。台湾将加速对接国际标准的经贸规范，建立具备可预测性与透明度的法制环境；并在互信、互利基础上，深化与美国的产业对接，共同打造具备韧性与竞争力的全球供应链网络。
他坚称，经贸谈判的成果，攸关台湾经济繁荣与人民福祉，因此期盼后续立法院审议台美关税协议，能够秉持专业、依法审查，跨越党派的藩篱，跳脱政党之间的窠臼，共同守护这份得来不易的谈判成果。
美台关税谈判历时九个多月，已于1月16日就15%税率达成共识，但台湾在野阵营质疑政府让利过多，已扬言在立院严审。　
美台达成关税协议后，星期二（1月27日）再就人工智能（AI）、科技和无人机合作进行讨论，并签署两份声明文件。

何立峰：春运是重要政治任务 要防范重大风险管控危险源
中国星期一（2月2日）开启2026年春运工作，国务院副总理何立峰当天在北京、天津检查综合运输春运工作，强调春运是重要的政治任务和民生工程，要求交通运输各部门各单位深入开展安全生产隐患排查整治，强化重大风险防范和危险源提级管控。

中国科学家发现糖尿病防治“黄金靶点”
中国科学家发现，介导锌离子积累的转运蛋白ZnT8可作为糖尿病预防与治疗的“黄金靶点”。
据人民日报海外版，同济大学李维达教授与高绍荣院士团队在糖尿病研究与治疗领域取得新突破，首次揭示锌离子积累是糖尿病患者的胰岛β细胞功能失活的关键致病因素，并证实介导锌离子积累的转运蛋白ZnT8可作为糖尿病预防与治疗的“黄金靶点”。
针对胰岛功能衰竭、需长期依赖胰岛素注射的晚期糖尿病患者，研究团队借助干细胞技术，在体外培育出可移植、具有耐逆特性的人源胰岛类器官，有望实现“一次移植、长期有效”的功能性治愈希望。
中国有高达2.74亿的糖尿病前期人群，他们目前无药可用，只能等待疾病进展，其中近半会发展为糖尿病。针对这一临床空白，研究团队同步推进全球首款靶向ZnT8的小分子抑制剂研发。这款药物旨在模拟ZnT8蛋白功能缺失天然的保护状态，于糖尿病前期为患者脆弱的胰岛β细胞提供保护，延缓甚至阻止其向糖尿病发展。
据李维达介绍，中国国内多家顶级医疗机构的临床专家已与他们接洽合作，共同推进后续临床转化研究。

马尼拉 — 眾議院多數黨領袖、伊羅戈斯北省第一選區代表桑德羅·馬科斯（Sandro Marcos）提出法案，主張對國會議員實施「不工作不領薪」的薪酬制度。 
根據眾議院法案7432，議員只有在出席並履行正式立法職務的日子才可領取報酬；無正當理由缺席者，該日報酬將被沒收。 
法案列舉的正當缺席原因包括：經執業醫師證明的病假、經議院領導授權的代表性公務、由議長核准的官方出差或任務，以及經核准的休假。 
馬科斯在說明理由時指出，「不工作不領薪」是普遍的勞動原則，現行做法使議員無論出席與否皆領薪，引發社會對曠職、問責不足及納稅人資金浪費的關切。該法案將薪酬與出席、參與全體會議、委員會聽證及其他公務掛鉤，以促進透明與廉潔治理。 
對於該法可能遭質疑違憲、構成特定階層立法的說法，馬科斯援引1987年憲法第VI條第10款指出，國會議員薪資應由法律決定，立法規範屬合法範疇。 
他強調，通過此法有助於維護公共服務高標準，確保民選代表落實為人民服務的理念。
