中国官媒评整治内卷进展：将持续破除地方保护和市场分割
中共中央机关报《人民日报》刊文盘点政府推出的整治“内卷式”竞争系列措施，指出下一步将持续破除地方保护和市场分割，助力纵深推进全国统一大市场建设。
《人民日报》星期五（3月20日）在第二版刊登题为《针对典型行为推出系列举措—综合整治“内卷式”竞争进展如何》的报道，从三方面罗列当局整治内卷的具体措施。
第一，对强制促销、商业诋毁等内卷式竞争加强规制。今年初，中国市场监管总局发布《互联网平台反垄断合规指引》，明确互联网平台企业合规的内容和具体要求。文章称，该指引有助于提高市场预期，指引企业合法经营，防患于未然。
去年6月，中国修订的反不正当竞争法加强了对新型、典型内卷式竞争的规制。文章指出，一系列制度规则不断完善，旨在以实招硬招破解内卷式竞争难题。
第二，对市场上明显低于成本价格的产品加强认证监管。市场监管总局认证监督管理司副司长杨冬认为，强制性产品认证是降低制度性交易成本、守护产品质量安全底线的重要手段。
杨冬称，监管部门将深入开展强制性产品认证制度“守底线”专项行动，加强相关机构的日常监管，提高精细化管理水平，要求认证机构加强风险识别等。
第三，减少重点行业无序竞争，规范地方政府行为。市场监管总局竞争政策协调司副司长赵春雷介绍，深入实施公平竞争审查制度，强化滥用行政权力排除、限制竞争执法，进一步为政府行为划清界限，是规范政府不当干预市场竞争行为的重要举措。
文章在结尾处表明，下一步，市场监管总局将推动各地区、各部门深入实施《公平竞争审查条例》，持续破除地方保护和市场分割，推动有效市场和有为政府更好结合，深入整治“内卷式”竞争，助力纵深推进中国全国统一大市场建设。
