他也说，首日有78户、269人已登记上楼。在居民可以上下楼次数方面，港府没有设硬指标，取决于现场人手，但希望大家都遵守三小时的限制。执拾时有稍微超时情况是可以理解的，但原则是不能影响到下午清场，因为这会影响到下一轮居民上楼。

新华社星期一（4月20日）引述据中国工业和信息化部电子信息司网站消息，中国工业和信息化部、国家发展改革委、市场监管总局、国家能源局等部门上星期五（17日）联合召开光伏行业座谈会，部署规范光伏产业竞争秩序相关工作。
会议强调，要深入贯彻落实中共中央、中国国务院关于规范光伏产业竞争秩序的工作部署，深刻认识治理“内卷式”竞争的重要性和紧迫性，扎实推进光伏行业“反内卷”工作。

彭博社报道，中国政府今年已与光伏产业制造商至少会面三次，这些企业均面对因产能过剩和激烈竞争导致的亏损情况。
路透社报道，光伏产业被广泛认为是中国产能过剩最严重的行业之一，虽然推动了创纪录的贸易顺差，但也压缩了制造业利润，并加剧了中国与他国的外交紧张。
中国光伏产业制造商表示，伊朗战争引发的石油供应冲击可能带动全球新能源需求增长，但这一增长不太可能显著缓解目前行业内产能过剩的问题。

圣玛尔塔（哥伦比亚）— 超过50个国家将于下周在哥伦比亚加勒比港城圣玛尔塔出席史上首场“化石燃料退出”会议，背景是伊朗相关冲突引发的燃料短缺与价格飙升，加剧各国对燃煤、石油和天然气依赖的担忧。 
部长级代表将于4月28—29日就能源安全与气候优先事项展开讨论。主办方称，霍尔木兹海峡局势所致的能源冲击进一步凸显逐步淘汰化石燃料的必要性，但部分国家短期内可能回归煤炭以保障供应。 
本次会议由哥伦比亚与荷兰共同主办，参加国涵盖澳大利亚、加拿大、挪威、安哥拉、墨西哥、巴西、土耳其、越南等产消两端国家，以及德国、法国、英国和许多小岛国家。但美国、中国、沙特与俄罗斯等主要生产国并未出席。 
组织者表示，会议旨在打破联合国共识式谈判的僵局，开辟更“坦诚”的对话空间；批评者则担心参与国利益分散，难以产生明确成果。会谈成果将纳入由巴西牵头的自愿退出化石燃料路线图。

莫斯科，4月20日——俄罗斯官员周一表示，乌克兰对黑海港口图阿普谢发动大规模无人机袭击，引发火灾并造成至少一人死亡。此前4月16日的类似攻击引发的火情当日才被扑灭。
图阿普谢是俄罗斯重要的南部港口，既是成品油出口枢纽，也处理散货（如煤炭和化肥），港内还设有俄罗斯石油（Rosneft）同名大型炼油厂。
克拉斯诺达尔州州长维尼亚明·康德拉季耶夫在电报（Telegram）上称，"图阿普谢遭到又一次大规模无人机袭击，港口发生火情。"他表示，港区一人死亡、一人受伤，无人机残骸损及市内多座建筑，包括幼儿园、小学、教堂和居民楼。
俄罗斯国防部称，当夜摧毁了112架乌克兰无人机。
