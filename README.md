中国电动车走向超大尺寸使道路养护财政承压
中国新一代电动汽车正朝着“超大尺寸”狂奔——部分车型车长超5米、重达3吨。这不仅令国内道路不堪重负，也让高度依赖燃油税来养护道路的政府财政面临日益严峻的考验。
彭博社星期四（7月16日）报道，中国乘用车市场信息联席会秘书长崔东树提供的数据显示，今年上半年推出的新车型中，高达六成车长超过5米，体型堪比福特探险者等大型SUV。相比之下，车长4.5米以下的紧凑型新车占比已从去年同期的13%骤降至仅2%。
在中国市场，大型汽车仍是身份地位的象征；加之其定价与中型车相差无几，进一步推高了消费热度。以比亚迪新款旗舰SUV大唐EV为例，这辆车的车长达到惊人的5.3米，整备质量高达2970公斤，起售价却仅为23万9900元人民币（约4万5000新元）。
此外，为缓解消费者的里程焦虑并制造营销卖点，车企不断在底盘上堆砌更大、更重的电池包，甚至将车载冰箱、多屏娱乐系统等配件塞进车厢。
大尺寸、大重量汽车的风靡，正让中国地方政府陷入财政困境。长期以来，中国由中央政府出资建设主要高速公路，并通过征收燃油税下拨地方以养护普通道路。然而，电动汽车通常比燃油车更重，对道路损耗更大，但随着电车市占率不断攀升，相关的燃油税收入在萎缩。这迫使政府亟需对道路资金筹集方式进行彻底改革，如引入基于行驶里程的道路使用费。
这种财政压力已将中国的公路网推向临界点。中国交通运输部下属研究机构的调查发现，中国普通公路养护与管理面临约50%的年度资金缺口。约四成地方道路陷入严重瓶颈，被形容为列入计划却无钱可用，亟待修缮却缺乏预算。该机构2023年的研究估算，每年的资金缺口高达3000亿元人民币。
为寻求长期解决方案，决策层正计划全面重构道路资金筹措体系。目前在海南等地，已在试点利用卫星导航系统追踪目标车辆，为未来征收动态的“里程税”奠定基础，这或将终结电动车“零路税上路”的时代。
中国多家官媒也开始呼吁车企为超大型车辆“减肥”，终止汽车尺寸的膨胀现象。

路易威登诉中国知识产权局案开庭未宣判
法国奢侈品巨头路易威登马利蒂起诉中国知识产权局案件星期四（7月16日）开庭，但未当庭宣判。
据中新社报道，北京知识产权法院当天公开审理这起商标行政纠纷案，原告为路易威登（LV）品牌母公司路易威登马利蒂，被告为中国国家知识产权局，广东汕头服装经营者黄民耀为本案第三人。
公开资料显示，黄民耀申请注册商标后，LV针对该商标向国家知识产权局提起异议/无效宣告申请，国家知识产权局作出裁定，未支持LV的主张。LV不服这份行政裁定，向北京知识产权法院提起诉讼。
本案庭审核心为，国家知识产权局作出商标行政裁定是否合法，商标申请人黄民耀作为案件第三人参与庭审。
法庭开庭审理前，北京知识产权法院诉讼服务中心大厅里，安检通道前排起了长队。
红星新闻也报道，在法院门口看到不少人希望能够旁听这起商标行政纠纷案的庭审。法院的工作人员解释称，由于案件审理的法庭可容纳人数较少，不能满足绝大多数人的旁听请求。
