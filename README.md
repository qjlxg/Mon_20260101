【乌拉圭总统：中国“十五五”规划为世界敞开机遇之门】金十数据2月15日讯，乌拉圭总统奥尔西表示，中国的发展规划为世界开启了更多机遇之门。中国向世界传递的发展信号与制定的发展规划，让我们得以预见，更多机遇之门正在向我们徐徐打开。乌拉圭是一个人口稀少的国家，亟需创造财富以实现更优分配，更好地增进人民福祉。我们的企业家和劳动者们能够在乌拉圭与南美地区之外看到更广阔的发展机遇，同时，这些机遇也为当今世界点亮了一束希望之光。

【2月14日全社会跨区域人员流动量完成28566.2万人次】金十数据2月15日讯，来自综合运输春运工作专班数据显示，2026年2月14日（春运第13天），全社会跨区域人员流动量28566.2万人次，环比下降0.8%，比2025年同期（星期日）增长7.5%。

【罗文同志现场检查春节期间值守应急工作】金十数据2月15日讯，2月15日上午，市场监管总局党组书记、局长罗文同志到总局机关检查值守应急工作，电话连线地方市场监管局，慰问一线值班人员，对进一步做好春节假期值守应急工作提出明确要求。总局党组成员、副局长柳军同志参加检查。罗文强调，2026年是“十五五”开局之年，维护好春节期间市场秩序和安全稳定意义重大。各地市场监管部门要紧盯春节消费重点领域，强化“三品一特”安全监管，加大重要民生商品和服务价格监测监管力度，认真做好安全风险隐患排查整治，加强应急值守，及时回应社会关切，切实维护市场秩序和消费者合法权益，确保人民群众度过欢乐祥和的春节假期。

【千问总裁吴嘉回应千问爆火：我们不是要和谁卷】金十数据2月15日讯，千问C端事业群总裁吴嘉今天回应千问爆火时表示，千问做这件事的初衷，并不是为了和谁卷，而是让AI融进老百姓的日常生活场景中，“这是未来一定会发生的事情。” 2月6日阿里旗下的千问App推出春节大免单活动，2月14日千问启动第二波免单活动。 吴嘉表示， 第一波活动是能力验证，验证AI能不能在真实生活场景里跑通“理解—决策—下单—支付”这条完整链路。第二波则真正把阿里巴巴全生态放进来了，飞猪的机酒、火车票、门票，大麦的电影票，以及即将接入的高德团购、打车等，都在千问里。“AI正在从一个聊天工具，变成一个能办事的助理。”
