你可能已经在为AI买单：四项意想不到的涨价
全球人工智能（AI）投资热潮正开始反映在消费者账单上。
美国联邦储备局部分官员和华尔街评估认为，大型科技公司投入数千亿美元建设AI技术和数据中心，可能正在推高美国通胀，并通过消费电子产品、电费、软件订阅和部分电脑硬件价格影响普通消费者。
《华盛顿邮报》报道，AI企业为建设数据中心，大量采购晶片、土地、电力和专业人才，带来连锁成本压力。
巴克莱投资银行美国高级经济师斯里拉姆（Pooja Sriram）说：“我们正经历这个痛苦时期。未来消费者会获得更好的计算能力和软件，但也必须面对略高的价格。”
首先受到冲击的是消费电子产品。
AI企业为数据中心采购大量晶片，导致笔记本电脑、智能手机、汽车、流媒体设备和电子游戏机所需晶片供应趋紧、价格上升。任天堂上月宣布，把最新掌上游戏机Switch 2价格从450美元（约580新元）调高至500美元。公司称，今年将承担相当于6亿2400万美元的额外成本，其中包括晶片和特朗普政府关税带来的压力。
运动摄像机制造商GoPro也警告，AI带动的晶片涨价和短缺，已使公司面临持续经营风险。
其次是电费。数据中心大量耗电，正推高部分地区电力系统成本。
马里兰州居民公用事业倡导机构估算，典型家庭每月电费已从2022年的约122美元升至今年约181美元。虽然难以精确计算数据中心应承担多少涨幅，但数据中心正在推高账单的多个组成部分，包括维持电网可靠性的相关费用。
第三是软件订阅。高盛上月分析指出，微软Office、Adobe摄影工具等软件价格在过去18个月内最高上涨约50%，许多涨价与新增AI功能有关。
第四是电脑硬件。苹果Mac Mini今年成为部分用户安装个人AI代理程序的热门设备，需求上升导致供应紧张。苹果约一个月前停止销售最便宜机型，目前Mac Mini入门价格从599美元升至799美元。苹果没有公开解释这项调整，但公司高层指出，晶片成本和供应压力已影响其生产iPhone和电脑的能力。
分析指出，AI公司自身也承受成本压力。
Meta和微软近期说，建设AI数据中心的支出将比预期多出数百亿美元。换言之，AI企业因争夺晶片和电力推高成本，而这些成本又通过商品和服务价格，逐步传导至消费者。

中国央视《财经调查》栏目星期天（6月7日）晚报道，在接到相关举报后，栏目组暗访了江苏扬州市的一些加工厂和小作坊。在当地一处废品收购站内，堆积的废旧塑料包括装化学制剂的白桶、旧家电拆卸下的面板、电风扇盖子、破旧轮滑鞋等。
工作人员称，这些回收塑料经过简单粉碎后，会流向各类塑料制品加工厂，其中相当一部分专门用来制作牙刷。
另一个小作坊里，堆积着各种破旧塑料布、蛇皮口袋、黑色垃圾袋、塑料绳等，据指这些垃圾也是做一次性牙刷的原材料，还有一些回收口罩，可直接加工成制作牙刷的塑料颗粒。
还有一家“回收料”加工厂里，原材料竟是鞋厂制作拖鞋剩下的边角废料，在被加工成一粒粒白色塑料颗粒后，将直接送往牙刷厂制作牙刷。
劣质的塑料垃圾在加工前没有清洗、消杀流程，仅经过高温处理、简单滤网过滤。不少小型工厂大量使用这些低价的回收塑料，与新料混合生产牙刷，原料配比根据牙刷的定价弹性调整。
这些超低价格的一次性牙刷出现在多地的酒店用品城里，大量销往各地的小型酒店与民宿，带来多重健康隐患。
据《新京报》报道，央视曝光之后，扬州市市场监督管理局星期一通报称已连夜开展核查处置工作，截至目前，已对涉事经营主体进行调查取证，并开展抽样检测，先行登记保存涉案物品4.4吨。
该局说，下一步将深入彻查案件，从严从快作出处理，同时立即组织开展牙刷产品质量安全隐患专项排查整治行动，严打各类违法违规行为。

法国人口预计2037年见顶 其后数十年将持续回落
法国最新人口预测指出，法国人口预计将在2037年见顶，并在之后数十年逐步回落至接近2014年的水平，而这一峰值时间比2021年时的预测提前了七年。
路透社报道，法国国家统计与经济研究所（INSEE）星期一（6月8日）发布新预测称，法国人口将从2026年的6910万，增加至2037年的6980万左右，之后转为下降趋势，预计到2070年将降至约6590万，几乎回到2014年的规模。
INSEE此前在2021年的预测中估计，人口峰值会在2044年出现，规模约6930万。
法国长期以来在人口结构方面优于大多数欧洲国家，一度被视为“人口相对年轻”的例外。但最新数据表明，随着人口老龄化加剧、出生率持续走低，法国同样难以避免面临与欧洲其他国家类似的财政与社会压力。
INSEE指出，法国自2025年起已出现“自然人口增长为负”的情况，也就是死亡人数超过新生儿人数，这一趋势将持续下去。直到2037年之前，人口净增长几乎完全依赖移民流入来支撑。
报告也提出风险情景：如果未来移民减少，或生育率低于当前假设的“每名女性平均生育1.45个孩子”，法国人口到2070年可能会大幅萎缩，最低或跌至约5460万。
除了总量减少，人口年龄结构也将发生显著变化。INSEE预计，到2070年，法国每三人中就有一人年满65岁或以上，65岁及以上人口比例将约为20岁以下人口的两倍。
在高龄群体中，变化尤为明显：80岁及以上人口预计将超过目前的两倍，达到约900万；百岁以上人口则可能增长四倍，增至约16万人。
