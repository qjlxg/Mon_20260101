中国象棋协会星期一（4月13日）发布情况通报称，洪智、谢靖，以及徐超等三名棋手，因参与“买卖棋”被终身禁赛。
买棋即通过金钱或其他方式让对方故意让步输棋，卖棋则是在接受好处的情况下，自己故意让步输棋，让对方获胜。
据“中国象棋协会”微信公号消息，中国象棋协会星期一发布关于对洪智等违规人员进行处罚的情况通报。通报称，根据公安、司法部门向中国象棋协会移交的象棋“买卖棋”问题材料，查实洪智等象棋运动员存在通过行贿、受贿等方式不同程度参与买棋、卖棋等违规行为，对象棋运动发展造成严重损害。为正风肃纪、惩前毖后，推动建立风清气正的象棋运动发展环境，根据《中国象棋协会纪律准则和处罚规定》等相关规定，综合考虑违规事实情节及本人态度，经中国象棋协会纪律与道德工作委员会研究决定，对有关人员分别作出处理。
六名棋手在禁赛期内均被禁止从事和参与中国象棋协会及其会员单位组织或授权组织的所有象棋赛事和活动。其中，洪智、谢靖、徐超被处以终身禁赛；申鹏被禁赛八年，禁赛期自2024年7月24日起至2032年7月23日止；蒋川被禁赛五年，禁赛期自2024年7月24日起至2029年7月23日止；徐崇峰被禁赛两年，禁赛期自2024年7月24日起至2026年7月23日止。
除了徐崇峰外的五名棋手，均被撤销中国象棋协会技术等级称号。
中国象棋协会在通报中指出，赛风赛纪和行风行纪既是象棋运动发展的生命线也是高压线，更是象棋行业作风建设的关键一环，关乎象棋事业的健康发展。
通报称，上述人员的违规行为严重破坏了象棋赛事活动的公平公正，严重冲击了象棋运动发展的正常秩序，严重侵蚀了体育事业的公信力。
据新华社报道，2023年4月起，网络平台上陆续出现录音片段，同时配有文字，称象棋运动员王跃飞、郝继超在商讨买卖协议。同年10月，象棋名宿柳大华及党斐在自媒体平台实名举报，要求彻查其他乱象。2024年9月，中国象棋协会公布关于对王天一、王跃飞违规行为进行处罚的情况通报，两人因长期存在买棋、卖棋、操纵比赛等行为被给予终身禁赛等处罚。
中国国家体育总局棋牌运动管理中心在2025年1月的媒体通气会和全国棋牌领域赛风赛纪警示教育会上回顾和总结了象棋行风行纪专项整治的工作情况，同时通报了对赵鑫鑫等41名违规人员的处罚决定。

能源价格飙升 分析指英国普通家庭每年减近500英镑收入
分析指出，一个典型的英国普通家庭将因能源价格飙升而每年减少近500英镑（约638.11新元）的收入。
彭博社报道，英国智库决议基金会（Resolution Foundation）星期一（4月13日）说，根据市场对能源价格的预测，英国中位数劳动年龄家庭的收入在本财年将下降0.6%，而此前预期为增长0.9%。
尽管智库分析显示，收入最低的五分之一家庭仍会增长，但在以色列和美国对伊朗发动袭击之后，这一群体的收入增长预计将明显减弱，仅为1.2%，低于此前预估的2.8%。 
决议基金会首席经济学家史密斯说，尽管近期地缘政治紧张有所缓和，但“今年对家庭财务的损害在很大程度上已经发生”。
他说：“能源价格仍远高于战前水平，这意味着许多家庭今年的购买力将下降。对于中等及以上收入群体来说，原本微弱的增长预期已经转为负增长。”
伊朗战争被认为是自工党政府上台并承诺提高生活水平以来，对家庭财务造成的最大冲击。这次冲突可能削弱英国民众的个人财务状况，而这些状况此前正从2022年俄罗斯入侵乌克兰引发的通胀冲击中逐步恢复。
经济学家预测，通胀率可能接近英国央行2%目标的一倍。英国司机已因战争爆发后燃油价格快速上涨而承压，贷款机构也因市场预期央行将加息而提高按揭成本。随着夏季来临，能源账单预计还将因天然气和电力价格上限上调而进一步上涨。
与此同时，根据德勤（Deloitte）的另一项调查，伊朗冲突也使英国企业信心跌至疫情初期以来的最低水平。
德勤的季度调查显示，3月下半月净信心指数从2025年底的-13%暴跌至-57%，为2020年第一季度以来的最低水平。
德勤也发现，净79%的公司首席财务官预计未来12个月的招聘人数将会下降，高于去年年底的55%。

中国股债罕见步调一致 伊朗战争推升避险需求
中国股债市场两年来首次同频共振，表明在美伊战争期间，中国资产因被视为更安全的投资押注而从中受益。
据彭博汇编星期一（4月13日）发布的数据，沪深300指数与彭博中国国债总回报指数的90天相关性，从3月18日开始转为正值并持续。中国保护经济免受油价冲击的举措已见成效，其资产表现优于全球主要资产。
与此同时，金融体系流动性充裕提振债市，有关中国通缩压力正在缓解的预期也对股市形成支撑。
此外，多年来低配中国资产的全球资金显现回流迹象。美国银行的分析显示，截至2月底，主动型多头基金的持仓、自2022年以来首次达到基准中性水平。
中信证券首席经济学家明明表示，随着人民币走强、流动性保持宽松，以及中国经济企稳并逐步复苏，中国资产的吸引力持续上升。这表明股债市场的相关性，可能仍将维持在较高水平。
自伊朗战争爆发以来，中国资产相对成为避风港。这得益于抵御能源冲击的能力较强、政策支持以及对冲突的敞口有限。
中国A股跌幅小于亚洲其他市场，而10年期国债收益率截至上周五仅上升3个基点。相比之下，法国、美国和德国的10年期国债收益率，至少上涨了40个基点。
中国经济复苏势头也在增强，工业生产者出厂价格指数三年多来首次转正摆脱通缩，推动市场对股市和债市“再通胀交易”的预期升温。
截至上个星期五（10日），沪深300指数实现周涨幅4.4%，为四个星期以来首次出现单周上涨。与此同时，20年期和30年期国债收益率下跌约六个基点，其中30年期国债收益率创下11月初以来的最大单周跌幅。
不过，股债之间的正相关性未必能够持续。一旦中东冲突缓和，投资者可能重新将注意力转回国内问题，消费持续疲弱以及房地产低迷仍对经济构成压力。
澳新银行资深中国策略师邢兆鹏表示，近期股债正相关主要源于中东局势的消息面影响，其背后的逻辑可能是投资者正在追踪聪明外资（smart foreign money）的动向。

中国官媒发文敦促警惕以苦熬代替苦干倾向
中共中央机关报《人民日报》星期一（4月13日）发文敦促警惕以“苦熬”代替“苦干”的倾向，批评只愿苦熬、不愿苦干，在思想根源上表现为一种投机心态。
该报在文章中称，苦干与苦熬，虽都有“苦”字，但境界完全不同，前者照见为民担当的姿态，后者却有为己谋算的“苦肉计”。“混淆干事创业为了谁的逻辑，就会衍生出错误的政绩观，将事业发展引向歧途，必须警醒与廓清。”
该报写道，只愿苦熬、不愿苦干，在思想根源上表现为一种投机心态，以为把表面功夫做到位，拼不了政绩至少能拼资历。只会苦熬、不会苦干，在实际工作中也体现出一种本领恐慌，没思路、不会干，无可奈何只好熬。
该报也列举称，看作为，比的不是为群众解决了多少实事，而是办公室的灯亮到多晚；看心得，在岗位上“守”了时、办公室里“熬”了夜，目的是让领导看到自己的“苦”。文章续称，工作需要坚守，但坚守不等于熬年头、混日子，这种“苦熬”看似在岗在位，实则只是浪费时间的虚功内耗。
此外，文章也批评“苦熬等于苦劳”这一思维，强调“熬”不创造价值，有“劳”才有“绩”。这里的“劳”，是实实在在的付出，是真刀真枪的作为。倘若只装装样子、搞假把式，不过是少数干部精致利己的权衡罢了，即便侥幸“熬出头”，也“熬”不出足够威信与百姓认可。
该报也强调，为官一任，就要造福一方。如果面对新形势新挑战，缺乏破题能力，只能一拖再拖，工作无进展、生活无改善。如此“履职”，显然组织不认可、群众不买账。
文章指苦熬会消解干部的担当，也会绊住发展的脚步，并说杜绝“熬”的心态，就要改变多干多出事、少干少出事、不干不出事的情况，完善政绩考核评价体系，健全正向激励机制，促进党员干部把心思花在“想事”上、能力用在“干事”上、结果显在“成事”上，最终将成效绽放在群众笑脸上。
文章最后强调：“好日子都是干出来、拼出来的。我们需要警惕的，是以‘苦熬’代替‘苦干’的倾向；应该反对的，是以‘苦熬’敷衍、不以‘苦干’成事的做法。做实干家、行动派，当内省，当诫勉。”

据彭博社报道，在美以对伊朗发动打击并使霍尔木兹海峡实际上关闭六周多后，希望规避额外成本的客户正越来越多转向更环保的解决方案。

多家中国企业包括阿里云和知谱科技近期选择不将部分最新人工智能模型开源，转而通过官方付费渠道获取使用价值。 
本周阿里发布三款仅能在其云平台或聊天机器人网站访问的专有模型，其中包括增强编程能力的Qwen3.6-Plus与可处理文本、音频、图像和视频的多模态模型Qwen3.5-Omni。 
今年9月发布的上一代多模态模型Qwen3-Omni为开源版本，但阿里表示新版不会开源，理由是基于开源平台Hugging Face的下载数据，Omni系列在开发者中的受欢迎程度较低。 
业内认为，顶级模型规模不断增大、难以在本地硬件上部署，使厂商更倾向于通过集中托管和商业化服务变现，同时并未明确放弃开源策略。 
阿里云为阿里巴巴集团旗下的人工智能与云计算业务单元，阿里巴巴集团为《南华早报》的所有者。

阿里巴巴集团在CEO Eddie Wu Yongming 领导下成立高层技术委员会，以加速人工智能研发并应对国内模型与智能代理领域日趋激烈的竞争。

周三，中国人工智能公司智谱AI开源其最新旗舰模型GLM-5.1，并同步将API价格上调10%，显示在与美国对手竞争加剧的背景下，开始加强对高级模型的商业化变现。此次为公司今年第二次涨价：2月对编程订阅的调整曾上调逾30%，本次幅度较小但覆盖包括云端输入token在内的多项计费类别。消息发布后，智谱港股盘中上涨约11.5%。
定价方面，智谱对输入token每百万收费约1.40美元、输出约4.40美元；而Anthropic的Claude Opus 4.6截至2026年2月分别为每百万输入5美元、输出25美元，差距仍在。GLM-5.1上月末先作为付费编程订阅产品推出，此举引发其从开源策略向更强调变现的猜测。
基准测试机构Artificial Analysis认为，按综合智能指标计算，GLM-5.1为全球最强开源模型，领先国内竞争者如MiniMax，但仍落后于OpenAI与Anthropic的顶级模型。智谱在X上称，GLM-5.1可在复杂工作流中持续运行约8小时，能从零构建可用的Linux桌面，反映对可长时自治运行的agent型AI工具的需求。
智谱定位为中国硬件生态中的重要软件合作方，并称已与多家国产芯片企业合作适配其GLM模型，包括华为、摩尔线程与寒武纪。摩尔线程表示其MTT S5000服务器已实现对GLM-5.1的“Day-0”兼容性。
智谱CEO张鹏在3月中关村论坛表示，依赖低价竞争不利行业长期发展，提价旨在建立“更健康的商业闭环”，支持高质量AI服务的长期交付；小米AI模型负责人罗福立也表达了类似看法。

新华社科伦坡4月9日电 中国公安部周四表示，斯里兰卡警方已于3月29日将此前多次收网行动中抓获的125名涉嫌电信诈骗人员遣返回中国，遣返工作得到中国驻斯里兰卡使馆和中国警方协助。
公安部称，此次联合行动是中斯两国深化执法合作、加大打击跨境电信诈骗力度的成果。自今年年初以来，双方针对部分涉诈人员及相关地下产业由他国转移至斯里兰卡的态势加强了协作。
公安部透露，斯里兰卡警方近期多次开展清查行动，摧毁数个诈骗窝点并抓获多名涉案人员。参与此次遣返工作的中国湖北省公安机关正在对相关案件进行侦查。
公安部一名官员表示，将继续加强与更多国家的执法合作，共同构建打击跨国犯罪的务实高效新机制。来源：新华社

深圳报道 — 中国国务院总理李强在周五举行的高层座谈会上，要求加快在制造业推广工业人工智能，推动制造业的数字化与智能化升级。李强在与新兴独角兽Moonshot AI创始人杨志林等企业家和业界专家座谈时指出，要更深入落实“AI+”战略，加快把人工智能嵌入制造环节，强化先进制造与现代服务的融合，增强产业体系整体实力。 
李强并表示，尽管外部环境今年更为复杂多变，但在及时出台积极的宏观政策支持下，经济已开局良好。他还强调要抓住新兴产业发展机遇，扩大就业、提升收入、拉动内需，并鼓励企业家在创新和带动就业方面发挥更大作用。 
受邀参加座谈的还有广发证券首席经济学家郭磊、中国人民大学金融学教授马广荣以及招商局港口控股董事长冯伯明等。

美国联邦通信委员会（FCC）周四表示，正考虑更广泛的措施，禁止中国电信运营商在美运营数据中心，并进一步限制其接入美国网络与基础设施。草案同时建议禁止美国运营商与这些被认为存在“重大国家安全风险”的公司互联互通。中国移动、中国电信和中国联通均被列入受限实体名单，可能被要求出售、转让或关闭在美的数据中心和网络节点，甚至被迫退出美国市场。
分析人士称，此举标志着华盛顿对中国科技企业多年打击的升级，尽管短期内对相关公司财务影响有限。塔霍塔律师事务所合伙人王宇祥表示，草案将管制从禁止向公众直接提供服务和限制硬件，进一步延伸到对底层互联网基础设施和互联协议的更深层控制。

美伊在伊斯兰堡举行的首轮面对面谈判首阶段已结束，双方专家团队继续就关键问题交换书面文本。伊朗国家广播电视台（IRIB）援引接近谈判团队的消息称，谈判可能在今晚晚些时候或周日继续。
