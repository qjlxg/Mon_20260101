德国团队用无人机收集数据 预测未来火山喷发
德国研究团队正在测试一套新的火山气体测量系统，以便更好地预测未来火山喷发。慕尼黑大学一名火山学家7月9日操控无人机，在西西里岛的“火山岛”武尔卡诺岛（Vulcano Island），收集拉福萨（La Fossa）火山口的气体排放数据。
无人机沿着传感器的轴线调整位置，传感器则发射出一束不可见的激光束，穿过火山气体喷发物，然后照射到反射器上，算法会计算出无人机在10到15分钟内沿预定路径飞行期间的气体浓度分布图。研究人员说，使用激光可以让传感器避开气体羽流，从而避免遭到腐蚀。
无人机的最大优势之一在于灵活度极佳，能够绕过烟柱中较为稀薄的区域，并迅速改变方向；它还能去到人类无法徒步到达的地方，从而保障研究人员的安全。（法新社）
