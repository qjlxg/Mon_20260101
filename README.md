日本6000米深海开采发现稀土 盼降低对中国依赖
（东京综合电）日本政府星期一宣布，在一次试验任务中，日本深海探测船从约6000米深的海底采集到含有稀土的沉积物。
日本说，这次任务是世界上首次尝试在如此深的海域开采稀土元素，旨在降低对中国稀土的依赖。
稀土是制造晶片、电动车、武器等多种产品的关键材料。中国掌握全球近七成的稀土产量。
日本政府发言人佐藤佳星期一（2月2日）说：“我们将对样本进行详细分析，包括其中具体含有多少稀土元素。”
他说，这是在经济安全和综合海洋开发方面具重要意义的成果。
日本深海探测船地球号上个月启航前往最东端的离岛南鸟岛，在被认为蕴藏丰富矿产资源的周边海域进行勘探。
据《日经新闻》报道，南鸟岛附近海域位于日本专属经济区内，拥有全球第三大稀土储量，估计蕴藏着超过1600万吨稀土，包括镝和钇等重稀土元素。
镝（dysprosium）用于制造晶片，钇（yttrium）则用于制造发动机特种合金和耐高温涂层。
国际战略研究所（IISS）研究员上须贵弘（译音）告诉法新社：“如果日本能在南鸟岛周边持续开采稀土，这将保障关键产业的国内供应链。同时也将成为高市政府重要的战略资产，大幅减少对中国供应链的依赖。”
与西方盟友一样，日本一直在减少对中国关键矿产的依赖。随着与北京爆发重大外交争端，这一努力变得更加紧迫。
去年11月，日本首相高市早苗发表“台湾有事论”触怒北京，中日关系因此跌至谷底。过去两个月，北京从经贸、军事及人文交流等领域反制，施压高市撤回相关言论。

日本将拍卖30年期国债 市场视为高市“公投”
日本大选进入倒计时之际，首相高市早苗正迎来比选民更早、更严峻的考验，来自债券市场的“投票”。
路透社报道，日本财务省将于星期四（2月5日）拍卖约7000亿日元（约57亿新元）的30年期国债，时间距离投票开始仅三天。市场将此次拍卖视为对其扩张性财政主张的一次“公投”。
高市早苗承诺暂停食品消费税、财政纪律或进一步放松的背景下，超长期日债收益率已多次在拍卖前后飙升至历史新高，凸显投资者对日本高达国内生产总值（GDP）230%债务水平的深层担忧。
瑞穗证券交易策略主管大森祥纪（Shoki Omori）说：“这次拍卖本质上就是一次公投，反映投资者如何看待此次选举带来的财政风险。”他预测拍卖需求可能偏弱，因为投资者会保持谨慎，且有充分理由这样做，并称拍卖结束后收益率可能会上升。
这些担忧体现在期限溢价上，即投资者为持有更长期债券而要求的额外回报。大森祥纪测算的30年期国债期限溢价为2.8个百分点，“明显高于”10年期国债的1.6个百分点。
在过去五次30年期国债拍卖中，有四次收益率在拍卖前夕或结果公布后立即飙升至历史新高。其中包括去年10月7日的剧烈抛售，当时距离高市赢得自民党党魁之争、即将出任首相，仅过去三天。
这是高市1月19日宣布提前大选、并承诺暂停消费税以来，首次进行的30年期国债拍卖。最新民调显示，自民党有望大胜，令债券市场神经紧绷。
日本国债市场主要由国内投资者支撑，基本不存在资本外逃的实际风险，但海外账户（主要是对冲基金）在日本超长期国债中的作用日益突出。随着寿险公司和养老金等传统买家退场，这些“快钱”填补了空缺，也加剧了这一历来相对平稳市场的波动。
日本证券业协会的最新数据显示，去年12月，外国投资者占超长期现货日债交易量的约46%，高于一年前的13%。
大和资本市场欧洲研究主管斯奇克鲁纳（Chris Scicluna）说：“在超长期端，进场的并不是真正的长期资金，而是快钱。”
他补充称，价格持续下跌令日本的长期持有型投资者选择观望，因为“当你看到如此剧烈的波动时，没有人愿意去接一把正在下坠的刀”。

【特朗普拟设120亿美元关键矿产战略储备】金十数据2月2日讯，据外媒报道，多名政府高层官员称，特朗普拟启动一项战略性关键矿产储备计划，初始资金规模为120亿美元。项目名为“金库计划”（Project Vault），将把16.7亿美元的私人资本与美国进出口银行提供的100亿美元贷款结合起来，用于制造企业采购并储存矿产资源。该计划类似于现有的战略石油储备，预计储备范围将涵盖稀土和关键矿产，以及其他具有战略重要性、价格波动较大的元素。目前已有十多家公司参与该项目，包括通用汽车、Stellantis、波音、康宁、GE Vernova以及谷歌。三家大宗商品贸易公司Hartree Partners、Traxys North America和Mercuria Energy Group已签约，负责采购原材料。美国进出口银行董事会将于周一晚些时候就这笔创纪录的15年期贷款进行表决。特朗普计划于周一与通用汽车CEO玛丽·博拉以及矿业亿万富翁罗伯特·弗里德兰会面，两人分别代表关键矿产的使用方和生产方。

【CBOE拟重新推出二元期权 对标预测市场】金十数据2月2日讯，据华尔街日报，芝加哥期权交易所全球市场正计划重新推出面向个人投资者的二元/非有即无期权（all-or-nothing option），与预测市场展开竞争。交易者使用二元期权对某一事件进行“是或否”的押注。与预测市场合约类似，这类衍生品允许交易者对特定事件结果下注，若事件发生，获得固定现金赔付；若未发生，则损失全部本金。一个关键区别在于，CBOE打算坚守金融市场本业。该交易所希望随着时间推移，参与二元期权的投资者能够逐步过渡到更复杂的合约交易。CBOE曾在2008年推出与标普500指数和CBOE波动率指数挂钩的二元看涨期权，但在当时期权交易主要由华尔街专业人士主导的市场环境下，这些产品反响平平，随后被下架。
