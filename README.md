此次决定是在美国总统唐纳德·特朗普强化对格陵兰收购努力并威胁新关税后作出的，标志着美欧紧张关系再次升级。此番对峙已震动全球金融市场，刺激了有关贸易战与报复性关税的担忧。 
周二美欧股市普遍下跌，欧洲市场连续第二天下挫；道指下滑逾1.7%，标普跌超2%，纳指收盘约跌2.4%。美元走软，欧元一度升逾0.8%至1.1749美元后回落，英镑收盘微涨至1.343美元。长期国债遭抛售推动包括美、英、德在内的30年期国债收益率上升，全球借贷成本随之上行。 
背景上，7月在特恩伯里达成的协议将大多数欧洲商品的美国关税降至15%（低于最初威胁的30%），换取欧洲对美投资及有利于美国产品出口的调整，但该协议须经欧洲议会批准。 
在特朗普就格陵兰问题威胁关税数小时后，德国资深议员曼弗雷德·韦伯表示“目前无法批准”，负责国际贸易事务的议会委员会主席伯恩德·朗格称，鉴于对格陵兰的主权威胁与以关税作为胁迫手段，别无选择只能暂停相关工作，直至美方回到合作路线。 
暂停决定也令欧盟是否对美采取报复措施悬而未决。欧盟曾列出约930亿欧元的美国产品清单作为可能加征关税的目标，并在谈判中将该计划暂缓；但该缓刑将于2月6日结束，若不延长或不批准新协议，欧盟关税将于2月7日生效。法国总统马克龙等人呼吁考虑包括“反胁迫工具”（俗称“贸易大炮”）在内的报复选项。 
在达沃斯，部分美国官员敦促欧方保持克制并等待总统到场表态；同时美方也警告不会对任何报复坐视不理。美欧互为最大贸易伙伴，2024年商品与服务贸易额逾1.6万亿欧元，约占全球贸易近三分之一。 
此外，围绕特朗普去年宣布多项关税是否合法的美国最高法院待决裁决也在背景中掣肘局势。 来源：BBC/通讯社

评论：伊朗动荡会迫使中国重新审视其中东战略吗？
自去年12月底以来，伊朗多省爆发的抗议持续蔓延到1月，遭遇严厉镇压并加剧民众愤怒。此次危机不同以往之处在于，其持续性正瓦解外界（尤其是中国）对伊朗“可控且可预测”的既有判断。
长期以来，北京把伊朗当作虽受制裁但政治上持久、对中国战略有用的伙伴，视其内部动荡为不会影响能源供应与合作的“背景噪音”。但在制裁下，伊朗对华的经济依赖已超出常规贸易关系——中国几乎承接了伊朗大部分石油出口，通过折扣且脆弱的安排获取廉价能源，这种不对称格局长期有利于中国。
然而，持续的政治不稳让这种模式暴露风险：能源安全不仅看产量，更取决于政治延续性；动荡从偶发变为常态时，即便廉价石油也将带来战略风险。面对以保守求生的伊朗政权与广泛逮捕、致命镇压，中国长期秉持的不干涉政策面临两难选择。
