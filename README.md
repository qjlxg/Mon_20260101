台军更名恢复军官学校毕业生反共爱国教育课程
时隔多年，台湾军方今年起将军官学校毕业生的爱国教育政训课程，更名恢复为反共爱国教育。
据台湾军闻社报道，台湾国防部参谋总长梅家树上星期三（7月1日）在台湾国防大学复兴岗校区主持“民国115年三军八校院反共爱国教育”精神讲话。他说，正因有一代又一代的官兵坚守岗位，台湾才能更安全，民众对台军也才更有信心。
《联合报》报道称，根据军史研究者考证，早年台湾军校分别自办毕业典礼，1962年首次举行三军官校联合毕业典礼，1967年加入政工干校，1986年再加入国防管理学院，成为三军五校联合毕业典礼。
除了联合毕业典礼外，各校毕业生集中到政战学校，进行两周反共复国革命教育课程，通过将领、专家学者反共讲演及国防部的思想教育，以及分组讨论，强化反共意识。
随着时代演变，课程陆续改为反共爱国革命教育、反共爱国教育等名称，2006年起改为爱国教育；课程精简浓缩，加入区域安全情势、领导统御、压力管理与情绪调适、生心理卫教等内容，时间缩短为一周。
报道引述参与过的不具名官员回忆，随着台湾民主化与政党轮替，传统反共抗俄与光复中国大陆的反共爱国教育逐渐转型。台军教育的重心开始转向认识大陆敌情，以及深化全民国防理念等。
据报道，今年台湾军方重新恢复反共爱国教育，除了开训日由梅家树精神讲话及各军种司令勖勉外，陆续由台湾陆委会主委邱垂正、国安会咨询委员黄重谚、调查局国家安全维护处处长叶丽卿等台湾国安高层官员上课指导。
据《自由时报》报道，邱垂正上星期四（7月2日）在课程上指出，中国大陆近期制定“民族团结进步促进法”，更是扩大大陆对境外的管辖范围。他称，此举无疑是在破坏两岸关系的和平稳定，并希望初任官员能认清这些手段，坚定自身信念与意志。

她说：“想到我们已经走过了250年，而这个国家仍然有人无法真正享有自由，这真令人失望。”
来自波多黎各、现居华盛顿的冈萨雷斯也有类似的看法，他称对“事情的处理方式，以及国家的运作方式感到某种悲伤”。

据介绍，登记指引适用于在民政部申请成立登记的国际科技组织，即由中外高等学校、科研院所、科技组织、企业等单位或者个人共同发起，在中国境内成立登记，旨在促进自然科学和工程技术领域国际合作、国际交流和国际治理的国际性（或者区域性）、非营利性、非政府的社会团体法人。在经济、教育、文化、卫生、体育、环保等领域发起设立国际性社会组织，可参照适用。
登记指引要求，成立国际科技组织须有适当的中外发起人，发起人为初始会员；有规范的名称，应按照国家有关规定经过批准，冠以“国际”“世界”“全球”“亚洲”等体现组织国际性或者区域性的字样；有合法的资产和经费来源，成立时的活动资金为捐赠资金，不低于200万元（人民币，约38万新元）等。
在申请程序上，主要包括五个环节，分别是明确业务主管单位，业务主管单位审核，登记管理机关审查，准予登记和发放证书，以及刻制印章、办理税务登记和开立银行账户。
