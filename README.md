演唱会引流加高端客支撑 澳门3月博彩收入同比增15%
受益于大型娱乐活动带动客流，以及高端客需求保持韧性，澳门3月博彩收入同比增长15%，超出市场预期。
综合彭博社和澳门广播电视股份有限公司报道，澳门博彩监察协调局星期三（4月1日）公布数据显示，今年3月博彩收入达226.1亿澳门元（36亿新元），同比增长15%，高于分析师此前约11%的预估中值。
目前，澳门高端博彩业务继续展现韧性。韩国流行音乐组合ATEEZ成员及艺人段宜恩的演唱会等大型娱乐活动，带动客流回升，也为博彩收入提供支撑。花旗月度调查显示，尽管3月贵宾厅高端客数量同比有所减少，但人均投注额上升17%，使整体投注规模大致持平去年同期。
客流方面，澳门2月入境旅客达420万人次，同比增长33%；3月访客数据预计将在4月稍后公布。
不过，在澳门政府与六大博彩运营商持续加码演唱会等非博彩元素以吸引游客的背景下，如何将客流有效转化为博彩收入，仍是行业面临的关键挑战。尽管2月博彩收入在农历新年带动下超预期，但增幅明显落后于旅客人数增长。CreditSights在本月报告中指出，2月澳门人均博彩收入同比下降21%，反映出春节访澳旅客中相当一部分属于非博彩消费群体。

业界：伊朗战争或致英国食品饮料涨价近10%
英国食品饮料制造商游说团体警告，受伊朗战争影响，英国食品价格到今年年底将上涨近10%，涨幅约为之前预测的三倍。
路透社报道，英国食品饮料联合会（Food and Drink Federation，下称联合会）星期三（4月1日）说，根据此前预估，到今年12月，英国食品和非酒精饮料的通货膨胀率本该下降到3.2%，如今却预计攀升到9%以上。
这个团体代表英国约1万2000家食品和饮料制造商。上述预测的前提是，伊朗航运要道霍尔木兹海峡在两到三周内重新开放货运，而且包括石油、天然气和化肥在内的大多数关键设施在一年内恢复正常。
联合会说，如果供应中断持续时间更长，或者能源供应所受影响更为严重，英国食品通胀可能进一步恶化。
联合会还指出，食品和饮料行业能源消耗量大，且依赖复杂的全球供应链，因此极易受到油气市场冲击的影响。
此外，大小企业在这场危机中的应对能力也存在差异。大中型企业正为能源价格飙升做准备，而通常在现货市场购买能源的小型生产商，已经面临成本激增的局面。
联合会的首席经济师达尼拉（Liliana Danila）说：“虽然企业竭尽全力不把（能源）价格上涨转嫁给消费者，但很明显，未来几个月食品通胀仍会加剧。”

赖清德：台湾对陆投资降至3.7% 美国成最大投资目的地
台湾总统赖清德说，台湾对中国大陆的投资比重已从2010年的83.8%降至2025年的约3.7%，目前美国已成为台湾最大的对外投资目的地和出口市场，并称台湾将持续与美国深化合作，维护全球供应链安全。

韩国政府将研究可否经红海运原油
韩国总统李在明发出指示，要政府研究从中东经红海运输原油的可行性，以便缓解因伊朗战事导致的供应中断。
路透社报道，韩国总统办公室星期三（4月1日）发布上述消息。 
随着美伊战事和霍尔木兹海峡封锁的持续，全球能源危机日益凸显，在探寻原油运输的替代航线的同时，韩国政府也再次上调针对原油的资源安全危机预警至“警惕”。 
韩联社报道，韩国产业通商部星期三宣布，经过有关部门会议决定，从星期四（2日）零时起，将针对原油的资源安全危机预警，从第二级别的“注意”上调至第三级别的“警惕”。
针对天然气的危机预警，也会从第一级别的“关注”上调至第二级别的“注意”。　　
韩国资源安全危机预警从低到高共分四级，分别为关注、注意、警惕、严重。
中东战争打响之后，韩国政府3月5日首次针对原油发布“关注”预警；之后因霍尔木兹海峡持续封锁等导致能源供应告急，又于18日将预警级别上调至“注意”。
当局说，由于国际原油市场采购出现部分问题，加之国内原油库存已减少20%，开始对经济和产业产生影响，政府决定再次提升预警级别。
韩国政府试图加强供需管理，鼓励民间部门从其他渠道获取原油，并收紧民官需求管控力度，扩大实施车辆尾号限行范围，同时加强对石脑油和石油制品供应链的管理。
