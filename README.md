美联储主席通常会在任期结束时离开央行，但鲍威尔选择继续担任理事一职，任期将持续到2028年。鲍威尔说，他留任后将保持低调，但在宣布决定留任时指出，政府持续对美联储施加法律压力。

LV诉茉莉奶白商标侵权 舆论称“四叶花中国古代已有”
奢侈品牌路易威登（LV）起诉中国茶饮品牌茉莉奶白，称后者涉嫌品牌视觉侵权。法院认定茉莉奶白侵害LV注册商标专用权，判其茉莉奶白主赔偿1030万元（人民币，195.8万新元）。
综合《羊城晚报》、新黄河大鱼财经等媒体报道，中国江苏省苏州市中级人民法院6月29日就路易威登诉茉莉奶白商标侵权案作出一审判决。法院审理认定，深圳市茉莉奶白餐饮管理有限公司，侵害了LV 七件四叶花卉图形注册商标专用权，茉莉奶白主体公司应赔偿LV经济损失1000万元、维权合理开支30万元，合计1030万元。
茉莉奶白创始人张伯丞星期四（7月2日）回复媒体称，公司将依法上诉。目前，茉莉奶白申请的相关四叶花卉图形商标处于“驳回复审”或“无效”状态。多家媒体援引律师分析称，二审完全推翻侵权认定难度较大，改判主要在于赔偿金额的调整。
判决消息传出后，茉莉奶白将小程序、客服头像等界面的品牌标识从黑色切换为彩色，被视为“停止侵权”的补救措施。针对此举，官方回应称这“只是一次常规运营决策”。
不过，中国互联网上的舆论却偏向茉莉奶白。有网民指出，LV纹饰使用的“四叶花卉”与中国古典纹饰“唐代宝相花”非常相似，“是属于全体国人的公共传统文化纹样”，为茉莉奶白叫屈。
平台数据显示，茉莉奶白短视频账号7月1日粉丝总量为171.4万，截至7月3日已涨至181.4万，两日间逆势涨粉10万。
