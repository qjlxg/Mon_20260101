受气温逐渐变暖影响，中国辽东湾海冰开始消融，预计将于2月下旬到3月初基本消融。

亚伯拉罕·纽曼是美国政治学者，现任乔治城大学外交学院与政府系教授，研究关注经济相互依赖与全球化如何改变国际政治。他与亨利·法瑞尔合著的《地下帝国：美国如何把世界经济武器化》于2023年出版。此次采访中，他谈到“武器化相互依赖”如何塑造当下全球地缘政治，尤其是对特朗普若连任的影响。 
许多人把全球化想象成权力去中心化的过程，企业主导、国家冲突式微。但纽曼与法瑞尔认为这是误读。无论是金融、通信还是生产网络，国际经济内部往往高度集中：iPhone的关键芯片主要来自台积电或三星；跨国金融交易经由少数几家银行；手机操作系统被苹果或谷歌主导。这种集中性让国家能够利用网络进行胁迫——通过监控或将对手排除在关键网络之外。因此，“武器化相互依赖”时代意味着全球化从单纯追求效率转向一个市场既是利益来源又是脆弱点的权力格局。

海牙 — 国际刑事法院（ICC）对前菲律宾总统罗德里戈·杜特尔特的确认指控听证将于本周一在海牙开庭。该案自2025年3月他在马尼拉被逮捕、面临三项危害人类罪指控并被移送至预审庭以来，已近一年。
确认指控为ICC预审程序，用以判断证据是否足以进入审判阶段。ICC菲律宾法律团队助理Ross Tugade指出，这不是“小审判”，而是决定是否进入全面证据评估与有罪或无罪裁断的重要步骤。根据《罗马规约》第61条，预审庭可确认指控并移送审判、拒绝确认或延期以便检方补证。虽少见，预审阶段仍有可能不确认指控，达尔富尔案件即为先例，但程序上与杜特尔特案有所不同（传唤与逮捕令之别）。
预审庭表示将给予被告更多时间陈述，并强调对被告正当程序权利的保障。法律通常要求被告出庭，但可放弃出席权。现年80岁的杜特尔特上周放弃出庭，称法庭对他“无管辖权”，并指自己被“强行带上飞机送往海牙”违背宪法与国家主权，称“年老、疲惫、虚弱”，将由首席辩护律师Nicholas Kaufman及外、菲律师团队代为应诉。
药物战争受害者代表不接受其“神志不清”说法，援引ICC独立专家认定杜特尔特适于出庭。辩方另请求撤销受害者律师Joel Butuyan、Gilbert Andres及案管Nicole Arcaina的代理资格，庭尚未裁定。预审庭在收到检方证据后有60天时间决定是否将案件移送审判，预计不会在原定的2月27日当日作出最终结论。
