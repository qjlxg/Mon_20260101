中国拟5月起限制硫酸出口 冲击金属和化肥行业
随着伊朗战争引发原材料供应瓶颈，中国已释放信号，拟自5月起停止硫酸出口，此举势必将冲击本已承压的金属与化肥行业。
彭博社引述知情人士报道，部分中国硫酸生产商近期已接获相关通知，一家大型买家也从中国供应商处获悉这一消息。此次限制措施将涵盖中国铜、锌冶炼过程中产生的副产品硫酸。
自伊朗冲突爆发以来，硫酸价格持续上涨。霍尔木兹海峡事实上被封锁，阻断了来自中东的硫磺运输，而硫磺是石油和天然气炼制的副产品。中东地区供应全球约三分之一的硫磺；硫磺是生产硫酸的重要原料，广泛用于铜提取和磷肥生产。
在农作物播种旺季临近之际，中国此举意在优先保障国内供应，但也将进一步收紧全球市场。供应趋紧预计将冲击智利、刚果民主共和国和赞比亚等主要铜生产国的采矿行业。
定价机构阿格斯（Argus Media）星期四（4月9日）率先披露相关出口限制，并称措施可能持续至2026年。中国商务部尚未就此作出回应。
目前，智利市场价格已显著上涨。智利每年从中国进口逾100万吨硫酸。作为全球最大铜生产国，智利约五分之一的铜产量依赖硫酸参与的加工工艺。阿格斯编辑马洛（Sarah Marlow）指出：“若限制措施持续全年，智利将面临比当前更高的价格压力。”
咨询公司CRU分析师哈里森（Peter Harrisson）也说，在硫磺原料同样紧缺的背景下，中国供应的减少将难以被其他来源填补。
