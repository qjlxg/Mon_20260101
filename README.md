警方称，Yves Sakila上周五在首都一条繁忙的购物街因涉嫌入店行窃被保安扣留，现场失去反应，随后被宣布死亡。随后在社交媒体广泛传播的视频显示，至少五名男子将他按倒在地近五分钟，两人将其面部压向地面，有人短暂停在其头部或颈部上方跪压。
Sakila的中学同学、35岁物理学生David Kaliba称，“这是一个乔治·弗洛伊德时刻”。黑人民权组织Black Coalition Ireland发言人Yemi Adenuga表示，视频令人震惊，政府未能为不断增长的移民人口提供有效融入措施，可能导致混乱与冷漠。
近年来爱尔兰反移民抗议明显增多，2023年都柏林市中心曾发生大规模骚乱，地点接近事发地。议员称视频“令人不安”，总理Micheal Martin呼吁彻查。议会外抗议者高呼“No cover up, no delay”，举着“Justice”和“cead mile failte is for everyone”等标语。
警方周四称已完成尸检，但出于行动需要暂不公布结果。消息来源：路透社。

阿联酋与沙特、卡塔尔共同敦促特朗普勿重启对伊战争
阿布扎比消息——知情人士称，鉴于担心若冲突重燃德黑兰报复将使海湾经济陷入混乱，阿联酋近日加入沙特与卡塔尔行列，分别与美国总统特朗普通话，敦促给谈判留出空间。三国领导人在电话中均表示，军事行动难以实现美国对伊朗的长期目标。
阿联酋此举显示立场转变：尽管曾在伊朗袭击中首当其冲并对德黑兰更为强硬，但现在与海湾邻国在避免冲突升级上达成共识。此次劝和源于2月底至4月初爆发的冲突记忆——以美以对伊发动攻势后，伊朗及其在伊拉克的盟友发射数千枚无人机与导弹，造成多人伤亡与港口、能源设施巨额损失。
伊美于4月8日停火并通过巴基斯坦交换谈判信息，但双方均表示随时可恢复交火；美国国务卿表示5月22日谈判有“些许进展”。阿联酋虽曾在4月决定退出由沙特主导的欧佩克，但随后与海湾合作委员会成员关系回暖，并强调任何协议须涵盖伊朗的核能力、弹道导弹、无人机及其代理武装。
同时，海湾国家担忧德黑兰仍保有大量打击能力：5月17日UAE核电站遭无人机袭击，阿联酋指责受伊朗支持的伊拉克民兵所为。多位知情人称，沙特与阿联酋倾向利用海上封锁与巴基斯坦调停施压，卡塔尔则持续主张降级局势。
阿联酋总统顾问称，与伊朗达成协议“概率约五成”，但警告若重启军事对抗只会使局势更复杂。各方呼吁以外交避免“第二轮”冲突，因战争已令美方付出巨额代价并推高能源价格，且国际与地区稳定受到威胁。

迪拜报道 — 伊朗当局在德黑兰大肆张贴颂扬“国家团结”和“战胜超级大国”的宣传画面，并在街头举办军主题集体婚礼、清真寺公开械训等活动，以在国内外塑造抵抗形象。此举发生在当局以大规模镇压平息抗议、并在战争中加剧民众经济困境数月之后。
与过去强调宗教革命话语不同，官方宣传更多使用民族主义与历史英雄符号——例如将抗英游击领袖莱斯·阿里·德尔瓦里与革命卫队并列，或以“关闭霍尔木兹海峡”及捕捉美舰的画面嘲讽特朗普——试图动员更广泛民众基础。
国际危机组织伊朗项目主任阿里·维兹称，这是从神权向军事化转变的一部分；圣安德鲁斯大学阿里·安萨里教授认为，这类表演能在一定程度影响中间选民，但多数伊朗人仍持怀疑。国家电视甚至播出未戴头巾的女性采访，反映宣传口径的调整。
尽管发动近夜集会以示支持、并以武装化场景威慑异见，民众反应分化：有人指责当局应优先解决经济问题，亦有人对公共场合男女互动表示愤慨。分析人士指出，宣传意在显示政权强硬与团结，但能否扭转民间不信任尚存疑问。

麦加——尽管中东局势紧张、停火脆弱，超过120万名穆斯林正齐聚麦加参加自5月25日开始的2026年朝觐（哈吉），这场一年一度的大型宗教活动笼罩在战争阴影之下。伊朗对沙特及海湾邻国发动多轮袭击，目标包括能源设施、机场、出口码头与霍尔木兹海峡等民用基础设施，令地区敌意持续燃烧。
沙方力图将冲突与来访朝觐者隔离，但来自德国的36岁家庭主妇法蒂玛说：“我们毫不犹豫来到麦加，觉得这里是世界上最安全的地方。”
受“战时局势”影响，伊朗官方媒体IRNA称，预计8.67万伊朗朝觐者中仅约3万人抵达。历年朝觐曾成为利雅得与德黑兰紧张关系的导火索：2015年拥挤踩踏致逾2300人死亡，其中464名为伊朗人，随后两国关系一度破裂，直到2023年在中国斡旋下恢复外交关系。
沙国内政部本周重申，朝觐期间严禁任何政治口号或煽动性、宗派性旗帜。与此同时，面对酷暑威胁（预计多数时间气温超40摄氏度），沙方在去年500多人死亡的惨剧后加强了防暑与医疗部署，本届提供逾5万医护人员与3000辆救护车。
尽管有战争与高温隐忧，朝觐者情绪依然激昂。47岁的艾哈迈德·阿布·塞塔说：“朝觐是我的终生梦想，如今终于实现。”
（法新社）

世卫组织将刚果（金）本迪布乔（Bundibugyo）埃博拉病毒在全国蔓延的风险评估上调为“非常高”，并将区域风险评为“高”、全球风险为“低”。世卫总干事谭德塞表示，该病毒株目前无获批疫苗或特效药，世卫已于5月22日将其列为国际关注的突发公共卫生事件。
目前刚果（金）已确诊82例，确诊死亡7例，疑似死亡177例，疑似病例近750例。乌干达形势相对稳定，发现两例由刚果（金）入境的感染者，其中一人死亡。
世卫应急响应主管阿卜迪拉赫曼指出，该病毒有高、甚至非常高的快速传播潜力，改变了防控态势。谭德塞称，乌干达通过加强接触者追踪并取消大型集会等措施，已在一定程度上遏制传播。
一名在刚果（金）工作的美国公民确诊后被转送德国救治；另一名与高风险病例有密切接触的美国人被转运至捷克。
世卫首席科学家西尔维·布里安表示，可在接触者中考虑使用名为Obeldesivir的抗病毒药物做预防性干预，但该药为吉利德公司用于新冠的口服实验药，须在非常严格的方案下使用。
