“棋圣”聂卫平逝世 中央有关领导表示哀悼
有“棋圣”之称的中国围棋协会名誉主席聂卫平上个星期三（1月14日）在北京病逝，享年74岁。官媒新华社星期天（18日）报道，聂卫平同志逝世后，中央有关领导以不同方式表示哀悼并向其亲属表示慰问。
聂卫平在1980年代的中日围棋擂台赛中，连胜多位日本超一流棋手，被当时的国家体委授予“棋圣”称号。
聂卫平也是中国国家主席习近平的青少年密友，两人是北京市二十五中的同学，年轻时经常一起观看足球比赛，习近平也曾向聂卫平讨教围棋。
聂卫平生前接受韩国媒体采访时披露，习近平为时任中国副总理耿飙当秘书时，耿飙认为下围棋能训练大局观，要身边所有工作人员都学会下围棋，习近平为此专门向聂卫平讨教。
已故著名武侠小说作家金庸也是聂卫平的忘年之交。金庸晚年酷爱围棋，1983年他以三鞠躬仪式，拜比自己小28岁的聂卫平为师。
综合新华网、人民网与《星岛日报》消息，聂卫平女儿聂云菲上个星期四（15日）在社媒平台发布讣告，指聂卫平因病医治无效，星期三晚10时55分与世长辞。
根据讣告，聂卫平2013年11月历经直肠癌手术，术后仍活跃于围棋赛场。2025年3月，他突发脑梗昏迷12天，苏醒后积极康复；本周三他在医院突发疾病，经抢救无效离世。
新华社星期天报道指出，聂卫平是中国著名围棋大师，第六届全国政协委员，第七届、八届全国政协常委，民盟中央原常委，中国围棋协会名誉主席，国家围棋队原总教练，中共优秀党员。
