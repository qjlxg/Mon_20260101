日媒：中国加强管制后已批准多项对日稀土出口
日本媒体报道，中国加强管制后已批准多项对日稀土出口。
日本共同社报道，多名贸易消息人士星期五（2月6日）称，中国有关部门在1月加强两用物项对日本出口管制后，已批准多项对日稀土出口。与此同时也获悉，日中两国之间贸易进出口接连出现通关延迟情况，其中包括工矿业领域。
消息人士称，日本一度担忧加强管制将导致出口暂停，但实际并未全面禁运。
中国商务部1月宣布，加强两用物项对日本出口管制。共同社报道，对象涵盖七类稀土、稀有金属、电子设备等数百种产品。
中国商务部在公告中说，禁止所有两用物项对日本军事用户、军事用途，以及一切有助于提升日本军事实力的其他最终用户用途出口。
贸易消息人士称，管制加强后，包括七类稀土在内的对日出口已有多项获批。共同社称，这些出口申请被认为是在加强管制前提交的，日本企业正在密切关注今后能否继续获批。

马尼拉（ABS-CBN News）—— 代理财政部部长罗兰多·托莱多（Rolando Toledo）周五表示，他反对在筹编中的2027年国家预算中取消“未编列拨款”(unprogrammed appropriations, UA)。
他解释，未编列拨款没有固定资金来源，通常作为备用，仅在政府出现金额超收、新税利得、出售国有资产或获批外援贷款等情形才动用。托莱多表示，问题不在于拨款本身，而在于使用范围，应严格限定纳入的计划与项目。2026年的UA他只限于外援项目、风险管理计划及菲律宾武装部队现代化计划。
他指出，UA对那些在年度支出计划既定后仍获批的外援或公私合营项目尤其重要；项目一经投资协调委员会批准、且出现超收时，才会啟用UA以继续推进工程。2013年—（注：原文为2023年）2023至2024年间，公共工程和公路部曾以UA拨付数十亿比索用于防洪工程，这一时期也出现所谓异常工程激增的国会调查披露。
托莱多称，他是到2025年总统费迪南德·马科斯在国情咨文中揭露相关问题后才注意到这些争议；部门通常依赖工程单位及审计署的文件与报告。前国家发展署长温妮·蒙索德（Winnie Monsod）对此表示失望，质疑经济学院出身官员在位时竟发生此类腐败丑闻。
托莱多承诺将加强预算监管并推行改革，包括发布国家预算通告595-A，要求机构首长对应付账款出具签署并公证的承诺书，确保所支付项目真实存在且合规。
关于谁有权核准UA款项释放，托莱多与前财政部长本杰明·迪奥克诺（Benjamin Diokno）意见不同：迪奥克诺称需总统核准，而托莱多认为预算部长在满足UA特别条款参数时即可签发特别拨款释放令（SARO）。蒙索德因此质疑总统早在2023、2024年就应知情。ABS-CBN已向马拉坎南宫求证回应。

【上海市人社局：全力促进高校毕业生等青年群体就业】金十数据2月7日讯，今天（7日）上午，上海市人民政府举行记者会。上海市人社局局长杨佳瑛表示，就业是民生头等大事。全力促进高校毕业生等青年群体就业。上海是广大青年学子就业逐梦的首选之地，我们针对青年群体的特点，强化政策支持，深入实施就业服务攻坚行动，充分发挥“一站（乐业上海第一站）、一圈（15分钟就业服务圈）、一行动（“乐业上海优+”行动）”的联动效应，加大对青年群体的就业帮扶力度。
