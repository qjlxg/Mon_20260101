中国称日本根本没资格要求成为安理会常任理事国
中国常驻联合国代表团临时代办孙磊称，日本根本没有资格要求成为安理会常任理事国。
据中新社报道，孙磊星期三（1月21日）在关于安理会改革政府间谈判的联大会议上发言说，安理会是国际集体安全机制的核心，对于维护战后国际秩序、捍卫国际和平与安全肩负着特殊重要使命。
他说，80年前的1946年1月，远东国际军事法庭正式开庭。东京审判严惩了日本战犯，伸张了国际正义，维护了人类尊严，也是对任何复辟军国主义、再行侵略扩张不法企图的强力警示。然而，日本军国主义不仅未被彻底清算，反而改头换面、潜滋暗长。日本右翼势力极力美化侵略历史，矢口否认南京大屠杀、强征“慰安妇”、强征劳工等历史罪行，推动修改历史教科书，企图为侵略历史翻案。多名日本现职领导人曾参拜军国主义的精神象征靖国神社，对甲级战犯顶礼膜拜。
孙磊说，从日本首相高市早苗发表涉台错误言论、对中国发出武力威胁，到日本高官公开发表拥核言论，再到推进修改“安保三文件”，声称修改“无核三原则”，充分暴露出日本右翼势力推动“再军事化”、企图让军国主义死灰复燃的险恶用心，对地区乃至世界和平与安全构成新的威胁。这种对历史罪行不知悔改、违背国际关系基本准则、挑战二战胜利成果、公然践踏战后国际秩序的国家，无法承担维护国际和平与安全的职责，无法取信于国际社会，根本没有资格要求成为常任理事国。
高市早苗去年11月初在国会发表“台湾有事”论以来，中日紧张局势不断升温。中国常驻联合国代表傅聪去年11月18日称，高市早苗的言论公然背离了日本所作的走和平道路的基本承诺，“这样的国家根本没有资格要求成为安理会常任理事国”。

中国对非洲贷款在2024年减半 转向人民币计价
数据显示，2024年中国对非洲的贷款规模几乎减半，降至21亿美元（27亿新元），创下冠病疫情以来首次年度下降。此外，中国对非洲放贷明显转向以人民币计价，并更加聚焦审慎、具有战略意义的项目。
据路透社报道，波士顿大学全球发展政策研究中心星期三（1月21日）发布的数据显示，2024年的贷款规模还不到2016年288亿美元历史峰值的十分之一。
研究中心指出，这一变化反映出中国正逐步淡出铁路、公路等大型基础设施项目，转而支持规模更小、商业可行性更强的项目。
报告称，随着动辄数十亿美元的大型项目时代逐渐落幕，中国不断演变的金融工具，正在塑造一个更为审慎、选择性更强的新阶段。2012年至2018年间，中国对非洲的年度贷款规模曾长期保持在100亿美元以上。
冠病疫情带来的经济冲击，促使赞比亚、加纳和埃塞俄比亚相继发生债务违约，北京在部分贷款上遭受损失。数据显示，自2000年以来，中国对非洲的放贷模式正在转变，逐渐告别“一带一路”倡议早期以美元计价的大型项目，转向以人民币计价、规模更小、目标更明确的融资安排。
报告指出，中国越来越多采用人民币计价贷款，通过非洲国家本土银行向中小企业转贷，并扩大对外直接投资（FDI）。数据显示，2024年中国对肯尼亚的所有基础设施贷款均以人民币计价。
此外，肯尼亚去年10月将来自中国的35亿美元贷款转换为人民币，埃塞俄比亚也在考虑类似安排。中国国家开发银行去年也与南部非洲开发银行签署协议，开展首个人民币计价的融资合作。
同时，单个项目融资规模超过10亿美元的案例明显减少，资金更多通过非洲地区性银行渠道投放，重点流向被认为具有商业可行性的项目。
2024年，中国在非洲仅资助了六个项目，其中两个位于安哥拉，肯尼亚、埃及、刚果民主共和国和塞内加尔各一个。获得14.5亿美元用于电网和道路升级的安哥拉，成为当年最大受援国，凸显北京更倾向于长期合作伙伴和具有战略意义的项目。
波士顿大学全球发展政策研究中心总结称，这些数据整体显示，中国对非直接放贷趋于保守，同时更多运用市场化金融工具，以降低成本、缓解债务风险，并支持可持续增长目标。
