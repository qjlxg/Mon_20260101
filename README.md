赖清德吁朝野年后开议速审国防特别条例 避免军购延宕
台湾总统赖清德表示，台军当前最迫切的需求，是能够及时获得先进的武器装备；国防特别条例迟迟未获通过，可能导致台湾跌出军售优先名单，延宕关键武器交付。他呼吁立法院朝野政党在农历新年后开议，尽速通过国防特别条例。
综合台湾《联合报》和TVBS新闻网报道，赖清德星期三（2月11日）在总统府举行“国家安全不能等！支持国防采购特别条例”记者会上发表谈话，防长顾立雄与参谋总长梅家树随后也就相关议题进行简报。
赖清德说，提升国防预算、维护国安绝非挑衅，而是展现台湾自我防卫的决心，这一方向也获得国际社会的高度支持。他并感谢美国政府持续支持台湾，已为相关军购项目提供报价，同时台湾强化自我防卫的立场，也获得美国白宫及参众两院跨党派议员的公开肯定。
他指出，面对日益复杂的区域情势，台军最迫切的就是及时取得先进、精准武器，以守护台澎金马安全；但若预算迟迟无法通过，不仅可能影响军售优先顺序、延误关键装备交付，也会让国际社会质疑台湾守护自身安全的决心。
针对近期舆论将预算延宕与政府未采纳在野党提案、为志愿役官兵每月统一加薪3万元相连结，并批评政府不照顾军人，赖清德澄清，这是完全错误的连结。他说，民进党政府比国民党政府更照顾台军，过去10年，执政团队除四度为军公教加薪、累计调幅达14%外，也推动台军装备更新、老旧眷舍改建，并完成23项加给调升；仅去年就调升五项加给，每年增加支出约138亿元。
赖清德强调，政党之间可以竞争，政策也应充分沟通，让人民作出选择；但攸关国安、主权与基本生存的国防议题，更应超越党派，团结一致、对外一致。
他提到，许多国际友人一再提醒台湾，强化国防不能再等待、也不能被拖延，并呼吁立法院朝野政党在年后开议后立即审议、尽速通过国防特别条例，“一起强化国防、守护国家，让台湾持续繁荣发展”。

据央视新闻报道，长征十号运载火箭系统低空演示验证与梦舟载人飞船系统最大动压逃逸飞行试验在星期三（2月11日）取得成功，标志着中国载人月球探测工程研制工作取得重要阶段性突破。

长征十号运载火箭与梦舟载人飞船完成最大动压条件下的逃逸飞行试验，意味着火箭在速度最快、气动压力最大的阶段若出现异常，飞船仍能在极端气动环境下将航天员迅速带离危险区域，并确保安全返回。

评论：美国如何从中国“产能过剩”中获益
西方媒体与政策制定者常指责中国存在“产能过剩”，但这更准确地反映了中国在太阳能电池、电池、 电动汽车、钢铁等领域以更低成本、更高效率生产的能力。
确有补贴，但这些行业同时受益于快速的技术积累和规模经济。问题不在于中国建得太多，而是其生产效率令竞争对手难以适应。
主流论述把中国描绘为建造“影子城市”、生产无人订购的商品并依靠海外市场消化剩余，西方把这解读为浪费、倾销和必然崩溃。然而我们一方面为中国商品价格下跌担忧，另一方面又为美国的通胀焦虑——对价格涨跌的双重标准暴露了意识形态分歧。
这种分歧反映出两种不同的增长观：在中国，增长体现为关键实物商品的价格下降；在美国，增长常以资产价格（股市、房市）上涨来衡量。两国不仅在如何增长上有分歧，也在于何为“增长”。
