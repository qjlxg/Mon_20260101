台国防部长批国民党版军购案三不合理 吁朝野理性审查
台湾朝野围绕对美军购案持续僵持，台国防部长顾立雄指国民党版“3800亿+N”军购案有三处不合理，包括所设2028年底的执行期限等于实质上封杀五个军购案。
台湾在野的国民党星期四（3月5日）拍板“3500亿元（新台币，下同，141亿新元）＋Ｎ”的国防特别预算条例版本，不过在稍后发布的新闻稿中又调涨数字，修改为“3800亿+N”。
“+N”的意思是，除了已公布的军购内容外，只要美国启动下一阶段对台军售，正式提出新的军售报价书，主管机关应立即研拟第二阶段特别条例草案，送请立法院审查，立法院会从速完成三读程序。
综合《自由时报》、“Newtalk新闻”等媒体报道，台湾立法院会星期五（6日）将朝野国防特别条例草案付委审查。国防部长顾立雄受访时直言，国民党版草案设定3800亿元上限，条文中并不存在“+N”的概念，因为条例规定若要增加，须另提特别条例，意味着上限金额就是3800亿元，“+N”的逻辑完全不存在、也不确实。
此外，顾立雄质疑，起草国民党版条例的人没看过国防部专案报告，把已纳入年度预算的项目也列入军购条例。
第三，顾立雄说，条例执行的年限只到2028年12月31日，包括精准火炮、远程精确的海马式、托式、标枪及反装甲的无人系统不可能在那之前交运完毕。因此草案相当于实质封杀这五项军购案，顾立雄三次强调“不可能执行”。
顾立雄称，行政院提出的1.25万亿元版本，缺了任何一块都会产生重大缺口及下行。他呼吁，希望行政院版付委后，不分朝野共同支持国防预算，一起理性面对、合理审查。

中国将于2027年在广东惠州点火一台加速器驱动亚临界系统（ADS）示范堆，预计成为全球首个兆瓦级原型。该项目由中国科学院与国有核企联合研制。中国科学院近代物理研究所副所长何元在3月2日《科技日报》报道中称，ADS既能发电又能将长寿命放射性废物转化为短寿命同位素，是“国际公认的核燃料增殖与废物处理理想途径”。
ADS为反应堆与粒子加速器的混合体，不能在无外部加速器的情况下维持链式反应，因而降低了失控风险。何元表示，该设计利用超导直线加速器产生高流强质子束，以约0.8倍光速轰击铅铋液态靶，产生大量中子，能将铀‑238转化为钚‑239，把“废物”变为新燃料。
中科院自2011年开展ADS研究，2021年研制的原型达到适用于工业的运行强度。该所正推进关键超导加速器的安装，力争今年完成。何元称，该装置可使铀燃料燃烧效率提高约100倍，并将核废物放射性寿命缩短到目前的千分之一以内，有望为人类提供可持续的“千年级”稳定能源。

研究：尽管失去巴拿马运河控制权，中国仍在全球港口投入239亿美元。
美国威廉与玛丽学院AidData实验室表示，过去25年间，中国国有机构在海外363个港口及相关项目累计投资约239亿美元，旨在防范东西方供应链脱钩及美国遏制中国供应链影响力的举措。
报告指出，约45.1%的港口融资流向20个“高收入”地区（包括澳大利亚、文莱、新西兰、西班牙和新加坡），覆盖30个港口。研究称，“中国在世界主要港口的广泛存在，使美国在和平或冲突时期难以脱离中国供应链。”

AidData列出的受资助重点包括斯里兰卡汉班托塔港、澳大利亚墨尔本港与纽卡斯尔港、喀麦隆克里比自治港及以色列海法港，单个项目融资规模在约11.3亿美元至19.7亿美元之间。
