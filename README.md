饶舌歌手出身的政治新星巴伦·沙阿（Balendra “Balen” Shah）领导的Rastriya Swatantra Party（RSP）在周六的大选中取得压倒性胜利，预计将组建新政府。 
选举委员会数据显示，35岁的巴伦在Jhapa-5选区以68,348票击败曾四度任相的共产党（统一马列）主席K.P.夏尔马·奥利（K P Sharma Oli，74岁）的18,734票，胜幅约5万票。若就任，他将成为尼泊尔史上最年轻、亦为首位来自马德希族的总理。 
RSP成立于2022年，党内由拉维·拉米查内（Ravi Lamichhane）发起，截至清晨已在117个已公布结果的席位中赢得91席，包揽加德满都谷地全部15席，并在全国34个席位领先。主要传统政党失利：尼泊尔国大党13席（领先3），CPN-UML仅7席（领先3），NCP4席（领先3），RPP1席，另有1位独立候选人当选。 
选举投票率约60%，165个单席选区的计票仍在进行。印度总理莫迪向尼泊尔人民和政府致贺，奥利亦祝贺巴伦能顺利执政五年。 
其他结果包括：RSP主席拉米查内在Chitwan-2大胜，前总理普什帕·卡马尔·达哈尔（Prachanda）在Rukum Purba当选，亲王党RPP的吉亚内德拉·沙希在Jumla胜出。此次共有10名女性当选（其中9名属RSP）。 
分析称，选民对反腐、结束裙带关系和政坛代际更替的诉求促成本次高票表现。来源：PTI

美国总统唐纳德·特朗普周六表示，英国正在“认真考虑”向中东派遣两艘航空母舰，但他称美方不需要这些舰艇来赢得与伊朗的战争。他在社交平台Truth Social上写道，英国——曾经的伟大盟友——现在想派航母到中东，“没关系……我们不再需要他们，但我们会记得。我们不需要在战争已赢后才加入的人。”
英国国防部称正准备将“威尔士亲王号”航母作可能部署之用，但一名官员表示尚未作出最终决定。
特朗普多次批评英国首相基尔·斯塔默，称伦敦阻止美方最初利用英国基地对伊朗发动攻击，伤害了两国长期密切关系。斯塔默为不允许美军使用英国基地支持初期打击辩护，称须确认任何军事行动合法且周密；随后他批准美军使用英方基地对伊朗储存或发射导弹的设施实施所谓防御性打击。
斯塔默今年早些时候也批评过特朗普欲购格陵兰的提议，并称其关于欧洲部队在阿富汗避开前线作战的说法“令人愤慨”。
