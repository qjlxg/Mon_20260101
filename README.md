费恩利证券（Fearnley Securities）分析师指出：“鉴于近期局势发展，我们预计船东将采取‘观望’策略，运输效率可能进一步下降。”

全球规模最大的海上保险互助协会中，逾半数已取消进入波斯湾船只的战争风险保险。

英军塞浦路斯基地遭无人机袭击 近40年来首次
一架无人机于星期一（3月2日）凌晨袭击了英国皇家空军位于塞浦路斯阿克罗蒂里的基地，造成轻微损毁，无人员伤亡。塞浦路斯总统和英国国防部证实了这一消息。
路透社报道，这是自1986年利比亚武装分子发动火箭袭击以来，该英军设施首次遭袭，事件被视为冲突明显升级。
塞浦路斯总统赫里斯托祖利季斯（Nikos Christodoulides）在讲话中称：“共和国所有相关部门都已进入警戒状态，并保持全面运作准备。”
他补充称，这架伊朗制造的“沙赫德”（Shahed）型无人机于凌晨12时03分坠入军事设施，造成轻微损坏。
他强调：“我要明确表示，我们国家没有以任何方式参与，也无意参与任何军事行动。”
目前尚不清楚这架伊朗制造的无人机从何处发射。两名不愿具名的消息人士透露，英军基地还拦截了第二架无人机。
尽管这些基地被视为英国主权领土，但塞浦路斯本身是欧盟成员国，目前正担任欧盟轮值主席国。
阿克罗蒂里位于塞浦路斯西南部、沿海城市利马索尔附近，是英国自1960年塞浦路斯独立以来保留的两处主权基地之一。除军事设施外，该基地也居住着服役军人的家属。
阿克罗蒂里位于塞浦路斯南端一块呈方形的半岛上，过去曾被用于伊拉克、叙利亚和也门的军事行动。
3月1日，英国首相斯塔默称，英国已接受美国请求，允许美方使用其基地，对伊朗储存的导弹或发射装置进行防御性打击。

（2026年3月2日12:17，政府办公厅）外交部长色哈萨克·普旺盖凯（Sihasak Puangketkaew）在国家安全委员会会议后表示，美伊冲突可能长期化，政府已准备协助撤离处于风险地区的泰国公民。若在伊朗要求撤离人数超过100人，将派空军包机接回；目前已与空军初步联络。
他称伊朗约有200多名泰国公民，驻伊朗使馆与社区保持密切联络，提醒提高警惕并待在安全区。现在该地区领空关闭，欲回国者需先经陆路前往土耳其边境再转机，现阶段约20人表示愿意回国，预计可由商业航班接回，并设立边境运作中心提供协助。
以色列约有6.5万名泰国人，目前局势可控，尚无撤离需求；阿联酋（迪拜、阿布扎比）约千余人表示想回国，可经阿曼机场转机。巴林、卡塔尔、科威特等受波及国家的使馆已联系在地泰人，安排经最近城市（如沙特）搭乘商业航班撤离。外交部已设立联络中心，供家属查询。
他并指出，若要从伊朗撤回20多名公民，需经千里陆路并通过多处检查点，必须与伊朗方面协商确保人道便利与安全通行。对冲突是否会延长，他判断可能性大，因美方目标不仅限于核问题，伊朗也将采取报复行动，影响范围已扩展到有美军基地的多国，泰国需全面准备。驻外使馆目前能与在外国民联络，但对补给问题表示担忧。

监察长赫苏斯·克里斯平·雷穆拉周一证实，他曾与国际刑事法院（ICC）工作人员会面。此前，上周有律师列维·巴利戈德与18名自称前海军陆战队员向监察长办公室提交联名宣誓书，称雷穆拉在任司法部长时与ICC人员接触，指控其对政府造成“非当性损害”，并提出与ICC“勾结”“密谋”的可能性。
雷穆拉在一次偶遇采访中表示：“我说我有谈过。现在怎么办？”他称会面时间大约在2025年3月之后，不到一年，强调要保护证人，但更关心当事人提出的具体指控，已要求取得这18人的个人宣誓书，称这些指控很严重。
巴利戈德声称这18人曾充当涉弊基建回扣的“跑腿”，并被指示将200万美元现金交给参议员安东尼奥·特里拉内斯以贿赂ICC。特里拉内斯对此予以否认。ICC检察官办公室上周反驳所谓与菲律宾政府“勾结”“密谋”的说法，称其调查独立并遵循严格证据标准。菲律宾海军随后澄清，18人中有4人从未服役，多数提出指控者已被不光彩除名。
