其中，日本甜面包价格涨幅最大，较2020年同期高出46.1%，其次是碳酸饮料（45.7%）和饭团（45%）。
熊野指出，日元疲软与全球大宗商品价格上涨，导致日本出现成本推动型通胀，赏樱活动也难逃影响。（法新社）

荔枝新闻星期一（3月23日）报道，江苏淮安一名董姓男子在未获危化品经营、储存许可的情况下，租用停车场违规囤售33.38吨石脑油。

伊朗在霍尔木兹海峡部署海雷的行动引发全球警觉。美媒（包括CBS、CNN）报道，截至3月23日，伊斯兰革命卫队在该海峡至少布置了“马哈姆‑3”和“马哈姆‑7”型近海/底置水雷，开源情报称数量“至少一打，可能更少”。
马哈姆‑3为锚定式、具磁声传感器的水雷；马哈姆‑7为贴底“蛤钳式”低可探测性水雷，均能精确感测并在船舶接近时引爆。情报显示这些水雷处于固定位置，表明伊朗很可能掌握其确切坐标，从而把霍尔木兹由随机威胁转变为可控的战略瓶颈——可维持“安全航道”、选择性放行并在需要时迅速清除。
目前海峡并未封闭，油轮在卫队指定航线通行，但航运几近停滞，主要因革命卫队的无人机与水面无人船持续袭击；迄今无船只被水雷确认击中。尽管数量有限，小规模水雷即可造成保险成本飙升、船公司避航、国际油价波动，战略影响远超部署成本。
应对方面，美英法日等已表态参与护航与清雷。现有清雷手段包括MH‑53E拖曳扫雷、无人水下航行器、高分辨率声纳及适合猎雷的近岸战舰（如独立级LCS）；此外美方据报亦对疑似布雷舰艇和导弹掩体实施先发打击。
局势仍在演变：对伊朗而言，水雷既是威慑工具也是谈判筹码；对国际社会而言，清除并保障霍尔木兹通航已成当务之急，以避免对全球能源和经济造成更大冲击。

马尼拉讯 —— 在赢得大量消费市场并进军平价笔电领域后，苹果正进一步切入企业服务市场。 
苹果周三发布了即将推出的“Apple Business”平台预览，这是一个面向各类企业的一体化数字枢纽，旨在集中管理设备、专业沟通与客户营销。平台计划于4月14日全面上线，分为两大支柱： 
一是内部管理，便于IT部门对员工进行设备入职和在千台设备上同步分发应用；整合了可使用自定义域名的专业邮箱托管、iCloud文件同步与集中账户安全。为吸引中小企业，苹果还在同一界面内捆绑“AppleCare+ for Business”，提供专家技术支持与硬件维修服务。 
二是面向外部增长与客户获取。苹果宣布商家将能更主动管理在Apple Maps上的展示，新功能包括“自定义操作”，用户可在应用内直接预订或预约；“地图广告”（Ads on Maps）将于今夏上线，为商家提供付费触达附近顾客的新渠道。 
苹果称该平台可“一站式”管理人员与设备、拓展客户并获取支持，但指出各项功能在不同国家和地区上线时可能有所差异。对于菲律宾市场，苹果未明确说明技术与硬件支持将如何落地——该国多依赖合作经销商，而非设立本地企业办事处与服务中心。

近日，伦敦、费城等地多段松鼠握住或啃咬一次性电子烟的短视频在社交媒体走红，造成“松鼠在吸电子烟”的误读。野生动物专家指出，松鼠并非在“吸烟”，而是被电子烟中甜美的香味吸引，误以为是食物，因而咬毁或携带设备。
班戈大学红松鼠专家克雷格·沙特尔沃思表示，与散落的烟蒂不同，带有水果或糖果香味的一次性烟具对依靠嗅觉觅食的松鼠更具吸引力。
电子烟液常含草莓、芒果、口香糖、棉花糖等香精，蒸汽味接近甜点而非烟草。弃置的一次性烟具含尼古丁、锂电池、塑料和化学香精，动物咬食后可能中毒或受伤。动物保护组织RSPCA称此类事件提醒公众注意垃圾对野生动物的风险，并敦促将垃圾带至合适的回收或投放点。
这并非个例：RSPCA 2023 年记录一只新西兰鸟类吞食电子烟致死，亦有威尔士松鼠尝试掩埋电子烟的影像。尽管画面看似“上瘾”，但暂无证据表明松鼠对尼古丁成瘾，主要风险是误食有毒物质。
专家认为，此事反映人类废弃物对野生动物的隐性危害：松鼠并非自愿吸电子烟，而是在与我们的垃圾互动。
文：Saarangi Aji

菲律宾国家气象局（PAGASA）发布厄尔尼诺（El Niño）观察通知，称模式预报显示今年7、8、9月发生该现象的概率上升，若在未来六个月内发生概率至少为55%即发出观察通知。
PAGASA高级气象专家约瑟夫·巴斯孔西略博士在记者会上表示，6、7、8月出现厄尔尼诺的概率已达62%，并可能持续至少到2026年底。厄尔尼诺表现为中太平洋与赤道太平洋海表温度偏高，常伴随降雨偏少，引发干旱与旱季加剧。
历来厄尔尼诺对农业与公共卫生影响最为显著。PAGASA副局长特尔玛·辛科指出，土壤含水量下降会影响农作物，但农民已有通过调整作物种植等方式应对的经验。
气候专家普遍预测6月至12月可能出现弱厄尔尼诺，海表温度上升约0.5°C；9月至12月则有可能发展为中等强度，升温约1.0°C。
随着暖干季来临，各地气温预计升至30至40°C，呂宋北部和平地尤为炎热。大都会马尼拉4、5、6月气温或达35–39°C。PAGASA提醒民众防暑保湿，户外劳动者备足饮水，老年人避免正午高温时段外出。气象局将持续监测ENSO情况并发布必要预警，协助各部门与公众应对可能影响。
