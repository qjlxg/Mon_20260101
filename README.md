中国进入七下八上防汛关键期 官员预测防御考验会更复杂严峻
中国正式进入“七下八上”防汛关键期。官员预测，防御工作将面临更为复杂严峻的考验。
气象上，“七下八上”一般是指每年的7月下旬到8月上旬，中国华北、东北地区降水最为集中的一段时期。
综合中国人民网和央视新闻报道，中国水利部副部长王宝恩星期四（7月16日）在新闻发布会上说，中国正式进入2026年“七下八上”防汛关键期。
王宝恩也说，即日起水利系统全面启动防汛关键期工作机制，强化部长“周会商+场次洪水会商”机制，水利部机关和直属单位与防汛工作有关的干部职工全面进入防汛关键期工作状态。
中国气象水文部门预测，今年防汛关键期，中国南北方均有多雨区，涝重于旱，局地暴雨洪水多发重发，有较强台风北上影响内陆，防汛工作依旧面临严峻考验。
王宝恩指出，今年入汛以来，强降雨过程多，时空分布不均，中小河流洪水频发，极端性凸显。中国主要江河共发生25次编号洪水，有609条河流发生超警洪水，数量较近五年同期均值偏多35%，97条中小河流出现超保证水位洪水，14条河流发生有实测记录以来最大洪水。

北海道首次举行被强掳中国劳工追悼仪式
日本北海道首次举行被强掳中国劳工的追悼仪式。
据日本共同社报道，日本北海道室兰市星期三（7月15日）首次举行了564名被强掳中国劳工的悼念仪式。
太平洋战争期间，这些劳工被强掳至军需产业兴盛的室兰市，在被迫从事严酷劳动后丧生。约50名室兰市民齐聚，对遇难的中国劳工致以追思。
据主办仪式的市民团体“维护宪法的室兰地区网络”代表增冈敏三（82岁）介绍，为填补战时劳动力不足，有超过1800名中国人在1944年至1945年被强掳到室兰市，主要在港湾和工厂被迫从事搬运煤炭和铁等危险的重体力劳动。营养失调和严酷的劳动环境导致564人死亡。
增冈认为，造成平民丧生的空袭和舰炮射击的历史在室兰市被传承讲述，但“加害的历史或许没有被充分讲述”，于是带着这种想法策划了上述追悼仪式。
室兰市市民上西英子（82岁）听过被强迫劳动的中国人叙述当年的情况，在参加悼念仪式时眼含泪光地说：“同样的牺牲不能再度发生。”
今年6月30日适逢黄花岗事件81周年，日本秋田县大馆市当天在濑野公园墓地举行了悼念仪式。中国驻日大使馆公使衔参赞等人出席了仪式。
在悼念仪式上，父亲被强掳至花冈矿山的寇书兴（66岁）在致辞中强调，不能让历史悲剧重演，为了中日两国的友好与世界的永久和平，将持续不断努力。
大馆市市长石田健佑则说：“为避免重蹈覆辙，将继续向后人讲述这段历史。”
花冈事件发生于1945年6月30日，在鹿岛组（现鹿岛公司）花冈作业场被强迫劳动的中国人不堪忍受恶劣的工作环境和虐待而起义。包括起义前暴行及镇压后拷问致死者在内，死亡人数据称有400人以上。
