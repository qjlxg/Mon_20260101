中国去年铝产量创历史新高 钢铁产量降至七年来最低
中国去年的铝产量升至历史新高，突破官方设定的产能上限；钢铁产量则跌至七年来最低，凸显出两种最常用金属截然不同的发展前景。
中国国家统计局星期一（1月19日）公布的数据显示，2025年铝产量同比增长2.4%，达4502万吨；12月产量更是创下387万吨的单月纪录。相比之下，钢铁全年产量同比下降4.4%，降至9.61亿吨，自2019年以来首次跌破10亿吨关口，12月产量仅为6820万吨，创两年来新低。
铝广泛应用于可再生能源、电力输送以及电动车车身结构；钢铁是建筑业的基石。作为全球最大的铝和钢铁生产国，中国对两大行业在不同程度上实施了产量调控。
在铝行业，中国政府自2017年起设定4500万吨产能上限，以遏制结构性过剩、降低碳排放并控制高耗电问题。钢铁行业虽未设定明确的产量目标，但决策层已多次警示，长期存在的过度生产亟须化解。
在需求增长与供给受限的博弈下，铝价具备上行动能，但行业自律能否持续仍存在不确定性。高盛集团在上周的一份报告中指出，产能上限总体得到遵守，但“并非不可逾越的硬性红线”。
集团预计，到2028年铝产能仍将增加90万吨，可能导致全球铝供应过剩规模高于此前预期，并在未来几年对价格形成下行压力。
另一方面，尽管钢铁行业正收缩产量，但在中国国内需求长期疲弱的背景下，调整空间更为有限。政府虽承诺对新增产能实施更严格约束，但迄今尚未推出足以扭转供需失衡的实质性减产措施。

总统秘书处召开会议审议无现金经济与数字身份证等项目进展
2026年1月19日，总统秘书处在数字经济部长、总统阿努拉·库马拉·迪萨纳亚克主持下，召开会议评估推动斯里兰卡无现金经济的系列计划进展，会议由总统媒体处发布消息。
会议重点审议了政府机构数字化、推广二维码交易、建设云基础设施中心、全国高速宽带计划、单一窗口服务、数字身份证项目以及交通现场罚款电子化等项目的推进情况，并讨论了2025年在数字经济部下实施项目的绩效与2026年预算拟推新项目。
总统指出，农村地区大量经济活动仍属非正规部门，须纳入正式记录以便为未来经济与发展规划提供依据。
会议还回顾了斯里兰卡CERT、数据保护局、电信监管委员会等隶属机构的运行与规划，并讨论了负责落实数字化方案的GovTech机构现状及新进人员情况。副部长埃兰加·维拉拉特内、总统秘书南迪卡·库马纳亚克、数字经济高级顾问汉斯·维贾亚苏里亚等多位高级官员出席。
