海事数据库Equasis显示，嘉祥大号悬挂巴拿马国旗，但由总部位于香港的嘉祥大航运公司（Jia Xiang Da Shipping Co Ltd，音译）所有并运营。公司尚未回复彭博社的置评请求。
嘉祥大号似乎是自战争爆发以来，第二艘被发现进入波斯湾的与中国相关船只。战争爆发数小时后，一家上海公司拥有的另一艘小型散货船也被发在波斯湾内，此前报告的位置是阿曼湾。
另一方面，中国招商轮船旗下超级油轮（VLCC）“凯景”号，星期一成功穿越红海曼德海峡。这是美以伊三国开战后，首艘中国超级油轮绕道红海载油回国。
据财新网报道，凯景号装载220万桶（30万吨）原油，预计4月初运抵福建湄洲湾港。这是自美以伊开战后，首艘中资超级油轮运回中东海地区的石油。
凯景号原计划本月3日在阿联酋的富查伊拉港装载原油，但由于富查伊拉港遭到伊朗袭击，装油计划无法进行，只能绕道千里，前往沙特阿拉伯位于红海的延布港完成原油装运。
凯景号之后，招商轮船的“新先锋“号超级油轮也在延布港完成装运，中远海运集团的“新连洋”号、“远新湖”号两艘超级油轮正同时在港口装油。

本次修订将土地分区类型增至15类，新增4类用途，包括“极高密度居住区”和“森林保护区”，以明确区分城市区与自然保护区。交通网络方面，批准18条新建道路以开放新开发区，同时取消7条不符合现状的既有道路。
在环境与工业管控上，规划对有害社区的经营活动加强限制，特定区域禁设油库、气体储备库或屠宰场，并提高对工业园区与工厂的监管标准。
拉利达指出，规划旨在在打造北部经济中心与保护文化、生态之间取得平衡，整个修订过程已依法通过公众听证。内阁并已要求内政部吸纳交通部与国家经济社会发展委员会的意见，进一步完善方案，力求长远提升清迈民众生活质量并守护城市文化遗产。

马尼拉讯 — 参议院议长文森特·索托三世（Vicente Sotto III）提出法案，要求废止《燃油放开管制法》（Oil Deregulation Law）。索托表示，他怀疑现行油价是否准确公平，指出当初并未支持该法，因对决定油价依据的透明度存疑。他质问加油站或业者何时、以何价、购入多少燃油，若无透明机制，应恢复政府管制，已到废法之时。 
代理参议院议长潘菲洛·拉克森（Panfilo Lacson）承认，若暂停或减少燃油消费税，消费者最多只能省约20披索。他说，参院草案授权总统在国际油价连续两周平均每桶达80美元时，立即实施税务减免，并主张将补贴针对最弱势族群；他警告石油公司可能成为主要受益者。该案为紧急法案，总统费迪南德·马科斯二世已核定紧急性，国会可在同日完成二、三读。参院倾向仅将授权限于原油暴涨情形，且希望尽速避免两院協商冲突。拉克森并鼓励政府开拓除西亚外的油源，举台灣从美国进口80%原油为例。 
参议员伊梅·马科斯（Imee Marcos）亦质疑减免燃油特别税是否能惠及一般民众，主张一并取消增值税（VAT），以便在销售点直接反映折扣，避免优惠被进口商吸收。 
关于生物燃料修法，索托表示在圣周休会前难以通过，须评估对糖业及相关产业的影响。草案允许总统在掺混燃料比纯燃料贵至少5%时，核准为期一年之生物乙醇与生物柴油进口，并将该期间相关关税与税收收入用于国内生物燃料产业农工的社会救助方案。
