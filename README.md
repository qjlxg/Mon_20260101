告别低端内卷 本届世界杯的“义乌制造”转型
（上海／北京综合讯）2026年美加墨世界杯开赛，有“全球超市”之称的中国浙江省义乌市再度迎来体育用品出口热潮。在本届赛事亮相的义乌小商品迎来转型，与往年相比，原创设计、自主授权类产品的占比增加。
综合上观新闻与中新经纬微信公号报道，作为全球最大的小商品批发市场，义乌约占世界杯周边商品70%的市场份额。早在一年前，这里已进入“世界杯时间”，大量球衣、旗帜、喇叭、球队玩偶、球帽等周边产品从这里陆续发往全球。 
义乌海关数据显示，2025年义乌体育用品及设备出口额同比增长20.3%，今年第一季出口额达28.3亿元人民币（5.3亿新元），同比增长12%。
义乌小商品长期依靠低价复刻、代工走量抢占市场。义乌市体育健身用品行业协会会长吴晓明说，早年行业没有专利、原创和壁垒，“世界杯订单体量看似巨大，实则利润微薄”。
但从本届世界杯订单来看，义乌经营户从“拼价格”转向“拼创意”，逐渐告别过去粗放发展的模式，转而以设计、专利、品牌赢得市场。
一家从事球迷球衣行业的公司自2022年世界杯开始转型，目前所有球衣款式均为自主设计，拥有60多项相关专利，本届世界杯期间拿下的单笔最大订单出自墨西哥，总量达10万件。
另一家产品合作覆盖53个国家和地区的球衣商家称，创意化、个性化定制近几年的需求越来越多，年轻消费者不再偏爱千篇一律的传统款式，更青睐有特色的原创设计。
还有企业与体育品牌开展合作，拿下多支参赛队伍的全品类品牌授权，实现品牌化运营的跨越。
除了成熟完备的供应链，义乌的高效物流也是商家参与国际竞争的一大优势。义乌国际邮件互换局整体网络目前覆盖220多个国家和地区，小商品抵达日本最快一天就能完成。

中国国安部：境外间谍机关利用“谍龟谍鱼”窃取海洋数据
（北京讯）中国国家安全部警告，境外间谍情报机关在海龟和鱼类等海洋动物上安装传感器，窃取中国海洋敏感数据，并通过卫星向境外传输。
中国国家安全部微信公号星期五（6月12日）发文说，近年来，境外间谍情报机关通过多种新型间谍设备，持续搜集窃取中国海洋敏感数据。
文章列举这些“海洋窃密手段”，包括利用探测浮标、水面航行器、船舶电子设备等，指这些设备可实时搜集周边声波数据和中国潜艇声纹信息、通过卫星系统实时接收指令，并向境外传送与军事相关的海洋环境和舰船活动信息等。

神田透露，另有四个国家正在与亚行就援助贷款进行磋商。“预计会有越来越多国家向我们求助，情况确实非常严峻。”
